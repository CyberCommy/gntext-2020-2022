# 孤島戰爭 二

# 1

“在走之前，我們還有一些事要做。”凱林補充道,隨後走到剛才襲擊我的敵人身邊，拾起了他的武器。

“看來是一種新的毒素，他們已經得到了我父親的部分技術。”凱林對著那形狀奇怪的武器自語道。

“那是毒液槍嗎？”我問道。凱林沒有回答，也許她沒有聽到我這麼小的聲音，也許是她覺得沒有必要回答這麼顯而易見的問題。

她徑自走到一張桌子前面，桌上有各種實驗用品，我猜想生物學專家的女兒在這方面也應該有不錯的研究。

首先，凱林從槍裡推出了幾枚“子彈”，那是一些用小的針管裝著的紫色液體，透著致命的光。接著，她又從破碎的養生倉中取樣了營養液，並把毒液和營養液混進一個試管。試管裡的液體似乎慢慢起了反應，經過幾次震盪，顏色變成了淡淡綠色。

“你的身體因為這種反應而起變化是偶然的，但這種反應應該不是偶然的。”凱林的話似乎是對我說的。 “同樣的物質，同樣的條件，應該會有同樣的反應吧。”

凱林隨即將新的毒液注射個一隻小白鼠，只見小白鼠在極短的時間內陷入了昏迷，但是並沒有死去。身體也沒有明顯的變化。 “營養液似乎能使毒液不那麼致命，而且儀器顯示實驗體的能量正在迅速散失，可能電擊只是起了催化的作用，沒有電擊，長時間後它也會變小的。”說著，凱林給小白鼠的身體通上電流，小白鼠的身體果然以肉眼可見的速度開始變小，凱林沒有斷開電流，不一會小白鼠就不見了。 “看來你身上的電流及時斷開也是偶然啊。”雖然凱林說的很輕鬆，但我不禁後怕，這許多的元素，少了一環恐怕我就要說拜拜了。

“那麼，這種反應對我們有什麼幫助嗎？”話出口，我才覺得有些多餘。

“當然有了，雖然現在還不知道能否逆轉這個過程，但是至少在充滿敵人的島上行走，需要有所依仗吧，現在的你沒法戰鬥，這種新的毒素就是我們賴以防身的武器啊。”雖然我也猜到凱林會這麼回答。

唉，看來短時間內我得用這樣渺小的身軀活著了，還要依靠我要救的人，真是可笑啊。

單純的凱林沒有體會到我的心思，只是很專注地看著眼前的毒素，片刻之後，奇異的事情發生了，凱林腰間的藤蔓似乎有了意識般地伸展起來，似乎是隨著凱林的指揮，藤蔓的尖端伸進了毒素中，吮吸著毒液，不一會，毒液被吸收乾淨，而藤蔓上居然以肉眼可見的速度生出了許多淡綠色的花苞。

面對我的驚詫，凱林解釋說：“植物本沒有思考與自主行動的能力，但是寄生在我體內的植物與我的身體不穩定結合，能夠被我支配，感覺……也許就像多了一隻手吧，你看。”凱林玉手輕抬，藤蔓輕柔地纏上我地身體，將我捲起送到了她身邊。

“我們出發吧。”這回是凱林說的，也說明她已經準備妥當了。而小小的我呢，則像個裝飾品一樣，被縛在凱林左臉側的髮髻上。不明白她為什麼給我選了這樣的位置，不過這裡視野很開闊。

我轉過頭，想看看凱林的那無暇的臉，卻發現一隻水靈的大眼睛（這裡看不到她的另半張臉）也正看著我的方向。如此近的距離，三目對視，兩人的臉同時變紅了。不同的是，她是無比的可愛，而我是無比的尷尬。

“呃……那個，”我趕緊岔開話題，“能告訴我那綠色的花苞中是什麼嗎？”

“哦，”凱林回過神“那就是武器了，那些花里包含著剛才調和的毒素……”她解釋了一通，我卻沒有在意，此刻，我的腦海中全是剛才我和她對視的那一剎那，我到底怎麼了。

凱林敏捷地在樹林中閃過，涼風迎面吹來，卻吹不走進在咫尺的花香，那是我旁邊花一樣的女孩的氣息。來時充滿危機的森林此刻被陽光射穿，蝴蝶停在凱林頭上盛開的花中，又被風帶走。一切似乎因凱林而美好，她似乎本就是來自這自然的仙女。

一路奔跑，都沒有遇到什麼阻礙，卻在日落將近碰到了敵人。那是我第一次看見凱林的“武器”。

本來接近黃昏，我們找了處空地打算歇歇（雖然我基本不費力，但凱林跑了一個下午了）。卻聽到有交談的聲音越來越近。於是我們迅速在樹後隱蔽好，凱林讓藤蔓纏滿了全身，偽裝成灌木的樣子。

“那個白痴以為自己能等到救兵，怎麼會想到通訊已經被我們截獲了，那些所謂特種兵還不是都落網了。”

“也沒有都落網啊，還有一個沒找到呢，隱患啊。”

“就是，就是，真是麻煩，這麼多人居然還找不到，早點出來，也好讓我們休息休息……”

來人有兩個，都和我在避難所看到的敵人一副裝束，他們走到空地，就找了棵樹坐在樹邊休息，並且摘掉面罩，點起了煙。

很巧合的是，我們就躲在他們休息的樹後面，兩個恐怖分子還在討論，卻不只凱林已經悄然出手。

藤蔓無聲地生長，很快延伸到他們腳邊，然後迅速將他們纏繞起來。他們非常驚訝，想要掙脫，卻已經晚了，藤蔓從各個方位迅速爬滿了他們在的身體，就在這時候，藤蔓上所有的花苞居然都朝向了裡面，無數道寒光沒入了兩名敵人的身體。

“那是什麼？”我問道。

“是種子。”

“種子？”

“是的，種子，有那種毒的種子。”凱林解釋。

似乎為了印證她的話，兩名敵人此刻已經不再動彈了。凱林也撤去了藤蔓。周圍靜悄悄的，沒有其他人，我們終於敢出來休息了。

“餓了嗎？”問我的是凱林，我非常尷尬，這麼小的我，生活已經沒法自理了，還要靠本來要營救的人照顧，這讓我非常不自在，本想說“不餓”的，沒想到肚子替我開了口，雖然聲音很小，但是我掛在她頭髮上，她的耳朵就在我旁邊呢，她又怎麼會聽不到？

她把我從髮髻上解下來，放在地上，這回是自下而上地仰望著她，又是一種不同的感覺，她的身體在夕陽下變得神聖而美麗。只見她在腦後一摸，手里居然多出了一串櫻桃。

“吃吧。”她把櫻桃放在了我面前，這櫻桃對於我來說有南瓜那麼大，我根本連一顆也吃不完。我疑問：“這櫻桃那裡來的，為什麼在避難所裡我沒看見？”

“啊？這個啊，是我身上剛長出來的，不信你看。”說著，她抬起手腕，藤蔓構成的手鐲上迅速長出了櫻桃。

“你身上的植物還真是有用啊，不過，結出這個櫻桃用的是你的營養啊，你不餓麼。”

“我啊，我吃他們。”凱林纖細的手指指向一邊，讓我大吃一驚，那邊躺著的，是那兩個昏迷的恐怖分子。

“你怎麼可以吃人啊。”

“人不可以吃嗎？”凱林用天真的聲音反駁。 “父親說，對待所有的生物要一視同仁，人可以獵殺動物，我就不能吃他們嗎？”

“可是他們都是和你一樣的人啊！”我還是不能接受。

“我可不是這種壞人的同類，再說，相比其它普通動物，這些壞人更該死，更該被吃掉。”

“……”我無言以對，仔細想她說得也沒錯，人類總是那麼自大，這樣的觀點也只有像凱林這樣真正認為自己是大自然的一部分的人才會有吧。我明白了，凱林是純潔的——自然的純潔，而非人類以自我為中心的道德觀修飾出的純潔。

事實上，凱林“吃人”的畫面沒有我想像中的血肉橫飛，反而讓我面紅耳赤。

從凱林那梅花鹿的可愛小尾巴下，深出了一條觸鬚般的藤蔓，藤蔓纏上“食物”的雙腳後，抖動了幾下居然有可見的電流通過，按照我的理解這應該是一種生物發電——類似虹魚那樣。而“食物”也確實如預期般開始變小。等到“食物”縮小到只有三十厘米左右的時候，凱林停止了放。藤蔓開始拽著食物收回，我才看清楚原來那藤蔓是從她的菊花瓣中伸出來的。

雖然是鹿的身體，但畢竟也是少女的屁股啊，我才意識到到現在為止她的屁股似乎一直是裸露的，可是我看過去卻沒有一絲不安，凱林的身體充滿了自然的美感，我想就算是裸體也不會讓人有絲毫猥褻的想法吧。而事實上，凱林自己是毫不在意的。

打消了扭過頭去的想法，我繼續觀看凱林“用餐”。 “食物”已經離開了地面，被吊向上方，等待他的，將是……

如我所料，藤蔓將“食物”的雙腳拉進了凱林的菊花瓣，接著凱林的屁股沒過了他的腰，然後是胸口、頭，直到他的雙手最後消失在凱林的股溝中，“食物”沒有掙扎過，只是像熟睡一般，而凱林似乎非常受用，臉上又出現了那可愛的紅暈，畢竟她在用比較敏感的部位進食。我開始想入非非，如果有一天，我也進到那裡去……

“餵，你在發什麼愣！”凱林的聲音打斷我的意淫，“我都吃完了，你怎麼還沒動口啊。”我環視周圍，果然兩個敵人都不見了，應該是進到凱林的腸道中去了。

我頓時尷尬，馬上埋頭啃起櫻桃。凱林也不介意，臥在我旁邊，像我解釋她那奇異的進食方式。

“其實我這鹿的身體裡面大部分是空的，主要都是植物的根，中了那種毒的動物會在裡面緩慢分解，釋放的能量被我吸收，而植物則吸收分解出的物質，也就是水跟二氧化碳，用做光合作用，光合作用產生的有機物又能夠被我利用……”真的是很節能的營養方式啊，再說那種無法下口的東西也只配用那裡來食用了，等到明天早上，那兩個倒霉鬼就完全成為凱林的營養了吧。

話說我們酒足飯飽之後，太陽也下山了，晚上的樹林氣溫開始下降，雖然自己受過良好的訓練，但是這麼小的身軀也無法與自然抗衡了，以往拂面的微風對現在的我來說就是凜冽的寒風，我已經忍不住瑟瑟發抖了。

不能生火，否則被敵人發現的話就麻煩了。可是這麼冷，怎麼過夜呢？我轉頭看看臥在身邊的凱林，她的身上已經蓋滿了植物，植物晚上只進行呼吸作用，可以放熱保溫。 “真是完美的身體啊，既美麗又方便。”我想。

凱林打個哈趄，似乎有了睡意，但她終究沒忘記我，她轉過頭，注意到了我的窘境。

“冷嗎？冷了和我一起睡吧？”她提議。

“怎麼可以，這個……太……”我雖然很想接受，可是出於人類的道德觀，還是拒絕了。但是凱林似乎不准備接受我的拒絕，又用她的理論來“開導”我：“有什麼關係呢？你不要想太多了，再說你現在的樣子很可愛啊，我想是女孩的都不會討厭你的啊……”

是啊，現在的我，和凱林一樣，都不能算是完全的人類了啊，對於這樣一個單純的女孩，還是不要用人類那一套所謂的道理來玷污她的心靈了，她有個好父親啊。

我不再拒絕，凱林“以藤代手”將我抓起來，我正思忖她要將我放在哪裡，卻發現纏著我的藤蔓是從凱林的屁股中伸出來的。

“餵，幹什麼，你不會要……”

“放心啦，我不會吃掉你的！”凱林的語氣中有了少許的不耐煩，但更多的是頑皮，也許她把這個當作捉弄我吧。

也放棄了掙扎，不但因為掙扎是無意義的，也因為我相信凱林，她不會傷害我。

方才還在幻想自己如果進到這裡會如何，現在真的臨近了，我乾脆仔細打量凱林的雙臀，那是一雙很豐潤的臀，不管在人類還是鹿中都是數一數二的，她父親為她挑選這麼完美的身體一定費了不少心思。似乎為了方便我進入，凱林稍稍將雙臀張開了一些，於是我能夠看見她的菊花瓣，不同於人類，亮麗的毛皮延伸到縫隙的周圍，留空的地方是可愛的粉紅色的皮膚，“粉色的菊花啊！”我想著。

隨著藤蔓的回收，我的雙腳最先進去，然後是腰和胸口，最後，我的身體被夾凱林的雙臀之間，只有頭露在外面，凱林的皮膚包裹著我，很溫暖，很輕柔，我不得不承認這是我睡過的最舒服的“床”。這裡不是我想像中的肛門，大概是因為她不需要大便的關係吧，我沒有聞到一點臭味，只有花草的清香瀰漫著。

等我躺好，凱林似乎收了力，雙臀合攏，我感覺稍微有了一點壓力，但是更舒服，就像被擁抱著。

“你知道嗎？”凱林的聲音有些朦朧“我把你當作我的伙伴，我的朋友，你是我在這個島上除了父親之外見過的第一個人。父親愛我，我很快樂，而你不是我的父親，卻給了另一種快樂，也許那是另一種愛吧。也許將來……算了，晚安吧。”

隨著凱林的一聲“晚安”，她的小尾巴也放了下來，輕輕蓋在了我的臉上。

“另一種愛嗎？”我不禁微笑。現在，我已經不再那麼厭惡自己的身體了，也許沒有這樣的身體，凱林對我就會對一個普通的人類吧，而這種快樂，也是只有我們這樣人之間才存在的。

將來會怎麼樣呢？不去想，快樂就好了，不是麼，我在黑暗中安然地進入夢鄉。

待序……
