# 1

没有想到，自己会以这样的方式，倒下……
这个一贫如洗的年轻人，正因为没有水喝，倒在了滚烫的地面上
他渐渐失去了意识，却又在朦胧中，听到了这样的话
“四公主大人，该怎么处置这个人？”
他被救下了
在四公主的宫殿里，他得到了水，得到了休息
脏兮兮的破旧衣服，被洗干净了。虽然还是破破烂烂的。
而饥肠辘辘的年轻人，坐在地上，四公主来到了这间房间
救下他，是因为他倒下的地方挡住了四公主的去路。
现在，他可以走，也可以选择缩小，成为四公主的宠物
如果成为宠物，不可以离开她的宫殿，直到他放弃自己的宠物生活为止。
他选择了后者。
今天是他与高贵的四公主大人相处的第一个晚上
回到宫殿的四公主，很温柔的让他知道了自己的处境
可以安心，随时可以离开，可以继续流浪
但是在这里，就必须以50cm的大小，并且对四公主作出对主人的礼仪。
今天的四公主，独自一人吃着一块蛋糕
她吃了五分之四，樱桃不是很好吃，便吐了出来
精致的小盘子，放在了地上
四公主的小嘴咬过的蛋糕，上面带着可爱的小压印和反着粼粼微光的唾液
嚼了一下的樱桃，泛着湿润的新鲜液体
桌下的他，爬着冲了过去
而四公主的裸足，则踩在了他的身上
脚后跟压着他的肩，脚心按着他的头
他苦撑着
原来，刚刚四公主说，“如果被踩到，一定不要让身体碰到地上，不然会受到惩罚的哦，请格外注意~”
是这样的意思
他苦撑着，看着眼前，吃剩的蛋糕，和因为使劲下压而蜷缩的脚趾
好重，真的好重，他艰难的苦撑着
没有人看到，四公主，正调皮的笑着呢

她使劲一用力，男孩应声倒下
不过，所谓的惩罚，并没有到来
他还是吃到了蛋糕
害怕被惩罚的他，主动的提出刚才的事情，道歉的时候
四公主温柔的用脚摸了摸他的头顶
“傻孩子”
这是一个八岁的主人，对十八岁的宠物的，爱的回答。
