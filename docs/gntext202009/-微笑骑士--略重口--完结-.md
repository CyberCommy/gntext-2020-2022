# 《微笑骑士》（略重口）[完结]

# 1

漆黑的重型盔甲，肩扛着一把半人多高的宽刃巨剑，厚重头盔的面甲上，纹着一个温暖的微笑。

他来自北方遥远的国度菲尔利文，凡是见到他的人，都叫他微笑骑士。

眼下，他正要穿过眼前的森林，继续自己的旅行。

突然，微笑骑士看见了前方森林间的缝隙，一阵阵冲天的浓烟，遮天蔽日。

想要看看发生了什么，随即往浓烟的方向走去，在一颗大树旁，他看见了一位面色焦急的少女。

看到了逐渐走近的黑甲骑士，她急忙跑过去。

“你好，骑士大人，请问你可以帮我一个忙吗？”

微笑骑士看着她无助的脸庞，点了点头。

“一头红色鳞片的巨龙昨晚摧毁了我们的城镇，很多人都死了。我的哥哥是本镇的守护骑士，他不久前带着镇上幸存的守卫，去深山中讨伐那头巨龙了。可他是一个白痴，根本不可能对付得了恐怖的巨龙，所以无论如何，我希望你能把他带回来，求你了。”

微笑骑士再次点了点头，表示同意。

“哈...太好了...”

少女急忙撕下了黄色裙摆的一角，放在了微笑骑士的手里。

“这是约定的信物!现在我带你去营地找找还有什么用的上的。”

牵着微笑骑士，少女来到了城镇废墟旁的营地里。

在这里，劫后余生的镇民们面无表情的静静坐着，一具具残缺不全尸体摆在旁边，被裹尸布盖着，整齐的排在一边。

少女带着微笑骑士先是来到一座低矮的哨塔旁，整个帐篷搭建的临时营地都是围绕着这座哨塔建立的。

在哨塔里，微笑骑士看见了一个肥胖的男人，对他和少女笑脸相迎。

在少女向他讲述完微笑骑士答应帮助并且约定的经过后，他轻捋着胡子点点头。

“哈哈，骑士先生，既然你答应帮助我的儿子讨伐巨龙，那么我就册封你为本镇的守护骑士!去找药剂师和武器商吧，他们会给你提供力所能及的帮助!等你们讨伐完巨龙得胜归来，我会把相关的身份文书给你!”

说完，他立刻拿起笔和纸，趴在他面前宽大的木桌上写了起来。

微笑骑士点了点头，表示答应。

在少女的陪同下，微笑骑士先来到了药剂师的帐篷。

“啊...这样啊，那就拿着这个吧，这是我最后一瓶恢复药了，其他都给受伤的人喝掉了。”

药剂师拿出一瓶恢复药递给了微笑骑士。

“太感谢您了!”

少女激动的对着他鞠躬。

“哈...没事没事，这是镇长提前叮嘱好的，讨伐巨龙能帮上忙我还是乐意的。”

两人来到了武器商人的帐篷前，此时，武器商人正站在帐篷口仔细端详着一个盾牌。

少女说明了来意，武器商人表示理解是点点头，随后将手中的圆盾递给了微笑骑士。

“巨龙摧毁了一切，我也只剩下这一块盾牌了。就把他交给你吧，这可是我的得意之作，希望它能派上用场。”

点头收下了盾牌，微笑骑士把盾牌背在背上，和少女离开了。

来到营地的边缘，少女为微笑骑士指出了前往巨龙巢穴的方向。

“我哥哥戴着一个绿色的头盔，拜托你了，骑士大人。请一定要把他带回来啊。”

微笑骑士郑重的点了点头，带着人们给予的东西，扛着大剑踏上了征途。

穿过幽深浓密的森林，穿过幽暗的峡谷，走上了崎岖的山路，终于，在接近山顶的地方，微笑骑士看见了一个巨大的洞口。

一阵阵恶臭扑面而来，微笑骑士看见了成堆的粪便和夹杂其中的森森白骨，这些是巨龙的排泄物，看来这里就是巨龙的龙巢了。

扛着大剑走进去，在宽阔的山洞中走着，转过一个遮挡视线的岩壁，微笑骑士终于看见了那头巨龙。

一对红色的巨大翅膀收拢着，雪白的肌肤，身上的隐私部位被红色的鳞片所覆盖。此刻，她正用细长的龙舌卷起一个身穿薄薄布甲的可怜家伙收进嘴里，然后一脸享受的微微仰头吞了下去，伸出手轻抚着自己微微隆起的肚皮。

她的面前，各种兵器和盾牌散落一地，看来那个少女的哥哥和他的手下们已经沦为她腹中的食物了。

把大剑插在地上，微笑骑士拿起了武器商人给他的盾牌挡在身前，然后重新拔出大剑，严阵以待。

听见声音，她也慢慢低下头，看向不远处小小的微笑骑士，暗红色的眼睛眯了眯，充满了凌厉之色。

大战一触即发。

对巨龙来说非常狭窄的洞穴，不可能直起身子迎战，所以只能用龙爪进行攻击。

巨龙被红色鳞片包裹的大手挥过来，微笑骑士立刻往旁边用力一跃险险躲开，但还没站稳，巨龙的另一只大手瞬间而至。

来不及躲闪，微笑骑士急忙举起盾牌抵挡，但龙爪接触到盾牌的一瞬间，脆弱的盾牌瞬间破碎。

完全没有意料到这个盾牌出奇的脆弱，微笑骑士结结实实的挨了巨龙的一击，直接倒飞出去，重重摔在地上。

挣扎着缓慢的爬起来，微笑骑士低下头，看着已经残破不堪的胸甲，要不是这厚重的盔甲，刚刚的一击足以致命。

因为失去了盾牌，回归双手持剑的微笑骑士活动了一下手腕，重整旗鼓举起大剑，剑刃再次对准了巨龙。

双方再次碰撞在一起，这一次，微笑骑士连续躲过了数次巨龙的攻击，然后对着她的右臂一剑挥下。

“吼!!!”

巨剑造成的巨大撕裂伤口瞬间让巨龙痛苦哀嚎着，巨大的身体碰撞岩壁剧烈震动着，不断有碎石落下。

巨大的翅膀合拢着，巨龙向洞口爬去，她想要逃到洞穴外面去。

但微笑骑士抓住机会，经过他身边时，用力一跃直接抓住了她臀部的鳞片爬了上去。

举着大剑踩着她柔软的腰部，微笑骑士奔向她巨大的红色双翼。

在接近洞口位置的不远处，微笑骑士对着她翅翼的根部，一剑劈下。

龙血飞溅，巨龙痛叫着，在距离洞口仅仅一步之遥的地方，无力的重重倒下。而微笑骑士也因为巨大的惯性飞了出去，摔在她的面前。

挣扎着迅速爬起来，浑身是血的微笑骑士喘着粗气，再次将剑刃对准了巨龙。

失去翱翔天空双翼的巨龙，此时已经不能称之为巨龙了。

“我...我认输...”

看着眼前的微笑骑士，巨龙眼里此时只剩下了恐惧。

“我愿意...臣服于您...我会改邪归正...绝不会再攻击人类...”

微笑骑士喘着粗气，听见她的话后，最终还是垂下了剑刃，走到一旁把巨剑用力刺进地面，然后靠着岩壁坐下。

胸口的伤口正在流血，微笑骑士想起了那个恢复药，随即将其拿出来。

望着那瓶子里晶莹的液体，略微思索了一下，还是喝了下去。

一股温暖的感觉传遍全身，微笑骑士感觉自己伤口的疼痛正在减弱。

缓缓松了一口气，又休息了一会，正当他想站起来时，一股深深的眩晕感却突然传来。

药剂被做了手脚。

双眼的视线变得越来越模糊，微笑骑士下意识的拿出了那裙摆的一角看了看，随即失去了意识。

不知道过了多久，当昏昏沉沉的微笑骑士醒来时，环顾四周，那头巨龙已经不见了踪影。

扶着岩壁坐起来，微笑骑士拔出剑走向洞口。

在洞穴外，微笑骑士看见了一堆新的粪便，而其中一处什么东西正泛着微微的光芒。

走过去抓住那东西的一角，然后用力拔出甩了甩。

这原来是一个造型怪异的绿色头盔。

拿上头盔，微笑骑士踏上了归程。

回到了密林中，微笑骑士看见松软的泥土道路上，每隔数米就有一个巨大的足印，超前而去。

当他回到被毁小镇旁的营地时，那里也已经被完全摧毁了。

遍地都是密集的交错在一起的巨大脚印坑和被踏的扁平的尸体。微笑骑士走到一个被踏扁的帐篷旁掀开，这里本来是药剂师的帐篷。

看着一地粉碎的瓶子，从中流出的液体已经干涸了，看瓶子的样式，和给自己的那一瓶恢复药一模一样。

摇了摇头，微笑骑士走向了武器商人的帐篷。

不出所料般，帐篷里摆着数面坚固的盾牌，甚至在巨力的踩踏下都没有损坏。

起身望向四周，微笑骑士看了看周围，想要看看还有没有别的有用的东西，结果却看见了已经倒塌短成两节的哨塔。

绕过数个大坑过去，在废墟的砖石间，微笑骑士看见了那个木桌。

走过去，把上面的碎砖石推到桌下，那张残破的纸漏了出来。

拿起来看了看，上面写着镇长的署名，但不是什么册封骑士的文书，而是一份通缉令，描述罪犯的内容就是头盔上画着一个微笑。

放下纸张，微笑骑士发现旁边还有一个小小的皮本。

打开看了看，上面的内容只是日记般繁琐的日常。

但翻到最后一页时，上面只有简单的几句话。

(太好了，机会来了，我那个傻儿子终于去送死了，哈哈，不可能有人斗得过巨龙!那个没有脑子的蠢货平时都在保护他那个贱人妹妹，现在他终于完蛋了，没有他护着，我终于可以品尝她的味道了...）

合上皮本放在桌上，扭头看向周围。

在营地里一具具尸体仔细的寻找着，终于，在森林的边缘，他找到了他要找的。一具身穿着黄色裙子的扁平尸体。

跳到脚印的大坑里，微笑骑士郑重的把她给自己约定的信物拿出来放在她的尸体旁，然后用绿色的头盔压住，防止被风吹走。

从脚印的大坑里爬了出来，微笑骑士回头看了一眼已经毁灭的小镇，扛着大剑再次踏上了旅程。
