# 1

**本文在紫心莲华yuxiaoqiu的授权和背景提供下完成
**主要人物：****莉莉丝.安琪拉多（莉莉丝大人，莉莉丝酱） ，太平，幽野社（小社），******琉璃** 琉璃蓝 太平绿 幽野社橙 莉莉丝红
如同宫殿一般的场所里，不像从外界看起来那般的神秘，内部的装饰虽然辉煌，却有几分乏味
乏味就像是种气味，弥漫在空气的气氛中
幼小的身影又一次软塌榻的在那里，不知道该怎样打发太过漫长的时间
“莉莉丝大人又在发呆了吗？最近好像特别的不精神呢……”
琉璃关心的语气让女孩抬头看了看，女孩看到的却是太平，心里小小的惊讶了一下，但继续说着心里想的话
“可以做的事情虽然有好多，想做的事情却根本没有”
“莉莉丝大人，您该找点爱好。”
“上次的那盆【盆景】，最后让我很恶心了呢”
莉莉丝皱了皱眉头，她想着之前发生的事情，不对任何人说着
“人类，真是太恶心了”
“对了，太平，你最了解人类，你有什么好办法吗？”
“嗯。莉莉丝大人，您可以这样……”
【此处省略若干】
一个类似于花盆的东西，四周有10厘米高的透明板，现在正安置在莉莉丝，琉璃和太平的面前。
“里面都是些小虫子呢”
“和上次一样，1000人的小村庄，25厘米的花盆。”
“莉莉丝大人，我可以给这些虫子们展示一下女孩子的力量吗？”
“请随意~”
琉璃甩掉凉鞋，白白的“小脚”伸向这群人的头顶 “等等，莉莉丝大人，我有一个好办法。琉璃，你玩的时候可以先让莉莉丝大人把人群暂时去掉吗？”
“唉？那还有什么意思？”
太平很吃力的聚气了一下，绿色的隔音光圈就罩在了花盆上
“让他们看着你去踩烂他们的家园，再把这些松软的土地踩成难以耕耘的硬土，莉莉丝大人就可以扮演救世主角色了”
“嘻嘻，好主意”莉莉丝拍了拍太平的肩膀，终于露出了已经一天没见的笑容。
光圈的时间刚好用光了，莉莉丝连法杖都不需要，抬了下手指，一千只蚂蚁人就被悬浮在了空中
急不可耐的琉璃甩掉两只凉鞋，立刻踏上了花盆，伴随着灿烂的笑容和小人们的注视，两只白白的“小”脚丫就对着小村庄一顿乱踩
村庄的中心严重的凹陷下去，琉璃高高的抬起砸出一双脚印形状的深坑后，满意的走出了花盆
“哎呀，脚弄脏了”
“辛苦了，琉璃酱~”
然而这一千多人已经完整的看完幼女踩烂他们的家园了
琉璃又把这盆盆景放到了一个阳光最好的阳台，夏季最明媚的阳光不间断的直射在这个“盆景村庄”中。
对于这些男女老少的村民来说，今天是恐怖和悲伤的一天
在阳光的暴晒下，看着被践踏殆尽的家园，他们很快的发现了一个大事。
那就是，没有水源。
酷热和失去房屋，让不少老人很快中暑
一千米的大脚印，踩出的印记里包含着不少的褶皱
一个脚趾就有上百米
两三米的巨型褶皱印记下，人们勉强的遮蔽着阳光
有些人在哭泣，有些人在怒骂，有些人在彼此争斗着仅存的一点点水
庄稼都毁了。然而本身就已经被缩小的村民们根本来不及顾及。
酷热的一天艰难的度过后，一千多人在夜晚忍饥挨饿。
第二天，不少人都已经无法再坚持下去了
太阳，似乎比昨天更加毒辣
虽然仅仅是他们的错觉。
有些人已经死了
被蒸烤干的干尸，被热死的人
绝望的气氛越来越浓重了。

咚……咚
是越来越近的声音
庞然大物出现了
巨大的，小孩子。
世界的主人，正看着这盆“盆栽”
她忍着笑，但还是忍不住了
扑哧的一声，笑的比想象中还要大声
淡粉色连衣裙的女孩子，莉莉丝看起来就像纯真无邪的天使
她抬起自己的裙子，横看胯在花盆上方
让自己淡粉色的裙子笼罩花盆
人们的天空被粉色笼罩，而头顶，则是小熊图案的内裤
内裤很快被脱了下来，人们都在抬头看着
莉莉丝并不会对这样的场面感到害羞
人类，尤其是这种处境的人类，她并不在意的。
就像上厕所那样的自然
娇嫩的下体，娇嫩到了难以想象的程度的可爱
金黄色的液体从上面倾泻而下
正下方的人们被水流冲走，而有些渴疯了的人却疯狂的朝尿潮冲去
不少人，都不知道该用怎样的心情去面对
面对自己靠喝尿为生的处境。
其实他们不知道，能够品尝到莉莉丝尿液的味道，是怎样的光荣和幸福。
少数人，还是在无知的咒骂着。
但更多的是在喝着
但很快，尿液就渗透了下去。
只剩下零距离的尿味
距离日落还有四个小时
加上漫漫长夜，有些不愿意喝莉莉丝的尿的人，已经渴死了。
早晨，莉莉丝惊讶的看到了一幕
大约三百多人，整齐的跪在地上
齐声高喊着
“欢迎主人”
虽然有一百多人或怒视，或东倒西歪的
莉莉丝感到了惊喜，有点兴奋的跨立在花盆上方，尿出积累了一夜的尿液
看着被冲得七零八落的人们，莉莉丝有种饲养宠物的感觉
心情还不错。
一个十五岁的女孩子，捧着手里的尿，尿反射着她狼狈凌乱的面孔
她把尿用力甩在了地上
发疯了一样的冲向了一块大石头
撞死了
而旁边的两个男子，却为了奸淫尸体而打斗着
充斥着莉莉丝尿味的这里，真的是天堂吗？

莉莉丝走出阳台，这次的盆景活动并不像上次那样让她恶心
上次的时候，盆景是直接安放在屋里的，既没有故意踩烂他们的房屋，也没有恶意的暴晒。
而且莉莉丝一上来就去“浇水”了。
但当时人们的反抗和咒骂，却又一次在莉莉丝脑内想起。
反倒是这样，却让人们的反应截然不同。
“人类，果然是愚蠢的呢”

“第几次了？”
“什么第几次？”
“那个小孩尿尿”
“三次？四次？记不清了”
在暴晒下越发积累和浓缩的尿味越发浓郁，
不知道是尿味熏得人们头晕，还是暴晒和高温的湿热让人心烦
还是有些人在倒下后再也没起来
但死亡率低了很多
现在还有将近400人吧
很快，人们又纷纷跪好“求雨”了
莉莉丝的尿液，成了这群人生存的支柱
有些人已经接受了这样的命运
一辈子在女孩的尿液支持下生存
跪倒在巨大小孩子的胯下

七天后
莉莉丝两天来，只去了一次阳台
兴趣已经失去了呢。
她找来了琉璃，太平和小社
“其实，有些时候，”
“怎么了莉莉丝大人？！”
“不要打断我”
莉莉丝顺手在小社身上做了个打断的动作
“因为某些事情，我觉得被人崇拜，被人需要的感觉还不错呢”
“是因为阳台的那个马桶吗？”
“是盆景！是盆景啦！！”
“可是莉莉丝大人自己都会说【上厕所】，然后往阳台跑了嘛”
“啰嗦！！”
在【教训】了调皮的两人后
“那个盆景，已经没用了，味道重，又麻烦”
“嗯？”
“唉？”
“你们三个陪我一起处置了他们吧，也算送他们上路”
“莉莉丝大人变得越来越善良了呢？”
莉莉丝红着脸，【用力】打了小社一下
“今天莉莉丝好暴力哦”
“啊。。烦死了，走吧”

干得快要死掉的二百多人，在龟裂的土地上苦苦支持着，等待着他们心爱的女神--莉莉丝的降临
然而，四个巨大的女孩同时出现在了他们的天空
“谢谢你们陪我玩哦，大家再见！”
“到最后不还是要玩死嘛，哈哈”
“做莉莉丝大人的玩具本身就是危险的……”【她流着冷汗说着】
“哈哈，我想起小时候用尿淹死蚂蚁了”
“嘻嘻”
她们齐齐脱下下身的衣物，露出形态各异的幼小下体
伴随着天籁纯净的童音，天真无邪的笑声
金黄色的水柱从四个方向合围喷射而来，大量的水注满了花盆
尿液的洪水将他们的【地面】彻底淹没，水位甚至突出了花盆，打到了透明的隔板
有些人沉底了，有些人在水面上漂浮着
尿注又将他们打进水底
“要不要把全世界都变成这样的厕所呢？”
“你这个小笨蛋，那样岂不哪里都是尿味了”

莉莉丝释放了一个法术，从这时每一个死去的人的灵魂都会被封印起来。
四个女孩走后，大水渐渐下降着水位
但一直不见底
人们在水面做着绝望的挣扎
直到被尿液吞没
傍晚的时候，莉莉丝带着她的三个仆人，又来到了这里
“怎么只有一百多个灵魂？”
“等等，我听到了什么？”
水面上，还有人活着
“主人，求求您放我一条性命吧”
“女神，救救我”
“主人，我愿意一辈子做您的奴隶”

莉莉丝犹豫了
她攥了攥拳头
“你们还有存货吗？淹死他们！”
“看看是虫子们厉害，还是我们的尿厉害”

第二天早上
莉莉丝来取灵魂的时候，发现还有十多个人没有死
她拿来了一根筷子，把小人按进水里淹死
却有最后一个人，死死抱住了筷子
“松开！”
那人便松开了
莉莉丝再按，那人还在挣扎着起来
“不许反抗！”
那人便不再反抗
莉莉丝准备把他彻底按进水中
但，她最终没有这么做
“上来吧”
洗干净那人后，莉莉丝对着跪地的那人问
“你叫什么名字？”
“报告高贵的主人，我的名字叫冰西瓜”

莉莉丝带着二百多个灵魂，定制了一双精致的凉鞋，凉鞋分别由一百块精密的天权石组成，莉莉丝将这二百个灵魂封印在了每块天权石中，保留着每一个灵魂的五感，莉莉丝迫不及待的穿上了这双鞋
鞋的主人还可以知道这些灵魂的思维。二百多个灵魂在脚底重压下的感受，莉莉丝如果想的话，随时都可以知道。
她跺了跺脚，感觉这双鞋有种特别的舒适。而在别人看来，莉莉丝，世界顶点的真神，是如此的可爱和高贵。
