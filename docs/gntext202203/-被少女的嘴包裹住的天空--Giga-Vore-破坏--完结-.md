# 1

近些年来，随着经济社会的发展，出现了许许多多的超级城市。

而海汐市，就是最早的一批超级城市，也是迄今为止最大的一个。虽然这座城市周围并没有海，而且整个城市都几乎是在山脊之中建造的。周围就是千米之高的山脉，耸入云端的山脉，使出入这座城市变得极其困难。但是，不便的交通并没有阻碍这座城市的发展，反而将这座城市的规模推到了世界顶尖的地步。

那是一个平和的周日，即便是在炎热的夏季，这座城市依旧凉爽。河岸边，随处可见成群结队前来戏水的人。大楼间，时而也有悠闲着交谈家事的朋友。街道上，熙熙攘攘的人群旁不时有几辆汽车穿梭而过，所有人都已经习惯了这种慢节奏的幸福生活。

“轰轰轰——！！！”

忽然，一阵突如其来的地震伴随着雷鸣般的巨响传遍整个城市。向着地震的源头看去，人们只看到环绕着城市的山脉，被无数根垂直指向地面的红色柱体贯穿。方才的响声，也是这些柱体压在地上，碾碎山脉发出的巨响。

更可怕的是，这些红色柱体竟有一根落在了城市里。但奇怪的是，明明成群的高楼被压在下面，却只是发生了轻微的倾斜。

如果是从远处看，根本无法知晓这些柱体究竟是什么。但是，有了近距离观察的机会，人们自然而然也能辩识出这柔顺无比的柱体究竟为何物。

周围忽然变暗，人们猛然抬头仰望天空，在那遥远到就连云层都消失了的高空，只见一张和背景融为一体的脸，突兀地代替了原先蔚蓝的天空，进入了人们的视野中。

那一路扫平山脉，压在城市上方的柱体，原来只是少女柔顺的红色长发而已。

隐约之中，人们看到少女在笑。

随后，少女的脸颊骤然靠近，整座城市都被黑暗所覆盖。一切光线都被少女匀称的脸部所遮挡，人们依稀记得最后看到的景象是——少女的双唇首先突破大气，嘴唇的本色也能看得一清二楚。在空中不断扩大的嘴唇，仅用了不到一秒就覆盖了整个城市。随着距离的拉近，这娇艳双唇的巨大渐渐突破人们的想象，这座占地诺大的超级城市，即便有着以百公里计量的纵横，却在一瞬间就被少女的嘴唇遮盖住。更巧的是，城市附近山脉的形状似乎正好与嘴唇的轮廓相契合，似乎一切就是专门为了少女双唇的降临而设计的。

这下子，所有人都无法再抑制心中的恐惧了，骚乱在城市中如病毒般扩散，涌上街道的人们四处逃窜，却由于过度昏暗的环境而屡屡摔倒。驾驶着车辆的人，则四处乱窜，就算撞到了什么东西也无法让他们停下，即便知道前方就是排成长龙的车队，却还一遍遍撞击着前车。人类构建千年的社会秩序，竟因为少女双唇的来到而彻底土崩瓦解。

“呵呵呵~~~~”

少女默不作声地看着人类不惜动用一切武器，付出这么巨大的代价，只为了赶走她这个入侵者。

几枚导弹零散地打在少女的唇上，其中一颗还算得上是核武器。然而，这些一枚就能毁掉一座普通城市的超级武器，只在少女唇纹的深处留下了一点印记，就连烧穿她的皮肤都做不到。

见到小人们已经穷尽了所有的攻击手段，接下来自然就轮到少女的回合了。

就在少女的舌尖即将完成环绕嘴唇一周的壮举时，来自少女的惩罚突然来临了。

“咚——！！！”

一团团粘稠的唾液不断向着四周扩散，冲碎沿途的高楼，裹挟着建筑碎片的唾液洪流毫无阻力地推平一切，甚至还夷平了好几座山丘，直到横断的大山将其拦下，少女的口水造成的灾难才没有进一步扩大。

刚刚，少女舌尖覆盖的位置，被一个深度达二十千米，直径也有五千米的陨石坑所代替。向着深不见底的坑中望去，致密的土石中隐约的建筑碎块以及人类血迹像是在诉说着那些人生前的痛苦。在那粉色的巨兽落地的一瞬间，地面被彻底撕成了两半，一半黏在了少女的舌尖，另一半则永远留在了地底。

真是可怜，想必他们都没反应过来吧。

少女并非毫无原则，在毁灭一座城市之前，她都会公开地宣布这座城市的命运，以便让小人们有所准备。然而这一次，也许是由于过度激动了，她竟在没有作任何宣言的情况下，夺走了千万人的生命，这让她倍感愧疚。

“没有预先说明真是抱歉，但接下来#1F市，以及可能被卷入的大家…”

面对死亡，每个人都是平等的，而少女就是那个带来死亡的人。

时间一分一秒地流逝着，少女看着城市里的人依旧没有动静，感到有些奇怪。往常的话，人们应该已经簇拥着跑上大街，仓皇地想要从她的唇下逃走了。

显然，少女并不知道，她垂下的长发已经摧毁了所有的道路，就连航路也被她的头发所遮挡，已经有上百架飞机因躲闪不及撞在她的长发间。大江大河全被她垂下的红发截断，就算所有人都长出翅膀，也难以在一分钟之内就从少女的唇下飞出。

一分钟很快便到了。

少女如约张开嘴唇，让嘴里的热气尽情地冲刷着大地。人们看向天空，那两瓣紧紧贴着的樱唇已然分离，露出一个深邃的红色洞穴。所有人都知道，那就是少女的嘴——用来进食的场所，而他们每个人都是少女的食物。
