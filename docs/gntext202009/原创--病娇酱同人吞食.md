# 1

今天，是绫乃转入新学校的第6天

每逢春色烂漫的时候，那棵樱花树就会伴随清风，纷纷扬扬撒落下无数樱花，化作芬芳送走黄昏的那片晚霞

自从某一天在这棵樱花树下与学长意外地邂逅，一见钟情的绫乃酱就疯狂地爱上了学长

来这里的上学的新生都是结伴出行，一路打闹着跨进校门，在茫茫的人海中，那棵樱花树站立着一位黄发双马尾的清纯少女。她微微抬起头，睁开双眸注视那樱花飞扬的瞬间。她就是学长的幼训染，绫乃酱最大的情敌，此刻正期待着与学长再次相遇。

“早上好，绫乃酱。”一声稚嫩的问候从绫乃身后传来，她移开了对幼训染的注意，朝着身后望去。是一位名叫miyu的蓝发妹子，她就是绫乃酱的第一位“好朋友”

很快上课铃就被打响，学员停止了嬉笑打闹，陆续走进教室

“放学的时候记得等我！”绫乃酱一手搭在miyu肩上面带微笑

“那好吧 拜拜！”miyu答应后就离开了绫乃身边，走向自己的班级

望着miyu渐渐消失的背影，绫乃酱的嘴角不由地微微往上扬

“你这个小傻瓜……”

——————————————————————————

终于放学了，学员们结伴走出校门。等到几乎所有人都离开了校园，绫乃酱这时才直奔向医务室。那里四下无人，于是她打开最近那张桌子的抽屉，警惕地翻找着医用器具。只见她将一支针管和几瓶药剂塞进了口袋里，就飞奔离开了医务室

教室门外，miyu独自一人守候在这里，不耐烦地瞥了一眼手表，正要抱怨起来。这时，绫乃酱突然现身，紧紧抓住miyu的手不放。“喂！”miyu被吓了一跳，“你在干什么，好疼的！”

绫乃脸色突变阴笑道：“今晚跟我回去吧！”说着掏出麻醉针，狠狠扎在miyu光滑的手臂上

“哎呀——”miyu只感到一阵钻心的疼痛，渐渐意识模糊起来，昏倒在地上。绫乃将她抱起，带到自己的家中。

沉睡许久的miyu稍稍恢复了意识，在朦胧中打量这周围的一切，“我……我在哪……”一阵恶寒拂过miyu，她马上就清醒过来，由于全身被麻绳绑在椅子上，完全没有办法动弹。

“呯——”一脚踢开罐子的声音打破了这片宁静，那扇木门后，绫乃酱从阴影中走出，一步一步逼近miyu

“绫乃酱，你这是要做什么!”miyu出于胆怯而低声问着。

“我？我当然是要杀了你！”绫乃狞笑着一把揪起miyu的头发

“你要是杀了我，早晚会被抓获！”miyu十分气愤

绫乃听到了这番话后却大笑起来：“你以为我会直接杀掉你吗？来吃了这个！”说着将一粒白色小药丸塞进miyu口中，miyu只感到眼前一黑，昏睡过去。等到醒来时，miyu感到自己不再被束缚了，便站了起来，正觉得不对劲，抬头一看“绫乃酱！为什么你变得好大！”此刻miyu正被绫乃捧在手上

“傻瓜”绫乃微微一笑“死在我的腹中，谁也不会知道吧”于是张开血盆大口，将miyu送入其中

“不！！”miyu整个身体都被塞进了口中，在这潮湿的口腔中挣扎，不过口气很清新，以不至于让miyu感到闷热。

绫乃细细品尝着稚嫩的miyu，除了校服上洗衣液的味道，就只剩下miyu那甜美的体香。被鲜红有力的舌尖所触及的每一寸肌肤，都沾上了粘稠的唾液。

绫乃潮湿的舌头又将miyu顶到一边，用牙齿轻轻碰了一下miyu，富含弹性的肌肤犹如软糖，口感美妙到了极致。

“我可不想让你死的这么轻松”绫乃心想着，就干脆把miyu往下咽

miyu感觉到自身在被挤向身后的深渊，竭力抱住舌根。可是在唾液的润滑下，miyu还是滑进了狭长的食道中

在食道壁的挤压下，miyu几乎要失去意识，只感到自身在这管道中不断地往下滑。短短的几秒，对miyu而且仿佛是千万年。miyu脚下的贲门缓缓张开，“扑通”一声，miyu就掉进了胃袋里。这里到处是胃液的酸臭味和食物的残渣，粉红的胃壁上挂着几丝晶莹的黏液，胃壁慢慢地蠕动着，miyu好不容易才站住脚。小腿下方都浸泡在胃液中，滚烫的感觉刺激着双腿。miyu忍着痛环顾着，只见不远处一块没被消化的胡萝卜正浮现一半在胃液面上，miyu连忙攀爬上去。“好大的胡萝卜，绫乃一定没有细细嚼。”miyu感叹起来。好在双腿没有浸泡太久，皮肤只是微微发红。

“你好自为之吧！”绫乃的声音从上方传来。“我先睡了！”

miyu冷静了下来，如果一直在这里待下去早晚会死。

“绫乃，你居然这么狠毒！”miyu愤怒地一拳打在柔软的胃壁上，拳头被软肉弹了回去。突然整个胃里摇晃起来，miyu好不容易才站住脚。原来绫乃也感到了一点不适，翻了个身接着睡。

miyu擦了擦汗液，平静下来，发现暴力并不能解决问题。而绫乃现在睡着了，刚好有机会逃出去。

miyu仰望着闭合的贲门，伸手在软滑的胃壁上揉了揉，捏了捏。

“这也太滑了，没有办法攀上去”miyu的内心几乎再次陷入了绝望。

“咕噜——”四周的胃壁猛然挤来，压住了miyu娇嫩的身躯，她紧紧抱住那块胡萝卜。胃壁随之又舒展回去，miyu看见，一些被消化地差不多的残渣都流进了胃底的一个小洞。

“难道我要从那里……”miyu突然感到脸颊发烫，两手抱住头“可是……如果不这么做，我就会死掉……”

求生的欲望总是胜过一切，幽门张开之际，miyu猛跨一大步飞跃过去，一头栽进幽门里，在幽门即将闭合的瞬间，被周围的肉壁挤进了小肠中

“哎！——”这突如其来的冲击让绫乃感到一阵疼痛，一手捂住肚子忍耐着。

这狭窄的小肠十分拥挤，小肠绒毛也有些油腻，这让miyu更加不适，只能任由肠壁缓慢地蠕动推动她前进。“这样下去要等到什么时候嘛！”miyu竭尽全力用双臂撑开肠壁，两脚踩在肉壁上往后一蹬，“叽咕”一声往前移动了不少。这一举动让绫乃感到肠道中仿佛有一个庞然大物经过，疼出了少许汗液。

小肠的长度是人身高的好几倍，而且miyu现在又十分的小，想要通过消化道可不是容易的事。miyu终于累的难易支撑，在肠道中睡着了。

绫乃感觉到肠子里没有了反应，心想她一定是死了，于是安心睡着了。

“咕嘟”小肠的蠕动惊醒了miyu，她擦拭着手表面上的肠液

“已经是凌晨了吗，如果绫乃一觉睡醒就不好出去了。”后面的软肉再一次挤压，miyu掉进了大肠里，没有如何防备就重重摔了下去。毕竟大肠壁要厚实多了。

大肠里弥漫着腥臭的味道，伴有少量的污物积在滑嫩的肉壁上，可以看清肠壁上一丝丝血脉。

miyu捂住鼻子，在这空旷的大肠里奔跑起来，这里可没有小肠里那么多的拐角。跨过一节节阶梯般的肉环。拐过几个弯道，污物逐渐越来越多，不过没有挡住去路。肠壁渐渐平缓，“这里就是直肠了！”miyu的心中又充满了希望，望前面跑过去，来到了另一个出口。

miyu望着眼前粉嫩的洞口闭合着，试探性地将手凑上去用力扒。好在绫乃已经睡着了，菊花稍稍有些松弛，勉强扒开一个小洞。miyu将身体挤过去，半个身体钻了出来。“咿呀——”绫乃感觉到下身的瘙痒，菊门立刻紧了些。miyu屏住呼吸，不敢动弹。在绫乃放松了身体后，miyu后腿往肉壁蹬，双臂撑开菊花，于是整个人从里面挤了出来。趴在绫乃圆硕的屁股上喘着气。

刚刚这一举动惊醒了绫乃，她将手伸进内裤抽出了miyu

“你居然没有死。”说着将另一只手凑过去

miyu已经没有力气挣扎了，闭上眼睛等待死亡的来临。

“还没有动手吗？”miyu睁开眼睛看着，绫乃伸手抚摸着她的头。

“我错了”绫乃满怀歉意的目光注视着miyu“今后我们还是好朋友。”

后来绫乃将miyu的体型恢复到原来的大小。

不过因为这件事情，倒使她们在校园里的关系更加亲密了。
