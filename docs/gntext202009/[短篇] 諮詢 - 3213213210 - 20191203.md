# 1

「那你今天是想問什麼?」

「就是那個啊，你記得我們之前撿到的人嗎?」

「嗯，巷子裡那個，怎麼了，你不是帶回家了?」

「對啊，他最近感覺很奇怪耶。」

「怎樣，生病了?」

「感覺不像，但是好像一直在迴避我。」

「怎麼樣迴避?」

「動不動就躲起來，不肯待在開放空間。」

「說不定他喜歡密閉空間啊?」

「可是只要我出門，桌上的東西都會被動過。」

「所以你覺得他平常都在探險，你在的時候就不願意出來?」

「對啊，一開始還不會這樣。」

「一開始? 嗯……那是不是你做了什麼?」

「做了什麼?」

「對，讓他覺得不想跟你相處的事。」

「沒印象欸，我通常都做自己的事啊。」

「沒有跟他互動?」

「偶爾啦。有時候覺得他很可愛就盯著她看。」

「……那他躲著你超級合理的好嗎。」

「咦～可是我超貼心的，為了他的自尊一直沒笑出來。」

「……問題不在那邊。就算是我，突然被陌生人盯著看也會不舒服。」

「我也會不舒服啊，這樣很公平吧。」

「呃，不舒服為什麼還看?」

「嘴巴很乾啊，但是喝水會打破那種感覺，還是盯著看比較重要。」

「嘴巴乾才不算是……」

「…………」

「怎麼了?」

「你該不會……」

「該不會?」

「該不會一邊盯著他看，一邊憋笑，一邊舔嘴唇?」

「好像是耶。怎麼了?」

「……怎麼了，真是好問題啊，怎麼了……」

「所以我不該舔嘴唇，應該多喝水?」

「……這也是個改進的方向，但問題也不在那邊。」

「所以……不要看他就好?」

「絕對不是，你每次都只顧自己開心，沒去想……」

「………………」

「又怎麼了?」

「我之前推薦你的那個，你說很好用的那個。」

「哦，對啊，拔出來的時候好像連靈魂都抽出去了。」

「我不是問你感想，你用的時候有沒有迴避?」

「迴避什麼?」

「你跟我講很好用，是在撿他回家以後的事。」

「嗯……對啊。」

「你有沒有，至少，去浴室用?」

「沒有啊，我喜歡躺在床上的感覺。」

「……然後在你自慰的時候，他就待在距離三公尺的桌子上。」

「嗯，對啊。」

「……在此之前或之後，你盯著他看還舔嘴唇。」

「之後，沒錯。」

「然後你不知道為什麼他一直躲著你。」

「不知道。所以你知道了?」

「大概猜得到。」

「所以我要怎麼辦?」

「呃……至少給他個獨立的空間。」

「獨立空間?」

「對，我知道你租的套房不大，但至少讓他覺得有屬於自己的地方。」

「屬於自己的地方?」

「意思就是，他感覺自己活在你的存在之下，你得幫忙畫出一條界線。」

「他感情也太豐富了吧。」

「……對。就麻煩你稍微顧慮一下他豐富的感情了。」

===================一周後====================

「結果呢?」

「好像有效耶，他沒在躲了，會待在我弄的小房子裡。」

「很棒啊。那為什麼今天還要找我出來?」

「因為有種他現在在準備逃跑的感覺。」

「你怎麼知道的?」

「我之前看到他在研究排水孔，看到我就嚇一大跳。」

「……排水孔? 你的浴室不是有台階? 他怎麼進去的?」

「沒有啊，我就把房子放在浴室啊。」

「……什麼?」

「我想說獨立的空間嘛，套房最獨立的地方就是浴室了嘛。」

「……那你洗澡怎麼辦?」

「淋浴間門拉上就好了啊，他的房子我放在馬桶前面。」

「……馬桶前面。」

「拿來大便的時候墊腳超方便的。」

「……拿來墊腳。」

「哦對，現在我也在浴室自慰了，這樣清潔超方便的。」

「……在浴室自………………」

「………………………………………………」

「幹嘛啦?」
