# 1

昏暗的小货车车厢被打开了，里面蜷缩着二十几个青年，已经坐了三个小时的车的他们，神经却一时一刻也没有松懈过，因为等待着他们的，是死亡。
每个人被两个警卫押下了车，由几个拿着手电的女仆带路，默不作声的拍成队走向别墅的大门。
虽然是夜晚，但是别墅周围灯火通明，美丽的喷泉，花岗岩地砖，精心修剪的榆树墙，都可以看得一清二楚，然而他们无暇欣赏这些。
长长的队伍被带到了别墅侧面的地下室，辗转了好久，最后所有人被推进了一个圆柱形的电梯一样的房间里。电梯门关上之后依然没有人开口，都在等待着最后的审判。
突然有人在外面启动了什么开关一样的东西，电梯里的灯光由绿色变为了红色，里面瞬间变得像桑拿一样热，同时电梯开始剧烈的震动起来，所有人的关节都感受到剧烈的疼痛，开始在里面惨叫着跌跌撞撞。不过仅仅过了十几秒，周围就安静了下来，里面变得一片漆黑。
不过很快，强光突然照射进来，所有人都捂住了眼睛，等他们视线再次变得清晰时，电梯的顶盖已经被移开，取而代之的是上空巨大的微笑着的少女的脸庞。
这里的所有人都是因为无力偿还别墅主人的私人贷款而被带来的，他们的命运就是被空间压缩机缩小到2cm，成为别墅主人——这位少女的食物。
尽管已经被剧透了自己的命运，但大家见到这一幕时仍旧震惊到失去了控制。在茶杯大小的电梯里哀嚎，拼命攀登光滑的墙壁，或是徒劳的敲打着电梯的门。
（很荣幸大家能到人家的肚子里面来做客，为了迎接大家，我可是特地一整天都没吃东西哦～）少女摸着自己光滑的腹部舔了舔诱人的嘴唇，可爱的声音令这些人陷入了更深的绝望。
（大家不要吵啦，每个人都有份～我的肚子里面可是很温暖很舒服的哦，有没有人自告奋勇尝试一下？发出声音就算数哦。）吵闹的圆筒里瞬间变得死一般的沉寂。
（切，大家都好害羞，那就一起来吧。）
穿着睡衣的巨大少女像拿起杯子那样拿起乘满生命的容器，简单的抖动一下，这些人就被聚集到了一个角落。少女伸出自己的舌头，把人们逐个倒到自己的舌头上，慢慢的缩回去。
（咕噜）
咽喉的一处突起沿着脖颈向下移动，最后消失了。几个漏网之鱼也被少女用舌头精巧地挑出来咽了下去，他们就这样在短短的数秒内被少女打入了无底的深渊。
（这感觉真是令人上瘾啊）少女满意地抿了抿嘴唇，感受到了小人在她胃里的活动，呼吸变得急促的她情不自禁地把手伸向了自己秘密的花园。
已经被少女囚禁在胃里的人们，正在绝望地做些最后的挣扎。然而空洞的胃袋感受到了猎物的到来，很快开始分泌大量的胃液。接二连三的惨叫声在少女的体内回荡着，却没有人能听到他们的呼救。
（大家……呼……被我消化掉，营养……被我吸收，成为………哈哈……我身体的一部分………）
三十分钟后，活生生的地狱恢复了沉寂，只剩下残破的衣服碎片和脱钙的骸骨。
