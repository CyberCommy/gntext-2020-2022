# 1

*本帖最後由 月熙哥哥 於 2018-10-22 09:31 編輯*

标签： 勇者大战魔物娘\胎内回归\吸收「啊啦…已經醒了嗎？魯卡醬，早上好～❤」     魯卡從沉沉的睡夢中醒來，不知何故，肚子上有一種被異物突入的感覺，腦中的意識朦朦朧朧記不清事，在這片伸手不見五指的黑暗裡，他感到一片茫然。似乎有誰在他的眼前，輕輕地對自己說著話，聽著這熟悉的聲音，他的內心感到平靜，睜開的眼睛也緩緩地眯起，無力的樣子就好像是睡著了一樣。

「還是在睡覺嗎…呵呵，都已經是勇者了，卻還是像個小嬰兒一樣呢。」

說話的聲音是從上方傳來的，但他卻不知道是誰。明明是很熟悉的聲音，但他此刻卻無法專心思考。而且不知怎麼回事，身體也感到沉重，難以動彈。明知道自己的身體出現了異常，但他的心頭卻沒有湧起任何不安與焦慮感，而是繼續安心地睡著。他發現，自己的身體好像被什麼柔軟的東西給包裹住了，肚子裡好像還被插進了什麼奇怪的東西，想側個身子都很困難。全身被溫暖的液體浸泡著，好舒服……魯卡伸出手指，戳了戳包裹住自己的東西，軟軟的充滿彈性，就好像是……媽媽的子宮一樣。

「魯卡醬，終於醒過來了呢，呼呼……變成如今這麼個小嬰兒的感覺怎麼樣？」

外面好像有人在說話，是一個相當親切的聲音呢。魯卡慢慢清晰的腦海中，終於朦朦朧朧地浮現出昨天晚上所發生的一切。原來昨天晚上發生的一切都不是夢啊，在船上被阿魯艾露瑪攔截，戰鬥失敗後被她的尾巴吞食，之後的記憶就開始模糊，一切都顯得好虛幻，就好像還沒從夢中醒來一般。

「啊啦啦，明明已經在我的子宮裡面了，但是卻沒有什麼明顯的反應嗎？我一開始還以為你會用盡全力掙扎的呢……」

原來包裹住自己的柔軟肉壁，真的是阿魯艾露瑪的子宮。魯卡的身體浸泡在溫暖的羊水裡，大腦一遍遍地回味著阿魯艾露瑪說的話。他現在終於知道，這插入身體的異物，是連接了他與阿魯艾露瑪身體的臍帶。

「魯卡醬，還是不想起床嗎？那好啊……從今以後，我們就永遠在一起了呢～❤」

阿魯艾露瑪溫柔的聲音從外面傳來，隔著厚厚的肉壁，魯卡似乎能感覺到她在撫摸自己鼓起的肚子。

魯卡明白了自己身體的反常，現在他的身體已經變成了一個小嬰兒！但是，不可思議的是，他的內心卻沒有一點焦慮的感覺，這種寧靜安詳的環境仍然讓他昏昏欲睡。

「魯卡醬，你知道嗎？我們魅魔族，最喜歡可愛的男孩子的精氣了❤魯卡醬身上的精氣又比其他人更加美味，大姐姐我……真的好想一個人獨佔魯卡醬呢～」

子宮的蠕動開始加快，溫暖的羊水快速地流動著，摩擦魯卡敏感的小小身體。不知道是不是錯覺，原本緊窄的子宮，開始慢慢變得寬闊起來。先前只能像嬰兒一樣蜷曲著的魯卡，現在也能漸漸伸展四肢了。

「啊啦啊啦～可愛的魯卡醬❤已經開始被我吸收了呢……這樣下去的話，魯卡醬就會成為姐姐身體的一部分了呢」

自己的身體正在面臨消失，阿魯艾露瑪的話語中透出危險的信號。魯卡任憑自己的身體越來越小，也不打算進行反抗。好溫暖的感覺，像是回到了媽媽的肚子裡……好想就這樣住在這裡面，再也不出去……魯卡放棄了逃脫的機會，閉上眼睛，靜靜享受著住在子宮裡的快感。

「啊啊啊～❤上等的精氣，開始流入我的身體了……好舒服啊～魯卡醬❤」

阿魯艾露瑪鼓起的腹部開始漸漸收縮，從外面也能清楚聽見吸收的水聲。魯卡的身體正在越來越小，他的意識逐漸模糊不清，腦海中浮現出的，是阿魯艾露瑪的記憶。他知道，自己的意識也即將和阿魯艾露瑪融為一體了。不知為何，明明知道自己要消失了，魯卡的內心卻只有安心的感受。勇者的任務已經無所謂了，阿魯艾露瑪的身體才是他最終的歸宿。魯卡舒展手腳，平靜地等待著最後時刻的到來。

「魯卡醬的力量，沒有什麼反抗就融入我的體內了呢…真是坦率的好孩子啊，最喜歡你了，只屬於我的魯卡醬❤」

阿魯艾露瑪站起身，輕輕拍了拍沉甸甸的乳房。滿足地扇動翅膀，飛向天空，她那平坦的腹部，已經聽不見吸收的水聲了。原先躺著的船甲板上，只剩下一灘愛液，粘稠的愛液結成一條晶瑩的絲線，一直連到阿魯艾露瑪裸露的股間，隨著阿魯艾露瑪的騰空，絲線被越拉越細。最後，斷開的絲線離開了阿魯艾露瑪的身體，回到了船上。過不了多久，這些愛液會慢慢蒸發掉，在這條船上，魅魔吃掉了一個少年的證據，恐怕也會消失殆盡吧……

本章完……我知道這次很短啦……這次是寫的是短篇集，所以總字數會和前面的魔物娘同人持平。阿魯艾露瑪篇總共三章，彼此劇情獨立，下一個短篇是寫女騎士塞西爾的故事的，敬請期待～ by 月熙（2018\10\21）
