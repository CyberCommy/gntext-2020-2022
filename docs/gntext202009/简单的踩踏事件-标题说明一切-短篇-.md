# 简单的踩踏事件（标题说明一切；短篇）

作者：嘟囔u影

TID：19766

 

# 1

在纳鹿贝的魔法师之塔，身为二级理论探索法师的蒙蒂安.贝拉是最被人嘲笑的女孩。作为法师塔二级法师的存在，除了发布几篇魔法实践性论文，她从没有真正给别人展示她所会的魔法才能到底有多少。她的指导老师都暗叹她实在是不应该做法师，而应该当一名学者。

这些贝拉都是知道的。但她很不甘心就这么走了，最起码，也要学会释放一门简单的法术啊。

并不是贝拉完全没有魔法师的天赋。只是她曾经被遗迹里一枚古金币的诅咒而导致她的魔法网无法与世界上任何的魔法网进行链接，这就导致她无法借用魔法网的力量来施法。

而她潜心来到法师塔的理由就是找到解除自身诅咒。只有法师塔海量的古藏书籍才会找得到诅咒解除的方法。

而功夫不负有心人，在一个月专研古帝国历史，她终于找到一种可以解除自身诅咒的方法——找到一种遗迹祝福的物品，就可以中和掉自己身上的诅咒了。

而且凭贝拉多年的专研古历史，她推测到法师塔背靠的隆德剌山脉里的古老密林里可能有一座未被发现的古国遗迹。

这是机会。贝拉不止一次对自己说着。而后她在一个月内准备好装备，决定自己一个人前往那座遗迹。

在踏入山脉密林里一个星期后，贝拉慢慢靠近那座封存数年未有人知的神秘遗迹。

但视线要转换一下，转换到生存在遗迹周围的一个“小”村落里面。

这里是一个还不曾有人闯入的地方，并且早有生物群聚在这个地方。

他们也可以说是被“祝福”，或者该说被“诅咒”的人类了。

他们的先祖由于被古老帝国的雅诺神灵所祝福，所以学会魔法锻造。而他们先祖由于在诸神之战中站错位置，抛弃了雅诺之神，遂被其加强祝福，在得到魔法锻造的能力的同时，会失去自身的定性，最终就会变成1厘米大小的人类。

为了不被灭族，他们举族迁移，最终在一片高山密林里找到住所。

但万万没有想到的是，他们会在今天毁于一个人类女孩的脚下。

=============

贝拉在遗迹的山脚下停下了脚步。她欣喜激动，险些要失去理智。不过多年的处事让她学会如何冷静看待事物。所以她平复了心情，踏步踩在密密麻麻的低矮草丛上。

但贝拉不知道，她马上就要毁灭一个村子。

（小人角度）今天村子有些不对劲。是西边的施拉姆开始暴乱了吗？一直在那边传来震动，而且还越来越大。

可能是我多虑了，还是昨天迅游太晚而产生疲惫感了？

反正只要村子不出事就可以放心了。

“咚！！”

什么声音？还震的屋子歪歪扭扭的。出去看看，施拉姆不可以有这么大的动静。

而出来后看到的一切都让我惊呆了，不，是我周围的所有人都惊呆了。

那是什么？一根黑色的柱子从高大的草丛上方露出，而后下面裹挟的奇怪的物体随着柱子一抬在落下，震的使人都快站不住脚。

这是怪物吗？从没有见过的黑柱怪物会做什么？

很快我就知道它要做什么了。

只见那柱子裹挟着奇怪的东西一抬，马上落下就落在村子门口，震的外围木墙发出“卡兹卡兹”的声音。而正巧在村口站岗的那个家伙张大了嘴，跌坐在地上，可能已经被近在咫尺的黑柱裹挟的东西给吓傻了。

而之后那怪物又抬高了，然后……“轰”的落在村子外边门口，还附带那倒霉的站岗的人。

我脑袋一下子懵了。随后我马上醒过来，想要逃跑。

可时间不等人，我还没有逃出门口半步，那巨大的怪物携带着一股不可抗拒的威严高高悬挂在我家屋子上方，不给我任何喘息的机会，“轰”的落下。

等我反应过来之后一切都太晚了。那巨大的怪物一下子就把我连带我是房子都压近地里，我无法反抗，只能在骨头被压的发出“吱吱嚓嚓”的声音里慢慢失去思考……

============

（贝拉视角）奇怪，为什么感觉踩到什么了，草丛里有什么东西或者小动物被踩到了？

算了，不管了，遗迹就在山上，快点上去，我都迫不及待解除诅咒的那一刻了。

不过还是很在意，刚刚脚下是不是真的踩到什么了？

===========

在贝拉不知道的情况下，她毁灭了一个村落，虽然他们只是一厘米大小。
