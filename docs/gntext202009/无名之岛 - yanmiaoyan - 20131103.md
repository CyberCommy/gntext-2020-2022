# 1

听说帖子少，开个渣开头抛砖引玉
关于那座奇妙岛屿的故事，我是从母亲口中得知的。当然，那时她还没有去世，那年我六岁。
「那座岛上四季如夏，周围是一整片雪白的沙滩和颜色有如昆虫血液的珊瑚礁。岛的尽头有座旅馆，里面住着一位年龄和出生地都如谜一般的老板娘。她每天举行聚会，岛上的居民们会来参加」
「天天举行聚会？那大家不觉得腻吗？」
母亲仰躺在房里的床舖，而我正依偎在她的怀里，享受着她轻柔的抚摸。
「每天都有不同的人啦」母亲答道：「不过那里是座特别的岛屿，聚集了来自世界各地的人，而且每天聚会前大家都不知道这天的活动。」
「为什么？」
「在那座岛上，有着一股神奇的力量，能够改变人的体型，那个老板娘可以使用这股力量，她每天都随便调整不同的大小，你爸爸就曾经像你这样大小依偎在我怀里呢」
「这么说来……甚至可能比婴儿还小啊？」
母亲不知道想到了啥，露出了古怪的表情，点了点头。
「确实有运气这么差的人呢。」
那是我和母亲最后的谈话。后来母亲就由于去买菜遇到意外去世了，就和我的父亲一样离开了我，只留下我和奶奶生活在一起。
如今我已经19岁了，甚至有了个可爱的女友，但是脑海中却还是回想着依偎在母亲怀里的感觉。于是在冬季将近时，我和女友一起踏上了旅程，去那座神奇的岛屿。
我靠在小船前端的栏杆上，任凭头顶的炎炎艳阳和纯白甲板反射的剌眼阳光猛烈夹击，环视周遭，看见两种仅浓淡略有不同的蓝色，漫天盖地地包夹住整个世界。
那座没有名字的神奇岛屿，现在还在不在呢？真希望它就此沉没算了。如果这艘船永远靠不了岸，永远漂荡在汪洋当中该有多好？一旦抵达目的地，答案便呼之欲出了。虽然，那是我早已心知肚明的答案，我想莎莎应该也早就明白了吧？男人追寻答案本身，女人则追寻得到答案的过程。印象中的这话是为了说明男女对于性欲的追求各不相同，但说不定这其实只是我自己胡思乱想出来的句子。
无论如何，我还是带着莎莎来了。远离陆地后，脑袋又持续暴露在耀眼的艳阳和浓郁的海风之中，不禁令我深深体会到自己早已疲惫至极的事实。尽管在小说中写过很多体型差的故事，但实际上不过是我的妄想而已。一旦想象过后，只会更加饥渴，让人忍不住找更多的这样的故事。最后就这样迷茫下去。虽然莎莎接受了这样的我，但却让我更加的痛苦，更让我恐惧的是我内心居然在嫌弃她的娇小，更加恐惧如果母亲所说的那个小岛的故事是假的该怎么办。
我从口袋里拿出被压得扁烂的香烟，叼住其中一根并将其点燃。
一阵生锈金属的摩擦声从背后传来，我回头一看，正好看见舱门开启。一位年轻人自舱内走了出来，看来约莫只有十五、六岁，有着高挺的鼻梁和土耳其蓝的眼眸。所以当他以流畅的日语说出「您好」时，我不禁吃了一惊。
「我们快到了呢。已经看得见这座岛的轮廓了。」
我顺着这个外国人的话转头望向船头，面对眼前的地平线，我缓缓吐出香烟的烟雾。
「来访这座岛的旅客，就有人在岛上结婚了，也有人离开了。毕竟这里只是一座小岛，知道的人也寥寥可数。虽然超自然现象杂志曾报导
过，人们之间也偶有传言，但实际上亲身来访的人大多还是透过口耳相传，或者是本身曾经来过。」
年轻人说得没错。如果不是只有这儿可能实现我的欲望，我也不会千里迢迢地跑来这种鬼地方。
「请问，那些再度来到岛上的人……呃……」我迟疑了一下。
「都多带人来了吗？」
「是啊。」
年轻人脸上的笑容消失了。
「还有带小孩子来的，不知道想啥，不知道小孩子有时候没轻没重吗？」
我脑海中不由出现了这样的图片，一个可爱的小女孩怀里抱着一个成熟的男性，那个男性在她怀里如同婴儿一般，脸上露出痛苦的表情，但小女孩却恶作剧般的用力搂着他，让他都难以呼吸。
就算明知如此，我还是带莎莎来到这里了，只为了证明这件事。
「这次与您一同前来的那位……」年轻人在我身旁小声询问道：「我刚才在船头附近看见她，那位娇小可爱的小姐是……」
「是我女友。」
「哎呀，果然没错。」
「那你可得让她到时候动作轻点了。」
就在这时船猛然一震，到岸了。
