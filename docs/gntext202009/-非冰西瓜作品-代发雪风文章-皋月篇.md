# 1

（镇守府 提督办公室）

“皋月，你过来一下！”我朝着门外正好经过的名为“皋月”的小女孩喊道

“司令官！您叫我？平时可是不怎么关心我们远征队的孩子们呢！今天有事吗？”

皋月一脸期待地站在我的面前，金色的双马尾在脑后摇啊的

“嘛...今天又有一个特殊的任务交给你...”我坐在书桌后面，搭起的双手摆成了碇司令的模样（I have a special mission for you）“是...新的远征任务吗？”皋月的脸已经嘟成了へ字形，一脸失望地看着我“并不是...而且，这是一个单人的特殊任务...”（15分钟后）“真是的真是的！司令官是存心寻我开心吗？说好的特殊任务就让我在房里等着！”皋月不满地在床上滚来滚去，用手敲着自己的靠枕发泄着不满

可就在此时，房间内却产生了诡异的现象，一个浑身带着电火花的球形出现在了房间的上空“噼啪-噼啪-”（咦？这是？）皋月小心翼翼地用食指触碰了一下电球没想到电球瞬间消退了，而从中似乎掉出来了什么东西掉在了地上皋月满脸地朝着地上看去，却惊讶的发现掉下来的居然是一个只有3厘米大小的小人儿！而在小人的身旁，皋月发现了一张字条上面写着：

皋月，鉴于某个家伙的强烈要求，我把他从另外一个世界中拉过来了。似乎这个家伙是你的狂热粉丝，最大的愿望就是被你玩弄。所以，今天就放你一天的假，好好享受玩玩具的感觉好了~毕竟皋月还是小孩子嘛~

爱你的提督 留

P.S. 这家伙是个抖M，使用他的时候请不要手下留情“司令官对这个人是有着多大的仇……”

“不过，似乎很有趣的样子呢！”

皋月将小人用两只手指捡了起来，放到了自己左手的手心里于是，按照字条上的指示，当这个名为冰西呱简称呱的男子醒来之时，他的身子上已经覆盖上了皋月那粉嫩的大脚趾

“嘻嘻，没想到皋月我也会有粉丝呀”

这样说着的皋月，左右扭动了一下大脚趾，将男子的身体不由自主地也带动了起来，在地上弯曲着而此时瓜的眼中所能看到的，就是那完全覆盖在自己眼前上空，皋月那高耸入云的右腿，以及上面那可爱的雪白胖次

“好可爱的白色……”男子下意识地咕哝了一句

“什么白色……啊！”刚反应过来的皋月，马上脸红地用左手扯住短裙的下摆用力往下压，将那一抹春光完全遮住“可……可恶！明明已经被我踩住了，心里居然还想着的是偷看我的胖次！”这样想着的皋月，脚下不由得加重了几分力道

“呃……”男子似乎是被压到了胸口的肋骨部位，痛苦地在皋月脚下扭动着刚被传送过来，意识不清吧？

由于想起了这是司令官的友人，还并不能就这么踩死他的皋月，慢慢地松开了脚趾，将右脚收回到了男子的跟前而由于吃痛的关系，男子也完全清醒了过来

“这……这里是哪儿……”疑惑的他，看了看房间四周的布景，下意识地吐出了第一句话

随后，便看到了皋月那对于他来说过分巨大的身体“咿呀！！救……救命！”皋月那光滑洁白的裸足，在男子的眼中，却是和自己一般高度的大小，甚至还比自己高出一截来。而不断扭动着的脚趾，就像是巨大的魔物一般，在对着男子发出无声的愤怒咆哮

于是，男子慌不择路地朝着后方逃去，由于过度的恐惧，双腿发软的他用两手颤抖地撑着地面的毯子，连滚带爬地向着后方滚了出去

“喂，我说……好歹是我遇到的第一个粉丝，居然是这种反应……”皋月不悦地叹了一口气，朝着男子说道虽然男子的确是个皋月控，也曾无数次的在屏幕外头幻想过与皋月的各种亲密场景，但绝对没想到的是，自己居然在她的房间里和她本人见了面。而更要命的情况是，自己对于她来说，只是一只小虫子的大小“轰……”皋月朝着男子的方向走了过来，每走一步，就在地上引起了一阵剧烈的震动，让男子不由得一屁股坐在了地上被吓坏了的男子想要继续逃跑，但是他的速度，相对于皋月来说，简直如同蜗牛在地上爬一般。很轻松地，皋月就重新用右脚踩住了他。不过这次与上次的情况不同，男子被夹在了皋月右脚的指缝里

“真是只小虫子……好弱啊。为什么我的粉丝居然是只小虫子呢。”“不过，司令官说过，每样东西都有它自己的用途呢。”“吶！告诉我，你的用途是什么呢？”皋月把用脚趾缝夹着的男子举到了半空中，用力地摇晃着脚。可此时可怜的他已经被剧烈的抖动都给晃得快失去思考能力了，哪里还有半分力气来回答皋月的问话？于是，皋月失望地什么都没有听到“好无聊，本来以为能好好玩的……”正当皋月一脸失望地想把男子从脚趾缝中取出扔到一边不管之时，突然感觉到大脚趾有点痒“滋溜——”皋月睁大了眼睛朝着自己的趾缝内看了过去，却发现是男子竟然伸出舌头，在舔着自己大脚趾的内侧。“呼呼——没想到小虫子居然还有这用途呢”皋月本来沮丧的心情，瞬间变好了，而在她的脸上，出现了一抹坏笑“那么小虫子，既然你这么喜欢我的脚的话，皋月大人今天就委派你做我足底的清洁工作了哦”将男子从趾缝中释放出来的他，用大脚趾轻轻地点着跪坐在地上的男人的头。随后，在男人惊愕的目光中，那如同高大房屋一般的巨足，就这么立在了男人的身前远处，是坐在地上的皋月那张坏笑着的脸“快点干活吧小虫子~”慑于对自己性命的担忧，男子开始吃力地从皋月的足底开始攀爬起来，如同一只真正的小虫子一般。每到一处地方，就用力地舔起了面前的肌肤墙壁。可这哪里是一个3厘米的小人能完成的工作啊一个小时之后，累瘫了的男人也没干完一半的活，更别说高耸在玉足顶端的五个巨大的脚趾了“真是没用的小虫子……”皋月失望地咕哝着，不过这种情况，其实也在她的预料之内了。根本没有奢望男子能真正完成工作的她，从地上站了起来，将两只脚放在了已经累瘫的男子的两边。“哈……哈……”正当男子还在猛烈地喘着粗气之时，巨大的阴影已经将他完全地覆盖了起来。男子猛地抬头，看到的却只是皋月放下来的离他越来越近的脚底……（突然想直接让皋月踩死呱了，这样我就可以不用继续写下去了）正当男子以为自己死定了的时候，皋月的脚，却轻轻地放在了他的面前。“吶~刚刚根本就没奢望你能完成这工作，看在你还算努力的份上，爬到我的脚上来吧！”可是，刚刚已经耗光了力气的男人，根本没有再爬上来的可能了“真是麻烦……”念叨着的皋月用右脚轻轻地将男人夹了起来，放到了自己左脚的脚面上

（真是可爱，意外地是个努力的孩子呢……呜呼呼——）

皋月眨着星星眼看着脚面上的小人。刚才他努力地为自己服务的心意，自己已经全部感受到了。对于这样乖巧的孩子，应该奖励才是呢。

这样想着的皋月，悄悄地抬起了右脚，用大脚趾一下子将男子戳倒了。

“呜——”男子试图发出抗议，但整个身体又被皋月的大脚趾给压住了，动弹不得。

不过这一次，皋月并没有用太大的力气，而是用左脚来回地晃着，右脚轻柔地在男子身体上来回地搓动着，简直就如同在给他按摩一般。

随着左脚轻柔地来回晃动着，皋月哼起了轻快地歌谣。

（几分钟后）

皋月松开了大脚趾，将左脚抬了起来。

“咦？居然睡着了？难道是因为太舒服了的关系？”似乎是由于太享受皋月的按摩的关系，男子蜷缩在皋月的左脚脚面上睡着了。

“真是拿你没办法。算了，等会去司令官办公室问问吧。看在你这么喜欢我的份上，不如以后就当我的宠物好了！”这样想着的皋月，似乎在遐想着与男子以后的生活，眼里冒出的星星越来越多了……

（一天后）

提督办公室

“不是吧……还真要留下来……”我双手抱头，痛苦地看着书桌上的一份报告，上面清楚地标明了作者：皋月

“小西瓜~今天有没有乖乖听话啊？”刚结束一天远征任务的皋月，从浴室中回到房间，第一件事就是去找寻自己新收养的“宠物”

而被她口中所称为“小西瓜”的男人，现在的身份已经转变为了她新的宠物，被饲养在了她的房间之中。

“嗯嗯~中午给你准备的午饭都吃完了呢，真乖。”皋月在检查了男人的进食情况之后，坐在床上，用手指不断地抚摸着男人的背部。

“既然你这么乖，那么，今天特别同意你提一个要求好了！”

“那个……皋月sama……我可以亲吻您的脚趾吗？太可爱了！”从男子的口中，冒出了这么一句话。

“…………”皋月沉默了

“怎么了？是这个要求不行么？”男人看到一言不发的皋月，以为她生气了，小心翼翼地问道

“没想到小西瓜居然这么喜欢我的脚呢~”皋月开心的笑了，银铃般的笑声在屋内回荡着，“可以的哟”

说完，皋月将男人用右手捏了起来，缓缓地凑近了自己左脚的大脚趾

“chu~”男人向着自己心里所喜欢的人的玉足，献上了自己深情的一吻

“嘛，真是受不了~”虽然这么说着的皋月，看到如此迷恋着自己脚趾的小西瓜，内心还是有着一丝丝的甜蜜
