# 3

由于这次的作品我早已构思好完整的结构，因此全文有大量的伏笔；另外，要符合这篇的语言风格并试图描写更细腻的人物情感，所以我没办法很直白地写出很多说明

结果就是，许多读者可能看了一两遍还是有些地方看不明白，觉得说不清楚

我个人是有意用比较隐晦的描写手法的，基本上读者的疑问应该都能从文中发掘出答案，不过这种解密一样的写法阅读起来肯定很累，如果各位对于情节有不明白的地方都可以在回复中发问，我会一一解答——当然，剧透的内容没办法说

【其实这是一种写作技巧哦：遇事不决杀女主】我这么说大概会被砍死

不过因为有上一章的关系，相信大部分看过的人都能猜到蜜希尔会死，但是绝对想不到会死得那么快

其实我是很心软的写手，没办法描述自己笔下可爱的角色的死亡，所以我在那一段直接切换到了洛夫的视角，尽量用很平静的语句讲述本篇最悲情的一幕。

顺带一提，把视角放在凶手身上后杀掉主要角色反而特别带感呢！（喂说好的心软呢！！！）

蜜希尔这个角色可以说就是为了死去而创造的，对她的一切刻画一切描绘都是为了让她的死更有冲击力与落差感，作为作者，许多时候不得不做出这么残酷的事情也是没有办法的，因为一个角色往往就是在死去的瞬间才能爆发出极致的魅力，如同流星，死亡是真正生命的开端。

是的，死亡，这是我这部小说的一个主题。

死亡的恐怖，死亡的魅力，死亡这个词所引发的一切联想都是由于它的一大特性——不可逆转

一个意志的消亡，一个思想的瓦解，这是不可挽回的悲剧，所以才值得铭记。如果死者可以复生，那生命反而会变得一片灰白无趣。

所以杀人者死，因为他犯下了无可挽回的罪。

大概也有很多读者猜到接下来的主线发展了：亚伦会误杀无罪之人，之后可能会死。

这是剧透，但我完全不在乎——正如猜不到蜜希尔死得如此之快一样，我相信大家也猜不到故事的结局会怎样发展

即使你真的猜到了，相信也只能猜到第一层真相

我其实是喜欢写悬疑故事的，但我的个性又不喜欢剑走偏锋地用奇策，所以我会布下重兵，重重伏兵，堂堂正正地以力服人：读者可以猜到故事，可以猜到结局，可以猜到哪条伏笔对应的会是什么照应，甚至可以猜到我会在结局上做出怎样的转折——但是，我不会让人猜到第二重甚至第三重真相。

我现在主要练习的是如何减少自己的闲笔，换句话说，我希望自己写的每一句话都能够发挥它的作用，比如烘托气氛，比如侧面刻画人物，比如揭示世界观，还比如......成为伏笔

【任何不太合理的描述，都是一个伏笔】

↑这是努力的方向

蜜希尔是个伏笔
