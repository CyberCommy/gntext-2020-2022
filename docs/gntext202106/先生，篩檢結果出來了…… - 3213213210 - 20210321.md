# 先生，篩檢結果出來了……

作者：3213213210

TID：30553

 

# 1

診間內。

「先生請坐，你的自費篩檢結果已經下來了。這次的項目包含縮小病……」

「等一下，縮小病是什麼？」

「縮小病是一種非常新興的疾病。傳播途徑、潛伏期不明，而且也還沒發現治療方法。提前篩檢是為了讓病人可以有所準備。」

「準備？」

「就是在自己失去行為能力前處理後事，還有製造縮小後的自己也能生活的環境。」

「……製造環境？」

「現在市面上還沒辦法充分應付新興的尺寸需求，所以大部分的人會請看護，解決各式各樣的生理需求。或是請親近的熟人……」

「我還有多少時間？」

「先生，不好意思，您的縮小病結果是陰性。我只是在列舉篩檢項目而已。」

「哦，真是太好了。」

「不過你得了胰臟癌。」

「啥？」
