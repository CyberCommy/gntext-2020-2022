# 1

此时，我完全处在了无意识中.......

第三天一早，也就是5月2日，姜颖从外地回到了上海。到家后，觉得混身劳累，脱了鞋便倒到床上呼呼睡了。快到中午的时候，电话响了，是她同事李芸打来的。她说她这两天不舒服,最近经常便秘，想去医院看下，可是家里老公又不在，问姜颖方便不方便陪她去。

‘我在睡觉呢,'姜颖不大想去。

‘算了，就难得陪我次吧。’

‘那好吧，下午一点我到你家来找你’说完姜颖就挂了。此时她的睡意也没了，从床上坐了起来，看着鱼缸。忽然她发现海带没了。。。到哪去了呢？不会被她们几个前天吃了吧.连忙电话给了沈婷,得到了确认。着么办？ 对了，现在还有一丝希望，就是李芸刚好象说她便秘，希望。。。能从她的大便里发现什么，不过那也太恶心了。姜颖想着，也在家呆不住了，便打了辆车往李芸家赶去，在出门时拿了个小火柴盒，打算万一真的还有希望，那可是要捞李芸大便的呀。。。

到了李芸家，李芸说‘算了，还是不去了吧。我刚刚去过厕所,你就在我这里吃了晚饭再走吧’

‘哦，不用了，我家里还有事先走了，不过我能用一下你家的厕所吗？‘姜颖听到李芸已经上过厕所了，觉得没什么希望了，但是她还不肯放弃。

’厕所在那里，那你自己去吧'

姜颖走进了厕所 关上了门，可能是李芸刚用好厕所吧，里面还有股恶臭，姜颖忍着臭，发现在马桶内壁上，还粘着一点黄绿色的大便，里面竟然还有一点没完全消化的海带。

‘不知道行不行了，先捞了再说吧.'想完姜颖便从马桶壁上捞起了那没完全消化的海带丝,弄进了火柴盒，同时为了表明她上过厕所，便冲了水后走出去和李芸说‘那我先走了哦，有事再找我吧'便往家里赶去。

到家后她忍住不断要呕吐的欲望，把那残破的海带丝捞了出来，用水洗干净后想‘现在再着么救呢？恩。。。。只能这么试试了’说着她拿出了个很大的盆，把洗干净的海带丝放了进去，用手一指。。。

一年后，我苏醒了过来，发现自己躺在自己家里，身上仍然盖着毛巾。姜颖看到我醒后，变把整件事都告诉了我。

原来她当时竟然把那仅存的一点海带丝变成了一些细菌，然后在水盆里放上营养液让我自己繁殖，过了段时间她又买了个更大的缸放在我家，把盆里的水连通菌一起倒了进去。直到今天，她以缸里所有的菌看成一个整体，尝试着把我变回来，结果竟然成功了。她说在那之前她也一点没有把握。

由于暂时我也没有工作，也没去找，于是我们在后面的日子里天天呆在一起，她再也没有把我变任何东西了。终于在七月一日，我们结婚了。

在结婚的那个晚上，她又和我说了个秘密。她说她和我结婚后，我将有变物的能力了，但是只能变自己，而且在变的前面要先设定好时间，在当物体其间那设定的时间是不能变的，要直到时间到了才会自动恢复。而她以后不能再对我施法把我变来变去了，而我变成的东西她也无法感知，会以为我就是那件普通的物品.虽然她还能变其他人，但只能把他们变成没有生命的物品了,虽然还能把他们恢复，但是如果不能变成各种生物而只是变成没有生命的物品的话，那就是如果损坏了就无法 自动修复，也就是再也变不回来了。而且我们有了女儿后，她的能力将彻底消失，直到女儿到18岁成人女儿才获得这个能力。

‘为什么这么肯定是女儿呢？’我笑着问

‘因为。。。由于家族遗传的原因，在我体内携带Y基因的精子是无法到达子宫的'她说到

‘你最好在自己变的前面和我说一下，省得出事情。‘她说完这个秘密后又对我说。

’会出什么事?'我问

‘其实。。。我爸那时就出事了。他和我妈开玩笑竟然设定24小时把自己变成了一只虾去和买来的虾呆在了一起，结果我们当然不知道，把虾都吃了.直到第二天，看到有些虾壳变成了人皮才知道，把我妈都吓死了。最后只能把那剩下的人皮埋了起来，成了永远的秘密。。。‘

‘我岳父真弱智'我暗暗的想到‘不过如果我将来自己变的话有时侯可以告诉她，但有时可以不告诉她，正好可以顺便探一下她的秘密，呵呵’

我变化的命运完全掌握在自己的手里了，最重要的是姜颖也无法知道，从此我开始了全新的生活了。

第二章:邻居张军的结局

由于姜颖婚假打算到圣诞节去澳大利亚过,所以休息了几天后,她就要去上班了.此时我有点舍不得和她分别,正好现在我也还没找到工作,就对她

柔声说'宝宝,你今天穿什么衣服去上班呀,不如我变件连衣裙你穿着我去吧.'

'好呀,'其实姜颖也舍不得和我分开,听到我这么说,高兴的答道.'不过,要变一件最漂亮最好的哦.'

'恩,我先变12小时吧,反正明天又是星期六可以休息了.'

于是,我想了想,把时间设成了12小时后便开始变化起来...全身开始变薄变软...当变完后,我就倒到了床上.我变成了一件纯白色的完全用牛奶凝结后做成的最新式的连衣裙,同时保留了自己所有的感觉.

'真漂亮呀',姜颖看到躺在床上的我,情不自禁的向我摸来,'好滑呀.'我们同时说到,当然我是不会发出声音而是在心里说的.她不断得摸着我,虽然摸得我也很舒服,但我不禁想到再这样下去你上班就要迟到了.

终于她回过了神,把我拿到卫生间脱掉了睡衣睡裤.此时看到她那美丽的胴体,洁白光华的肌肤,虽然还穿着内裤带着胸罩,虽然前几天天天看到,但我还是兴奋了起来.她并没有察觉,拿起我往头上一套,我便象流水一样的从她的头滑到了胸,腹,臀,大腿,小腿,一气而成,此时我因为被她穿着而不在是平平的,也有了完美的曲线.

'恩,大小真合适,真不错呀.'她边照着镜子边说到'可惜就是裙子长了点,算了,上班去了.'说着便拿起了包出门了.

呵呵,当然我要把裙子部分变得长一点啦,不然让色狼看你的腿吗?我暗暗的想,只是没想到就因为我这么个小小的想法,竟然害死了我们楼下的邻居.

到公司后,别的同事看到她都先笑着问'做新娘的感觉着么样呀?'然后看到了我,她身上的连衣裙,惊叹我的完美,不断得向我摸来,弄得姜颖很得意,说'这可是用牛奶做的哦,是我老公给我定制的.'

她走到座位上,坐下来悄悄对我说道'你刚刚是不是被摸得很爽呀,以后不准变手感这么好的东西,听到没?' 呵呵,她竟然吃醋了,我想.接着,她便工作起来.其间她去了两次厕所,可她怕我看其他女孩子非要等厕所没人的时候才去,难道她忘记结婚前还把我变成厕纸的事情吗?结婚前和结婚后女人变化真的好大呀,我想,以后变的时候还是最好不要都告诉她.

中午吃饭的时候她和沈婷坐在一起,沈婷不断得夸着我,她身上的连衣裙,还问是哪买的,她也想买一件.

终于到了晚上下班姜颖回到家了,虽然我比较透气,但是由于是酷暑时节,她身上还是都是汗,我也吸了不少她的汗,粘粘的粘在她身上.因为恢复的时间快到了,她把我脱下来放到床上等我自己恢复,然后自己去卫生间洗澡了...

时间一到我果然马上复原了过来,可是我觉得自己懒懒的,不想起来,就这么躺着吧.一会姜颖洗好了来到房间,看到我已经复原了,还躺在那.

'你脏不脏呀,还躺在这,还不快点去洗洗?'她说

'嘿嘿,我这么脏还不是你造成的?我不洗,你帮我洗吧.'

'你这么大个人我着么帮你洗?'

'那我再变回去变一小时,你就把我当衣服洗了吧'

'好,不过我事先声明我要用洗衣机洗的哦.'

'恩,没问题.'我就是懒着不想动不想自己洗了,于是又变回了连衣裙.

她走过来拿起我,把我拿到卫生间扔进了洗衣机,然后胸罩内裤都扔进来后开启洗衣机后就走开了.

'汗,还真拿我完全当件衣服了呀,陪都不陪我会'我想

接着,当洗衣机里放好水后,便运转了起来.此时我才知道什么叫自作自受了,那个....真是转得我晕呀.最后洗完了还竟然甩干,如果我还是人的话我要把我内脏都要吐出来了.

接着把我放进了烘干机,由于刚洗好,虽然甩过了,但身上还是有不少的水,如果是冬天在烘干机里本来应该挺舒服的,不过现在是夏天....我又忍受着热浪的袭击.早知道这样还不如自己去洗澡了.

终于等我被烘干了后她把我拿出来放到了床上,而其他东西则放到了她床头柜的抽屉里.

她坐在我旁边,等着我复原后,我们又是一夜的激情...

第二天快中午的时候,我们还蒙胧的半梦半醒,门铃响了.原来是我家楼下的邻居张军.说来他也挺可怜的,七岁的时候父母就死了,后来毕业后也一直找不着工作,靠给别人打临工过活,到现在快四十了,至今都没有女朋友.我结婚前经常来找我一起喝酒,喝着喝着关系也挺熟了,都是朋友了.现在他可能看到我结婚了想到他自己还是孤单一人,便来找我喝酒聊天了.反正她和姜颖关系也挺熟的,我们起床洗涑完毕后就一起坐下来边吃边聊.我和他喝着啤酒,而姜颖则是橙汁.

大概到了下午两点的时候,门铃又响了,我和张军继续喝着,姜颖去开了门.

'沈婷呀,呵,快进来,我们正在吃饭呢,一起来吃吧'姜颖开门看到门外的沈婷后说

'不了,我来是想问你借你昨天穿的那件衣服穿穿,明天白天我要和老公一起参加个活动,到明天晚上我就来还给你,好吗?'

'哦,你等等...'

姜颖走进来后看向我悄声问'要么你变了去借给她?'由于大门到客厅有个玄关,所以沈婷看不到里面的情况.

'我想这两天和你在一起么...'我看到已经喝得半醉的张军,说'要么...你把他按照我昨天的样子变了?反正他也喝醉了,你把他变好后把他所有的感觉和意识先暂时都去了,等沈婷明天晚上还回来我们再把他变回来.反正他以前也经常喝醉了在我家睡一天的,就和他说在我们家睡了一天好了,他不会知道的.'

'那...好吧.'说着姜颖对张军一指,张军就变成了和我昨天一模一样的连衣裙.

'真的挺漂亮的'我看着想到,手也忍不住伸过去摸了一把.

'去,你个大男人去摸男人干什么'姜颖开着玩笑,把张军变的连衣裙拿出去给了沈婷.

'谢谢呀,明天晚上就还给你哦'说着沈婷高兴的走了.

而我们收拾好张军的餐具后继续吃着,到伴晚的时候陪着姜颖一起去公园走了圈

'还好变的不是你,你能陪我来逛公园,呵'姜颖说道.

第二天星期天我们又去外面玩了一天,晚上到家后,我坐在客厅看着电视.姜颖刚烧好晚饭,门铃响了.姜颖开了门,听到了沈婷的声音'谢谢你呀,我来还衣服了.'

'进来坐会吧,一起吃个晚饭着么样?'

'好吧,反正我老公下午后来去加班了,那就蹭你们顿饭了呀,呵呵'说着便拿了衣服走了进来.

'呵,我也饿了,我刚烧好,那我们现在就吃吧'姜颖说

'呵,我给你们个惊喜.你看这件衣服现在着么样呀,是不是比昨天更好点了呢?我老公可是著名的服装设计师,他说这件衣服很完美,唯一的缺点就是裙子太长了,于是他昨天晚上把裙子改短了点.姜颖你不会介意吧.呵呵'

'啊?'姜颖说到

我顿时也惊呆了,她着么能自说自话改别人借给她的衣服呢?姜颖现在的能力只能把人变成没生命的物品了,而且有一点损坏就变不回来了呀.这么说...张军就永远是这件连衣裙了?直到他(或者现在用它了)被穿破穿坏? 我对不起张军呀,我想

'呵,周斌又给我买了件这样的衣服,我还是比较喜欢裙子长一点的.要么...这件就送给你吧.'姜颖无奈的说到,谁叫她们关系这么熟呢?又谁叫姜颖没把自己的秘密告诉沈婷呢?不过这个秘密还真的不能说呀.

'呵,那着么好意思呢?'

'别客气了,你就穿着回去吧,顺便给我们看看改得好看不好看?'

'那真的谢谢你了呀,你不会因为我擅自改了你的衣服生气吧,呵'

'不会'

于是沈婷拿着张军(现在只能说连衣裙了)去卫生间换衣服了.

我看向了姜颖,她也看着我,都露出了无奈.

沈婷换好衣服出来后,我发现那件衣服真的和她是完美的结合.她本来身材就好,而连衣裙更把她那美好的身材展露无疑.只是此时,我也没什么心情欣赏了.

吃完饭后,沈婷便穿着连衣裙(张军)高兴的回去了.此后的夏天,我每次看到沈婷,她都穿着这件她心爱的连衣裙,而我看到这件连衣裙,只有深深的内疚,我想如果当时我去的话,我还能把自己变成可恢复的动物自我修复,那样张军也不会有去无回了.不过再想想,也许那也是他不错的结局吧.

第三章

夏天的一个傍晚姜颖下班回家,告诉我说对面开了家一兆伟德的健身房,里面设施都很全的,还有游泳池,于是就办了张卡.她说她晚上吃好饭就想去.我说'那你去吧,呵.多锻炼锻炼对健康有好处的.'暗暗想她告诉我这个干什么呀,想叫我一起和她去健身吗?

'我...我上次买的分体式泳衣下面那半找不到了,你帮我变一件嘛.'她摇着我的手撒娇的说道.

'我想在家看会书么'我说

'你不肯变是吗?那我一会去游的时候裸泳哦,嘿嘿'姜颖玩笑的说

'你会吗?'

'呵,你知道我不会去裸泳.那我随便去马路上找个人变成我下面那半分体式泳衣吧.我会保留他所有感觉的哦.'她又笑着对我说

果然不出她所料,我还是吃醋了.我老婆着么可以随便给别人抱呢?虽然只是变成我老婆的东西后也不可以.

'那还是我来变吧'.我妥协的答道.

'恩,还是老公最好了,你反正最近也一直在家里，这个星期我每天都要去游泳的，不如你就直接变一个星期好了，省得麻烦。哈哈.'说着跑过来亲了我一口把分体式泳衣的泳衣部分拿出来放到床上后去吃晚饭了.

我来到房间,想了想,她晚上去健身房游泳,但不知道会游多少时间,那里是晚上十点半关门的,到家最晚11点也到了,现在是星期一的五点半,她要一个星期么那就七天加五个半小时吧,至于变什么样子的,当然是要和上面那个泳衣部分配套的咯，记得以前下面好象是短裤样的粉色的，想着以前那分体式泳衣下面半部分的样子开始变了起来。。。先是两条手臂围成了裤腰，然后两条腿渐渐往上，最后两只脚和手合并起来，变成了短裤的后面半边，而身体和头变成了前面半边，方向都是朝内的。
