# 1

今天是西元8787年 8月7號

一個下午...對 就是一個下午

我從剛放假的時間點就必須趕緊去打工

但是...我他媽只是在員工休息室睡個覺 ....

醒來

居然發生了這一件連我跟牆壁結婚或許我馬子都不會相信的事

我居然變得只有1cm....1cm

不過我擔心的不是這個...

而是我工作的地方——女僕咖啡廳

最慘的是...我只是裡面的新員工

完全沒有一個認識的人可以求救OAO

這時候巨大的腳步聲響接近

走進來的是咖啡廳最正的女員工...但她有遠視

齁呦~~又要上班了啦 我不想阿QQ

她在休息室抱怨著

(╯°□°)這畫面....

她開始將上半身的衣服脫去 露出了那纖腰秀項

皮膚白的如同嫦娥姐姐一般

頭髮更是秀麗的不得了

胸部又是那麼的令我嘆為觀止....(๑♡∀♡๑)

這時....不好的事情就這樣降臨在我身上 （囧）

你這臭蟲~看的很爽嘛(｡ì _ í｡)

我被她的手指捏了起來

看的那麼爽 都不會不好意思嗎

那我接下來就要懲罰你了 可惡 ‘ ︿ ’

我被她放到了她的乳頭上

剛開始的感覺是那麼的軟 如同在棉花糖上面躺著

但是我卻後悔了...不到1分鐘 她用她的手壓著我

一直來回搓揉著 發出了聲音超高的叫聲

乳頭漸漸的硬了起來 我被她壓的不得動彈

她越搓越用力 那個頻率連她自己都受不了了

阿~~臭蟲~~嗯~~爽不爽

阿~~阿~~阿~~ 姐姐的胸部如何

我已經無力的癱軟在那裡 但是更可怕的還沒結束

她把我又在次抓了起來

但是我已經全身都沒力的四肢下垂

她卻發出了淫淫的笑聲(๑¯ω¯๑)

哈哈哈 臭蟲 剛剛看的那麼爽 現在怎麼沒力了呢

差不多也該給你最終懲罰了 就死在我的幸福下吧

別...我是 （我最後有氣無力的喊著 但卻沒有用

她把我往下塞到了她的陰唇 開始來回摩擦

嗯~嗯~嗯~嗯~~~~

啪~~~~淫水濺了我滿身都是

她那副滿足的表情 我應該完蛋了

那味道 讓我意識更是模糊

好了 臭蟲 去死吧 我聽到的時候已經沒感覺了

只有聽到蹦的一聲

被丟到地上的我 那巨大的黑影

重重的壓在了我的身上

來回 來回 來回  阿 ~阿~阿~ 臭蟲 爽不爽阿

器官 被碾成一團  

這也是我最後聽到的聲音

血色就那樣留在了地上.....
