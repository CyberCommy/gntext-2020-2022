# 1

下面是無邊的大西洋，蔚藍的海水讓人覺得窒息，只有上方直升機的螺旋槳拼命提醒著時間的流逝。

“13號，有新的任務，請回總部。”

13號，我不喜歡這個數字，但是作為一名特種兵，能帶著這樣的番號經歷大大小小數十次任務而活到現在，也是件足以令人自豪的事。回想著接到任務的時候，我乘著自己的小艇在海上釣魚，現在居然又回到海上了。

對面的兄弟已經睡著了，我拿出這次任務的詳細說明，想再看看有什麼值得注意的地方，雖然出發之前我已經將其翻過千萬遍了。

任務：援救

地點：大西洋的某海島，諾頓伯爵私人研究所

要求：不明恐怖分子襲擊了研究所，營救島上研究所的諾頓伯爵和他的女兒凱林。最好全殲敵人。

任務相關：諾頓伯爵是出色的人體生物學專家，由於其出色的理論與實驗而被封以伯爵的頭銜，海島是他的封地，伯爵的女兒，出生於XXXX年……

伯爵的女兒啊，從照片上看很美麗啊，今年應該有16歲……我在想什麼，我是特種兵，不是王子，我還要為正義事業奮鬥……但是這無法阻止童年王子與公主的童話在腦海中閃過。

還有諾頓伯爵為我們提供的地圖，在恐怖組織入侵之前，伯爵將他的女兒轉移到了島上的某個避難所，需要我們先確認他女兒的安危。

海島終於出現在視野中，飛行員做了簡短的報告，大家都打起了精神。

這是個典型的火山島，島中央是一個不算很高的死火山，而山腳則是茂密的森林。而拒伯爵所述，研究所是建在火山的山體中的。

“準備戰鬥！”雖然我們已經很小心，但是登陸的時候還是被察覺了，直升機被火箭彈擊中，只有我和幾個已經下了飛機的隊員倖免，但是前面還有敵人向我們開火，我們只好邊打邊從側面退進樹林，至少那裡比較容易隱蔽。

敵人似乎不是單細胞生物，他們開始形成半個包圍圈，似乎是想將我們逼到某個地方，我們目前的人手也不足以反向突圍了，只好且打且退。然而我們終究漏算了一步，樹林利於我們隱蔽，同樣利於敵人隱蔽，就在我們的陣型變長的時候，側面又殺出一路敵人，我們很不幸地被沖散了。

現在只有我一個人，但是即使只有我一個人，我也不能辱沒了我胸前那閃亮的軍徽。我拿出GPS辨認了方位，不幸中的萬幸，我現在離諾頓伯爵所說的避難所非常近，進到什麼程度呢？我一轉身便跌進一個草叢，但是這從草下面似乎沒有土壤，而是鋼板！

“嘣！”聲音很響，隨即我聽到不遠出有人叫喊“什麼人！不許動！”糟糕，這裡還有敵人，我正準備舉槍反抗，腳下的鋼板忽然消失——準確的說，這是一扇隱蔽的門，而它現在打開了，我當然就掉了進去。

“啊！”自由落體之後，我摔在了柔軟的草地上，沒時間理會疼痛，我站起身來打量周圍的環境，這個地下洞穴有兩個足球場那麼大，而四壁和頂部則是完全靠樹根支撐著，地上長滿鮮花野草，如果不是頂部照明的燈光和周圍封閉的牆壁，我會以為自己在地面上。

洞穴的中央有一棟別墅一樣的建築，如果這裡是伯爵所說的避難所，那他的女兒很可能就在裡面。

我推開房門，裡面並不是想像中的豪華房間，而是一個研究所一樣的地方，裡面擺滿了各種儀器，而最讓我吃驚的，房間的中央有一個連接著許多管子，盛滿液體的透明罐子——或許我應該叫它養生倉吧——罐子裡有一個朦朧的人影，說是人，但是他的下半身有4條腿。

“上帝啊，真是不可思議！”我走進那個養生倉，更加不可思議的事發生了，雖然我早有預感，但是當我通過容貌辨認出裡面的人就是伯爵的女兒時，我還是下意識壓出了肺中的空氣，我仔細打量著伯爵的女兒，她的上半身還是完美的人形，但是從腰部以下，居然是梅花鹿的身子，雖然很怪異，但是我不得不承認這是完美的銜接，最美的人和最可愛的動物合在了一起，就是達芬奇在世也要稱讚這樣的傑作。

伯爵的女兒，或許在接下來的文字中應該稱她為凱林小姐了，她似乎陷入了沉睡，淡綠色長發掩蓋著的稚氣未脫的面龐上寫滿了恬靜，身上沒有穿任何衣物，但是從腰部延伸出一些帶葉的藤蔓，巧妙地遮掩了胸部和私密的部位。

即使在軍中被戲稱為“和尚”的不好女色的我，在這樣一個另類而完美的女孩前，也感覺內心裡有一根琴弦被撥動了一下。

良久，任務的概念才重新回到我的大腦，強迫自己把視線移開，我注意到手邊的桌子上有一個筆記本。

“X年X月X日：理論已經成熟，實驗也差不多了。哈，只有我這樣的天才才能想到用植物細胞與動物細胞的不穩定結合來消除排異。”

“X年X月X日：凱林馬上就能夠自由地奔跑了，雙腿癱瘓不再是奪取她的青春與幸福的兇手。”

“X年X月X日：凱林從小就喜歡梅花鹿，那確實是一種美麗而高貴的動物。”

“X年X月X日：結合似乎很成功，但凱林的新身體還需要調整。作為人她的內部器官都被保留了，那麼梅花鹿身體裡的內臟就不需要了。但是這個空間用來做什麼呢？總之應該有用的，也許……”

我正專心地看著筆記，但是身體一陣麻木打斷了我的專注。我意識到不妙，迅速回過身子——果然是敵人！應該是在我掉進避難所之前發現我的敵人！帶著面罩的敵人手中拿著我從未見過的武器，那似乎是一種毒素，我的意識迅速消散，在昏倒之前，我用盡最後的精力瞄准他的眉心，開槍，敵人中彈後慌亂地開槍，但是我的視線已經模糊，只聽到玻璃破碎的聲音，然後感覺到有清涼的液體濺落在我身上，接著是電流貫穿我的身體，我終於失去了意識……

稍微有了一點感覺，我費力地睜開眼睛，卻被進入我視線的一雙巨大的眼睛下了一跳，那是一雙碧綠得如同翡翠的美麗的眼睛，卻是如同車輪般巨大的存在。難道我死了？

我掙扎著站起，卻險些摔倒，因為我所在的地面是柔軟而有彈性的。我爬起來，發現自己身上的裝備都完好，只有中彈的地方衣服上有個破洞，而傷口已經不見了。台起頭，那雙綠色的大眼睛又映入眼簾……這次我定神把這雙眼睛的主人看了個仔細，結果讓我的下巴都快掉在地上了。

那張臉，分明是我前面還看見的凱林小姐，只是變得巨大了。我掐了自己一下，確認自己不是在夢境中。四下看看，我發現自己居然站在一隻大手上，這隻手無疑是凱林的。而四周的一切似乎都變的巨大了。我思索一會，然後逐漸明白了，一定是剛才的敵人慌亂中擊中了養生倉，但是所幸沒有射中凱林，然後營養液從破碎的養生倉流出濺到了我的身上，並且造成了短路，使我遭受電擊。本來我中了毒，應該馬上就要死了，但是可能是毒素的化學成分與營養液在電流的作用下起了某種反應，解了我的毒，也同時導致我變成現在的大小。真有這樣的巧合麼？我現在該怎麼辦？

為難之際，還是凱林小姐先開口了：“看您的肩章，應該是父親請來的援救人員吧，但是能解釋下您為什麼這樣……”聲音非常動人，也非常響亮——當然是對於現在的我來說，感覺就好像站在影院的音響前聽音樂。

“呃……這個，也許是這樣……”我把自己認為最完美的邏輯道了出來。 “很抱歉，現在的我沒法幫助您了，小姐。”我對自己現在的身體感到無奈，這樣的身體，怎麼去完成任務呢！

“還是非常感謝你，如果不是你的話，我可能還在沉睡中，現在當務之急是阻止他們獲得父親的技術，不然會造成非常嚴重的後果。”她所說的“他們”當然指那些不請自來的恐怖分子了。

“呃……那個……凱林小姐，這樣不太合適吧，在下太失禮了。”我示意她我還站在她的手上。

“噢！不要緊。”她似乎也才意識到，趕緊將我放回桌子上，臉上紅暈閃過，甚是可愛。 “叫我凱林就好了，請不要客氣。”

“對了，你父親之前有和你說過什麼嗎？”我向她詢問。

“恩，有的，他告訴了我他的一處秘密基地，他可能藏在那裡，也許那裡能找到對我們有幫助的東西。”

“好吧，我們就去那裡，幫我定位吧。”

“好的，地點在西經XX，北緯XX。”

待序……
