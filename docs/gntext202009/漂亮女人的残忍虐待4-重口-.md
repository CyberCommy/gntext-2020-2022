# 1

因为他们的老大已经死去，所以其余的九个小人只能是拼命的逃窜。第一个小人是死在亮宵的脚下的。当时这个小人一直在看这四个美女，可能是因为亮宵今天这身衣服是情趣水手服，那个小人觉得亮宵应该比较温柔，所以就向那边走去，可是没有想到还没有等他反应过来，亮宵已经来到了他的身边，他抬头一看，亮宵的小短裙里什么都没穿，突然一只大脚出现在自己头上，亮宵抬起自己的裸足，使劲跺下，表情和脚的力度上来讲没有一点怜悯。从视频上看，在短短的4秒不到的时间里，亮宵对着那个男人使劲跺了十下，而那个男人，也消失在了视频中。只有亮宵脚下的一滩血。。。。。。。

随后是第二个男人，相比较第一个男人，他就有点痛苦了。不用多说，他是死在韵篮手上的。韵篮从地上把这个男人抓起来，用手折断了他的双脚，但是并没有把脚扯下来。韵篮是个残酷的美女，她喜欢那些小男人在没有任何攻击力的时候，把他们慢慢折磨。韵篮把这个男人扔在地上，可以看出男人是非常痛苦的，他并没有向远处爬，因为他知道那样没用。所以他向着韵篮的蓝色网袜脚爬去，视频上一个穿着蓝色网袜的长腿美女站在那里，而脚下却有一个老鼠大小的男人向她求饶，这画面会让每个男人都感到兴奋。韵篮蹲下去，抬起右脚脚掌，让男人把自己的上半身放到脚下，然后轻轻踩住他，用双手直接扯掉刚刚已经断去的双腿，只见男人的双手在韵篮的蓝色网袜下挣扎。这时，韵篮笑了笑，便站起了身子，因为韵篮起立的时候全身都重量都放在了踩住男人上半身的脚掌上，所以那个男人就像一支香蕉一样，一下在韵篮的脚下软了下去，变成了一滩泥。。。。。

接着，韵篮抓住了第四个男人，她叫来张怡一起处死这个男人。她和张怡都躺坐在地上，然后伸出一只脚，把那个男人放在两只脚之间，左边是韵篮的蓝色网袜脚，右边是张怡的白色网袜脚，两个美女开始慢慢用力，两只脚之间的间隙渐渐变小，最后两只脚合在了一起，已经完全看不到中间还夹了一个男人。。。。。

终于看到了梦玲的出场，死在梦玲手上的男人是幸福的，也是痛苦的，那个男人不知怎么的就自己跑到了梦玲的黑丝脚下，男人抬头看着这个绝世美女，而梦玲也像一个夜场女神一样在这个小男人面前卖弄身姿。男人已经看傻了，忘记了逃亡。梦玲不断的抚摸着自己的豪乳，从腰再到胯下，在到腿。梦玲腿上的纹身，虽然隔着黑丝袜，但也显霸气。那个男人看到这里，突然反应过来了，但是他已经无法逃掉，等他一转身，梦玲便一脚跺了下去，踩在他面前，因为梦玲用力过猛，鞋的后跟已经深深陷入水泥地里。那个男人被吓得软坐在了地上。梦玲把脚从鞋里取出来，这时梦玲的脚底依然可以看到老大的血迹若隐若现。梦玲对着那个男人说，给她一分钟的时间，如果他能把鞋跟从水泥地里抬起来，就可以再死之前和他做一次爱。这句话说到了一个重点，就是这个男人怎样都得死。而死之前可以和梦玲做爱，那比较前面几个兄弟，还是值了。那个男人鼓足了劲，但是鞋跟还是一动不动。“时间到”，梦玲对那个小人说。那个小人一听，知道自己的下限到了，一下就开始往旁边跑。梦玲一脚下去。男人被踢开了很远。梦玲又脱下了左边的鞋子，走到男人面前，男人面前出现了一双黑丝脚，但是梦玲并没有踩死他，而是把他抓起来，把他的手用透明胶粘在地上，承大字型展开，穿回左脚的高跟鞋，走到男人面前，对着那个男人，梦玲只是笑了笑，左脚便一下踩了下去。金属鞋跟刚刚踩烂那个男人的头，并同样深深的插入到水泥地里，接着，梦玲抬起左脚脚掌，把全身的力量都压在左后跟上，金属鞋跟又被压下去了一点，这时，梦玲才抬起脚离开。

这时一个男人突然出现在梦玲面前。对梦玲说如果他能把鞋抬起来，希望能和梦玲做爱。梦玲想都没想便答应了他。梦玲和他来到右脚那支高跟鞋前。这个男人居然一下就把鞋跟拔了起来。梦玲看的很惊讶。突然，梦玲把那个男人逮了起来，在原地退下自己的黑丝袜和咖啡色丁字裤，在退内裤的时候，内裤底部居然湿了一大片，看来梦玲她们这群美女是相当享受这次的虐杀的。那个男人还没等反应过来。梦玲便一下把他插进了自己的下身，穿上内裤和黑丝袜，，，，，那个男人就这样被封在了一个满身纹身的美女YD内。。。。。。梦玲穿好后摸摸胯下，小声说“这就是我说的做爱，用你来做爱，哈哈哈，快，使劲挣扎吧，如果你能让我的水湿投我的内裤，并湿透我的黑丝袜，你就是小男人中最棒的，哈哈哈“

还剩五个男人，已经有三个男人被韵篮她们软禁起来准备最后的”表演“，剩下的两个男人便是她们现在的目标。

这两个男人一直逃，最后被三个大美女围在中间，一个被韵篮一把抓住，韵篮左手抓住男人的上半身，右手抓住他的下半身，然后扭成两截，扔在地上。。。。。。而另一个男人，也被韵篮抓住，放在张怡的白色网袜里面，那个男人在张怡的网袜里挣扎，弄得张怡很痒，张怡便慢慢把脚踩在了地上，而这个男人就被张怡踩死在网袜里，男人的残骸便随着网袜的小孔留到了地上。

而剩下的最后两个人，便被放进了一个容器里，这个容器有三根管子，现在已经拿在了三个美女手上，三个女人把管子的一头扣在自己的屁眼上，深呼吸一口气后，开始放屁，之间三个女人翘着自己的屁股哈哈的笑着，而容器里面的男人开始呕吐。梦玲这时候出现了，她给了三个美女每人一个灌肠器，让她们自己灌肠，灌完后，三个美女用肛塞把自己的肛门塞住，又开始对着容器撒尿，，，，，不一会儿，尿已经到了男人的腰部，这个时候，韵篮解开自己的肛塞，再次把管道口对着自己的屁眼，而张怡和亮宵也照做了。最后，睡着噗的一声，三个美女开始同时向容器里大便，，，，，直到把容器灌满，而她们，便蹲在容器旁边，看着里面两个小人被自己的屎尿淹死。。。。。。。。

视频到这里结束了。。。。。。梦玲笑着对我们说感觉如何，，，，，，结果没有一个人回答，梦玲也没有说话，只是邪恶的笑了笑，再次退掉了自己的黑丝和内裤，从YD里拖出一个小人，这个小人就是之前和她做爱的那个小人。。。。。。。梦玲把那个男人握在手里，目光看着我们，慢慢把手捏紧，血液便从手指缝里流了出来，韵篮把手一松，男人的残骸就掉在了地上，随后这四个美女也转身离开。。。。。。。
