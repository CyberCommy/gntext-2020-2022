# 1

我是一名刚刚过完大一的大学生，无聊的暑假和燥热的天气让我无处发泄，今天下午和同学浪了两把狼人杀，我成功成为了第一把被首刀和第二把被自己波的狼人给失误票死的倒霉催的孩子。给我个妹妹，我要回家。（唉，为什么我会想到妹妹？？？我没有妹妹啊。）
在和同学们告别后，我回到了家里，爸爸正在做晚饭，妈妈一脸严肃的对我说：我和你爸马上就要出差半个月，晚饭后就走，你妹妹已经来了，你一定要照顾好她。偶，我有气无力的回答到，一想到我又要自己孤独的生活了，心里一阵感触。等等，我好像遗忘了什么关键的东西？？？这时，后背上贴上来两片柔软同时伴随着一阵娇呼：哥，你回来了。我回头一看，鼻血差点喷出来，虽然天气很热，但你也不能穿这么清凉吧，那里简直连长裙都要包不住了，那天使般的面容怎么我感到带着一点邪恶。我飞速的跑到厨房问我爸：爸，我什么时候多了个妹妹。我爸一脸淫笑的回答我：这是你姨家的妹妹，但她和你没有血缘关系，她自己不知道，那岂不是你想干什么就干什么(=_=)。然后就发出了一长串的笑声hia hia hia我为什么有个这样的老爸（一脸蒙蔽）
晚上送走了老爸老妈，我哆哆嗦嗦的回到了家，心里想着是不是到飞哥家住一晚上，来逃避我这个看着就想让人犯罪的妹妹。回到家后，发现妹妹正在自己屋里做作业，并没有来诱惑我的意思，我放心的回到屋里去打游戏了。过了11点，我准备去睡觉，我偷偷来到妹妹的房间里，看到妹妹已经睡着了，突然，我看到了妹妹穿着白棉袜的脚，完美的脚型让我差点把控不住自己，妹妹一米七的个子有着39码的玉足，脚趾一钩一钩的，好像在召唤我过去这不是在诱导我犯罪，不行不能看，我决定去喝杯水冷静一下。拿着水杯去厨房接水，路过鞋橱，我鬼使神差的看到了妹妹穿来的Nike运动鞋，我的呼吸开始有些急促，心里仿佛有个声音在催促我去拿起来闻闻，但同时又有个声音说：哥哥怎么能闻妹妹鞋子那，被发现就完蛋了。最后理智战胜了邪恶，我决定就闻一下，肯定不会被发现的。对，肯定不会被发现的。

# 2

捧起妹妹的鞋子，看向鞋里，却发现一件不正常的事，鞋里怎么黑乎乎的，什么都看不见？我好奇的把脸靠近鞋口，想仔细观察一下，可这时我感到了鞋子里传来一股吸力，同时我眼前一黑晕了过去。当我醒来时我发现我处在一个陌生的世界，这是哪里？？难道我穿越了，地面软软的，空气中混合着女生身上淡淡的香气，还有一种说不清道不明的味道，但让人忍不住诱惑，我又多吸了两口。突然我身边传来一个声音：你好啊，小虫子。我当时就跳了起来，他妈的是谁吓唬老子。可四处我查看了一圈，并没有发现人影。难道我见鬼了？“小东西，在我手指尖上动的舒不舒服啊”，那个声音又传来了，我才察觉到了一丝不对劲，我脚下的大地怎么那么像一个人的皮肤。这时，我才看到天空中一个美丽妹子的面孔，不过这也太大了吧。“已经好久没有小虫子进来了那，女神难道现在才想起补充？可只有一个人进来怎么够啊。”她突然说话吓了我一大跳，“喂，我这是在哪里，你又是谁？”我问到。她好像不想回答我的问题，继续自言自语到“一个人我就可以玩了吧，估计玩坏也没事”我还正处于自己在一个漂亮妹子的手上这个事实的震惊中，就发现我在快速移动，冲着一个星球撞去。啊不要啊，这是哪里怎么还有星球，我尖叫着被扔到了地上，我这才发现这个星球上还是有居民的，他们都拿着一种害怕的眼神和愤怒的感觉看着我，我又没招惹你们怎么都这样对我？
