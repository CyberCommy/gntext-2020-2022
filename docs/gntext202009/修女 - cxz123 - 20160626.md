# 1

我是个无神论者，直到有一次我喝醉了把教堂的东西全砸了，一个少女的出现，我才开始信神。
那一次我酒醒后，发现干了什么后我很惊慌，这时候一个女孩子的声音传来，渎神者。我转过身去，看到一个穿着修女服饰的精致少女，少女长得很漂亮仿佛来自天堂天使，少女严肃的看着我开口道，渎神者承认自己的罪过吧，神会原谅你的。我看着她笑道，世界上根本没有神。不肯承认错误么，那么，少女喃喃道，接受惩罚吧。一道光芒闪过，看着眼前巨大的少女，我懂了我变小了！！！！！！！！
少女走到我的面前，脱下鞋子，露出了穿着黑色长筒袜的脚，一把捉起我放到鞋里，对着我说，渎神者要是你能在我脱下鞋子后还能活着，你的罪孽就洗清了，要是中途被我踩扁，但你就永远当我的鞋垫吧，所以祈祷吧。不你不能这样。我大叫，少女无视了我的话，把鞋穿上，我被巨大的黑色丝袜脚压住，少女开始收拾教堂，我则在她收拾的过程中，不断地被踩踏，极度的痛苦，我感觉到自己的肠子胃骨髓身体正在变扁，最后随着少女脚一用力我就被踩扁了失去了意识，我醒来以后发现自己躺在地板上，四周依然是那巨大的物体，修女少女看着我说知错了么。知错了但我还是不相信世界上有神。少女听到我认错笑一下但听到后面一句脸色立马变了，看来只能让你在死一次了，说完，再次把鞋子脱下，把巨大的黑丝袜脚放到我的身上，由于之前收拾教堂，剧烈运动导致她的黑丝袜脚，气味异常浓烈，我差点被熏晕，少女看到这一幕生气起来，先把我的四肢踩断，然后把脚掌放在我的肚子上，我不禁害怕起来，难道她想......我还没想完，她的脚开始慢慢挤压我的肚子，随着压力我的肚子越来越扁，在我的肚子快被踩扁时，突然将脚收回，然后又踩，反复几次我的肠子被挤压了出来，我一边吐血一一边痛苦的大叫，她看着我把整只黑丝袜脚放到我的身上，用力一踏，噗呲，我被踩成了肉泥，而少女依然在左右碾踩，直到我彻底被与少女的黑色长筒袜融为一体后，她才停下。
她沉默了一会，叹了一口气，没办法，再次把我复活后，我已经被她变回原来的样子了，少女看着我说，既然你一直执迷不悟，那我就跟在你的身边，直到你信仰神为止。我愣住了，你确定。我确定。那你别被我攻略了，我这个人除了不相信有神，其他方面可是很优秀的，也包括人格魅力哦。你放心修女是不会结婚的，而且在这期间你要是敢侮辱神，我就用之前的方式惩罚你。

几个星期后，我和她谈恋爱了。一年后，我们结婚了。
教堂外，我突然想起一个问题，我问她，樱，修女不是不能结婚吗？
（之前的修女少女）樱说，应为我特别啊，还有要是你敢花心你就死定了。
我，樱，有你陪着我，是我这一生最大的幸福。
樱（脸红），哪有这么好，好啦快进去啦。
走进教堂，一名神父突然大喊，圣女殿下回来啦。
