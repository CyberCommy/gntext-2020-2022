# 2

怎么突然变成这样了，真是伤脑筋呢。哥哥本来跟我说可以长高一点的，可是现在超过了一点的范畴吧。就不应该相信那种什么吃了这种药就可以长高的鬼话呀。不止可能无效，也可能药效过头，这是我给各位的忠告。我现在就遇到了这种药效过头的情况，比起来完全无效可是更加让人头疼。不了解的各位只要看了新闻，或者处于亚洲的某处，就可以看到我的身影了。这可是乱吃药遇到药效过头的后果。

好啦，不遮遮掩掩的了，因为服用了奇怪的药物，本来想要长高5CM的我突然变大了五百万倍，从185厘米高变成了九千二百五十公里高。打个比方的话，勉强可以说我的身高已经和地球的直径接近了。这样的情况可是预料之外的呀！想要长高到190厘米也只是为了达到加入篮球国家队的身高要求，而不是一个脚趾就可以抹平我所在的小国家的程度。

万幸的是，我的衣服还没因为突然长大而被撑破，所以就说那个奇怪的药物一定不是正常的东西了！哪有吃下增高药，衣服也会受到影响的道理啊？而且哥哥也没有受伤的样子，虽然变成这么巨大，但是还是能看清楚哥哥，还有他裤腿上的破洞很不可思议，不过就是这样的情况了。毕竟这是我的故事呢。而哥哥自然被我放在了一个安全的地方……我的运动鞋里面。如果你像我一样大，就会知道这样做的必要性，虽然我很确定他一定迷失在我的鞋里面了。

当然了，人类（请原谅我使用这个词来区分我和你们的不同）的政府是不会眼睁睁地看着我坐在太平洋里的。至少他们无法容忍我一屁股摧毁了要以三位数计算的舰队，或者阻碍了无数条航线吧。不过，这个世界上好像没有一个足够我休息的地方，而我相信，比起一屁股坐在欧洲上，你们一定更喜欢坐在大西洋里的我。

变成这样已经一天了，还是没有人来想办法帮帮我，真不知道都是在做什么呢。也许，我应该引起你们的注意，提醒你们我不是一座坐落在太平洋里面的山，而是和你们一样的人呀。嗯…也许可以略过一样这两个字，不过你们也不能就这样忽视我吧？

抱着引起注意的想法，我伸了伸腿，穿着棉袜的双脚踩到了澳大利亚的土地上，此前，我的运动鞋已经被我脱下来放在身旁。你无法想象那种感觉，我的两条腿横越了印尼而直接踩到了澳洲，同时我的屁股还坐在太平洋中没有移动。这种感觉让我觉得自己很伟大。人类坐飞机才能越过的距离，我连腿都无须伸直即可横跨。这样的话，也许人类会注意到我吧。

这个想法是很好的，对于我来说。可是我脚下的那些人有不同的想法。我能看到，感觉到他们在做些什么，这让我有些为这种行为后悔。在我双脚落下之前，被阴影笼罩的地区里面的人类都疯狂了，有些人拼命地开车，或者干脆靠着双腿想要跑出去，也许这样在他们看来很快，但是这么巨大的我，光是一只脚的长度就超过一千公里，宽度也接近四百公里，对于人类来说，这可是遮天蔽日的大小，许多岛国连这只脚都比不上呢。

或许是调皮吧，我故意慢慢地放下脚。在脚掌接触地面之前，最先抵达地面的应该是味道吧？因为运动鞋的关系，出了不少汗呢，在我闻起来味道的不小，对于人类来说就和毒气没两样了。潮湿温热的少女的脚的气味弥漫着，像是瘟疫一样肆虐着，让人们呼吸困难，甚至倒地不起，这还只是我连脚都没踩下去的程度。

人类真是太弱小了。

但是真正踩下去的一刻，才能体会到人类真正的弱小程度，或者说，这个星球的弱小程度吧。无论是高山，或者人类花费大量税金建造的大楼，和繁华的城市，都非常公平地在我的棉袜下崩坏。但是我却感受不到任何的阻力，真是可悲呢。超过五十个城市就被我一只脚抹平了。其中至少有三百万人呢。

一只脚就踩死了三百万人，我大概成了人类眼中的魔鬼吧？

应该说这样引起注意力的方法是有效的，因为我很快发现，许多舰队开始朝着我行驶。你们大概很迷茫吧？为了攻击一个女孩，是要朝着她的脚，或腿，还是屁股的方向前进，这种考虑应该是第一次呢。也有不少战机群飞来了，不过怎么都不觉得是来友好地谈判的。很快，人类就开始攻击我了，果然是不热爱和平的物种啊。有趣的是，无论是战机还是舰队，都拼命地对着我踩在澳大利亚的右脚猛攻着。大概是我太大了吧？你们都无法把坐在太平洋里的我和踩在澳大利亚的脚联系起来呢。

不过，这种弱小的攻击有什么意义啊？连我的脚指甲都无法攻击到，还妄想伤害我嘛？本来观看你们拼命地进攻我，结果连一点效果都没有那种狼狈不堪的样子是很有趣的，不过事事和我作对的人类实在不能安静一会，居然开始发射核弹了。笨蛋，这种对我毫无杀伤力的东西可是会对你们的同类造成伤害的呢。没办法，随便伤害这些本来属于我的人类是不可饶恕的，谁让你们迫害我的私有物品的，必须赶快阻止才行。

行动总是比说要快得多，这群被我定义为“破坏者”的小人们在一秒之内就被全部镇压啦。毕竟体型差距太悬殊了呢，用我的左脚来消灭他们都是一种浪费呀。也许是太平洋，不，全世界最强的舰队吧？但是在我脚下就显得弱小的可笑了，小人们也就是看着填满头顶天空的脚掌，连抵抗都不会想呢。

踩在舰队上的感觉和陆地不一样，深一两毫米的海水并没对我的脚产生任何阻力，而舰队在和我脚接触的一瞬就被踩破了。连海水里的潜艇也没能躲藏，全部被重重地压在和海底。

…“就这么结束了？”

几乎一点触感都没有，连海水都被我的棉袜吸收，搞得湿漉漉的，可是却感觉不到被踩破的舰队和士兵们。海底大量的烂泥粘在棉袜下，让我有种不舒服的感觉。就像是在下过雨后上体育课长跑一样，黏糊糊湿漉漉。这就是人类的无敌舰队嘛？连我的一只脚都不能抵抗，怎么样保护自己呀。但其实还是我太强大了吧，人类的力量怎么也没可能抵挡从天而降的大陆吧。

第一次沟通的尝试是在我踩破舰队两个小时之后，一艘在人类眼中相当巨大的军舰接近了我，虽然在我看来，连看到它都很难。军舰在离我二十厘米远的地方停下了，虽然那其实是一千公里的距离，不过我可以相信，如果它要接近我，会很浪费我的时间，而他们也会被我的体型吓到吧。听着从广播里面传出来的声音，我很无奈，一千公里远的距离，难道他们以为一般人的声音可能被听见吗？

忍住贬低这艘军舰上人智商的冲动，我耐心地听着长篇大论的讲话，无非是一些讲着法律之类的东西。大约十五分钟以后，我轻声问，“说完了吗？”就是这一声彻底毁灭了这艘军舰，从嘴里呼出的空气穿越了超过一千公里之后依然轻易掀翻了这艘军舰，钢铁要塞被飓风吹得飞起几公里高，然后重重地摔回海面，四分五裂。我不再看它一眼，因为可以确定没人能够生还。假装自言自语，我却以几千公里外的大陆都能听到的声音说道，“这么弱却和我讨论法律吗？”

第二次沟通迟迟没有到来，让坐着的我越发无聊起来。一个无聊的少女是很可怕的，因为你永远不知道她要做什么。虽然人类不能预料我要做什么，不过无论是什么样的动作，对于他们来说都是灭顶之灾吧。因此，当我把吸足了海水的棉袜从左脚上脱下来，然后随意丢到俄罗斯的时候，他们应该体会到了绝对的体型差距带来的绝望感吧。被一千五百公里长的超巨大棉袜压住，就算只是棉布也是压倒性的质量，你很难想象山被一只袜子压平的情景，可是现在它正在上演。袜子的余温融化大量的积雪，但是又被棉袜吸进去，足够人类饮用数十年的水量就这样被吸进我的袜子，变成混着汗液的脏水。

我把赤脚肆意移动着，拖在地表上。势不可挡的大脚随便移动都可以抹平几百公里宽的土地。无论是发达国家，还是发展中国家，穷人还是富人，在我脚下都是公平的。隆隆前进的巨大肉山不会因为你是有钱人就放你一马，也不会因为你穿着名牌就绕开你。我的脚抹过的地方，都留下了几十公里深，数百乃至上千公里宽的巨大盆地，很快海水开始填充这些巨坑，然后又被地下涌出的岩浆所蒸发。

这样玩弄了两分钟之后，东南亚靠近太平洋的部分已经在我的赤脚下被抹平。我决定换一个地方。屈起双腿，依旧穿着棉袜的右脚和赤裸的左脚并在一起，然后踩在了太平洋里，移动着重心，我慢慢把屁股抬离了地表，质量是天文数字的污泥粘在我的运动裤上，并不断地掉落。随着重心的移动，我的脚周围出现了宽阔得连一个城市都能掉进去的裂缝，轰鸣的“咯吱”声好像木质地板发出的声音，令人不舒服。接着，我慢慢直起腰，伸直双腿。第一次站直的我，骄傲地向全世界展示着九千二百五十公里的身高。

# 4

就这样而已吗？应该有几百万人吧？还有城市呢？山脉呢？

什么都感觉不到。无论是花费大量税金建造的大厦，还是存在了几万年的山脉，又或者是恐惧的人们。连降低我的脚下落速度都没有，就这样踩在了地上。有点失落的感觉。我已经大到这种地步了吗？

接下来，迟到的地震才开始缓缓波及四周，因为故意用力踩踏，地震的强度让我也不禁咂舌呢。先是离脚最近的地方，大约几百公里的距离，都全部开始下陷。在我眼里几厘米的深度，对于你们又是多深呢？

就在我扳手指头算的时候，一群小小的黑点接近了我。本来以这种大小，是看不到他们的，但是偏偏就是知道他们的存在，连他们的样子都一清二楚。

当然，这样也就知道，他们并不是很友善的主人呢。没有人会开着坦克和战机来欢迎客人的呀。人类眼中的战争机器现在看起来这么可笑呢，难道你们真的会相信这种东西能阻止我嘛？

连你们自己都不会相信。

开到最前面的战机开始发射导弹。曾经听哥哥说，这些大威力的导弹都是很昂贵的呢，几十万、几百万一个也是正常的事情。

这样想着，突然觉得很不好意思。麻烦大家用这么多昂贵的东西来对付我，简直和把钱往我身上砸没两样呢。但是，更对不起的是，这些浪费掉的钱，根本就没有效果。

又想起了日本小说中常见的那句话，“不要浪费纳税人的钱！”

就这样等待了十几分钟，经过各种轰炸之后，还是没对我造成任何损伤。连他们脸上不敢相信的表情都可以一览无遗。觉得有些无聊，我轻声说，“辛苦各位啦，不过，请不要挡着我的路哦。”

随后，带着满满的恶意，我把还在海中，穿着棉袜的右脚抬起，然后缓缓（当然是对我而言）移动到了军队集结的地方，这不是很难，因为他们的大小决定了他们一定可以被我的一只脚覆盖，尤其是我的一只脚超过一千公里长的时候。

在我踩下去之前，吸饱海水的棉袜开始释放汗液和海水的混合体。虽然我怀疑其中汗液的含量比海水还要多，味道也一定不是很好，不过这其实无论对于脚下的人类，或者我来说，都无关紧要了。

因为我决定踩下去，这样可以更快地结束他们的痛苦，也可以更快地扑灭希望的火花吧。

因为找到了一个借口，所以这一脚踩下去也格外轻松。最先被毁灭的是战机，虽然是以十几公里高度飞行，但是对于我来说，实在是太矮了呢。这么说伤大家的心了吗？没办法，就是这样弱小卑微的东西。战斗机群碰到脚底的时候就撞毁了，飞行员也全军覆没了呢。

到地面的十几公里对于脚下的军队是最漫长的吧？不过，再长的时间也会过去，所有的人类都下意识地举起双手来阻挡我的脚。真是可笑呢，那么小的人类，和我的脚对比起来，没可能注意到吧，居然还想阻挡我。对于被踩中的人类，这也许不是很可笑的实情，不过无所谓，只要我开心就好了吧。

搓动了一下脚趾头，湿湿黏黏的袜子紧紧贴在脚上。这种感觉让我觉得有些不舒服。只花了两秒钟，就决定把袜子脱下来。因为不想坐下，我抬起右脚，并以左腿保持着平衡，试图把袜子扒下来。因为重量的移动，脚下再次发出了吱嘎声，大量的裂缝也开始不断出现。

只能说这个做法是一个愚蠢的错误。贴在脚上的强悍程度超过了想象，拉了两下之后，我用力一拽，结果袜子虽然脱了下来，我也失去了平衡。

就这样，九千二百五十公里高的超巨大肉山崩塌了。朝着北美大陆，整个压了下去。

很奇怪，我清楚地感觉到，我身下的人惊恐地抬起头，毫无意义地乱喊乱叫着，并发狂一样乱跑。但是无论做什么，也改变不了他们的命运了。

好像一个世纪一样漫长，但是又好像瞬间就结束了，我倒在了地上。虽然不疼，但是却也感觉很奇怪。可以感觉到，盐湖城在我左膝盖下灰飞烟灭，芝加哥则被大腿抹去。波士顿、纽约和华盛顿在小腹下消失。

上半身很没面子地倒在了大西洋中。

接下来，和星体一样重的质量引起了无法想象的地震。整个南美洲四分五裂了，加拿大几乎被冲击力掀翻。海水被冲击后飞起几百公里高，地球历史上曾有，并将会有的最强海啸开始朝着海岸线推进着，大概几小时后，半个欧洲就会被呼啸的海水淹没吧。

几小时后，欧洲还能在我的玩弄下幸存吗？

终于站起来的我，浑身泥泞不堪，简直太丢脸了。上半身几乎全都是海底的烂泥和海水打湿的痕迹。头发也乱糟糟的。腿上则是大陆的黄土染脏的污渍。

不过，看到地上印着我形象的一个陨石坑，还是很有成就感的。

有些遗憾地看着加拿大和南美，本来是很好的玩具，因为我的疏忽毁掉了，真是粗心。我有些懊恼，下次应该更小心……如果有下次的话。

当我走到欧洲大陆的时候，我惊讶地发现这里的人们统一做出了一个动作，跪在地上，手放在脑后。我看过足够的战争电影，不过，这……这应该是投降的动作？

楞了一下之后，我偷笑起来，最后还是忍不住笑出声音。

那个是英国吧？还有德国人？勉强的话，可以看到其他国家，不过地理比较差的我，只能分辨出几个地方。即使如此，我还是觉得很惊异。他们居然可以放下尊严来对一个十六岁的少女投降吗？即便这个少女比普通人大一些。

这样做的其实是聪明人吧，当所有武器都不能让我感觉到的时候，只有服从才可以活下去呀。就算接下来要苟延残喘地活在我的阴影下，不还是活下去了吗？这样的领悟让我很满足，但是又让我有点失落。如果我是他们中的一员，我会这样做吗，为了活下去而放弃自己的尊严？

突然感觉到，自己好像是恶人。

就那样杀死了多少人？要以亿来计算吧？就这样无所谓一般，嬉笑着踩下去，把其他人都不当回事，这就是我的真实面目吗？以前那个会按时上下课，见到街头艺人都会停下来认真聆听的我，都是假的吗？

我茫然地向前走着，第一步踩到了欧洲大陆的中心部分，但是又怎么样呢？都已经害死那么多人了，再死几千万，几亿人也无所谓。

反正我已经是罪人了。

就这样两步来到了俄罗斯，这个星球最大的国家……不过如此。看着小的可怜的大陆，我发泄一般地抬起脚，准备用力踩下去。

然后，头突然疼起来，好像一根针扎进去一样疼。

再次倒在地上，我无法再去观察感觉自己压到了那里，一种奇异的感觉遍布全身，并不陌生，当初我变成这样的时候，也是有着同样的感觉的。然后，四周的景物在不断缩小，缩小，缩小……

疼痛逝去的时候，我面前不再是天空和大陆，而是一颗蔚蓝的星球。我看着比手掌还小的地球，有些自嘲地笑着。所谓的第二段药效居然还真的存在吗。制造这种药的人，一定是疯子。

我将两只脚抬起，然后放在这颗星球的左右两边。就算人类小的和细菌一样，我还能感觉到他们的情绪。那是一种绝望，和刚才不同，这是一种完全放弃，完全不想要抵抗，已经明白他们和我差距的绝望。连碰运气都不想，就等着灭亡的感觉。

我并没有再玩下去。毫无犹豫地，两只脚并拢，脚底踩在了已经被玩弄得乱七八糟的星球上。超过百分之七十的大陆面积就在我的脚丫底下被压平，从未发生过的地貌改变瞬间完成了。脚下的人们不可能生还，事实上，面对如此巨大的双脚，他们早已放弃了希望。

其他没被踩中的人类也没多好过，双脚的浓烈气味迫害着他们的神经，一滴脚汗落了下来，碰巧落在了日本。连天文数字都无法形容的汗水冲刷着这个岛国，无论是富士山，还是东京塔，都无法承受这可怕的力量，在眨眼间被冲毁了。人们狂叫着，被洪水带走，拼命挣扎着，试图能够逃出来。

在一个少女的一滴脚汗里挣扎，很可悲吧？

观察了一会，我终于决定，这个星球已经没有价值了。双脚慢慢用力，星球发出悲鸣，然后终于无法承受我双脚的力量，被压成一个难看的圆饼。又一阵轰鸣传出，然后整个裂成了几瓣。

我不再看这个我曾经居住十六年的星球，转过身，然后又想起什么似的，仔细看了看几片地球的碎片，终于找到了那个，我的鞋子还在上面。小心翼翼地拿起来，无需去看就知道，哥哥还在里面。

“我们走吧。”
