# 1

王霞刚拿起垃圾桶，薛媛就知道了，她要被当做垃圾被倒掉了，薛媛希望王霞在到垃圾时，能多看一眼，也许能看到薛媛。但是，薛媛在王霞的脚缝中间和王霞亲密接触时，都没有被王霞感知，现在被当做垃圾丢弃，能被王霞看到呢？薛媛在摇晃中抓紧了身旁的丝袜筒，在起伏摇晃的环境中，这是薛媛能想到的最安全地方了。柔软的丝袜现在成了薛媛的保护伞，保护着薛媛这个弱小的生命。虽然，那只丝袜，曾经就像牢笼，让薛媛困在了王霞的脚趾间。薛媛现在却对这只丝袜有了些依恋，那只丝袜上散出的淡淡的气味，也让薛媛感觉有些依靠。那曾是王霞脚上的味道，但是，这个味道，也让薛媛熟悉，薛媛感觉她自己身体也有着类似的气味，好像是同源。如果这只丝袜也有意识，薛媛似乎都能与其成为姐妹。过了一会儿，在剧烈的摇晃中，薛媛连爬带滚的摔倒了丝袜的底部，就是丝袜尖的部位，这时薛媛觉得她自己多么像王霞脚上掉下来的脚垢，被毫无意识的掉在了丝袜里。连同被王霞毫不在意的废旧丝袜，一起丢弃。王霞走到了楼道拐角处的大垃圾桶旁，将手中小垃圾桶里面的东西，一并的到进了罩有黑色大塑料袋的垃圾桶。在急速落体中，薛媛在这几天里第一次见到了光线，很是耀眼，她感觉自己好像坐上了垂直的过山车，顺着巨大的丝袜，一起落下去。还好，丝袜在桶底，被翻转后，却是落在了其余物品的最上方。薛媛的想法是对的，丝袜起到了很好的缓冲作用，虽然薛媛相当于从几十米的高出落下，但却柔软的落地了。薛媛赶紧调整了视线，赶紧朝巨大垃圾桶的上方看去，她看到了表情轻松的王霞，一手轻拿桶，一手轻扣桶壁，然后很是自然的扫视了一下掉落的垃圾，当王霞看到了薛媛所在的肉色丝袜时，露出了很不满意的表情，轻声说道，“太不结实了，穿两回就破，真是的，下回买些质量好的，”带着鄙夷的眼神，王霞轻松的转身走了。当薛媛看着王霞扫视垃圾物品时，尤其在丝袜上面略作停顿时，薛媛大叫起来，希望能让王霞注意到，薛媛真的太小了，声音太微弱了，王霞没有丝毫注意，她看到的只是被自己穿旧的丝袜，怎能想到里面还有一个带有自己丝脚气味的意识，正在用一双渴望的眼睛看着自己。

王霞轻松的走开了。当看到王霞巨大的面庞从垃圾桶的边缘移开时，薛媛还保持着仰望的姿势，她多希望王霞的相貌留在自己的眼前。但是王霞走时留给薛媛的眼神是鄙夷的，或者说是嫌弃的，薛媛觉得她自己真的好像王霞脚上的一只卑微的污垢，被主人扔掉了。薛媛无力的坐在丝袜里，淡淡的丝足味道还弥漫在薛媛的周围，好像提醒着薛媛，她的主人曾经是王霞。过了一会儿，薛媛冷静下来，在垃圾桶里最后只能被当做垃圾处理掉。要想办法从巨大的垃圾桶跑出去。白天会有很多垃圾扔下来，会慢慢堆满整个垃圾桶，这样就有机会顺着各种物品爬到顶端，薛媛知道她有机会跑出垃圾桶，虽然她不知道，现在跑出来及桶后，能去做什么。因为她太小了，那些巨大的同事们能发现她么？她都没有人家的高跟鞋底高，能引起别人的注意么？薛媛都不敢想，当她努力逃出垃圾桶，会被无意识的同事踩到么？会变成人家漂亮高跟鞋下的一粒尘土么？薛媛决定不想了，她慢慢爬出了丝袜，果然，一会又有不少物品被到进了垃圾桶。薛媛觉得有机会，就要努力，她小心的沿着物品边缘爬行，这样可以有效的保护她自己。时间很快，上午快过去了，垃圾桶里多了些饮料瓶，还有用过的医用棉花，医用废纸，棉签，甚至还有很多瓜子皮。渐渐的，垃圾桶已经填满了一多半，薛媛也渐渐变成了攀岩高手，她甚至可以利用一些物品的弹性，来跳跃式的攀爬。“还好没有什么恶心的东西。”薛媛劝慰这自己。这一次，薛媛又征服了一个巨型的废纸团。到了中午，薛媛还在复杂的垃圾结构里攀爬，显然，她的速度有些赶不上垃圾倒入的速度，她还没有到达所有物品的顶端。希望能快点够到垃圾桶的顶端。这时，薛媛听到了一个熟悉的声音。“蕊蕊，你怎么了。”“刚才买的包子好难吃啊，你看我在包子肉馅里好像不干净，好恶心，好想吐!”那是闫蕊的声音，导医台的一个年轻护士，什么，闫蕊要吐，薛媛听到了闫蕊和别人的谈话，啊，薛媛现在可是在垃圾桶里啊，难道闫蕊要吐到这里来！这时，薛媛看到很远的地方已经有一个巨大的半块包子飞进了垃圾桶，脚下的平面都跟着包子的砸入震了一震。“啊，”还没等薛媛反应过来，闫蕊的巨大脸庞已经出现在了薛媛头顶天空。哇，闫蕊真的吐了，薛媛透过上方有几片碎纸间的缝隙，看到了一片巨大的淡黄色的粘稠流体的快速的洒落下来，啊，薛媛赶紧抓紧了跳入了旁边的一个塑料袋的下面，还好，大量的泥浆流似的东西没有掉在薛媛的头上。但是，一些零散的还是砸到了薛媛的头顶，砸到了薛媛上方的塑料带上。这些闫蕊的呕吐物压着塑料袋带撞到了薛媛，虽然这些东西的实际重量很轻，但是薛媛更是微小。薛媛差点没有被砸晕过去。她好庆幸自己躲到了塑料袋下。要么，她就要被埋在闫蕊的呕吐物里了。那是多么恶心和危险的情况啊。随着闫蕊的呕吐，食物混合着胃酸的味道瞬间充满了薛媛所在的空间，好难闻，薛媛想吐，但是，她更要隐藏好自己，如果真的再被闫蕊的呕吐物砸中，薛媛不知道她自己能否挺得住，尤其是还带有胃酸，会被闫蕊无意识的消化在体外么？薛媛一遍干呕，一边找物品躲藏。还好，闫蕊不是有意找薛媛来的，所以只是冲着一个地方吐，过了一会儿，闫蕊感觉好些了，最后吐了一口，自言自语的说“真恶心，”就快速的走了。估计是去洗手间漱口了。可是，她却不知道，有一个同事去幸存在了她的呕吐物旁，身临其境的忍受着她呕吐的气味和温度。

等闫蕊走了，薛媛从遮挡的地方爬出来，她尽量爬向闫蕊呕吐相反的方向，薛媛知道她怪不得闫蕊，谁能想到，自己会变得这么微小，被丢弃在了垃圾桶里呢？什么时候才能从这令人作呕的地方出去呢？过了一个小时，薛媛爬到了一只斜躺的矿泉水瓶的上沿，她又一次的爬到了垃圾物的顶层，等待着后续垃圾的填入。薛媛头顶上是楼道天花板组成的天空。她太累了，只好躺在还算干净的瓶子上歇息，她凝视着头上圆形的天空，她忽然想起了修依，想起了自己工作生活的事情。修依还好么？他会在王霞的家里么？还是鞋里？还是和王霞的丝脚融合了？他还活着么？薛媛突然没了勇气，如果一会儿有人把什么东西倒在她的身上，她准备不反抗了，随便怎样，她算是什么呢？人家丢弃丝袜里的一点脚垢？在薛媛胡思乱想中，突然一只喝了半瓶的可乐被人丢进了桶，正好砸中薛媛所在矿泉水瓶的另一端，巨大的惯性和重量，让这只塑料矿泉水瓶弹跳了起来，而在其上表面的薛媛，就被巨大的弹力瞬间沿着斜上方弹入了空中，飞出了垃圾桶，薛媛想过她自己很多种逃脱出来的方式，但是，她怎么也没想到是这种情形。啊，薛媛在大叫中被弹入了空中。她不知道自己被弹向了何方，要落在什么地方。等待她的，会是什么物体？是美丽的过路护士？还是冰冷的医疗器械？或者是坚硬的水泥地面呢？她会得救么？薛媛子在大喊中闭上了眼睛，不敢看她自己将要被弹入哪里。如果是地面呢？她真的会提前变成美丽高跟鞋下的一粒尘土？薛媛全身收缩，紧闭双眼，等待着未知的落点。她会怎样呢？
