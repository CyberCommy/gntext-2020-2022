# 1

感觉写的有点平淡，最近又不敢写刺激的，等等看情况吧

这天，公寓里的一对夫妻有事出去，便拜托孝治照顾他们的女儿丽子。

“孝治哥哥，真高兴你来陪我。”丽子高兴的对孝治说。

“能和可爱的小丽子一起我也挺开心啊。”孝治低头看着可爱的小丽子美丽的面容有点心动的说道，同时觉得奇怪丽子怎么看上去比原来要高了一些。

“太好了，那孝治哥哥，我们来做一些我父母爱做的事吧。”丽子猛然对孝治说道。

“额，丽子，你还小，没到这时候呢。”被丽子的大胆吓住了，孝治连忙说道。

“哼，我已经不小了，孝治哥哥，你没发觉我和你的差距都比我父母的差距要小了吗？”丽子比了比自己和孝治的身高，得意的说道。

“额，确实是啊，你怎么突然长高了这么多啊。”孝治这时正好说出了自己觉得怪怪的地方，同时想起了丽子娇小的母亲和高大的父亲。本来丽子应该是比她那娇小的母亲还要矮一些的，可现在显然还要比她母亲高一点，才几天没注意，不应该长这么高啊。

“我只是用了父母没带走的的道具而已。”丽子得意的指着房间里的一个遥控器，孝治认了出来，那是最新款的大小控制器。

“妈妈经常把爸爸变得比我还要小呢，然后两人做一些好玩的事，他们没想到我在外面通过门缝看到了呢。”

说完后丽子拿起那遥控器又按了一下，顿时丽子变得更高了，比孝治还要高了一点，“孝治哥哥，让你说我小。”说着丽子又按了一下那控制器，孝治的视角发生了变化，变得和刚才的丽子差不多高，两人的身高一下子发生了倒转。

“哈哈，孝治哥哥，这下是谁小了啊。”丽子得意的对孝治说道。

“丽子乖，别调皮。”孝治举起手摸了摸丽子的头。

“哼，哥哥你还当我是小孩子。”说着丽子又按了一下那控制器，顿时孝治发现周围的东西又变的更大了，尤其是丽子，如今的自己身高只到她的肚子下方，就算挑起来也难摸到她的头了。

“哈哈，孝治‘大’哥哥，你这样好可爱。”丽子低头俯视着孝治，得意的说道，还回敬的摸了摸孝治的头。

“哥哥，你这样好可爱，让我把你变得更可爱点吧。”丽子兴奋起来了，她又按了一下那个遥控器，顿时孝治变的更小了，如今的他只到自己要照顾的可爱小女孩的大腿那么高。

“丽子，别继续胡闹了，听哥哥的话。”孝治抬头对丽子说道。

“哈哈，就算我坐着也没我胸部高的大哥哥吗。”丽子坐在椅子上，俯视着孝治笑道。

“看样子我得让‘大’哥哥认识到是谁在照顾谁呢。”说着丽子拉着孝治来到了卧室外，往洗浴间走去。

“大哥哥，能帮你的小妹妹打开热水吗，或者把洗发精拿给你可爱的小妹妹。”孝治看着高高悬在半空中的那些物品，明白了自己现在是多么的小。

“这下明白了吧，可爱的孝治小弟弟，现在你需要的是丽子大姐姐的照顾，所以要听大姐姐的话。”丽子带着孝治回到了房间，对他说道。

然后丽子让孝治站到了床上，“看，小弟弟，你就算站到床上还是没有大姐姐的胸部高，让大姐姐来照顾你吧。”

“好了，丽子，别闹了，快把我变回来吧。”孝治有点无奈的说。

“看样子你还没认识到啊，孝治小弟弟。”说着丽子又按了下那个遥控器，顿时孝治眼前的一切变的更大了，他发现自己还没有丽子的膝盖高，抬头往上看去，只见丽子的大腿如柱子一样耸立在自己面前。

丽子又把孝治抱到了床上站着，孝治沮丧的发现哪怕是站到床上，他还是只到丽子的大腿高。

然后他从床上走了下来，看着曾经的小女孩，如今在他面前就如同一个太古的女巨人。“好吧，丽子大姐姐，现在你要我怎么做。”

孝治认命了，他发现继续违背一个女巨人不是个好主意，抬头对丽子说道。

“这才乖吗。”丽子低头看着自己可爱的大哥哥，“让大姐姐给你奖励。”丽子有点脸红的说道，然后离开房间走了出去，孝治好奇她去干啥，可丽子临走的时候把门关上了，孝治没法打开，只好等待丽子回来。

过了一会儿，丽子回到了房间，这时候的她换上了一套成熟性感的内衣。“小弟弟，大姐姐漂亮吗。”

丽子在孝治面前转着身体，脸红的问孝治。

孝治张大嘴看着眼前的性感女巨人，真不敢相信那是他今天准备照顾的青涩小女孩。
