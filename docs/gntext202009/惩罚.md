# 1

终于从昏睡中醒了过来，头还是很疼，我仔细回想起晕倒前的点点滴滴，作为一个大学二年级学生我像往常一样没课就提早回我的出租屋和女朋友过二人世界，走在离出租屋不远的小道上问题发生了，我的身体仿佛被电击一样不能动弹，身后传来越来越近的笑声和高跟鞋踏地声“蹬 蹬 蹬”我失去了意识。

我观察我周围的环境，我的周围是一片粉红色的墙壁，脚下踩着是木质的地面，我这是在哪？

开门的声音传来，我听见两个很熟悉的声音“你确定他爬不出来？”

“放心吧 对现在的他这可不是件容易的事”

“要不是他手机忘在了你那里我们还没有那么快就能逮住他劈腿的证据”

“我想我们可以去尽情的玩弄他了”

是我女朋友小燕和她姐姐小玲的声音，我有种不好的预感，她们的脚步声越来越近，当看见两张巨大无比的面庞时我意识到我被缩小了，小燕的手抓向我，我根本没有办法抵抗，我知道的我刚刚只是在她们梳妆台的抽屉里，粉红色的墙壁只是她的钱包。

“亲爱的。你想知道发生了什么吗”小燕用一种极其蔑视的眼神看着我 对我说道。

“你现在可欺负不了我了欧 你最好还是听话一些 这样我和姐姐会让你少吃一些苦头的。”

我被她抓在手里不断的尖叫大喊谩骂 其实我怀疑她们根本就听不见我的声音。

“老实点 别乱跑 不然我不确定我会不会一脚踩死你” 这是小玲的声音。

我被放了下来 放在了地板上，我看见了我正对面的两双巨大的脚掌。她们都没有穿拖鞋，也许是为了更方便的玩弄我吧。小燕穿着灰色的丝袜，大脚趾不断的上下挑动。“滚过来，跪在我脚趾面前”我知道我没有反抗的余地，走过去跪在了她巨大的脚掌面前，望着面前巨大的脚趾我发现我甚至没有她的半个脚趾大，她想要踩死我甚至都不要抬起脚掌。“把你的身体贴着我的脚”我只能照做。用力把身体挤进她脚趾间的缝隙处，四周散发着一种奇怪的酸味，有余是夏天长时间穿着鞋子导致她的丝袜上已经被潮湿的脚汗浸透，现在我感觉我正浸泡在脚汗的海洋里。

“感觉怎么样 亲爱的 你以前不是很喜欢亲吻我这双脚吗”伴随着嘲笑的话语不停的讽刺着我。这时小玲穿着船袜的脚也踏在了我的面前，“也过来闻闻我的袜子吧”我顺从的爬了过去把脸贴在她的船袜上，船袜上的味道比小燕脚上小很多，只有一股淡淡的脚臭，我用力吸着这股气味，竟然感觉被缩小也不是一件坏事。
