# 1

因為太陽會發光發亮，所以沒學過天文學的人們不會注意到太陽比地球大一萬多倍；
因為護士小姐太過漂亮，所以患者們也注意不到護士小姐無微不至的護理；
因為校花國色天香，所以光顧著看美人的同學們同樣也不會注意到校花堪稱十項全能的能力。
有時候，人們總是會因為一件事物的某項特別突出的特征而忽略了其他並不差勁的特征，或者是其他注意不到的特征。
就像你在吃一個冰淇淋，你的目的是為了消暑，就因為這個冰淇淋實在是太冰了，你才注意不到這個冰淇淋並不符合你的口味——但是它冰涼啊，在這個冰涼的口感的引導下，你自然而然就忘記了冰淇淋那糟糕的口味。
在籃球場上，一位英姿颯爽，帶動著籃球，也帶動起全場觀眾的少女正在做著一個又一個高難度動作，配合著隊友，進了一個又一個的球，但是大部分人卻對這些熟視無睹——因為這位打籃球的少女實在是太漂亮了，好像“會打籃球而且打得非常好”只像試卷上無意義的附加題一樣不重要。
某項太強大的特質確實會掩蓋事務身上的其他小一些的特製。
一位少女因為氣場強大，就連眼神也好像可以殺人一般，大家就忽視了少女明眸皓齒、花容月貌的特點；另一位少女因為氣場很弱，所以即使非常的漂亮，也沒有人能注意到她。
而這樣一的另一位少女，她很巨大，如果她穿著涼鞋的話，你甚至沒有她的鞋底要高；而如果她沒有穿鞋，你同樣也不會有什麼改變，因為對於她來說，你只是一個兩釐米不到的小人——在一個一百八十米的女孩子面前，你只有她的小腳趾那麼高，就算你把頭抬得老高，也根本看不到她的臉，就像你的朋友站在在55層的高樓朝下叫你，但是你抬起頭來卻什麼都看不到是一樣的。
但你卻不覺得這位少女有著一百八十米那麼高，因為她的容貌實在是清麗脫俗，那完美到黃金比例也沒辦法比她更完美的身材更是無法形容；你覺得她就站在你面前，她並沒有一百八十米，而是和你一樣只有一百八十釐米。
你的認知思想一直在對你強調著：
“嘿！你個白癡，她的腳趾都能把你踩扁。”
但就因為少女的漂亮和氣質的甜美，你的認知放棄了繼續按照常理推斷少女的大小。
是的，誰也不知道怎麼形容少女的完美，因為她完美到可以讓人忽視她有著一百八十米的身高。
也許從五百米外你可能會忽視，但，但那是近在咫尺，連她的腳趾都像一堵肉色的墻壁一樣立在你的面前，甚至那巨大的軀體帶來的熱量、腳趾上的氣味都竄進你的探知系統里了，你怎麼可能會無視掉那巨大的身高呢？
但你就是無視掉了，在少女的容貌和氣質面前，她的身高你也可以無視掉，因為和她的容貌相比，這比全世界96%的建築物還要高的身高就像你憋了一秒鐘的呼吸對你的身體帶來的的影響一樣無足輕重，可以無視掉。
或者，也許剛認識少女的時候，你會因為她的身高而驚訝，甚至嚇得尿褲子。
但是，和少女相處久了之後，少女身上平易近人的鄰家少女的氣質，讓你覺得很親切；就這樣，時間一天天過去，你也一天天和少女相處……
終於有一天，即使你被少女放在那堪比兩個籃球場的手掌上也不會害怕時，你才注意到你已經很久沒驚訝少女巨大的體型了——你已經因為少女親切的氣場而自動無視了這個特質。
“有時候，我會忘了你有一百八十米高呢，哈哈哈。”
……
然而，這樣的容貌和氣質，到底是什麼樣的？誰也不會知道。
