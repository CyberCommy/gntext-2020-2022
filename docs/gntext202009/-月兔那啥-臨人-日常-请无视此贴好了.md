# 2

海岸邊緣，少年躺在水與沙的界線上，慢悠悠的想著事情。

就這樣舒服的躺著，不斷感受著溫潤的水湧上身體，把衣服弄濕，軟軟的粘攤在身上，帶著異樣的舒適。

就算是虛的海，還是可以帶來這種享受啊。

就這樣永遠的躺在這裏，什麼都不去思考，也不失為美好的生活啊。

少年打著這樣的念頭，頭上卻突然傳來冰冷的觸覺。

「該死，突然就變成暴風雨天氣了麼？」

少年忍不住咒罵了一句。

確實是突然的變天，傾盆的大雨毫無徵兆的來臨，甚至天都沒有完全暗下來，幾下的閃電把天空輝映得更加光亮，才逐漸暗下來。

少年跳起來，帶著狼狽不堪的樣子，跳上一臺倚向一邊的摩托。摩托的操控看上去很複雜，幾個不同顏色的計量表堆在小小的車頭，連普通的把手上也有幾個看上去很危險的按鈕。

但少年很單純的無視了這些按鈕，伸出右手，似乎是憑空拉出了一個控制面板的東西，上面的指令看上去很簡單：↑↓←→。

少年就像玩賽車遊戲一樣，一直按住↑鍵，摩托發動起來，噴出不太自然的灰煙，呼嘯著遠離暴風雨。

「登出。」

少年的身體變得破碎了，碎片化成一片片的星屑飄上天，然後消失不見。非要形容的話，大概是那種IMAX片的特技感覺。

「登出確認。」

冰冷的提示音後，少年睜開眼睛，強光已經褪去，望著自己周圍那些熟悉的亂七八糟堆雜在身邊的東西，內心莫名的失望了一下。

周圍的空間很小，亂七八糟的衣物和金屬器具讓少年幾乎轉不過身來。少年伸出手，把眼前的布拉開。

刷的一下，陽光刺進了這一片小小的空間。少年眯住了眼睛。

放眼望去，都是無垠的沙漠，大概是沙漠中心的風景吧？

但少年帳篷裏的地圖，卻給予了否定的答案。

滿地圖上，都是一個顏色。

倒不如說，地圖上除了標識城鎮的紅色，滿滿的一片土黃色——

那是代表沙漠的顏色——

世界，已經變成了一片沙漠。

目前是叫做新歷年的年代。

世界，在無數的輝煌下，終究歸於一片塵土。

給予裁判的是，一顆威力無法估量的氫彈。

「焚一切於虛無啊~」

少年從地下室裏爬出來，看著周圍只有火焰的世界，突然唱起了奇怪的歌。

人類，要陷入絕望了麼？

……

似乎還沒到這個程度。

就在那天前一天，被投入少量使用的「虛」，卻發揮出他的功效了。

「虛」的原理據說，是把人的靈魂與身體的聯繫切斷，接到另一個世界的軀體去。

聽起來天方夜譚的遊戲終端，卻救了不少人的性命。

但說是不少人，其實比起全人類來說，也不是很多。

有人說過：說話拐彎抹角的話，意思是無法傳達的。

所以非得直說的話，只能說，被「虛」連上線的，只有寥寥千人而已。

就是說，70多億人，如今只剩下千人。

少年爬出自己的帳篷，重新把帷帳拉好。沙塵暴把帳篷吹得嘩嘩作響，如果少年沒拉上的話，估計幾個小時後就滿是沙子了。

「啊喂——————」

少年突然放聲大吼起來。

然後，有誰聽見他的聲音而趕來了。

「轟隆、轟隆。」

每一步都會揚起很多沙塵，飛灑得四處都是，讓人有點看不清來人的身影。

但，就算是再大的沙塵，估計也遮不住她的身軀。

幾十米的高度，不是說遮掩就可以蓋住的吧？

那樣的一個幾十米高的女孩，就站在了少年面前。以少年的身高，連女孩的小腿一半都沒到，只是抬頭望著，也會很累吧？

所以少年沒有抬起頭，反而坐下在沙子上，目光隨便甩到某處，以有點不禮貌的姿態說：

「找到水果了嗎？」

「有哦。」

女孩用手撥了一下撩起來的衣服下擺，掉下一堆就顏色看來亂七八糟的水果，向少年頭上砸去，少年慌亂的抱頭躲避著，引起女孩一陣偷笑。

儘管少年不斷的躲避，也被幾個蘋果砸中了頭，痛倒是其次的，卻因為打下來的慣性而很丟臉的趴倒在地上，水果堆在附近，看上去總有種鄉村感。

「如果你更直白一點的話，應該說是在賣水果的時候不小心絆倒摔在水果堆上的老農樣。」

女孩蹲著俯視少年，口中吐出嘲諷。

「啰嗦！」

少年立刻反駁。

這是日常麼？

大概就得看日常是個什麼概念了。

若是以現在的生活來判斷的話，這就是日常了。

在「實」的世界裏，經歷著不可思議的事情。

而在「虛」的世界裏，卻是很平靜的一天又一天。

這就是少年，臨人的「日常」了。
