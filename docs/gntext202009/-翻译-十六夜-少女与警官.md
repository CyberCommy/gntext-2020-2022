# 【翻译】十六夜：少女与警官

作者：1945911

# 1

这是十六夜老师较早的一部小短文，比较平淡。——————————————「你已经被完全包围了！还不乖乖地投降！」「居然被要求投降了…」皱起眉毛表情显得为难的少女，脚下是无数的小警察和警车。在周围包围着自己的光脚，全体人员举起枪瞄准了自己。「我做错了什么嘛」少女把手叉到腰上俯视脚下的警察。警察们的身体在发着抖。这是一个1000倍身高的超巨大少女。即使眼前用力踩着地面的脚的小趾头也比自己乘坐的警车都大。如果警车被这个脚趾压一下的话眨眼间就会被压扁。「喂，你如果继续向市区入侵的话就真的开枪了！」「唔，没有那样的打算。只是散散步。一会儿就回去了」「骗谁啊？停止徒劳的反抗！」「那个额…」少女焦急地吧嗒吧嗒动了动脚尖。对警察们来说庞然大物一样的东西。那是能轻松覆盖一整片住宅街的脚。「没有去市区之类的打算啦。尽管如此你们还是这样围着我…。我说，这样的包围圈我能简单地跨过」有数千警察和数百警车包围着少女的脚。那个包围圈是里三层外三层的。不过，只要少女向前走一步就能轻松地跨过。「唉，好吧，我投降了，请赶快逮捕我。这样你们满意了吧」叹气的少女举起了双手，做出投降的姿势。几个比较勇敢的警察面向少女跑了过去。跑到了趾尖前，然后被那个压倒性的大小震撼到了。眼前，巨大的肉色的圆物体，都不过是少女的脚趾腹的底边。颤微微地伸出手，警察从腰间取出手铐。但这个小手铐又能干什么呢？　这是为了铐住一般情况的犯罪嫌疑人的手脚的东西。但是，难道要用这个把高耸入云的可怕的巨大少女铐住吗？铐住手？不好意思，手铐拿不到那个高度。铐住脚？即使连那个高度都也完全不行。手不行，脚不行，那脚趾呢？难道能铐住眼前比电车还粗的可爱的脚趾吗？警察摇起手铐。他该怎么办？警察只能仰视眼前巨大的脚趾头。「怎么？」少女俯视着过了这么久什么都做不了的警察。警察，望向那个俯视自己的少女的脸，她什么表情也没有。不论嘲笑、愤怒抑或焦躁什么的感情都没有，只是发呆地俯视着。只是因为大小的差距，警察的手铐就成为了毫无意义的东西。面对压倒性的力量的差距，被那个无力感包围的警察垂头丧气。心都碎了。站在这里已经没有什么用了。其他警察把那几个人叫回去以后，这次用上了数十台重机（啥东东？起重机？吊车？重卡？）。打算在这个巨大的脚趾上系上绳子拖走。现场谁对那个行为都没有疑问。他们已经被过分的力量的差距打垮，已经不能理性地思考了。缠上绳的工作开始了。但是，缠绳这样的事必须有谁爬到那个巨大的脚趾上。那谁去呢？所有人都退缩了,最后有几个人被强迫上去。消防车在脚趾的旁边部署，警察们用它来爬上去。眼前肉色的墙只是少女的脚趾的一部分，但是连消防梯的高度都够不到，最后总算勉强安全地上到脚趾的高度。警察们在那个肉色斜面上搭上手，在脚趾上面走着。确实能轻松地行走。肉色地面。这是少女的“小”脚趾，而自己则走在上面。走到脚趾的顶部的话，在那个对面能看见肉色的平原。那是脚尖。巨大的脚趾在眼前伸展着，警察们只想尽快结束工作。拿着绳子，警察走在那个少女的小趾头和无名趾之间。周围渐渐变得昏暗潮湿起来。在地面上的警察钻过小趾头腹下的隧道向外边出发了。现在，无法考虑自己行动有多么悲惨。只要稍微想一想，就能让意志崩溃。警察来到外边结结实实地系上绳与重机连接了。一圈又一圈地缠上，最终缠了有十圈，重机用绳子将少女的小趾头和自己连起来。这期间，少女一动不动地俯视着他们。他们的意图显现出来了。是要这样啊。少女还是不动，只是静静地看着。不久，重机的引擎大声轰鸣，牵引开始了。绳子突然绷紧，吱吱嘎嘎地呻吟。履带摩擦着地面。但是，重机完全开不动。烧着引擎，冒着浓烟。只有引擎的声音，和和履带摩擦着地面的刺耳噪声，除了那个以外，就没有别的动静了。少女的脚一动不动。少女看着脚下。绑上自己的小脚趾的重机拼命地拉着。脚趾感到被拉动的微弱触觉,不过，毕竟微乎其微。而自己都没有用一点力。绑上脚趾的重机升起浓烟地奋战着。少女往重机们拉动的反方向慢慢地调动了脚。于是重机们随着脚的活动被拖着向后。简直就像被主人拖着的宠物一样。真可爱。少女那样想着。伟大的警察们即使动用了重机也没有作用。如果不能逮捕，就只能当场击毙。这给予了警察们开枪的许可。「?」少女注意到了警察们正向自己的脚开枪。但是一点都不愤怒。刚才的宠物的感觉还残留着，少女简直就是用嬉戏的态度面对他们的行动。「呼呼，可爱」少女举起一只脚在他们头上悬着。是脚后跟只使之接地轻松的体态。警察们由于害怕跑出去了,不过还有一部分人拼命攻击着。少女倾斜了脚。只是稍稍降低了点，脚下就瞬间变得一片漆黑。脚底，还是有无数的子弹打进,不过，少女感觉不到。只是，想象着自己受到大量武器的攻击。不久，察觉到警察们的子弹已经打光，少女在原来的位置返还了脚。「叔叔们辛苦了。现在你们满足了吧？还是还想逮捕我？」少女再次动脚，用趾尖用力踩住一部分人。瞬间大量的警察和警车被压扁在了那个脚尖底下。其余警察一哄而散四散逃跑。少女笑着看着逃跑的警察再次抬起了脚。「这样的话包围就解除了。那么我回去咯」少女对着脚下的警察们摆摆手，把脚后跟翻过来走着离开了。
