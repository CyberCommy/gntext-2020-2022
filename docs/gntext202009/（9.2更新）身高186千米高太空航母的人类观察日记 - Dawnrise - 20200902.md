# 1

*本帖最後由 Dawnrise 於 2020-9-2 12:55 編輯*

**我从没有见过那样的人。**所以决定好好珍惜他。今天总部把他送过来时才告诉我，原来舰长不是消耗品，这是最后一个有舰队航母执照的军官了，如果照顾不好他，我就再也没有舰长可用了。为了记录最后的舰长能活多久，我决定写小人观察日记，直到和往常一样不小心把他玩坏掉。总部好像对我玩弄小人的行为感到很生气，但细菌一样的小人对我还有什么用呢，也不能当炮弹打出去，扔进反应堆当燃料又有点浪费了。
回到总部的时候，总部一直在路上叮嘱我不要玩坏他，说得我有点烦，所以前进三多挂了1秒，以超过规定的速度冲向了港口，用雷达能清楚的看到港口上细菌一样的小人四散逃跑，也不想想我真的撞上去了逃跑有什么用。不过这个舰长一动不动，好像一点也不怕我，数据库里他的名字叫林澈，很仙侠的名字，看起来也年轻，应该比普通的小人更有活力。
假装用手撑着码头的护盾停下来，护盾和鲜奶上面的皮一样一碰就会破，还好只是轻轻的碰了一下，涟漪都让小人躲在桌子下面了。不出所料的，总部又开始骂我了，说起来总部好像也没说过什么好话。停下之后，我没有把自己挂到港口上，很麻烦的，要不是怕捏碎细菌一样小的舰长，就直接用手指把他拿起来了。
让一架信使型运输机从嘴里飞出去降落到港口上，我想试试年轻舰长的随机应变能力，就把她的客舱关掉了。嘴是飞行甲板，胃里是机库的设计看起来很合理，但大量起降时总感觉怪怪的。过了半天信使才慢慢飞回来，港口刚才躲起来的小人都已经跑出来给我拍照了，真是一群色色的细菌。
从信使体内出来的林澈全身都黏糊糊的，看他狼狈的样子我确实给了他一个下马威，还有几个总部没告诉我是干什么的人也跟着他一起过来了，可能是赠送的消耗品吧。从甲板跑到舰桥之后，林澈在中控室翻看我的信息，还跟查户口一样抓到人就问，我告诉他在舰桥里有舰长的房子，刷脸就能进去，他果然进去了，真想今天就享用他，不过要矜持一点，离开总部之前把他吓跑了就不好玩了。
放到房间里没人陪他说话就开始和我说话了，问我多重，多高，体内有多少人之类信息里能看到的数据，看得出来是很努力的试图和我找点话题，打好关系，求生欲很强的样子。不过和查户口一样问了一些问题之后就开始往奇怪的方向跑偏，问我胸围多少，胸里面护盾发生器的值班情况，阴部里面L级舰船制造港的值班情况，然后更加过分的问护盾发生器的循环液体能不能当奶喝，制造港的爱液是怎么在L舰船进港时润滑的。
虽然有点想用流氓罪把他扔出去，但被主动问这些还是有点兴奋的，所以我都认真回答了，还给他拿了一瓶护盾循环液，他竟然真的当奶喝了。以往只有不小心掉进管道的小人才会被动的喝循环液，这让我乳头有点发硬，不过在内衣的包围下，除了胸部里面的小人没人会发现这个，小小的林澈也不知道。
怀着兴奋的心情，我用自己的爱液做了一份松饼给林澈，他完全没有发现异常，很开心的都吃了，还说我做得好吃，有其它地方没有的独特味道。这真令人感动，一直以来我对细菌一样的小人都是用道德经的无为而治作为行为准则的，不干涉他们内部的事情，让小人安心在我肚子里繁衍生息，偶尔控制体型小的舰娘去偷两个来玩一下，再把活下来的送回去，既不会让小人们恐慌，也满足了我自己的玩心。至于本体，只能用体型几千到一万米的驱逐舰和巡洋舰了，所以我很少用左手的港口，总让她们停在阴部的港口。
吃完松饼的林澈又跑出房间，去找他带来的小人了，那就不再去干预了。今天是见到林澈的第一天，写下一篇日记，期待着以后他能活多久，关系能有怎么样的进展，他会跨越种族和体型爱上我吗，像肚子里的宅男小人那样。想到这里，禁不住挥手，打断了总部的一根港口连接器，这次被占用了整整一万个信道挨骂，好讨厌。

*本帖最後由 Dawnrise 於 2020-9-2 00:26 編輯*

闲聊了好久才第到二天，我都没想好怎么温柔的玩弄林澈，时间一到起床点他就提出要检查我身体各部位运行的情况，明明在舰桥上都能看到，也许他只是想旅游一圈，最好不是对那些部位有不太好的想法。刚想到一些玩法，发现他带着一起上船的几个小细菌一起走，把我的玩法都打乱了，得等下把小细菌都除掉，不能打扰我们。
先把他们送到了嘴里的甲板里，都咬不到这么小的细菌，等等再除掉吧。林澈他们有说有笑的竟然直接跑到M级甲板上去了，这样就创造机会了，安排一艘轻型护卫舰降落到那里，果然把他们都压住了，不过等了一会全互帮互助的跑出来了，真失败。他们还大喊大叫的，跟受委屈了一样，真不知道谁才是受委屈的，换个地方再除掉吧。
到胃里的机库之后，竟然跟空军的师长遇到了还谈个不停，懒得想除掉他们的阴谋了，让师长闪一边去，派了几个没事的S级M级飞机和船去一对一把多余的小细菌都抓走了，告诉她们这是送给她们的小玩具，别玩碎掉就好。分出去几个信道看看她们在怎么玩弄，小细菌总是要挣扎和逃跑，真有活力。处理完灯泡一样的小细菌之后林澈变得很生气的样子，真不知道有什么好生气的，难道他想让我控制小型机体去玩弄他嘛，那我记住了。
林澈自己跑去坐电梯跑到了护盾发生器里，本来以为他想到了什么和胸部有关的玩法，却不听我说话，又爬了半天楼梯去乳头外面，还让我把一块内衣和衣服变透明了他要看着我的脸，我低头看了半天也没用眼睛看到他，真是太小了。一想到细菌一样小的舰长连我乳头里面的楼梯都要爬很久，就感觉乳头发涨，忍不住用手指隔着内衣捏了捏乳头，林澈就直接掉下去了，在护盾发生器的空管道里坠落了几分钟才回到乳腺深处，在他摔死之前才帮他减速，放下之后就被吓得和烂泥一样瘫在地上。
