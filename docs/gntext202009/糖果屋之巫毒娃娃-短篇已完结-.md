# 1

最近填坑还有挖新坑整的头痛，所以整个短篇冷静一下。“晴川，你这个笔袋好丑哦，我帮你扔啦！”男孩子说完将一个可爱女生的小熊笔袋扔进了垃圾桶里。‘可恶，又是这个家伙，总是欺负我。’我叫晴川，是个高中生，因为发育的比一般人要完，所以个子不高，只有155，刚刚那个讨人厌的家伙叫木，性格古怪，反正就是十分讨厌的那种，看我各自比较矮小就总是欺负我，今天又把我新买的笔袋扔了，真的好想报复他。糖果屋的女巫，这家在小巷角落里的店据说不是卖糖果的，听爷爷说这里住着个女巫，专门替人下巫术的那种。“有人在吗？”晴川小心翼翼敲了敲门，“没有人回应诶。”我推开了门，一股古老的气息扑面而来，据说这家店的女巫比爷爷的爷爷年纪还要大，那该多老啊。晴川小心翼翼的走进小店，店铺里的桌面上放着一个玩具娃娃，大概十厘米大小，“有人吗？”晴川又稍微大声的向屋里面喊了一声，顺手拿起了玩具娃娃看了起来。“很喜欢这个娃娃吗？”突然从里面的房间传出一声女声，晴川吓了一跳，赶紧把手上的娃娃放回到桌上。“您好，请问您是糖果屋的女巫女士吗？”从里面的房间渐渐出现了一位女性的身影，曼妙的身形看起来不像是一位老人家呀，女子走了出来，看上去完全不到30岁啊，晴川心想，好漂亮的大姐姐，不会是也是来这里找女巫的吧。“请问女巫今天不在吗？”“啊，还没有自我介绍，我叫墨羽，是这家店的老板。”女子回道，“所以你今天来是想要报复那个坏孩子吗？晴川小妹妹。”“对！没错！”晴川立马回答道，“诶？你怎么知道我的名字的？”“这个娃娃可爱吗？”墨羽拿起桌上的娃娃，并没有回应晴川的问题“可爱。”“这个娃娃可不简单呢，他叫巫毒娃娃，可以帮你实现梦想哟，让我来教教你使用方法吧。”女巫脸上露出的阴森的笑容，而刚刚晴川也没有细听，好像里面的屋子里传来了若隐若现的人类呼喊声。“真的只要在纸条上写上木的名字，就可以了吗？”已经放学之后，晴川一个人留在教室里，拿着巫毒娃娃观摩着，‘今天又被那个讨厌的家伙欺负了，一定要好好报复他。’“哟，这不是晴川吗？”教室外传来了熟悉且让人厌恶的声音，“怎么还没走啊？难道是因为想我了吗？”‘果然这个家伙一直在外面等着欺负我’我也没打算给他任何机会，将写好他名字的字条贴在巫毒娃娃身上，“呐，你看这是什么？”我晃动着手里的娃娃，故意给木看到。“不就是一个破娃娃嘛，也太丑了，快让我帮你扔掉。”说着，木伸出了右手想过来抢走娃娃。‘右手是吗？’我左手拿着娃娃，右手狠狠的捏住了娃娃的右手，用力一扭。“啊！我的右手。”木痛苦的抓住右手，倒在了地上，他应该看到了我刚在做了什么，“你做了什么？你这个婊子！”“哈哈，舒服吗，被人欺负的感觉。”我又扭了一下娃娃的左腿“啊！”木痛苦的抱紧左腿，“对不起，我错了，求求你放过我。”木意识到了我手里娃娃的恐怖，赶紧向我求饶“那可不行，你之前欺负了我这么久，我得好好报答报答你呀。”我把娃娃平放在地上，把女巫姐姐给我的药水取了出来，滴在了娃娃身上木在一旁看着晴川把一小瓶黄色液体滴在了娃娃身上，他惊恐地等待疼痛的发生，可是痛苦并没有如期而至，而是一阵眩晕袭上了大脑当木在睁开双眼的时候，发现自己已经不在教室里了，眼前的空间很大，木坐了起来，“可恶，这个婊子，头还是有点晕。”木晃了晃头，发现这里好像是一个巨大的房间，是个女生的房间，“巨大的床，”木小声嘟囔着，一边环顾四周，“巨大的门，还有……”“你醒啦？”“巨大的晴川！”木转过身，看到了惊为天人的景象，巨大的晴川就蹲在他的身后，晴川的身边放着那个可恶的娃娃，但是那个娃娃已经变得无比巨大了，比晴川还要大一点，“简直就是，跟之前的我一样大……”“哈哈，你那芝麻大小的脑子反应还挺快嘛，你和巫毒娃娃互换了大小哟！”晴川说着，站起了身子看着晴川站了起来，木瘫软在了地上，他充分感受到了身体的差距，也深深的感受到了绝望，晴川抬起了自己穿着白色学生袜的巨脚，阴影笼罩在了木的身体上方“不！！！”巨脚如同山一般压下，木被巨足按在了地板上，巨大的压力让他喘不上一口气，窒息感让他两眼发黑，“晴……求……”他用尽全身的力气想要求救，可是却发不出一点声音“呐，完全不能动了嘛，感受到自己的弱小了吗？大坏蛋！”晴川又回想起之前被欺负的时候，把自己全身的重量全部压在了右脚脚下的木只感觉身上的压力不断地增重，眼前一黑，失去了意识晚上，晴川认真的在做着老师布置的作业，穿着自己可爱的兔子拖鞋，只不过在拖鞋的里面，巨大白洁的脚丫下面，压着一个十厘米的小人，巨大的脚掌轻轻地压在了小人的身上，让小人动弹不得，小人只有脑袋露了出来，被夹在巨脚的脚趾之间，小人用尽全力的抬起头，奋力的舔舐着巨脚的脚趾缝“呐，好好舔哦，不然我可要踩死你咯。”晴川一边写着作业，一边漫不经心的说着。脚下立刻就传来了酥痒的感觉“作业做完啦！”“求求你，把我变回去吧，我以后再也不欺负你了。”木跪在拖鞋上，望着头顶的巨脚，以及巨脚背后遥远高空中的那张娇美的脸。“不可能！”如同死刑的审判，木痛哭着，被晴川捏在手里，晴川拉开自己的鞋柜，把木放进了自己今天穿着的长筒靴里，又把今天穿着的袜子也塞了进去，“呐，这里以后就是你的住处啦，今晚就盖着我的袜子睡觉吧。”晴川踩着兔子拖鞋，蹦跳着去洗漱了，完全不在理会自己靴子里痛哭的木……“诶，你听说了吗？木失踪了，他已经好几天没来学校了，据说他父母已经找了私家侦探了。”“真的吗？据说是那个赫赫有名的大侦探，小五郎先生呢”晴川默默听着同学之间的交谈，嘴角露出了一丝微笑……糖果屋里：“请问女巫在吗？”一个漂亮的小女孩走了进来，顺手拿起了桌上的放大镜。
