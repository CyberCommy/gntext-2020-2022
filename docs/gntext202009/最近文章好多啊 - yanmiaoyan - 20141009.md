# 1

"总觉得快累死了."一从门口传送到自己的房间,我差点就瘫坐在地上,对我来说,为了买到新游戏,今天实在遇见太多人了......
   哦,我是谁,我的名字叫安田响,是个高中勉强完成了注册却没有去过的家里蹲,不过现在由于科技的发达,像我这样的人也不少了,而且由于最新的缩小传送技术的出现,只要一进家门,我就自动被传送到了自己的房间,由于设置的缩小10倍大小的缘故,原来小小的房间现在是如此的宽敞,让我更加不想出去,有个缺点就是由于输水管道不好布置,导致我洗澡之类还得走出自己的小房间,到外面那相对巨大的房间中去.
   我本来想钻入被窝休息,但可能是精神太亢奋,身体很疲劳却毫无睡意,这种时候,洗个澡放松一下或许是个好选择.
   我下定决心之后便走出了自己的房间,来到了外面那个对我而言的巨人的国度,我的家人没有我这样充分利用空间的自觉,只设置成缩小差不多2.5倍的样子,所以对我而言就是来到了一个四倍大的巨人之家.
   我来到了浴间那边,还好家里人知道我设置成了不同的大小,所以浴间门一般不关,我用力推开了浴间的门,按下了智能放水按钮之后,便先在边上洗起脸来.
   "咦,怎么自动放起水来了......喂,你在干嘛啊,哥哥?"
   这时候一个可爱的女孩从里面的浴间走了出来,是我的亲妹妹安田洋子.
   "嗨,嗨.抱歉,我想放松一下......"
   "想放松,你没去学校,整天都在放松吧?"
   妹妹满脸不服气,目光锐利地低头瞪着我,配合她那足有我四倍的体型,让我无言以对.
   自从装了缩小传送门之后,我大多数时间待在房间里,所以一直没有与妹妹再见面,没想到第一次见面就是在这种尴尬的情况下,而且那巨大的身体也让我感受到了极大的魄力.
   妹妹整理起了自己那还湿润着的长发,看样子她也正准备享受热水澡,想不到却有人打扰,我完全能够体会这种不悦的心情.洋子一语不发,又走进了浴间.伤脑筋啊,该怎么讨她开心呢......
   "真是的,一个大男生傍晚就泡澡,这太奇怪了,你还发啥呆,快进来啊."
   意料外的话语让我不知所措,呆在了原地.
   "还不快进来!"洋子加重了语气.
   "......那个,你要我也进去洗吗?"
   "干......干嘛?你可没有拒绝的权利哦!我都是在这个时间洗澡！是哥哥你的不对,但大度的我就让你一起洗吧."
   洋子看我一直不进去,又走了出来,气呼呼的用双手抓住小小的我,带进了浴间里,大概是感觉低头看我太吃力了,她跪了下来,而这更让我感觉到了自己的渺小,因为就算如此,我的身高也只能到她的大腿.<title>2</title> <link href="../Styles/Style.css" type="text/css" rel="stylesheet">
