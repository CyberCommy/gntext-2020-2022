# 1

一

我愿成为你鞋子里的一粒尘埃，

当你穿鞋时，脚趾将天空压下，

那是多么壮观的场面。

你便是我的神，

你的鞋子便是我生活的世界，

你动下脚趾，便是地动山摇，

当你抬起脚，天地星辰为之转动。

二

我被一只蚂蚁追杀，

路过的你轻松把它踩死，

比珠峰还大的鞋子，

彰显着我的渺小，

矗立于天外的小腿，

那样洁白神圣。

你笑着伸出手指，让我爬上，

你是那样强大，主宰一切，

成为你的宠物男友真是幸福。

三

我喜欢抱着你，

更喜欢抱着你的手指，抱着你的头发。

四

你便是我的神，我的全世界，

除了你，我何须世上其他。

五

我踮起脚尖，

吻你的脚趾。

等我以后有女朋友了，要介绍性癖时可以发给她感受下
