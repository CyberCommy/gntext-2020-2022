# 1

這是發生在小光和他妹妹手術成功後6年的事了。

妹妹今年12歲，剛好小學畢業。壹年以前，小光發現妹妹的身體開始有了微妙的變化，比如嗓音變得高而纖細，胸部開始隆起，自己生活空間稍微寬敞了些，後來那條小縫周圍也開始長出淡黃色的絨毛。

雖然那還不是真正的陰毛，但是小光很興奮，因爲受過壹些“教育”的他知道，妹妹發育了，不久的將來，她壹定會更迷人。

終于，暑假的壹天，小光看到了那期待已久的壹片血紅——這是妹妹的初潮啊！小光興奮極了。

如果說以前妹妹的私處有壹顆含苞待放的花蕾，那麽現在，這顆花蕾開始綻放了，它將要生出漂亮的花瓣，流出醉人的花蜜……小光興奮地想著，雖然他覺得妹妹還小，自己這麽想很不道德，但還是常常抑制不住心跳加速，在那蜜穴中撫摩壹番，而妹妹則體貼地用溫軟的肉壁抱以按摩。

伴隨著時間的流逝，蜜穴中的氣氛越來越醉人，但是小光卻發現自己無法真正地“興奮”起來，自己發育得應該比妹妹要早，但自己卻從未失控過，這是爲什麽。每每想到這裏，小光總覺得懊惱，也覺得慶幸，懊惱是因爲自己失去了正常人的情感，而慶幸是因爲他還無法確定自己與妹妹之間的感情，現在的自己是無法對妹妹造成終身的傷害的。

直到有壹天，妹妹去做體檢，小光看到了體檢的門診，腦海中靈光壹閃“我好象有聽那個做手術的阿姨說‘等妳妹妹發育了再來找我’。”對啊！自己怎麽就把這件事忘記了呢。

于是，小光第二次領妹妹去了那家診所，與6年前相比，診所毫無變化，“這些人有夠低調啊。”小光這樣想。

有見到了那個女人，小光向她道出了自己心中的疑惑。

“不用擔心”那女人回答，“我是考慮到先發育的妳可能會對妳妹妹過早地造成傷害，所以‘封印’了妳們部分的機能和情感，現在看起來時機成熟了，我叫妳們回來就是爲了恢複妳們兩人所有的情感與機能。”

“可是”小光有些疑惑，“現在妹妹才12歲，我們做那種事會不會太早啊”

“呵呵，妳誤會了，不是妳們，而是妳妹妹。”女人答道。

“我妹妹？這又是怎麽回事”

“是這樣的，”女人顯然很耐心“發育的少女，尤其是妳妹妹的這個年齡，不容易控制自己，常常會有激動的行爲，比如自慰行爲，但是妳妹妹對這方面的知識還是空白，不科學的自慰不僅不衛生、不健康，而且可能對自己造成無法彌補的傷害。妳知道嗎，有的婦女用活鳗魚自慰，結果被咬破直腸喪命了呢。”

“這樣啊，那妳不准備恢複我的全部能力喽？”小光似懂非懂地問。

“可以這麽認爲，但是我要賦予妳新的能力，手術完後我再和妳詳細解釋。”女人笑著說。“又賣關子。”小光心理想

……

兩小時後：手術完畢。

“現在妳可以告訴我了吧。”小光急切的問。

“好，妳仔細聽。從現在起，妳可以自由控制自己身體的大小、形狀。”“這是爲什麽？”

“不要插嘴，聽就可以了。作爲哥哥，妳現在要挑起另壹份擔子，每當妳妹妹情緒激動的時候，會有神經刺激傳到妳的大腦中，這時候妳就要運用好妳的新能力來撫慰妳的妹妹。尤其是晚上，要避免她不正確的自慰，比如手淫，妳明白嗎？”

“那麽，我的新任務就是當妹妹的按摩棒了？”對于自己的新身份，小光並不覺得難以接受，雖然自己的任務只是當個工具，但是從前不也是如此嗎？早在第壹次手術時，他就把自己的壹切毫無保留地奉獻給了妹妹。其實他不知道，給妹妹這樣的美人當按摩棒，是多少人求之不得的事啊。

“不要這麽想，妳做的事是爲妳妹妹好，妳不會委屈的，我們再次修改妳的皮膚神經，在妳妹妹亢奮的時候，妳的皮膚會變得敏感起來，到時候妳就知道了。”

……幾年以後

小光已經能夠熟練地完成自己的新任務了，而自己與妹妹的感情也在悄悄地激增，是兄妹之情，也有壹點朦胧的暧昧。

貴族學校的校舍是單人間的，晚上房間裏沒有別人。因爲妹妹把自己的房間布置成粉紅色，並且把床做得寬大柔軟，在這種環境下十分容易“進入狀態”，所以小光的任務幾乎天天有的執行。

又是壹天晚自習結束，妹妹回到自己的寢室後簡單地洗漱後就寬衣入睡了。燈光熄滅不久，那熟悉的輕柔的呻吟聲又經過妹妹的肉體傳到小光的耳朵裏，小光知道自己該幹什麽。

首先，他從妹妹微緊的小穴探出頭，不知道是做手術的人動了手腳，還是妹妹天生麗質，那樣的事做了不只多少回，妹妹的小穴卻依然紅潤而富有彈性，所以小光的工作量也維持不變，這如果讓其他女生知道，恐怕會嫉妒得要死吧。

環顧四周，小光不禁感慨，如今妹妹已經15歲，當年小穴周圍的絨毛早已經長成黑亮順滑的長發（說是長發是因爲妹妹陰毛的質地太好，就和真的頭發壹樣柔順，有光澤）。撫摩著這些順滑而且疏密有致的陰毛，小光不禁想起妹妹沐浴是自己替她梳理陰毛的情景，他細心地將每壹根陰毛分開梳理，使它們不至于糾纏而折斷，所以妹妹的陰毛不像普通人壹樣錯雜無序，而是像歐美人自然卷曲的頭發壹樣，妹妹也常贊歎這樣的傑作，是以小光還變成了妹妹的“發”型師。

“開始喽！”小光賣力地用臉、用手臂，用整個上身摩擦妹妹的小穴，這是替妹妹做准備活動，而妹妹早就習慣不自己動手，把自己的身體交給了這個疼愛自己的哥哥，她知道哥哥會盡力滿足她的。

不久，小穴周圍已經濕漉漉的，而且兩片花瓣壹張壹合，小光知道花蜜就要傾瀉而出，所以敏捷地從床邊抓過早就准備好的道具——壹個吸氧器——這是小光突發奇想問醫院買的，本來用來罩在口鼻上的氧氣罩剛好能罩住妹妹的小穴，氧氣罩的另壹頭接著壹個水壺，這樣，妹妹的花蜜就不會沾濕床單了。

感覺到身後的洶湧，小光停止了前線工作退居二線，他把全身藏入妹妹的尿道只露出頭在陰道裏，花蜜洪水般湧過粉紅的長廊，也只有在小光這個角度才能如此別致地“觀潮”。這還不是最受用的，他那小小的頭顱轉瞬間就被洪水淹沒，然後沖擊，甜蜜的淫水包圍著小光，但他卻很清醒，壹點淫亂的感覺也沒有，只是覺得很舒服，很爽快，這要歸功與那奇妙的手術啊。

洪水過後，小光重新回到陰道。他伸手在肉壁上推了推，殘留的花蜜如雨般落下，如果小光穿著衣服，可真是應了那句“沾衣欲濕杏花雨”了。

感受著妹妹身軀的顫動，自己的皮膚也變得敏感起來，每壹次觸碰都好象有電流通過全身，說不出的暢快。估摸著妹妹已經進入了高潮，小光將身體陡然漲大，將妹妹的密穴撐起，然後有節奏地扭動身軀，或者將身體變小然後再變大，此刻，他是那麽的投入，就好象自己真是按摩棒壹樣。

小光閉著眼感受妹妹的動作，他能看到妹妹因爲陰部的刺激而嬌軀微顫，就連呼吸聲都在顫動著。雖然蜜穴的內壁劇烈地擠壓著，但是身體卻保持著睡姿，遠看她的臉，恬靜中透出壹抹微笑，但除了小光知道這微笑中包含著怎樣的滿足，誰也想不到妹妹的蜜穴正如壹顆初戀的心壹樣劇烈地顫動著，“自己的妹妹多麽有教養啊，多麽淑女，就連到達了高潮，也不將自己內心的激動輕易示人”，小光這麽想。當然，小光對妹妹的狀態是壹清二楚。

深夜裏，壹男壹女，壹大壹小，就這麽無聲的膠著在壹起，這個世界在此刻，只屬于他們兩人……

或許真的是手術的功效，妹妹的高潮持續了很久很久，但是消耗的體力卻及不上壹夜恢複的百分之壹，大概是由于不用妹妹親自動手，她只需要在夢中享受哥哥的關愛就可以了。

直到黎明來臨，最後壹波浪潮才湧過，妹妹睜開了睡眼，看著不遠處接花蜜的水壺滿滿當當，不禁臉紅了壹下：“又流了這麽多啊。”掀開被子，發現哥哥正躺在自己股間，享受著被窩中香甜的氣息，兩眼出神，不知道在想什麽。于是，妹妹趁哥哥不主意，惡作劇地將哥哥按進了自己的小穴。

“哇，妳幹什麽啊，又調皮！”裏面傳來哥哥不滿的抗議。

“想什麽呢，都天亮了，我要沐浴了（這是貴族學校按照貴族的生活習慣定的校規）。妳今天給我做個什麽‘發型’啊？”

“唉，這麽快就早上了，時間過的真快啊，要是時間能靜止多好啊！”

“想得美，壹晚上了還不夠嗎”說到這裏，妹妹不禁臉紅了，其實他們知道他們之間尚未發生真正的關系，因爲小光的分身連他自己都不知道在哪裏呢。

“啊，沒關系，反正今天晚上還有的！哈哈”小光調侃地說。

“要死了妳”妹妹的臉更紅了，但是也更可愛了，不再說話，徑直到門口拿了新鮮的牛奶——她需要補充壹晚的流失。

新的壹天開始了……
