# 1

**-------------------------------------战争的开幕------------------------------------** B国家边界上，
军人们都在严阵以待的面对这边境外的大海，
海面一片平静，看不出有什么敌人会突然出现，
然而就在一天前，A国的领袖受到了信息，
“马上投降，否则就踏平你们的土地--B国”，
国王无比的愤怒，决定抢先进攻B国，让他们为说大话而付出代价，
在B国的边境上，国王派人发出信息，
“B国国王，我军现在在你国家的边境处，我打算带领我的10万人马到你的首都转转”，
很快，便传来了回应，
“A国国王，虽然是我先对你们发动战争的信息，但是想到我们的首都转转，可没那么容易”，
A国国王微微一笑，又派人用电脑发送信息过去，
“那么试试便知道了”，
B国也不甘示弱，
“试试就试试，不过，你们的对手可不是我，而是她们”，
A国国王有点糊涂，她们？是什么？
“还没发现吗？亏你们10万双眼睛，连这个都无法看见，A国国王，你们马上就成为我的俘虏了”，
B国国王更加骄傲了，他在B国首都里，向她们下令道，
“姑娘们，请帮我击败他们，那样你们才能在这个星球上有国休息的地方”，
下令完后，B国国王回到卧室中，打算休息一会，在去听获胜的好消息。
而在此时，A国国王以及10万大军都在思考着，敌人在哪？
等等，地面有些震动，
等等，海面有些波涛，
等等，怎么突然出现了一些柱子？
有点不对劲.......
终于，悲哀的人们终于发现，
这些不是柱子，而是少女的腿，
亭亭玉立的少女，雪白的腿，不仔细看，真的会当成柱子，
这可是是正常人身高1000倍的少女啊！
军队开始慌张了起来，一个人接一个人，不敢想象这眼前的事实，
但是，国王的命令，依旧是战斗，
只是主要任务，是保卫国王罢了。
人们已看清楚，估计有10个少女，将他们包围了起来，
虽然很荒唐，但是仍旧是10人包围了10万人，
“恩！姐妹们，可别让他们跑了”
一个短卷发的少女，18岁左右，第一个走上前去，面对着依然反抗的人们，
上去就是一脚，
几百米长的鞋子，将来不及举枪的士兵与来不及逃命的士兵一起吞没了，
一下就阵亡了一千人左右，
国王慌了，这次是脸上的慌，是藏不住的慌，
“学姐，昨天那个人让我们消灭这只军队，我还以为有多困难呢？”，
“没办法，谁让那家伙威胁我们，说不帮他就用强大的核武器消灭我们”，
一名戴眼镜的女生一边说：
“依我看，他们是奈何不了我们的，为什么要帮他们？”，
一边伸出手，抓地面上的小人玩，
“还是小心为妙。”，卷发女说，
很快，伴随着少女的走动，10名少女已经消灭了约1万的士兵了，
而剩下的9万士兵，已经完全有挣扎变为逃亡了，
“啊呀，他们都在逃跑呢！”，
卷发女生似乎有点厌倦这样的玩法了，
不是厌倦，而是胖次里面早就忍受不住了，
“好想要啊，好想，我......”
她趁几个姐妹没有注意，趁机从地面上抓出一把小人，
没有男生的她，早就忍受不了了，
不是没有男生，而是没有那么高的男生，
今天竟然在这，而且地面上的小人都是男性，不管大小了，只要满足就行了！
她以最快的速度，将小人们塞入胖次里，
小人们在挣扎，在反抗，那些与他们一样粗的阴毛与汗液，却使他们无法动弹，
很快，爱液就代替了汗液的存在，它比汗液更浓，更黏，
有的人，在爱液的表面上，想呼吸有无法忍受这种味道，真是要了命！
有的人，身体完全被爱液覆盖，动弹不得，活活憋死在爱液中，
还有的人被阴毛缠住，不能说是缠住，应该是包围住，直到被几根阴毛压的粉身碎骨，
胖次中的人们，都在挣扎着，只是结局不会改变，
一片亮光，胖次中的人们以为得救了，
然而，他们没有出去，只是又有人们进来了，
这次进来的人们的命运更加悲惨，
他们没有停留在阴毛形成的密林中，
他们没有停留在爱液覆盖的沼泽中，
但是更让他们毛骨悚然的是，女生的玉洞，里他们越来越近了，
士兵们开始从手掌上跳下来，但是下面仍旧是沼泽和森林，死亡，只是时间问题罢了，
少女的玉洞，与士兵们原本在家中所能满足的玉洞，是不同的，
这个玉洞，深不见底，
并且伴着洪水一般的爱液，
滴落在地面上的爱液，虽然不起眼，但是依旧吞没了地面上的部分士兵，
而胖次中渐渐进入洞中的士兵们，比地面上的更加悲惨，
进入洞中，被肉壁慢慢的碾碎，吞噬，
或者被洞中积累的爱液淹死的，
死人越来越多，
突然，卷发少女思考了一下，说：
“为什么要踩死他们呢？”，
“反正都是死，用能使我们更新高兴的方法就是了。”，
其他9个少女听见了，其中一个呵呵一笑，
“学姐也真是的，自己爽了那么久，现在才分享给我们！”，
“咦？”卷发少女一脸疑惑，
“我们早就知道学姐你在干什么了。”
卷发少女听后有些脸红，又说：
“你们早就知道了？随便你们了。”说完，有陷入自己的世界中。
“开始吧!”,
其他9个少女异口同声的说，
随后，又是一片银铃般的笑声，
而此时，处于她们鞋边上的士兵与国王更加紧张了，
只剩下5万人了，
而且接下来的不是死亡，而是揉捏。
“不能这样下去，”A国国王一边远离其中一人的鞋一边下令说，
“快联系空军部队，陆军部队，海军部队，带上最先进的武器，将他们赶出这里！”
可这又能否得逞呢？
那个通讯的士兵一边逃跑，一边求救，
“请求增援，我军几乎到了全灭的地步，敌人......”，
啪-------------------
也许是少女们太不注意了，有个女孩脱下了鞋子与袜子，随手扔在一边，
估计她想脱下鞋子后才与小人们互动，
不过不管怎样，反正通讯的士兵，是死了，
先是被从天而降的物体---袜子覆盖住，
然后是从哪个袜子散发的淡淡的臭味，将方圆500米包裹住，使那个士兵剧烈的呕吐了起来，
最后是一个更似坚固的物体---鞋子，压在了袜子上面，
同时，也很巧的压在了士兵的上面，
噗嗤，没有人听见的声音，很轻很轻，
即使在鞋子落地的一瞬间内，死了许多人，
那个拖鞋子的少女依旧不知道，她的揉捏，已经开始了，
她光着脚，轻轻的踏着大地，
“该怎么玩你们呢？”，
“这似乎是一个挑战呢！”
她快乐的思考着主意，
只是她并不知道，她的脚，已经对这些士兵们开始了折磨，
尽管平日了，少女们相互之间都闻不到她们的异味，
因为只是淡淡的臭味，太淡了，
但是对于地面上的士兵来说，这真是让人生不如死的气味，
士兵们的希望已经完全的变成了绝望了，
“恩，不走了吗？这样也好！”
光脚少女小心的移动的脚掌，使气味扩散的更充分，
虽然脚掌移动的很慢，但是对于地面上的士兵来说，脚掌移动带起的风，已经将他们卷人空中了，
在脚臭味形成的风中，士兵们饱受着身体和心灵上的折磨，
轰！！！
那只光交落在了地上，形成了更加激烈的旋风，
士兵们被迫飞得更高了，
而在地上的士兵就连反应都没反应过来，
就被那只美脚给碾碎了，
尸体粘在脚上，连脚上污渍的大小都没有，
光脚少女有些失落的说到，
“就这一点小人了吗，你们也给我留一点吗！”
眼睛女说，
确实，10万人，还剩下1万左右，
每人分一千？少女们都感觉有点少了，
但是少归少，不可能因为少，他们就不玩了，
她们依旧是那么开心啊，
很快，A国边境上的飞机就到达了，
只是飞行员并不知道，他们要迎击的，是10个少女，
更不知道的是，10个1000倍高的少女，
因为命令就是增援，
而A国的雷达中并未扫描出B国的军队武器，
增援的人们不知道他们遇到了什么麻烦，
飞行员们稀里糊涂的驾驶着飞机，看着导航仪上的地图，
轰！
一架飞机爆炸了，似乎是撞到了什么，撞到了什么导航仪上不存在的东西,
是什么？
“啊呀呀，这是什么？”
巨大的声音响彻天空，飞行员们也听见了，现在他们才发现，那架飞机撞到的是少女的胸部，
少女用手抹去了衣服上的飞机残骸，不高兴的说到，
“真是讨厌，这个星球的小人真是色呢！”
“怎么了？”
“哼，他们开飞机，向我的胸上撞！”
“等等，你说有飞机！在哪儿！”
“在....你看，哪里全是的，都往这来了。”
“啊呀，这飞机很小啊！”
“是啊，竟然比我们的手指都短！”
于是，厌倦了与地面上小人玩的少女们，开始与飞机互动了起来，像捉蜻蜓一样
但是地面上的人们却不会因为少女不与他们玩而消除了危险，
因为她们那巨大的脚更加活跃了，走动的范围比原来更大了，
当然，士兵们也处于她们的走动范围之内，
几分钟过去了，虽然少女们只抓到了几架飞机，但地面上的人们，已经所剩无几，
而飞行员们，全都驾驶着飞机，仓皇离去，
A国国王已死，他的尸体，在光脚少女的大拇指上，在酸臭混合的气味中，紧紧的卡在指纹中，
A国的10万军队，在30分钟内就全灭了，
在少女的玩耍中全灭了.
