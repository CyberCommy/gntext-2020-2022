# 以前的一个作品……QAQ【鲛珠】

作者：嘉言懿行

# 1

西安闹市的夜空可以用璀璨形容，高楼大厦和车辆的灯光闪烁交错下，五彩斑斓。行人们接踵摩肩，相当繁华。当然，这是一个大城市所必备的，但西安可是一座历史厚重的古城。大雁塔、钟鼓楼、古城墙、大唐芙蓉园，这些古迹上挂满了金子一样的灯光，那么光彩夺目，但却没有夺去其中的古朴，而是相得益彰。这是每个西安人所当之无愧的自豪，因为这不是大自然日积月累雨打风吹所形成的鬼斧神工，而是西安一代代的名人与百姓共同创造的奇迹。这也是中华儿女向世界展示其辛勤和智慧的又一次有利见证，也是中国文化的伟大的最好证据。

这是我发给女友小华的一段文字。小华是南京人，和我在网上认识了五年了。今天，小华向我提议，她想要去西安找我。西安的美食、历史这些都耳熟能详，我也没有必要再多介绍，于是就向她介绍了一下西安的夜空景色。她的话语间能看得出来，颇为兴奋。我不知道是因为我的关系，还是西安本身的魅力？应该是兼而有之吧。我便和她约好了：下个月的今天，我去机场迎接她。【后加：我当时不知道之后会发生什么，如今时过境迁，回想往事，只觉得就跟昨天一样，历历在目，让人不能自已。那是多么的惊异神奇，又是多么的美丽自然。我有高兴，有悲伤，有懊悔，有庆幸。过往无法返回，未来也不能预知。也许，这就是人生的一个魅力吧！】

”文冬！文冬！“机场外面，小华在向我招手。虽然我和她认识五年了，视频聊天和爆照都是家常便饭。但我见她这一面，还是十分惊讶，视频和照片果然和真人不同！”欲把西湖比西子，淡妆浓抹总相宜“这句话常被人认为是美丽的最高境界。但我见到了她之后，便认定了这无异于扯淡。化妆对于她简直是侮辱！增添一点颜色显得多余，抹淡一点就显得不够，那种清新淡雅，还需要什么妆容？发明”完美“这个词的人是不是见过和小华一样的女子？我不禁这样想着，不觉痴了。

”小...............小华“她跑到了我的面前，我方才反应过来。她一米七左右的个子，比我矮一个头，但在女生里面算比较高的了；身材高挑匀称，面容白皙无暇。我拍了拍她的肩膀，笑着道：“饿了么？我带你去吃葫芦鸡，西安的夜市也很有名。还有回民街，也是应有尽有。要吃什么给我说，我请客“我这么说不仅仅是为了岔开话题调解刚才的尴尬，也是为了让她真正爱上这座城市。对于自己的城市，我不可能不自信。

夜市有什么？”小华看起来迫不及待，跃跃欲试的样子。我淡淡笑着，打趣道：“想知道吗？垂涎欲滴了吗？好奇心充斥全身了吗？就不告诉你！要不然，你叫我一声夫君大人~”论斗嘴，还没有几个是我的对手。  “你不说也可以，反正我迟早要知道”小华却没有中我的套，而是若无其事的样子。看来她对付我的经验也不少啊！“女人要懂得三从四德，你爹没让你裹脚真是错误”嘿嘿，我换一种方式来挖苦她。  “裹就裹，你舍得吗？”小华却根本不吃我这一套。我就低头看着她的玉足，做出看着出神的样子，嘟囔道：“不裹脚也挺好的…….看看看看，真是长一点显大，短一点显小…….这样刚合适，才是十足的美人。”正在小华脸红不好意思的时候，我又话锋一转：“可是这性子实在是应该改一改了，还是裹脚来得好。鱼和熊掌不可兼得，难啊！”欲取先与，欲擒故纵。论语言的艺术，实在是不可以轻视的。 “taix”说笑着，我打了一辆车，准备送她去吃烧烤。“还是先回家吧”小华道：“我有事给你说。”我感觉有些奇怪，但又不便询问，便答应了她。“师父，祭台村”。 出租车师父开的很快，我们很早就到了家。虽然父母早就买了新房，但因为我是一个专栏作者，新房环境太喧闹，我又念旧。是以平常就在旧家里打字办公，而周五、六、日三天去新家陪爸妈。我生活很不规律，平常基本不收拾家。但因为小华要来，我几天前就开始了大扫除，将家里收拾的整整齐齐、一尘不染。“没想到你家这么小，却这么整洁。”看来我这几天的努力没有白费，她一进门就给我一个高评价。 “美人居之，何小之有？一屋不扫，何以扫天下？”我拍了拍她的肩，笑着调戏道：“这里不是金屋，但藏娇绰绰有余了。”“谁说不是金屋？”小华走到我的床前，拿起了我床前的一本书“韩非子…………..金屋不就在这里面吗？” “知我者，小华也！”我勾了勾她的下巴，调戏道：“哈哈，你也不赖，我们可以多比试比试。”我无话可说了，只能由衷感谢命运，感谢上苍，让我有了一个如此的境遇，遇到了如此的一个美人、爱人、同好、知己。我该怎么配得上她？我暗自惶恐起来了。 小华拍了拍我的背，道：“你这个人不懂怜香惜玉，经常欺负我，所以我哪里敢跟你比？”之后她退了两步，腰杆直了起来，郑重道：“我现在要说事了，请你做好惊讶和恐惧的心理准备，并且要保持严肃和冷静，不要对这件事惊慌失措。我知道这个很困难，但请你理解、坚持。我知道这件事很可能破坏我们二人的关系，请你尽量包容。谢谢。”她说这话的时候，带着一点哭腔，我顿时懵了，不知道怎么回事。她家里有困难吗？是的话，我应该尽量帮她。我稳了稳心神，道：“放心，你无论遇到了什么困难，我都会是你的坚甲利兵，为你披荆斩棘，绝不退缩。” “这时候就别表忠心了。”小华强笑了一下，手中动了动，竟出现了黄色的闪光。我还没反应过来，眼睛就一花。等我再睁开眼睛，真是被眼前的一切吓了一跳，坐倒在床上。 我四处张望，发现我家里的一切竟然如此巨大。我床边那本被小华翻过的《韩非子》的厚度竟然跟我一般高！再转眼看看小华，她更像一棵大树，比我大了不知道多少倍。我诧异级了，也害怕极了，真不知道如何是好。但她总不至于害我吧？而且为了保持一个男人的尊严，我迅速站了起来，笑着看看她，并没有显露出一点恐惧。 “对…..对不起….”小华显得比较意外，然后问我道：“我接着说的……你应该信了吧？  这样我怎么能不信？实在是无法用常理解释了。因此她再说出什么违背常理的话，我可能都不敢不信了。就算她说雪是白的，我也得附和说煤是黑的了。于是点点头。 “我不是人类，我是….鲛人。怕是有鲛人在岸，对月流珠。知道吧？要不是这一次事情实在紧急，我也不会暴露了，我宁可和你过一辈子凡人的生活。可是这一次事情紧急….我这么跟你说吧：有一伙鲛人要上岸，意图给人类带来一场灾难。” “灾难？”我有些费解。 “你知道为什么我国王朝最长也只有三百多年，而且到了王朝后期肯定社会矛盾尖锐吗？” “人口”我自信地道：“中国这片土地地大物博，但古代生产力不足，难以承载太多的人口。人太多的话，土地、物资都会出现缺乏，因此就会出现争夺的情况。要是土地真的实在容不下那么多人，那么必然会导致天下大乱。”我有点跟上了她的思维。 “是的。工业革命之后，人类的科技发展突飞猛进，生产力成倍提高。但生产力，本质上还是改造自然、掠夺自然。起初，人类在大陆上掠夺。后来，人类掠夺向了大海、太空。我们居住在大海，人类的扩张掠夺已经开始威胁到了我们的生存。因此有人提议用武力手段减少人类的数量，以求双方尽量保持一个平衡。所以，我需要你，人类也需要你。你是我见到的人里面，最有胆识、最出色的一个，我相信你能够帮助我阻挠这一次计划。”
