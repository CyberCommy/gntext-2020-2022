# 三国杀之左慈传人

作者：miniboy

TID：19279

 

# 1

2012年6月8日  终于考完了，看着面前书桌上堆积如山的各类参考书辅导书，我自己也不得不佩服自己能把这么多书都一页页的看完，题目做完。虽然考试感觉不是很好，但是都已经过去了，接下来的就是要面对大学生活了。每次妈妈都和我说‘现在好好学习，等到了大学就有得你玩了’，我终于盼到这天了。今天晚上和几个兄弟准备通宵玩一玩，不过我们本身就都是宅男，即使再着么疯，也疯不到哪去。但我们都酷爱桌游三国杀，由于高三学习紧张，都有一年没玩了，听同桌小黑说晚上他会带上新出的山包，里面都是新武将，真想能快点看到啊。大家约好了晚上七点在避风塘碰头，那是我家附近的一个休闲的茶坊，18元一个人饮料畅饮，可通宵，我们就打算去那里玩个通宵的。吃完晚饭我就冲冲的出了门，骑上我的自行车出发了。等我到了那里的时候，大家都已经开玩了，我也立马加入了。也许他们已经玩了几圈，而我刚到，新武将技能我一点都不知道，结果成了他们嘴里的猪队友，给了我一套正版的山包，让我去一边先好好研究，而他们玩的则是盗版的。小黑说正版的不舍得玩，怕玩坏，我也是醉了。这卡牌不就是玩的吗？于是我只能坐到一边一张张的看起来，也许是高考刚结束人彻底放松了，忽然觉得有点困，当我看到左慈的时候，眼睛竟然渐渐的模糊了。。。然后惊奇的发现，周围的人都消失了，好像自己也不在茶坊了，周围一片混沌。而我面前站着个白发老人，微笑的看着我说‘没想到能进入这个空间的有缘人是你。小玄你好啊。’‘你是谁？我这是在哪啊？’我看着眼前的白发老人总觉得有点面熟‘我叫左慈,这是我的空间，只有在特殊的时候通过特别的方法别人才能进入。既然你来了，就说明我们有缘。’老头说左慈？我记得刚刚手里拿着的卡牌就是左慈，再仔细看看，果然和卡牌上长得一模一样。左慈在三国里也是个仙人的级别，听他这么说，是不是要传我什么法术了啊，我不觉兴奋起来。好像他能看穿我的心思似的，他点了点头说’既然你有缘能进入到我的空间，那我打算把我的技能传授于你。不过你所在的世界人物繁多，又没有什么特定的人。我这化身技能却是不着么好传啊。‘他略作思考后说’不如我传你化物技能吧。万物苍生，幻化由心。你能变成这世间任何物品，生物，但就是不能变成其他人。你每次化物后，只有受到伤害破坏，才能变回自己，而自己的位置你可以设置个地方，比如你的卧室等。然后你每受次伤害，就能多变化一种物品生物，如何？‘我听着，这可是神级技能啊。我连忙学着电视里古代人的样子拜了下来，’多谢老师！‘’不必多礼。‘说着伸出手把手放在我头上，忽然感觉全身暖洋洋的，好舒服啊。。。’既然已传你此等功法，你我缘尽于此。。。‘忽然感觉自己被谁推了一下，睁开眼看到坐我旁边的小牛看着我说’我说我们让你来熟悉武将技能的，你着么就睡着了呢？我玩累了要休息会，你上吧。‘此时我一看对面墙壁上挂着的钟，已经半夜十二点多了。着么做了这么奇怪的一个梦呢？摇了摇头，去把小牛替换了下来。。。后续可能发在这里，也可能发在QQ群里，有兴趣的也可以一起进QQ群讨论
