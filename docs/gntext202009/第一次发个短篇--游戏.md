# 第一次发个短篇——游戏

作者：杨小洁

TID：21630

 

# 1

大家好，我是新人小洁，这次是下了很大的决心发了这个短文，文笔不好，还请见谅～

“那么就开始咯～”

银铃般的话语，夹杂着些许的笑声，那本只存在于梦中的声音却让我感到深深的恐惧。

几分钟前

“规定的时间内，从我的脚跟跑到脚尖逃出去。”

让我逃出的，是头顶上巨大的棉袜脚。

“你的初始身高是一厘米，随着时间的推移会慢慢变小哦，所以你要快一些才好呢。”

巨大的脚跟踩在我的面前，把我完全笼罩在超过50米的脚底的阴影下，我完全可以看清脚底棉袜上的一条条纹路。

“小东西，快一点哦，人家这样举着脚很累的～”

我拼命的向脚尖跑去，本来坚信10秒之内绝对能到达的，可当我跑了一分钟才发现只到达了脚心部位的时候感觉到了一丝绝望

“她本来就没想放过我！”抱着这样的想法，就算拼了命也得逃出去！

“你还没有出来啊？我的脚掌距离地面只有一厘米了诶。”

“！”一声“善意”的提醒让我开始观察四周的环境，如果她说的是真实的话，我头顶上的应该就是脚掌了，微微换算一下，我的身高大概只有2毫米了。

本来这游戏就是不公平的啊！望着视野中近公里的脚尖，它和地面之间的空隙就是终点，跑出去就能活下来，可惜，我注定逃不掉了。

“如果快到了的话要更努力一点啦，我要踩下去咯～”

更努力一点吗？我躺在地上，面朝空中几万米的棉袜脚底朝我慢慢靠近，而我却不断的在缩小，我知道，与其挣扎，还不如享受这最后的时光。

“嘿咻～”女孩把脚踩在了地上，并且用力把脚掌踩实，在搓碾了近一分钟后，才把脚抬起来。

“看来是我赢了呢～”女孩笑着离开了，没有人能想到一条生命刚刚在她的袜子下消失了。
