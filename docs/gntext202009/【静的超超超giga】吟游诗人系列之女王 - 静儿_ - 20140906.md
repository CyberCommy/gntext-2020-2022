# 1

        “女神大人是有很多手下的，多到可以数以亿万记，当然，你们可能没有见过，因为见过她们的人没有多少生存的机会。”女生清脆的声音响了起来。围着她的人们越来越多，大多数都直接坐在地上，认真听着。
        “当然了，这些手下也是我们眼中不可超越的，据说其中最大的在女神大人眼里和细菌差不多，那可是很可怕的大小呢……”
        “我们在女神大人眼里有多大呢？”一个眨着大眼睛的女孩仰头问。
        “唔，这个……很难表达，刚才所说的手下是女神大人的百万分之一大小，大概是百万分之一的百万分之一的百万分之一的百万分之一？我也不太清楚……”女生歪着头算了半天，最后下定结论。“总之，无论是哪个大小，都不是我们能想象的。”
        “总之，据说这些……唔，叫她们女王好了，这些女王是可以通过吸收来增强力量的，如果一个女王毁掉一个城市，那么她就吸收了这个城市的能量，而这种能量会让她变得更大。”
        见所有人都在认真听着，女生继续说，“大约在……几十亿年前吧，当然，是人类纪年，对于女神大人来说可能就是几天的时间？曾经出现过一个女王，据说是女神大人的仇敌，被抓来之后为了凌辱她，把她变成了身上的一个随从，那个女王的精神崩溃了，因为意识到了女神大人不可侵犯的力量，然后迷恋上了那种力量。”
        “然后，她就开始疯狂地屠杀所有的城市，国家和聚集地。当初在女神大人的一根发丝上杀掉了很多人……我记不太清了，几百兆总是有的，因为这样的关系，她变成了当时最强大的女王，其他的女王都是她随意欺负的对象。”
        “这个女王的名字叫做咲月。”
        说到这里，女生停了下来，然后看向远方，好像在努力回忆着。
        “后来，咲月觉得自己已经有资格称霸女神大人的身体了，简单地说，她自封最高神王，也就是说，她成为了女神大人身上的寄生虫的领主。”
        “后来呢？为什么我没听说过咲月这个名字啊……”一个头发胡子都已经花白的老爷爷问着，“我也活了近两百年了，却没有听说过任何她的故事。”
        “那是自然的。”女生面色不改，“因为她已经离开了。”
        “离开？”所有人都倒吸了一口凉气，“这个世界，除了女神大人之外，还有其他的空间？”
        女生的表情也变得庄严肃穆，“有的，而且，在那个存在的眼中，女神大人不过是个小不点，是可以随意玩弄、碾压的东西。那位的名字是一个禁忌，任何人都不能提起。”
        全场陷入个一种诡异的寂静，在听众的心目中，连女神大人的随从都是不可想象的巨大，那么连女神大人都可以玩弄的那位，又是什么样的呢？
        “总之，出于没人知道的原因，那位带走了咲月。”女生继续说道，“咲月离开了之后，世界……呃，女神大人的世界，再次陷入了混乱之中，所有的女王都对咲月曾经的权势红了眼，因而对人类展开了疯狂的屠杀，那甚至没有一丝玩弄的意思，根本就是在机械地重复着踩踏、吸收的过程，所有女王都想成为最高女王。”
        “这件事情最终引起了女神大人的注意，毕竟，虽然人类非常小，但是女神大人是无所不知的。就这样，女王的行径引起了女神大人的愤怒，因为所有女王都在争权夺势，她们本来的工作却被搁置了……”       
        又有一个青年忍不住打岔，“本来的工作是什么？”
        “既然是女神大人的手下，自然是要负责女神大人的身体清洁的。但是因为争权夺势的关系，没有人去进行清理的工作。女神大人就决定，这些东西没有存在的必要了，也是那天，女神大人将所有的人类和女王都聚集起来。这是女神大人第一次用肉眼看见我们的存在，虽然还是在超高倍显微镜下。”
        “接着，女神大人就吐了一口口水……”女生的身体忍不住颤抖，“那是……那是最可怕的景象，无数的人类和亿万女王全部被一滴口水包裹起来，然后就这样被留在那里风干，人类几乎全部融化掉了，而女王却要忍受着口水一点点风干，而变得越来越粘稠，最后不能移动而活活被粘在原处不能动的痛苦。即使是一滴口水也比最宽阔的海洋还要大，是根本逃不出去的地狱。”
        女生草草的结尾让人们有些不满，但是却没人敢说什么，也许是最后一段的描述让他们害怕了。
        “能让女神大人抽出时间吐一口口水融化，是当时人类的幸运。”女生总结道。
        “你是怎么知道这些故事的？”老爷爷最先回过神来，问。
        “我是一个吟游诗人。”女生这样说着。然后看着原处，脸色变得铁青。

        “不要！”我抗议着，“学姐大人！这不公平啦！”
        我平视着学姐大人的脚踝，“这样和我打篮球，我怎么可能得分呢！”
        学姐大人却投给我一个微笑，然后再一次投篮，比分变成了1152:0。
        “不玩了！不玩了！”我怄气似的跑到一边，脱下了运动鞋和袜子，然后让微风吹在一双光脚上，一滴汗水渐渐成型，然后顺着脚丫的弧线滑下。

        洪水在一瞬间席卷了数十亿个国家，而一个小村庄也不能幸免。
        带着酸臭味的洪流的速度快的不可思议，拼命喊叫逃跑的人类却是那么的渺小，所有的建筑、山脉都被洪水席卷而冲碎。自然的力量在一滴脚汗面前是如此的不堪。
        因为女神就是自然。
        ……
        几百年后，从废墟和汗垢中爬出一个人。
        女生呼吸了一口空气，还是带着浓浓的酸臭味。
        “真倒霉，下次果然不能在脚上讲故事，太危险了。”
