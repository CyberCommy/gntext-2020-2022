# 1

在宇宙中一个神奇的异世界里有一块陆地，在那块陆地中心的一个庄园住着一对十几岁年纪可爱的小姐妹，姐姐身高1.8米，妹妹身高1.6米。姐姐时常嘲笑妹妹矮小，妹妹总是不服气，她天天朝思暮想地要去世界的尽头寻找女巨人，于是有一天姐妹俩决定亲自去世界的尽头寻找女巨人。

她们收拾好行装后离开了家开始长途旅行，姐妹俩走过了许多村庄，翻过几座山跨越了几条大河，经过半个多月大概跋涉了一百多公里来到一个牧场，姐妹俩经过多方打探，有人告诉她们在十多公里外的山上出现过身材很高的女巨人。姐妹俩继续赶路，这一天她们来到了一片草原，这时她们的食物也快吃光了。她们在草原里寻找食物时，发现这里的女人都很高大，于是妹妹上前问一个身材近三米的高个子姑娘，“巨人姐姐，这里就是传说中的巨人村吗？”

那个高大女子笑着说，“我可不算什么巨人，你们要找巨人可以去那座山脚下，那里有一个非常高大的女巨人，不过她身材十分巨大，你们要小心不要招惹她”。

姐妹俩继续赶路，中午就来到了大山脚下，猛然间发现在前面不远地方有一块彩色巨石，这块大石头长十几米、高有五六米，看起来上面非常漂亮并且很光滑，就像在盖上了一块鲜艳的幕布。姐妹俩被吸引着走过去，她们很轻松地就爬上去，在那上面非常柔软，于是她们开始吃东西并且躺下来晒太阳，等吃饱了她们开始嬉笑打闹。不料这个巨石突然抖动起来并且升高了，她们俩从上面直接滚落到地面上，幸好这里不高又很柔软落下来没有摔伤。姐妹俩站起来仔细一看，原来那并不是什么大石头，而是一个正在睡觉的女巨人的腿，她的身材十分匀称容貌非常漂亮，穿着一条有将近20米长的白色连衣裙，脚踩一双2.5米高的黑色高跟鞋。她们俩就站在这个女巨人的双脚面前不到两米的地方，姐妹俩站着还不及她的脚踝高。姐妹俩既兴奋又有些恐惧地看着这个女巨人，女巨人伸着懒腰说道：“谁这么讨厌打搅了我的好梦？”说着她低下头揉揉眼睛看到了地面上的姐妹俩，惊奇地说，“哎哟，这里有两个可爱的小人儿啊！嗨，你们好！”然后坐起来伸出胳膊把一只长3.2米、宽2.4米的大手摊开平放在她们面前，示意姐妹俩上来，女巨人手掌心距地面高不到一米，只到姐姐的胯部，所以她很容易就迈上去了；而妹妹身材矮，女巨人的手掌心高到她的腰部，费半天劲爬不上最后被姐姐拉了上去。姐妹俩站起来在女巨人的手心上大喊：“女巨人，我们终于找到你了！”

这时候女巨人慢慢地站起来有七八层楼房那么高，女巨人手托着姐妹俩走了几步然后坐下来，把她们俩凑到自己眼前，妹妹说：“巨人姐姐，你真的好高大呀！你的手指比我的胳膊还粗，你的手掌比我身体还大，伸直臂膀有十多米长呢！”女巨人笑着对她们说：“我身高才25米，不觉得有多高呀，主要你们俩是太小了！其实我可不算真正的女巨人，据说在很远的地方还有比我高大许多倍的女巨人呢！我也想去找真正女巨人，我带你们一块去找真正的女巨人吧！把你们的东西都给我吧，你们就在我身上休息好了。”说完她把那姐妹俩放在地面上，从衣兜里掏出一块面积四十平方米的巨型手帕，伸出手指轻轻夹住姐妹俩的腰部提起放在上面，然后把她们俩的所有行李也放上去，一起包起来塞到上衣兜里并对她们说，“小宝贝们，你们在里面好好休息吧！”这时姐妹俩已经趴在女巨人的衣兜里睡觉了。

女巨人带着小姐妹又赶了几个小时路程，天色已晚感觉有些饿了，于是女巨人走到一块平原上搭起一个30多米高的帐篷，然后钻进去把姐妹俩从兜里掏出来把手放在地上让她们下来，可是姐妹俩不敢跳下来，女巨人把手心倾斜使她们滑下来。女巨人问道：“小宝贝，你们饿了吧？我这里有些食品足够你们的，咱们一起用餐吧！”说着，从随身布包里拿出一块直径五米的大烙饼掰了一小块给了姐妹俩，然而这一小块也有2400克重，又切了几块20厘米宽的香肠和菜叶给她们吃，用一个直径半米的瓶盖倒满水给她们喝，姐妹俩狼吞虎咽地边吃边说：“谢谢女巨人，这已经太多了，够我们两顿了！”吃完饭，女巨人拿出个直径五米高两米的大碗倒满温水，这里共有五十升水，对姐妹俩说，“你们身子看着很脏了，应该洗澡了！”然后亲手脱下姐妹俩的外衣，姐妹俩把衬衣乳罩和内裤脱下来扔在女巨人手里，就跳到大碗里非常舒服地洗了起来，洗完后请女巨人把她们捞出来，却没有把衣服给她们穿，女巨人笑眯眯地说，“小宝贝，你们现在已经属于我了。”这时姐妹俩面带疑惑地摇摇头，女巨人接着说，“让我好好看看你们，小姑娘挺漂亮的没穿衣服就更美啦！”姐妹俩恳求道，“求求你把衣服还给我们吧”，女巨人微笑着摇头，姐妹俩没有办法只好乖乖听任摆布，女巨人轻轻捏起来姐妹俩夹在手指头间抚摸着她们溜光的身体，用一根手指头轻轻逗弄着她们的乳头和屁股，有用舌头舔舐赤身裸体的小姐妹，她们在女巨人手心里扭动并呻吟着，女巨人又抓着姐妹俩的双脚把她们头朝下荡秋千般玩了十分钟，小姐妹在倒悬在空中惊吓的大声呼喊，恳求把她们放下来。女巨人看姐妹俩开心地笑着，后来看姐妹俩精疲力尽了，就把她们俩放在自己的乳沟里让她们休息，小姐妹俩很快就睡着了，女巨人很快也睡了。

第二天吃过早餐后，女巨人开始收拾随身物品打包，女巨人的随身背包就有7500公斤重，就连最小的腰包也重达150公斤，可是姐妹俩合力也抬不起来，女巨人嘲笑着说，“你们俩小东西力量太小了”。说着把这对姐妹塞进衣兜里又继续赶路，大概有五百多公里行程，她们的食物也快用光了，这时前方出现了一个很奇怪的黑色大山。这里有个洞穴很宽阔，里面却软绵绵的，女巨人走进去把姐妹俩从兜里掏出来放在地上，她们在这里休息了。姐姐从地下抓了一块柔软的东西闻了一下非常香甜，于是就咬了一下非常好吃。这时候整个洞穴突然剧烈摇晃起来，女巨人抓起姐妹俩赶紧往外跑，这时整个洞穴突然迅速上升了一百多米高，幸亏姐妹俩被女巨人用手拿着她们而没有摔下去。这时她们抬头望去，只见前面坐着一个非常美丽并且极为高大的女巨人，比这个女巨人还要高大许多倍也更美丽，她抓起那座小山原来是这个更大的女巨人的随身挎包，长度有120米厚达40米，而姐姐吃的是挎包里的糕点屑。姐妹俩齐声惊叹：“哇！竟然有这么巨大美丽的女巨人呀！”
