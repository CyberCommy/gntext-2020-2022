# 1

**旅途**   我们度过了非常美好的学生时代。她是一个善良的人，与同学相处很好，朋友很多；成绩优异，能力突出，老师非常喜欢她。我读书时，每次闻到她身上的气味，好像坠入了芳洲，然后似乎有远方悠悠白鹭，叫了两声。然后只见淡淡的孤霞。她身材修长，常带有笑容。那是我少年时代最美好的回忆。那时候风很淡，天很蓝，考试考得好。好多书都还没有读，对她保有一种憧憬。那时生命有一种张力。

我爱写诗，喜欢思辨，喜欢古代的文学，喜欢书法。这无疑与教育体制脱离了。于是有一段时间，我迷失于海，我迷失于灯光。我爱在夏日看幽花，舒蘸明月的霜；在秋后听回旋曲，同风君与东篱菊痛饮。她在好好地读书，成绩优异。那是往日的日子了。我的少年时光。

我们毕业后，在了一起。我们去天台幽会，我们也时而去繁华的街区隐去自我，只留下爱情的抽象者，观察，再观察。

这年夏天，天气异常地热。我们相约去海边。夕阳，海浪，宁静，鲸之音。空气中都有一种美的节奏。这，**似乎是一场美妙的旅行。**

海滩旁有酒吧，我们喝得半醉。海风与温度撩动了我们的情思。我们相约好，回旅馆云雨一番。渔人自远海回来。一些看起来很美味的鱼。

''嘘-------''一个尖锐的声音自远方而来，然后一阵猛烈的海风卷起。萧冷寒远。**她的眼睛动了一下。**

''我们回去吧，我怕''她说，那柔美的眼神，像玉兰花。

回到旅馆，她洗了脚，然后沐浴。那圆润丰满的脚掌，如玉，如凝脂。她坐在床上，那脚幽幽点地。像落花一样点地。那自然优美的体态，和她上学时一样。她拥有瘦简的身材，不是什么**丰乳肥臀，生死疲劳**。肌肉很舒美，一看就是健康的表现。

我们化而为一。

***梦***

我在奔跑。

因为松涛震怒。

我在思考。

因为我怕。

''站住''，一声熟悉而巨大的声音传来。我回过头去，在我面前的是一只洁白无暇的巨大脚掌，上面有着''蝴蝶玫瑰''之纹身。踏在木板地面。空气中有木头的香气。巨大神女坐在一个王座上。翘着腿。我抬头望去，不再地的一只柔美脚掌，悬在我的头上。除了神女的距离我最近的部位--那巨大的柔美脚掌。**我周围一切，都是干燥的。**

**         那巨大玉足傲然翘着。她的眼神，傲然俯视着我。**

**  那是她，那美丽的面容，是她。**

她慢慢地站了起来。我意识到了我的渺小。她足足有一千多米高。玉腿骨态很美很美。我骇然望着。

''嘘-----------''我身边顿时出现了一批人。好多。成片成片的出现。

''都给我像小虫子一样伏地！''人们都跪下。神女微微抬起了她的脚。然后他们似乎知道自己该做的事，进入缝隙疯狂轻吻舔舐。很多人都是跪着进去的。边跪进边疯狂的舔。好多人好在哭。她似乎很享受这种感觉，仰起头眯眼体会。我迷迷糊糊地也到了她的脚底下。

''好了，现在开始最后一步了。卑贱的虫子们''她带着残忍的笑容，说出这句话。这时，人们的哭泣更加大声。我看到，一些女生都害怕地蜷缩起来。她慢慢地放下了脚，似乎想要将这些人的痛苦延续。但就那么一秒都没有，几千人的体积瞬间减为没有。哭泣声也休绝。

''好舒服.....好舒服啊....''.

她再次眯起眼睛抬头享受。

我还没死，似乎就我还没死。当她抬起脚，我发现她的脚玉质柔美的依旧。

她的脚掌太巨大，几千人在上面只是有些红点。贴在如玉如脂的脚掌上。述说着生命的可悲。空气中是血腥与芳甜。刚才奋力的舔舐，换来无声的死。

**可在我的周围，我突然有了一种悲恸，尸身残缺，满地血骸，成千上万。有些人还在爬着，缺没有了半边身子。**

她调皮地拍了两下脚，又用脚在地上涂抹了两下。这下彻底死绝了。我是肉体。雪白的玉足，每一次踏在我身上，我都感觉我的体积消失。我都流泪。不痛，却无穷悲伤。为我，也为那些死去的人。

''你知道吗，我爱你哦。就好像我踩在这些人身上，是无情的。可我爱你啊，所以不会让你死去。''她说。

她又坐在了神座上。翘起了腿。笑眯眯地向后梳展，整理头发，然后撑直了玉质骨感的双腿。我得意看到那玉质的柔美的脚的全态，巨大，雪软的脚趾轻轻舒展。软润雪白的巨大脚跟在我旁边，我这时发现，那些死去的人们，加起来也就四分之一脚掌的面积。在这脚跟面前，是那么的渺小。脚掌上粘着一些尸体。已经没有体积了。

她双腿撑着，且笑眯眯的表情，是我们云雨时，我作为跪下那一方为她示爱时，她的一贯动作。

我渺小地立着，痛痛哭泣。

她又撑在腿上，下身怜爱地看着我。

''别不高兴啦，人家还要你....还要你....嘻嘻''

**清晨**  

''喂喂，怎么了，怎么了，为什么哭呀''

我睁开眼，看到她关心地问我。我一阵恐惧惯麻全身。不能动。

旁边是闹钟，木质的床柜，书桌，还有海风静静撩开窗帘，想把新鲜送进来。一切如初。

原来是个梦。

''没事，做了个噩梦''

她眼神复杂地看着我，然后有了一缕轻放地笑容。

''你梦到了啥，我昨天也做了个梦''

我没说话，有些害怕地看着她。

她突然也害怕地看着我。

''你是不是梦到你....变得很小''

我害怕的一句话也说不出。

''不是的，不是的，我不会那样的.....''

她抱着我哭泣

她出去买早饭。我打开新闻，里面播出这样一则消息;百慕大三角洲，突然出现四千余人残缺不全的尸体，现在事态.......''疑云与害怕，占据了我。

**疑云**

我喜欢向她示爱，一个女人，或许可以经常处于性爱的快感中，但除去那种形式，被舔舐玉门，花庭，与足部敏感地方，却是少能享受到的。我提出来时，她很害羞。但也同意了。早上的不快没有后，她似乎兴趣来了，去浴室精心清洗一番后，走过来柔美慢漫漫蹲在我的脸上，她也懂得了用这种无理来遮去害羞。经过锻炼的健美雪白的臀部。粉嫩干净的花庭。她也已经喝的微醺。手脚酥麻。我用舌头去尽量让她舒适。她很愉快，很愉快。可她站起来时，我的**眼睛**里闪出这样一幅画面；

''神女啊，不要啊，小虫们.....呜呜呜呜''小人们的舔舐更加卖力。

''舔，要来了要来.....啊......''她似乎上到了天国。

巨大的花庭，巨大雪白的蜜桃臀，把那小人显得那么渺小那么渺小....''花''突然紧闭，小人们变成了一团血液，与污物混在一起。排泄物夹杂着血肉，喷涌而出。马桶下面的小人，被打乱。她画上了浅黛粉霞。半咬着嘴唇，眯眼享受。玉手拖着下巴。她穿着美丽的古代丝纱。她的巨大玉足下面，有几千人在舔舐刺激着她的巨大玉足。他们在哭泣，因为神女，时不时按压涂抹他们，追求那种舒服的滑腻感觉。莲足圣洁，从拖鞋下仰望，那揉动的山一般的巨趾，令人震憾。上方的天塔般的玉腿，垂下的巨大的酥胸，令我发狂。爱，害怕，对小人的同情，无限交织。

''好了，开始吧''

无数的小人哭泣着，涌向她的花庭，用舌头清理着那污物，她酣畅淋漓。偶尔收缩，上千死去，再有哭泣着的人来....小人裹挟着污物，自觉跳入水中，为后面的同伴让位。最后小人不停舔，在用香膏涂抹清洗。

我回过神来，又是现实世界。她笑着把柔足放在我脸上。等待着我的舔舐。我呆呆地舔舐着。然后她几乎一字，再后屈腿，等待着我的回应。我木木的舔舐着。她很享受。

我留下了眼泪.我想到了供我读书的父母，我那死在战场上的战友，求学时候看重我的老师。对她的恐惧，害怕，爱。这些情感的两极啊！我要爱我的家人，爱她啊！！！我之后疯狂地舔舐起来，留下热泪.....

樱花飘舞着,海边的夏月，章鱼壶里也朦胧.....

川端的雪和鹤！

写于一个午后。

来这个圈子好久了。也是第一次发文。受影响最深的是媛媛吧，我在一个暑假，崇拜着她。青少年时期，魂牵梦萦，她的足，她的臀，她身上最卑下的地方。但我还算能把这些东西独立于生活。有时因为情欲我也挺自责的，但有时候又用庄子奇人来安慰，认为这是全其天。也许纯文学东西看惯了，写这种也带一点......算是我的某种表达吧。
