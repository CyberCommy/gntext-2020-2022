# 1

最近在youtube上看到一个视频，讲的是女巨人在迷宫中抓捕小人的故事。
这个设定让我很感性趣，于是就写了这个短片。
原视频好像是deaf giantess的系列作品，画风很像，感兴趣的朋友可以去搜，不过是很久以前的视频。

本文情节与视频无关，设定也会有较大出入，请不要在意。

# 2

“不计一切拦住她，决不能让她再变成巨人了！”对讲机中传来科尔的咆哮声。
“明白！”雷恩一行人接到命令，一刻也不敢停歇，奋力追赶爱丽丝的步伐。

自从被<诛神之矢>射中之后，爱丽丝的身体已经缩小了很多，现在的她只有原来身高的八分之一。
而更让爱丽丝烦躁的是，这种变化还在持续，每隔一段时间，她就要缩小为原来的一半。
但即便如此，爱丽丝也有着数倍于常人的庞大身体，这足以让她与这些小虫周旋。

在迷宫中追赶别人的确不是一件容易的事，何况对方是个巨人。
追击队伍在迷宫中走散了。

雷恩一路穷追猛赶，但还是被爱丽丝甩开了几百米的距离。
更不幸的是，队伍中的其他人早已不知去向。
【只剩我一个了，无论如何也得完成任务】雷恩感觉自己责任重大。
突然，远处的高大爱丽丝突然停下了脚步，回头看向了他。
这个举动反而让雷恩更加警觉起来。
【不好，她想干什么？】雷恩担心她会和自己鱼死网破。毕竟，爱丽丝目前依然是个将近四米的小女巨人，孤身一人与她正面对抗，雷恩不觉得自己能有几分胜算。

爱丽丝喘着粗气，站在迷宫中的某个岔口。在岔路的尽头，金色的巨人祭坛在朝阳的照耀下闪闪发亮。她得意地回头远望，很显然，那个小男人几分钟之内都不可能追上自己，巨人的步伐为她争取了足够的时间。

突然，远处爱丽丝的身影再一次缩水。
【哇，天赐良机！】事态的变化让雷恩很兴奋【现在她看起来像个正常人了】
不过雷恩的小算盘还是落空了，爱丽丝并没有鱼死网破的打算，她一个闪身消失在道路中。
【我艹，又是转角】雷恩暗骂道。

爱丽丝就不是很开心了
“md！”突如其来的缩小让爱丽丝吃了一惊，看来迷宫并没有给她沾沾自喜的机会，现在的她也就只有两米出头了。
眼看自己很快就会恢复原本大小，她得赶紧走了，巨人祭坛就在前方。

几分钟后，雷恩也终于气喘吁吁地追到了拐角处
【她好像是在这里拐弯的，希望不要再有岔路口的好】
刚转过弯，雷恩就看到了远处金光闪闪的祭坛。
【是终点，太好……哦，不】他发现自己高兴的有点早。因为，在那宏伟的祭坛之上，有一个小小的人影，正是爱丽丝——她就要变回巨人了。

爱丽丝站在巨人祭坛上，一切都已准备就绪，她正在静待程序开始。
这时，在道路的入口处，一个小小的身影姗姗来迟。
【哼，太晚了】爱丽丝很确定，那个男人已经无法阻止自己了。
面前的这条道路足足有400米，而自己巨人化的速度，根本不会给他任何机会。

“巨人化程序成功启动，现在的身高是一米七三”人工智能的女声在道路中响起。

爱丽丝跨过高墙俯视自己的恐怖画面在雷恩脑海中挥之不去。
他再也不想有那样的经历了；
想到这里，雷恩又有了动力；
于是，他冲了进去；
他要阻止那个变态的女人。

爱丽丝的身高开始缓缓增长
“一米八三”
“一米九四”
“两米零六”
人工智能每秒都会播报她的实时身高。

“两米一八”
“两米三一”
“两米四五”
雷恩并不知道阻止爱丽丝的方法，或许根本没有。但现在不是犹豫的时候，他明白爱丽丝正在不断变大，而自己唯一能做的就是尽可能快的赶过去，兴许还能有一丝希望。

“两米六零”
“两米七五”
“两米九二”
爱丽丝倒是很轻松，只是远远地看着他，一脸不屑的表情。

“三米零九”
“三米二八”
“三米四八”
长时间的迷宫追逐让雷恩很疲惫，无法发挥自己的最佳速度，他仅仅跑了60米左右。

“三米六八”
“三米九一”
“四米一四”
“四米三九”
雷恩现在十分紧张，他不知道到达祭坛时将有一个多大的爱丽丝等着自己。。

“四米六五”
“四米九三”
“五米二三”
“五米五四”
【不行不行，得快点了】雷恩越发的不安起来。
而爱丽丝，她似乎都有些困倦了。

“六米二三”
“六米六零”
“七米”
七米，已经是一个惊人的身高了。
【我艹，这长得也太快了】

“七米四二”
“七米八七”
“八米三四”
爱丽丝的身体以每秒百分之六的速度生长的，这意味着，她的身体越巨大，她的生长速度就越可怕。

“八米八四”
“九米三七”
“九米九三”
爱丽丝的身影越发清晰起来。
【你tm就不能再跑快点么！】雷恩咒骂着，他对自己有些恼火。

“十米五三”
“十一米一六”
“十一米八三”
【我的天？我….我在仰视她吗？】雷恩感觉到自己的视角发生了变化。

“十二米五四”
“十三米二九”
“十四米零九”
爱丽丝一点也不担心，因为终止巨人化的方法只有一个，那就是让她的一只脚抬离祭坛。

“十四米九四”
“十五米八三”
“十六米七八”
【不不，别再长了！别再长了！】雷恩在心中祈祷，但无济于事。
爱丽丝还在不断的长大。

“十七米七九”
“十八米八六”
“二十米”
爱丽丝已经在闭目养神了。
作为一个二十米高的巨人，她有足够的自信，自己的脚根本不是那个小人能够撼动的。

“二十一米一九”
“二十二米四六”
“二十三米八一”
“二十五米”
雷恩有点不相信自己的耳朵，但他的眼睛明确地告诉他，爱丽丝绝对有这么大——她已经超过高墙了。

“二十六米二五”
“二十七米五六”
“二十八米九四”
【别紧张，别紧张！还有机会！一定还有…】雷恩安慰自己。
巨人化的进程已经过半。

“三十米三九”
“三十一米九一”
“三十三米五零”
爱丽丝的生长速度已经肉眼可见。

“三十五米一八”
“三十六米九四”
【不不不，这不是真的】
雷恩心中的恐惧随着爱丽丝的身体一起，正在不断地膨胀

“三十八米七八”
“四十米七二”
一些不堪的画面在雷恩脑海中闪过，那让他无比害怕。

“四十二米七六”
“四十四米九零”
【不，她不可怕…】雷恩强压着自己心中的恐惧

“四十七米一四”
“四十九米五零”
“冲啊——————”最后的100米，雷恩呐喊着开始了冲刺。

这是他最后的勇气。

“五十一米九八”
喊声虽然很微弱，但还是惊动了爱丽丝。这让她有点恼火，

“五十四米五七”
区区小人竟敢打断自己发呆的美好时光。
爱丽丝睁开眼，她要给他一点教训。

“五十七米三零”
爱丽丝握紧拳头，深吸了一口气，然后
“啊——————————”她突然爆发出雷鸣般的怒吼。
那声音震耳欲聋，吓坏了雷恩，他一下子失去平衡，猛地摔倒在地。

“————啊——————”
身高播报完全被盖了过去。
爱丽丝的吼声还在持续，她的气儿可长着呢。

“————啊——————”
可雷恩快要受不了了。
他趴在地上，无比痛苦地捂住耳朵，他觉得自己耳膜要爆了！

“————啊！”
爱丽丝终于停了下来，吼声大概持续了五秒钟，但雷恩却觉得有一个世纪。
他被巨吼震得头昏脑胀，趴在地上低声呻吟。
【哼，让你知道知道什么才叫呐喊，你那蚊子叫算个p！】爱丽丝对这个结果很满意。

“量化处理，最终身高，六十九米二二”
“巨人化倍数：40”
“进程结束，哔——”
人工智能的声音再次传入雷恩的耳朵，他明白自己彻底失败了。
难以言喻的恐惧涌上心头。
雷恩挣扎着想爬起来，但他的身体抖得厉害，根本不听使唤。
看着前方，那双巨人之脚，就踩在离自己不远的祭坛上。
雷恩的眼中充满了绝望。
之前在迷宫中，他亲眼见过那些被爱丽丝踩成肉酱的人，那让他几天吃不下饭。而现在，他也要变成那恶心的样子了。
雷恩不知道现在该怎么办，恐惧让他无法思考。
他抬起头，紧张地仰望着爱丽丝。

而爱丽丝也正在看着他，用一种很轻蔑的眼神。
看得出来，小东西完全被吓傻了，没有哪个小人不会畏惧自己的巨人之身。
如果在平时，爱丽丝会很愿意和这个小可怜玩玩。但现在，她只想尽快回到迷宫，找那些的小混蛋好好算算账！
所以，处决的时刻到了。

对于爱丽丝而言，雷恩离她只有一小步的距离，她有很多种方式可以弄死这个小虫——踩扁他、踢飞他、碾碎他或者任何一个小动作。
“好了，你这个讨厌的跟屁虫，现在就让我……”爱丽丝犹豫了一下，想选个好玩点的方式，“……踢飞你吧。”
踢飞也许是个不错的注意，她有段时间没这么玩了。

雷恩双眼无神，空洞地望着这个女巨人，绝望使他麻木。
两根高耸的巨腿微微晃动。
接着，巨脚拔地而起。
还没等雷恩做出任何反应，这个庞然大物就飞快的向他冲了过来。
五根巨趾迎面袭来，那种速度，雷恩不可能躲得开，他甚至都来不及闭上眼睛。
这一踢足以让雷恩飞出100米远，必死无疑。

但是，巨脚并没有撞上雷恩。
千钧一发之际，爱丽丝改变了主意。
她稍稍翘起脚尖，脚掌擦着雷恩的头顶飞了过去。

巨大的脚丫倾斜着悬停在雷恩的上方，脚心与他只有几米的距离
不可一世的巨大脚底遮蔽了一切，背光的效果让它而显得格外阴森和恐怖。
这一刻，雷恩真正感受到了，来自死亡的威胁。

巨脚轰然落下。

“爆浆~”熟悉的触感从脚下传来，这让爱丽丝很舒服。
比起其他的方式，她发现自己还是更喜欢踩扁他们的感觉。

现在，是时候了回去找那些小东西复仇了。
一想到自己竟让他们弄得如此狼狈，爱丽丝就恨得咬牙切齿。
这一切，他们必须加倍奉还！

爱丽丝迈开巨人的大步，高傲地向前走去。
在她的脚底，有一片不起眼的红色。
“我想这也很适合你，小跟屁虫，你不是很爱粘着我么？”
雷恩残败的尸体粘在爱丽丝宽大的脚掌上，现在，他再也不用担心自己被甩开了。
