# 1

这里是雪风……抱歉我咕了_(:з」∠)_我原本以为年假的最后几天应该不会再出现任何打扰我写文的事情，结果主管在微信上一个新项目就给我拍过来，还要我马上熟悉好之后上班就得接手，于是我在看完一大堆资料头晕眼花的同时就毫不犹豫并且内心并不羞耻的鸽了（被打）

不过我不是说不更新，只是想先在投稿结束之前在这里先占个坑_(:з」∠)_我会在之后的几天里补完的，请不要吐槽我或者diss我，我什么都会做的（暴言）

于是先放出一段并不会涉及到剧透的东西（虽然应该已经预告过所以没啥好剧透了吧）

“咔嚓……”

随着人体扫描和我手上拿着的一个特殊金属饰品被扫描成功，本来对于我无比巨大的，人形所使用的输入密码所用的小键盘侧面，划开了一道精心制作的暗格，从里面弹出来了另一套适合我所用尺寸的密码输入键盘。

“呼……当初还好在设计基地安全系统的时候有给我自己预留过后门，不然怕是要累死。”

毕竟很明显，相比于人形的尺寸来说，仅仅只有一厘米多大小的我要是想要按动她们所使用的键盘按键的话，怕不是要操纵着自己的喷气背包，然后一头撞上去，才能勉强按下一个按键吧。说起来我也不得不佩服起自己当初所想到的双重保险设计，因为毕竟单纯扫描的话是有被侵入和伪装的风险隐患的，但是如果同时加上物理键盘输入方式的话，就算敌人能过得了扫描那一关，难道还能把机器开进来用来按键盘吗……

也就自家的基地能使用这种技术了吧，带着小小的得意，我的手指在键盘上跃动着，输入那一长串已经背的滚瓜烂熟的识别码。随后当输入完成之后，键盘便缓缓地缩回了暗格之中。而旁边那看起来犹如高墙一般的巨大机械门便慢悠悠地朝着两边滑了开来。

扑面而来的是一股虽说不上浓郁但很明显可以感觉得到的消毒水味，以及所有科研研究用的房间所特有的那股清冷气息，就算是基地内有着24小时的精准温度调节，我也不由得打了个寒颤。

“哈……”

轻舒一口气，我搓了搓双手，操纵着自己专有的飞行背包进入了人形实验室。

还好……某个白大褂并不在这里。现在应该是午休时间吧，就算像她那种工作狂人，也是需要进食，休息，解放脑力的。也是我特意要挑这个时间段进来闲逛的理由。不然要是被她知道了我在这儿，就算她不会嫌我碍事把我用胶带如同标本一般给贴在墙上，也会被犹如看虫子一般的嫌弃眼神全程盯着，同时还有诸如“不要碰坏我的杯子”，“不要干扰我实验”之类的话语充斥在耳边吧。

哎，到底谁才是老板啊，明明我才是你的投资人好不好……

说起来，自从人形变大以来，虽然我顶着个指挥官的身份，但是每次碰到玩心起来的人形，都免不了要随着她们的意愿去做一些事，虽然明明自己是这组织的掌权人，但赤裸裸的体型差距在你面前的时候，任何试图用自己的身份去压服她们的行为都是苍白无力的。更何况，我也并不想这么做，诶？我身为指挥官的威严？那种东西存在于世界上吗？

说起来，今天来这里的目的是视察一下新人形技术的进度啊。融合了军方技术，集合目前为止所有高端性能配备做出来的新式军用……呸……民用人形，会是什么样子的呢？

昨天接到帕斯卡的报告，说是已经进入到最后的调试阶段了，这两天应该就能入职了吧。

（是的你没看错，在进入正题之前我可耻的咕了）
（于是半夜先睡了，明天上班之前狗命要紧_(:з」∠)_）
