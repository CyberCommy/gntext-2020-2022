# 1

五十三章 阴谋交错

一股寒意涌上心头，的确，露露是宇宙的神，而我阿青不过是她曾经的男朋友，一个小奴…她又怎么会为了我而放弃整个宇宙呢？

我看到慕鲁德那泛着冷光的利爪，是那么的阴森！我闭上双眼，向前倒了下去…

“啊！！！！”

妹妹慕鲁娜发现了我的意图，一脚把我踢翻在地，我的嘴角泛出一丝血光，她冷笑着：“阿青大人，您想为神上尽忠是么？呵呵，没那么容易，我们雪狐家族的赌注可都在您身上了！”说着还用手勾勾我的鼻子。

慕鲁德走到妹妹旁边耳语了几句，慕鲁娜点点头，抬起右爪，变到一米大小，从鼻子里发出一声闷哼：“阿青大人，我们要带你去见主人了…你会高兴的。”我一惊，伸手去抓水晶球，但就在这时，她的巨爪落了下来！！

“啊！！！！”我又是一声惨叫…

我在这最后一刻把水晶球放到怀里，我看到了露露那心里的挣扎…

富兰切斯科的演奏仍然在继续，悠扬的钢琴声飘荡在大厅里，多少人听得如痴如醉！但露露女神怎么也无法平静下来…她的心就像一团乱麻，“剪不断，理还乱”就是说这样…

富兰切斯科心中也无法平静，是什么样的大人物能有这么大的能力？为什么一个音乐节能吸引全世界的社会名流和政府首脑？又是为什么今天大厅的气氛这么压抑？他的心逐渐脱离了乐曲，直到……弹出一个失误！！！！！

就在这零点几秒钟内，他心中大惊失色，难道自己的生命就要终结了么？！不！多年的音乐生涯使得他十分镇定，用一个外音把失误掩饰的毫无暇疵！不过露露听这首曲子不知多少遍，发现了这个完美的错误。

露露条件反射的弹了起来，大厅内的所有观众都面色如灰，齐刷刷的跪了下去！！磕头的声音掩盖了钢琴声！！

“神上饶命！神上饶命！……”

富兰切斯科被眼前的一幕吓的半死，连忙跪到舞台上猛烈的磕起头来，心中却在想：“神上？神话成为现实？这个世界上真的有神？这个貌美如花的少女居然是宇宙的神上？这就是大人物？哎呀…我金手指要把自己送入地狱了！”

露露向舞台缓缓走去，那位伟大的音乐家却在浑身战栗！他觉得死神正在靠近自己！

“富兰切斯科先生，起来吧…您真不愧对金手指的称号，居然把失误掩盖的如此的完美！不过我听这首曲子很多遍了所以很熟悉。是我吓到您了么？真的是十分抱歉！你曾经是我的偶像呢！快站起来吧…”露露继续着她招牌式的微笑，用手扶起了富兰切斯科。把他搞的又喜又怕…

“大家也都回到座位上吧！富兰切斯科先生，请您重新开始演奏，我跳一支舞好么？”露露仍然是微笑…所有人都惊呆了，没有人敢相信自己的耳朵！无比伟大的神上居然要为自己跳舞！！为悲贱的小奴跳舞！！

所有人又跪到了地上，感动的泪流满面！“谢神上！谢神上！…”高呼声一波盖过一波。“回到座位上！这是神上的命令！” 

# 2

五十四章 神上的舞蹈

这就是极度精神的力量，即使露露不用统治趾环，仅仅依仗自己的威信，也足以使人们臣服于她。 露露就是这么想的，她要让地球人对她忠诚无二，就像那些炽烈盆地在羽凡的威逼甚至死亡的时候仍然高呼“神上统治一切”的众魔君一样，以免当年英国的悲剧重演。

露露女神脱下了白色风衣，露出里面的白色T恤和粉色纱裙，而…鞋跳舞确实有点费力，仍然是那双露露来是穿的黑色超高跟！

完美的身材！！极致的曲线！！所有人都惊呆了，大厅里鸦雀无声！！ 

# 3

富兰切斯科先生首先反映了过来，因为神上正看着他！娇美的脸蛋上仍然微笑，用温柔的目光等待他的演奏。

他从来没有这么紧张过，也从从来没有这么平静过，一种纠结的心理正束缚着他，他不知所措了…

“富兰切斯科先生，您可以开始演奏了…”露露一句话冲走了所有的疑问，他现在心里只有一件事…把自己的才能都奉献给神上！

浮云流水般的钢琴声飘扬起来，弥漫了整个大厅，露露本来舞蹈就不错，现在有了灵魂之袜的力量，舞技让现场所有的明星、舞者都自叹不如！没有一个人说话，他们珍惜这一支舞的时间！没有人一知道在这个神之舞结束后会怎么样，是死是活没人清楚，他们的生命都是属于露露女神的…

露露在舞台上尽情的表演着，她知道所有人都已经被自己征服了…

终于…舞，跳完了

所有人又一次齐刷刷的跪了下来，砰砰的磕头，高呼：“神上万岁！谢神上！……”

露露穿上风衣，淡淡的说：“刚才发生的事不要让外界知道，在电视播放时把这一段剪下去…大家都起来吧，音乐会继续，你们不用用眼睛的余光看我，就当我不存在好了。”

露露回到座位上，跳了一支舞，她觉得心情好多了…

音乐会照常进行，一位又一位大家登台献艺，气氛活跃了起来…

菲儿快步走了进来，在露露身边耳语了几句，露露眉头皱了一皱，就让菲儿坐在自己旁边，和她小声嘀咕起来…

五十五章 平凡的间谍

我是听不到她们的耳语的，但是，水晶球可以！

“神上，我知道我不该打扰您的个人时光，可是事态紧急，我必须第一时间通知您，您听完后惩罚菲儿吧…”

露露依旧微笑，用她的手抬起了菲儿的头，喃喃着：“菲儿，有什么就说嘛，神不惩罚你。”

菲儿打算跪下磕头谢恩，但露露制止了她，于是，她只能坐着说了：我们派去带回夏彤的**，被…杀了…”

露露有些吃惊：“谁干的？不想活了么？！”
