# 1

α基因缺失，一种医学界极为罕见的病症，凡不幸患上此症者，肌肉会逐渐萎缩，身体机能持续下降，最后会以非常痛苦的方式死去，而有效的医治方法至今没有找到——甚至连并发症及其带来的刻骨痛楚都无法缓解和减轻。患上这种病的人，为什么偏偏会是我？我生在一个普通的家庭，过着普通的生活，从没跟人打过架，对身边每个人都尽可能的和善，这样的我始终觉得自己会平凡地度过一生——上学，毕业，工作，结婚，生子，直至终老——我也始终希望如是。可上天却极尽讽刺之能事，将我这唯一的小小心愿也无情地剥夺，还慷慨地把这全世界不到千万分之一发病率的厚礼奉送给我。这是什么意思？天罚么？那为什么那些终日游手好闲挑衅滋事的混子可以整日潇洒地出入酒吧，左拥右抱着那些毫不爱惜自己身体的贱女人；为什么那些终日糜烂在灯红酒绿温柔乡而无法自拔的有钱人可以挥金如土，却不肯施舍给穷困潦倒的人们哪怕是一个馒头甚至是视之为人的丁点尊严；为什么从没做过伤天害理之事的我却仅仅因为平凡而遭受这种罪孽？有道是鬼怕恶人，看来老天你也如同那些恶鬼一样，欺善怕恶啊。真是瞎了你的狗眼！这份厚利是在我18岁生日那天收到的。随着漫长潜伏期的结束，它作为一份丰厚的成人大礼献给了我。随后的几年，我逐渐失去了行动能力，如今只能躺在病床上靠仅有知觉的右手点击鼠标浏览网络而勉强度日。“花季少女轻生跳楼，众人围观起哄。”“一男子持刀劫持公交，满车无人反抗。”“两车闹市追尾，围观市民过多至交通中断数小时。”每一天，每一天，这种新闻充斥着媒体的版面，无聊而丑陋，可报道中那些丑陋的人却欢蹦乱跳得无比健康，他们就不该得这种病么？至于那些来看望我的人又为我送来了什么？同情？关怀？不，不，他们只会说：坚强点！你看那谁谁谁和你一样，却做出了如何如何的事迹来，你也不能这样一直消沉下去啊！听听，多么精彩的话语，振奋人心！可是你们这些看客，我不过是个普通人，我无法像那些与病魔斗争的英雄楷模一样有着似乎是与生俱来的坚强意志，我只想过平淡的，简单的，与世无争的生活。苍天，我恨啊！我的身体还有精神日复一日地经受着病魔的摧残，而那颗平静淡泊的心灵也逐渐开始扭曲，变得前所未有的狠毒，它的每一下跳动都是一次对无情命运的诅咒，对世间不公的宣泄。终于有一天，我唯一能动的右手也出现了肌肉萎缩的症状，心脏的搏动似乎再也无法把热血送达指尖，我知道自己的死期已到了。“如果世上那些狗日的杂种能给我陪葬就好了。。。。。。那些本该承受我现在所经历着的痛苦的杂种们。。。。。。”这恐怕是我最后的心声了吧，不是带着一个普通人该有的对生的眷恋，对家人朋友的留念，而是带着这样无比恶毒的诅咒，离开人间。<title>2</title> <link href="../Styles/Style.css" type="text/css" rel="stylesheet">

“你真的这样想么？”我听到了，仿佛有人贴着我的耳边对我莺声燕语般，一个略带魅惑的嗓音敲击着我的耳膜。“要那些人为你陪葬，你真是这样想的么？”我确定那不是濒死的幻听，那声音是如此真实，甚至连穿过牙齿间隙的气流都能感受得到，我认为自己必须要为这声音作出回应才行。“确实如此，我恨不得他们统统死光，只可惜现在这样的我又能做什么呢？”“我会给你1周的时间，在那期间你的身体会复原如初，然后你必须履行你的承诺。”“什么？你要给我1周的时间？你到底是什么人？还有你说的承诺是。。。。。。？”猛然间我的眼睛睁开了，一度因肌肉萎缩而瘦小的不成样子的身体竟然完好如初。我清楚地看见周围家人和医生那惊诧万分的眼神，此时病床边的墙壁轰然倒塌，一只无比巨大的手伸了进来！那手实在是太巨大了，远远超过了病房的大小，将我连同病房坍塌的瓦砾一同握在了手心。惊魂未定的我注意到那指尖柔美的弧度，略长的指甲上似乎涂了一层淡紫色的指甲油，这竟是一只女人的手！我不知道自己在急速升空的眩晕中挣扎了多久，当我感到那只巨大玉手停止活动的时候，便试图从掌心中挣脱出来，出人意料的是这只玉手的主人似乎也想将我放出来。随着手掌的慢慢摊开，一根擎天玉柱般的手指正轻轻地拨开我周围的碎砖，这时我才注意到这只手竟有半个篮球场大小，我鼓起勇气爬到手掌的边缘向下观瞧，就仿佛从摩天大楼的窗户俯瞰城市一样，那高度令我不禁感到阵阵眩晕，10多层高的住院部此刻正处在这手的遥远下方。天呐，这简直太不可思议了！我战战兢兢地抬起头，想看看这手的主人是个什么样子的时候，这玉掌好似回应我的愿望一般缓缓地向上移动起来，接下来我便看到了令我这一生都难以忘怀的绝景：晶莹剔透的双唇，小巧而不失挺拔的鼻梁——说是小巧是因为相对于整张脸的比例，在我实际看来那就像一座大楼——忽闪个不停地大眼睛，那卷翘的睫毛调皮地上下闪动着，乌黑的秀发从有着柔美弧度的脸颊垂落下来，这也让我顺便端详了一下她两腮白里透红的水嫩肌肤。这真是一张标志的美人脸，只是被放大了不知多少倍。我还在为这眼前的绝景惊诧不已的时候，濒死时耳边传来的声音再度响起了。那声音说道：“这就是我赐予你的力量，好好利用吧，注意你只有1周的时间。”好吧，也许留给我发愣的时间并不多了。我知道眼前这有着天使外表的巨大可人儿将会成为毁灭眼下世界的恐怖工具，时间紧迫，就让我们开始吧。<title>3</title> <link href="../Styles/Style.css" type="text/css" rel="stylesheet">

美人嘴角轻轻一扬，带起了一个美妙的弧线，她微微张开了无比巨大的朱唇，手掌一斜将我放入了口中。我随后来到了一个奇妙的世界，四周柔软而温暖，整座城市的风景尽收眼底，那巨大美人的视野与我共享了。这时我才发现她身着黄色的低领短衫和牛仔热裤，只可惜城里不可能找到能照出这曼妙身姿的镜子，要不然我一定要好好地欣赏一番。走吧，动起来吧，你的双足将成为人们的噩梦。大地震颤着，这美人的第一步便踩烂了一座小小的公寓楼，我能清楚地听到公寓楼的废墟中传来的惨叫声，清楚地看到血水慢慢从废墟中渗了出来。这感觉，前所未有的美妙啊。第二步踏出去了，这次消失的是一座商店还是邮局呢，管他的，我只需要让这美人尽情地破坏就好了。现在离市中心还早得很，周围最高的建筑连她的膝盖都不及，大部分只到她的小腿肚子，甚至那些低矮的民房都不及她的脚踝。所以，请尽情的踩吧。如果之前有人对我说，光靠散步就能毁灭世界，我一定会嗤之以鼻，现如今随着街道上遍布的无数血肉模糊的尸体，还有一片片建筑物的断壁残垣来看，这并不是个笑话。轻轻一脚踩下去，便能带起一片火光，一片惨叫，还有一片鲜血。哦？我看到了一大片人群在发疯似的向广场逃去，还有早已瘫痪的交通导致数百辆汽车拥堵在一起，这么好的机会怎能不把握呢？早已不知是多少步了，她每跨一步几乎能横贯一条街道，你们这些蝼蚁又能跑到哪儿去呢？光洁的脚掌并没有因为踩踏无数的尸体而沾上血肉，将一辆辆汽车踩扁时发出的金属声就像极富节奏的打击乐一样令我越来越着迷。对了，这附近应该有个飞机场才对，是的，我看到了，似乎有几架飞机正准备紧急起飞，还有一架正在空中盘旋不愿降落——机长一定看到了她。快去！别让那架飞机离开！随着步幅的加大，大地的震颤愈发剧烈。还好，那飞机不过飞到了美人胸前的位置时就被一把抓住，引擎小小的出力怎能敌得过巨大玉掌的力量呢？我已经可以透过机窗看到里面小人们的惊慌失措了——虽然他们和我一样大小。我还没有对她发号施令，只见她玉柱般粗细的手指轻轻一折，一片机翼便断开了；再一折，另一片也断开了；当尾翼也折断后，整架飞机被削成了人棍，被她不停地拿在手里把玩着。这是要干什么？美人脸上闪过一片绯红，用手中的飞机头部隔着热裤轻轻摩擦着。天呐，她还是个情窦初开的大姑娘呢！好了好了，我们还有正事要做呢，我可是只有一周的时间。美人似乎领会到了我的意图，手中停止了动作。这飞机怎么处置呢？美人，玩过纸飞机么？那就将它抛出去吧，这可是我给你们的逃生机会哦，毕竟坠机生还的概率要远远高于我这疾病的概率啊。病。。。。。。我怎么又想到这该死的病了。。。。。。飞机从美人手中如出膛炮弹般飞了出去，我便把注意力从疾病上转到了飞机场上正待起飞的飞机上。毕竟我的生命只剩一周，这里面可没有多少时间用来感伤。美人来到了塔台边上，俯下身子拨开塔台的屋顶，遮天的玉掌将塔台的里的人们压成了肉酱，对，再狠一点，用你的指甲切断他们的身体，对，就是这样。注意那边几架飞机，你还想那样玩玩吗？好吧，我只给你5分钟，抓紧时间享受吧！美人的脸上红霞浮现，不一会儿就将整个机场润湿了。我无心去听美人发出的阵阵呻吟，眼下有更重要的事情等着我去干。在这座人口500万的城市里，人口最密集的地方在市中心，那里才是最佳的行刑场所。去吧，美人，没有什么可以阻止你！去往市中心的道路同样变成了一幅人间地狱，美人将她脚下的房屋、车流、人群悉数碾碎，鲜血染红了整条整条的街道。途中我们遭遇了军队的抵抗，可那些枪炮子弹打在美人身上如同瘙痒一般，美人微笑着踩翻了一辆辆装甲车和坦克，她将手伸向了地面，巨大的玉手瞬间将数辆警车揽在手心中，可怜的小小警察们一同被挤成了肉馅，然后被送到了美人的口中，她开始大口的咀嚼起来。我看着此情此景，胃中竟有些泛呕。虽然她忠实地履行我的指令，但更多时候是自我“创造性”的行动，这个巨大的美人还真难驾驭。但现在顾不得这么多了，因为我已经能够看到市中心那标志性的建筑，这个城市的地标——黄金之塔。这座因在晚霞的照射下通体反射金色光辉而得名的塔状摩天楼，曾经一度是这个城市的象征，如今作为这早已腐朽不堪的城市的墓碑，还有什么比它更合适的呢？随着她一个大步迈过城区之间的标识线，整作闹市区的景象便呈现在了我的眼前。不愧是市中心，整座城市最热闹繁华的地区，之前凤毛麟角的高楼在这里是鳞次栉比，而且更高更具现代气息，很多数十层的高楼已经达到她腰部以上的位置，还有不少到达她胸部的高度，可惜这些极富现代气息的钢筋水泥马上就要报废了，它们都将彻底拜倒在我们这位美人的通天的玉足玉腿之下。继续前进吧，肆意地破坏吧，这里才是你尽情施展才华的场所啊，我的美人。美人微微颌首，以一种曼妙的姿势将右腿抬过头顶，那右脚脚尖的高度足以傲视整座城市了。就在我猜想她是否要先舞上一曲芭蕾的时候，高高抬过头顶的右脚猛地向下斩落，一座50层左右的公寓楼瞬间坍塌，大半楼层顷刻粉碎化作石雨纷纷落下。扯断的电线引发了火灾，火焰与从破裂的管道中涌出的煤气一触即燃，滚滚黑烟从勉强矗立半坐大楼废墟中冒了出来。一时间，惨叫声此起彼伏，废墟中不知埋着多少尸体。哦，天呐，多么美妙！既然苍天无眼，让这些腐朽的世人快乐地苟活于世，那我便代替你这瞎眼的苍天来行道吧！美人，继续，不要停下！<title>4</title>
