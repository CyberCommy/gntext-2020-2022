# 1

“喂，你听说了吗？4年级的米奈塔变成巨人了.”
我听到一个男生这么说着，心中又是一紧。
米奈塔是我的妹妹，也是我唯一的亲人，就在两天前，她开始长高，不平常的长高，在我们明白过来之前，她已经长到了170米高。
其实也够难为她的了，这么高的她，每天的进食量其实是普通人比例的十分之一，洗澡什么的更是难办。
其实米奈塔并不是学校内唯一一个巨人，还有另外两个女生也变成了巨人，而她们都是我的同班同学，12岁的静璇和玉。
自从她们变大后，麻烦就一个一个来了。
首先，我们学校没有一个教室能容下1个170米高的巨人的头，更别提2个人了。
而光是她们鞋子的高度，就已经超过了20米高的教学楼，为了照顾她们，让她们听课，学校只好设立特殊教室，将一间教室建立在一块长50米，宽50米的水泥板上，上课时放在这几个女生的腿上。
体育课自然没法上了，但是这几个女生的体力出奇的好，就算吃的很少，也能发挥出超过自身数倍的力量。
有趣的是，这几个女生和我的关系皆不一般，米奈塔就不用说了，但是静旋和玉都是我喜欢的女生，两个人也表示过可以接受我，而她们三个还是好朋友。
由于这个原因，每天在我们学校门口都可以看到3个巨大的女生在争吵谁带我回家……
