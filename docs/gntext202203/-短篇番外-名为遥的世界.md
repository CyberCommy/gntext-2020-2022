# 【短篇番外】名为遥的世界

# 1

在真琴家里几次逃跑被抓回来后，家住已经失去了对我的任何兴趣。以极为厌恶的表情看向只有身高1cm多的我，那眼神就像看待房间中的臭虫一般，而后表示再也不想看到我。就这样我被女仆遥带出了房间。

“喂，虫子。你已经没有存在的价值了，你是想怎么样消失？”遥昂着头俯视着坐在她手掌上的小人。

“遥女仆。不，遥女神。求求你放过我吧，把我交给里奈，让她把我变回来。”从坐姿赶忙请求的从坐姿变成跪姿，卑微的恳求道。

“。。。。”遥没有说话，寂静的氛围，甚至听到了她那震慑大地的心跳。

“只要我能变回来，你有什么要求我都答应你。”

“真的？”遥又鄙夷的低下头向这边看。

“是真的”

“怕不是这会儿骗我，到时再反悔吧？”说着就又昂起头冷笑道。

“我对天发誓绝不骗人，骗人的话我就。。我就。。”

“非常抱歉呢，你的要求没办法实现呢。”遥打断了话的继续。

“啊？”

“里奈已经把你抛弃了，对你已经不感兴趣了，让处理掉你呢。”遥将一只手插在胸前，原本不太的胸部，硬是撑起来了一些，凸起的两个小点甚至比我身高还高。

“。。。。”

“不愧是人蛆，看到胸部都能起反应。”

下体的帐篷不知道什么时候涨起来了，为了掩盖自己的尴尬，赶紧说道：“那是因为神圣的遥女神过于美丽，忍受不住了。”

“油嘴滑舌的。”

这时候可不能选择实话实说，万一跟游戏剧情似的说一句你那个不大，飞啊~机啊~什么的就没命了。。

“这样吧看在你恭维我的份上，你自己挑个消失的方式。喜欢哪个女仆，我就偷偷把你放到她的食物里，到时候你就能跟她永远在一起了。或者把你丢到马桶里，像开盲盒一样，猜谁会最先来上你。一想到生命中最后的时间，是跟女仆的尿尿时间长度有关就蛮有意思的。哦，说不定会刷手机视频。也可能是最后张合擦拭的时间。更甚至，万一来的是大的那是不是更有趣了呢”

“。。。。”我的脸已经吓得惨白了。联想到这些场景，这比世间任何的死法都要屈辱，赶忙求饶。“女神大人求您了，我不想死，饶我一条命吧”这时已经顾不得什么尊严了，那种死法不是人的死法。

“现在你知道自己的地位了吧，女仆们的粪便都要比你高贵呢。它们是利用后的残渣，而你连利用的价值都没有呢，只是被抛弃的乐色。”遥上下摇了摇手，就像掂量了下轻重一般。

“只要让我活着，让我做什么都行，我把所有财产都给你。”到紧要关头，什么都豁出去了。

“我可不是图你财产啊（他有多少钱？看样子也没多少吧）可是不让你消失我没法向两边交代，更不可能把你藏起来饲养，平时还有功课。”

“我账号是。。。求您了！”毕竟对方是个JK，应该不会卸磨杀驴，在还原无望下，给点利益对自己也有利。

“我要是带在身边或者放在家里早晚有一天会暴露。”遥女仆有些犹豫。

“只要让我不出现在她们视野就好了。”想着如何躲避其他人。

“对啊”

“？！”

“里奈给了我一个防臭喷雾，说是能缩小物体。要是把你缩小不就没人看的到了吗？”

“那样的话，我跟被抛弃死掉有什么区别”大吼着，头脑一片胀，无法理解言语。

“放心好了，我会把你永远带在自己身边。”

“啊？”

“有一个地方，既不会影响我的生活，又能让你过完后半生。”

“哪？”

“当然是我的脚了~  难道说你还以为在上面？人蛆当然是活在脚下才对吧。让你寄生在我的脚下已经是最大的恩赐了，卫生什么的放心好了，我每天都会按时洗跟替换黑丝的。当然你不用担心会被冲走或者吹走，因为只需将你的磁场转化为之被我吸引就行了。”

就这样我又缩小了百倍被放置在遥的足趾指纹当中，穿入了新的黑丝当中迎来了无尽的黑暗，那原本的足趾主人变成了大地。遥女神变成了星球。时间变得漫长，直到生命终结。。。
