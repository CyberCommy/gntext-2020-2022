# 1

有这样一个巨大娘与人类共存的世界。
巨大娘们不仅有着巨大的身躯，而且有着强大的力量，人类只能饱受不同大小的巨大娘的摧残。
如果是百米那么大的，人类可以逃跑，甚至可以利用武器赶跑她们。
如果是千米那么大，人类除了跑得远远没有别的选择。
可如果有万米那么大，人类连跑的选择也没有，因为人类能不能逃跑的选择权在巨大娘的手上……
在一个小村庄里，有一个牧童，他每天都会带着自己家里的几十头羊到村子背后的山上放牧。
可是有一天，牧童吹完笛子后闲的无聊起来，他看着蓝蓝的天，突然有了一个想法：
“我如果叫着巨大娘来了，那些人会不会被吓到啊？”
平常就喜欢恶作剧的牧童把羊牵回去，然后站在半山腰上朝着村子大喊：
“巨大娘来啦！巨大娘来啦！一百多米高的巨大娘来啦！！！”
牧童喊完之后，便看到村子里有的人落荒而逃，还有的人带着武器跑了出来。
他们找到半山腰上的牧童，问他巨大娘在哪？
牧童看着焦急的人们，笑的前仰后合，然后说：“我骗你们的啦，没有巨大娘。”
人们放下武器愤愤而去，留下笑的躺在地上捂着肚子的牧童…………
一个月之后，牧童把自己的羊再次牵回来的时候，又有了一个恶作剧的想法。
想到什么就做什么的牧童再次跑到半山腰上，朝着村子里大喊：
“巨大娘来了啊啊啊！千米高的巨大娘来了啊啊！！”
村子里的人听到后连忙跑了出去，整个村子顷刻间便变得空无一人，看到从村口焦急离开的村民，牧童趴在羊背上笑的合不拢嘴。
不一会儿，就有机智的村民发现四面八方并没有巨大娘出现，村民们揭穿了牧童的恶作剧，并狠狠地教训了他一顿，然后留下被骂了一顿还是开怀大笑的牧童回村子里了……
第二天，牧童照常带着羊群到山上放羊，可是，一直以来乖乖听话的羊群这一次却不听使唤，一个劲往外跑，牧童感到奇怪，他爬到山顶，站在那颗老树上。
当牧童从地平线看到什么东西冒出来的时候，地面突然开始一震一震的，就像一个面团被什么东西拍打，而且拍打得那样重，空气好像也被这恐怖的拍打撞击开了一样，一股大风朝这边吹拂过来，把已经不剩几片叶子的老树吹的摇摇晃晃的。
牧童的草帽也被吹掉了，甚至被这跟随一震一震的地震来临的大风吹的老远了，他也没有去捡。
“啊啊啊，巨大娘！！！啊啊啊，万米的巨大娘啊啊啊啊！！！！”
牧童发了疯一般飞下山，跑到村子里大声呼唤着——可是因为之前牧童骗了他们，没有人相信牧童，他们朝牧童投来同情和鄙夷的目光，继续做着自己的事。
“巨大娘啊！！…………不！！我没撒谎！！”
牧童已经被吓到小便失禁，简直是屁滚尿流了，可是仍然没人相信他，大家相信这只是他为了恶作剧而做的演技。
在牧童解释着震动时，震动突然停止了，村民们更不相信牧童的解释了——因为村子本来就在小地震频发带上，大家都以为刚才不过是例行公事。
牧童跑出村子，犹豫再三，然后跑向了那十多万米高巨大娘的位置。
虽然对于那样的巨大娘来说，牧童还不如蚂蚁大，可是有着天文望远镜和电子显微镜级别视力的巨大娘还是看到了朝她跑过来的牧童。
巨大娘放下右手，把牧童连同他周围比村子大好几倍的土地扣了起来，抓在食指和拇指上，细细地打量着这个因为害怕，站都站不稳的“人类”。
“小家伙，你为什么不怕我，反而还要靠近我？”
巨大娘轻声说着话，可是对于牧童来说就像噪音一样震耳欲聋。
多么巨大的存在啊！自己甚至没有武器，自己的笛子就算能当做武器，也恐怕伤不到巨大娘的一根手指吧。
可是自己必须要上；牧童知道自己必死无疑，可是还是必须要上。
“我……我要阻止你！不能让你毁掉村子！”
牧童就像风中摇曳的花骨朵一样脆弱，随时可能会消失，恐惧让他只能颤颤巍巍地趴着说道。
“你很勇敢啊，人类里像你这样勇敢的人可不多。”
说完，巨大娘站了起来，一步都不需要，就走到了村子附近，她把右脚抬起来，用小脚趾对准了村子和周围的群山所在的范围。
“你们这些小虫子居然瞧不起我，去死吧。”
尽管刚经历了一次无比震撼地地震，可是当村民们听到巨大娘的声音还有感受到变黑暗的天空时，已经来不及了，巨大娘的小脚趾已经将村庄连同周围的群山一同碾平了，变成了土黄色。
在面不改色地轻易杀死了几千人之后，巨大娘把目光放到了右手的牧童身上。
“至于你这个小家伙，做我的男朋友吧。”
随后，巨大娘带着牧童离开了这个地方，两人过上了幸福快乐的生活。
