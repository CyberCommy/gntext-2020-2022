# 1

随着科技的发展，缩小技术开始普及，大量公寓都安装了缩小门，让人们生活的空间大大增加，而孝治在大学毕业后便在一所这样的公寓成了管理员。
    “孝治哥哥，今天我妈要很晚回来，她拜托你‘照顾’我，早点过来，好吗？”这天下午，孝治正在管理室待着，和母亲一起生活的小学生玲美放学回来带着甜美的笑容对孝治说道，照顾两次上还加上了重重的语气，熟悉的孝治明白了她的意思，有点无奈的笑着答应了她。
    玲美高兴的先回房间了，孝治又待了一会儿，把工作交给了和自己一起管理的丽子小姐，便回到了自己的房间。
    为了能够更宽敞，孝治装的是能缩小到百分之一的缩小门，而且托这之福，他还发现了这个公寓的一个小秘密，在这个管理员房间里居然有小小的通道能到各个房间里去，而且正好适合他现在这样的大小，所以在当管理员之后，利用这通道，他得到了很多有趣的体验，有好的，也有坏的，也有和玲美的糊涂关系这种不知是好是坏的，想到这，他不由想起和玲美关系改变的那一天。
    那天，他突然童心发作，通过通道来到了玲美的房间里的游乐场中玩耍，由于玲美母女的房间是40分之一的比例，所以对他而言就相当于在一个2.5倍的游乐场中玩，结果没想到那天玲美提前回来了，来到游乐场看到了正乐在其中的孝治，上前抓住了他，虽然玲美没有说啥，也没有告诉自己的母亲，但从那时开始，两人间的关系就发生了微妙的变化。
    孝治想着的同时已经通过通道来到了玲美家所处的大房间,玲美所在的40分之1大小的别墅的门是开着的,孝治直接走了进去,来到了玲美的卧室,玲美正坐在办公桌那儿发呆,孝治走了过去,本来的小女孩现在就算是坐着也比孝治要高的多。
    玲美发觉孝治到了，回头站了起来，“孝治哥哥，你来了啊。”她拉着孝治来到镜子那儿，这是玲美现在最喜欢做的事情了，只见她哪怕是坐到地上，还要比现在的孝治要高，两人在房间里一起玩耍了一会儿，玲美累了坐到了床上，看着眼前的小哥哥，露出了笑意，先把孝治赶了出去，然后过了一会儿把孝治喊了进来，这时候她已经换掉了衣服，换了一身成熟的女牛仔装，孝治被她拉上床，两人又玩耍了一会儿，玲美感觉困了，便拉着孝治睡在旁边，很快睡着了，这时孝治爬了起来，看着玲美可爱的睡姿，忍住了自己心中的冲动，离开了这儿，看了看时间，这时候快到美沙到家的时间了，连忙通过通道往美沙的房间跑去。
    美沙的房间离玲美的房间不远，当然对只有百分之一大小的孝治来说，那还是花了一点时间，他还是没能在美沙下班前到达，这个房间又比玲美那个房间要大一些了，如果说在玲美的房间里孝治只感觉自己如同一个小孩，那么美沙的房间对他而言已经是巨人的房间了，还好美沙把门都打开着，不然孝治恐怕都难以进来。
    孝治来到了美沙的卧室中，发现美沙正坐在凳子上休息，看样子工作有点累了，孝治来到了这个膝盖都比自己高的女巨人面前，抬头往上看去，正好对上了美沙低头往下看的双眼。
    “孝治，怎么这么晚才到？”美沙站了起来，低头对着孝治不高兴的说道。
    “玲美她母亲不在家，我去照顾她的。”孝治抬头对美沙解释道。
    “是她照顾你吧？”美沙还是有点不高兴，不过没有再纠缠这事多少，两人都发觉这样子对话有点吃力，于是美沙把孝治抱到了床上，不过就算这样，孝治沮丧的发现自己也直到美沙的大腿高。
    “孝治，明天我们公司要去海边旅行，你也陪我一起去吧。”美沙低头对孝治说道。
    “你们公司不是都女性吗，我去不方便吧。”孝治疑惑的说。
    “方便了，你待在我的包里大小正好。”美沙说。
    “额，我好好想想。”孝治明白了美沙的意思，她是想把自己当作娃娃一样带过去！
    “哼，不愿意就算了。”美沙气呼呼的转过头去，背对着孝治来到了书橱那儿，孝治连忙走下床也跟了过去，可只能对着美沙背对着他的膝盖，还好美沙很快蹲了下来，让孝治不至于那么吃力。
    “美沙，你知道的，我要值班的啊。”孝治对美沙解释道。可美沙不理他，就在那对着书橱看。
    “好吧，我想办法请假陪你去，这总行了吧。”孝治无奈的说。
    “这才对嘛，我可爱的小孝治。”美沙站了起来回头看着抬头仰望她的孝治，露出了笑容，然后来到床边坐了下来。
    “不过你的包要收拾好啊。”孝治也提出了自己的要求。
    “这你就放心吧，小甜心。”达到目的的美沙高兴的说，然后从地上站了起来，往门外走去，孝治也连忙跟了上去。
    “孝治，这衣服还不错吧。”美沙换掉了内衣对孝治说道。
    “当然了，美沙，你好漂亮。”孝治抬头看着美沙赞美道。
    “虽然直到你是顺着我，但还是挺开心的，小可爱。”说着美沙带着孝治来到了洗漱间，“今天有点累了，所以我准备早点休息呢，正好也帮孝治你好好清洁下。”美沙低头对着脚边的孝治坏笑道。
    发觉不对的孝治想要逃跑，可他哪里逃得了有自己四五倍大的女巨人的手心，被美沙脱去衣服放到了洗水池里好好的冲洗了下，而在这同时，美沙自己则到旁边的浴间里也洗了把澡，回来把孝治身体擦干净，穿上了衣服。“孝治，明天我们要去的海边旅社有混合浴室哦。”同时换上新衣服的美沙边诱惑着孝治，边抱着他回到了卧室。
    美沙先回到工作桌那把自己的文件收拾了一下，然后回头来到了床上，“孝治，期待明天吧。”美沙微笑地对孝治说完就躺了下来开始睡觉。只留下旁边被她诱惑的欲火上来的孝治无处发泄。就在孝治考虑是否靠上去的时候，美沙的身体翻了下身，那庞大的身躯差点压到了孝治，也打消了他心中的那点邪念，想到眼前的是一个稍微动动就能让自己受重伤的女巨人，可那对着自己的美丽面容又让孝治舍不得就这样离开，还好这时美沙又翻到了另一侧，让孝治只好期待起明天的旅行。
    被挑起欲火的孝治这时候可睡不着,没办法之下,他只好硬着头皮冒着被榨干的危险往里奈家走去,好解决自己身体里的欲火。
