# 四戰紀實--奇里乞亞的上空 7797帕裡3突15天 光日

作者：clivic

# 1

持續了十六個帕裡（一個帕裡相當於0.76個“年”，其中年是宇宙時代之前的托斯德爾星球時間計量單位）的第四次世界大戰已經接近了尾聲，爲什麽這麼說呢？

德利米克聯盟軍可以說接近分崩離析了，自從盟軍的敵對陣營庫魯聯合軍成功施行美洲登陸戰後，托斯德爾星球這個盟軍的本土，也是所有托斯德爾星人古籍中稱為“地球”的母星，就不再是盟軍可以掌控的了。對火星的亞細亞大同主義聯合國的入侵陷入持久戰的僵局後，盟軍的托托里帝國再也難以抽出手來幫助遭遇本土登陸的十七區聯合自治國（另稱美洲自治）。本來大佔優勢的德利米克聯軍（德利米克在托斯德爾通用語中意為團結能力，擁抱和睦）只能接受速戰速決的期望之破滅，陷入了曠日持久的托斯德爾本土戰。

聯合歷7796帕裡，也就是戰事進行到第十五帕裡時，美洲已經全境“解放”，美洲自治也投降，建立了新的十七區傀儡政府。戰事的終結也指日可待了。亞細亞聯合國粉碎了托托裡帝國（托軍）的最後一次集團作戰“三號作戰”，於是也準備登陸歐羅巴大陸。可是托托裡帝國強大的宣傳機構和統籌機構使得所有的托托裡人義憤填膺，決不投降。亞軍佔領了歷史悠久的古城，也是托托裡巨大的商業中心米蘭之後，和成功擊敗美洲自治和另外幾個德利米克聯盟軍的盟國的庫魯人、帕斯提人還有埃克森星球的弗地崗人夾擊進攻已經遷都托斯德爾本土亞細亞的浦路克裡斯的托軍。

直到7797帕裡的1突（一帕裡有十“突”，相當於古歷中的月，但長度不同）。失去了所有的昔日輝煌的宇宙聯合艦隊的托托裡帝國，在人民的支持下負隅頑抗著，聯軍傷亡慘重，也無心戀戰了，畢竟托軍的科技也是一流的。庫魯民主主義聯合國（庫軍）情報部門樂觀估計也還要三帕裡才能結束戰爭。可是庫國總裁（相當於宇宙時代前的總統）出席了卡拉庫木星駐兵基地的絕密會議後，欣喜若狂地對書記們喊道：百姓們終於能夠迎來和平了！

當然，這裡聯軍所說的百姓們，目前是不包括托國的“邪惡狂妄殘忍”的人們的，因為。。。

在托國政府蠻橫地拒絕掉第三次無條件投降警告後，庫國總裁“傷心地”對托國外交部發表了一番演講，主要是說，希望古代第二次世界大戰的悲劇不要重演，人類需要和平。托國當然不會理睬這些“虛偽”之言，可是政府智囊團還是敏銳地覺察到，難道對方有像二戰中壓倒一切的最強武器原子彈一樣的科技研發出來了嗎？然而戰事不能就因此結束，不僅是盟軍所有托斯德爾人的臉面，還有人們對政府以及政治理想的期望。。。

7797帕裡3突中的一天，中西半島的奇里乞亞上空突然暗了下來，托軍立馬做好了警戒，可能是光能離子系統入侵之前兆。然而不一會兒就又亮起來了。然而，天空中的大氣層中，多出了一個巨大的藍色光環，雲氣在一旁繚繞，反射出似藍非藍似紫非紫的光，隱隱投來陣陣恐懼。傳送門在四戰中也不少見，畢竟當年聯合軍攻擊托斯德爾星球時就用上了傳送門科技，托星人在宇宙時代初也早已有實驗室開發出微型傳送門。可是這次光能離子作傳送門的目的僅僅是空投軍隊，也應該是不明智的，因為內陸空投難以接應的教訓，在當年十五區反抗聯合軍的戰事中已經讓各國軍方深深銘記。可能又是鐳射機空襲吧，托軍士兵們仍舊像以前一樣，疏散人們到地下避難硐，并部署自動化機械人警戒。然而那一千米高的的傳送門（是的，米的單位和古代相同）又不能不讓將軍們起疑是否是一次大規模登陸戰，但畢竟與情報上的敵軍軍力戰力不符啊。

十五克裡夸（時間計量單位，相當於0.33“小時”）後，傳送門隆隆作響，劈裡啪啦聲中，不時傳來陣陣閃光。這是正常的，但是又是對哺乳動物有潛在致盲效果的傳送門軍事武器化體現。人們紛紛低下頭閉上眼睛。過了一會兒，人們抬起頭，傳來陣陣驚呼--

那是什麽鬼東西？

那是什麽？

展現在人們眼前的，不是一群軍隊，她只有一個單位，就一個個體。

但是，那是一個身高八百六十六米的托斯德爾星人形態的個體，準確的說，是年輕的母性托斯德爾人，我們姑且用古詞“少女”或者“小女孩”來形容她吧。她擁有著光滑的黃色肌膚，看上去是古亞洲人的血統--雖然進入了宇宙時代後，托斯德爾人類的“微型進化”已經讓人種之間之區分越來越不明顯了。她還是穿著著基本的防輻射裝置服，但是只遮住了上聲的胸部，但用來吸收紅日光的乳頭則還是由透明的光透材料包裹，以便補充能量。可是這樣的往往只會出現在小說裡的“巨人”，也會和正常質量的小人一樣需要吸收光波能量（這是宇宙時代托斯德爾星人“微型進化”的一個體現）進行光合作用？巨人的確不多見，哪怕是和托斯德爾人類似的哺乳動物埃克森人（比如弗地崗種族）以及聯合軍合成軍人，都是和托斯德爾人差不多身高，但是。。。這到底是什麽？

巨大女孩轉過了頭，如果你在高空，看到她那明亮的清澈的眼眸相信你很有可能被她的純潔可愛所打動--如果她是常人之大小與質量，那看起來也就22帕裡克（形容年齡）的樣子，並且勢必會獲得許多大人的憐愛和“風流男子”的回頭，讓人愛不釋手。但畢竟她是那麼巨大的體型。她梳著可愛的盤起來的頭髮，臉龐出乎常態的光滑，除了必要的毛髮，正常的托斯德爾青年可能生長出來的青春痘絲毫不存在。她保持著微笑，不是明顯的快樂，也不一定是哀傷，只是淡淡的，淡淡的，藐視世界萬物的堅定眼神。

這就是一個現實，一克裡夸前托國人還在疑問這是什麽？現在已經得到答案--聯軍的必殺新科技！但是那個巨人現在除了轉磚頭，也並沒有進行移動，這讓人們有了“充足的”時間去避難，也有了“充足的”時間去集結軍力抵抗，但是，不要總是把事情往壞處想，誰知道那是什麽呢？

女孩仍舊微笑地看著地上渺小的萬物，沒有移動，只是靜靜地掃視。難道僅僅是窺探情報實際上不堪一擊？還是根本就是假的巨人--用少女的投影掩飾，只是個高空中的攝錄機窺探盟軍的情報？畢竟托國的蒲普璐研發中心就在奇里乞亞郊區的山穀裡，雖然武裝探測軌道機（古稱“衛星”）由於超強誇克波干擾極難發現，但是以此巨人所在之位置，如果發現的話地點座標就盡收眼底啦。所以不管怎樣，托軍都要去把這個威脅區處理掉。

37個編隊的自動化機械人集結起來朝五個不同的方向逼近女孩，作為托托裡人崇拜的吉祥數字37，真的是代表順利的任務嗎？還是連作戰都要迷信的悲哀？

當3號分隊的機械人接近時，影像出現了。女孩全身除了胸部和襠部用白色的看起來像絲綢的防輻射裝置服包裹，其餘全系赤身裸體，光滑的腳丫所踩之處使得地面下榻整整1.12米，四處繚繞的泥土成灰還未完全散去。女孩所傳送的地方是一座黃土山，曾經是古國巴基斯坦的軍事基地，叢林繚繞，而自從三戰爆發後，就光禿禿了兩百多個帕裡。而這時，它甚至已經不能稱之為“山”了。女孩巨大的脚板起碼有四十米，剛剛傳送時閃光的一瞬間，最高處的山頭就因為這隻可愛的小腳轟然下榻，引發了大量碎石墜落，現在還停留在機械人前進道路的沿途。巨大的左腳將這座山活生生劈成兩半，山石滾落到她嫩黃的腳丫上，其中不免尖利的碎石，但是看上去絲毫不能構成損傷，反而是那仍然光滑的腳丫將尖尖的部份磨平了，然後墜落至地面。爾右腳，踩在四百米開外的小渠上，小渠被攔腰斬斷，直接造成塌陷，下榻的部份河流流過，迅速集成了一個寬約三十米的小湖。這已經是一個小型淺水游泳池了。河中踩的部份那裡的小魚，完全被壓碎在那巨大的絲毫不能反抗的腳底，就算是倖免於難的魚，也被踩下時造成的巨大物理衝擊和水波衝擊彈飛到地面，然後也失去了生存的權力。灰塵在空中繚繞，似乎漫無目的。

看來不是影像！根據一個機械人傳來的數據監測表，軍官們分析得女孩質量能把山脈輕易切開還要下榻1米多很有可能在75萬噸以上，這等質量是哪裡來的？這樣的科技，蒲普璐研發中心也沒能結項這個課題。托軍軍官中間開始瀰漫恐慌。突然，一個機械人的錄像儀突然花屏然後損壞，從其他機械人錄影所見，它是因為靠的太近，電波干擾並且也被巨大女孩按照牛頓定理吸過去了，然後撞擊到女孩可愛的腳丫側翻在碎石中。

平常如果是庫軍，早就抱作一團，緊密退後嚴格監視。可是那可是英勇的托托裡人！哪有畏懼呢！指揮官馬上下令全番隊進行試探性進攻。剎那間考的最近的3號分隊機器人鐳射管轟隆隆地齊射，射出了道道紅黃色的激光，打在女孩光滑可愛的腳踝，腳趾，腳背上。之後面的機械人輔助以離子中空炮，墜落在女孩的腳丫和肉肉的小腿上。這對一個“小”女孩來說，未免太殘忍了吧？

離子中空炮的特點就是會起一大堆的煙滅煙霧，灰色透著紫色的煙塵包裹著女孩的小腿和腳丫，然而等它過去後，司令部的托軍軍官得到了他們意料之中的最不詳預感--女孩的表皮依然是那麼光滑，似乎是會自我修復似的還是有檢測不到的防護罩包裹，根本就沒有任何損傷，鐳射打上去竟然根本得不到燃燒的效果，難道這個人類已經根本不是人類的皮膚？改造過的皮膚綜合熔點看來及其之高，根本不會受到激光的高溫影響和爆炸影響。離子中空炮也似乎對小女孩來說只是玩具彈頭一樣的存在，看她現在的反應就知道，她仍舊是那副迷人的表情，搖擺著頭俯視著地上渺小的萬物，絲毫沒有理會，甚至可以說，注意到，那攻擊的存在。

無人機番隊在托軍接受了預料的最壞情況而發佈的正式攻擊令下立即出動了。一下就17個編隊，七百多架生物無人機，以每托裡夸（77托裡夸等於1克裡夸，時間單位）138米的速度（相當於古代每“小時”488000米）轟鳴而來，而那400多米的高度，最多也只是到女孩的襠部位置，迅速圍繞起了女孩，發出了陣陣光線和光彈。機械人編隊也展開瘋狂攻勢，那座女孩腳下的山徹底沒有了，這次是被狂轟濫炸而填平的。離子中空炮轟炸過的地面呈極黑色，用燒焦一詞也不夠形容它的狀態--就像一層極薄的黑色焦膜貼在之下1米寬兩米深的窟窿之上，而窟窿內的土質早已成燒紅之色，白煙裊裊。所以也不難怪為何小女孩腳丫如此光滑為何是“最壞情況”了。激光達到地面則是一個個激起一米多高的爆炸霧和半米的燒紅窟窿，而到了小女孩的身體，不管是腳丫還是飽滿的潤滑的小腿，都完全改頭換面，一聲不響，匿跡在那動人的皮膚上。

奇怪的是。小女孩仍舊紋絲不動。這是聯軍的實驗嗎？

狂轟濫炸持續了九個克裡夸。可能是出於托斯德爾人的文化習俗，無人機根本就沒有攻擊女孩襠部的脆弱的“衣服”--至少相比她那驚人的皮膚來講是脆弱的--但幾十架繼續攀爬的無人機卻密集轟擊，在女孩遮胸的防護服處輕易地炸出了好幾個窟窿。當然同樣的，那對可愛的飽滿的雙峰絲毫沒有塌陷，傷害的跡象，爆炸過後，還是原來的粉嫩之色，那粉黛的乳暈，伴隨著被炸個稀爛的乳罩防護服，在紅日的照耀和雲霧的映射下顯得格外迷人。甚至是那托軍引以為豪的遠程雷光導彈(一種導彈類型的光彈），射到女孩的身體時，大多改變航方向反彈回來，稀稀零零地墜落在地，然後引發一陣陣的爆炸，甚至將幾個托軍自己的機械人炸得面目全非，焦黑的零件四落。

女孩似乎還是沒有動靜--

直到她抬起了她的左腳。

左腳一抬，由於巨大質量與氣壓的關係，引發了四十米範圍內的一陣往剛才腳丫所在的中心颳的颶風，靠的近的機械人們紛紛被吹到空中，然後撞在一起，遠的也紛紛搖晃，站立不穩。然後，幾乎是刹那間的，女孩急速地往左邊幾十米的地方踩塌了下去，整個人的中心似乎稍稍偏向了左邊--這裡是許多機器人圍起來向她射擊的地方--幾乎是每托裡夸255米的速度（相當於古代每“小時”763000米）。刹那間一聲轟隆的巨響傳來，連司令部和逃難的居民都聽得很清晰。地面塌陷了2.7米，所踩踏的地方瞬時出現了條條斷層裂痕，顏色也改變了，估計溫度超出了7.0普卡（古代120攝氏度）。當然小女孩是不會有任何損傷的，然而機器人們就沒那麼幸運了。他們有的頭被當場瞬時踩得灰飛煙滅，完全變成顆粒一般的存在；有的就是被超強的氣流氣壓彈飛，然後跌落到女孩的腳背上或是倉痍的大地上粉身碎骨。接下來，小女孩將她的左腿快速勾起，不用說又是一陣巨大的氣流，然後，又是超快速地朝前鏟去--地面一下子被削去一塊曲面，上面的殘存的枯萎樹木還有昆蟲蚯蚓什麽的，隨著飛起來的土快速射向後排射擊的機械人--不消說，這幾乎瞬間的動作機械人根本來不及躲避，然後被這二十多米的大土塊給擊倒壓垮，起碼有十來個。

然後又是轟隆一聲，女孩踏下了左腳，伴隨著幾個弱小的機器人絕望的宕機聲又是一次強烈的塌陷，還有微小的地面震動。然後，小女孩開始往前走動，仍然保持著那個微笑，牙齒微露，如果她只是一個常人般大小的女孩，必是讓眾人憐愛的。右腳抬起時，那被截斷的河流隨著女孩大腳的抬起突然噴發好幾米高的水柱，伴隨著數不盡的灰塵和稀裡嘩啦水聲，然後轟隆一下，輕易地踩碎了河畔一個鈦制墻結構的屋子。

托軍軍官們有的冷靜，有的則按耐不住恐慌了，難道她是無敵的？這樣的質量，輕的物體不消說很有可能會被吸過去，就僅僅靠氣壓，就能滅掉一大群軍隊。

女孩遲疑了幾托裡夸，然後，她緩緩地舉起了她緊緊握拳的左手，當然又帶動了一陣狂風，然後伸出了她光豔動人的右臂，將筆直攤開的右手手掌搭在左手的拳頭上，呈L型直角狀。然後，她將那美麗光滑的右手小臂，瞄准了那座苟延殘喘的奇里乞亞郊外建築。突然，那少說也有150米的巨大手臂，突然閃現了耀眼的大面積白光，這是女孩的超級必殺技吧？這道光線若是發射出來，那絕是約100米寬20米厚的極具破壞力的巨大爆炸，那是100000個三角機甲中空炮（這是沿襲了古時聯合軍進攻托斯德爾行星的一種機器人用武器而研發的先進中空炮）也抵不上的巨大破壞，這樣一來，那裡整片土地，也將被瞬間蒸發而轟平吧。

可是奇怪的是，女孩的發光的巨手，又慢慢黯淡下去，接著逝而不見。女孩看來放棄了進一步全面的的殘忍抹殺，就算是敵人，也會有仁慈的一面。可是托國殘存的支援空軍偵測到了空中庫軍的電波--推測應該是女孩用電波和總部交流過--難道是庫軍讓女孩放棄了進攻郊外？他們意識到了那是他們的敵人的寶貴科研基地而決意保存下來？這暫時都不得而知。

女孩雙乳又噴出了陣陣汁液潤滑了殘破處處冒著黑煙的大地，給整片悲傷的大地帶來最後的安靜平和的死亡樂章。最後，她轉過身，伴隨著轟隆隆的巨響，向最先前那被她瞬間踩個稀爛的那個微不足道的小沙丘（從地上看足以稱之為一座山，當然，開裂的山）走去。女孩始終保持著裂開嘴的微笑，是那麼的高貴，那麼的迷人，有那麼的堅定。

奇里乞亞上空又突然暗了下來，隨之而來的便是那先前的傳送門。伴隨著幾次忽明忽暗的巨大亮光閃爍，雲氣環繞的巨大傳送門忽然打開。女孩便一步一步地，走向那裡，即將進入門中時，她停下了右腳的步伐，定了下來，隨後回過頭，向那滿目蒼夷的奇里乞亞大地投射了一道迷人燦爛的笑容--這大概就是真正的“回眸一笑”吧！隨後，轟呲呲呲。。。伴隨著持續始終的傳送門亮光閃爍，天空又突然全暗了下來，因為充斥在大氣中的光幾乎全被那巨大的橢圓形傳送門所吸收進去。等2個克裡夸之後天空重又回覆光亮時，那迷人的小女孩已是不見蹤影，傳送門的淡淡輪廓，也漸漸化為遠處的一道白煙，與大氣消融不見了。留給奇里乞亞的，是一片焦黑，開裂，斷層，和仍在劇烈地震崩塌的殘破土地。建築物的廢墟根本難以辨認，昔日100層的高聳建築，也早已沒有了那輝煌的身影，現在留下的，則是那根本難以找全，或者早已蒸發，的廢鐵。整片焦黑的土地，除了汩汩流動的殘留乳汁，根本毫無生機。人或其他動植物的屍體，也幾乎完全無法找到了，難道戰亂時代的人們根本沒有享用墓碑和骨灰的權利嗎？

7797帕裡3突15天（時間單位，相當於“日”），成爲了奇里乞亞必將久遠流傳的紀念日。死傷57萬托斯德爾星人，包括軍人，包括民眾。
