# 1

本来想写一篇VORE的文了，可是一想到静儿和学姐超长的评价，兔姐的奖励点就下次再说吧！

*****************************************************************************************************************************

太阳系中突然出现了一个少女，静静地漂浮在宇宙中。

她看起来是十九岁左右，身上穿着一套深蓝色的紧身制服，将她那完美的身材包裹在里面，只露出嫩白的双手与双脚，与她那垂到背部的乌黑长发有着鲜明的对比，而她那绝世的容颜足以让所有男人为之疯狂。但是，任何人都无法与她花前月下。她是那么的巨大，地球在她的眼里只是一颗漂亮的玻璃球，而人类只是细菌般的存在。几乎所有人都停下了现在做的工作，望着天空中的巨大少女。

少女用一种迷茫的眼神看着地球。一周过去了，她似乎明白了什么，一副粉色的细框眼镜凭空出现，恰好架在了少女的鼻子上。在眼镜的帮助下再看向地球时，眼神中充满了惊讶，又夹杂着一丝不满。但当她看了三天后，双眼又变得无神了。

任何人都不知道少女在做什么。各国首脑经过了一天的商讨后，他们决定对少女的耳朵中发射一支装载着友好问候的大型播放器的火箭，以表达地球人的友好意愿。但令所有人都意料不到的事发生了：少女身上的引力太强了，地球也只能勉强抵抗，但小小的火箭就在这引力的作用下撞上了少女的手指，变成了一团小小的火花。

少女的表情变得愤怒了。她对地球伸出了一根手指，点在了地球上，北美洲在这次接触中变成了世界上最深的盆地，环绕的山丘则是她的指纹的凹陷处印成的。少女的行为使人类感到了绝望，根本没有任何办法可以活下来。

随后，少女的拇指和食指伸向了地球，落点是亚洲和南美洲。人们只有呆呆地看着天空上那块肉色的陆地降下，等待着死亡的降临。只有一个穿着女仆装的萝莉拉着三个不断挣扎着想留下的少女，见实在来不及了，只得一咬牙，一人一拳都打晕，小声说：“兔姐、小静、学姐，对不起了！”打开一道空间裂缝，将她们一个一个扔进去，自己也钻进去消失了。

地球被少女捏住，随后被推进了少女的嘴里。洁白的牙齿轻松地将地球切开、磨碎，粉红的舌头则将地球的碎块。很快，地球便变成了一团混合着少女唾液的糊状物，通过长长的食道消失在少女的胃中……

*******************************************************************************

在探测器的引导下，我传送到了一个星系中，找到了那个探测到智慧生命体存在的星球。可是……

我诧异地看着面前这个还不如一颗葡萄大的小星球，上面只有三种颜色：蓝色的水层、少许的绿色苔藓与许多灰色的地衣，分布在星球的表面，哪里会有什么智慧生物？不仅如此，这个星系的其他星球也都不大，最大的是那个恒星，像是一个香瓜，放出的热量还不如一个吹风机大。不过探测器应该不会出错呀？

当我联想到这颗星球的大小后，这才恍然大悟，连忙戴上了一副最大倍率的放大眼镜，这才看清了这颗星球上的样子。那蓝色的水层，是星球上的汪洋大海；绿色的苔藓，是星球上的浩瀚森林；而灰色的地衣，竟然是人类的城市！在放大眼镜下这些人类也只有蚜虫大小，不过却全都在对我指指点点的。由于没有珠听装备，我不知道他们在说什么，但对远道而来的客人指指点点的也太过分了吧！

不满归不满，工作还是要做的。于是我开始向飞船上传输这颗星球的数据。可当传输到一半时，一枚小小的导弹从这颗星球上发射，击中了我的手指。虽然他们的攻击没有对我造成任何伤害，但这种行为令我十分愤怒。我中断了传输并删除了已经传输完的数据，因为这里可并没有任何星球哦！

用一根手指轻轻一点，这颗小小的星球的表面便清晰地印上了我的指纹，并偏离了它原本的轨道，向那颗恒星飞去。我连忙将它捏住，这样毁灭它可太便宜那些妄自尊大的小虫子们了。可是该怎么处理它呢？对了，不知道一颗星球是什么味道的呢？于是，我将它放进嘴中缓缓的咀嚼着。

嗯……吃起来像是质地较硬的软糖，而咬开后会有一股细细的暖流滑过口腔，虽然没有什么味道，但那口感却让人着迷！于是我将它咽下，又将目光看向了这个星系的其他星球……我开动了！

十分钟后，我一脸满足地回到了飞船上。坐在椅子上，我一边回味着刚刚的各种美味，一边上传了错误报告：探测器错误，坐标处无星系……**********************************************************************************************************************
