# 1

又是一个美好的早晨，我起了个大早，精心梳洗打扮了一下，出门之前还不忘往身上喷了点香水。

大清早的太阳还不至于这么猛烈，和煦的阳光映照在艾尔文森林草地的露珠上，一闪一闪的仿佛把空气都染成了金色。

今天是我作为圣光的学徒毕业的日子，我将前往暴风城参加毕业典礼，这意味着从今天开始我将真正成为一名[提尔](http://www.baidu.com/s?wd=%E6%8F%90%E5%B0%94&ie=gbk&tn=SE_hldp00990_u6vqbx10)之手白银骑士团的一名圣骑士。

其实从闪金镇到暴风城的路途并不遥远，但我确走的很慢，虽然从我出生就一直在期盼着这一天，但我怕我还没有完全做好准备。

“宏伟的暴风城！”当我见到那座伟大的城市时我不由地赞叹了一句，远方那高耸的台阶之上，便是国王的宫殿，但我要去的并不是那里，而是市中心的光明大教堂——我的圣光导师乌瑟尔·光明使者的居所。

“在这里，史蒂芬！”大教堂门口，一个身着长袍的女士远远地见到我就冲着我挥手，“你怎么这么慢，父亲已经在里面等候多时了。”

这位女士和我一同在光明使者名下学习圣光，在我眼中，世界上已经不可能有比她更美丽的女子，她就像一个女神，无时无刻散发着耀眼的光芒。我疯狂地迷恋她，但也仅仅是停留在幻想罢了，毕竟我们之间的地位天差地别——她是暴风城最受人尊崇的人之一：乌瑟尔·光明使者的独生女瑞秋·光明使者，而我则是闪金镇的一个穷小子。

“抱歉，[瑞秋](http://www.baidu.com/s?wd=%E7%91%9E%E7%A7%8B&ie=gbk&tn=SE_hldp00990_u6vqbx10)，我只是有些紧张...”我看着女神的脸，说不出话来。她是一个牧师，同样是圣光的信仰者，在我们这些圣骑士冲锋陷阵与冷酷的敌人肉搏时就是这些牧师给了我们勇气和力量。

“别多说了，进来吧。”[瑞秋](http://www.baidu.com/s?wd=%E7%91%9E%E7%A7%8B&ie=gbk&tn=SE_hldp00990_u6vqbx10)一把[拉住我的手](http://www.baidu.com/s?wd=%E6%8B%89%E4%BD%8F%E6%88%91%E7%9A%84%E6%89%8B&ie=gbk&tn=SE_hldp00990_u6vqbx10)，将我领往教堂中的一个施洗室，她的手柔软而细腻，还散发着牧师特有的温暖。施洗室的中央，一个苍老的人坐在书桌旁，正用鹅毛笔写些什么，那人就是我的老师[乌瑟尔](http://www.baidu.com/s?wd=%E4%B9%8C%E7%91%9F%E5%B0%94&ie=gbk&tn=SE_hldp00990_u6vqbx10)，虽然已经年迈，满头华发，但渊博的学识和强大的意志已然让我什么望尘莫及。

“啊，史蒂芬，你来啦。”老老师一抬头见到我，便挥挥手让我走他身边，“今天是你和瑞秋毕业的日子，老师我本该送你一件礼物的。但我一世清贫，也没有什么值钱的东西。”老师说着站了起来，走到了一个看起来已经尘封多年的箱子前。

“这把[灰烬使者](http://www.baidu.com/s?wd=%E7%81%B0%E7%83%AC%E4%BD%BF%E8%80%85&ie=gbk&tn=SE_hldp00990_u6vqbx10)，用黑铁筑成，举世无双，削铁如泥，曾跟着我久经沙场，饮过无数敌人的鲜血。如今我也年老眼花，再也抡不动这把剑了...”

“老师，莫非你要？！”

“我就将[灰烬使者](http://www.baidu.com/s?wd=%E7%81%B0%E7%83%AC%E4%BD%BF%E8%80%85&ie=gbk&tn=SE_hldp00990_u6vqbx10)赠与你，你一定要继承我的意志，用剑盾守护弱小,用圣光驱逐邪恶，誓死保卫暴风城...”

“[灰烬使者](http://www.baidu.com/s?wd=%E7%81%B0%E7%83%AC%E4%BD%BF%E8%80%85&ie=gbk&tn=SE_hldp00990_u6vqbx10)...不，这太贵重了，老师，我何德何能能接受这样的礼物。”

“收下吧，孩子，我相信你能驾驭的好。这世界现在并不太平，瑞秋是我唯一的女儿，[从今以后](http://www.baidu.com/s?wd=%E4%BB%8E%E4%BB%8A%E4%BB%A5%E5%90%8E&ie=gbk&tn=SE_hldp00990_u6vqbx10)保护她的重担就落到你肩上了。”

见导师这么说，我也不好再推辞，便单膝跪下，双手接过了这把剑。要知道[灰烬使者](http://www.baidu.com/s?wd=%E7%81%B0%E7%83%AC%E4%BD%BF%E8%80%85&ie=gbk&tn=SE_hldp00990_u6vqbx10)绝对不仅仅是把武器，更象征着一种无上的荣耀和责任。

从光明大教堂出来的时候天色已经不早了，我和瑞秋还要赶着去参加暴风城[战争学院](http://www.baidu.com/s?wd=%E6%88%98%E4%BA%89%E5%AD%A6%E9%99%A2&ie=gbk&tn=SE_hldp00990_u6vqbx10)一年一度的毕业典礼，那些当年同一届考进学院的小伙伴们从今天开始就完成学业，独挡一面，被送到联盟战场的各个角落。正如[乌瑟尔](http://www.baidu.com/s?wd=%E4%B9%8C%E7%91%9F%E5%B0%94&ie=gbk&tn=SE_hldp00990_u6vqbx10)老师所说，这世界并不太平，联盟和部落之间的战争全面爆发，燃烧军团的恶魔大军加快了对[中土大陆](http://www.baidu.com/s?wd=%E4%B8%AD%E5%9C%9F%E5%A4%A7%E9%99%86&ie=gbk&tn=SE_hldp00990_u6vqbx10)的侵略，北部地区的亡灵活动也一点儿也不消停，在如此严峻的形势下，联盟内部种族之间的矛盾也在加剧。

所幸教堂离举行毕业典礼的广场并不远，我们中途还有时间叫上约瑟夫和[吉安娜](http://www.baidu.com/s?wd=%E5%90%89%E5%AE%89%E5%A8%9C&ie=gbk&tn=SE_hldp00990_u6vqbx10)。约瑟夫是我从小玩到大的发小，一名战士，师从大名鼎鼎的战术大师[洛萨](http://www.baidu.com/s?wd=%E6%B4%9B%E8%90%A8&ie=gbk&tn=SE_hldp00990_u6vqbx10)，吉安娜则是他最近交上的女朋友——[大魔导师](http://www.baidu.com/s?wd=%E5%A4%A7%E9%AD%94%E5%AF%BC%E5%B8%88&ie=gbk&tn=SE_hldp00990_u6vqbx10)[安东尼达斯](http://www.baidu.com/s?wd=%E5%AE%89%E4%B8%9C%E5%B0%BC%E8%BE%BE%E6%96%AF&ie=gbk&tn=SE_hldp00990_u6vqbx10)的得意弟子，同时也是[海军上将](http://www.baidu.com/s?wd=%E6%B5%B7%E5%86%9B%E4%B8%8A%E5%B0%86&ie=gbk&tn=SE_hldp00990_u6vqbx10)[普罗德摩尔](http://www.baidu.com/s?wd=%E6%99%AE%E7%BD%97%E5%BE%B7%E6%91%A9%E5%B0%94&ie=gbk&tn=SE_hldp00990_u6vqbx10)的独生女。

从这一点上我和约瑟夫很像，我们都是出生平民阶层(约瑟夫甚至曾一度在西部荒野靠偷窃妓女的钱为生），却有幸结识了[吉安娜](http://www.baidu.com/s?wd=%E5%90%89%E5%AE%89%E5%A8%9C&ie=gbk&tn=SE_hldp00990_u6vqbx10)和瑞秋这样受过良好教育的富家女子。

毕业典礼算得上是暴风城的重大事件，许多联盟种族的领袖级人物都在场，当我们四人赶到的时候，发现暴风城的摄政王博瓦尔公爵、[暗夜精灵](http://www.baidu.com/s?wd=%E6%9A%97%E5%A4%9C%E7%B2%BE%E7%81%B5&ie=gbk&tn=SE_hldp00990_u6vqbx10)的领袖女祭司[泰兰德](http://www.baidu.com/s?wd=%E6%B3%B0%E5%85%B0%E5%BE%B7&ie=gbk&tn=SE_hldp00990_u6vqbx10)、矮人议会的铜须兄弟和[德莱尼](http://www.baidu.com/s?wd=%E5%BE%B7%E8%8E%B1%E5%B0%BC&ie=gbk&tn=SE_hldp00990_u6vqbx10)的英雄守备官玛尔拉德等等大名鼎鼎的人物都已经早早到场。

出于战事严峻程度的考虑，我们这一届[战争学院](http://www.baidu.com/s?wd=%E6%88%98%E4%BA%89%E5%AD%A6%E9%99%A2&ie=gbk&tn=SE_hldp00990_u6vqbx10)所招收的人数超过往届所有，包括人类、精灵、矮人、侏儒、[德莱尼](http://www.baidu.com/s?wd=%E5%BE%B7%E8%8E%B1%E5%B0%BC&ie=gbk&tn=SE_hldp00990_u6vqbx10)和[狼人](http://www.baidu.com/s?wd=%E7%8B%BC%E4%BA%BA&ie=gbk&tn=SE_hldp00990_u6vqbx10)六大种族和[圣骑士](http://www.baidu.com/s?wd=%E5%9C%A3%E9%AA%91%E5%A3%AB&ie=gbk&tn=SE_hldp00990_u6vqbx10)、战士、猎人、盗贼、[德鲁伊](http://www.baidu.com/s?wd=%E5%BE%B7%E9%B2%81%E4%BC%8A&ie=gbk&tn=SE_hldp00990_u6vqbx10)、法师、牧师和术士八种专精共计一千余人。

在[中土世界](http://www.baidu.com/s?wd=%E4%B8%AD%E5%9C%9F%E4%B8%96%E7%95%8C&ie=gbk&tn=SE_hldp00990_u6vqbx10)，一个人对于力量的掌握主要体现在期对于自身体型的掌控上，所有专精，所有种族，不论男女都是如此。但是女性出于未知的原因，身体对力量的适应性和掌控性都远远大于同等级的男性，也就是说同等法力的男性和女性，如果都要将身体变到最大，女性的体型要比男性大上不止一个数量级。

广场已经人山人海，我们找了个位置坐下，刚好赶上伯瓦尔公爵的开篇演讲。

“同学们，大家早上好。很高兴，今天你们通过了层层磨练，从而坐在这里，即将为了自己和联盟的荣誉而战！” 伯瓦尔仅仅说了一句话就赢得了大家强烈的欢呼声，毕竟暴风城的摄政王可不是天天都能见到的，“我们今天有幸邀请到了一个嘉宾，我相信她的强大能力能够给你们带来巨大的鼓舞，她便是泰坦创世时为中土留下的五条巨龙之一，红龙女王： 阿莱克斯塔萨。”

“我从书上读到过。”瑞秋小声对我们说，“据说五条守护巨龙分为五种颜色，分别是黑龙大哥耐萨里奥，二姐红龙阿莱克丝塔萨，三姐蓝龙玛蒂苟萨，四弟青铜龙诺兹多姆和五妹绿龙伊瑟拉。他们个个身躯浩瀚无比，拥有无穷无尽的力量。”

一条小红龙缓缓从天而降，然后从龙型幻化成一个血精灵的形态，引得在场的人一阵惊呼。

“有部落！公爵，快抓住她！”台下有人大喊。

“不得无礼！”公爵喊道，“大家放心，不要惊慌，这个人并不是部落的血精灵，她是红龙女王阿莱克丝塔萨的女儿阿蕾莎。”

“大领主，我们的母亲已经驾到。”血精灵对伯瓦尔说。

“那么，有情红龙女王！”

天空突然间变得黑暗，狂风阵阵袭来，远方传来巨大的轰隆声，广场的人们一阵惊恐。

“大家别怕，这是因为我母亲的翅膀遮住了天空而已，这飓风只是轻微拍打翅膀产生的小气旋罢了。”看着慌乱的人群，阿蕾莎安慰道，可这安慰却起了反作用，毕竟人们从来没有见过这么巨大的生物。

“英雄们，你们好。”我们头顶上方的天空传来遥远而深邃的话语声，伴随着轻微的地震声，暴风城两侧远方的天空都出现了一根巨大无比顶天立地的黑色柱子，每根柱子的直径堪比整个暴风城，柱子上端深入云霄，地上的人根本无法一窥究竟，也许在远处的人看来，暴风城就像一个玩具一样被夹在两根巨柱之间，只是那柱子的形状，看上去像……一条腿？圣光在上？

“很抱歉，英雄们。”天空上方再次传来空灵的话语声，“为了不破坏你们的城市，我已经尽力缩小我的体型了，但原本的身躯实在太过巨大，我几乎耗费了所有精力才保持着这么“袖珍”的体型，希望不要吓到你们。”

我这下才明白包围暴风的两条黑色天柱只是阿莱克斯塔萨穿着黑丝的两条大腿，整个城市就位于她的胯下，在红龙女王的脚下如同玩具一样，甚至只要她轻微挪一挪脚，一个脚趾就可以将暴风城全灭。

瑞秋显得有些害怕，她搂着我的肩膀瑟瑟发抖，我从手中慢慢释放出一缕金黄色的圣光，这让我们得以在红龙女王巨大身躯蔽日的阴影下看清一些周围的动静，身边的学员们无一不惊慌失措，毕竟我们中的大多数这辈子都还没见过如此巨大的存在，我不由得想到，如果她费了这么大劲还只能把身体缩小成这么大，那她原本的身躯到底是多么恐怖的程度。

“阿莱克斯塔萨，生命的缚誓者，暴风城恭迎您的降临。”伯瓦尔公爵单膝跪下超红龙女王致敬，其他人见状也纷纷向着巨龙叩首跪拜。

“不必拘礼，暴风城的子民们。我来知道你们中的有些人不久的将来就要上战场，我来这儿就是为了赐予你们我的祝福。”透过云层，我是不可能知道云层上端阿莱克斯塔萨在做什么的，可我隐约间感觉她像是对着我们吹了一口气，因为一股清新无比沁人心脾的微风从我们头顶拂来，顿时让我感觉精神百倍，浑身充满了力量。

“好了，英雄们，我此行的目的已经完成。如果你们有幸——或者不幸被派到魔多，我会在那里等着你们。”

阿莱克斯塔萨口中的魔多就是我要被联盟军派去的地方——事实上是我自己的申请的，毕竟作为乌瑟尔·光明使者的学生，我的人生最大的使命并不是对抗部落，而是驱逐邪恶。魔多作为燃烧军团在中土世界的主基地无疑是我实现价值的最好场所，那里终日被乌云和浓烟笼罩，原本清澈的河流被酸液和硫磺污染，火山中滚烫的岩浆似乎会随时爆发，天空被火山映照成红色，大地干裂，没有任何生命能在这样的情况下存活下来。

“那么，再见，英雄们。”伴随着剧烈的地震，矗立在暴风城周围的两根通天大柱开始移动，在一阵如烈日般耀眼的光芒过后，天柱消失，阿莱克斯塔萨化为龙形，停留在不知道多少公里高的天空，那无论用多少词语都无法形容的浩瀚身躯遮蔽了太阳，整个艾尔文森林都陷入了阴影中。巨龙拍打着巨翼，每一次挥动翅膀产生的气流对于底下的我们来说都是一场风暴。

“好了，大领主，我想我也该走了。”阿蕾莎跟伯瓦尔公爵握了握手，随后又变成了龙形态，有这么一瞬间，我觉得她似乎就在盯着我，对着我笑，“我会在魔多等你！”她的眼神里传达的信息完全就是这个意思。

“噢！”我只觉得我的手臂上钻心的疼，瑞秋又在我身上试验她的“掐指神功”了，她一定又因为我在看别的美女而心生醋意。

“人家是龙族，就是比我漂亮，看的你两眼睛都掉出来了。”瑞秋阴阳怪气地对我说，背对着我翘起了二郎腿，圣光的白袍顺势滑下，露出了里面的黑丝大腿。

见到瑞秋的光洁无比的黑丝腿，我哪里把持得住，那柔美的曲线简直是致命的诱惑，让我情不自禁的就把手搭上去轻轻地抚摸。

“大小姐，别生气了。”我来回抚摸瑞秋的大腿，那黑丝脚正套在中筒布靴里，一晃一晃的，好想让我一探究竟，“在我心中你永远是世界上最美的。”

“哼，谁知道你们男人哪句话是真的。”瑞秋嘴角露出一丝难以察觉的微笑，但还是故作镇定，我就喜欢她这样。

“大不了下次见到那个什么龙女，我就当着面骂她丑总好了吧。”
