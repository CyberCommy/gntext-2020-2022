# 1

A

无垠的宇宙中独一无二的蓝色星球——地球。 但是前面还有那个——一个巨大的屁股。 这是是十亿倍少女的屁股。 屁股的两个肉瓣的体积是地球的几百倍，其表面积也比地球大得多。 地球上的68亿人口可以靠她的半个屁股生活。 对她来说，地球是一个1.3厘米大小的小球，是比弹珠还小的弹珠。 她的屁股越来越接近地球，无声靠近的屁股给地球投下巨大的阴影，世界上大部分都笼罩在屁股的黑夜下，已经快到了碰上的可怕距离了。 月亮早就碰到她的屁股上粉碎了。 不久，屁股的动作停止了。 地球放置在了那条裂缝前。68亿人观看着，夜空中没有星星，只有皮肤的颜色。 就在此时。 原本应该是真空的宇宙空间里响起了轻快的声音。 10亿倍的少女放屁了。 地球正处于她的屁眼前，所以放出的屁直接击中了地球。 在这种气势下，地球都被冲击得后退了一些。 而且，曾经覆盖地球表面的大气层全部被吹散。 直击面的海面也因为强大的风压，一瞬间露出了海底。 海洋从地球的半球消失了。 原本是海洋的东西，现在已经变成了海啸，要么向另外一个半球袭来，要么散落在太空中。 臀部面对的半球，一切都从地表消失了。 山被摧毁了，森林被夷为平地，国家、城市和人类被夷为平地，瞬间变成了荒野。 吹走大气的屁被地球的引力而所吸引，覆盖在地球表面。 她的屁变成了地球的大气层，如果地球有人还活着的话，他一定会因为逃到世界任何地方都逃不掉的恶臭而流下眼泪。 放屁的女孩在“嗯”了一声之后离开了银河太阳系，只剩下赤褐色、散发着恶臭的荒废的地球。

B

10亿倍的小女孩看来地球只是一个1.3厘米大小的球，所以这个女孩对地球施了魔法。 这样地球就不会因为做一些一点点小事就坏了。 女孩用手指拿着那个地球带到眼前。 小蓝星闪闪发光，它的美丽吸引了她的眼球，就只有指尖那么大的小星星。 虽然因为魔法不会轻易毁了它，但也会因为它过于渺小而变得谨慎。 地球人惊恐地看着覆盖世界大半部分的巨大指腹，许多大陆和国家被压在下面，不过没有一个人受伤。 由于少女的魔法，连他们也变得强健无敌了。 少女以陶醉的表情注视着星星，过了一会儿，她把星星贴在自己的乳头上。她觉得自己还是比较丰满的，虽然和手指间的星星比起来没有什么意义，但看到自己的胸部比一颗星星还大，还是挺自豪的，地球只比乳头大一点点。 女孩开始用乳头摩擦地球。 虽然很小很可爱，但星星所具有的特殊粗糙感却是非常甜美的。 好刺激，好刺激呀，她为了快感而摩擦着地球。 尽管如此，地球并没有受到任何伤害。 但是，无论多么坚固，物质的性质都不会改变。 大海碰到了她的手指和乳头，一片狼藉。 再加上她不停地摩擦，大部分的海都粘在乳头表面被吸引住了。 海洋从地球上消失了，地球人也是如此。 每当地球在她的乳头上摩擦时，数以亿计的地球人就会被挤压在乳头上，被抛在后面，他们不能回到以正在以超高速移动的地球。 从现在开始，他们必须生活在这巨大的乳头上。 结束了对地球的爱抚的少女，这次把地球带到了下半身。 她用另一只手把狭缝撑开，这是女性的生殖器。 女孩把地球推进了那里。 往里走，往里走，被推进了几个地球的距离。 然后，手指就这样离开了地球。 地球被遗弃在黑暗的生物洞穴中。 紧张的脉搏就是生命的本质。 地球被少女的阴道包裹着。 也许是为了感受更好，阴道壁紧紧贴着地球，抚摸着地球表面，地表的一切，都被她的嫩肉舔过。 所有的海洋都被吸干了，取而代之的是足以淹没整个大陆的爱液。 地球人也和大海一起被内壁缠绕，四散而去。 数十亿的地球人都被困在一个女孩的阴道里。 舒舒张张的脉搏，由爱液组成的海洋，这就是他们今后生活的世界。 少女继续用生殖器吮吸着无数的地球人和地球。 过了一会儿，少女离开了太阳系，将地球留在她身体中。地球就这样从太阳系中消失了。
