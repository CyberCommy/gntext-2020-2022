# 本版将开展精品回帖活动

作者：铜锣门卫

TID：12792

 

# 1

本版为鼓励回帖，减少回帖水帖的数量，决定开展精品回帖活动，凡是回帖被认定为精品回帖将有不同程度的奖励，本次活动持续时间为1个月，望大家积极参与，期待更多的发出精品的回复。

感谢咯咯的补充，看到回帖关于什么是精品的问题，我和咯咯讨论了一下决定暂将标准定为一下

1回帖字数不少于50毕竟字数是一个很重要的标准来评判嘛,当然短小精悍的我们也欢迎的说，但是光有数量而没有质量也是不行的呦

2对文章有明确的见解或建议(废话嘛![](static/image/smiley/bigface/57.gif) 要不然怎么叫精品)

3回复中可以出现对作者不满的语言，但是不许出现不良信息如骂街对作者进行人身攻击等。如发生按引战处理![](static/image/smiley/bigface/43.gif)

4以上规则为暂定施行有如有疑问请点击论坛短讯功能发送给我们或在回帖中提出我们会进行认真的讨论滴
