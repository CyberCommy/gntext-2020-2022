# 1

我和妻子分开了四个月 在这四个月里由于我工作的地方没有女人 只有用自慰来解决生理需要的问题 但是自慰却需要去幻想让自己兴奋的事情 于是我的不幸开始了 因为我在一次自慰时突然幻想亲吻妻子的小穴 喝妻子的爱液 和她的圣水（尿）越想越兴奋 于是我每次自慰都这么幻想 当我四个月回家后 我迫不及待的把妻子内裤扒掉 但是妻子没有迎来因我大手对他小穴的抚摸 却迎来了我热烈的嘴唇对她小穴的舔吻 这种感觉也许让我妻子丽儿感到新鲜 所以她表现的很兴奋 小穴也流了很多爱液 可是我没想到 我喝她几口爱液后 我自己就射了 这使丽儿很是埋怨  之后一周更是让丽儿快要疯狂 因为我只要亲吻丽儿的小穴我就会射出来 可是不亲吻她的小穴 我又硬不起来 最后丽儿和我去了医院就诊 医生却查不出病因  。就这样我和丽儿都很痛苦的生活着 甚至丽儿都不让我去触碰她 于是我买了一个假阳具给她使用 她才开始让我触碰她的身体 但是她却开始对我虐待 比如她下班后第一件事 是让我喝她的圣水 她的脚都是我用舌头来清洗 她小便时我要为她吸允脚趾 甚至是舔她的小穴和菊门 做爱时我要先把她舔到高潮 之后用嘴把假阳具弄的润滑后咬着假阳具去插她的小穴 事后还要把她的小穴舔干净

一年后的一天丽儿突然问我 是否崇拜她的小穴 我当然崇拜啦 于是拼命的点头  丽儿又说她想让我变小进入她的内裤里 侍奉她的小穴  我当时说 要是你真能把我变小 我愿意永远在你的小穴里  待续
