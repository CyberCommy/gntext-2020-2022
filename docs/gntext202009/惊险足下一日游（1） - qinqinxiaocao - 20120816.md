# 1

玉婷20岁了，她1米7的个头，在同龄女孩中也算不低。双眼皮，瓜子脸，标准的美人身材。这几天，她说要创作一个剧本，经常往家里领一些陌生人。我想应该都是她同事吧- 我是他的弟弟，平时在家帮忙收拾东西，招呼客人。爸妈去上海要一个多月才回来，家里只有我们两个了，这更适合姐姐的创作

一天早晨，姐姐找了一个陌生人来我们家，说是要一起商讨事情。趁姐姐上厕所的功夫，那个陌生男人拉住我的手说：“你都这么大了，还在你姐眼皮底下办事，该自己做主了！”边说边把一罐可乐递给我，然后匆匆离去了。

姐姐出来问我：“他人呢？”我说不知道。然后我让她喝可乐，姐姐不喝。于是我就把它一饮而尽了。之后，姐姐去上班，我在家玩电脑。一直到晚上，姐姐回来了，她让我洗洗睡了，然后她坐在电脑旁查看资料......

我在床上睡着睡着，忽然觉得身体一阵酸痛，想起身却没有力气。我失去知觉了，恍惚中看到了陌生男人诡异的笑容，之后就在也记不清了。

不知过了多久，我在一片喧闹中醒过来，眼前的一切都变了，周围触手可及的东西离我那么遥远。我这才意识到是我变小了，我大喊大叫，可是周围的声音比我的叫声大多了。我抬起头，发现我正在一双巨大帆布鞋和高跟鞋中央。这时，我听到一个熟悉的声音，并且看到了特别巨大的身影：“今天穿什么鞋好呢，高跟鞋? 板鞋? 球鞋还是帆布鞋?”啊 原来是姐姐，我这才看到平时就个子高的姐姐，一眼望不到头！只能看到眼前两只巨大玉足在鞋边徘徊。这时，玉脚开始移动，我趁机抓住姐姐的脚趾甲跟着进到了鞋里.....

好黑！并且姐姐用脚趾甲顶住我的胸口，我想动也没办法动。我这才记起昨天那个陌生男人和那个可乐！真想抽死他！  可是现在我却动弹不得，黑洞洞的鞋里散发出女孩独有的脚的味道。我使劲嗅着，没有比这更爽的了！  我意识到，如果想变成原来的样子，必须要让姐姐注意到我。我立刻掏出随身带的钥匙，用力扎姐姐的脚趾头。姐姐好像有点反应，玉脚微微扭动了几下。凭我的直觉，姐姐穿的是帆布鞋，并且是36码的。平时我就关注姐姐的脚，没想到今天身临其境了，在粉色帆布鞋里我却感受到另一种气息。姐姐的脚开始扭动，本来鞋就不大，我在这个黑暗狭小的空间里更加拥挤。我的胸口发痛，一股前所未有的压力扑面而来。黑暗中，我什么也看不见。只能感觉到我身上越来越挤，我拼命捶打姐姐打脚趾头，可是姐姐毫无反应。是啊，我几乎变小了100倍。平时23厘米的小脚，现在变成了23米。脚趾头比我还大出好几倍。怎么来形容呢、 我就像一只蚂蚁。我就在这种折磨中度过了好久，不知什么时候，姐姐的巨大玉足离开了帆布鞋子，我开心极了。抓住脚上的皱纹跟着被带了出来。我站在姐姐足下，欣赏她23米的标准美人脚。

不知是哪位圣贤说过:"女人眼尖。"就在我陶醉于其中的时候，姐姐大叫一声：“虫子！”我听到上方传来一阵旋风，我本能地往旁边一闪。姐姐光着脚在踩我！把我当成虫子了！“姐姐，是我！”我大叫，可是姐姐的尖叫埋没了我的声音，随即而来的是身后玉足的践踏！我左闪一下，又闪一下。拼命地跑。可是仅仅只有1厘米的可怜的我，又能跑出多远？我被姐姐的脚压在地板上。我随即吐了一口血。现在的我，别说是一只脚的重量，只要姐姐把一只脚趾头轻轻放在我身上，我也跑不掉了！我敲打着姐姐脚心。姐姐大叫：“死虫子，我要把你踩成烂泥！”说着，我感觉头上有巨大响声。我摸摸身子，啊，我没死！原来是姐姐的脚链救了我一命。我拼命跑，跑到死角上，姐姐说道：“哼！让你跑。这次放过你，下次别让我看见你！”我喘着气，看着眼前极大地粉色帆布鞋离去。我叹了一口气：“啊，好可怕！还好躲过一劫”
