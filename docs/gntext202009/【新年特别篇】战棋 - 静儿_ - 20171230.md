# 1

寒风凛冽的北方前线上，一个人影也看不到。
北方联盟的士兵们躲在高耸的城墙后，即使这样也可以感受到外面呼啸的北风,几个人聚在一起喝着烈酒，抱怨着这样的生活。
北方联盟和神国开战已经七八个月了，这是自从三百年前以来，北方联盟第一次重新踏足这片曾经属于他们的土地。虽然进展十分缓慢，但是所有人都看到了希望，也许数年，也许数十年后，北方联盟可以重新占有神宫，再次回到霸主的位置。既然北方联盟在八个月内就能推进两百公里，那么将这篇数万公里长宽的大陆彻底攻下也不过是时间问题。
开战以来，许多北方联盟人对神国与生俱来的恐惧也逐渐消散了。神国的军队并没有传闻中一样攻无不克战无不胜，实际上，神国一直在北方前线采取坚守的战术，她们并没有利用自己身高马大的优势主动进攻。
“哎，可惜现在的天气不适合进攻，否则我们可能在更暖和的地方过冬了。”一个士兵这样抱怨着，其他人也纷纷附和，不得不说现在北方联盟内部的情绪非常乐观，而形势也的确对他们有利。
正当士兵们开始准备换班的时候，一阵刺耳的哨声响了起来，刚才懒散的士兵们一下子进入了紧张的状态，拿起兵器跑向城墙。
此时，城墙外面已经聚集了一支不知从何而来的军队，数千名五米高的神国女战士列了进攻的阵型，一名足有二十米高的女武神冷笑着，“是时候对这些小虫子发动反攻了。女神陛下的奴仆们，让我们一起踩烂他们！”
女战士们爆发出一阵欢呼声，然后紧跟着女武神发动了对城墙的冲击。
这座城池曾经是神国的领地，因此大多数设施也是按照神国女战士的身材打造的，北方联盟的士兵们在三十多米高的城墙上看着这支军队冲锋，几千名体重超过一吨的女战士一齐跑动，场面颇为壮观。那种地动山摇的感觉让许多士兵不可控制地颤抖，他们又回想起被神国统治的恐惧。
“别怕！放箭！”一名军官压住心中的恐惧，大声鼓舞着部下的士气，“这是我们的土地！我们不会让这些女强盗们把它夺走！为了我们的家人，勇敢地作战！”
士兵们开始渐渐进入位置，对这支冲锋的部队射出了箭矢。
对于身高五米的神国女战士来说，北方联盟射出的箭矢虽然不能无视，但是伤害也的确有限，往往要十数支箭射中要害才能对她们造成伤害，在近一千米的冲刺距离中，只有几十名运气不好的女战士丧失了战斗力。
“轰。”随着最前方的女战士冲到城下，她们用穿着厚重护甲的身躯撞击城墙，整座城池仿佛都为之一振。几名探头往下看的士兵一时没抓稳，从城头摔了下去。一名士兵摔在了一个女战士的身上，劫后余生的喜悦还没过去，女战士就一把抓住他的腰。健壮的士兵在这个女战士手中就像个大号娃娃，被她一把捏碎了许多根骨头，紧接着被扔到一边，另一名女战士的战靴踩了下去，这名士兵便彻底停止了挣扎。
经过最初的恐慌之后，士兵们发现，这座城池还是有一定抵抗能力的。女战士们的撞击除了让城墙震动一样，短时间内似乎不会破城。城门早已被军官用铁水彻底封死了，而援军也已经在路上，只要能够拖延住，这次防御战必然可以胜利。
只不过，事情并没有那么顺利。随着一阵更加强烈的震动，神国军队中的女武神开始大步冲刺。比起女战士们的冲刺，这一名女武神似乎更加恐怖。有的女战士躲避不及，即使强悍如女战士，被女武神一脚踩中也只会瞬间死亡。
当女武神撞到城墙的一瞬间，整座城池似乎都被移动了一般，此前坚固无比的城墙被撞开了一个六米宽的缺口，几名在城墙上的士兵随着坍塌的城墙摔了下去，被女武神那三米长的战靴踩烂。
女战士们欢呼着，开始从这个缺口涌入城内，城墙上的士兵们陷入了慌乱中。女战士进入城内后可以从城墙后的阶梯爬上墙头，北方联盟的士兵们开始逃窜，可是女战士们只要大步行走就可以追上拼命逃跑的士兵们，越来越多的士兵们被女战士追上，然后女战士随意的一拳一脚就可以把他们杀死。
“不要逃窜！抵抗！”刚才鼓舞士气的军官大喊着，他举起手中的剑，大踏步跑到一名女战士面前，和两个副官一起砍向这名女战士的大腿，女战士被这三个勇敢的军官逼退了一步，更多的士兵被这一幕激励，开始回头集体攻击女战士。城内北方联盟的士兵足有三万人，而进攻的女战士不过数千，采取了以多围少的战术之后，的确暂时挡住了女战士们的攻势。
不过，很多人忘记了那个恐怖的女武神，她发现了形势的转变，愤怒地锁定了军官的位置，气势汹汹地朝他冲了过去，几十名士兵试图围住她，却被她几脚全部踩死了，见识到女武神恐怖的实力，越来越多士兵开始退缩。
女武神冲垮了又一道防线之后，离军官已经不足百米，这点距离对于二十米高的女武神来说一秒就可以踏过，她发出愤怒的怒吼声，朝着军官发起冲锋，绝望的军官已经闭上了眼睛，准备迎接他的命运。
“轰隆”一声巨响，让军官疑惑地睁开了眼睛，此时女武神已经不见了。取而代之的是一只巨大无朋的黑色雪地靴。这只雪地靴占据了军官全部的视线，即使站在三十米高的城头，他也看不见这只巨大雪地靴的鞋面。费力地抬起头，他的视线随着雪地靴筒向上，看到了一条高耸入云的黑色柱子。军官激动地跪了下来，“公主殿下！”
北方联盟的公主此时穿着一套厚实的冬装，脚踏着两只雪地靴，正不断重复抬脚-踩踏的动作，在两百多米长的雪地靴下，即使女武神也和虫子无异，毫无抵抗能力。更小的女战士们则四散逃窜着，但是在公主看来，她们的速度慢得可怜，几千名女战士几分钟便被消灭殆尽。
公主的脸上露出了嘲讽的笑容，“你们辛苦了，能够坚持这么久也不容易，还好你没有逃跑，不然我可能会浪费更多时间。”
听到公主这样说，军官心中的恐惧更甚，他很清楚，那些之前偷偷逃掉的士兵恐怕已经被迎面而来的公主殿下踩死了。毕竟对于比士兵巨大一千倍的公主殿下来说，他们说是虫子都有些大言不惭，只能面对着公主殿下膜拜她的巨大吧。
“是……公主殿下，感谢您的帮助。”军官诚惶诚恐地说道，他生怕这名拥有绝对力量的公主有任何不满。在整个北方联盟中，虽然联盟领导人是公主的父亲，但是公主的权威大的多，实际上北方联盟能取得现在的战绩也和公主殿下的实力分不开，无论女战士，女武神甚至更巨大的，一百米高的神女，在公主殿下脚下都没有任何抵抗力。只有神国那名从未见过的女神才可能与之抗衡吧。
“呵呵……你终于肯出现了。”如果是刚才公主踩死女武神的一脚是惊雷，那么这道声音仿佛天崩地裂一般。只是声音而已，就震的城墙倒塌，无数士兵捂着耳朵满地打滚，就连公主也失去了她惯有的冷漠表情，惊恐地抬头看向天空。
一张绝美的容颜正俯视着公主，这位存在的巨大已经难以理解，公主在其眼中不过指甲大小，而这座城池也不过和她的手掌大小相当。此时趴在大地上，用轻蔑笑容看着公主的，正是这篇大陆上至高无上的存在，神国的女神。
军官此时的心情只能用由天堂到地狱形容。女神，这个让所有人敬畏的存在就在自己的眼前，而公主殿下，让自己无限恐惧的存在此时仿佛一只小虫一样在女神的淫威下瑟瑟发抖。
以至于，所有人都没注意到，整片大陆的天空都变成了粉红色，一股奇特的气味充斥于空气中。
女神疑惑地抬起头，看着粉红色的天空，露出了同样惊恐的表情。
粉红色的天空带着不可抗拒的威压缓缓下降，女神疯狂地巨大化着，抬起手试图撑住下降的粉色，只不过和遮蔽数万公里长宽的粉红色天空相比，女神再怎么巨大化也宛如小虫子，不，微生物一般可笑。面对拥有绝对主宰力的粉红色天空，女神只能像其他的一切一样被瞬间压制。无数个城池，山脉，都在碰触到粉红色一瞬间崩溃。直到公主、军官与这片四面八方延伸到无限远的粉红色亲密接触，让他惊异的是，这片粉红色意外地柔软。
仿佛一个女孩的足底一般。

“喂！好歹让我的小家伙发表胜利感言吧！”我一把抓住佳儿，但是她已经一脚把沙盘踩平了。
“才不要！怎么又是你赢了！这个游戏一点也不好玩！”佳儿生气地用脚丫碾着已经破碎不堪的沙盘，“明明是我占据优势的！你把那个微生物藏在什么地方了！怎么突然间出现了！”
“喂……你不会是在对玩具炫耀你的优越感吧……”我无奈地摇摇头，“是你太差了好吧……我摆了十几分钟的喂！就这样被你弄烂啦！”
佳儿这才放过沙盘，不过沙盘里只剩一个佳儿脚印的形状了，“反正这个游戏不好玩。下次换一个吧！”
“好啦好啦……”我把沙盘收回盒子里去，然后翻了翻装着游戏的抽屉，“你看这个……宇宙帝国的毁灭怎么样？是上次有个女神献上来的……据说是她把真的宇宙搬进去制作的，我还没开箱呢！”
“要我赢才玩！”
“哪有那种事啦……”
两个女孩打闹着，当然，没人注意被卡在佳儿脚趾纹中的小小女神，也许洗脚的时候，她就会被冲走了吧。
