# 1

我是一个无业游民，一天晚上我走到了一个荒无人烟的地方，忽然一阵阴风我晕了过去，等我醒来以后发现自己在一个不知名的空间里。

新的玩具啊，一个声音突然响起，我转过头去，一个穿着哥特装脚下黑白不同的长筒袜没穿鞋背后还有一双蝙蝠翅膀的巨大的萝莉正笑眯眯地看着我，不多说了现在就开始游戏吧，说完萝莉，一脚踩了下来，我还没缓过神，直接被她踩在脚下，唔，看来这个玩具不如之前的那么好玩呢，那就赶快解决吧？说完，我感到一阵巨大的压力袭来，萝莉脚上的黑丝袜，不知道多久没洗了，浓烈的气味，让我喘不过气，眼看我就要被熏死了，萝莉不知道又想的了什么，坏笑了一下，把脚挪开，然后把已经熏的晕乎乎的我的放到了一只黑丝袜脚上，然后另一只白丝袜脚也踩了上来，再见了小虫子，萝莉开始慢慢用力，两只脚之间的间隙渐渐变小，最后两只脚合在了一起，已经完全看不到中间还夹了一个人，最后萝莉的双脚开始不停地搓动，过了一段时间，才慢慢停下，把双脚分开后只能看到一滩血迹，至于身体已经被搓进了袜子里，成为了里面的污渍。

嗯，已经完全不存在了呢。小萝莉笑着说，拿出了一双黑色公主鞋，又从一个盒子里拿出了两个小人把它们放到鞋子里，鞋子里非常的热，两个小人不停的求饶，突然一个小人从盒子里跑了出来跑了出来，没跑多远，一只巨大的白丝袜脚，狠狠地踩在了他的身上，小人直接被踩成了肉泥黏在地板上，血液飞溅到萝莉的白丝袜上添加了一丝妖艳，还想跑，萝莉恶狠狠地说，脚不停地碾动，直到那滩肉泥完全跟地板齐平时她才罢休，随后看着鞋子里的小人，露出残忍的微笑直接穿了上去，在脚落地的一那一刻，小人就立马被踩成肉饼了，小萝莉不断地在房间里子走动一遍遍的踩着脚下的尸体，直到跟她的鞋垫溶为一体她才停下。

看着盒子里的小人，对他们说，我会放你们走，但是你们得陪我玩最后一个游戏，我会在你们身上加一个隐身术，任何人都看不到你们，你们要是能逃离传送到的区域的大门，那你们就可以复原。说完就直接把小人们传送到了一个不知名的地方，突然一个巨大的白丝袜脚踩到了一个小人身上，直接爆他踩得，仔细一看，这是一个教芭蕾舞的地方，小人们在中心地段，周围都是女生，而且不知道在做什么训练，一个个都没有穿芭蕾舞鞋，因为隐身术的缘故，她们都没看到直接踩了过来，小人们有的被踩的血肉模糊，黏在白丝脚上，有的下半身被踩扁生不如死，甚至有的被踩到黏在脚上还没立刻死，而是慢慢的在女生们的舞蹈中被踩扁内脏直接被踩出来，然后其他女生踩到粘在脚下，现在的场景就跟地狱一样，只不过除了小人以外，没人能看到，终于只剩下一个小人，并且快跑到大门口了，小人狂喜，终于可以脱离这地狱了，就在他快跑出去的时候，一只巨大的黑丝脚降临在他的头上，不！！！这是小人留在这世界上最后的声音，咚！！！！！黑丝袜脚直接把他踩成肉泥，这双脚的主人一个相貌迷人的少女，对着芭蕾舞蹈室喊了一句，妹妹，该回家了。我在外面等你。说完就到了位面穿上鞋子，至于最后那个被踩成肉泥的小人，也随着巨大的黑丝袜脚，被踩入鞋内，在少女走动的时候，全身压扁，黏成泥，又压扁，又碾成泥一直反复着，不远处，一个小萝莉微笑着说了一句，游戏结束.
