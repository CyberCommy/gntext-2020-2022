《母なる大地》

这是发生在不远的未来的故事
20××年○月△日，某种存在突然来访了……
那一天，世界一瞬间被强光所包围……
我所知道的只有这个了。
因为在那一瞬间，我突然被扔到了一片什么都没有的非常广阔的平原上。
我不知道为什么会变成这样。
我：“这里是哪里！？”
因为太突然了，所以会怀疑是不是在做梦。总之先拧一下脸颊。
我：“果然很痛啊。”
好吧，看来我没有在做梦。
我的名字叫武，23岁，单身，职业是上班族，出身于九州的乡下。
好的记忆也没错，那看来我的脑袋也应该没什么问题。
身体和头脑都没有什么特别的异常。那说回来，这里是哪里呢？这里还存在着最大的一个疑问没有得到解答。
就算这样一直呆着，也是没办法找到眉目的，所以我开始毫无目标的在这旷达的荒野中前行着然后我走了起来，总之是走了一段路程。附近没有看到城市，这个荒野别说是人，连植物都没见到。但是，就在某一刻，我在远处看到了一个女孩子的身影。
我：“喂！”
我大声呼喊着，而那女孩也注意到了我，并向这边走来。
但是随着女孩逐渐接近，我感觉到的违和感越来越强。
我：“喂！我在这边啊~！……卧槽！？这是骗人的吧！？”
ズシィィィィィン！！！
巨大的脚踩在我的旁边，差一点我就要被踩扁了。
令人吃惊的是，那个女孩子的体型是我的100倍。
女孩子：“那个~！？好奇怪啊~我确实是在这一带听到了声音的……”
我：“喂！我在这里啊！拜托请细心注意下~！”
在这个时候不管你是巨人什么的都可以，反正我是不想继续独自在这平原上徘徊了，因此我大声地呼叫着那女孩。
嘛，因为我是抬头向上叫的，所以看到内裤什么的都是不可抗力。
女孩：“咦？声音是从下面穿来的！？哇！！虫子！！……咦？不是虫子吗！？”
女孩把我捡起来，并放到手掌中。
对于被放到手掌中的我很感兴趣，所以女孩目不转睛地盯着我，我的视野被这女孩巨大的脸庞填满，能看到女孩的眼中也带有惊讶。
即使是那样巨大的体型，女孩看起来也非常的可爱啊，如果有这么可爱的女孩子陪在身边的话，真的很想让她做女朋友啊，等等……说起陪在在身边……
我突然觉得好像在哪里见过这个女孩子。
我：“如果是我认错的话抱歉……你难不成是‘见崎’吗？”
见崎……那是我的青梅竹马的名字，虽然已经好几年没见过了，但总觉得和眼前的女孩有点像，所以这个巨人不会真的见崎吧？
见崎：“为什么你会知道我的名字？……难道你是武君！？”
我：“真的吗！！真的是你吗？这么久不见你变得非常可爱了啊！而且，你还长得那么大个了！！这已经不是时间长可以做到的了，你这不是变成巨人了吗！到底是吃了什么变得这么奇怪的啊！！”
见崎：“没有啦变得可爱什么的……不对，我才不是巨人呢！！对着女孩子说她是巨人什么的，太失礼了！话说不是武君你变小了才对吗！？
我：“为什么会是我变小啊！我怎么一点头绪都没有！”
见崎：“我也不记得我有变成巨人啊！”
那么到底是我变小了，还是见崎变大了呢？即使看向周围，也没有什么可以比较的东西，所以根本不能证明谁才是对的。
我：“嘛，不管是我变小，还是见崎变大都可以啦，我想问下你知道这里是什么地方吗？”
见崎：“emmmm，当我回过神来就发现我站在这里了，所以我也是什么都不知道啊！所以，这荒野到底是延伸到哪里呢……”
我：“总之现在只能向前走了啊！那个，你能和我一起走吗？”
见崎：“好啊，是武君的话大欢迎！！但是这个衣服没有口袋呢……对了！武君可以进来这里！！
我被见崎拿起，放到了她的巨乳间。
我：“那个！把我放在这里真的……好吗？你知道这里是哪个位置吧？”
见崎：“这里可是为了武君准备的特等席哦！不要客气，尽情享受吧！”
我说：“嘛，你这样说的话，那我也就不担心什么了！”
就这样，我进入了巨乳谷间的特等席，和见崎一起在这荒野前进着。

见崎：“我累了~！走了那么久，城市什么的都看不到啊~！呐，武君，我们来交换一下吧！！”
我：“你在说什么呢！现在的见崎可是我的100倍啊！我怎么能忍受你那样的体重啊！”
见崎：“你是想说我重吗！？这么坏心眼的小人是要受到惩罚的~！”
我：“对不起，请原谅我吧！”
见崎：“嗯嗯坦率是一件好事！……但是这荒野真的很奇怪啊~！完全无边无际！有谁能来帮帮我吗~！”
？：“那么我来帮你吧？”
我・见崎：“………！！！”

# 2

突然从上空传来了很大的声音，我们抬起头去，完全说不出话来。那里竟然有一个比见崎还要大1000倍的超巨大少女。
看了看年纪，大概是小学高年级的程度吧，背上也背着书包，与那年幼的外表不同，这名少女好像非常成熟懂事。
少女：“呵呵‥‥这里居然还会有小人先生呢！为了不会一不小心把姐姐你们踩到了，请坐到我的手掌上吧。”
当我和见崎坐到少女的手掌上时我们非常吃惊，在这里还有着数百个体型和见崎一样大的人，看来大家都是被传送到了这个奇怪的世界了吧。
我：“啊，这个荒野到底是什么地方？你知道吗？”
从少女的角度看的话，我大概是10万分之一的大小‥‥差不多是0.01毫米的程度吧？这种程度的话少女是根本听不到我的声音的，所以我拜托了见崎帮我传达。
少女：“荒野？‥‥啊，你说的是这山谷吗？”
少女说着，把载着我们的手掌举高。于是，我们看到和刚才截然不同的景象，我们一直认为的荒野，其实只是这山谷底部的一部分。
然而这个山谷有着即使是这么大的一位巨大少女也比拟不了的高度，可见这是一个多么巨大的山谷啊。
少女：“其实我也不知道这里是什么地方啊，我知道的只有这个山谷一直延伸，没有尽头，我也已经走了好久呢，但是完全看不到出口。”
见崎：“不管怎样，我们必须想办法离开这里！能不能带上我们一起走啊？”
少女：“好啊！我从一开始就打算这样做来着。”
然后我们与这位超巨大少女一起寻找着山谷的出口。

# 3

女警：“啊啦，你也迷路了吗？”
把我们放在手掌上的巨大少女被突然出现的要更加巨大的女警给叫住了，那女警的大小比少女也还要大1000倍。
从我的角度看的话，大概是我的1亿倍吧，啊，太大了总觉得已经理解不了啊。
少女：“啊那个，这里到底是怎样一个地方啦！我完全走不出这个山谷的说……”
女警：“山谷？啊，你说这个啊！你坐到我的手掌上看一下的话就知道这里是哪里了！”
就这样，少女来到女警的手掌上，看了下四周后吓了一跳。这巨大的山谷不仅仅只有一个，好像还有好几个，这远远看着，这山谷群好像是什么熟悉的东西。
少女：“这里看起来好眼熟啊，这里到底是哪里呢？”
女警：“说出来你可不要吓到哦！…………这里其实是比我还要巨大的女子高中生的手掌上哦！你认为这里是一个巨大的山谷，但这只不过是她指纹的沟壑而已！”
少女：“这样的事，怎么会……”
女警：“我可没有在说谎哦！你看看那边，那里高耸着的五个塔其实是她的手指！”
听到这么一说，少女往女警所指的方向看去，在远处确实有五个塔，但是距离非常遥远，看起来非常的模糊。

女子高中生：“女警小姐！你在说什么呢？难不成是偷偷地在说我的坏话吧！？”
突然间，无比巨大的女高中生发出了声音，她的体型是女警的1万倍，在我看来就是1兆倍了……简直不能相信她原本也是和我一样的人类。
不如说把她比作世界也毫不夸张的说。
女警：“没……我没有！我只是在和小女孩说话而已……”
女子高中生：“你说什么~？你如果不说得大声一点，我是完全听不到的哦~！你怎么说也是个警察吧！？就不能说的大声点吗？！”
女警：“怎么这样……我已经说地很大声了啊！”
女子高中生：“所！以！说~！我完全听不到啦~！啊，算了，对于不听话的女警小姐，我必须要给予你惩罚！就请你进入这里呆着吧！”
女子高中生突然摘下胸罩，露出了她那比星球还要巨大的巨乳。那个巨乳如果有人住的话，到底可以住进几千亿人？”
然后，女警就消失在了女子高中生的乳腺里。
女子高中生：“我没有把你碾碎就好好感谢我吧！但是，女警小姐只能在我的乳腺中度过一生咯！”
但在这里不仅仅只有女警，我，见崎，少女以及很多其他人都被关在这无边无际的乳腺里，然后女子高中生就像什么事都没有发生过一样，重新穿上了胸罩。
女子高中生：“但是说起来，这个粉色的平原到底还会持续到哪里呢？”
女子高中生继续向前走着，一想到每每向前踏出脚步就可能会踩到那些不知有多小的小人，女子高中生就非常的兴奋。
但是脚下除了粉色的地面以外什么都看不见，说不定脚下什么都没有，又或许真的有小人也说不定。但不管怎么说，女子高中生在走路的时候显得很兴奋。
女高中生：“虽然不知道为什么我会来到这里，但这里意外是个非常好的地方呢，在这里我是最强最大的！小人们看着我说不定就像看着神一样？啊~只是想一下就很兴奋了~！！”
由于女子高中生太兴奋了所以没有注意到，周围不知不觉变得像夜晚一样黑暗了。
女子高中生：“咦？怎么变得这么暗了？难道说已经是晚上了吗？但是，明明刚才还亮着的啊，怎么突然就变暗了呢！到底是怎么回事呢……………哎！？……………”
充满疑惑的女子高中生抬头看向天空，发现天空居然倒塌了下来。虽然太大了很难理解，但仔细一看，还是能发现，那是一双覆盖了整个天空的巨大的脚。
女子高中生：“骗人的……骗人的……那不是真的……我才是这个世界的神啊！！！”
在女子高中生大喊大叫的时候，巨大的脚也越来越接近了。那只脚实在是太大了，根本就没有逃出去的可能。

女子高中生消失在了那巨大的脚之下。

不管怎么说，女子高中生还好好地活在那巨大的脚下，虽然这只巨大的脚确确实实是踏在了地上，但是，脚掌有着一部分是拱起的，不过，对于女子高中生来说，这拱起的部分，是不管她怎么努力也触碰不了顶部的。

知瑠久：“啊~又失败了~！”
一个身穿着纯白色长袍的少女，看起来好像是白魔导师。但是，即使是那样宽松的长袍，也遮挡不住少女那极其雄伟的乳房，顶着白袍的胸部高高地隆起。
这个样子看上与其说是白魔法师，倒不如说是一种羞耻的cosplay。
梅亚：“不是吧又来！？这已经是第几次了？……所以这次又变成什么情况了啊？”
向白魔导师搭话的少女，看起来是一位黑魔导师。比起白魔导师的少女，黑魔导士少女的胸部比较小，但与普通人相比的话还是相当有分量的巨乳。
知瑠久：“啊啊，好像是转送魔法和缩小魔法暴走了，而且……还是发生在那个地方……”

知瑠久：“不……不是！转送地点好像是我的乳头……”
梅尔：“哈（叹气）……你为什么总是犯这种奇怪的错误呢？顺便问一下，你把什么给传送了？”

《知瑠久的乳头上》
知瑠久：“呐！梅亚你有找到小人吗？”
知瑠久向缩小了1万倍的梅亚问道。体型这么小的话，即使是知瑠久看不见的小人，梅亚应该能看到了吧？
梅亚：“呐，这里真的有小人吗？我什么都没有看到啊！”
梅亚就是做梦也想不到，在她的脚下有个女子高中生，与梅亚现在的状态相比，女子高中生只是10亿分之一大小，是微生物以下根本察觉不到的微小存在。
知瑠久：“诶~怎么会，很奇怪啊~？我总觉得有小人在啊……”
梅亚：“这可能是你的错觉吧？算了，不管这些了，呐，昨天晚上的那个……要不要现在继续啊？”
知瑠久：“诶！？梅亚酱，还在还是早上哦！”
梅亚：“那些都无所谓啦！就算是知瑠久酱不愿意我也是要动手的哦！”
梅亚恢复了原来的大小之后，吻上了知瑠久的嘴唇，然后两人就这样相拥着倒在了床上，两人从白天就开始做爱。

从今以后，人类将在知瑠久的乳头这个新的大地母亲上繁荣起来。
