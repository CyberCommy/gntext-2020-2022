# 2

或许身为一个神，说出这样的话来真的很奇怪吧。

“真的好想被小人反杀。”

嘛，真的很奇怪吧。

当走到这个高度以后，自己变得已经强的离谱的时候，一切就变得好无聊好无聊了起来。哪怕是这个宇宙最强的生命，最强的文明，最厉害的家伙，现在变成了连我的脚趾都抗衡不了的存在……不是，好像吹口气就再也看不见了……

我真的太强大了。

强大到，可以将这一切随意践踏在脚下，就连不可撼动的物理法则，也可以随心情改变。任何武器都无法对我造成伤害，任何质量的存在也不会让我的身体有一丝痛感。

过分的强大，导致了我内心的孤独，我明明之前只是一个普普通通的女高中生而已。而现在，仅仅只是稍微加快的呼吸，就差点让这个星系毁灭。

我真的太强大了。

我的内心，一直在渴望着什么，但无论我怎么做，都无法满足我内心的渴望。

好想……好想被小人狠狠地欺负一下啊……

明明是肉眼都看不见的渺小存在，却能让身为神的我感到羞耻的快感……

或许，正是因为我的巨大和强大，才总是让我产生了渴望“对手”的想法吧。

于是，怀着这奇怪的想法，我开始全世界收集强大的文明，然后……然后把他们抹在我的身体上……将我身体最敏感最薄弱的部位展示给他们，让他们在被巨人转移家园的恐惧和愤怒中向我的身体倾斜他们的炮火。

请，请尽情地攻击我的身体吧。

眼睛，眼睛应该很娇嫩吧。我把眼睛凑到她们的星系前，将睫毛笼罩在他们的星球下，无数的炮火轰击在我的眼睛上，却连让我的眼睛发干都做不到。

好不容易有那么一点点触感，却被我一时的激动一眨眼就“消失”了。

真的就是一眨眼，就毁灭了一个文明啊……

为什么，为什么都这么弱小啊!

我把收集来的文明小心地放在敏感的部位上，小穴、阴蒂、肚脐、后庭……然后极力忍住身体的颤动，又极力地去感知那些小细菌们在我身上的活动。好像有点了，好像有希望了，加把劲呀，加把劲吧，大家快把我干死呀!

无数个撕裂炮、歼星炮、激光炮、温压武器、生化武器、原子弹、维度炸弹……如同烟花一样打在我的身体上，我的身体终于感知到一些微弱的触感了，但是和我想要的粗暴的对待差的太远了，还不够呀，还不够呀……拜托大家再用点力嘛!

我甚至变得和他们一样小，赤裸地降临在他们的城市，面对毁灭他们星系的女神，无数的酷刑倾斜着他们的怒火攻击着我的身体。

但大家实在太弱了，一点值得兴奋的感觉都没有啊……

无数次实验的失败，我都感觉到有些绝望。

嗯？怎么…屁股那里？有点……发痒？

后庭那里，似乎闪烁着些酥酥麻麻的感觉？

我扭过头看着自己翘起的臀部，仿佛有一只看不见的蚂蚁在我的后庭褶皱深处啃咬着我的肌肤，那种感觉越来越明显，甚至超过了以往任何文明造成的感觉。

看来…好像有上次的文明，在我后庭上幸存下来了？

我用神念扫视着我的后庭，却看不到一点生命的迹象，我几乎动用了所有的神念，才隐隐约约地看到一点点灰点点附着在我后庭细微的褶皱上，无数个看不见的倒刺攻击着我这巨大十亿倍的褶皱大陆。

这，这是有多小啊？!小得我几乎都看不到它们的存在，但却让这么巨大的我感到发痒……

我感到我的全身在发抖，呼吸也止不住地加快。这次的文明，看来真的很棒啊～!

我操动着时间，加速着这个可爱文明的发展。后庭的感觉，开始一点点令人兴奋起来，我强忍着股括肌想要收缩的强烈欲望，徒扰地晃动着自己的屁股。后庭的感觉，已经从微弱的触感变成了抓心的瘙痒!
