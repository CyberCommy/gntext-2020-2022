[http://giantess.uu1001.com/read.php?tid=1323](http://giantess.uu1001.com/read.php?tid=1323)

每周五，发一个新章节本来是惯例。

原计划，我是想在全部写完之后，在这里做一个汇总的。

但就在今天，又有人把《红唇狩猎场》的上部弄到“百度”的贴吧去了。

所以我就提前在这里发这一章节了。

一方面，至少给转载的人多制造些麻烦。

另一方面，我也想鼓励大家在国内的几个坛子里互相多转转。

还有，就是想多听听不同地方的声音。

那当然就只能这样了。

于是我把《红唇狩猎场》叉开，单数的章节发在这里。复数的章节发在“女神”。

还请大家多多支持吧。

（这样可以对比出双方读者的阅读态度，似乎是一件有趣的事儿。） 

# 3

从下部开始，我也开始学但丁。从单纯的“GTS”文学向简单的小说方向靠拢。

这也就意味着《红唇狩猎场》的下部不会像上部那样有许多的“看点”。

在这几章中，我试了试一些悬疑的笔法。

个人希望还是想通过“文学手法”而不是“GTS描写”来吸引读者。

即使是“GTS”向，我也希望能真正塑造一个让人喜欢的角色，而不是单纯的让人兴奋。
