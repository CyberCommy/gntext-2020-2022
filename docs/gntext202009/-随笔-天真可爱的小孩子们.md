# 1

我自己撞坏了自己的蛋，已经四个月了，现在正疼的要命，蛋蛋朝不保夕，可是我想要女儿啊，特此写下这篇随笔，女孩都是按照心目中女儿的形象写的哦！我想生好多好多女儿，希望蛋蛋起码存活一个。。

“为什么……”

一个晒得黝黑的男人，背着沉重的行囊，在说完遗言前就永远的倒了下去，死在了烈日下的沙漠里

自从那两个小孩庞大的影像出现在天空，这个村庄的人就再也没有过宁静的生活

现在，所有人都被逼迫着把全部的家产转移到一块指定的荒地

“你们的村庄要被改为洗脚盆啦~，所以虫虫们统统搬到那边的荒地好啦，我们之后会给你们安排水源的~嘻嘻嘻”

这是几天前以女童调皮的笑声为结尾的一段消息，而经历了之前两个小孩种种暴行的村民们只能不得不无助的背负行囊赶去避难

“对啦，到时天上要下大雨哦，虫虫们准备好房屋住所来躲雨”

“哈哈哈……”

女童们眼中的玩具，如同蚂蚁搬家一样的爬行着。

在损失了近一半的人口，无数人失去亲人，家破人亡的代价下，终于全体搬迁完毕了

“虫虫们做的不错，现在请去那边的小土堆上等待指示！”

在山顶，人们看到的是一只从天而降的巨大的脚

那只脚非常的短小，又嫩得晶莹，只有幼女才会有的水嫩奶气的肌肤，完美的“小”脚

脚笼罩着新搬好的村庄，

脚，在村庄的上空扭动着脚趾

得意而调皮的扭动着，可爱活泼的在村庄上空蜷缩和放松着

“小虫虫们都没发现吗？这片区域刚好是一只脚的形状嘛”

“发现了又怎么样，还不是主动把家伸到人家脚底下”

“哈哈哈×2”

人们纷纷跪下，膜拜着巨大的巨大的脚，祈求着最后的宽恕

“还真跑到这里避难了呢，刚好把你们最珍贵的东西统统一脚踩扁！”

巨大的脚趾，按在地上拍打了几下

可爱的行为，让上百人失去了住所

“逗你们的哦，我怎么可能那么狠心呢？不过当然还是要玩一玩了”

脚抽走了

咚！

又重重的落下了

原来并不是为了抽走，而是为了蓄力而抬高。

不管那些刚刚得到希望就化为绝望的人群，两个女孩继续聊着

“不知道刚才那一脚下去，虫虫们是什么心情呢~”

“一会抓只活的采访一下吧”

“虫子们好像连之前的十分之一都没有了呢”

“是啊，都怪你玩的时候太浪费了啦”

“虫子就是用来玩死的啊”

“哈哈，听到了吗虫子们？你们全部都得被玩死哦！”

人群们的耳中，身旁的亲人，街坊的哭声，和头顶轰鸣的女童嘲笑声混在一起，天堂，或是地狱

“谢谢你们一直挣扎到现在，现在为了奖励你们的愚蠢，给你们约好的大雨哦！其实就是我们俩的洗脚水啦~~”

巨大的水盆将几千米宽的巨湖一般的洗脚水泼向了挤在一起的人群，灭世的洪水，就是这样。

“好像还有活下来的虫虫呢？等洗脚水干了记得给我们讲故事哦~讲你们一点点被我们玩死的故事，哈哈”

这就是原本繁荣美丽的村庄，也是小孩子们的玩具，天真无邪游戏中的消耗品

想不想加入玩具的一员呢？抬头看看天空吧，她们在笑哦~
