# 千秋外传：玛利亚的视频工坊【排泄】

作者：水泥彼岸

TID：26748

 

# 1

　　苏珊是为数不多的流亡者。　　她的城市在战乱中被作为胜利者的战利品，整个都被缩小了，放在了一名叫做千秋的婴儿的房间里面作为她的玩具。　　其实被缩小也还好。　　听说有些城市更是在战斗中的时候就直接被摧毁了，整个城市被巨大的脚印覆盖，更有些城镇没有留下任何的痕迹，直接消失在了这个世界上。　　虽然千秋一般只是在城市周边好奇地观察着，不会出手毁坏城市，整个城市里面很安全，但是依旧有着一小部分人选择离开城市，不过很少有人能够出去之后活着罢了。　　这些人被称为流亡者。　　他们每一天都看着巨大的婴儿看着自己，在城市的周围爬来爬去。他们的内心被恐惧填满，他们相信城市终有一天会被千秋摧毁，惶惶不可终日的他们会逃出城市，在房间里面寻找着地方藏起来，在屋子里面到处流亡着寻找食物。　　苏珊就是这样的一名流亡者，她当初在城市里面发现了一条通道，经过了几天暗无天日的洞中行走后，她发现自己来到了另一个房间。这里是厨房的食物储藏室，里面有着数之不尽的食物。　　接下来的一个月，苏珊都在洞穴中穿行，她发现这个洞穴里面四通八达。厨房就只千秋的卧室隔壁，通过洞穴行走不远就可以从厨房来到千秋房间里面的一个柜台上，看到房间里面的景象。……　　这一天，菲儿（千秋的家族）家到来了一位尊贵的客人，米娅小公主，不过这些跟苏珊没有什么关系，真正有关系的是随着米娅的出现，千秋多了一个看起来还是高中生的女仆。　　这名叫做玛利亚的女仆在米娅走后就成为了千秋的专属女仆，每天负责照顾千秋，同时还要照看千秋房间里面的城市。玛利亚每一次给城市里面添加能源的时候都会挂着微笑，城市也在她的手中变得越发繁华。但是每当玛利亚将头转离城市的时候，苏珊就会看见玛利亚不怀好意的笑容。　　苏珊回到城市中，但是没有任何人听她的，他们被玛利亚照顾的非常好，甚至想要将苏珊也留在城市里。　　见此苏珊也就没有多说什么，趁着晚上悄悄地离开了城市。　　几天之后，苏珊出现在了柜子上方，观察着千秋的房间。　　又到了玛利亚给城市补充物资的时候，但是今天的玛利亚手中拿着一个不一样的机器。她在城市当中放置了什么之后使用了手中的机器，整个城市瞬间消失了。随后玛利亚蹲下身子伸出手，从地上轻轻的拿起了什么。　　不！城市没有消失，而是整个被缩小到了巴掌大小，躺在女仆的手心中。　　苏珊看着这一切的发生，但是没有任何的办法。　　艾迪现在非常的后悔之前没有听苏珊的话，现在当他想要离开整个城市已经是不可能的了。整个城市在瞬间缩小，化为了女仆玛利亚手上的小小玩具。没有人知道现在这个女仆想要做什么，但是绝对不会是什么好事。　　玛利亚的脸上挂着跟以往完全不一样的邪恶笑容。在这一刻艾迪终于相信了苏珊的话，可惜已经太晚了。整个城市的命运已经完全掌握在了玛利亚的手中。　　就在这时候，千秋从不远处爬了过来，好奇的看着玛利亚的手中。　　一种淡淡的异味出现在了玛利亚的鼻子当中，提醒着女仆应该给千秋换纸尿裤了。看着千秋的样子，玛利亚的脑海中突然冒出来了一个大胆的想法。　　她将千秋的纸尿裤从后面微微的掀开一点，当手中的城市放了进去。　　随后玛利亚将千秋挪到了原本放着城市的地基之上，看着千秋好奇的在上面爬行。　　一瞬间，整个城市瞬间笼罩在了一片黑暗之中，城市中的人们在快速的移动中难以分辨发生了什么，但是随后一个有些特殊的臭味出现在了城市当中。　　当城市里面大部分的人都还在不知所以的时候，几名带着孩子的母亲已经猜到了她们在什么地方了。随后，城市里面巨大的探照灯亮起，验证了她们的猜想。　　她们现在在千秋巨大的屁股瓣上，纸尿裤覆盖了整个城市的天空，而千秋依旧在不断的爬动，整个屁股瓣都在随着一起不断的晃动，让城市在随着渐渐的向着后方滑去。　　随着滑动的进行，那种异味变得越发明显，整个城市的倾斜感也越来越明显。“那是……”　　千秋今天早上的排泄物并没有被女仆及时的清理，而是留在了纸尿裤当中，当城市里面的人们看到的时候，他们已经来不及做出任何的反应了，这个城市在一瞬间滑落，坠落在了那一滩排泄物中。　　一名婴儿纸尿裤当中的一大滩排泄物瞬间化为了一个人间炼狱，整个城市的人都在里面挣扎着，企图逃脱这片由千秋生产出来的海洋，但是他们注定不会成功。千秋依旧在不断的爬动着，整个纸尿裤也在她的动作下晃动着，不会有任何的人能够在这样由半固态的物体形成的滔天巨浪中幸存下来。　　在千秋爬行的痕迹完全掩盖了这个城市原本存在的痕迹后，玛利亚将千秋抱起，用手晃了晃已经明显下垂的纸尿裤，随后贤向着一旁的柜台走了过去。　　看到玛利亚向着自己所在的柜台走来，苏珊立刻跑回到了洞穴里面，但是她仍旧控制不住自己，顶着难闻的气味在洞口看着外面。　　巨大的纸尿裤在她的面前被揭下，显露出里面如同泥浆一样的屎黄色婴儿便便，眼神稍好一些的苏珊还能够看到里面一些灰黑色的小黑点，但是几乎完全被便便遮盖了。而女仆也是同样拿着一个机器，似乎在录制什么。随后女仆将机器收好，纸尿裤团成一团扔进了垃圾桶，抱着千秋去清洗身体了。
