# 1

安迪是一个标准的好青年，虽然是大学毕业刚工作两年，但是因为是IT男，所以收入还算可以。他的性格略有腼腆，但是身高长相还可以，目前一段时间困扰他的是没有找到女朋友。他现在为了省钱，与人合租住在一套大的公寓里，这套公寓三室一厅，安迪住在一个主卧，房东是一个同住在一栋楼里的年轻阿姨。另外的两个房间总是换房客，对于大城市的青年来说，这再也正常不过了。安迪本来觉得生活过得很平淡，直到他碰到了对门房间新搬来的女生，直到他一次偶然的深山旅游，让他的生活大大的改变了。这一天，安迪躺在自己的床上陷入了沉思，就算是手边的烟掉了他都没有想明白这到底算是什么呢。
事情要从一周前的旅行说起。安迪是一个旅行爱好者，尤其偏爱大自然风景，有时会找友人结伴出行，有时他甚至会自己去幽静的地方一探究竟。这次恰好是安迪一个人，让他遇见了一个自己也解释不清楚的事情。他撞了仙缘！是的一个IT男撞了仙缘。安迪怎么也没有想到自己的一次好奇，独自探查了一个环境优美的山洞，之后就像是做梦一样，不知道触动了什么，他进入了一个奇异的空间，然后他也没有看见谁，也没有碰到什么奇珍异兽，后来他也没有记住自己到底看到了什么，只是记忆里就多了一样神奇的功法或者说能力，他的能力是奇门遁甲，说的简单点就是穿墙术，多么牛掰的一项技能，他也是靠着这个能力，穿过了一个古老的石门回到了现实的山洞。等他回到自己的公寓，都没有缓过神来。他的想法就是“这太不科学了！太不讲道理了！”这回他喝了半瓶白酒，第二天清醒过来后，也许是冷静了许多，他忽然想，是不是可以那这个技能干点什么呢?这时，他第一个想起的竟然是对门的美女，罗琳。今天是工作日，罗琳还没有回来，安迪本来是请了一周假的，由于提前回来两天，这两天他还是比较清闲的。想到这里，安迪突然想看看这个美女的闺房，他没有别的想法，只是想了解一下罗琳的喜好，有的放矢的与美女搭讪而已。说做就做，安迪想好了，就走出自己的房间，来到宽敞的客厅，目前是上班时间，另外的房客小郑也在上班，所以公寓里只有安迪自己。安迪偷偷的笑了笑，关好了自己的房门（安迪回想起来，为什么不在自己的房间里再试一试穿墙术的效果呢，虽然依靠这个技能从山洞里的出来了，但那毕竟是密境，这法术在现实世界到底有多大效果，非要跑到人家美女房间里试一试，每次想到这里，安迪都顿足捶胸。）也许是安迪太高兴了，他可是从来没有看过美女的闺房，所以太兴奋了，很多事情也没考虑。当时他倒是想起，拿后背去撞门，从而穿门而过，如果技能有问题，也只有后背撞一下门而已，不至于正面呼在门上。他默念的法术，然后一闭眼猛地朝门撞去，啊，没有疼痛啊，有感觉，但是怎么感觉不对劲，好像动不了！安迪纳闷了，他睁开了眼，他看到了客厅里的景象，但是从旁边的门框视角来看，他正好附在了门上，而且从后背略有轻风的感觉，他的后背到了门的另一侧，简单说他好像附身在了门上，他的正面成了门的正面，后背面成了门的后面。“啊，”安迪一阵的恐慌，这是怎么回事，这仙家的法术怎么是个坑？难道默念的不对？难道穿墙的姿势不对？难道不应该用来做不义的事情？难道跟门神似的等着人家姑娘回来？啊，这该怎么办啊!安迪不知道想了多长时间，不知道需要多长时间才能从门中再穿出来，直到天黑，另一个房客（小郑，同样是一个刚毕业的单身小青年）从他身边经过，或者说从罗琳的门口走过，他都没有理出一个头绪来。他当时想呼喊小郑帮忙，但是他在门中，是行动不了的，叫不出声音来的。也许他叫出声音来，一个会说话的门也会让小郑困惑（害怕）吧。小郑只是在客厅里呆了一小会儿，就进了自己的房间安静的休息（疯狂的玩网游）去了。又在内心挣扎中等了两个小时，安迪等到了这间卧室的主人，罗琳。一个清秀柔美的姑娘，只是当时的脸色比较红，安迪听到了门外清脆的高跟鞋声音，然后客厅的门开了，罗琳走了进来，换了拖鞋。安迪感觉很是尴尬(激动)，他是第一次这么毫无掩饰的直视着罗琳一步步走过来，他的视角高度正好与罗琳持平，罗琳也是直视着他一步步走来的（人家是一直看着自己家的门而已）。快到了跟前，安迪发现罗琳的脚步有些不稳，他还隐隐的闻到了酒气，置身门中，他还是有感觉的。马上走到门前，罗琳的脚步一滑，她本能的抓住东西稳住身体，所以她稳稳的抓住了门把手，阻止了身体的下落。需要说明，这是卧室的门，所以门把手是普通的圆柱形式（后面细颈前面圆头的那种，圆头上带锁孔），可是，安迪却在这突然的动作中，身体不由得一振，啊（内心中大喊），好酸爽，他竟然发现自己的小弟弟竟然覆在了门把手上，而罗琳单手攥着门把手，其实她也是使劲攥着安迪的小弟弟， 啊， 安迪酸爽的（激动的）在内心高呼起来。罗琳扶着门把手稳了几秒钟，自言自语“这几个丫头，不就是这个月我的业绩好一些吗，非要让我请客，还让我喝那么多酒，哼，下回喝酒记得给你们灌回去。”门是锁上的，所以罗琳想转一下门把手，打开门是不可行的，而此时的安迪感觉自己的下体就像是木棍一样坚硬挺拔（其实就是），罗琳的小手怎能转的动呢，但是安迪在酸爽之余觉得好像哪里不对劲的事情要发生了。是的，罗琳见没有打开门，说以在背包里掏出了钥匙，然后，安迪就感觉到了一个坚硬（尖锐）的东西捅进了他的马眼里面，啊（内心中狂喊），他还感觉那个坚硬的东西在马眼里顺时针转动了一圈， 啊（内心狂喊，一千头草泥马奔腾而过），太疼（太酸爽了）。安迪感觉自己身体里的某个机关开了，然后，罗琳坚韧的小手握住了圆柱型的门把手顺时针转了九十度， 啊（安迪内心狂喊，一万头草泥马奔腾而过），门开了，罗琳从安迪的身边走过，走进了公寓，随后把门关上了，如果安迪现在能出冷汗，罗琳的房门已经是汗流门面了。安迪表面平静的（内心狂呼的）置身于房门中，面对着安静的客厅，等待着自己从门中穿出。他需要等到什么时候呢？美女罗琳还会对着他的马眼多捅几回么？他只能面对客厅，倾听美女闺房中的声音么？<title>2</title> <link
