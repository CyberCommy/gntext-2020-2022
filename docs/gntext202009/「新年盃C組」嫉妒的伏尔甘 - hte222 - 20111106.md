# 1

两边大楼向我涌来，更为恐怖的是，在不断涌来的大楼背后，掩盖了地平线的肉色高墙不断摧毁着我身边的文明痕迹，高墙上沾满了一些灰尘和红色的斑点，在高墙中的人们，即使极力仰脖也看不到天空高墙的尽头，他们也绝对想不到那无尽的高墙只不过是可爱少女的一双脚而已，在巨大的玉足面前，几百米高的大厦与几十米深的地下室没有任何区别的翻起，即使日本引以为傲可以阻挡八级地震的钢筋水泥的建筑物没有给予她哪怕一丁点阻力就与后面的泥浆混在了一起。甚至她都感觉不到脚下的究竟是建筑还是土地，房屋坍塌所引起的瓦斯和电力的爆炸引起了火灾，但玉足的前进速度远比火灾蔓延的要快，在玉足还没有波及的区域因为巨足造成的冲击波樯倾楫摧，大厦的玻璃大多震碎，漫天的沙尘笼罩了整个城市。在沙尘后一个想起银铃般的一声“哼！”

    在城市奔跑的人群中，一个披着奇怪的防护罩戴眼镜的少年正站着的极力的调整自己的思绪，“这就是传说中的NICE BOAT么，果然同时攻略七位女神太危险了,本以为恢复神格即使出现心灵裂隙也不会让古恶魔趁虚而入，但没想到入侵夜月的竟然是七罪之一——深渊领主嫉妒，泰坦级的恶魔甚至中和了她的神格，终于在一次为密涅瓦恢复神格的时候爆发了......”。“喂喂，听的见么”恶魔通讯器显示出了马尔斯。“很抱歉，我们无法帮助你，我们只要靠近你就会增加深渊领主的能量，不过.......”光环退去，一个帅气的...嗯... 女生出现“无论发生什么，我一定会保护你的，桂马。”

    轰一声巨响，城市里最后完好的玻璃也碎光了，月夜坐了下来，美丽的臀部形成了一个几十公里长几百米深的桃形圆坑，半边城市与土地融为了一体，其压缩的密度甚至不能用人类现有的科技分开，掀起的泥土海啸甚至淹没了屁股后面的一座城市，幸存的人类仰望那高耸云天的雪白，恐惧让他们想逃跑，而墙中散发出芳香却让他们本能的想靠近,几架自卫队飞机徒劳的在白墙和玉足上射尽所携带的一切火力，而他们的地面部队已经被刚才的冲击全部摧毁，而月夜甚至不知道有人在抵抗，因为他们他太弱小了，即使最大吨位的战车属于即使放入鞋中都不会感觉到不舒服的沙子，而其他的几乎如灰尘一般。即使她知道，也会认为有幸与女神接触是一种荣耀，她并不吝于赐予这种荣耀。

    坐下后月夜调整了一下坐姿让臀部下最后一批幸存者也融入这片大地，然后她轻声说了一句：“桂马！”

    虽然只是轻轻一声，但足以让底下城市大半人耳出血，连旁边的城市玻璃也被震的粉碎，少年在那个像防护罩的东西里才逃过一劫。

    看到没出现想要的结果，月夜扫视一眼，选择了一座最高的大厦伸出了手，所产生的气流卷使几架飞机在她身上爆炸，修长的巨大手指落下时摧毁了大厦两边的建筑。不过她不在意也没法在意，轻轻的用指甲把夹起来送到眼前，起码她认为是轻轻地。而大厦里边的人类仿佛经历一场风暴和地震，随着月夜手的抬高，大厦里的人不断因为气压变化而猝死，幸存者也因为缺氧而不停大口的喘气。

   “桂马，如果你不出来，我就把这个捏碎了”说完月夜顽皮的动了动手指一些幸存者被甩出了大厦，从远处看仿佛一些灰尘落了下来。

   “不要，你知道你拿的那是什么，是KEY大厦，你摧毁了它等于摧毁了游戏界的良心。”少年从防护罩出来，发现上当的他想要躲回去，可惜已经太晚。

    “将军!”月夜笑着捏碎了大厦！
