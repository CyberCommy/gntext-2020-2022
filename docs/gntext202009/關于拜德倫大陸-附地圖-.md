# 1

大家不要誤會,關于拜德倫大陸的故事不局限在馬修壹個人身上,這是壹個世界,壹個世界會有許多故事,講不完的故事,馬修倒下了,還有別人:P

現在給大家介紹壹下這個所謂的拜德倫大陸。

拜德倫大陸，東西南三面環海，北方有崇山峻嶺。人、魔、獸三大種族在這片大陸上居住著。

其中，人類集中在大陸的東北，所有的人類都是修斯帝國的子民，帝國據守著斯特魯要塞的天險以抗衡魔族的侵犯，北面與群山相接。人類多才多藝，相貌和我們這個世界的黃種人壹樣，但皮膚稍白。人類能夠學習所有的魔法，同時和北方龍谷內的巨龍定有盟約，所以也擁有壹定數量的龍騎士。

獸人分布在大陸的南面，獸人領地土地肥沃，物産豐饒。獸人有狼人、豹人、熊人、狐人、蛇人等多種，他們以種族爲界限聚居在不同的領地內，早期的獸人只是壹些零散的部落，互相敵視並且總受魔足欺壓，後來獅人部落獅心王查理率領部衆難征北戰，統壹了獸人，建立羅克帝國，從此不受魔族擺布，自力更生。

魔族人的迪坎達帝國在大陸的西側，首都索萊城。由于這裏土地比較荒蕪，而且氣候寒冷，所以魔皇壹直想攻打人類的領地，以求發展。魔族有各種各樣的魔獸，據說都是上古的物種，魔族人是天生的黑暗魔法修行者，壹般十幾歲的孩童也有中級法師的實力，由于黑暗魔法的改造以及氣候的影響，魔族人皮膚白皙到極點,頭發烏黑，瞳孔是紫色的，身材纖細修長，男性大多容貌冷俊，女性則美麗動人。

除了這三大種族外，還有壹些數量較少的種族，比如，三族交接的界林中有精靈族，他們是自然的兒女，害羞怕生，可以和自然交流，能夠潛伏在樹木中。

龍族居住在北方的山嶺中，而90%以上的龍居住在修斯帝國北面的龍谷中，他們是最強大的生靈，但是數量不多且繁衍能力不強。

種族之間並非無來往，貿易是不分國界的，所以除了軍用物資，沒有哪個國家限制貿易。種族之間也常有通婚，而其後代往往具有父母的雙重特征。

職業：法師是奧術的控制者，壹般爲了追求高級的法術，法師們會選擇專修壹門魔法，法師們從不睡覺，他們依靠冥想同時恢複魔力和體力。法師的等級由低到高爲學徒、中級法師、高級法師、魔道士、魔導師、以及傳說中的大魔導師。多名魔導師可聯手施放憾天動地的禁咒，而大魔導師可以獨立施放禁咒。

戰士，壹般在軍隊中服役或者是傭兵，等級按其掌握的鬥氣由低到高分爲入門級、精通級、了然級、大地級、天空級以及傳說中的斷山級，天空級鬥氣已經允許戰士禦氣而飛，而斷山級的鬥氣據說能在百步之外將山岩削得如同鏡子壹樣光亮，而掌握這種鬥氣的人，大家都會冠以“戰神”之名。

另外還有壹些其他職業，比如煉金師，他們能制造各種奇怪的玩意，可以通過魔法將物質等價交換爲別的東西。

刺客是戰士的旁支，他們壹般有矯捷的身手，但通常力量不足，他們依靠自己精巧的隱匿技術以及狠辣的刺殺手法置人于死地。
