# 1

课堂上的一些瞎想，暂时还没写到含有GTS内容的桥段

大学的课堂在理想之中的样子大多是洋溢着激情和思维的火花的，无论学校本身的好坏，大家似乎都在不遗余力的塑造着这点几乎可以算作做“偏见”的认知。

然而实际上，垂垂老矣的教师，无精打采的同学，闷热且充满着各种奇怪味道的空气才是课堂最常规的模样。

除了前排少数几个励志保研或者心血来潮听几耳朵的人之外，大多是同学的课时不是在后排网上冲浪就是在梦香中度过。

“无所谓啦，反正这老头儿给的平时分高，到时候你卷子借我看一眼大家就都能过啦～”

我的好哥们儿拍了拍我的肩膀，接着用几乎是“掩耳盗铃”的方式弓着腰从边上溜出了教室。

以他的块头，除非眼镜片比防弹玻璃还厚，不然不可能不会注意到他。然而事实有的时候就是这么魔幻，不仅台上滔滔不绝，吐沫星子翻飞的老师没有注意到，就连我边上认真听课的同学也没有注意到。

“或许这就是如饥似渴的求知欲吧...”

我在心底默默吐槽。

虽然我也不好说自己是什么混子，但是要我像他们那样学习？饶了我吧....我宁可永远的睡死在课堂上也不愿意做到前排。

但是现在我为什么坐在前排呢？

“呵...鬼知道”我自嘲的笑了笑，可能是进了课堂之后就坐在这里了吧，反正整个教室的后面都被占满了。

“大家都知道，所谓梦境，是我们在无意识中，对于我们的人格所认知的事物进行的再演，就好像你看了别人日记，然后带入自己情感之后.....”

平心而论，这节课前排没人坐，不是毫无理由的..

“那个...你好～”就在我胡思乱想的时候，一股轻柔的香气从边上传来。我扭过头，看见一个曼妙的少女伏着身子，贴在我的旁边。

“请问学长，这里有人吗？”

黑色的长发如瀑布般洒在雪白的肩头，淡蓝的的格子裙散发着青春的气息。

与之在视觉上鲜明对应的是丹数明显不高的黑色裤袜，淡淡的肉色若隐若现，再往下则是纤细的脚踝和可爱的、扭成八字的乐福鞋。

成熟中透着淡淡的生涩，这样的感觉简直....

“学长？”

“哦 啊——这里没人，你做吧”

对不起了兄弟，今儿午饭晚饭我全包了，但是请一定在厕所都抽几根烟，最好别回来！

“好的，那就麻烦啦～”

说着，她解下书包，甩了甩头发，从包里拿出一本书，一个精致的笔袋和一个小盒子。整个过程优雅而简练，我看着她精致的脸颊，透水的眸子，微微有些出神。

“嗯？学长有什么事情吗”

大概是我的视线太不加以掩饰，以至于让她产生了实感 她歪过头，有些困惑的看着我。

“呃呃...不不不，只是很好奇为什么现在还会有人来上课。”我指了指墙上的挂钟，一般的迟到倒也说得过去，可是现在已经上了接近一个小时的课了。

“大概是...等到睡熟了吧？”说着，她朝后面那些已经陷入沉睡的人眨了眨眼，有些轻快的说道。

真是奇怪的解释....不过大概是我问得问题太奇怪了吧...不行，我得赶紧找个别的撬开话题的方法，可能不能让这个女孩儿就这么白白的走了。

不过我为什么会突然对她这么感兴趣呢？

“然而实际上，还有一种解释。在你沉睡的时候，你的思绪实际上寄托在了另一个位面的生物上，它们的世界是只读的，也就是只可感知而不可改变...”

抑扬顿挫的声音从讲台上传来，打断了我的思绪。

“他们的世界就像一个虚拟的网络游戏一样，控制每个人的皆是本我中最基础的东西，也是人的两大原动力，即性欲和与他人斗争的渴望。”

我看着我边上的少女，她正在用钢笔迅速的记录着课上所讲到的内容，字体精致娟秀，就是总感觉有些眼熟，仿佛在哪里见过一般。

“管它为什么感兴趣，这么可爱的女孩子当然要接着往下发展啊...我之前都在想些什么嘛...”

我拍了拍自己的额头，感叹自己之前的想法是多么的莫名其妙。

“那个...你刚才说学长吧？你是大一的新生吗？”

她放下手中的笔，嘴角扬起一抹坏笑。

“当然咯...毕竟喜欢学妹的人大概会多一点吧！”
