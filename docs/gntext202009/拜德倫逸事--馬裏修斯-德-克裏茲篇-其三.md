# 拜德倫逸事--馬裏修斯·德·克裏茲篇 其三

作者：wobuzhidao

TID：806

 

# 1

回聖西索城的路上，我也漸漸和銀兒熟了起來，按莉莉亞的說法，是多了壹個分享我的姐妹。其實這也是我選擇步行回去而不是傳送回去的原因。

終于到家了，莉莉亞說放假了她要回去陪她的父親，所以這幾天我要好好陪陪銀兒。因爲銀兒不是學院的人，也不好住在學生寢室。我幹脆在離學校不遠的地方購置了壹套大房子，並到學校辦理了走讀手續，搬出來和銀兒住在壹起，這洋也省得別人發現她是個白狐人。

白狐人的服飾很特別，女生們只穿壹件寬松的袍子，平時赤著腳，那天在樹林裏發現銀兒時，她的袍子已經劃破好幾處，才讓我看到了她曼妙的身材。爲了讓她習慣，我在屋子所有的地面都鋪上了厚厚的地毯，床和沙發都是獸皮的，好讓銀兒能赤腳到處走而不把腳底弄葬，同時也省得她尖尖的指甲劃壞東西。莉莉亞臨走的時候去了趟魔法店鋪，帶回來壹車的藥，大部分是水下呼吸，這種藥可以支持人在無氧環境中長時間呼吸，還有就是曾經暗算我用的麻痹藥，她說開學以後她也要搬過來住。

我利用這壹個月的時間，好好地了解了銀兒，她已經有了18歲少女的外貌，身材，身體都已經基本成熟，但是智力只有人類10歲兒童的水平。不同于莉莉亞有時閑靜，有時野蠻，銀兒天真爛漫，活發可愛，當然這洋我業很好哄她。作爲白狐人，她有著人類沒有的光潔如絲綢般的大尾巴，觸感極佳，腳底板非常柔軟，雖然看上去和人類壹洋，但大概有壹曾軟軟的肉墊保護他們赤足行走時不會被地上的碎石沙礫劃傷腳，而且，她的跨下與腳趾縫間都有發達的體腺，分泌出壹種極具媚惑力的體味，說不上是狐臊，因爲似乎還有濃濃的麝香味，總之叫人聞起來如癡如醉。像這洋有著神兵利器卻頭腦間單的女生真是可遇而不可求啊。

由于銀兒十分天真，總是纏著我，而我叫她做什麽她都會照做。在家裏反正沒有別人，她幹脆不穿衣服，確實她的裸體更讓人著迷。正常大小的時候，我常充當她的坐墊，她非常的輕，大概只有40公斤左右，坐在我的胸口，屁股朝著我，大尾巴在我臉上掃來掃去，十分舒服，同時，我也能享受那種迷人的體味。

而更多時間，我會保持5厘米的身高，這洋我可以做更多的事，比如藏在地毯的絨毛中，等待銀兒壹腳踩在我身上，本來40公斤的體重壓在我身上我是絕對吃不消的，但她的腳底因爲有了肉墊而非常柔軟，卸去大部分重量，我感覺整個身體都被埋入了她的腳心，和她的玉足融爲壹體了，而她的腳底觸覺並不靈敏，因此很少發現我就在她腳底下，後來我幹脆在身體正面塗上膠水，貼在她的腳底上，而把臉埋在腳趾縫中，瘋狂地呼吸著腳趾間媚惑的氣味，我經常讓她在跑步機上鍛煉，不壹會，腳趾間就會有大量的腳汗，我會吮吸個夠。

關系發展到壹定程度，我也經常光顧她的前後庭院了。她的跨間也生有陰毛，但是不像人類那麽黑亮，她的陰毛是潔白如雪的，而且非常非常細密，從小蜜穴的周圍延伸開來，摸上去和細膩的狐皮沒有區別，大概因爲這原始而野性的地方保留了壹點狐狸的特征吧，前面已經說了，她對我非常順從，我會喝下呼吸藥水，然後讓她用手把我送到小穴的最深處，她那裏非常緊，肉壁充滿彈性，而且不管我進去多少次，她那裏都好象新的壹洋，永遠保持緊繃。我進去後四肢無法動彈，只好直挺挺地被夾在肉壁之間，只能呆在裏面喝水，不過這些難不倒我。有壹天晚上，銀兒入睡以後，我偷偷起來，掰開她的嘴給她灌了壹點昏睡藥劑，雖然弄醒了她不過她很快就倒下了，我把她倒過來堆在牆角，把兩腿分開，于是她的小穴便朝天開放了，我喝下水下呼吸藥後變成10厘米高，然後跳到她跨間僅有的壹點平地上，向她的小穴內倒入刺激性的藥物，不壹會，她的兩片小花瓣變得粉紅，濕閏，並且微微阖動著，最後，竟然像壹張貪吃的大嘴，壹張壹合。我走到小穴邊觀察，本想等她的愛液噴出後再下去，但是她的花瓣非常潮濕，加之不斷顫動，我腳下壹滑，雙腿已經滑進了花芯中，頓時，我感覺下方傳來強烈的吸力，由于再往下還比較緊，我沒有被馬上吸進去，就像蛇吞鳥蛋壹洋，我壹點點地往下沈，先是下肢被吞沒，然後到胸口，到下巴，最後，我消失在她的跨部中。

經過漫長的擠壓，我終于被帶到最深處，由于這裏面媚惑的氣味太重，我已經有些神志不清了，失去理智前，我用空間魔法把銀兒丟到了床上。然後，我被她愛液的氣味激的失去控制，完全成爲她陰道的玩物，既然手腳被夾得緊緊的，我就幹脆像蟲子壹洋蠕動，然後被她的陰道夾得更緊，直到我因擠壓無法呼吸而張開嘴，那壹刻，就像有預謀的，陰道中的愛液在擠壓下全部勇如我的口中，爲了不被撐死，我只好在喉嚨中開了個空間門，把她的愛液都傳送到其他地方——或許有的地方現在正在下酸雨吧，我管不了那麽多了，不過，這些愛液可是壹滴都沒少地經過了我的舌頭。把水放光後，陰道裏也稍微平靜壹點了，我這才感覺到雙腳已經踩在銀兒的最後防線上了，不過我並不急著進去，我還沒好好享受這緊緊的陰道呢。似乎這裏以前從未有過空間，周圍的肉壁不留空隙地壓著我，想把我這個入侵者擠碎。想到銀兒醒來後我就是頭朝下了，于是我用盡全身力氣將她的陰道撐開壹點，然後調換了身體的朝向，當重新被夾得直挺挺時，我已經筋疲力盡，于是合上眼，沈沈地睡了，我喜歡睡女孩的體內也是從這壹天起的。

壹陣震動聲把我叫醒，我知道已經早上了，銀兒已經醒來，她開始走動，我能清楚地感覺到因爲她跨部的扭動而引起陰道的肉壁來回摩擦我的身體，不久，摩擦停止了，我聽到不遠處有嘩嘩的水聲，應該是她在小解把，因爲在陰道裏所以聽的很清楚，我要是再呆在外面壹點就被沖出去了。

由于什麽也看不到，我就用空間魔法轉移視覺到外面，她洗漱完畢後清醒過來，回到臥室發現昨完躺在床上的我不見了，于是到處尋找。我想要是找不到我她又要哇哇大哭了，于是就傳音給她：“不用找了，我在妳的陰道裏面哦。”“那大哥妳快出來啊！”她急道。“我倒是想出去啊，但是妳把我夾得太緊了，我動不了，動不了就沒法出來啊！”我故意逗她。她聽後不說話，似乎開始想辦法“恩，也許這洋吧。”只見她深吸壹口氣，小腹微收，我周圍的肉壁開始蠕動，壹開始，蠕動雜亂無章，我被攪得頭昏腦漲，後來，蠕動開始整齊，我才發覺自己在擠壓下開始向外滑，她原來是想靠控制陰道的肌肉把我擠出去，雖然覺得有些不可思議，因爲人類做不到這壹點，不過她是個以靈巧見長的白狐人，想到她的五個腳趾和手指壹洋靈活，還能弓起腳用腳心把我夾起來，我突然覺得控制陰道的肌肉對她來說應該很輕松。

最後，我終于被吐了出來。“真有妳的啊，看不出來，論控制自己的身體，恐怕莉莉亞也比不上妳。”“怎麽洋，我厲害吧”她笑得很甜。銀兒的首次體內探險完畢，接下來的幾天，我又光臨了幾次，不過不用藥物，她可以用自由控制自己的陰道，我在裏面全憑她擺布，她能能控制我在裏面進進出出，能控制肉壁擠壓我的胸口讓我張嘴喝愛液，騰挪翻轉，都不在話下，最後我突破她處女膜的時候，還多虧她在我咬住膜後把我用力往外扯，才撕破了她那充滿彈性的屏障。

對于銀兒，我的另壹個嗜好就是她尾巴後面的菊花瓣，因爲那裏面有著最密集的體腺，而在直腸中，迷惑人的氣味蓋過大便的味道。同洋，她的臀部也十分靈巧，只要我壹接近她的股鈎，先是因爲瘙癢，兩臀自然地壹夾，感覺到我的存在後，她便控制臀部的肌肉將我的頭對准肛門，然後送進去，她的肛門裏十分溫熱，發現這壹點後我冬天常在這裏面過夜，銀兒沒法像莉莉亞壹洋把自己裏面洗的很幹淨，雖然後來莉莉亞搬過來後常幫她清洗屁股裏面，但之前的工作都由我來做，開始我並不情願，但是她很會撒驕，弄得我沒辦法，只好答應，她控制我在她的直腸和股鈎裏遊走，張著嘴吞下殘留的排泄物，並且把褶皺處舔幹淨，大便的味道並不好，幸好因爲這裏體味很重，將臭味蓋過去了。在這裏，我的身體可以再大壹些，大概30厘米高，身體在她的直腸中，脖子穿過肛門，臉露在外面。仍舊被她的大尾巴掃過來掃過去……

和銀兒壹起生活，許多事都會變得有情趣，而這情趣都來自她的壹雙玉足，吃飯的時候，我保持正常人的體形，而她則用她的腳來給我喂飯，通常我讓她跑1000米後再來給給我侍餐，因爲那洋她的腳上那特殊的腳汗會很多，她用腳指甲切開牛排，然後用腳趾甲當叉子把肉喂到我嘴裏，經過她腳的牛排也帶著她腳上的氣味了。和飲料時，她就單腳站在桌子上，另壹只腳擡起，將飲料倒在她擡起的腿上，飲料順著她的玉腿，流上她的玉足，帶著她的腳汗，最後流到她腳下那個張開嘴接著的我，飯後，還用腳趾甲幫我剔牙。而洗澡的時候，她躺在浴缸裏，用壹只腳的腳心夾著5厘米高的我，在另壹只腳心上塗滿肥皂，然後兩腳心搓在壹起，算是給我洗澡。

這洋的生活壹直持續到莉莉亞搬過來住，她壹來，銀兒有了她的指導，我頓時失去了地位，經常認她們擺布，打打鬧鬧恩恩愛愛，半年不知不覺過去了。

這年，魔族10萬大軍再次進犯修斯帝國，邊境的斯特魯要塞已經進入壹級預警，但是另人類和魔足雙方都感到不可思議的是，魔族大軍在出發後第三天在壹液之間消失了，十萬大軍，連辎重都消失得幹幹淨淨，又由于他們當晚紮營在沙地上，所以壹點線索都沒留下。事情是怎麽洋的呢，只有三個人知道，其中壹個是我，另外兩個是我的美人們。當我們得知魔族軍隊的方位後，我用傳送術帶著兩位美女來到魔族大營二裏外的壹個禿山上，在這裏可以清楚地觀察軍營裏的壹舉壹動，魔族的軍營因爲周圍有山地，要防著騎兵偷襲，在營周圍挖掘了壹條看起來很深的壕鈎，但他們不知道的是，正是這條鈎成爲了他們壹部分人的黃泉。突然，銀兒說她內急，就要就地解抉，莉莉亞說等壹下，她讓我將壹顆藥丸傳送到銀兒的膀胱中，估計著藥丸溶解了，然後叫我按照她的吩咐，開了壹對空間門，壹個正好覆蓋了銀兒的小巧的蜜穴，另壹個開在魔族軍營的防禦鈎裏，當然，要大許多，不過魔族的軍人沒發現。“現在妳可以方便了，能尿多少就尿多少。”莉莉亞開口。銀兒如蒙大赦，長舒口氣，面色陶醉而輕松，兩腿見沒有壹滴金黃流過，但二裏外魔族軍營外的壕鈎裏卻已經波光粼粼了。按照計劃，莉莉亞把銀兒的眼睛蒙住，然後把她推進了我開的空間門，銀兒再出來時，已經在魔族大營中了，而且身高有200米，她以爲我們和她捉謎藏，于是四處摸索，而我則用傳音的方法誘導她在營盤裏轉來轉去，頓時，大營裏亂作壹團，因爲銀兒降落在大營的中央，而他們的將軍很不巧呆在中央的營帳裏，所以他是被銀兒巨大而尖銳的腳指甲穿透帳篷直接刺穿了，而不知情的銀兒只覺得腳上有什麽東西挂著，于是蹭掉之後踩進土裏，這下恐怕大軍連自己將軍的屎體都找不到了。因爲無人號令，軍營的士兵很長時間處于混亂狀態，等他們看清楚來自上方的威脅，紛紛逃往營外時，才發現那條鈎——臊氣沖天的鈎，許多士兵抵擋不了銀兒尿液中媚惑的氣味，紛紛跳下去，但是壹入水，就再也沒上來，後面的人見狀想退後，但是巨大的銀兒在我的操縱下不斷將軍營裏的士兵趕出來，于是外圍的士兵退無可退，紛紛被擠如鈎中。

幾個時辰後，大營中再無跑動的軍人，我把銀兒傳送回來，她已經暈頭轉向了。接下來是莉莉亞去打掃戰場，200米高的莉莉亞用魚網從壕鈎中打撈出壹尊尊石像——原來前面她在銀兒的尿中下了石化術的藥劑。她把十萬大軍的石像堆在壹起，我把他們全部變成沙子大小，裝進壹個口袋，然後莉莉亞將解除石化藥劑倒入，讓這些沙子們重新變成活的。最後的工作，由1000米高的莉莉亞來完成，她在沙子中刨開壹個大坑，將魔族的營盤全部掩埋，最後鋪平沙子，就好象從來沒有什麽魔族軍隊在這裏駐紮過……

幾分鍾後，三個人出現在魔族的城市中，所有市民都已經入睡了，他們潛入壹個大護人家，壹個15歲的魔族小女孩正在自己的臥室中熟睡，白皙的皮膚在在月光下像玉石壹般，這三個人叫醒她，說要給她壹件禮物，並告訴了她使用的方法，小女孩接過壹個袋子，隨後三個神秘的人瞬間消失了。只見魔族小女孩劈開雙腿，將袋子的口塞入自己細小的縫，然後把袋子了東西都倒了進去。當晚，臥室裏傳來女孩歡娛的叫聲……

天亮了，我和自己的兩個美女站在斯特魯要塞的城牆上，我感歎：“和平多麽美好啊！”“是啊，”莉莉亞接口道，“終于把十萬大軍都送回他們自己的地盤了，幾年之內他們都不會攻來了。”

與此同時，魔族的壹個富翁正在自己女兒的床前鐵青著臉，因爲她女兒的床上有壹灘血，但是女兒堅持說自己沒有和人偷情，隨後，醫生的診斷讓富翁大喜過望，女兒的貞潔並沒有失去，但是新的問題來了，這灘血又是哪個人的呢？

其實，不是哪壹個人的血，而是十萬人的血……
