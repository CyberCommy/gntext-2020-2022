# “战斗女友”（9.25更新序章和第一章）

# 1

"致每一位反法西斯战士，不论你们是荣归故里，或者是默默无闻，甚至是牺牲在异国他乡......"

序

柏林，1945  胜利了！伊乌什金从坦克里探出身来。 在满是废墟的街道上，庆祝胜利的士兵和官员拥抱在一起，紧紧相拥，伊乌什金看到这一幕，一想到自己的女友在战争早期就牺牲了，不由得潸然泪下。喜极而泣的伊乌什金从上衣兜里摸出一张泛黄的相片，这是他和他女友的唯一一张合照，是在战争开始前三天拍的。 照片上的女友依旧和牺牲前一样年轻，虽然是黑白的照片，但是在伊乌什金的眼中，她依旧是有色彩的，甚至是栩栩如生的——因为，他只保留了唯一的一张她的照片。"热尼娅...... 我们...... 胜利了。 "伊乌什金在柏林国会大厦的红旗下举着他和热尼娅唯一的合照，"看哪！ 热尼娅！ 我们在柏林，插上了胜利的红旗！ "伊乌什金的语气中难掩激动，对着热尼娅的相片是又亲又吻，一时间让周围共同庆祝胜利的苏军士兵面面相觑。但是他们很快就理解了，随奔向满眼噙满泪水的伊乌什金，和他拥抱在一起。"达瓦里氏，我们获得了胜利！""乌拉————!" 大家不约而同地爆发出了欢呼声，声音震天，似乎是想让去解放天堂与地狱的红军战士们也听到一样。大家拥抱在一起许久，在伊乌什金看来有一个世纪那么长，那么久。庆祝着胜利的苏军官兵，相互亲吻着对方因为战火硝烟而被熏得漆黑的脸颊，政委和士兵们在柏林街头上少有的空地上跳着欢快的士兵之舞，用着他们自己的方式庆祝胜利。5年，我们赢了！1941，全世界注视着我们！1945，全世界依旧注视着我们！攻克柏林的不是以英美为首的西方盟军，而是诞生在严酷的一九一八的苏维埃工农红军！"你总能战胜一切敌人！ 哪怕是凶悍的法西斯匪帮！ ”这会是纳粹德国的投降书生效的前一分钟，不远处依旧还有枪声，炮声，夹杂着远方的沉闷的滚雷的声音，一起传到了在柏林市区庆祝胜利的苏军官兵的耳朵里。但是，很快的，到了柏林时间的中午12点50分，这是投降书上协定的停火时间，德国境内瞬间鸦雀无声。 天空中尚有太阳，但是阳光却因为浓厚的硝烟而在地面上的双方士兵看来显得苍白无力。 德军的士兵想起了自己在法国看到的成建制投降的法军高举的白旗和双臂，而苏军士兵则想起了用于包扎的纱布——是时候治愈自己和周围的人因为战争而留下的伤疤了（不论是生理上的还是心理上的）。伊乌什金的同车组的战友也从坦克里出来了，几人踏着柏林城内被炮弹和炸弹搞得坑坑洼洼的道路走到了一面尚且完整的墙下。 招呼来了一名随军记者，为他们拍下了第一张，也是唯一一张的合照。伊乌什金并没有因为胜利而精神抖擞，反而由于经历了几天彻夜不眠的战斗，让他脆弱的神经一直紧绷着，直到他从战友们那里得到了胜利的消息，他才稍微安心下来。看着满是残垣断壁的街道，他不由得想起了他的家乡——斯大林格勒。 在那里的战斗中，每一条街道，每一栋房子，甚至每一个房间都要和敌人反复争夺。 每一次的争夺，都会让双方丢下几具尸体，战斗极为惨烈。 坦克，战斗机，火炮，在这场战斗中的作用全无，唯一可以依靠的，就是士兵，以及他们手上的步枪冲锋枪，或者机枪手榴弹。这场战斗让他留下了很深的印象，而同在一个师服役的他的女友热尼娅也在这里牺牲——她死在了敌方的狙击手的冷枪之下。从那之后，他的部队被打散，建制被消灭，他被编入了一支新组建的坦克连中，被任命为一个坦克车长——只是因为他在斯大林格勒战役中凭一己之力干掉了4辆德军坦克。为了向德军报仇，他把自己的坦克命名为"战斗女友"号。 在之后的库尔斯克会战之中，他所指挥的"战斗女友"号T-3476坦克击毁了3辆豹式坦克，一辆斐迪南坦克歼击车，还若干的火炮，并且轧死了至少50名德军步兵。 这样的战绩，即使不能活动任何勋章，也足以彪炳史册了。 而不幸的是，他所在的坦克被一辆潜伏着的三号突击炮击毁，全体车组人员只有他一人逃脱，后来他把他的新座驾再度命名为"战斗女友"号...... 期间他的战友和座驾换了四次，终于在第五辆战斗女友号T-3485坦克上，他和第五任车组乘员一起熬到了战争的胜利日。靠着他的战斗女友号战车，伊乌什金缓缓地坐在坦克的首上装甲的备用履带上。 体力不支的他又渐渐躺下，背部完全贴合到了坦克冰冷的且布满弹痕的装甲上，这一瞬间的凉爽让伊乌什金彻底放松下来，仰着头看着天空中的那苍白的太阳。"真好，战争终于结束了......"脸上洋溢着笑容的伊乌什金一边嘴里喃喃道，一边拧下随身携带的，满是凹痕且褪色严重的跟随自己五年的军用水壶中沾满泥土的盖子，将里面从河里取得水一饮而尽，然后随手将之丢弃。 失去了瓶盖的水壶在满是碎石瓦砾的废墟上滚出了一小段距离就卡在了坑坑洼洼的废墟之中，不仔细看的话根本无法辨认。睡着了。 

# 2

伊乌什金睡着了，因为战火而显得肮脏的脸上带着笑容的伊乌什金躺在坦克上睡着了。 他做了个梦，梦到了他的坦克变成了他女友的样子，只是，梦中的她如此高大，即使是有着接近一米八身高的伊乌什金在她面前也只是堪堪够到了她的脚踝而已。 梦中的她是那么的漂亮，洁白的脸蛋，漂亮的连衣裙，一头金黄色的长发，湛蓝色的双瞳，长长的睫毛，还有脸上的微笑...... 让人看一眼就欲罢不能...... 所不同的是，她手上还有一门作为坦克兵的伊乌什金非常眼熟的武器——T-3476装备的F34火炮，那门在他眼中两个人抱起来都费劲的坦克炮，在她手上就好像一把小巧的手枪一样，任凭她差遣。"来吧，同志，跟我一起，把德国佬赶回老家去！" 她发出了富有磁性同时又相当具有号召力的声音，伸出手来，似乎是邀请着伊乌什金上去。 而伊乌什金也像是听到了塞壬的歌声的水手一样，想也没想，就义无反顾地向着她巨大的手掌走去了。梦中的她是那样的强大，德军不论是步兵还是坦克，甚至是天上呼啸而来的战斗轰炸机也不能伤她分毫，这和现实中弱不禁风的热尼娅相去甚远。 在伊乌什金看来远在天边的德军阵地，在她看来不过只是十几步的距离，德军一直以来引以为傲的虎式，豹式坦克，统统在愤怒的热尼娅的脚下变成一片片的废铁，德军的防线在热尼娅的脚下化成了一片火海，惊慌失措逃命的德军士兵也纷纷在热尼娅的巨足之下变成毫无生命力的肉饼。......梦中的热尼娅，在伊乌什金看来是那么的勇，那么猛。 她的双眼中似乎有说不尽的怒火，那是她对侵略者的无尽的憎恨。 每一次下踏，都会有至少一个排的德军在她的粗糙且带有锋利的花纹的鞋底下被碾碎，蹦出的尸块和血迹飞出很远，让本就惨烈的战场变得宛如地狱一样。 即使是坦克，在热尼娅的足下也如同玩具一样脆弱不堪，哪怕是让苏军谈之色变的虎式坦克，也在热尼娅的脚下发出了可怕的金属扭曲的声音，呻吟着随着热尼娅的脚逐渐下降高度而被压扁，最终竟然变的像二维平面一样，被热尼娅的体重狠狠地压在了地里和大地融为了一体。伊乌什金在热尼娅肩上坐着，并且用热尼娅柔顺的金发把自己的腰部捆得牢牢的，而就在他准备再度坐稳，看着热尼娅如入无人之境一样冲散德军的阵地，粉碎德军进攻的时候，伊乌什金脸上浮现了自从战争开始以来就没在他脸上看到过的满足的笑。突然，伊乌什金感觉缠在他腰间的头发丝逐渐从自己手指般粗细变成了手腕粗细，而且他再也不能坐在热尼娅的肩膀上，而是被他眼中的，手腕一样粗细的热尼娅的头发吊在半空中，随着热尼娅前进所产生的的风在热尼娅的肩膀上随风摆动，像一个热尼娅耳边的挂坠。伊乌什金很清楚，毫无疑问地，热尼娅又变大了，她现在身上背负着越来越多牺牲的苏军将士的英魂，还有那些一直和坦克兵不离不弃的，战车的英魂，此时都背负在热尼娅的身上，等着热尼娅向着侵略他们家园，屠杀他们的家人和同胞的纳粹法西斯德军复仇。入夜，热尼娅身先士卒，把阵地上的德军全数赶走（没能赶走的都被热尼娅像臭虫一样踩死在脚下），之后后续的苏军立即巩固了阵地，在热尼娅留下的鞋印之中构筑了新的阵地，严阵以待德军所有可能的反扑。 而热尼娅，则靠在附近唯一的山丘上，半躺卧着休息，和伊乌什金一起。 在夏夜的月光下，回忆着过去的点点滴滴。"伊乌什金......"热尼娅低声说着，但是，即使这是热尼娅几乎听不到的声音，都差点让躺在热尼娅肩膀上的伊乌什金的鼓膜差点报废，"你说，战争什么时候会结束啊...... 我好想，和你结婚啊......""......"伊乌什金没说话，他也不知道战争何时结束。 眼下，部队刚刚跨过苏德边境，向着波兰进发。 而他们，正在波兰的土地上休息。"我不知道，但是我认为应该不远了，我等着那一天......""哈......"热尼娅捂着嘴，但是还是笑出声来了，好在他们所在的部队属于大后方，周围不可能有德军的身影，所以热尼娅即使放声大笑，也不会招致德军的打击。就是退一步讲，就算招来了德军，凭借着热尼娅绝对的体型与力量的优势，也能像踩死蚂蚁臭虫一样让他们全部死在自己的脚下——热尼娅是完全不在乎脚下再多一些德军士兵的尸体或者德军坦克的残骸的。
