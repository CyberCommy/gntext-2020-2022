# 2

就在平凡的一天，在地球的某個洲上，開啟了一道星球的傳送門，一隻巨大的腳掌從裡頭踏出來，降落於地表上，揚起重重灰塵

“轟隆”

一名極為巨大的少女，出現在城市的邊緣，她巨大的乳房有如山脈那麼高，整個城市都被她的雙峰陰影所掩蓋住，山峰的尖端，是兩座粉紅色的高塔，比起城市中的大廈還要來得巨大，人們看不到少女的臉龐，但是從高空的噴射機所帶來的畫面，少女長了一對黑色的牛耳朵，以及一條純白的牛尾，其他地方都跟地球的女孩子一模一樣

「哞~~」

城市中迴盪著乳牛少女說話的聲音，但是在小人的耳裡，就像是隻巨大乳牛的叫聲，低沉的聲音迴盪在整個城市，人們開始驚慌失措，政府開始疏離人群

但巨大的乳牛少女並沒有在意這些事情，只是在原地好奇地左顧右盼，彷彿在找尋些什麼，巨大的腳掌抬了起來，往前走了一步，在毫不知情的狀況下，毀了城市中的熱鬧區，那些渺小的居民，伴隨著尖叫聲與他們最喜歡的地區一同消失

嗡….嗡…

一些聲音吸引起了乳牛少女的注意，幾架客機在空中飛過，城市中的大老闆們，不用與其他平凡的住民們受困在混亂的交通裡，他們在政府還沒提出警報前早就已經登上了自己的飛機，飛機吵雜的嗡嗡聲，使得少女皺起眉頭，黑色的牛耳不自覺的搧著，幾架客機在不經意的舉動下，被當成蚊子給搧去，煩躁的白色牛尾，在空中揮舞著，將一架接著一架的客機打落

咕嚕嚕嚕~
「哞…」
低沉響亮的聲音迴盪在整座城市中，少女面有難色地看向地面，發出了虛弱的聲音，這是提醒他該進食的生理反應，少女俯下身子，在灰色的城市中爬行著，她面有難色地看著灰白相間的城市，四處找尋著她所能吃的食物

就在一個轉彎處，巨大的少女看到了佈滿綠色的中央公園，儘管上面有許多的黑點，但是餓極了的少女已經忍不住了，低下頭，張開嘴，一口吃下整片中央公園，許多來不及逃跑的小人，就這樣被送進了嘴中

少女反覆咀嚼著，品嘗著嘴中的食物，因為是人工草皮，所以沒有綠草的甘甜味，只有濃厚的金屬味，以及一些奇怪的蟲子們，蟲子在你口中擩動著，少女做了個鬼臉，勉強地將口中的食物嚥了下去，摸了摸肚子，露出難受的表情

當少女再度俯下身，巨大的乳房擠壓著都市，漆黑的柏油路經不起乳房的重量，裂成兩半，一輛滿載疏散的民眾的地下鐵，直直闖入了雙峰之間，整輛地下鐵被包覆在柔軟的巨乳中

「哞!!」

感受到地下鐵的少女喊叫出來，抬起上半身，看著那輛陷入她乳峰中的地下鐵，雪白的臉龐脹紅了起來，雙手支撐住豐碩的乳房，地下鐵經不起乳房的壓迫，終於粉碎成末

就在少女勉強的解決完一餐的同時，胸前那對乳白色的山峰突然間不斷脹大鼓起，少女緊閉著眼睛，雙手抱著那對不斷脹大的乳峰

「哞…哞…哞…」臉頰泛紅的少女擠壓著自己不斷膨脹的乳頭，微張的小口喘息著體內濕熱的氣體，發出嬌嫩的聲音

雙乳已經脹大到少女無法抱起的地步了，爆乳掙脫少女手掌的束腹，純白濃稠的乳汁隨之湧出，巨大的乳汁從天而降，淹沒了鄰近的另外一座城市，原本熱鬧的城市，變成了一攤濃稠的液體

「哞…哞…哞…」心滿意足的少女發出了幾聲嬌喘，便放軟自己疲憊的身軀，巨大的臀部坐臥而下，疏散的居民全數消失在少女的屁股底下，30萬平方公里的土地，一轉眼全數消滅
