# 【静的超超超giga】吟游诗人系列之圣战

作者：静儿_

# 1

“很久很久以前，人类还是愚昧的……”

“大叔！愚昧是什么意思呢？”一个坐在地上认真聆听的女孩打岔道。

男生丝毫没有因为故事被打断而恼怒，也没有被称为大叔而有异样的表现，而是微笑着解答道，“就是笨吧，什么都不了解却非常自大呢。”

停顿了一下，见女孩没有其他问题了，男生又恢复了那不快不慢的语调，“那个时候，居住在女神大人右脚小脚趾上的人类自以为征服了整个世界，却不知道女神大人的存在，所以，在科技发展到他们可以一窥女神大人的全貌的时候，人类的心理就崩溃了。”

讲到这里，男生停了下来，然后指着一个听故事的男人，问，“你，知道女神大人的伟大吗？”

男人立刻坐直起来，“当然，我们所有人都是为了女神大人而存在的，如果没有女神大人，我们就也不会存在在这个世界上。”

“没错，就是这样。我们作为女神大人身上的寄生虫，其实我们都没有存在的必要，是因为女神大人的恩赐，我们才能苟且地活下来。”说到这里，男生似乎有点激动。

不过，很快又平静下来，“当时呢，人类认为自己这么高贵的物种不可能是一种寄生虫，所以，当时的领袖联合了几千亿人一起反抗女神大人。这其实很可笑，因为这么多人也不过是女神大人一条指纹里寄生的一小部分而已。”

看着聆听的人们露出了惊讶的表情，男生的笑容更加灿烂，“其实女神大人到底有多大，没有人知道，据说，一个星球……啊，你们没见过星球吧，那是一种让人类居住的东西，圆形，直径在几万公里之内，一般可以住个百八十亿人，一个星球在女神大人的指纹里，就像峡谷中的一粒沙子。”

男生眼神里流露出了短暂的迷茫，那是一种叫做思念的表情，“可惜，我很久没见过星球了，女神大人身体的引力根本不是那种东西能抗衡的……”

“总之，这些人发动了被他们自称为‘圣战’的行动，然后，就这样开始对女神大人进行攻击……”

“攻击？”

“是的，就是攻击，不停地发射导弹和各种武器，因为都是寄生在女神大人身上，所以只对地面拼命地攻击。”

“结果呢？”一开始发问的女孩又打岔道。

“三年之后，人类还是没有检测到女神大人的任何异常行为，甚至有人怀疑，女神大人其实并不是生命体。但是，事实是女神大人根本没有感觉到有人在攻击她，所以自然就没有反应了。”

“然后呢？”

“然后，圣战一直持续下去，人类相信，只要不停地攻击，早晚有一天能够打倒女神大人的，这样的圣战持续了三千年，在这个过程当中，人类所有的资源都用来制造武器，武器的发展程度几乎是日新月异，但是其他方面却没有任何进展，直到三千年后。”

说到这里，男生又停顿了一下，然后说，“也许是感受到了瘙痒，也可能是单纯地无聊，女神大人用手指碰了一下脚趾。”

然后，男生站了起来，收起了水杯和折叠椅，然后离开了。

大家还沉浸在故事中，唯独那个女孩追了上来，“这个……后来呢？”

“没有后来了，如果你能想象整个世界崩塌的样子，那就是当时发动圣战的人类的心情，无可抵挡的绝望。”

“大叔！你是什么人呢？”

“我？”男生回过头来，笑了一下，“我只是一个吟游诗人而已。”
