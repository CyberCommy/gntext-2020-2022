现在，我正被两个山一样的巨大少女注视着。
为什么、会变成这个样子呢。

当时我正在做外勤工作。
这个季节做外勤工作是痛苦的。直射的太阳光、那桑一样的温度，对穿着整齐的西装的身体来说都是强大的敌人。
我早已将西服上衣脱了夹在左腋下，右手不断用手巾擦拭着汗水继续步行的状态了。这一带的住宅区没有便利店、咖啡店这样的东西、也没有能够在树荫下乘凉的公园。

至少去车站的话......

这样一边想着一边走的时候，忽然和一个女孩子擦肩而过。她有着黑色的头发，神采奕奕地樣子非常相稱。
相貌也相当不错啊。擦肩而过的时候，目光不知不觉地跟随着，有點失禮的程度。

相当可爱的孩子呐......

这样想着，视线一边看着前方一边继续走着的时候
呼啦...

意识越来越远......

醒过来的时候，我呆在巨大的房间里的地板上，被两个坐在地板上，看上去就像山一样巨大的女孩子俯视着。

突然注意到，其中一個女孩子就是剛才在城鎮里遇到的女孩子。
變得这么巨大，那種可愛也意義不明了啊。

到底發生了什麽事情，完全不能集中精神去思考。
但是我明白的是，我的周圍有幾個和我相同情況的人仰望着。我們都無法理解站在兩個坐在地板上的巨大少女之間這樣的現實。

其中一個有长长双马尾的巨大少女發出了聲音，但是那個聲音太大，我們都忍不住捂住耳朵悲鳴起來。

明日香  『是呀。到這裏來的路上順路捉到的。』

那個時候擦肩而過的女孩子說。

她的聲音太打让人痛苦，但是從那個發言我明白了，我是因為她才會處於這樣的狀態。

小春  『可以使用了嗎？』

短發女孩呆呆地說。從巨大的聲音的內容可以得出，我們似乎是被縮小了，真是難以置信。
這樣的事情是有可能的吧，因為這樣的疑問已經靠自身的實際体验得出了答案。

應該怎麽辦，思考這樣的問題應該是之後的事情了。現在應該是用盡一切辦法打破這難以置信現狀的時候吧。

但是，应该怎么做......

一个惊悚的阴险的笑容。

能把房子都抓起来的巨大的手逼迫过来。看上去漫不经心的样子，但是对于我们来说太快了，没有一个人能逃脱，都被巨大的手指抓到了空中。
如同一个人身高那么粗的手指，被抓到之后，根本无法逃脱。

简直是地震。

晃动停止后，因为身体疼痛不住在地面打滚，当脸朝向前方的时候，一双巨大的裸足踩了下来。

那个双马尾少女抓着一个人举到面前一边观察一边呵呵地笑着。

小春  『那么就请多多关照咯。』

被夹在趾缝里的人已经完全看不见了，他的身体全部埋在了趾缝里。

还在思考的时候，手再一次向我们伸来，又有别的人被抓了上去，然后被放在了其他脚趾之间。
注意到了少女行为的人们纷纷躲避少女的手指，可是仍然被连续夹住放入趾缝。

就像巨大双马尾少女的巨大发言那样，这次是左脚的行动，和右脚一样站立，碰的一声踩在了地板上。

这实在是太巨大了我根本不清楚自己在哪里，也许是食趾和中趾的趾缝里。
和我身高一样长的脚趾把从左右把我结实得夹住，无法动弹。约束着自己的巨大脚趾传来可怕的力量让我的脸庞都扭曲起来。

想当强烈的酸臭。因为太恶劣以至于感觉时间就像静止了一样。

包含我自己在内的八个人被困在了巨大少女的脚趾缝中。
巨大少女像体育坐那样坐着。我承受着巨大脚趾的压迫和难受的臭味。而上方，巨大少女双手靠在膝盖上，同时下巴压在手上饶有兴趣地俯视着这一切。

没有被困在脚趾缝中的人们发出了悲鸣并且开始逃亡。

明日香  『哎呀。』

咚！！！！

还有一个人，那个短发的巨大女孩，和双马尾少女一样把我们夹在中间，双腿八字张开，全长70--80米左右的巨大的大腿让人们无路可逃令人绝望。

缩小被捉来的人们，被两个少女的身体完全困住，在压倒性的巨大力量面前，没有人再去思考怎么逃脱了。
突然，夹住我们巨大的脚急速得上升。
一瞬间，我们遭受到了急剧的上身和下降，身体不断翻滚。
双马尾的巨大少女脚后跟着地抬起了脚尖，左右脚尖在空中上下摆动。
虽然只是这样很平常的动作，但是在脚趾缝中的我们来说确是令人抓狂的运动。身体快速得在上下将近10米的长度翻滚着。
身体被摇晃着，脑袋都被摇晕。这种忽上忽下的感觉实在是不好受。伴随着脚趾缝中的气味，这一切快要让人恶心到吐出来。但是再这样的摇晃中，连呕吐都做不到。

我们经历着至今为止最激烈的地震，但是对少女来说是一件很寻常的事情，她也不会在乎我们的感受。

明日香 『过一会。臭味正在被吸收，大概还需要一分钟左右就可以结束吧。还有啊，小人一定会死掉吧，吸收了那么强烈臭味。』
小春  『是吗。那么就好好欺负到底吧~』

听到双马尾少女欢快的声音，我全身的血液都快凝固了。
足部上下高速运动停止了。
紧接着——

咯吱。

咯吱...
脚趾握得更紧了。我和其他一样被困在脚趾缝里的人发出了就行临死前痛苦的声音。

『啊啊啊啊啊啊啊啊！』
『不要啊！！！！！！』

身体受到巨大的压迫无法呼吸。身体中所有的东西都快被巨大脚趾的带来的压力挤出。

我们这样悲惨的叫声，让地板上的其他小人感到恐怖。

巨大少女声音响起的同时，握紧我们的脚趾，又一次开始了上下运动。左右巨大脚趾传来强有力的压力，巨大脚趾上下高速的运动，这一切都让我们痛苦万分。

轰！
惊人的轰击让我们全身剧痛。就好像是两辆高速行驶的汽车撞在了一起，非常的有气势。
想起来了，这是巨大少女左右脚尖碰撞引起的。

轰轰！！ 轰轰！！

不断碰撞的脚尖。对我们来说就像是不断发生的车祸一样。与之前不同，这样的冲击给我们带来了直接的伤害。每次脚尖的相互碰撞，全身就受到了非常强大的冲击，似乎骨头都要被震碎。
在这样不断碰撞的脚尖之中，我们的身体蜷缩起来。我们就像是在不断碰撞的脚趾之间的一团垃圾一样，被随意的玩弄着。对我们来说，这巨大的脚趾是最残酷的刑具。

在乱动的脚趾之间，悲鸣声此起彼伏。
我也差不多快到极限了。
就在这个时候。

巨大少女这样说着，松开了脚趾。
我们终于从来个恐怖的牢笼中逃脱出来。咚咚地落在了地板上面。尽管身体的疼痛让我的脸扭曲起来，但是能从巨大的脚趾之间逃脱出来，这样的事情也算不上什么了。

大约有三个人被手指捏住。巨大手指的面积容下几个人的数量绰绰有余。

『...哈...』

睁开眼睛，我靠在公园的巨大树荫下的长椅上，是一个非常凉爽的很普通的公园。没有奇怪的地方，也没有在巨大的房间里。
检查了自己的身体，没有异常，也没有痛处。
当然了，那两个巨大少女也不存在。

『...梦吗...』

脑中浮现出了两个巨大的少女的身影。
真是度过了一段相当怪异的时间啊。
难道说这个就是大家白天都会做的白日梦嘛。

视线不经意落在了手表上，工作时间也没有过去。
要继续工作的我，扭了扭脖子站起来，离开了公园。

男子看不到树荫里的两人。

小春  『这样好吗？就这样消除了记忆让对方回去。』
明日香  『就这样吧。留下来令人满意的回忆，我也很开心。下次给我的鞋子吸汗吧~~』
小春  『啊，真狡猾啊。』
明日香  『啊哈哈，马马虎虎啦....对了，他们怎么样了？』

明日香把视线停留在了小春的鞋子上。

小春  『嗯--好像已经不动了。大概筋疲力尽了吧。』

小春也看向自己左右晃动的脚尖。

明日香  『嘛--就是这样的啊。小春的袜子和鞋子里会非常闷热吧。虽然对小春来说只是从家里走到这里的程度。但是对脚趾缝中的小人来说，应该如同变天了吧。』

小春  『就是这样。』

小春笑着用脚尖拍打着地面发出了咚咚的声音。

除了那个男性以外的小人都囚禁在了小春两脚的脚趾间。一个脚趾缝夹住几个人的状态。如果只是单纯的用脚趾夹住也没有什么问题，但是那双脚被黑色的过膝袜包裹着，还穿上了鞋子。鞋子里的温度和湿度已经达到了不可想象的程度吧。

除此之外，小春的移动对他们来说也是如同地狱一般的拷问。对于被缩小了一百倍的他们来说，悠闲的步行也是相当大的灾难。
小春行走的时速是四公里。但是对于脚趾之间的他们来说就是时速400公里的速度。而且这个速度不是平均的，而是以惊人的冲击力砸到地面一瞬间停止，然后马上有向前踢出去，以此重复着。
脚踏在地面的冲击，脚尖踢出的一瞬间，对他们来说如同是从高处坠落一样。超高速得突然下降，突然的停止，以及火箭似得超加速前进。
小春的不行，对小春来说只是简单的行走，但是对关在鞋子里发出凄凉惨叫的小人们来说，是闻风丧胆的可怕拷问。

小春  『小人除臭剂效果持续多久呢？』
明日香  『大概三天吧，长的话会持续一个星期。』
小春  『这个样子啊。那么到效果结束之前，我绝对不会浪费的。』

小春俯视着自己的鞋子库库的笑声。
外面是看不出什么。不过现在小春动动脚趾的话，脚趾之间的人们就会转圈。在巨大的脚趾的动作下，闷热的空间中人们对这样的举动发出了尖叫和哭泣声。
这样想着，更想欺负下去了。

明日香  『口渴呢，难得出来，我们去哪里休息下吧？』
小春  『要去车站前的心蛋糕店嘛。』

两人这样说着离开了公园。
小春依旧走着。那鞋子中不断的尖叫又继续想起。只是，那个声音连小春在内的所有人都听不到。
