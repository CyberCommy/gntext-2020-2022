# 1

《女神的乳房》

“大家好，初次见面。”

突然，全世界的人们的头脑中听到了声音。

不知为何，无论哪个国家的人都能理解那个内容。

然后，此时能看到宇宙的人，发现宇宙中有一个巨大的少女。

一位披着长长黑发、赤身裸体的巨大少女在地球前荡漾。

那个身高大概有160万公里吧。是人类的十亿倍大。

地球看起来有1cm大小的球。

“嗯，实际上再过50亿年大家所在的星球就会毁灭。”

在宇宙中少女一说话，脑子里就听到了声音。

人们呆呆地听着那个。

地球的寿命还有50亿年。

但是比起50亿年前的事，现在地球眼前的10亿倍巨大的少女更是个问题。

“所以我来保护大家。来，上车吧”

说起来，少女的胸部比那颗木星还巨大。

直径是地球的10倍以上大小的乳房。

从夜晚的地球上，可以看到头顶上的星空染上了粉色。

巨大乳房的巨大乳头。地球漂浮在少女左边乳房的正面。

即使是突然跳出乳头的乳头也和地球差不多大。

当巨大的乳头被靠近的时候，地球上的所有人都开始从浮在空中的地球上飞出来，向少女的乳头移动。

70亿人全部移居到了超巨大少女行星大小的乳头上。

少女微笑着确认了那个星球上所有的人都移动到了自己的乳头上。

“那么这边的星星已经用过了，要弄碎了。”

说起来少女把手伸向了漂浮在胸前的蓝色小星星。

人们从那个少女的乳头这一粉红色的大地上仰望宇宙，看到了巨大的手指将漂浮在那里的地球捏得粉碎。

一想到像是轻轻地摘取地球一样巨大的手指将地球摘除，地球就在手指之间一下子被击溃了。

人们在乳头上看到了笑容少女超巨大的手指相互摩擦，地球完全被磨碎的样子。

少女气喘吁吁地飞到手指上粉碎的地球上，俯视着他们所在的左侧乳头。

“马上就要找大家住的星星了。我觉得还不到5千亿年，请稍等一下”

微笑着说的少女就这样从太阳系飞走了。

人们不得已在少女的乳头上生活了数千亿年
