# 1

随笔区我投不进去文了。。只能暂时在这里发
今天，大家要学习小学魔法课程的最后一节课，那就是把物体缩小的能力。
萌萌眨着水汪汪的双眼，没有人知道这个外表娇小可爱的小女孩正想着复仇
眼熟的她，就在刚刚向心仪的男生表白了，
"你好怪哦，大夏天穿着长筒靴，你的脚很臭不敢穿凉鞋吗？哈哈哈哈"
那个男生上下扫视了她几下，嘲讽的这样说着
说话的男生也许无心，却正中了事实
萌萌的脚总是会出汗。而且气味很不好
不过在夏天她这样捂着，气味已经恐怖了
这正是萌萌的痛处
缩小的能力学会了后，在愤怒中，魔力被无限放大
整个星球都被缩小，如同鸡蛋一样大
虚空宇宙中的幼女，把脚从鞋里抽出，白色的丝袜，完全发黑的袜底和袜尖，真空中没有空气，气味在积蓄着
黑色的汗滴悬在袜底，粘稠而浓黑
萌萌把袜子撑开，套住了整个星球
笑容终于恢复在了小女孩的脸上，多么美好的天真无邪的笑容
星球的地面，一只黑白相间的巨大的丝袜笼罩了天空，100%湿度，以及恐怖的气味
丝袜轻轻的笼罩着大气层，实际上基本没有接触大地。只有通过大气层传导着，一个可爱小女生脚的味道。
这一天，星球上的无数人因此丧命，无一生还。
