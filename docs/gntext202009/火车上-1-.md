# 火车上(1)

作者：夢也千尋

TID：2101

 

# 1

放寒假了,河大的小敏和她的男朋友也随着春运大潮踏上了南下的列车.只是他们的旅途生活并不好过,车上可以说是无立锥之地,尽管他们有座位,但是基本上她们属于不能动弹的境地,只能在原地一直坐着.哪怕只是想挪动几米远的距离上趟厕所都很难.地上到处都是人,有的在蹲着,有的躺着,有的站着,总之你如果想要动地方,就一定得小心脚下,因为,地上都是人,不小心就会踩到正在熟睡的某人.

小敏依偎在男朋友阿刚的怀里,她浅灰色的牛在裤,脚上穿一双白色的休闲鞋,忽闪的大眼睛,美丽的小嘴唇,只是头发和皮肤显得比较干枯,这样的环境中,再水嫩的皮肤也会失去光泽的.

火车行驶了六个多小时了,小敏喝了好多水,因为实在太热了,身体里的水分蒸发很快.也因为现在的火车上任何人想挪动一步都很难,但是,现在小敏必须得动了,她必须得上趟厕所.但是看着眼前的人堆,她又很为难,她知道,如果她要动的话会很艰难,并且,这人堆当中大部分是民工,也有一部分学生,一定会有人趁机吃自己豆腐的.小敏是一个很传统,很内向的女生,想到这里,小敏感到很怕.她扭头看自己的男朋友,希望他能够帮助自己,但是,阿刚睡的很熟,小敏实在不忍心把他吵醒.于是小敏站了起来,她要自己走过去,管不了那么多了,她没有别的办法.她使劲的推开别人,因为靠身体去挤她会很吃亏的,当中不知道有多少人正巴不得她来挤他们呢.一个一个的推开别人,但是还是有的人她根本就推不动,她娇小的身躯在满身臭汗的民工堆里面穿梭着,有很多人故意当住他的路,好让自己的身体能和她亲密接触一下.小敏走啊走,大概走了有二十多分钟,终于走到车厢连接处了,她长舒了一口气,可是她抬头一看,把自己吓一跳,车厢连接处的人可以说已经到了饱和了,没有一丝空隙,这种场景,别说小敏,就是一只蚂蚱也不可能蹦过去.小敏绝望了,她长途跋涉走了那么长时间,难道再走回去吗?

小敏身体已经支撑不住了,如果再不上卫生间可能要出事情的,这时候小敏想起了自己小时侯拥有的一种超能力,就是可以缩小和放大任何生物,包括人.因为在她七岁的时候她顽皮的把自己家的一只养了八年的小狗缩小.缩小到只有蟑螂那么大,她只是感觉好玩,并没有想伤害它,她从小就是个很善良的小女孩.但是惨剧还是发生了,小敏上学的时候忘记把小狗复原了,她每次都记得的,只有这次忘掉了.母亲下班回家看到地上有一只白色的虫子,以为是毛毛虫,就一脚把它踩扁了.后来小敏知道之后哭了半个月,是她害死了那只和她一块长大的小狗,她很自责.家人都以为小狗失踪了,而小敏知道它被妈妈踩死了,她到垃圾桶里找到了它的实体,把它复原然后埋了.从那以后小敏就再也没有用过自己的超能力,她想一辈子都不在用这种能力了,她有时候都害怕自己的这种能力.

但是现在,眼前的这种情况,让她不由自主的想到了自己的这种超能力,她想,也许她用一下也不会伤害到任何人,这个时候的女生是那么的自私,那么的残酷.小敏从前就连一只蚂蚁的生命都很当回事的,现在居然想拿几十号人的生命开玩笑.但是没办法,小敏的身体实在受不了了,这个时候我们又怎么能奢望她作出多么高尚的决定呢.小敏想"我只把他们变小两分钟,两分钟而已,我上完厕所出来一切都会恢复原貌的.不会发生什么事情的",她这样不停的安慰着自己.终于,她默念了她的咒语,突然,她的眼前变的那么豁亮,车厢连接处突然那么空旷,这时车厢里的人们大部分都是在熟睡或者闭目养神,没有人注意到三十多口子人不见了,小敏把他们缩小到只有蚂蚁般大小,所以没有人会注意到的.

而被缩小的这些人里面,大部分都是民工,他们感觉到了周围情况的变化,可能他们每个人都以为自己在做梦而已.只有四五个大学生意识到了他们被缩小了,而且就是自己头顶上那么漂亮可爱的小女生做的,因为这个小女生显得很慌张,而且嘴里在不停的说:对不起,对不起,大家忍一下,我没有想伤害你们.然后小敏轻轻的惦着脚从他们身上跨过去,生怕会踩到谁.小敏轻松的走到了厕所门口,她推开了门,哇!!!吓了她一跳,厕所里居然还有人,并且还是一家三口,妈妈抱着六七岁的小姑娘,爸爸在不停的抽烟,边抽边抱怨这破火车这么慢.火车上的人太多了,连厕所都站满了人,小敏很无奈,但是她这时候顾不上那么多了,她一咬牙,把这一家三口也给缩小了,只是没有外面那几十人那么小,只把他们缩小到有十几厘米那么大,因为小敏害怕如果把他们缩小到蚂蚁那么小,会被厕所里的水,或者尿淹死的,这个破厕所地面上积了好多水.
