# 1

“女神大人和那位玩游戏的时候，就是我们大多数人的末日了。”戴着眼镜的男生推了一下镜框，“当然，这种游戏不会经常发生，历史记载中也只有三次而已。其他的时候，我们可以想象，游戏的破坏力让所有人都灭亡了，因此就没人能活下来讲述当时的情景。”

很安静，几百个听众坐在大礼堂中，目不转睛地看着拿着麦克风的斯文男生。

“每次游戏，女神大人都要征集很多很多人类，这些工作都是由女王负责的。”

“女王又是什么？”一个坐在前排，穿着名牌时装的男人问。

“这个问题你可以去参考上一篇故事，作者说写出来像是凑字数，[点我查看上一篇](http://giantessnight.com/gnforum2012/forum.php?mod=viewthread&tid=17703&extra=page%3D1)。”

“哦。”

“七百亿年前，女王突然行动起来，从各个区域征集青年男子和学识丰富的科学家，当时，每个国家都十去其八，甚至引发了一场大灭亡的灾难。而这些被女神大人征集的人，都被送到了沙盘里。”

那个男人再次打断了男生，“什么是沙盘？”

“沙盘是女神大人之上的那一位随手创造的，对于女神大人来说，有一个房间那么大，是巨大无比的游戏机。”

“当时的人被送进去之后，分成了两队，一队由女神大人控制，另一队由那位控制，要进行沙盘模拟对抗。每一队人都被给予了五千年的时间发展科技、培养军队。”说到这里，男生停顿了一下，“五千年的时间，在女神大人眼中也是弹指一挥间，但是那些人类却经过了几百代的更替，在压力之下，他们的发展速度飞快，一些科技比现在还要先进。”

“你怎么知道？”还是那个男人。

“你再打岔信不信我阉了你？”男生终于忍不住了。

“但是最后，无论科技水平多先进，双方开始对抗之后，女神大人还是被打败了。因为那位手下的军队，是喝着那位的汗水成长的，因此身体的强度被极大地改善，总之一个打十个没问题。”

“关键在于，对抗的奖惩。”男生继续说着，“这场对抗的胜利者，可以获得迁徙的权利。”

男生瞪着前排的男人，男人只好放下了举起来的手。

“众所周知，我们都寄生在女神大人的身上，但是也有优劣之分，比如能够生存在女神大人脚上的无疑是幸运儿，但是也有一些非常不适合居住的地方，比如……”男生想了想，“比如腋下。女神大人是非常喜欢运动的，如果居住在女神大人的腋下，就要不停地被汗水冲刷，甚至忍受着可怕的味道而生存。”

“对抗中胜利的一方，就有权选择改变自己的居住地，而失败的一方则被强行迁徙到恶劣的环境中自生自灭。当时失败的一方，由于实在输的太惨，而被女神大人直接扔到了下体，在上厕所的时候被全灭了。”

男生放下麦克风，与此同时，礼堂的大门被撞开，一个女生走进来，高举一把手枪，“我现在怀疑你宣扬动摇政府领导的信息，请你跟我回去接受调查。”

男生一点也不惊讶，站起来离开了礼堂。

“你怎么来了？”走到一个没人的角落，男生问道。

“我来巡视自己的领土，不行么？”我笑着挥挥手，手中的枪就这样消失了，“倒是你，怎么又到我的脚趾上来了？”

“随便走走而已。”男生耸耸肩，“以神魂的形态出现，就不怕被人伤害吗？”

“谁敢？”我轻蔑地笑着，“如果他们知道我就是信仰了一辈子的女神大人，应该是跪下来全部自杀吧？用自己的视线玷污了我。”

“你越来越霸道了。”

“我有这个资本。”我毫不在意，“对了，这次来主要是上次输的不甘心啦，所以打算召集一点人去再和学姐大人一决高下。”

“你也打算用汗水喂养他们了？”

“开什么玩笑，他们可不配。”

“学姐不也是这么做的么？”

我低下头，“我和学姐大人不一样啦……”        

“知道了，你娇生惯养的，看不起人类是吧。”男生摇摇头。

“你敢说我哦？”我瞪了一下眼睛，男生立刻缩小到了我小腿那么高，伸出一只脚，我踩在了男生的头顶。

“何苦呢，你知道我不怕的。”

“没关系，我踩的爽就好了。”我用力踩了下去，男生就这样倒在了地上。“又晕死过去了？没意思。”

“你才晕死了呢，笨猪。”男生一下子跳了起来。

“好了，不玩了，要回去了，在自己的脚趾上和别人聊天真不习惯。”

“再见，笨猪。”

“叫静儿女神大人。”

“静儿。”

“是静儿女神大人。”

“静儿。”

“静儿女神大人。”

“静儿。”

“好吧。”

“哼，刚才不想欺负你，现在还玩不动你吗？”神魂回到了肉体中，我坐下来，翘起二郎腿，然后看着自己的右脚，“是在哪个脚趾缝的哪条纹路里来着？”

看了半天，也分辨不出来了，“那就都去死吧。”

我把脚向前伸出，立刻，一个女仆爬上来，把五根脚趾含进嘴里。

“乖，咲月。”我摸摸她的头，然后闭上了眼睛。
