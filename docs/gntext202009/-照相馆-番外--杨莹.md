# 1

平常的周一，杨莹接到了来自电视台的任务--采访一家照相馆的老板娘。

“又是什么有趣的人呢?”杨莹在车上不禁想到。

杨莹今年大学刚毕业，毕业前一直是大学内公认的笑话，白暂的肌肤，顺畅的长发，以及淡淡的体香，让她成了每个男生梦寐以求的女伴，但是，杨莹始终没谈过恋爱，因为，没人知道，她心中的秘密--杨莹小时候登上过哈利法塔，从此她就迷上了居高临下的感觉，因此她经常坐飞机，就是为了享受一下那种快感。“还是太小了啊，真想用脚搓着地球玩啊。”说罢，杨莹看了一眼脚下，圆润的大脚趾，纤细而又充满肉感的指头，加上省吃俭用买的白色露趾Gucci高跟鞋，露出白白嫩嫩的脚背，这样的脚，就算是比宇宙还大，都会有人喜欢吧。

“到啦，摄影师大哥你别跟过来了，看上面写着男人禁止入内呢。”“好，我在这等你。”摄影师大哥看了一眼杨莹，白色衬衫与白色超短裙。“啊，好像在这片白色里游泳啊，摄影师感叹道。”

杨莹进了照相馆。“你好，杨莹。”一阵悠扬的女声传来，只见一名光脚的女生走了出来，黑色风衣与迷彩超短裤显得娇小而又帅气，杨莹看了一眼那女生的脚，那是一双完全不输给自己的玉足，但是，自己却莫名的有种敬畏感，仿佛，这只脚刚刚踩碎了地球一般。“诶诶诶，你知道我的名字诶。”

“不但知道呢，我还知道你的兴趣爱好等等，是来采访的吧，我们开始吧。”老板娘说道

“诶诶诶，怎么知道的呀！”

“emmm你之前进来过一次，但是刚好遇到我在自拍，然后我变大时你爬上了我的脚趾，我们聊了很多，在我大到能夹爆地球之前，然后我就重置了世界，所以就这样咯。”

“等等，变大！我也可以吗！”

“当然，等采访时，我带你体验，首先，来个最舒服的身高吧，一千倍。”

说罢，二人放大了起来。此时门外，摄影师大哥。。。“噫，什么声音，怎么会有震动?”“诶，那不是杨莹的鞋子吗，杨莹被赶出来了?不对啊！她的脚把房子撑破了，好大啊！啊啊啊啊啊，过来了，我得有所行动。”摄影师大哥麻利的爬上了杨莹的露趾高跟鞋，此时，杨莹的大脚趾已有一栋小房子的大小，而先前载杨莹过来的那辆面包车，以消失在了鞋底，只有小脚趾呢么大。杨莹此时以停止了增长，她发现自己的“小脚”上有一个人“诶，摄影师大哥，记得好好拍摄哟！”说完，杨莹调皮的动了一下大脚趾，摄影师大哥在上面被震得发麻。“好好好，去吧，无人机。”一个不起眼的小黑点飞了出去，记录下了那有小山一般大小的脚，及，如女神一般降临的杨莹与老板娘。

老板娘利索点找到那六百多米高的企业大楼，脚抵着街道，踩在两百多米的“小房子”上，而杨莹，来到了企业对面同样六百多米的电视塔总部坐在了上面，双腿微微内靠，倚在电视塔上，两百多米的小脚也只有这“椅子”的三分一大小。

(采访开始)

杨莹:“请问，您从事这项工作多久了?”

老板娘:“几万年了，我从远古时代就掌握了这种能力，我喜欢与漂亮女孩子一同变大，皇宫里的爱妃啊什么的。”

杨莹:“爱妃?”

老板娘:“对，她们太可爱了，总喜欢变大到皇城只有一个脚指头的大小，然后叫全城人来舔。”

杨莹:“那皇上呢?”

老板娘:“皇上也喜欢啊，所以那些好看的香妃脚踝上都有一个脚环，你别小看它，那东西看起来像是装饰品，其实里面有个宫殿呢，皇上一般就戴里面，知道唐玄宗为什么宠杨玉环吗?因为杨贵妃的足弓，刚好能容纳下神州大地！”

杨莹:“哇，我也行吗?啊不对，那么，你有没有给什么明星拍过照呢?”

老板娘:“获得我认可的女星不多，我目前只帮奥黛丽赫本拍过照”

杨莹:“可是，赫本不是去世好久了吗?”

老板娘:“那个是假的，在她拍完罗马假日后，我就把她变大了，现在我们这个宇宙在她的脚趾缝里，你知道XX盆地吗，那是她离开地球前用脚趾尖点出来的。”

杨莹:“难怪澳大利亚现在是个盆地，人们还说哪里有女性的芬芳”

老板娘:“这个身高有点没意思了，变大点吧”

说罢，老板娘与杨莹便缓缓增大了起来，那两百米的大楼被推倒了，老板娘的脚像压路机一样，直线摧毁着沿路上的一切，而杨莹，大脚趾撞毁了六百多米的电视塔总部，白色Gucci将整个省压在了脚下，老板娘玉足的阴影笼罩了一千多公里外的首都，二人现在已是原来千万倍的大小。此时，杨莹大脚趾上的摄影师大哥:“天哪，那是汗毛吗，跟哈利法塔一样巨大，杨莹皮肤真好啊，变得这么大，还是呢么光滑，看着足弓！能塞下多少城市啊！”

杨莹:“天哪，我现在有多大了！我们这么做不会被军队打吗?”杨莹不安的晃了晃脚，白色Gucci越过了山和大海。

老板娘:“你看看我大脚趾边上的疙瘩。猜猜是什么”

杨莹:“不知道”

老板娘:“是珠峰哟，我们现在可是一千万倍的大小，你的脚可是比一些国家都要大的呢”，老板娘调皮的翘起了大脚趾，“碰！”珠峰变成了一个盆地。

杨莹:“天哪，军队呢?”

老板娘:“你把右脚鞋子脱下来”

杨莹脱下了尊贵的白色露趾高跟鞋，瞬间重心不稳，连退好几步，将许多国家的人变成了足下亡魂，身体一倾，欧洲大陆便被压在了香肩之下，而稍有幸存的地方，也被直径百来米的秀发横扫一空，在杨莹出丑的同时，老板娘用高跟鞋往太平洋一捞，亲手给杨莹两百多公里的“小玉足”穿上。

杨莹:“好尴尬啊，怎么感觉脚下有东西，好舒服啊”

老板娘:“那是日本哦，整个日本都在你脚下呢，富士山在哪足弓哪里呢，看呢，军队毫无还手之力”

杨莹:“这样子啊，那你摄影时有出现过什么尴尬的事吗?”

老板娘:“我记得早些年，我应邀给一个新娘拍婚纱照，结果把人家变大了，原本以为她会超生气，结果还要求要更大，现在人在半人马星系呢，她的丈夫在她的大脚趾上建立了一个帝国呢，还有还有，帮一个学生妹拍写真，不小心把她变大了，结果她强烈要求把她的祖国穿在脚下，出了很多钱，而那个国家，就是亚特兰蒂斯了，并非沉了，而是一直在那个女学生的小皮鞋里。”

杨莹撒了个娇:“那我也行吗?”

老板娘老脸一红:“当然，啊，我开始喜欢上你了，现在，让我结束这一切”说完，老板娘便变成了一兆倍的大小，有半个地球大的杨莹连老板娘的小脚趾都不如。老板娘趾间微微张开，一夹，地球炸裂，但随之恢复了正常。

此时，杨莹在办公楼里发了呆“奇怪，我刚刚不是还有半个地球大吗，噫，怎么突然好想站起来”杨莹站了起来，脱下了一只脚的鞋子，然后目视前方，突然失去了意识，变大了起来“杨莹你在干。。。。。啊啊啊啊！妖怪啊，跑啊”杨莹的头撑破了大楼，越过了山峰，杨莹的脚缓缓扩大将整个zhong国踩在了脚下，没有意识的杨莹走到了太平洋中间，另一只鞋子别扔到了北美大陆，太平洋几千米的海水还不及杨莹的指甲盖厚，全世界都震惊了。此时，杨莹的内心只听见了老板娘的声音“我要去找赫本了，这个宇宙就交给你玩耍了，不过要百年后才能行动哟，放心，你是不老不死的，先让他们崇拜你百年吧！”

过了一年两年，全世界人民发现杨莹好像不动了，便纷纷开始了尝试，先是探险家们，脚趾探险，指甲探险，高跟鞋探险，他们还在足弓内发现了富士山！而且，都已登上杨莹白玉般的脚踝为荣。然后是政府，凿空了高跟鞋的鞋跟，建成了通天大厦，虽然天就是杨莹的后脚跟，并在杨莹右脚的五个脚趾上建立了一个国家，桥将五个脚趾头连接在了一起，这个国家只给有钱人居住，因为这里能享受到杨莹美妙的脚香，美国人在那只扔过去的高跟鞋上建立起来了高原城市，五个脚趾印是热门居住地，而杨莹的左脚，还穿着高跟鞋，开发难度大，但还是在五十年内建成了娱乐圣地--杨莹公园，及联合国总部等等，这些只有一个脚趾盖呢么大，脚背上被弄成了地质公园-地板就是杨莹的脚。而当下最热门的活动，莫过于坐宇宙飞船，去杨莹身体上的个个部位探险--国际空间站建立在了杨莹的胖次附近，人们安居乐业，在杨莹身体上生活的不亦乐乎。

百年后，人们不安的发现，那大陆般的大脚趾，调皮的动了一下，天上，杨莹在微笑。。。。
