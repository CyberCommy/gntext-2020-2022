# 1

没什么营养也没有剧情的压路机短文。
想试着模仿一下十六夜的风格，不知道有没有那种感觉‥

≯≯≯≯≯≯≯≯≯≯≯≯≯≯≯≯≯≯≯≯

伴随着清雅的鼓声响起，繁华的闹市上空，突然如雪花一般飘下洁白的花瓣。
理解了这代表着什么的人群爆发出了惨叫，原本已经拥挤不堪了的街道现在更是变得滞塞不通，人们疯狂的互相推挤，试图逃离这个地方。
但是这一切都已经太迟了。

首先到来的是巨大的侍女们。
身着漂亮的和服的她们，如蝴蝶一般翩翩而来，几乎没有一丝多余的动作，整齐划一的分作了两排。
然而她们的脚步可就没有看起来那么轻盈了，百倍大小的她们，只是纤足落地就足够让这整座城市天摇地晃。洁白足袋包裹的纤纤细足毫不在意的踩塌行进路上的楼房建筑，道路上可怜的人们更是被成片成片踩作了肉泥，里面还混杂着被踩作铁片的汽车。

这仅仅只是灾难来临的前奏而已。
随着天空中通道门的打开，一只几乎可以将这些巨大的需要人们爬上楼顶抬头仰望的侍女们像小虫子一样踩扁在脚下的华美裸足轰然降临，横跨踏平十数条街道，置于分站两侧的侍女们中间。
高楼大厦在这只奢华玉足面前简直就像沙粒一般渺小，就算只是一只精致圆润的足趾都足够简单的压碎大片街区。纤细可爱的足踝处系着一个纯银打造的小巧铃铛，纤足轻动，铃铛就泠泠作响，对于玉足下微菌一般渺小的人类来说，这清脆的铃声就如同宣告毁灭与死亡来临的审判之钟一样。

人们不知道这位少女的一切来历，唯一所能知道的只有侍女们称呼这位少女为『公主』。
身着华美的和服，留着几乎要垂到地上的乌黑长发，发簪上叮叮当当缀满了漂亮的宝石，漆黑的大眼睛乌黑灵动，精致的面容几乎不似现实之物，更像是画中倾画师所有爱意所绘制的尽善尽美的绝美少女。
不过这一切都是人们通过卫星照片所得知的情报。
这位美丽的公主巨大到人们无法以肉眼望及全貌。
连脚趾头的高度都几乎要触碰到云层的她，需要低下头俯视这个微小到可怜的世界。和服长长地下摆遮掩了公主修长的双腿，只露出两只粉妆玉砌一般的雪白玉足，不过这也足够让微尘一样的小人们仰望了。

仅仅只是存在，就足以称之为天灾。
人类用尽一切手段，试图与这位巨大的公主达成对话。可是没有一点成果，打从一开始，公主就没有把脚下这些渺小的人类视为与她对等的存在
这个世界都不过是她的玩具而已。
公主在闲暇时间最喜欢的娱乐，就是以压倒性的巨大来欺负脚下这个微小的世界。

公主开始了她的玩乐。
只是向前走出了一步而已，华美的裸足就踩平了大半座城市。
防空警报在简单的就能遮蔽半座城市的天空足底面前变成了笑话，只要被这阴影覆盖，逃跑就变成了毫无意义的事情。
顶多只是能微调一下自己被公主足底碾碎的位置而已。

公主优雅的前行着。
由于和服的下摆狭窄，她没办法迈出太大的步子。
然而这却又进一步降低了小人们的生还概率。
因为实在是追不上公主过于遥远的步伐，百倍大小的侍女们干脆飞了起来，聚集在公主精致的面庞前。
可爱的女孩子们说说笑笑，就像高中女生们放学回家的光景一般美好，连周围的空气都好像染上了粉色的气息。
然而在公主的脚下，却又是另一副光景。

没有任何东西能阻拦得住公主的脚步，连山都会被她放下的脚随意的踏平。
公主的身体似乎还在无意识中持续的变大着。

一支军队驻扎在了山顶，准备在制高点对公主进行突袭。
然而，随着天空突然变暗，整座山化作了公主优美足印里的碎石。

飞机甚至难以飞到与公主的膝盖平齐的高度。
随着公主的不断接近，飞机上逃难的乘客们透过机窗，逐渐感受到了飞虫看人类的视角是什么样的。
飞机飞行的航速完全逃脱不了公主步行的速度，哪怕公主只是一小步一小步磨蹭的向前走着。
终于，像一粒浮尘一样的客机在公主精细考究的和服布料上爆炸了。
甚至没能给和服的一根线头留下黑色的痕迹。

公主十分享受人类文明建造的高楼大厦在自己足底像沙堆一样崩溃的触感。
哪怕是能防御住十级地震的坚固建筑，在自己遮天蔽日的脚趾面前还是像纸糊的一样乖乖塌掉。
在她原本的世界之中，公主她已经是万人之上的高贵存在。
但是在这个渺小的世界之中，公主她是凌驾于世间万物之上的伟大女神。

不知不觉，公主的玉足已经可以轻而易举的踏平一整座都市了。
她所放下的每一脚，都可以轻松写意的收割数十万人的灵魂。
公主并不怎么在乎这些异世界的小人——谁会没事去在乎地上看不见的微生物呢？她所享受的，是将一整个人类文明轻松葬送在自己脚下的过程。
倘若只是想弄死这些小人们的话，直接变得比星球还大，然后用手指捏爆这个星球，60亿的生命可以一下子葬送在自己的弹指之间。
但是，公主喜欢享受整个星球在逐渐变得越来越大的自己脚底下解体的感觉。

今天的游戏结束的很快。
公主玩的太尽兴，不小心一下子变得太大了。地面在她眼里都变成了球形，漫游世界一圈已经不过是几步路的事情了。
公主的每一脚都在改变着地貌，地球的表面变得凹凸不平，满是要以公里作为最小单位的巨大足印。
渺小的人类差不多已经灭绝了。
只是因为公主的散步，整个星球变成了死球。

“哈啊……”公主打了个满意的哈欠。
在自己的世界处理公文积攒下来的压力，在这个微小的异世界已经释放的差不多了。
是时候该回去了，公主张开手，将小小的侍女们收入手中，打开了回宫的传送门。

公主离开这个世界的一刹那，时间被重置了。
这个世界的时间被拨回到了公主降临之前的时刻，地球还是一个漂亮的蓝色圆球，城市也还没有全部变成峡谷，被踩死的人们依然完好的站在原地，看着天空发着呆。
一切都好像没发生过一样，只是刚才的一切全部都铭刻在了人们的记忆里。

每个月，巨大的公主都会降临于这个世界。
时而带着同样巨大的侍女，时而独自来访，有时穿着典雅的和服，有时穿着漂亮的旗袍。
只不过，每次降临，她都会用自己巨大美丽的身体将这个世界蹂躏成一片死地。

再过几天，玉洁雪白的足底就会再一次遮蔽天空盖在所有人的头顶上吧。
人们没有任何反抗的力量，只能任凭公主将他们当作足底按摩的消耗品。
这已经成为了这个世界的日常。
被巨大的美丽公主踩踏蹂躏的日常。
