# 2

“这个世界是由因果组成，由逻辑连接。我们的职责，就是找到这个最前端的因果，并且理清其中的逻辑关系。为了这个目标，我做这些是值得的。”实验室内，一个年轻人满头大汗，用尽全身的力量说道“为了先辈们的遗志，让我试一次吧！”他的语气，像是请求，也像是命令。如此的低下与谦卑，又如此的坚定，不容任何人质疑。 “你已经迷失了……”中年人则满脸惋惜得看着他“你会毁了一切的……” “不试试怎么知道！”年轻人已经控制不住，泪水和吼声一同喷泄出来。他左手拳头紧握，右手抓着栏杆，暴起青筋。 “让他去吧……”一个须发皆白的长者遗憾地看了看中年人“我们阻止不了他的……年轻人……” “可要是激怒了‘她’，我们全都得……”中年人不甘道“博士，我们……” “让他去！”老者拉住了中年人的手，慢慢往后拽“至少我年轻的时候……也想这样……不做些什么，又怎么能后悔呢……” “可是”中年人还想再争辩什么，看了看老者，摇了摇头，一同走了。<title>3</title> <link href="../Styles/Style.css" type="text/css" rel="stylesheet">

“这下终于完成了……”年轻人坚定且骄傲的目光，诠释着自己的身份——真理研究社最年轻的社员，也是“因缘”工程的总策划者。三年了，他从研究社开始游说，到民间工会、达官贵人，乃至女王陛下，都说了个遍。这些人一开始都不支持他，反而对这件事避之如虎。但他像皮球一样，越打下去，弹起来便越高。终于，那些恶语相向的反对者，一个个的认输，甚至成了他的簇拥。举国之力建造的通天塔、无底洞，终于完成了。他知道，只要在足够的高度和深度同时召唤，必然能召唤出这个世界的主宰和创造者——这样他才能探索和询问这个世界的法则，以找到这个“最终点 ”<title>4</title> <link href="../Styles/Style.css" type="text/css" rel="stylesheet">

“……预备——唤！”他喊了一句，就跪倒下来，脸朝着天。无底洞和通天塔里的人也跪了下来，口中念念有词。接着，旁边的所有人也都跪了下来，期待着主宰的出现——这是主宰隐匿之前最后说的召唤秘法。虽然经年累月之下多数人遗忘，但还是有有心人发现了斑驳在时光中的那一卷典籍。大家紧张到了极点，不仅害怕无功而返，更害怕她的降临——谁也不知道她会做什么。相传，她可是一个暴君。<title>5</title> <link href="../Styles/Style.css" type="text/css" rel="stylesheet">
