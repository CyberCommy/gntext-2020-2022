# 2

肚子里一直有轻轻的碰撞感，赵悦停下刚刚还在敲打键盘的左手，挺直了腰，左手成勺状不轻不重的拍在自己的小腹，肚子里的碰撞感立刻便消失了，赵悦端起咖啡杯，小小的抿了一口咖啡，嘴角带着微微的笑意，她耳朵上的无线蓝牙耳机并没有播放歌曲，而是有一种细小的声音从她的身体里发出来。

她抿着嘴，克制着自己的笑意，左右环顾了一下，看了眼周围还在埋头工作的同事们，肚子里的碰撞感又开始了。

赵悦装作伸懒腰一样，摇晃了一下自己的身体，肚子内便又恢复了安静。

时间倒退回24小时之前。

赵悦是一名普通的OL，今年刚28岁，原本是长发的赵欣因为工作的原因换成了一头干练的短发，虽然已经快三十岁了，但是脸上岁月的痕迹并不重，反而留下了一些成熟女人的韵味。

是夜，九点半，赵悦关上公司的电脑，靠在椅背上，

她晃动着已经有些昏沉的脑袋，漫不经心的打开手机，社交软件上的几个小红点让她有了一些精神，她嘴角带着盈盈笑意，双手捧着手机，快速的打字。

等了一两分钟，看对方没有回复，才站起来把包跨在肩膀上，划动着手机屏幕，漫不经心的走进电梯。

公司外强光闪过，一颗直径十几米的彗星自太空拖着长长的白色拖尾直冲赵悦飞过来，然后突然消失不见。

只留下外面街道上，过往行人的尖叫声。

赵悦没有注意到外面发生了什么，只是像往常一样，发动汽车，一路行驶回家。

赵悦脱下鞋，有些轻轻的酸臭味，她的黑丝长筒袜经过一天的工作已经出了一些脚汗，褪掉黑色长筒袜露出一双白皙丰满的大腿。

手机被随意扔到床上，突然嗡的轻响了一下，赵悦顾不得小腿上还有半截黑丝没有褪下来，胡乱的用右脚踩下去，扑到床上，打开社交软件，噗的一声轻笑出来。

她精致的半张脸埋进被子里，盯着那个带着黑色笑意的头像，喃喃自语。

“好像把你从里面抓出来。”

赵悦说着，右手食指大拇指对着那个黑色笑意的头像做了一个捏的动作。

一个只有赵悦食指三分之一高的小人从手机屏幕里掉了出来。

赵悦惊得从床上一跃而起，从右边滚下去。迅速抓起早上随手扔在床边的拖鞋。小心翼翼的从床边弓着腰，向手机探去。

那个小人被赵悦巨大的体型所带动的被子牵连，他从被子中的一个褶皱处爬上来，跳到床上，观察了一下四周，左手掐着自己的下巴，低着头沉思起来。

相对于只有赵悦三分之一手指高的小人来说，赵悦等于一个身高300余米的巨人，她举着拖鞋，左膝轻轻贴在床上，想看清楚是不是从天花板上掉下来一只蜘蛛。

床铺的瞬间倾斜让毫无防备的小人向后倒去，滚了两圈，撞到被子边，抬头看见遮天蔽日一般的赵悦。

赵悦也同时看清了小人。两个人惊讶的对视了半分钟。

半小时后，赵悦左手轻轻揉着自己的太阳穴，小人站在她右手手心上，他甚至没有一粒瓜子高。

刚才的半小时里，赵悦把小人放在耳边，确认了他的身份，在社交软件上用了各种方法给黑色笑意发消息，全无应答。
