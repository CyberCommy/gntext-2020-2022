# 1

這幾天臨時起意，就把之前寫過的巨大美少女超人王瞳這篇文章做出修改，以前有同好反應過第一次我在舊論壇貼上的這篇文章看起來很亂，應該說沒把文章格式用好，所以翻開這篇文章之後我就先調整好這方面的問題。

　　接下來就花了一些時間把裡頭的一些段落做了修改，看起來應該會順眼些，當然我寫文章的方式還是沒有改變，從巨大少女的觀點來寫加上小人的視野看著巨大少女的一舉一動來寫，所以有時候只是一些劇情或動作時就花了很多文字描述，沒辦法，這就是我的習性。

　　原本打算來說把這篇文章的女主角王瞳改掉，換成別人，例如糖果，但是改掉之後畢竟這邊文章當時就以王瞳為範本所編寫的，突然換掉，會看起來感覺有偷懶的嫌疑，而且現在王瞳年紀也２０幾歲，形象來說不像之前１０多歲的感覺了，但是最後我還是決定沒這麼做，當然沒這麼做也是有其他的因素在，有時間在談這些吧！

　　希望修改後的巨大美少女超人王瞳這篇文章大家會喜歡唷！
