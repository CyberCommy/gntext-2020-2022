# 1

         我叫阿东，是一名自由职业者。大学毕业后，我终于找到了一份合适自己的工作。我打算把父亲留下的老房子租出去。

        今天一个听声音很动听的女孩子联系我，想要来看房子。

        早早的我便起床把屋子和院落收拾的干干净净。

        站在清爽的院落里，这里将要暂时属于别人了。

        就在这时，头脑一阵猛烈的眩晕，我失去了意识。

        模模糊糊的醒来后～我摸了摸自己的脑袋，我是怎么了？怎么突然就晕倒了～

         摸着晕乎乎的脑袋站了起来。

         ……这是！？

         四周的环境熟悉而又陌生，父亲的院落～却又异常的巨大！

        我向前走一步，却没注意到身前的巨坑，我栽了进去。

         小型石砖直接的空隙对于现在的我来说就是一个巨坑！我到底变得有多小！？

        从坑中艰难的爬上来后，我的身上衣服上沾满了坑里残留的湿润泥尘。

        就在这时，地面微微的有些震动。 一个巨大的身影出现在了院落的门口！

        一条白色短衣裙披着一件白色的外套～一个美丽的女孩出现在了门口。

         女孩走进了院落喊了几声“请问有人在家吗？”

          没有人回答。她似乎也不是很着急，自己在院落里参观了起来。

        看起来她似乎很喜欢这里的环境，闻了闻院落里的花朵～脸上露出了淡淡的笑容。（图1，图2）

        她向着院落的中心走来。

        看着她巨大的身影一步步向我走来，我连忙挥舞着双手向她大喊。

        不知是我的呼喊声起到了作用，还是别的什么，总之她注意到了我～而她的脸上绽放出了让我不知为何的美丽微笑。（图3）

         看着她一步步走近，那修长而又巨大的双腿犹如通天巨塔一般，双腿之间一条巨大的内裤遮挡着女孩神圣的地方，她的双脚犹如巨型游轮一般巨大。

        当她巨大的身影离我越来越近时，地面的震动甚至让我有些站不稳。

        我吓坏了，转身没命的奔跑。

        地面的震动忽然停止了，随后头顶的光线突然暗了下来。

        我停了下来抬头看去。

        她庞大的身影蹲在了我的上方，脸上依然挂着让我不知为何的笑容，清澈美丽而又巨大的瞳孔在努力注视着地面渺小的我。

         “喂！！我变小了！帮帮我！”我努力的呼喊。

         “好久没玩虫子了。”

        虫子？我看了下身上，黑乎乎的泥土布满了我的全身，她认为我是一只虫子！

         她巨大的手指出现在了头顶向我袭来。（图4）

         “不～不～不～不！别这样！！”我转身想要远离手指。

        巨大的手指从两侧袭来将我夹在了中间，紧接着猛的上升。

        女孩蹲下捏起了地上的虫子，托着下巴看了看四周（图5），没有人注意到童心未泯的她的这一举动，捏着虫子站了起来，将虫子藏在手指里垂下又四处看了眼（图6）。

        在女孩巨大的手指里，女孩的内裤就在不远处，一股淡淡的特有香气刺激着我的神经。

        女孩捏着虫子继续在院落里走着，走到了小木桥上，她抬起了左腿～将小虫子放在了大腿上，让小虫子在她的大腿上爬~单脚摇晃站立玩耍着(图7）。

         趴在女孩女孩光滑白皙的大腿皮肤上，再加上女孩腿部剧烈的晃动，我顺着皮肤就往下滑。好不容易～才在膝盖处稳住了。

        “哎呀～好脏！”女孩看着自己的腿上，虫子经过的地方留下了一道泥痕。

        她伸出手将虫子拍落到了地面。

        巨大的手指将我挥飞，我从高处追落到了地面。本以为会就这么摔死了，可没想到却只是一阵疼痛。

        我无力的躺在地上看着头顶巨大的女孩。修长美丽的双腿以及可爱的内裤，以这种大小欣赏一个漂亮女孩子的胯下我应该是第一个人了吧。

         她似乎没有注意我了，我要赶紧离开。拖着沉重的身体，我艰难的站起来行走。

        “有点热了，怎么还没人啊？”女孩不满的嘟着嘴。

         她低头看了眼脚边的虫子，准备把不满的情绪发泄在这只倒霉的虫子身上。

          我的头顶再次暗了下来，一股危险的感觉袭来。

         抬头看去～女孩巨大的黑色拖鞋鞋底正在我的头顶压下（图8）。我想要跑，可我双腿却已经不听使唤了。

         女孩踩死了脚下的虫子～并将鞋底横移了一下（图9）。

          虫子已经彻和鞋底德灰尘融为了一体，不复存在。

        “好无聊～去别的地方转转。”女孩带着美丽的微笑转身离开了（图10）。
