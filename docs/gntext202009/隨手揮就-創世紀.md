# 隨手揮就：創世紀

作者：wobuzhidao

# 1

2008年，年的今天。

在一個平行於主物質面的位面，那裡的世界才剛從混沌中誕生，當秩序形成，黑暗和光明就分開而形成了對立的兩極。創世神在做完這一切後感到前所未有的疲憊，於是他從主物質面，也就是我們生活的世界挑選了兩個女孩作為接班人。

這兩個人原本是來自不同高中的校花，但是卻在巧合下走到了同一個世界。她們分別從創世神那裡繼承了光明與黑暗的力量，成為了新一代的神明。

不要以為這個世界的創世神是色鬼，他挑選的繼承人的秉性是符合她們所繼承的力量的，容貌倒在其次，畢竟繼承了神力之後，總會有天人般的容貌。

兩個女孩繼承了神力之後，開始掌管這個世界，光明掌管著人間的希望，美麗與純真，而黑暗庇護著欺騙，犯罪與褻瀆。

本來一切看似可以這樣平靜地下去直到世界末日。但是作為女神的兩個女孩在成神的時候還處於正常人的青春期，又由於在體形上與凡間的黎民有著巨大的差異（女神嘛〜肯定對於普通人來說是GTS上啦） ，無法與人溝通。唯一和自己相似的人又處於對立的陣營，為了維持世界的秩序是不能搞的GL滴。兩個少女心中的寂寞隨著時間而膨脹，終於有一天她們不得不尋找凡人來慰藉她們的心靈。

但是她們對待被選中的人的方式不同，也導致她們在凡人中口碑的不同，於是也就形成了整個社會嚮往光明，而黑暗總是伺機而動的局勢。

光明女神從虔誠的神職人員中挑選自己的僕人，善良的他們通常在死後或者達到了某種境界後被送往光明女神的住所-一般人稱那里為天堂。女神賜予這些人以不朽的生命，用神力賦予他們無盡的活力，而作為代價，女神將他們的"貼身保管" ，以滿足自己的"需求" ，女神用神力將小人們編織在一起，做成了胸衣，內庫，項鍊和鞋，女神穿上這些衣物就能時刻感受到她的自民的親吻與撫摩。還有一些更為幸運的人則被留在女神的嘴裡，陰道中還有菊花瓣裡，為女神提供最直接的服務。這就是凡人所說的天堂的快樂。

黑暗女神也有她自己的做法，不過也因此人們稱她的住所為地獄。她將生前罪大惡極的人帶往她的地盤，在那裡她吸取他們生命的精華，讓他們的身軀變得脆弱不堪，同時又將他們的靈魂禁錮在他們脆弱的肉體上，讓他們即使肉體毀滅了也依然能保持意識。黑暗女神喜歡用這些人來浴足，她將雙腳放進盛滿小人的盆中，然後不斷地研磨，直到裡面再找不到一個完整的身體，她喜歡小人在她腳下掙扎時給她帶來的瘙癢以及小人的身體貼著腳底破碎時帶來的爽快感覺。當然，她也有其他方面的需求，比如她會將小人整把整把地囫圇塞進自己的小穴和菊花瓣，任憑他們在其中互相推搡，掙扎，小人們將她帶入潮後，她又會"過河拆橋" ，

她會用自己蜜穴與菊花的劇烈蠕動來將小人們變成肉泥，這些小人的下場是，靈魂要么隨著早已殘缺的肉體被女神的液體衝出體外，要么就粘在女神的身體中。離開女神身體的小人成為了鋪在地上的爛泥，等待著永世的踐踏，而留在女神體內的小人則成了陰道與股溝中被遺忘的殘渣，痛苦地用自己殘缺的身體被動地為女神服務。

當然，服侍女神的人也不是一成不變的，不論是光明女神還是黑暗女神。不同的是，光明女神的僕人是主動提出要轉生，那麼女神就會讓他回到人間。而黑暗女神則要定期進行"大掃除" ，她一次性讓所有的靈魂獲得解脫，然後再去尋找新的小人，罪惡的靈魂獲得解脫，世人稱之為救贖。

不同的更新方式也有不同的周期，顯然光明女神的周期要長很多，這也是為什麼好人長命百歲，壞人出門撞車。

一切，冥冥中自有定數。
