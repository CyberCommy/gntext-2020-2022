# 舔老姐的内裤

作者：huhuhu

TID：17097

 

# 1

*本帖最後由 huhuhu 於 2014-6-22 00:08 編輯*

我明白.我做了一个非常可怕的决定.很清楚接下来要做的事多么危险.一旦曝光我接下来的人生就彻底的完了.即便没有曝光我想我自己也不可能原谅自己.即便这样.我也已经不打算压抑自己了.

现在已经凌晨九点了.应该不会回来了..从昨晚开始就一直策划着.直到现在内心仍然惴惴不安..我要潜入姐姐的房间.使用姐姐的内裤.丝袜让自己变态的嗜好得到满足..事到如今我也不想否认自己变态了.我的确是个一见到女孩穿着丝袜的美腿就兴奋的变态.想贴着女孩穿的内裤舔她们的小穴.想被女孩征服.想当她们的奴隶.我即便努力也改变不了自己的这些想法.好吧.好吧...确认门窗都锁好后.拉上窗帘.尽管知道没有人会知道.却改不了我胆颤心惊的心情.我讨厌这种感觉.深呼吸一口.我快步走进姐姐的房间.笔直走到那个姐姐专门存放内裤的小抽屉里.每次仅靠路过老姐房间那点时间偷瞄幻想.马上幻想就要变成了现实.胆颤的心情平复了.另一种感觉开始沸腾了.我看起来的确是个不折不扣的大变态.啊..看起来太过于激动了.呼..因为拉开抽屉时用了过度的力量.抽屉被过于迅猛的力量扯出来.老姐的内裤也因此飞出来散了一地.好吧.直接抓了一把内裤.使劲嗅了一回.除了洗衣液的味道外.没有任何味道.更别说想象中的让人欲罢不能的气味.的确是的.又不是刚刚脱下来的.心中顿时觉得很失望.对了.看看能不能从内裤上面找到姐姐的阴毛.我想舔姐姐的阴毛.我开始认真寻找着.我因为这个想法变的极度兴奋.

10分钟后.我开始有些厌烦了.眼睛随意一扫.我看见了一个白色内裤上面有个黑点.是黑色长长的.很细小的东西.我心情大振.

“哼啊哈哈哈”忍不住笑出声了..我把内裤带有阴毛的那一面叠起来含着.舌头不停的触碰那个小黑线.

好异样的感觉.刚刚好像感觉到内裤动了…内裤的确动了.哗的一下.我把内裤立刻吐出来..再确认内裤的确会动后.我吓的不轻.窗帘.门都被关上.拉上.一个阳光没办法进来的昏暗的空间.我想跑了.几秒的时间.内裤变大.变的更像个人的形状.几秒后.一个类似人类女孩的东西出现在原地.

“你终于想改变…”抬起头看到我后突然沉默住了..

头顶着山羊的独角.背后一双蝙蝠的翅膀.腿部一根牛一样长的尾巴扫来扫去.

“救…呜呜呜呜”我想大声呼救.却被她以看不清的速度来到我背后捂上了我的嘴上.我的内心充满了恐惧.

“好了！别喊.不然拔掉你舌头.”虽然她叫我别喊.但现在被她捂住嘴的我想喊也喊不了.

“听好了.我是可以帮你实现愿望的精灵.我没有一点恶意的”的确.她要是想杀我的话不用那么麻烦.遇到那么非现实的事情.我竟然能那么快冷静下来.我自己都忍不住佩服自己起来.

“你要是不喊了就点头.我就会松手.让你说你的愿望.那么你是点头还是摇头”我点头了.这个女精灵的确松开了我的嘴.虽然她自称是精灵.但我总觉得她像恶魔多点..

“那么你说出你的愿望吧.是要很多很多钱吗？还是要很多很多糯米糯米...”家里又不缺钱我要钱干什么？糯米就更奇怪了.唔…许什么愿望好.

“是不是什么愿望都可以？”

“恩.是的”

“那么.请让我变小活在女孩的内裤里.年龄在18岁左右.要很漂亮的那种..”我手舞足蹈说出了自己最变态的想法.第一次吧自己变态的位置暴露给别人.虽然极度羞耻.却又种莫名的快感.很畅爽的感觉.但是看到这个女妖精完全被我的变态愿望吓到了.一脸惊愕的表情.却让我有些悲伤感觉.唉.也是.就连妖精也没办法接受.

“太棒了”突然就高兴的跳起来的妖精真让人弄不明白..她并不讨厌吗？我的心里竟有些高兴.我竟然会为这点小事觉得高兴.呵呵

“你真是解决我的一个大难题.”诶..？我有些糊涂了.

“么么么么!~~~~~~~吱~~~~ 么么么~~~”突然就被这个妖精强吻.实在是措不及手.

“你想变的多小”. 至少给我点害羞的时间再问这种问题啊.第一次被女孩接吻

“那就1毫米吧.”诶.擅自做什么主啊

“喂.你….”好奇妙的感觉.“哇啊..…”..超级巨大的脸.出现在我眼前..

“你现在在我指尖上.现在就带你去见你新家的主人去.嘻嘻.谢谢哟.帮我一个大忙.还有.那个女生长的很漂亮.很漂亮的.你放一百个心吧.”虽然这张脸大的离谱.但的确是刚刚的那个精灵.
