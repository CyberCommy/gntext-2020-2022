# 1

微波荡漾，天地同色，这样美好的景象，也许只有在海上才能见到吧。

尽情驰骋于这片无垠深海中的舰娘们，对此感触良深。这仿佛能包容天地万物的大海，让她们极为享受。然而，随着时间的推移，她们却开始不满足于此。再广袤的海洋，也终有边界，而这边界便是陆地。

遍布海洋各处，形状各异的陆地，大小也有着极大的差别。有些仅仅只是岛屿一样的狭小地块，而有些看上去甚至比海洋还要广袤。有些荒无人烟，仅仅是一片沙漠戈壁，而另外一些，则绿意盎然，高楼林立，一派繁荣。

终于，在获得能走上陆地的能力后，舰娘们又反倒对造出她们的人类感到不满。自然，拥有轻易超越世界上任何生物的强大躯体，又为何要屈服于这些渺小的人类。渐渐地，在舰娘的操纵下，她们的身体变得越来越庞大，有些的身高甚至能达到十几千米之高。

在获得了足够强大的力量之后，舰娘们发动了对人类的战争。

被彻底奴役的人类，就此成为她们的仆从。在掌控了整个蓝星的资源后，拥有无尽寿命的舰娘们，并没有选择就这样一直改造自己的身体，直到撑破地球的界限，最后甚至统治整个宇宙。相反地，她们主动限制了自己的身高，转而运用一种从名为『塞壬』的阵营中缴获的新式舰装。

持有这种舰装，舰娘的身体便能得到不同程度的巨大化。当然，至于巨大化的程度，则要根据每个舰娘各自的相性所决定。有记录的最大舰娘，名为『阿尔及利亚』，最大能达到千米左右的高度，这也巧妙地与世界上存在过的最大舰娘相匹配。

也即是说，千米大小，已经成为公认的最大舰娘尺寸。因而，这种能将舰娘凭空变大的舰装，虽然看上去十分恐怖，但实际上也并没有多大的力量。

结果便是，舰娘们的身体，在一次次的改造中反而越来越小。

平时以不到十米的姿态出现，在需要的时候再进行巨大化，变到合适的尺寸。这样的规则，一直持续到现在。直到那一天，从世界各地城市的上空，开始不断出现空间的裂痕。

被舰娘统治之下的人们，当时还远望着深邃的天空发着呆，他们丝毫没有意识到，一场足以改变世界局势的灾难，正在悄然靠近。

那绝对是不属于这个世界的怪物，有的长着比一般章鱼多上数倍的触手，体型甚至能比肩百米身高的舰娘。还有一些只是单纯地与地球上现有的动物相似，只不过被赋予了前所未有的体型，例如薮猫、猿猴乃至蜥蜴等，都是现今较为常见的怪物。

类似这样的怪物，每隔一段时间就会出现，而一般来说，它们出现的地点都是有着空间裂痕的区域。虽然人的肉眼能明显辨认出那样的裂痕，但以人类的力量，很显然是无法和这些动辄百米身高的怪物对抗的。

虽说统治了整个地球，但舰娘们却并不打算就此抛弃人类。因为，假若这个蓝星上没有了人类的存在，她们舰娘生活的方方面面都会受到极大的影响。与其说是想保护人类，不如说是不想自己的生活受到太大的影响。

就此，一批舰娘被赋予保护城市的职责。她们长久地驻扎在陆地上，呼吸着干燥的空气，同时也帮助人类抵御巨大怪物的侵扰。

以上，便是一般舰娘所了解到的事实。

但实际上，怎么可能这么巧合。不管是偶然发现能让舰娘变大的舰装也好，还是恰好每次都出现在大陆的怪物也好...这一切，都是由一位名为「圣路易斯」的舰娘，一手操纵的。

受到「塞壬」所带来的神秘力量侵蚀，圣路易斯成为了世界上第一个与这种力量接触的舰娘。

能随心所欲地干涉世界，一直以来都是人类与舰娘所追寻的梦想。然而，花了千万年的时间，至今无人能实现那样的梦。

而圣路易斯，则成为了历史上的第一位获得那种力量的舰娘。巧合的是，她最先得到的能力也是能让自己身体变大的能力。趁着任务的空隙，她还曾跑到无人的海域，想要验证自己力量的极限究竟在哪。

可惜，直到身体突破大气，险些将整个地球都摧毁掉的时候，圣路易斯依旧没有找到答案。

也许只是单纯地出于好奇，圣路易斯制造出能够量产的这种舰装，并让其广泛地运用于每位舰娘的身上。

当然，她绝不会允许第二者拥有她这样的力量，因而这种舰装的极限便是千米的身高。不过即便如此，在舰娘们看来，这也已经足够巨大了。

能够无限度地干涉世界，拥有这样恐怖的力量，圣路易斯也几次想过要将整个地球全都毁灭，随后按照自己的喜好重建一个。但是，每当看到港区里大家露出的笑容，这样的想法总会烟消云散。

由于实在太强，每次任务的时候，只是动动手指而已，那些敌人便凭空湮灭了。就连塞壬的舰队，在圣路易斯的面前，都如同蝼蚁一样渺小，轻易便被抹杀了。

也许，圣路易斯究竟有多强——这个问题，只有她本人能知晓答案了。

“哈啊～～好舒服。”

躺在大陆的中央，百米身高的圣路易斯，正静静地守护着她身旁的城市。尽管可以用上各位巨大的体型，但那样做的话，估计一睡下来，城市就被圣路易斯的身体压碎了吧。

虽说任凭自己的身体碾碎城市，一边惬意地享受着阳光，一边感受着底下城市爆裂的感觉，确实算不上坏。但那之后，为了不让别人发现，圣路易斯就又要在原来的位置重新造出一个城市，还需要将所有人都记忆全都修改一遍。只是单单为了短暂的快感，而要承受这样的麻烦，圣路易斯并不觉得这是一个好方法。

“嗯？痒痒的...怎么回事？”

感到腿间有些异样，圣路易斯随意地翻了个身，脚边随之传来一阵浆果爆裂之类的奇妙感觉。但是，圣路易斯对此却十分熟悉，因为——这无疑是小人在她身体下，被无情的重压碾碎身体产生的感觉。

这时，圣路易斯才隐约有了印象。

前几天，由于睡觉的时候身体不慎滚入了城郊中，这巨大的身体，一瞬间就让数十公顷的田地化为了烂泥。而种植在上方的蔬果以及遍地的稻田，自然是随之消失于圣路易斯的身体下了。

利益蒙受侵害的人们，愤怒地聚集在圣路易斯的身体边，全然忘记了他们与圣路易斯的体型差距。就这样，在圣路易斯未察觉到的情况下，大约百人已经葬身她的足、胸、腿下。

抗议逐渐升级，规模也渐渐扩大。

尽管以圣路易斯的大小来看，地面聚集的千人，同时喊出口号，这样在普通人看来震耳欲聋的巨响，在她的视角也只不过是蚊虫的叫声罢了。只不过，正是这样的声音，让她越加地烦躁与愤怒。

“咿！真是够了！”

圣路易斯猛地站起，一脚朝着人群落下。超过百人瞬间变为肉饼，迸出的鲜血都染红了大地，将周围的人吓得魂飞胆裂。

仅是这样，抗议的人群便四散而逃了。想到人类如此懦弱，圣路易斯便忍俊不禁。

“好了，也是时候，惩罚一下愚笨的人类了。”

以现在的体型，推平一座城市简直轻而易举。但是，圣路易斯却不想亲自动手。

展开手掌，伸向天空。抬起的右手，直指苍穹。随后，一个大洞，出现在了城市的上空。圣路易斯撕裂了空间本身，准备将她想象中的生物制造出来，随后扔到城市中。

“那就，选个稍微大一点的吧。”

人类所创造的艺术作品，舰娘们也略有染指。其中，让圣路易斯最为在意的，是那些灾难题材的电影。其中不乏许多天马行空的怪兽，而现在，圣路易斯正打算让其中自己最喜欢的一只降临现世。

在脑中构建其基本的特征，随后缓缓让它变为现实。

“咚——！！！”

一阵巨响，响彻了整个城市。

烟尘落地，一个比肩圣路易斯的巨大身影，逐渐由朦胧转向清晰。当人们见到那覆盖着鳞片的躯体时，顿时都开始尖叫不止。

圣路易斯无言地退回城市的边缘，随后赋予了这个被自己创造出来的小生物自由行动的意识。顿时，一声远古巨兽般的嘶吼直击城市，惊吓到的人们在恐惧中仓皇而逃。但奈何，圣路易斯并没有要让她们逃跑的想法。

展示完创造生物的力量，似乎还不足以让小人们相信圣路易斯拥有能随意干涉这个世界的能力。接下来的一幕，便是要让这些固执的小人。彻底信服。

圣路易斯翘起玉指，一道光线随之出现。由于平时便经常运用舰炮，圣路易斯的视力自然极好。于是，这道足以融化大地的耀眼光束，光芒甚至盖过了太阳。就这样，那束光匀速地在城市的边缘绕了一圈，烧穿沿途的建筑，在地上划出一道百米深度的沟壑。

“咿呀～～姐姐真是不小心呢，不好意思咯。”

从远方看着城市内横行的百米怪兽，推平试图阻挡它的一切。慢慢看着整座城市变为废墟，这种掌控全局的感觉，让圣路易斯欲罢不能。

但是，这还远远不够。

既然人们把自己也称为怪兽，那不妨满足他们的愿望，就真的化身一次蹂躏大地的“怪兽”吧。

“设定巨大化程度，完成。”

“呵呵～到底，会变成什么样呢？”

还在躲避那突如其来的巨大怪兽之时，城市中慌乱的人们，却被天空的异象吸引了目光。只见在正对着城市的方向，一个身影正在缓缓变得清晰。但让人不解的是，一开始人们还能见到她的全身，但后来，眼前的景色就被两只矗立在地上的玉足所遮挡了。

白皙无瑕的皮肤，细腻无比的纹理，甚至在阳光下还散发着象牙般的光晕。辅以那婀娜的脚型，以及裹着整个脚掌的纯白高跟，这一切，赋予了这双美腿全新的价值。整个城市的人们都看呆了，这哪里是少女的两只脚，简直就是这世界上最为完美的艺术品！

但是，最为瞩目的还是那高跟的大小。为了容纳如此巨大的圣足，高跟的尺寸自然也变得极为恐怖。从城市中的每个角落，都能一窥那高跟的雄伟，就算是漂浮着的云层，也不及脚跟的一半高度。就这样，那按照程序设定的怪兽，在城市中横行践踏，杀死了不少因为呆滞地望着圣路易斯的脚而忘记逃跑的小人。

“啊呀？难道，我的脚有这么美吗？”

圣路易斯设定的巨大化倍率，为一百倍，也即是常人的一万倍大小。

尽管可以不用这些繁琐的程序，而直接任凭自己的想法变大。然而，由于不够精细的感觉，圣路易斯常常会变到比预定还要更为巨大的体型。一般来说，误差大约在两倍之内。

虽然这数字看上去很小，但在百倍的倍率之下，两倍的误差，造成的便是千米乃至万米的差距。况且，在极端情况下，圣路易斯还有可能直接撑破大气，变得和地球一样的大小。

以两万一千米的身高，圣路易斯骄傲地俯视着整座城市。在她的脚掌面前，万物都不得不俯首称臣。遍地的百米高楼，曾经作为人类工程的奇迹屹立不倒，但圣路易斯的出现，却让它们又低下了头。

星罗棋布的街道，尽管占地面积巨大，但圣路易斯只要降下脚掌，便能连着毁灭好几个这样的区域。不过，圣路易斯此行可不是为了毁灭这座可怜的城市，而是为了让卑贱的人类意识到，在神祗一样的她面前，人类究竟是多么弱小的存在。

“哦呀？看来，我的小宠物好像被你们看作是怪物了呢。”

将高跟摆放在城市中，数万人因此丧命。原因无他，圣路易斯特意挑选了人群最为密集的一片商业区落脚。而由于本来就没有让那里的小人存活的想法，圣路易斯毫不留情，就这样任凭脚掌碾平下方的一切。

光是其中一只脚掌，就有两千米的长度。所覆盖的面积，更是能遮盖住两个以上的街道。这样的庞然巨物，在人们的眼中已经是十足的山丘了。一整座山丘砸在人口稠密的商业区，后果可想而知。

再次抬脚，那里除了一个镌刻着圣路易斯鞋子纹路的落寞脚印，以及其中变为土黄色的大地之外——其余的一切，都已经荡然无存，片瓦无余了。

只是落脚，城市原有的区域就被彻底抹去。拥有这等强大的力量，造成了如此严重的灾难，圣路易斯却只是微微一笑，吐舌致意。在她看来，这只是为了消灭那只在城市中为非作歹的怪兽，而不得已做出的正义举动。

很快，被圣路易斯的落足声所吸引，人们纷纷望向远方，而那只在人们眼中无疑是庞然大物的怪兽，到了圣路易斯的脚掌面前，居然连一只虫子都算不上了。

“看上去挺小只的，跑起来却快得惊人呢。”

圣路易斯的巨足，就这样在城市中肆意滑动，追赶着前方那只可怜的怪兽。虽说有着在人们眼中无比巨大的身体，但与圣路易斯的脚比起来，还是远逊于此了。

享受着一路推平高楼，碾碎楼房，抹去街道的快感。圣路易斯只需用脚轻轻一压，那些遍地的建筑就成片垮塌，最后就只剩下一个坑洞了。这种力量的悬殊差距，让她甚至只要动动脚趾，就能让大地为之震动，以此最大限度地榨取人们的恐惧。

直到刚刚为止还举着横幅，喊着口号对圣路易斯口诛笔伐的人们，在那碾压一切的巨足面前也都闭上了嘴。当然，有一些人可能永远都没法再说话了。

圣路易斯让自己的每一个步伐都尽可能地小，防止真的将自己创造出来的可爱宠物卷入足下碾死。以此，借着追赶怪兽的名义，圣路易斯一路将连成一排的十几个街道一齐推平。许多小人甚至还没能看到圣路易斯脚掌的真容，就被飞来的高跟连同他们周围的建筑一起撞成了齑粉。

“嘿呀～～小人们，快跑呀！不然的话，姐姐的脚就要来咯。”

圣路易斯抬起脚掌，将其高高举到身体一半的高度。已经玩腻了在地上滑动脚掌的游戏，圣路易斯打算尽快结束这个无聊的游戏。

于是，她将脚掌对准城市的中心——那建筑密度最高的区域。随后，操纵着她制造的怪兽主动朝着市中心进发，在那只怪兽恰好到达的瞬间，圣路易斯猛地压下脚掌。

被阴影吞没的数千栋楼房，里面居住着的小人倾巢而出。然而，当他们意识到头顶的脚掌究竟有多么巨大时，纷纷丧失了逃生的欲望。大部分人只是呆在原地，瞻仰着这巨足的唯美，同时想象着究竟会怎样被它杀死。就在远方传来一阵急促的脚步声的同时，头顶的天空忽然崩塌，所有人都在来不及反应的情况下，平等地变为了一滩血水。

“下次，要不要创造一个比我还大的怪兽呢？不过那样的话，人类就真的完蛋了吧，呵呵呵～”

圣路易斯转动脚裸，碾动着下方的土地，将已经被碾碎一次的街道，再一次变为更加细碎的粉末。人类所建造的一切，在她这位能随意干涉世界的神面前，都显得太过渺小了。

然而，在刚来的时候，圣路易斯确实能老老实实地睡在山脉的外边。后来，由于翻阅山脉有些繁琐，她索性一下将倍率调整至一万倍，也即是两千一百公里的身高，一脚便将那些山脉全都抹碎了。

“啊呀，既然怪兽已经被消灭了。作为被我救下的人，你们是不是应该给姐姐我一点奖励呢？呵呵～”

一番蹂躏过后，城市依旧留存有大部分完好的区域。兴致大发的圣路易斯，打算就用这小小的城市，作为她的新玩具。
