# 《对她说》（诗作)

作者：libido

TID：31946

# 1

刚才洗澡时偶然产生的灵感，感觉诗的形式会比文章更有表现力，就决定按诗的格式写了下来

原本打算发在「随笔发想区」，不过那里现在似乎没法主动发帖了，姑且先发在「原创文章区」，如有需要，烦请管理员善加处理

本人之前没正式学过现代汉语诗歌，这次算是初投稿，献丑了

标题来源于同名电影，里面《缩水情人》的片段大概是本人在H方面对gts的初体验，至今记忆犹新

你和她相约在那片空地虽然在她看来，未免有些拥挤

你到了，长途跋涉的

路上没看见半个人影她到了，轻快的脚步惊动了整片森林

你仰头，她俯身她按字面义把你捧在手心里

你望着她的眼镜，她盯着你的全身最后决定用嘴唇打破沉默你暗中庆幸她没有选择湿吻，毕竟换洗的衣服你没有带够

你们谈情她低声细语，你大声回应——虽然在彼此眼中可不一定

你们做爱但或许更准确的说法是她的自我发电中也包括你

你用全身来满足她，而她不得不再添上自己的手你射在了她深处，作为回赠她把你吹飞出她的身体

你们能彼此相爱但不能事后彼此擦干，更不能两两相倚而去

她提出要带你回城里你担心麻烦，只得回绝并把她家作为下次相约的地点

而当你回到地上后才发现忘记了那爱的告别

你向她招手，她已走远你喊她名字，她听不见你望其项背而不得追只能呆呆地伫立在她留下的脚印里面

# 2

感谢各位的支持，这里集中解释一下“望其项背”的用意

“望其项背/望其肩顶”字面义是「能看见（对方的）脖子和后背」，比喻追人时距离很接近，能赶得上

这里是用此时其字面义与转义间的矛盾，来突出小人与巨人的悬殊差距（或者说，整个处境的非日常）
