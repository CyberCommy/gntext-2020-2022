『魔法少女』。

这是所有生活在这世界上的少女都无比憧憬的存在。

要得到这个称号，不仅需要魔力的天赋，对容貌的要求也很高，从小还要接受严厉的训练。

只有通过了被称为『觉醒考试』的人，才能获得这样的称号————

＊

秋意渐浓。

即将变红的树叶仍在接受阳光的照耀。今天是难得的，仿佛惋惜夏天离别的残暑。

像往常一样离开家门的卡尔娜恨恨地望着天空。

明明都快凉下来了…又搞得浑身是汗怎么办啊？她撩了一下栗色的头发，啪嗒！跺了一下平底皮鞋，快步走向学校。

蓝色的格子花纹裙子配合着卡尔娜的脚步，有节奏地摇晃着。

女孩子独有的纤细双腿穿着学校指定的黑色丝袜，由于纤维度①比较低，她耀眼的洁白肌肤散发出不少魅力。

藏青色西装的胸口上，只有得到魔法学习训练许可的人才能获得的金色校徽在闪耀着。

＊

今年是最后一次穿着中等部的制服了。一直憧憬着魔法少女的我，迎来的15岁的生日。

为了今天，我学习了许多东西，也经过了不少的锻炼。

是否拥有魔法少女的资质，便看今天能否承受住未知的考验了。

只要成功，我就能当魔法少女！

「今天，加油吧！」

从自己家里出来到车站约10分钟。穿过检票口，登上楼梯，我来到了站台。

不久，电车来了，车门哗啦一声打开，我精神抖擞地坐在空座上。

「早上好～卡尔娜！昨天睡得好吗？」

欢快地向我搭话的是我的同学梅蒂。虽然班级不同，但从第一年开始我们就是好朋友了。

她担心地凝视我，也许是想缓和一下我紧张的情绪。

「早上好，梅蒂。我没事，睡得很好。话说，我的脸真的看起来很紧张吗？」

「当然啦，可爱的脸都被糟蹋了」

关系一直很好的梅蒂能看穿我表情下的紧张。

「迎来人生分歧点的美丽少女，是多么坚强啊！想安心下来吗？来，拥抱我的胸膛吧，来嘛♪来嘛！」

虽然知道梅蒂是这种不认生的、满是温柔的性格，但从早上开始就怀着紧张情绪的我，觉得她有点烦。

像是要展示我从未见过的丰满胸部一样，她张开双手邀请着。

「喂，停一下，梅蒂，太难为情了啊」

「在有人的地方不好意思了？那么那么，到学校之后好好地帮你散散心吧，咕呼呼呼」

「别说啦！」

虽然一脸嫌弃，但不知什么时候，我还是浮现出了笑容。

多亏了她，紧张的情绪得到了不少的缓解。

15分钟后，电车到达了学院前站。学生们纷纷下车，我们也跟着人流走了过去。

穿过检票口，再走5分钟左右，就能看到熟悉的学院大门了。

虽然平时走过时都觉得理所应当，但今天抬头一看，校门口周围的树木，似乎散发着季节的气息。

红叶摇曳着，像是在激励着我们这些为了未来而努力的学生们。

「考试是下午开始吧，今天加油哦！下周我也要考试了！大家一起努力成为魔法少女吧！」

为了成为魔法少女而举行的『觉醒考试』，是参加过特定课程之后的人，在15岁生日那天参加的。

因此，参加考试的很可能只有当天过生日的一个人。

为了一周之后梅蒂的生日，我要努力当上魔法少女。

我轻轻向梅蒂道谢，分别走向各自的教室。

＊

上午是同往常一样的训练。

吃完午饭，我一个人走向了考场。考场是位于体育馆地下的魔法训练场。

我走下楼梯，打开厚重的门，哒哒哒地走了进去。

魔法训练场不是给不会魔法的我们，而是给那些高等部的魔法少女用的。

不管怎么说，我以后也会在这里练习魔法的…怀着这样的幻想，我向深处走去。

在昏暗的灯光下，班主任玛丽娜老师和两个陌生的女生在等着我。

「按时到了呢，卡尔娜」

玛丽娜老师翘着长长的双腿，坐在椅子上。

紧身裙配上肉色丝袜，脚下是黑色的高跟鞋，白色的衬衫和黑色的西装包裹住呼之欲出的胸部。

富有光泽的黑色长发延伸到肩膀上，鼻梁挂着黑色的细框眼镜，真是经典的女教师形象！

虽然穿着并不出奇，但那足以称之为妖艳的成人魅力依然满溢着。即便作为女性，我也觉得她很美。

旁边的两个学生穿着高等部的制服。

「这两个孩子，是今天考试来帮忙的高等部一年级学生」

「…我是米拉」

米拉用平静的语调打着招呼，脸上挂着和善的微笑。

她粉色的长发一直延伸到腰部，即使在昏暗的考场上也散发着美丽的光辉。

她的瞳色也是粉红，圆润的瞳孔非常吸引人。

米拉穿着高等部以白色为主的西装上衣，裙子到膝盖位置，端庄的姿态仿佛西方故事里的公主一样。

纤细的双脚上穿着洁净的深蓝色袜子和富有光泽的黑色平底皮鞋。

「我是妮娜，请多指教～卡尔娜酱♪」

接下来向我打招呼的是一头蓝色短发的妮娜。她语调轻快，似乎很容易亲近的样子，与米拉形成鲜明的对比。

她的胸部也非常有料，感觉能和玛丽娜老师比一比了。

妮娜个子很高，腹部和腰部都很紧致结实，也是深蓝色的袜子，不过穿着的是深棕色的鞋。

虽然很不甘心…但可以说身材上我完全败北。

「请、请多多指教…」

「这两个人，也是一年前在这里成为魔法少女的」

「不用那么紧张」

「好可爱～！虽然说是考试，但没什么难的啦，放轻松——♪」

米拉和妮娜像是察觉到了我的惴惴不安，给我加油鼓劲。

不过话说回来，『觉醒考试』到底是什么？

「那么，早点进行考试吧」

「明白了，我去准备一下」

「那，我去拿『妖精』吧♪」

高等部的两人快步朝着准备室走去。

「上午是体育课吧？做了些什么？」

玛丽娜老师站起来向我搭话。突然说些日常的话题，是缓解我紧张的情绪吧。

「今天是垫上翻滚，然后还有跳箱…」

「嗯，这样啊，是不是很热？今天是偏离季节的酷暑呢」

「是，温度湿度都很高，出了不少汗…」

「体育馆的窗户还少，里面更闷吧～」

「是呢，穿了运动短裤，还是热…」

「啊啦是这样啊，那么之后真是值得期待呢」

「…诶？」

我没听懂老师在说什么，有些吃惊。

不久，米拉和妮娜回来了，抱着小木箱一样的东西。

「老师，带过来了～」

妮娜将宽约20厘米的木箱放到桌上，米拉走到了我的面前。

「准备好了吧？」

「啊…诶…准备…？」

望着还不知所措的我，玛丽亚老师直接进入正题。

「卡尔娜，魔法少女和普通人决定性的差异，是魔法的有无，你明白吧？」

「是，拥有魔力，能用出『魔法』，然后用这样的力量裁决罪恶，这就是魔法少女」

「对，而魔法少女原来也是普通人，那普通人如何获得魔力呢，如何将魔力存在人体中呢？」

「需要一种被叫做『妖精』的生命体所拥有的能量。魔法少女能将『妖精』的能量变成真正的魔力储存在体内」

「学的真不错。也就是说，『觉醒考试』是迎来15岁的少女，首次『使用』『妖精』，从而得到魔力的仪式。

而所谓的考试，就是看本人能否做到了。」

「这些，就是课上学过的东西」

「那『妖精』的使用方法…嗯？妮娜？」

「哇哇…不好意思，『妖精』掉下来了！」

不仅是我，现场的所有人都望着慌张的妮娜的脚底。在那里的是摔在地上、赤身裸体的…男…人？

「诶!?」

难道，那就是『妖精』？

像小人偶一样的？

而且不像是人偶，是真的在活动着的？

我以前认为的所谓『妖精』，是长着漂亮的羽毛，闪闪发光，更具幻想感的存在。

而作为『妖精』出现的『那个』，更像是人的形状。或者说，虽然大小不同，但外表是普通的男孩子。

与其说这是妖精，不如说是小人。而且，尺寸相当小，只有3~4厘米的样子。

被称为『妖精』的小人突然爬了起来，从妮娜的脚下跑开了，说不定是要逃跑！

「啊…！」

我惊讶地望着眼前突然发生的事情。

因为对他来说无比巨大的脚踩了下来，发出了无比夸张、仿佛威胁一般的轰鸣声。

接着，米拉沉默着俯视小人，然后右脚稍微抬起，闪闪发亮的黑色皮鞋就踩在了小小的身体上。

小人毫无抵抗地被踩在了脚下。

「谢谢，米拉酱」

这个学院里，有被称为魔法训练场的设施。训练场在体育馆的地下。

为了避免受到冲击而崩塌，天花板和墙壁由被魔法强化的钢筋、水泥与砖头加固而成。

地面更是铺满了被称为比钻石还坚固的黑金刚石。

入口的大门也很厚，设置了拒绝无关人员与入侵者的魔法结界。

这天，卡尔娜迎来了15岁的生日。

虽然外面仍是如夏日般炎热，但这与外界隔绝的训练场里却充斥着凉爽的空气。

因为上午体育课上出了很多汗，所以对现在的卡尔娜来说，这种冰凉感非常舒服。

如果成为魔法少女的话，在炎热的日子里，随时都可以在这凉爽的设施里训练吗？

做着如此幻想的卡尔娜，听到了班主任玛丽娜老师的声音。

「让你久等了，卡尔娜，你的『魔法少女觉醒考试』正式开始」
