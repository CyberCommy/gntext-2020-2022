# 1

      不知道发生了什么，在东京里产生一声巨大的声响，所有的人都被整得头晕眼花，当在东京里的人们恢复过来，慢慢站起身子。但是现在的场景却让人无法让人们所平静下来，周围的地面变成了粉色的地毯，而且还有着无数的粉棉的绒毛如同柱子一般，当然这些并不是人们无法平静的原因，在东京高高的上空，一层阴影裹住了整个东京，那是一只能够完全把东京围住的巨大白色物品，从高空中还有着一股淡淡的香味从那白色的巨物里散发出来。但当人们看到那另一个和上空的白色巨物及其相似的存在放在地上，那完全就是一只穿着白丝袜的脚，巨大的白丝足被平静地放在地面上，还从那巨大的白丝足不断散发出的香味，很明显白丝足的主人平时很注意脚的保养。即使这只白丝足接近乎完美的美丽高贵，还有那股诱惑的气味，但现在的人们却完全无法欣赏过来，因为另一只完美的白丝足现在就悬挂在他们的上方，只要这只脚的主人愿意，随时都能把整个东京像虫子一样碾碎在脚下，东京里的人们被这只白丝足造成的巨大压迫感不断冲击着他们的内心，有些人不管现在的东京发生了什么 现在只想要全力逃离女神的足底对他们造成的压迫感。
        “啊，好累啊，终于把作业都给写完了，一写完作业就感到肚子好饿啊，还是去做点东西吃吧。”
        穿着学校的制服，在脚上穿着白色长筒丝袜的少女翘着二郎腿写完了功课，慢慢地把那只搭在右腿上的左脚慢慢地移下来，人们看着天空中的那只白丝足好像有了些动静，仅仅只是因为白丝足移动下产生的风就让东京里的所有人都站不稳脚，都被白丝巨足产生的巨浪掀倒在地，当他们再次仰望上方的时候，所有人的眼神都变成了惊恐的神色，巨大的丝足带带着巨大的压力不断的压力不断地想他们压下来，在这完全把整个东京围住的白丝足下，人们根本躲无可躲，巨大的尖叫声在东京里面响起，但那白丝足底就像是女神要把恶心的虫子碾碎一般，丝毫没有停下碾压的迹象。
        “轰。”整个东京都消失在了白丝足底下，没有任何幸存者，全部都变成了少女的白丝足底的一个毫不起眼的红点，东京变成了白丝足底的一堆没有任何用处的灰尘，被永远地消失在了少女的白丝足底下。在女神的脚下，弱小的生命无论做些什么都没有任何意义，只要女神把脚随意地放下，弱小的蝼蚁只有被女神的神足碾碎，没有任何的能力反抗。
        少女把翘着二郎腿的左脚往地面踩了下去，好像在脚底感到了痒痒的感觉，把头稍稍地往下看到自己踩着的粉色地毯的上方，但在那个位置上只有一堆灰尘污垢在地毯上。
       “总是感觉自己刚才好像踩到了什么，唔，可能是我 写作业太久产生错觉吧，还是快点去做点东西吃吧。”
