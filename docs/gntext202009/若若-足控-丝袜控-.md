# 若若（足控，丝袜控）

TID：27950

 

# 1

“放开我！你要干什么？！”

若若指尖捏着的小人一边挣扎一边大喊。若若笑着不回答，松开手指，任由小人落入下面的丝袜之中。

若若提起两只丝袜，看着散在里面的十多个小人，笑道“你们上次被我拿出去的小伙伴已经没有精神气力陪我玩啦，现在轮到你们来陪我啦。在里面乖乖的哟～”

说罢不顾小人的哭喊和挣扎穿起了丝袜。为了延长小人的生命，若若抬起脚用手指调整着在丝袜里挣扎的小人的位置，将他们移到了脚趾下方的空隙。若若扭动了下脚趾，感受着小人们的挣扎，哈哈一笑，穿上了帆布鞋。几分钟后，出门上学去了。

一路上，若若控制着自己的脚步，避免挤死踩死丝袜里的小人。教室里，正值课上，若若感受着丝袜里小人的扭动和挣扎，好像有的已经放弃了开始舔弄若若的脚趾，搞得若若痒痒的，不由得面露笑意。课上太无聊，若若不自觉屈了屈脚趾准备挑逗下脚下的小人，谁知竟有几个小人顺着屈趾出现的丝袜与脚掌之间的缝隙，逃往脚心处，仿佛以为那里有出口，可以逃离若若的玩弄。若若一边为小人的天真感到想笑，一边为了多让小人活久一点，便顺着他们，屈着脚趾任由他们往脚心挣扎爬去，直到他们到了相对安全的足弓处才扭动脚趾，断了小人的退路。小人这才意识到不对，更加拼命捶打若若敏感的脚心，却已然晚了，若若感受着小人的按摩，享受着度过了一中午的课程。

放课后，经过一中午的玩弄，双脚下丝袜里的小人们早已精疲力尽。回去的路上，除了脚心处的几个小人还有精力捶打两下，别的都已没了动静，只能随着若若的脚步，在丝袜里滚动。若若笑了笑，继续和自己同路的小姐妹聊着天，仿佛自己就是一个再正常不过的有点姿色的高中生。

“啪！”若若关上了家门，慢慢的脱下了帆布鞋，看着丝袜里失去了精神的小人，若若夸道“你们今天真的很努力呢，居然还能逃到我的脚心处给我按摩。”边说边用手指轻轻的戳了戳脚心处的小人。小人受到惊吓，挤出力气，又扭动了几下。有幸钻到脚趾缝里的小人，抬头看着若若，早已放弃挣扎，渴求的望着女神，希望女神可以看在自己这么努力的份上，放自己出去。

若若毫不在意小人的渴求，抬脚便走向卧室，上床躺下，对着脚下的小人说“你们就乖乖的在里面好好休息吧，明天可是有体育课呢，要好好养好精神哟，渴了就舔舔我的脚汗吧～”若若说完，缓缓的闭上了眼睛，感受着双脚丝袜里小人们突然强烈的挣扎，笑着拉了拉丝袜，睡着了。
