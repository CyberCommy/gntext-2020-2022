# （海贼王同人）日奈的遭遇

作者：jingmom

TID：32247

# 1

*本帖最後由 jingmom 於 2021-12-15 20:07 編輯*

辽阔的海面上，一艘海军军舰行驶着，这艘军舰没有目的地，因为它执行着守卫海洋和平，清楚海贼的日常巡航任务。而这艘船的舰长，黑槛日奈，正靠坐在舰长室中、“真是无聊，已经连续几天没有碰到有趣的对手了”日奈叼着烟抱怨道“都是一些被我一个人跳上船，然后全团就被我解决掉的三流货色。”“不经历一些挫折，怎么能成长，怎么能有足够的力量，对付像四皇那样的对手呢。”心烦意乱的日奈将烟头扔进垃圾桶，从椅子上跳起来，去甲板上走路散心。巧合的是，当日奈走到甲板边的时候，视野里突然发现了远处飘荡的一艘小木船，利用见闻色霸气，日奈看清楚了小木船的东西：最醒目的是船中央阳伞，伞的后面堆着大大小小的各种包裹，而伞下，一个女孩懒洋洋的躺在折叠椅上，旁边还放了一个小桌子，上面盛了一碗冰淇淋。女孩身裹着一套貂皮大衣，或许是之前被海风冻得不轻，但是两个脚倒是光着，一对米色靴子放在女孩的脚边，女孩的头上有两个奇怪的发饰，使得女孩的绿色短发稍显正常了起来。“来得正好。”日奈正憋着一股劲没处发现，指挥手下海军开到女孩的船前，逼停了女孩。而这女孩，正是我们在海上四处游荡的布兰缇什。布兰缇什坐起身，从下面看着高高在上，站在船头的日奈，无奈的大声问道：“海军大人，有何贵干啊。”“少废话，出海有许可证吗？”“就一艘小木船飘来飘去，碍着谁了？”日奈确懒得废话，直接跳了下去，正好落到布兰缇什的椅子前面，她使用体术，控制力道，使船不至于侧翻，但布兰缇什已经叫了起来“喂喂喂，小心点啊，翻船了怎么办。”“少废话，今天就给你点教训吧。”日奈虽然为人还算正派，不准备过多的为难一个小女孩，但毕竟违反了出海规定，自己又正好在气头上，正好给她一个教训常常，也算发泄一下，于是将两手变成了栏杆准备伸长把布兰缇什困住，可是突然，她感觉自己的身体快速的从空中落下。日奈一瞬间没有反应过来发生了什么，但他忠心的属下在船上用望远镜看到了这令他们大吃一惊的一幕：一瞬间，他们的长官身体突然缩小，变成了在空中的一个小人，然后迅速的下落。然后，布兰缇什把右脚伸了出去，用大脚趾和二脚趾在空中把落下来的日奈夹住。日奈也是海军新生代的代表人物，在被夹住后立即镇定了下来，准备伸长两手变成的栏杆去反击。谁知布兰缇什先开了口：“我劝你不要轻举妄动，我现在可是为了防止你摔下去轻轻地夹住了你，要是你一不小心刺到了我，让我条件反射动了下脚，你不是被我夹死也可能掉下去摔死哦。”说完，还调皮的轻轻地夹了下脚趾。日奈还没反应过来，身体自行做出了危机反应，调动武装色进行防御，可惜武装色制造的铠甲轻而易举碎掉了，同时碎掉的还有日奈的多处骨头。在疼痛感和羞辱感的双重作用下，日奈的脸涨的通红，吼道：“你知道我的能力，你到底是谁？”布兰缇什又轻笑了一下：“呵呵，可不要搞错，我可不是什么坏人，只是个无名小角色，倒是你黑槛日奈，有名的海军大人，出海混的谁不知道您？可您为什么要为难我呢？没想到，平常用黑槛将敌人困住的日奈大人，现在被我的脚趾困住了，看来还是我的脚趾略胜一筹啊。”说罢，布兰缇什不再理会羞愤交加的日奈，对着船上的海军大声喊话：“快给我让开，不然你们的舰长大人，就要成为我的脚下亡魂了。”船上的海军群龙无首，在一段慌乱后，还是很快的让开了道路。布兰缇什继续让船开始飘荡起来，她重新躺了下去，但两脚继续紧紧地夹住日奈，在飘过军舰时，她调皮的对海军又喊话了一次：“也不许追上来，你们的舰长大人，在我安全到岸后，我会放过她的，你们在家等着就好。”

。。。。。。一小时后

“好热啊，这海上的天气真是反复无常，之前那么冷，现在又这么热，害我出了这么多汗。”边抱怨着，布兰缇什把貂皮大衣脱下来，往身后一甩，身上就只剩下了一套内衣，任谁看了都心潮彭拜，但被夹在脚趾间的日奈并不好受，脚总是人总容易出汗的部位之一，布兰缇什的脚汗的气味浸入日奈的身体里，让日奈无比难受，这时，听到布兰缇什发声了，胆怯了一个小时不敢说一句话的她也终于开始大胆问道：“我看你这船好像根本就是随着洋流乱飘的，你要去哪？”布兰缇什有些莫名其妙“不去哪儿啊，就是乱飘啊，放心，等到了岸，我就把你变回去。”日奈生气点有点失去了理智：“那我得等多久？我吃什么喝什么？你要把我活活弄死？”布兰缇什像是突然想起来什么，把手往后面伸，够到了下面的一个包裹，从里面拿出来一些面包：“你这么一说，我好像还真有点饿了。”日奈更是难以接受：“快给我也吃点，我保证，不会把你通缉的。”布兰缇什有些不难烦了，她把脚伸进了脚边的米色靴子之中，在穿上靴子之后，布兰缇什松开了日奈，让日奈困在靴底和她的脚底之中。日奈落到了靴底之上，才意识到了发生了什么，她大吼着，用全身的武装色，对布兰缇什的脚底发起了一次攻击，但是只打落了一些角质和死皮。布兰缇什觉得脚受到了一点冲击“还挺舒服的。”她这样想着，等着这样的感受多来几次，但日奈已经彻底的躺倒，双眼流出泪来。。。。。。。又过了不知道多久日奈又渴又饿，她没有吃，也没有喝，之前的修炼不是没有这种情况，但之前的修炼，是为了更好的锻炼自己，是为了朝着那光明的未来，而现在，她的眼前只有布兰缇什的脚底所造成的黑暗，她只能用所剩不多的见闻色来确认自己的处境。“等一下。”在见闻色的洞察中，日奈终于意识到，自己并不是没有吃的东西，喝的东西......。。。。。。七天之后“啊，终于到了陆地，在海上飘着我都快要疯了。”当小木船靠岸之后，布兰缇什伸了一个大大的懒腰。这七天，布兰缇什一直躺在床上，整个人都要麻掉了，最严重的是穿着靴子悬在空中的双脚，都快要失去知觉了。布兰缇什决定先把双脚轻轻地放在船上，休息一下，恢复过来再起身，走上岸。在把脚放在船的地板上的一刹那，布兰缇什突然感觉到了脚下有什么东西。“。。。原来我靴子里还有个人，幸好没有直接站起来，不然你可就惨了哦。”想到这，布兰缇什把靴子脱了下来，往陆地上倒了倒，一个小人从靴子中掉了出来，布兰缇什对着空中的小人释放了魔法，一个高大的女人就这样趴着，掉在了地上。“好臭，这是什么味道。”布兰缇什赶紧站起身，捏住鼻子，往远处走了好几步，然后才观察起日奈来。此时的日奈，全身都被浸湿，而这并不是正常的水，全是布兰缇什的脚汗，因此日奈的全身散发出一股恶臭，但更为恶心的是，日奈洁白的海军服，标志性的粉丝头发上，黏的到处都是黑色的污垢，似乎是被放大了的，布兰缇什的脚泥。此时的日奈受到的伤害，与其说是生理层面的，不如说是精神层面的；她胃里满满的脚汗和脚泥，甚至足以支撑她发动反击，或是选择逃离。但她什么也没做，只是静静的趴在那里，像是魂魄被吸走了一半。不如说，之前的时候，也正是这种精神崩溃，让她疯狂的用脚汗和脚泥胀满自己的胃。“这都是什么跟什么啊”布兰缇什一脸嫌弃的靠近日奈，毕竟她的船还在那儿。当她把船缩小放进口袋之后，她一溜烟的快速小跑离开。留下日奈一个人，静静的趴在海边。
