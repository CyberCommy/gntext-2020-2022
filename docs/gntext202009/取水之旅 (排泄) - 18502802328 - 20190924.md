# 1

水，万物之源。对生命而言，水是不可或缺的，无论多么强大多么巨大。

‘我的喉咙。。。哪里有水。。。’这里有一名口渴万分的少女，正为寻找水源而在太空中徘徊。在宇宙中水并不是非常常见，然而她却幸运的碰上了一颗有着珍贵的水资源的行星地球。‘啊，一颗蓝色的行星。。。一定有水’少女被太空中那一抹蓝色所吸引，她决定立刻前往地球。

很快少女降落在了还算广阔的太平洋上，可是落地才发现这看上去很广阔的海洋实际上将将漫过少女脚底。‘这甚至算不上一个水坑啊。。。不过总好过一点也没有吧’少女弯下身子让四肢着地，低下头让脑袋靠近水面，伸出了舌头，小小的尝了一口，不过在人类视角来看那就是铺天盖地的粉红色肉墙从天空中席卷而下，仅仅舔了一下便引起了数百米的海啸。‘唔。。。这水能喝，太好了！我开动了！’说玩少女便张大嘴巴，尽可能多的汲取水资源。这对太平洋上的人类来说是个噩耗，就在粉红色肉墙消失不久之后，云层里出现了一片干枯的嘴唇，当嘴唇张开，他们在劫难逃。由于少女嘴唇产生的巨大吸力，在海面和嘴唇之间瞬间产生了数十千米高的水龙卷，并引起了全太平洋规模的超大海啸。海啸破坏力巨大，太平洋沿岸地区及太平洋中的各个岛屿都受到了数千米高的海啸袭击，人类伤亡上亿人。全球总计13.7亿立方千米的海洋在少女吸收下也不过坚持了数十分钟，不过少女并没有感到满足。‘我记得在另一头还有着冰块来着，我还想要更多的水。’少女站起身来，在干枯的海洋里向着南极前进。对于地球上的人类来说，在几十分钟之内就失去了全部海水简直闻所未闻，难以置信，不过伴随着急剧的气温下降人们相信了这个突然出现的少女确实喝光了所有的海水。在卫星的帮助下，各国首脑发现了少女正朝南极冰架前进，紧急商议要阻止她继续夺取水资源。联合国别动队紧急出击，不过常规武器不可能对少女有任何效果，各国决定直接使用大当量的核武器攻击少女。少女十几分钟就走到了南极，正准备享用南极冰架，这时候数枚携带核弹头的洲际导弹袭来，直接命中少女和冰架。当然，核武器也不可能对巨大少女有任何威胁，只是核武器爆炸所产生的热量瞬间融化并升华了冰块，南极冰架直接变成水蒸气，消失的无影无踪。少女看着手中突然消失的冰块甚是难过，好不容易找到的水就这样没了，不一会儿这难过转化成了懊恼与愤怒。‘不就是喝了点水嘛，你们这么想要水，我这就还给你们！’少女脱下裤子，让胯部靠近已经干涸的海洋，一边憋气一边向下腹部注入力量。终于第一滴金黄的液体洒了下来，经过少女新陈代谢过后的废水，充满了盐分和各种废物，被飞洒到了新的海洋里。又一次，人类见证了奇观, 几十分钟填满海洋的奇观，只不过这次的海水是温暖而又腥臭。。。
