# 1

    满天星斗，散落银河，星光璀璨却不耀眼，若隐若现，若即若离，若明若暗，若。。。。。。若你妹啊！这黑的跟锅底一样的天上，别说星星，连月亮她老人家的脸上都蒙着一层昏黄的马赛克。好吧，好吧，今天的空气污染指数，悬浮颗粒，你们再次立功了。
    我合上杂志，脑海里还隐约浮现着插页上的那张星空照片，想想照片里的美丽天河，再抬头看看灰蒙蒙的夜空，便终于体会到了现实的残酷。
    我不是什么天文爱好者，也不是每晚眼巴巴望着夜空盼流星的小姑娘，只是那张星空的照片勾起了我灵魂中些许的多愁善感——这漫漫天际中，是否会有和我们人类一样的智慧生命体存在呢？外星人？UFO？数以亿万计——也许还要多得多得多的行星中，总该会有那么一两颗上存在生命吧。即便那里可能没有适宜的气候，没有纯净的水，没有新鲜的空气，但生命在起源时就一定要如此的优雅吗？生命的诞生就一定需要奇迹吗？
    ——生命本身岂不就代表着奇迹？
    好吧，好吧，这转瞬即逝的多愁善感很快就从我脑海中溜走了，因为我本就不是个酸溜溜的人，何况眼前还有一份无比重要的报告等着我去处理。你也知道，部长那个人可不是好惹的，或许连外星人都惹不起。
就在我胡思乱想的时候，我错过了一个非常重要的新闻，这则新闻的重要性或许可以改变历史的进程，改变人类的命运。或许在一刻，全世界都知道了这则新闻——除了我——毕竟在太阳周围出现千艘UFO的壮观景象不是随随便便就能看到的，更何况每一艘的直径都超过了地球。
    当然，在事后听到这则新闻时，我除了嗤之以鼻外，还有不屑一顾。毕竟，幻想是幻想，现实是现实。

# 2

    夏末，气温不降反升。夏末的一天我突然间心血来潮，非要去海边转一转。我居住的城市本就靠海，一向懒得出远门的我——不得不承认，去海边这种行程对于我来说已经算是远了——忽然想去看看海，想看得紧。
    夏末的海边，来享受日光浴的游人依然不减。每个人都想抓住夏天的尾巴，再尽情享受与温暖阳光的肌肤之亲。我光着脚丫子走在沙滩上，体会着脚板踩着沙子的感觉，又忽然多愁善感地想起了冬天穿着鞋子走在雪地上的感觉，并津津乐道地把这两种感觉进行着比较。
    当然，这次的多愁善感来得也快，去的也很快，因为我看到天空中一道流星划过。
    白天怎么会看见流星？事实上，不光是我，我相信海边的每一个人都看见了，而且瞧得真真的。
    一个巨大的发光物，拖着长长的尾巴，从云端坠入到了海天的交界处。
    脚下在震动，由弱到强的震动。我看见一朵浪花在遥远的海平面升起，绽开，然后像滚雪球一样越变越大，最后变成了一座山，朝着海滩打了过来。
    海啸！那巨浪源于海啸！好容易来次海边，没遇到来拍泳装写真的美女也就算了，居然碰上海啸！还玩笑呢吧？！
    但是人们都没有逃，不但没有逃，反而都呆在原地不动，所有人的目光都直愣愣地盯着流星坠落的地方，那些似乎在热切期盼着什么东西出现的目光似乎比太阳还要灼热。
    貌似大海的包容性远远超出了我的预料，之前还是数百米高的巨浪在推进的过程中渐渐变小了，等到那山一般的浪头打到沙滩上的时候，已经变成跟一次普通的涨潮退潮差不多了。
    但我眼中看到的东西，比山要高得多，巨大的多。
    就好像海底突然冒出一座山来一样，我看见水平线上冒出了一个头。
    为什么是头？
    因为随着那东西一点一点的由远及近，我看到脖子——脖子上面岂非就是头？
    等到那东西又近了一些的时候，我渐渐看到了更多的部分。等到那东西的绝大部分都已经浮出海面的时候，我窒息了。
    因为那根本不是东西，那是一个女人，一个浑身赤裸的女人。
    湛蓝的海水将将没过她的膝盖，海面之上是一双修长而笔直的腿。她的臀丰满而浑圆，腰很细，胸饱满而坚挺。海浪打碎在她那洁白而细腻的肌肤上，在阳光的照耀下折射出水晶般的光彩。她每向前迈出一步，我都能感受到从海底传来的震颤，一直传至心灵。以她如此巨大的身躯，在海中漫步的身姿竟看上去无比轻柔，无比动人，让人一时忘记了她的巨大，眼中只留下她的美。

# 3

    但这些都不是最主要的，随着我的目光继续上移，我看到了她的脸。
    没有人能形容这是一张怎样美丽的脸——美的让人窒息。
    这是一张亚洲人的脸，或者说，是一张每一个东方人看上一眼就会终生难忘的脸。这张脸上，你绝对挑不出任何瑕疵——何况这又是一张带着微笑的脸。
    有人说，微笑是瞬间的，如果一个人的脸上时常挂着微笑，那一定显得很做作。
    但这张脸上的微笑，你非但不会觉得丝毫做作，甚至会认为笑容本就该属于这张脸。如果这脸上出现了其他表情，反而会让人觉得别扭，尤其是那笑容带起的眼角旁的微妙弧线，更让这微笑有一种摄人心魄的美。
    这美丽的巨人就这样一步，一步地走向海边，走向沙滩，走进所有人的心里。
    因为这是一个靠海的城市，所以在海滩上你不光可以看到出海的渔船，也可以看到巡逻的舰船。就在美丽的巨人一步步向海边走来时，已有5搜巡逻舰船迅速向她驶去，正面三艘，两侧各一艘，似乎试图要包围她一样。
    哎，我叹了口气，不光是我，恐怕在场的所有人都希望这几艘小船别被她带起的海浪掀翻就不错了。
不知道为什么，这里的所有人，包括我在内，都没有对这个无比巨大的美丽女人产生丝毫的恐惧心理和敌对情绪，仿佛她是一位远道而来的贵客。这位贵客看到迎接她的几艘小船时，便俯下身去，慢慢伸出右手，用两根指头轻轻捏起了一艘船。
    她的手指如玉，指尖的弧度非常柔美，那小船在她手中本该向酥饼一样就被轻易捏碎，可是她没有捏碎它。她只是把船捧在手心里，船里的人此刻是惊慌？是恐惧？是晕眩？我不知道，我只看到那美丽的女人凝视着手心的小船，仿佛在和船上的人做心灵的沟通一样，过了一会儿，便又轻轻把船放在了海面上，轻的好像生怕溅起一片浪花。
    她本就毫无敌意，对吧？
    她本就比这世上任何一个女子都要温柔，不是吗？
    巡逻船散开了，刚才还仿佛是要驱逐这高入天际的女人，现在却围绕在她的双膝周围，变成了开道的护航舰。
    我感到脚下的震颤变得大了些，原来这巨大的美人已经来到了海岸前，我已经可以看见她的脚踝和在海水中时隐时现的脚背。我抬头仰视她完美雕塑般的胴体，所有应该暴露的地方，和所有不应该暴露的地方，全都一览无余。她的脸已经完全看不清楚了，因为那高度实在是太高了，我想只有站在城里的高楼大厦中才能仔细端详这张美的不属于人间的脸吧。只有那坚挺的乳房，还有傲然挺立的乳尖，在白花花的日光下显得分外的耀眼。
    我忽然萌生出她和城里最高的大楼乳交的念头，然后又随即打消了这个荒淫的想法。当一个女子赤裸裸的出现在我面前时，我那些色色的想法反而逐渐消失了，只剩下纯粹的欣赏。
随着空中呼啸着的直升机螺旋桨的噪声，还有不断赶来的军队车辆的鸣笛声，原本热闹的海滩显得更加喧哗，但这份喧哗很快被一道道禁止靠近的黄色警戒线打断了。
    我站在远处，看到盘旋在那美丽的女巨人肩头和腰部的直升机，又想到消失在水平线上的那颗流星，忽然间很想跑到她的脚下，大声地问她一句话。
    你就是外星人吧？

# 4

    “她”登陆已经三天了，三天之中一直很安静，而我也在一旁的山上远远地看着她——这座城市不但靠海，而且多山。我发现自己已经被这百米多高的——也许还要更高得多——的美丽女性深深吸引住了，我甚至向老板请了年假，只是为了在山上搭个帐篷，好方便自己每天都能看到她。我觉得她即使是坐着，也要比这城市里大部分建筑物要高很多，而且她的身体仍然是赤裸的。
    她静静地坐在海边，低头看着自己身边这些小虫一般的人类忙来忙去，她有几次忍不住要抓起一两个来仔细端详，可是手指还没接触到他们时就有无数的枪炮对准了她，她也很知趣地收手了——虽然我相信这些东西对她造不成一丁点伤害。
    一周过去了，军方不知从哪儿跟她弄来一件衣服，但在我看来，那与其说是衣服，倒不如是一件特大号的比基尼。那巨大的衣服被几架军用直升机吊着，送到了她的面前，而当她穿上之后也不过是勉强遮住了那几点不该暴露的地方而已。
        15天过去了，一直闭口不语只是微笑的“她”居然开口说话了，她说的第一句话竟然是：
    “我想四处走一走。”
    天啊，一个像你一样巨大的女人，莫说是在城市里走一走，光是你现在站起来就足以引起恐慌了吧！
没有等到答复，她居然就这样站起来了。现场所有人好容易习惯了她坐着时候的高度，现在这位巨大的女人忽然间站了起来，仿佛瞬间又长高了一倍，起身时衣服带起的气流让一直盘旋在她周围的直升机几乎失去了平衡。
    透过望远镜，我看到她两步就迈出了100多米的警戒范围，果然朝着市区的方向走去了。我想，现在那些一直监控她的人一定乱成了一锅粥。我如果再在这小山丘上坐下去，可就离这巨大的美女越来越远了。
发动引擎，挂档，加速。我的小车尾随着那双通天的长腿，紧跟其后。
    车身随着她每一步踏下去的时候，都会颠簸一下，那感觉就像开在山路一样，我相信不光是我，所有在她周围开车的司机都会有这种感觉。我想她的脚得有5,6辆车子那么大吧，每跨出一步就超过了几十辆车子。好吧，好吧，我知道那些司机已经吓的不知道是该踩油门还是踩刹车了，好在这十五天里无论是新闻还是广播，都每天24小时不间断地滚动播出她的动态，市民看来已经有点习惯这位巨大的女人做他们的邻居了，只是今天邻居要来串门了。
    邻居来串门，主人们却吓得不敢动了。
    她好像刻意避开了车流和人流拥挤的区段，想想也是，从她那个高度往下看去，这城市的交通状况还不是一览无余。她边走边仔细观察身边的建筑，一般的公寓楼刚过的她的脚踝，她小心翼翼地迈过去，遇到一些写字楼或者大型商场等能到她腰部的建筑，她会停下脚步，蹲下身子来用手轻轻地摸着楼身，似乎这些小人般的建筑物提起了她极大地兴趣，看得她眼中好像放出了光。每到这个时候，我就把车子停在一旁，仰起头来慢慢欣赏一个巨大女人抚摸高楼大厦的绝妙风景。
    当她发现和自己身高相仿，或者比自己还要高的建筑时，那种难以言表的兴奋劲头似乎更加强烈。如果你问我是怎么知道的？我只能说，她胸前的衣服有两个非常明显的凸起，这难道不是兴奋的表现么？
她会将整座楼身摸个遍，仿佛是抚摸情人滚烫的身体一般，然后还会试图用双手扶住楼顶的两侧，轻轻地晃一晃。天呐！这不是要了建筑物里那些小人的命嘛！好在她对力道的拿捏十分准确，到现在还没有晃散一撞建筑，否则这城区里早就四处废墟，尸横遍野了。
    我在车里仰望着这个比摩天楼还要高，比晚霞还要美的女人，竟看得痴了。
然后，就是一通警笛的鸣叫，还有直升机的轰鸣破坏了此刻的温馨，那些监控她的人再一次将她包围了。
哎，你们这些人就不懂得什么是浪漫吗？
    “她”又被“请”回了海边，继续像一只安静的小猫一样坐在海水中，时不时地用手指尖碰一下舰船，车辆还有那些小人，可我在望远镜中看到了她的眼神中有了一种异样的深邃，那种眼神之前完全没有出现过。
    后来我才回想起来，她的笑容在那一刻似乎也发生了变化，但当时的我完全没有察觉出来。

# 5

    没过多久，各大媒体纷纷报道了一则消息，在那位巨大女人出现的海域将进行一场军演。我啃着面包，看着不远处那个正在玩弄一辆装甲车的美人，忽然间又多愁善感地幻想巨大而又美丽的女神消除世界上所有战争的美妙画卷来。
    演习如期举行，只见一个航母编队浩浩荡荡地从远海驶来，耳机中新闻直播传来的什么驱逐舰，巡洋舰之类的海军名词我完全搞不懂，也没有兴趣搞懂。我只知道望远镜中那位一直安静而优雅地端坐着的女性，慢慢站起来了，然后朝着那航母编队走去。
这次军方好像是有所准备一样，并没有惊慌失措，只是让数十家武装直升机在她头顶盘旋，数十艘舰艇在她身后尾随。
    我根本无法形容出当时的壮观场面，因为根本没有人能想象，一个女人，一个无比美丽的女人漫步在湛蓝的海水中，海水只不过没过了她的细腰，她的头顶盘旋着直升机，她的后面有军舰，她的左面有军舰，她的右面有军舰，她的前面有军舰，还有航母。
    最要命的是，每一艘军舰跟她比起来就像玩具一样。
    更要命的是，即便是航母，在她身前就像是一艘独木舟！
    在这种时候，在这种地方安排军演的人疯了吗？！
    耳机中继续传来了现场直播的报道，而我只听到了四个字，便跳起来扔掉了耳机。
    “实弹军演。”
    就在我的心中笼罩着的那片不祥阴云还未散开时，眼前发生的景象已经让我完全绝望了。
    几乎在同一时刻，所有的舰艇，所有的直升机，所有的枪炮，开火了。
    炮弹在她光洁如玉的肌肤上炸开了，火光，烟幕，就像烟花一样，璀璨而夺目。
    炮弹一发接着一发，似乎不全弹消耗完就不会停止一样。
    这一刻，我手中的望远镜掉在了地上，我相信所有人手中的望远镜都掉在了地上。
    十分钟，足足有十分钟，直播也被中断了十分钟，这期间到底发生了什么，已经没有人能知道了。
    每个人能知道的只有一件事情，她还活着，像十分钟前，像十天前那样美丽、端庄、优雅的活着。
    十分钟的炮火宣泄，似乎只让她洁白如雪的肌肤沾染了一点烟尘，所以当她一头钻入海中，又钻出海面的时候，她就像第一次出现时候一样美丽、端庄、优雅，溅起的海浪让她身边那些玩具一样的舰船摇摆不止。
    我傻了，我不相信眼前发生的一切，我只知道她还在微笑，而且笑得比以前更加好看，好看的让人心里发毛。
    后来，官方给出的解释是：所有舰船的电子系统在同一时刻被干扰，导致武器系统全面失灵，无法控制。但我只知道，永远不要轻信一件事情，直到官方出来辟谣。
    军演结束几天后，官方忽然宣布要为上次军演的失误埋单，为了补偿这位温柔巨人受到的一点点惊吓，特意为她准备了一份礼物——允许她做这个城市的荣誉市长。我叹道，这种蠢主意不只是哪个猪脑袋想出来的，难道你被人一顿几乎是要命的胖揍之后，给你个荣誉市民你就一笑而过了？但眼下这位巨大的女子，似乎非常乐于接受这份殊荣。

    他正聚精会神地瞄着一个人，一个女人，一个无比巨大的女人。
    她的臀翘的很美，很性感，正好抵在大楼的墙壁上，而包裹着巨大而又性感臀部的只有一件不比内裤大多少的布料。也许正是因为这美的令人窒息的曲线夺去了那男人一瞬间的注意力，所以我才能趁机去夺他的那把枪。

    那男人已经用匕首地主了我的喉咙，我的喉头咯咯作响，一句话也说不出来。显然我的喉咙就要被刀刃划开，热血即将喷出，生命随之流逝。
    我只是想在临死前再看她一眼，而已。
    于是我用尽生命中最后一股力量，挣脱了他钳住我喉咙的铁锁一般的手臂，相应的，我的身体向反方向飞了出去。

最后，我看到了她的笑——对我的微笑，那时我便知道，我死不了了。

随后，官员们莫名其妙的结束了热闹的仪式，随后，她也默不作声地向海边走去，又像一只全世界最乖最乖的小猫一样，坐在了海边。

    我抬起头，看着天空中这张无暇的脸，叹了一口气。
    “难怪你这么美，美的好像不属于人间，也难怪你并不介意被一个陌生男子偷窥。”
    她嫣然一笑：“我叫你来，一是为了谢谢你，二是为了报答你。”
    她摊开半张篮球场那么大的手掌，掌中有一个在月光下微微闪着银光的灰尘一样的东西浮了起来，掉在了我的手上，赫然是一颗子弹。

    我点点头，我从一开始就相信，地球上的科技是不可能对你造成半点伤害的。
    “那为何你还要做那种傻事？”
    我不知道，我真的不知道，也许那一刻是我的身体下意识做出的反应，也许只是因为你是个女人。
    男人保护女人，岂不是天经地义的事情？

    “因为你的出现，官方的一部分人感到了恐慌，他们处心积虑地想除掉你，所以才组织了那场演习。”我抬头对她说道。

    她是她们星球的公主。
    她所在的星球，资源已被开发殆尽，环境也日益恶化。
    为了生存，她需要将她的子民移居到新的星球。
    所幸她的星球上有足够发达的科技，足够强壮的战士，所以很快她便找了其他适合居住的星球。
    但是，这些星球上同样也有生命，和她们类似的智慧生命。
    她憎恨赤裸裸的侵略，憎恨依靠武力的掠夺，憎恨用鲜血和生命换来的生存空间。
    但是为了自己的子民，她不得不去占有那些本来不属于她们的星球。
    于是，她和元老院中的元老们争吵不停，直到双方达成了一个折中的协定。
    “如果一个星球上的生物是肮脏而丑陋的，那他们就不配生存，我们用武力占有这个星球的行为就不是侵略，而是履行王的道义！”
    而执行这判断的人，就是她。
    她们已经占有了数十个星球，元老和子民都相信她的判断，事实上她的判断也从未出错。
    她这一次的目标，就是地球。
    “幸好有个男人一直在偷窥着你却没有心存邪念；幸好有个男人在关键时刻遇到了另一个端着枪的男人；幸好这个男人以为自己能够英雄救美，却反而被美人所救；幸好这个男人，就是我。”我笑道。
    她也笑了：“幸好，他在临死的愿望，不过是再偷偷看我一眼。”
    我又觉得脸上在发烧，却没注意到这位顶天的公主大人两颊也闪过一片不易察觉的绯红。
    所以，你才要报答我，我也竟在无意间成为了救世主。
    “你们。。。地球上管两个人得结合。。。好像叫结婚。。。是吧？”
    我从没想到这位公主大人会忽然间冒出这么一句话来。
    “嗯。。。是的。。。不过轮到我结婚，估计要十年之后了，哈哈。。。。。。”我尴尬地笑道。
    夜已深，月更圆，漫天的星斗织起了一张珠帘，盖在了公主的脸上，让我瞧不真切她的脸。
    “我要走了，再见。”
    她只留下这句话，便消失在一道光中。来得如此突然，去的也如此突然。
    我只能呆呆地望着无边的星空，也不知道就这样站了多久。
    第二天，NASA撤换了太阳黑子的照片，并解释照片上那些所谓的上千艘巨大太空飞船不过是压缩图片造成的。看到这里，我又想起了那梦一般的巨大公主，不禁笑了笑。
    “永远不要轻信一件事情，直到官方出来辟谣。”

    太阳系星门附近，一艘看上去像是旗舰的超巨型宇宙飞船正在准备空间跃迁，指挥舰桥内，一个娇滴滴的声音说：
    “公主大人，看来这个星球有些与众不同呢。”
    “或许吧，副官，务必记好这次航行的坐标，这个星球是重要的资源点，我们今后还会回来复查一次的。”
    “大概多久以后呢？”
    这时公主的声音发生了一些变化，但是那位年轻的副官并没有听出来。
    “十年之后吧，结婚什么的。。。。。。”
    “哎？！！结盟？！！您要跟这个星球结盟？！”
    “你听错了副官！即刻起航！”
    “是。。。。。。”
