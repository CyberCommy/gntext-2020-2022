# 1

各位大佬们，看官老爷们好，潜水两年以上的我今天终于决定要写文了......第一次写文，文笔一般，还请各位多多包涵。

序幕
我的名字叫苍，是一枚重度足控，又有一点M属性。家有娇妻一枚，名叫梅露，外表娇小柔弱，软萌可爱。她那雪白娇嫩的可爱玉足，经常沦落为我晚上发泄的对象。然而，不久之前，我已经不满足于每日每夜重复着舔脚足交的玩法了。通过一些特殊渠道，我得到了GN牌缩小器，可以缩小指定物品（或者生命体）至指定倍数，并且被缩小的生命体在失去生命体征之后可以复活。于是，每天趁着老婆不注意，我就偷偷的钻进她刚穿过的鞋子或者袜子中去手冲，冲完之后再钻出来，好像无事发生，因此也从来没被梅露发现过。可谁知，不久前，意外发生了......
第一章
和梅露一起完成日常两公里的夜跑，我们终于回到了家中。“呼啊，累死我了”只见梅露四仰八叉摊在沙发床上，两只小手像翅膀一样向脸上扇着风，希望能吹干脸上的汗水。边上搁着一双刚脱下的白袜，袜底已经被踩的发黑，凑近一看仿佛还能看见一缕热（jio）气。
“不行了不行了，我要先去洗澡了，你等我洗完再洗嗷！”梅露连拖鞋也不穿就光脚走向了浴室，光滑的大理石地板上留下了一连串的可爱小巧的湿足印。
“咔嚓”待听见梅露把浴室的门锁上之后，我偷偷的拿出来了藏好的缩小器，坐到沙发床上，白袜的旁边，把倍数调整为15倍，对准了我自己. .....“biu”的一声，我变成了12cm的小人。
梅露洗澡时间大约是四十分钟左右，我手冲的时间一般是15-20分钟左右，时间还算赶趟儿，不过还是得赶紧整活儿了。
我迫不及待的脱下了裤子，朝着白袜里面钻去。袜底上沾着的脚汗还保留着余温和余香，越往深处爬，温度和湿度就越高，气味也就愈加浓厚。爬到脚尖处的位置，我一边开始了手中的针线活儿，一边还不忘舔食着棉袜中，由老婆可爱的小脚上落下的角质和死皮以及脚汗构成的脚垢。大约过了二十多分钟，我的阿姆斯特朗回旋加速喷气式阿姆斯特朗炮终于射出了白色的炮弹。与此同时......
“老公，我洗完了，你也来洗叭！”恰巧我射精完，梅露便洗完澡从浴室中出来了。我的内心突然收到惊吓，下体瞬间痿了下去。
怎么办！我怎么也没有预料到梅露今天洗澡时间竟然比平常短了一半，要是被她发现，又要怎么解释啊......我只好祈求梅露洗完澡回卧室玩手机。
然而，事与愿违，我只听到外面的脚步声越来越大，离我越来越近。最后，梅露提起我所在的袜子，失重感传来。梅露掂了掂我所在的袜子，将我的姿势由趴着变为平躺着，又变为倒立着，头在袜底，冲着袜口。“怎么感觉袜子变重了，是我的错觉嘛？算了不管了。”最后，我的视野被湿湿的，粉红软嫩的巨大脚掌脚趾所占据。脚趾仿佛推土机一般，一张一合的向我推进。
脚趾掠过我的双腿直达我的小脑袋，脚底上带着的汗水和洗澡没擦干的水全都抹在的我的身上和脸上，巨大的脚掌狠狠的踏在了肚子和大腿上，脑门被食趾紧紧扣住。我的视野中一片漆黑，恐慌之中，我只能嗅到梅露足底上沐浴露的香味和脚趾缝残留的脚味的混合味。
又是一阵失重感，“啪”，身上的巨脚毫不留情的踩下，将脚下的小生命体踩至半死。浑身传来巨痛，完全感觉不到双腿以下的部位了，是被踩成浆糊了么？腹中的器官也基本被踩的失去功能，大脑也因此受到冲击，险些晕厥。殊不知，这只是梅露坐在沙发上施加了一条腿的重力所带来的伤害。
“苍这个笨蛋，叫他去洗澡怎么还不去，又在卧室打他那足控黄游呢吧，我现在就去卧室好好的收拾收拾这个笨蛋！”身上的玉足突然增加百倍的重量，把原本半身不遂的我瞬间踩成浆糊......
第一章写到这里就先结束了啊，短短一千多字却是我从不到十二点构思到经历四个多小时才写完的，具体后面的发展我目前还不是特别有思路，恳请看完我第一章的看官们大佬们给我提提素材建议。因为疫情问题推迟开学，寒假又没有作业可写，大概这篇文不会鸽掉叭！
