# 1

随着人口越来越多，除了农业人口外，剩下的人都住进了高达云霄的巨大楼城中，王秀便是一个住在高楼几万层上的一位技术人员。不知道为何，高达几万层的楼城中只有运送货品和垃圾的通道，却没有供人上下的电梯，必须步行来上下楼层，所以从王秀爷爷那一代开始，就没有下过高楼，在工作之余，王秀就一直想要到没见过的地面去看看，最好还能看到传说中的大海，可惜他身边的朋友没人愿意和他一起。

“王秀，在楼上待着多好，要啥有啥，为何要下去看所谓的大地呢，而且听说越往下的越高大，而且还很粗鲁，如何能和楼上那些娇小文雅的小姐们相比？”他的好友，刚和他们上面30层上一个娇小可爱的姑娘定亲了的里德劝他道。

“而且，你忘了当年的事了吗？”里德看王秀还是想下楼去，只好说起了1年前的一个趣事，当时有一个人估计也是想去楼下看看，走到了自己所在层的600层以下，结果发现在该楼层而言原来高大的他如同一个婴儿，本来是去在自己交的小女友面前展现自己的雄风的，结果被那小女孩逆推了，差点被那小女孩就养在了家里，还好趁那小女孩一次出去跑了出来，也因为这事，机构专门发了个公告，让人们最好都在上下50层内活动，而且不要和层数相差太多的人谈恋爱，免得离开所处层太远而由于体型差距过大发生各种问题。

“放心吧，里德，我在网上联系的都是不错的朋友，到时候住在他们那里挺让人放心的。”王秀对里德说道。里德看劝不了他，只好再三叮嘱王秀小心，也让他开完后回来和自己好好说说路上的见闻。王秀在单位请了假，收拾好了必须品，便带着期待开始了自己的回归地面之旅。

“我这该下了300层了吧。”王秀拿着导航系统在楼梯口那边休息，毕竟是下楼梯，还是比较轻松的，他也没有着急，出了楼梯开始找起了自己在这层所订的旅馆，他看着路上走着的人群，在直接的视觉效果面前还是惊讶的张开了嘴，虽然本来他就对低楼层的人们高大有了心理准备，但真看到了还是挺让他震撼的。本来身高1.8厘米的他在自己的楼层已经是非常高大了，可路上来往行走的路人都比他要高大太多，就连那些行走的小女孩他也只能勉强到肚子的位置。

“看样子得具体了解一下身高的差距了。”王秀心里想到，毕竟还有几万层要爬，如果按这种差距，恐怕后面的楼梯都会非常难爬下去了。王秀往自己订的旅馆走去。

“王秀，你终于到了啊。”一个女孩跑过来把王秀搂在了她的怀里，“你看上去真的好可爱。”王秀努力推开了这个女孩，哭笑不得的说，“丽丽，你太用力了。”“对不起，我太激动了，上层的真的像宣传的那样挺小的呢。”那女孩低头看了看王秀，吐了吐小舌头可爱的说道，说完帮王秀拿起行李，带他来到了所订的房间。王秀进了房间便往床上一躺，不想动弹，毕竟一直做技术工作的他第一次走了这么长时间的路确实感到很累。

丽丽也进来了，坐在床边帮他按摩脚，“王秀，你的脚比女孩子还娇小细腻呢。”丽丽赞美王秀的脚道。“是你们太大了。”王秀无奈道。“对了，丽丽，你身高多少啊。”王秀问丽丽到。“人家才不大呢，才2.3厘米高，是你太矮了。”丽丽争辩道，还故意用力捏了下王秀的小脚。

好高，王秀心想，在自己的楼层还没有这么高大的人，而在300层下，这样的身高居然只是一个娇小的小女孩，那再往下走，人又该多么的高大。王秀不禁思考起来，越往下，人越高大，那后面人该有多么高大，自己的路又该怎么走下去呢，王秀不由产生了一点迷茫。

“秀哥哥，你在想啥啊，说实话，你说要下来时我真非常高兴呢，毕竟很少有人离开100层以下的。”丽丽看王秀在发呆，用在网上对他的称呼说道。“我也很高兴见到你啊，丽丽，谁能想到一直说自己娇小的丽丽居然如此的大。”王秀调侃她道。“讨厌了，秀哥哥，是你太小啦。”丽丽躺在王秀旁边，气呼呼的道。

这时候门口传来了门铃声，丽丽去打开了门，原来是她的男友小何来了，两人招待王秀吃完了晚餐便告辞离去了，留下了王秀在房间里休息。“今天第一天，所以我只下了300层，而明天我得下500层，中午就在楼梯那吃了，主要是到时候要去见的里奈该是多么高大啊。”王秀期待中又带着点恐惧，毕竟丽丽和他只是朋友关系，而里奈却是网上和他聊出了感情的女孩，也可以说他想回归地面计划最支持他的一个人，不过看到了丽丽和他的差距后，他不由担心起到时候里奈如果强行将他留下该怎么办了，算了，不想了，温柔的里奈应该不会这么做的，王秀安慰自己道，然后早点休息进入了梦乡。

第二天早上丽丽招待王秀吃完了早餐，帮他准备好了中午的便当，送他去了楼梯那儿。“回来一定要再来啊。”丽丽招手对他告别道。王秀对她点了点头，继续了自己的旅程。随着往下的行进，王秀明显感觉到了楼梯的变化，变得越来越大，估计都要有原来的两倍大了，这也使他越来越吃力，就在他不怎么吃得消的时候，看到了下面楼梯那有一个熟悉的身影，是里奈，她穿着他在视频里曾经看到过的衣服等在下面，让王秀又有了力气，努力往下跑去，随着靠近，里奈在他面前也越来越大，等他下了楼梯到与里奈一样地面的时候，他甚至只比里奈的膝盖高了一点点，里奈这时候正在玩手机，没有注意到他。王秀抬头看着高大的里奈，不禁想要逗逗她，便用手机拨打了里奈的号码，“里奈，你在干啥啊。”“我在等你啊，小秀秀。”自从里奈知道王秀所在的层数比自己高800层后便一直这样喊王秀。

“你太好了，里奈，可我今天路上有变故，估计到不了了。”王秀低声说道。“怎么了，你没受伤吧，现在在哪。”里奈紧张的说道。“哈哈，被我骗了吧，看你的脚边。”里奈低头就看到在自己脚边抬头对自己微笑的王秀。“你居然敢骗我了，小秀秀。”里奈好气的笑道，用一只手就轻易的托着王秀的屁股把他抱在了自己的怀里，看着王秀身上的汗，怜惜地拿出块手帕帮他擦干净，带着他回到了自己的房子。

可是回到了里奈的房子后，里奈将王秀的行李放下却没有带他去床上休息，而是直接带他到了洗浴间。里奈将王秀放了下来，自己也蹲了下来，盯着王秀看，“里奈，这是干啥啊。”王秀被里奈看的不自在了，只好问她。“当然是帮你洗澡啊，快脱啊，还是说小秀秀不会脱衣服要我帮你啊。”

“你这样看着，我怎么脱啊。”王秀无语的看着兴致勃勃的里奈。“你在网上不是说要和我坦诚相见吗，结果路上就骗我，难道你在网上说的话全是假的啊。”里奈幽怨的看着王秀。王秀被她看得受不了了，横下了心来，不就是脱吗？就脱掉了自己的衣服站在了里奈的面前。“小秀秀，你皮肤好好。”里奈用手抚摸着王秀的身体，欣赏的说道，不过还好她没有做再多动作，让王秀松了一口气，不必纠结于反抗还是不反抗。里奈倒满了一盆水后把王秀抱了进去，拿了条毛巾给他，“嘻嘻，小秀秀，你把身体洗干净啊，我洗完了在床上等你哦。”说完里奈拿着王秀脱下的衣服就笑着离开了这个房间到另一个洗浴间里面去了，只留下了王秀在这个水盆里。王秀听着传过来的冲水的声音，压下了心中想去看看的欲望，躺在水盆里放松下身体慢慢洗了起来，洗完后王秀披着一条毛巾走出了房间，往卧室走去，显然考虑到他的体型，里奈把门都开着，王秀来到卧室，努力爬上了床，发现里奈正躺在床上，这时候里奈不像迎接他那时穿着长衣长裤，而是只穿了内衣，那双修长的白腿让王秀的眼都看直了，里奈看着发呆的王秀，嘻嘻笑着，把那双比王秀身体还长的腿抬了起来，诱惑着王秀，王秀实在忍不住了，松开了毛巾，上前抱住了里奈的一条大腿，整个身体都在上面摩挲了起来，很快下体就有了反应，“哈哈，小秀秀，你皮肤好光滑，还有个突起戳的我腿上痒痒的。”

感觉着王秀身体的变化，里奈把他抱了起来，更加起劲的挑逗起他来，很快王秀就忍不住射了出来，“这也太早了吧，不过也难怪，小秀秀你太弱了。”里奈皱着眉头用手指碰碰王秀的老二，说着让王秀脸红的话，“是体型的问题，你和我一样大试试。”王秀脸红道，这时突然想起还没问里奈的身高，“对了，里奈你多高啊。”“我才4.5厘米高啊。”里奈回答他道。800层下的女孩居然有他的2倍半高，王秀对自己下楼回地面的难度有了更深的认识，不由沉默了下来。

“怎么，现在小秀秀知道下楼的难度了？就留在我这算了。”里奈看王秀沉默了下来趁机挽留他道。“不，我一定要地面看看。”王秀坚持道。“就知道你会这么说，小秀秀，其实是有办法克服这困难的。”里奈说道。“真的吗，快说说看，好里奈。”王秀对里奈讨好的说着。“有些体型差爱好者就喜欢到不同的楼层去体验，你可以让他们带你下去的，几天后正好有个队伍想下去体验，你可以跟他们一起下去的。”里奈抚摸着王秀的头回答道。“里奈，你太好了。”王秀爬到里奈的头部，亲吻了下她。”

“哼，你别忘了我就好。”里奈说完按住王秀的头，用力吻住了他，直到王秀以为自己要窒息了才放开了他。接下来几天王秀就在里奈的家里度过了难忘的几天，直到里奈所说的下去的队伍的到来。里奈说的队伍五男三女，都是想到这一层下面1000层下去会见他们约好的伙伴的。显然他们准备的被王秀充分多了，居然能利用运送货物上下的通道下去，队长居然是里奈的弟弟真介，难怪里奈知道这次活动，里奈对他再三嘱咐，才把王秀交到了他的手上。

王秀便被真介抱着来到了一个大箱子那里，整个队伍走了进去。“秀哥，你真要去地面吗？”在箱子里大家都没事，真介好奇的对王秀说道，其他几个人也好奇的看着他，显然经常去下层的他们也没见过上层的人，都对王秀要去地面非常感兴趣，“当然了，这是我的梦想。”“可是听说越往下人越巨大，而且还有巨大的猛兽在，非常危险，我们上次跟随1000层下的朋友去2000层下的地方就非常害怕，都没敢离开朋友过。”说到这儿，几人的脸上都出现了害怕的表情，显然那次经历让他们心有余悸。“我会小心的，对了，真介，那下面的人到底有多高啊。”王秀心里一动，便问真介道。“我是4.6厘米高，而1000层下的人有13厘米高，2000层下的人足足有25厘米高啊。”真介回答他道，让只有1.8厘米高的王秀感到了巨大的压力。

就在真介和王秀他们聊天的时候箱子振动了一下，然后停了下来又动了起来，“应该是到地方了，正在被人搬运到樱子也就是我们会长的家。”真介对王秀解释道。
