# 2

人类开始放弃生存。

在生存竞争中胜利，站上地球生态系顶点的人类。

他们曾梦想着向太空扩张，创造新的世界，但如今往日的生气已无影无踪。

世界上的人们怠惰度日。人类失去了对未来的希望，数量不断减少，慢慢走向灭亡。

国家间的纷争、文明的发展竞争之类如今已经成为过去式。

人们在麻木中度过每分每秒，混着日子等待死亡。

一切都是因为数年前那个东西的出现。

即使是现在，那个东西也堂堂坐镇在那里。

没有任何征兆地，某个来自宇宙尽头的不速之客，落下了一件失物。

翻滚掉落到欧亚大陆上的，一只全长足有7200km的巨大室内鞋。

鞋面由白布制成，鞋底则是酒红色的橡胶，一只女孩子用的室内鞋。

脚背的地方写有名字，但是地球人无法辨认的文字。

它的样子看起来有点旧，显得有些脏。并非是主人不爱干净，而是被长期使用留下的痕迹。

鞋的内侧着地，横卧着矗立在欧亚大陆上。

宽2700km的这只小行星级的超巨大室内鞋，在日本也能看得清清楚楚。

连地球的弧度，也无法盖过它的巨大。

它的落下给地球带来了无可估量的破坏。

欧亚大陆上的所有国家瞬间被毁灭，袭向地球全域的大地震与海啸将地球的人口数削减至不足原来的十分之一。

所有的人类文明就此崩溃，寥寥无几的生存者，站在已经天翻地覆的地球上，茫然失措。

据幸存下来的学者计算，这只室内鞋全长7200km，宽2700km。

是地球人使用的室内鞋的3000万倍大。

看到得出的计算结果，人类重新认识到这只超巨大室内鞋的恐怖。

然而更恐怖的是，宇宙中还存在使用这样巨大的室内鞋的人。

全长7200km的室内鞋。

这是足以竖着放下两个日本的大小。

整整一个国家，可以被一只室内鞋完整地容纳。

使用这只室内鞋的，究竟是一个多么巨大的存在啊。

女孩子用的室内鞋。

在地球人看来足以容纳国家般巨大。

也就是，反过来说，在这只室内鞋的主人——也许是一名少女——看来，地球上的国家只有可以放进鞋里般渺小。

日本在少女眼里不过10cm长。

穿着同样室内鞋的地球人，无论如何努力生长，也不过0.0006mm。

甚至不足头发直径的100分之1。

我们的地球在少女看来，也不过是40cm多一点的球。

甚至没有一个平衡球大。

不，如果穿着这么巨大的室内鞋的少女坐上来，地球一定会承受不住被坐坏掉吧。

宇宙中还有这么巨大的人吗…

直到刚才还站在生物的顶点，不知天敌为何物的人类。

这个认识在他们的心中种下了无尽的绝望。

甚至将幸存下来的人们生存的希望都夺走了。

然而就算还有希望，从此往后的生活也会变得及其痛苦吧。

所有的文明都崩溃了，无法再过上以前那样的生活。

亲密的人也都不在了。

环境也已经被完全破坏。

而且最要命的是，空气中的异味。

从这只横卧在地球上的超巨大室内鞋中，散发出混合了橡胶、布的气味与大概是室内鞋主人的少女的脚臭和汗臭。

这股气味乘着大气环流，随风渗入世界的每一个角落。

无论逃到地球上的哪个地方都能闻到这股臭味。

即使鞋已经落下数年，这股臭味依然没有消散。

仅仅偶然飞来的少女的一只室内鞋里的气味，就让存活下来的全人类痛不欲生。

不过是掉下一只室内鞋，地球已经濒临毁灭。

＊

又过了数年。

“啊～！ 居然掉在这种地方了～”

一个声音震响在地球上。

地球因为这个声音而颤抖，整个世界都受到了地震的冲击。

数量比数年前更加稀少的人类，听到这个声音望向天空。

他们看到了一名少女，从天空无比遥远的彼方正在靠近这边。

随着她的靠近，她的身躯逐渐变大，最终占据了整个天空。

“真是的～害我好找～”

苦笑着的少女把穿着白袜的其中一只脚伸向地球。

一大片区域瞬间陷入黑暗。

只因为那只超巨大的室内鞋的主人，把脚伸了过来。

她恐怕是要用那只脚扶起室内鞋，然后穿进去吧。

天空被无边无际的纯白色填满，室内鞋附近国家的人已经无法看到原本的天空。

少女在地表降落，用袜足把室内鞋踢起，把脚塞了进去。

跟着在地面敲了敲鞋尖来适应鞋子，然后用力蹬地高高跳起，重新消失在了宇宙的尽头。

人类就这样灭绝了。
