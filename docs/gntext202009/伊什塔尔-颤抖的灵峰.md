# 伊什塔尔·颤抖的灵峰

作者：不动明

TID：26834

 

# 1

彼，伊什塔尔女神周游于世界。

是日，游历于艾比赫山脚下，逢一女祭跪地而迎。

“Ah～伊什塔尔，伟大的女神伊什塔尔。”

女祭歌曰，声如玉石，婉转圆润。

“您携宇宙法典归于乌鲁克，将世间万千神权皆握于手；

“民众群王跪伏于您的脚下，皆叹服于您的美丽与强大；

“您以金星为饰，俯瞰万物；

“群星皆因您之容颜而黯淡；

“Ah~伊什塔尔，伟大的女神伊什塔尔；

“您之权能通达万物；

“您之智慧知晓万理；

“吾歌吾颂，难达其意；

“句词韵律，难赞此圣。”

歌毕，余音荡于空谷，幽响莺莺。

伊什塔尔心中欢喜，赏赐于女祭之国家风调雨顺，女祭唱诺拜谢。

然，伊什塔尔行至半路，以千里眼观之，只见先前颂其神德之女祭，于此正拜祭于艾比赫山脚，为灵峰献祭牲畜，咏歌娱之。

女神顿感不悦，返程而归，见女祭慌得俯首以迎，怒而问曰:女神伊什塔尔与灵峰艾比赫比之，孰为伟大？

女祭惊恐，战战兢兢，口不敢言。不待其口中言语，艾比赫山骄而斥曰:“小小丫头，怎敢于灵峰争雄？汝矮小之躯，须昂首尚能仰望吾百丈之躯，安能在此饶舌放肆？”

女神见灵峰高傲，斥其矮小，心中恼怒，于无边神力灌注其身，轻呼:“长。”瞬时之间，娇艳圣躯直冲云霄，绝美玉体顷刻增长百万倍之大小。

彼时，伊什塔尔女神立于天地之间，万物于她足下宛若齑粉。

四方众国皆仰见天际女神巨足，仅见云层于女神趾下游荡，山峦于女神足边宛若砂烁，而女神娇美之躯，蔽于云层之上，凡胎肉眼难以及其。

慌得四海万民齐齐跪地叩首，惧怕触怒巨大女神，云上巨足踏下，将四方诸国一同踏平。

女神之怒，上惊天界，下扰冥府。

伊什塔尔傲然屹立，轻抬足趾，蔽日巨影遮于灵峰之顶。

“汝尚且唤我小小丫头？”女神问曰。

“今我再问，女神伊什塔尔于灵峰艾比赫比之，孰为伟大？”

女祭惧，伏地抢头，高呼伊什塔尔之名。灵峰亦惧之，仰其遮天玉趾，央求阿安娜神居中调解。

阿安娜闻之，无奈，劝阻伊什塔尔女神念其初犯，得饶且饶。伊什塔尔心中不悦，充耳不闻，反进而放大玉躯，稍顷，灵峰已如女神足下细尘，再无一丝幸存之兆。

“汝既得意于百丈之躯，不若叫我帮你一帮，以这万丈之趾，助汝成万丈之谷可好？”

女神嘻然，摇晃足趾以挑逗之。灵峰惧其神威，忙颂其美貌权能，谄媚不止。伊什塔尔不屑，参天巨趾悠然按下，天界震荡，地动山摇。

女祭伏地闭目，不敢视之，待地摇稍止，环首四顾，不见女神，亦不见灵峰，唯见昔灵峰耸立之处，今已化为凹地，望而不见边界，殊不知此新造盆地，只为伊什塔尔足趾之印罢了。
