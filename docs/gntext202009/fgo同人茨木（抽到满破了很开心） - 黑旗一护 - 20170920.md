# 1

今天的迦勒底也很和平呢......
“今天很闲呢，刷副本吧肯娘还没满呢。”
咕哒子伸了个懒腰站了起来，这时突然响起来敲门声。
“谁啊？”咕哒子问道
“我是东山奈央”
诶？迦勒底有这个人吗？咕哒子打开门，站在外面的茨木笑着和咕哒子打了招呼。
“茨木啊，这么早有事吗？”
茨木晃了一下自己手里的葫芦
“当然是来找master喝酒了，难道master忘了吗？”
啊，牵绊4级的时候确实说过.....
“那个...我还未成年。”
本来打算含糊带过感觉去刷副本来着，但茨木并没有走的意思。
“这个我知道，这个里面装的是饮料。”
什么嘛.......
“那我进来喽。”
没等我阻拦茨木就跑了进来爬到我的床上
“喂喂，上床要脱鞋啊。”
说出这句话的我发现自己错了，茨木根本是光脚到处跑的嘛。除了身上的黄色和服和手链茨木就什么都没穿了，一想到这发现自己说了失礼的话觉得不好意思。
“我就喝酒，master喝饮料就好了就当陪我了。”
我做到自己的写字台前，看着用山大王的姿势坐在我床上的茨木。她就像是在自己家一样开始倒酒和饮料，话说杯子你也带来了啊。
“master接下来准备干什么？”
说罢便畅饮了一口，我跟着喝了一下感觉这个饮料很好喝。
“打算去刷副本啦，弗兰克不是还很弱吗？”
“这样啊，我听说master和很多满破的英灵做了快乐的事了。”
噗！我一下吧饮料吐了出来。
“这事你从谁那听的......”
“saberLily呀龙娘呀还有马修”
“呀，她们搞什么！”
“是我逼她们说的哦，听说master是足控呢虽然有点红但是要舔吗？我也很高兴呢，毕竟人类就是这样的存在。”
虽然茨木这种萝莉少女是我喜欢的类型.....但是，毕竟是鬼我还是比较抗拒这点天生抖s。
“我看还是算了嘛，这种事以后再说拉比较现在还是早上。”
“人类你已经没有选择了。”
诶？突如其来晕眩感让我失去重心躺倒在地，眼前的世界再不断扭曲就像做了时空穿梭机的感觉一样。混乱、迷茫、无助的感觉袭来。
当我眼前的一切静止后我看见的依然是自己房间里的天花板，她给我下迷药了？我打算站起来突然周围突然传来轰的一声.....
“舔把人类，呵呵真是如同虫子一样呢。”
抬头望去，茨木赤红色的脚屹立在我的面前。要比喻的话就好像一只b叔那么大（两张双人床），由于长期光脚脚底下占满了汗液与污秽的结合物在受力的地方格外的黑。刚才没有在意但缩小后便闻到的臭味，伴随臭味的是温暖的气息就好比冬天的暖气。
我慢慢走到茨木的脚前，面全身趴到她的足底舔了起来。
“哈哈果然人类都是这样呢，master也不例外全身肮脏的虫子。你现在只有蟑螂大小我可以轻松把你踩碎哦，好好享受这一切吧。”
说罢茨木开始畅饮她的美酒，我也因此开始畅饮她的汗水。她每一滴汗对我来说如同豆子大小，每每冒出我都情不自禁的去舔砥。汗水与污渍还有脚味交融的味道，散发着炎热的刺鼻气息。全都是让人致命的环境，虽然如此我依然开心。
“这里也好好舔舔吧，苍蝇。”
茨木把脚踩了下来，不过目地并不是踩死而是把我夹在他宽广的脚趾之间。比起足底更容易堆积泥垢的脚趾缝隙，同时也更加温暖味道更加浓厚。而我不知不觉间变的更小了，感觉的的气味更加浓烈。脚的温度如同没有水的沙漠，快要把我蒸发我只能从她的脚缝里摄取汗液来保持自己不会干掉。
“哈哈哈哈，master你现在跟寄生虫没两样了呢。”
随着茨木的笑声我感觉到自己的卑微 ，什么master我只是失去了她的汗水没法活下去的可怜虫。
“差不多了，来吧到我的这里来。”
说完茨木把我从脚缝里取了出来，因为一直被捏着所以根本搞不清楚到哪里。
“来吧人类，下来吧。”
不知何时，茨木的手停了下来。它把我送到了一个长着弯曲的金色深林的地方，那里时不时的传来少女的气味。虽然算不上香气但也是一种芬芳。
“你还会变小哦蛆虫，在变成阴蚤之前找到你想去的地方吧。”
这时的我已经迅速朝着气味最浓的地方跑去，绕开茂密的树枝朝着少女的那个地方......但。不知过了多久像树枝一样的毛发已经变得和树干一样粗了，丛林渐渐开始潮湿气味已经浓到我无法呼气。意识渐渐离我远去，我变得只会往前跑。
不知绊倒了什么东西我躺倒在地，同时也回复了些许意识.....这里是。粉红色的可爱小山，虽然对我来说算不上可爱但我知道那是女孩子的阴蒂。我全身都趴在她的阴蒂上，如同山丘一般挺立着。
“哦哦，已经到哪里了吗？不过这么慢你差不多也和跳蚤一样大了吧。”
我已经听不进去她的话语，我趴在那座小山上亲吻感觉肌肤间的触感。我开始玩弄自己的胸部和下体，她可能根本感觉不到吧一只阴蚤在她的阴蒂上自慰什么的。
“差不多该现楼鬼的性质了吧，就是生吞活剥哦。虽然现在你的大小用嘴吞会有点困难......那么就用我下面这张嘴吞噬你吧.”
她微微抖动了一下，但对在阴蒂上的我却是灾难性的。因为阴蒂过滑我根本抓不住，失足的我看向背后如同山脉般粉嫩的阴唇和裂谷般幽深的阴道正准备吞噬我。这也是我在外面世界看见的最后景象了.....
“master真的很美味呢，我还要再享受一遍哦....”
game over
“为什么玩fgo还会gameover啊？这是什么鬼？可恶好不容易抽到的好卡”
生气的我吧手机摔倒床上，突然窗外传来轰的一声.......
“什么鬼。”
我拉开窗帘，看到的景象令我惊呆了。周边10多层的建筑全都不见了，取而代之的是一只红色的脚。这时房顶突然裂开，上面几层就像被移开一样消失。然而出现在我上方的不是篮天白云而是那个茨木童子的美丽面孔。
“终于找到你啦master，让我们再来一次吧。”
