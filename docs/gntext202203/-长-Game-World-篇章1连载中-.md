　『序』 【开始游戏】

【黑光】事件之后的第324天

　　城市的生产力已经逐渐恢复至正常水平，受灾的人们也已经陆续得到治疗，回复到原来的状态，一切似乎都恢复到了常态。

　　三百多天前，奇妙的外星造物从天而降，圆圆的飞碟残骸剧烈爆炸，在大气层中放射出不可思议的黑色的光辉，笼罩了这座城市的各个角落。

　　大部分人仅仅只是看到黑色的光波从眼前扫过就失去了意识。小部分人则是一直清醒着，但他们被光芒扫过的部分竟不可思议地缩小了。受灾的大部分人都是整个人一起被缩的很小，他们大多是市内的居民。但也有一部分正在进入或者离开这座城市的人受到了影响，他们只有一部分身体被光芒扫到，惨不忍睹的身体状况让前来采访的记者都不忍直视。由于只有部分器官被缩小，被救援到的人员只能被插着各种管子维生。他们算是比较幸运的一批，后来的调查表示，有很多上半身被完整缩小的人，在车子受到碰撞安全气囊弹出的情况下，被饥饿和风吹雨打还有下半身传来的放大了几十倍的疼痛神经信号折磨了十几天。救援人员找到他的尸体时候，杏仁大小的脸上是无法形容的痛苦到极度扭曲的表情，同时还透着一丝死亡的解脱，扭曲肿胀的身体每一部分无不宣布着他遭受的折磨。

　　在灾害发生第一百二十四天后，该国政府终于宣布他们破解了飞碟残骸的部分系统。他们发现黑光原本是飞船上的武器系统，当然对外的说法是未知能源造成的污染。部分官员仅存的人道主义还是让他们把部分研究成果用在了医疗系统上。首先被恢复的是那些畸形的人们，不可思议的是，在身体所有部位被恢复成正常大小后，他们只用了一个星期就恢复到了正常的身体水平。

　　最初的一批人在经过三个星期的观察后就被放归社会和家人团聚了。观察的结论是他们各项身体素质都得到了比较大的提升。经过研究他们的寿命似乎都得到了不小的延长，不少人多年的顽疾被一扫而空。结论一出不少国家纷纷发来了建交申请，他们用尽浑身解数才换来了用部分成果延长寿命的机会，当然技术方面肯定是劣化版本。

　　外星飞船残骸带来的科技虽然不能一下子全部解析完，但该国政府还是一下子获取到了足以保护自己不被侵略夺取技术的力量。全国上下的官员被反复审查，直到他们的祖宗十八代的底细都被扒的一干二净。被扒出来的间谍们在确认资料的外泄情况后就被迅速处理掉，但还是有一小部分技术资料流到了外界。科研方面，完全解析外星人的技术预计要二十年甚至更久，但每天产出的各领域科技成果仍然比其他国家要多的多。

　　我们暂且把这个幸运的国家命名为a国，受灾的城市就命名为b市。a国的人民们和他们所得到或外泄的技术，在日后将会引起全世界的轰动与变革。但这一篇要讲的，仅仅只是刚刚开始的那一段时间里，【黑光】带来的后续影响引发的故事而已。

　【游戏1】『稚知』（节1）

这是【黑光】扫荡城市大地后的第二十个月，b市的社会秩序已经差不多要恢复到以前的样子了。人们仍然在这个国家一如既往的生活着。缩小的患者被恢复，新的科技被应用到生活，可以说现在的b市是世界上最适合移民的城市也不为过。

　　今天是a市中小学生们暑假的第一天，各种大型的游乐场啊公园啊，孩子们的嬉闹声不绝于耳。但也有小孩子们选择在谁的家里一起打一盘手机游戏啊，亦或者玩玩桌游或者下棋之类的游戏，炎热夏天里，孩子们欢乐的时光才刚刚开始。

　　芊芊今年上五年级了，等这个暑假放完就是作为六年级的学生步入学校了。在这个内卷的年代，来年就要面临一大堆升学补课班的她，自然要在这个假期好好玩乐。虽说小孩子的每个假期也都是那么欢乐，但今年的夏天有了一点不一样的新奇玩意。

　　芊芊走向书桌，从抽屉的最里面拿出来一个铁皮盒子，铁盒上了锁，但是上面被扎出了几个小洞。她捧着铁盒锁上房门，掏出藏在自己书包深处的钥匙，把铁盒上面的黄铜小锁打开了。

　　盛夏里，略带温热的铁盒被打开，里面静静地躺着几个袖珍版本的人。他们的身长只有三四厘米，确切的说，他们都是【黑光】的受害者。一年多以前，【黑光】带来的昏迷症状有轻有重，率先醒过来的不知为何是女性的小孩子们，其次过了一些时间醒过来的是年轻女性们。男性则是在稍晚的时间才醒过来，这种现象直到现在也尚在研究中。（问就是作者意志，别研究了）

　　满大街的超仿真人偶自然是部分好奇心重的小女孩的狩猎对象。事后调查发现，一部分缩小人的死亡原因正是被或慌乱或兴奋的小孩子们乱跑踩死的。而那些『收集稀有人偶』或者因为其他想法而收集缩小人的女孩子们，她们中的大部分虽然在不久后就被人发现私自囚禁缩小的患者，但还是有少部分因为家长粗心等一系列原因没有发现这些不幸的人类。政府部门也因为救济受灾等种种原因放弃了这一小部分人的追查。他们尚且不知道自己成为了社会的弃子，大部分人还在一天天地期待着救援。

　　虐待，玩弄，成为无知孩童的玩物的他们每一天都在饱受折磨。被饿死被无意中弄死的人比比皆是，受困于童真（或贞）构成的地狱的他们，今天也要继续生存下去，去等待那唯一的希望。
