# 「新人求罩」梦境

作者：金泽明

TID：19236

 

# 1

做了一个梦，烈火燃烧着残存的废墟，绝望的哀嚎着的行人，怀中抱着嗷嗷待哺的婴儿的主妇跌倒在地，被从天而降的巨大脚掌踩得粉碎，成为黑色丝袜上一个不起眼的红色斑点，再也听不到自己孩子的哭号，然后被其他的红色斑点覆盖。

而我，就站在街道上，看着无数的路人从身边匆匆而过，看着高楼大厦变成废墟，看着这一切罪恶的元凶——那个穿着黑色丝袜的少女——将这座城市的一切美好变为废墟而无能为力⋯⋯

突然，那个少女好像注意到了我，她开始向我移动。她的每一个步伐都如同地震一般，震源不止是她那巨大的双脚，还有从脚上不断掉落的，刚刚粘在她脚上的曾经繁华城市的废墟。

“轰”“轰”“轰”

随着少女的前进，变响的不只是她脚步的声音，更响的是建筑物粉碎时震耳欲聋的愤怒和无数人肢体爆裂的声音还有绝望时的咒骂。这些声音也许在少女的耳中不过是为她演奏时的交响乐，然而却如同一根根沾满剧毒的针一样刺痛我的心脏。

“为什么要这样做⋯⋯”我问。并不奢望少女能够听到，或者说，这句话根本就是在问自己。

她的双脚完全占据我的双眼，黑色丝袜上，红色的斑点异常醒目，有些仍是鲜红的液体，仍然散发着刺鼻的腥味，而有些已经变成了暗红色的硬块，执着而无力的黏在少女的黑色丝袜上。

“怎么样，这样的世界？”少女低头问我，巨大的乳房阻挡了她的视线，可她的声音却如同雷声般让人无法忽视。如果我没有见到这腥红的一幕，我想我会沉醉在少女温柔的声线之中。

我试图稳住自己的心中的恐惧，可是手心中的汗还是止不住的冒出，我只能尽全力站着，然后祈祷以少女巨大的体型无法发现我的的恐惧。

可惜少女还是发现了，她问“：怎么？你在害怕吗？明明是你自己创造的这一切？”

“别开玩笑了，我怎么会创造出向你一样的杀人的怪物！我连想都不会想！”

“哈哈哈哈哈！”少女突然大笑了起来，“是吗，果然你是这样想我的呀。这样的话，死吧。”当她说出最后这句话的时候，语气变得十分冷静，冷静的让人害怕。

她说的这句话并不是玩笑，当她说完这句话之后，她就抬起了自己的脚，巨大的，包裹着黑色丝袜的脚。

未凝固的鲜血不断滴落，血液还连着脚底板和大地，如同血做的擎天柱一般。随着脚的提升，血柱断裂，鲜血从天而降，宛若一场血雨。紧接着，建筑与尸体的残骸无法继续抵抗自身的重力，砸在了血液的湖泊中，溅起的鲜血覆盖大地。

然而这一切都只是绝望的前奏而已，真正的高潮还没有来临——那只脚，它拦住了所有光线，遮天蔽日，仿若北欧神话中，诸神的黄昏来临前那三年漫长的黑夜。

然后，她让脚落了下来。

巨脚的下落很缓慢，如同她升起来时一样缓慢，仿佛是为了让我尽情品味绝望一般的缓慢。巨脚慢慢降临，我蹲下来，抱住头，不停的告诉自己，这只是一场梦而已。但是，这场梦让我真实的让我无法相信这是一场梦，我能闻到鲜血的气味，我能看到凝结的尸块，我甚至能听到尸体不甘的咆哮！

“轰！！！！！”近距离的听到，果然更加震撼，震耳欲聋的脚步声。

我这样算是死了吗？在梦里。身体变得很轻，渐渐升高，穿过了少女的脚掌，看到了她的美腿，纤细的摇身，丰满的胸部和那极美的面庞。

而且那面庞之上，还有一抹悲伤⋯⋯
