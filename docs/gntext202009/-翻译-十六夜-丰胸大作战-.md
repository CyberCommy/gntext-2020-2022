# 1

草根也长不平原上

在那里，有数千人的男人们。

男人们在这抚摸，踩踏着地面，有几个用头在刺激着地面，哗啦哗啦地运动着说（什么鬼），数以千计的人也同样的动作的反复的样子很奇怪。

“瞧，好好干吧”

突然，大气和大地震撼的惊人的声音大小轰鸣，男人们全体人员向地面蹲下。

看地平线的对面山前有什么什么爬起来。

那是，巨大的脸。

巨大的下巴，鼻子，高的东西从依次出现，不久脸上俯视着一样的形式的男人们面前出现了。

年轻的少女的脸。

男人们现在，少女的身体上。

起伏的缓慢的、谨慎的胸脯上。

数以千计的男人们，北方在在床上只有上半身裸体少女的胸脯上。

少女的胸围是杯说的A。膨胀称之为谨慎的胸口（什么鬼）。

但是，对男人们来说的话，其范围广阔比东京巨蛋还要大吧。

仿佛沙漠什么，什么也没有不休延伸的大地上，男人们站着。

那太突然的事情。

蝉鸣的午后，街上回响着阵阵轰鸣。

如果极目远望，很远的地方有一个山一样的巨大的少女向这边走过来了。

横格的背心。牛仔短裤。赤脚凉鞋，看似普通的少女，但其身高超过1500米。

巨大的凉鞋住宅区和大楼等纷纷踏碎边走了。

这里是大城市是没有的。这栋大楼星星点点有，其他是住宅区和超市等的程度。

她，脚踝到达建筑物都不存在。

这个城市是对她来说平坦也同样如此。

街侵入发挥了巨大的少女腰手捂金刚了环顾四周。

然后在一角找到目的的东西，又各家踏碎边朝那边走了。

少女的眼眸前来到，它从正上方俯视了。

那是男校。

操场运动了男子的身影也能看见几个。

那样的操场，那样的大小程度，少女脚放下就踩碎了

少女的男子学校前蹲下。

男子们，巨大的脚如铁塔般高耸入云，巨大的上半身降下来也看不见了。

少女短裤延伸出来的丰满的脚的大腿和腓重叠，坠弯的肉终于露出了的鞭子。

“请你帮我们，有事情的说”

少女对着脚下的学校说。

那声音足以撼动整个街道。

那样的可怕的声音轰鸣起来的瞬间，在运动场出来的男子们叫苦着朝着校舍和体育馆朝开始逃跑。

校舍就在学校门口的目标。

“喂，逃不也很好吗？”

少女说着在场也没有男子停下，大家朝着校舍等目标继续跑了。

“喂，你在听吗？”

略微焦躁的轰鸣声从头顶传来。不只是空气，大地也不断振动起来的声音从空中传来。

但是，那个大声堵住耳朵奔跑的男人们，最终也全部建筑物里隐藏了。

“…哼，青森比较好啊。你那边比较省事”（什么鬼，看不懂）

少女向着男生们逃入的校舍伸出双手，把整个校舍连根拔起（什么鬼啊~~日文翻出来真奇怪的说），天空中时不时有着土块落下。少女双手捧着校舍放在脸前。

逃走了的学生们，玻璃窗的对面一杯广阔的巨大的少女的脸，看到了。

“是的是的，无用的努力呢。辛苦了。你们不逃跑的话我也不用做这样的事情呢”

然后少女拿着那男子学校再次在城市中开始走动。

少女的凉鞋下无数的房子和车相继被践踏，凉鞋，抬高了后它们完全被压缩地面的花样。

这样的城市里来回走着的少女还有其他几个男校捡起它们带町离开了。

之后，城市只剩下残骸了了。

然后被袭击的男子学校的学生们现在，被装在那个少女的胸罩中。

那些学校的学生都是站在也不太有也不太大的胸。

少女好好地努力踏出的一步也能使沉稳的大地（少女的胸口）上的男人们走投无路。

并向他们宣告的使命，那个巨人的少女的心刺激的事。

“男人触摸能让胸部变大么？”

少女笑着说道。（未完待续，后面才是福利章节）

10米左右的高度和粗细存在的吧？。
