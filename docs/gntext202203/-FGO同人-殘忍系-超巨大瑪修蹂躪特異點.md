# 1

之前不小心發錯地方了，所以重新在原創區放一次，真的很不好意思。

－－－－－－－－－－－－－－－－－－－－－－－－

西元2019年，人理保衛機構迦勒底－－

「就是這樣，這次我們偵測到一個微型特異點，藤丸立香君，希望你能和瑪修去一趟，實地調查一番。」

有著棕色柔順長髮的成熟女性如此說著，道出一大早將藤丸和瑪修找來的來意。

「達文西醬，為什麼會在這個時候出現特異點呢？難道是跟魔術王有關……」

「不，從觀測上看上去是個自然產生的特異點呢，雖說現在的規模不大，但難保放任下去的話不會對人理產生危害，就像是布料放久了也會自然發黃那樣，雖然沒有危害，但也時間一久就很難清掉那樣。」

達文西細心的跟兩人解說道，隨後兩人看向身後的投影螢幕。

「放心吧，這次的特異點反應很微弱，是目前已觀測到的特異點中最微弱的，我們估測應該不會有太危險的意外發生，跟你們以往已經跨越過的特異點比起來根本不算什麼，難度值：E-。

與其說是勘查，不如說是去打掃吧。」

「這樣呀！沒有太多威脅的話，或許可以順利地跟當地建立起友好關係呢。」

「嗯！」

瑪修抱有幻想對一旁的藤丸說道，至今為止的特異點除了在烏魯克的時候，其他特異點都和當地居民處的不是很融洽，也因此讓瑪修期待這次的旅行不會看到傷亡和犧牲。

「關於這點就是我接下要說的事了－－」

「誒？」

達文西突然切換成嚴肅的態度，讓瑪修收起放鬆下來的心，認真聽道：

「這是的特異點，雖說是微型，觀測的反應也很微弱，卻出現比以往都要更多的生體反應，其數量是現代總體人類的三分之二呦！更不可思議的是裡頭我們沒觀測出人類的存在。」

「三……」

「三分之二？！這怎麼可能！」

藤丸和瑪修紛紛感到不可思議，居然有這麼多智慧生命體，迦勒底在觀察特異點內的生體反應時，會自動去掉動植物等沒有智慧的存在，而剩下來便是智慧生命體和有魔力的生命體。

這意味著，瑪修和藤丸將面對成千上萬的非人類存在，即時威脅度不高，但一次和這麼龐大的魔物對戰的話，也必是一番苦戰。

「就是這樣，我們估測它的裏頭將會比外觀測出的規模要來的更大，年代我們也觀測不出來，是人類史上不存在的時間點，但文明貌似相當落後。」

「……不確定的因素太多了。」

藤丸謹慎的說道。

「這一點我們無可否認，雖然很不甘心，但我們能依靠的只有你們了去調查那個特異點了。」

「不，達文西醬，沒關係的。」

達文西笑了笑，隨後便用長官的語氣下達任務：

「這次的任務是勘查新出現的特異點，並解決其原因，拜託你們了，藤丸、瑪修。」

「是！」

－－－－－－－－－－－－－－－－－－－

【特異點X】

這裡是瑪修和藤丸等人這次的目標特異點，從文明的發展程度來看，與人類史上西方中古世紀類似，不同的是他們的人口過度發展，許多人流浪街頭和山野，現今這個特異點地表上佈滿人口和都市的建造，各個大國不斷吞併小國，擴張王池，平民的死亡率一天比一天高，一般百姓的生存空間遭王族剝奪，有人餓死這件事早已不被人稀罕幾乎每天都在上演，然而人口的密度仍沒有減緩，簡單來說，這是一個不合格的人類文明。

在一個和以往沒有不同的某一天，王成內富裕的人們花天酒地，王城外人們依然絕望度日，那東西來的很突然，毫無徵兆的，各地人們的頭頂不約而同的被一個巨大的黑影所覆蓋－－

「欸？天怎麼黑……？」

咚咚咚咚咚咚咚咚咚咚咚咚咚咚咚咚咚－－

只聽前所未聞的一聲巨響，闖入特異點內人們的生活，那些被陰影所覆蓋的人還來不及看清是什麼在他們的頭頂上，便再也無法發聲，沉眠在這片折磨他們已久的土地上。

「那……那是什麼？！」

沒有不被那東西所震驚的人在，從方才的變故中倖存下來的人們只能看到剛才被陰影所覆蓋地面，如今被兩根從天而降的漆黑柱子所佔據，現在那兩根柱子底下的文明和人也悲慘的被壓入低地之下，化作霽粉和可悲的肉醬，更慘的是剛才柱子的降臨造成周遭的大地震，使無數樓房倒塌許多民眾被壓死在廢墟之下，死傷慘重，而距離稍微離漆黑柱子遠一點的人們抬頭仔細觀察一番，從柱子漆黑的部分向上延伸，發現上頭有肉色的部分，並從其外觀去推敲，好一會兒後，人們才終於意識到那不是單純的漆黑柱子，而是一雙穿著高跟鎧甲巨靴，柱子的真身則是雲層之上那不可視的存在，這雙巨靴主人的雙腿。

－－－－－－－－－－－－－－－－－－－－

「這裡是……前輩人呢？難道是在剛才的靈子轉移中走散了嗎？以前從沒發生過這種情形……好擔心前輩會不會有事呢……」

幾分鐘前，瑪修和御主藤丸像以往一樣進行了靈子轉移，可這次在轉移後卻不見藤丸的蹤跡，這讓瑪修不禁擔心起藤丸的安危，內心也越來越焦急。

「話說回來，這裡是哪裡呢？我難道是被傳送到荒野之上了嗎？不知道附近的城鎮在哪裏呢……」

瑪修茫然地四處張望，此時的她還不知道自己擁有在這個特異點主宰一切的體型和力量，不管是人類、山脈、都市或國家，只能被她踩在腳下，要是她有心花點時間，便能將地面上所有的生靈踩入土壤內，然而心地善良的瑪修正常的情況下不可能這麼殘忍，現在從她的視角，只知道自己身處在一片未知的荒野，望去是一望無際的平原，腳下灰中帶綠的土壤也是沒見過的地質。

「腳下是沒見過的土地呢，地球上有這樣的土嗎？真不可思議，哼哼。」

瑪修總是留心觀察著旅行中的一花一木，這對外頭常年暴風雪肆虐的迦勒底而言，都是珍貴的發現和回憶，然而對腳下的小人們可就沒那麼美好了，當瑪修彎下腰，仔細觀察腳下的土地時，人們也得以仰望她的尊榮，在小人眼裡，瑪修的力量就如女神下凡一樣，當女神垂腰用那圓潤的紫色明眸「審視」他們時，那巨大的壓迫感使民眾中一大部分人當場下跪磕頭膜拜，在他們眼中瑪修可愛甜美的模樣，就是為這個腐敗的世界帶來淨化的女神，更有人認為是王族的貪得無厭，引來女神的憤怒，不過這些也都是他們在恐懼下產生的信仰和幻想罷了。

「啊對了！得先趕緊找到前輩才行！但是前輩在哪呢……

總之先找到附近有人居住的地方吧。」

重新專注在尋找藤丸的瑪修決定總之先行動起來，找到有人的地方，於是瑪修抬起左腳，邁開來到這個特異點最初的一步－－

「啊啊啊啊啊啊啊！她動了！那個黑色怪物動了！」

「女神大人開始淨化了……」

「不要！不要！求求妳不要踩死我！」

「這是審判！所有人都得將得到女神大人的寬恕！請帶我走吧！女神大人！」

面臨瑪修的玉足，許多人在臨死之前都展現出自己真是的想法，有的人失聲痛哭、有的人當場昏厥、有的人咒罵、有的人向瑪修求饒，而有部分對世界失望的人則期待瑪修的腳降臨在他們身上，期待著女神把他們帶往更好的地方，可當那靴底的巨大陰影籠罩他們時，更多的人都是迫切的想逃離瑪修的腳，然而，這一切都是沒有意義的……

因為他們以為的女神實際上根本沒認知，甚至不知曉他們的存在。

並非故意剷除，而是像走在路上不知道腳下存在的螻蟻蟲豸那樣，沒有人會在去注意到，隨後將之碾碎在鞋下，只是這次平時不小心踩死蟲豸，是他們自己。

明明只是一步的時間，對小人們好似漫長黑夜，隨著女神的玉足逼近，人們漸漸能看清女神紫色靴底的花紋，和在那上頭無數黏著的同胞肉醬，卡在靴底花紋的廢墟不斷掉落，砸傷無力的的人群，人們清楚的意識到，在神的尊軀面前，他們只有接受被粉碎的命運。

轟轟轟轟轟轟轟轟－－

沒有給予腳下可悲的小人們一絲慈悲，瑪修的高跟短靴終究會落地，將無數可悲的小人踩死，成為大地上瑪修足跡中數以萬計的紅斑之一，有幸的小人屍體殘塊肉泥和血跡則會粘在瑪修的靴底，成為不起眼的污漬，等著下一步被踩的更碎。

「欸？好像聽到人的慘叫聲？」

瑪修屹立在小人們屍骸遍野的土地上，貌似聽到人的慘叫……可當他安靜下來，又什麼也沒聽到，只有空曠安靜的平原和她自己。

「……大概是幻覺吧。」

「話說回來，地面好軟呀，跟爛泥巴似的，走起來有點噁心呢……」
