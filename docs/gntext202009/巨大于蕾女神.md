# 1

往上看..那2根柱子是從雲端到海面...其實那2根大柱子..

只是于蕾的雙腳..白雲只到她的膝蓋而已..這時候..于蕾已從

160公分的小女生變成16公里高的巨大少女..

從海上走到港口只有她的3步伐的距離..雖然她都是輕輕的往港口

方面走..但是在她的每一步卻造成極大的震動及大海嘯..

50m的海嘯奔向陸地..造成港口附近的建築物倒塌的倒塌..

淹沒的淹沒..那裡的人類當然死傷慘重..但是在巨大少女于蕾的眼

裡..港口的慘劇卻只像是腳踩在水漥產生的漣漪．．毫無感覺...

此時于蕾雙腳已經站在港口了..但在她雙腳附近的生物及建築物..

無一存活及完好無缺..

這時候整個城市的人類眼光都看著眼前的巨大少女于蕾..

根本只看到她白淨光滑的小腿..離她腳下不遠的人類只能看到

她的腳趾頭及她雙腳穿的低跟涼鞋..

此時軍隊已經到達剛口附近..上百架的噴射機及數百台的坦克車開始

朝向巨大少女于蕾開火了..但是在他們眼前的是巨大少女于蕾..不..

應該算是巨大少女于蕾的雙腳..雖然軍隊攻擊非常激烈..但是他們卻像

是蚊子跳蚤一樣的大小..

空中的噴射機集中火力攻擊于蕾的右小腿..

地面上的坦克車也朝著于蕾右腳巨大涼鞋的腳趾頭攻擊..

這樣子的攻擊..只能讓于蕾感覺到癢癢的感覺..

於是于蕾心裡竊笑著在想:

(嘻嘻!太小了吧!像跳蚤一樣小..還想要跟我戰鬥嗎?)

(哈哈!那就這樣子好了!我就站著給你們打吧!)

噴射機的飛彈攻擊..坦克車的砲彈攻擊..這樣子的集中攻擊..

可算是天驚地動..整個攻擊後..軍隊的眼前盡是煙霧瀰漫..

(多多少少也該有所損傷吧!)

軍隊裡的人認為在這樣子的攻擊下心裡就這麼想了:

但是..煙霧散開的時候..于蕾的右小腿還是一樣的白淨光滑..

右腳腳趾頭還是一樣泛著美麗的粉紅色..連腳趾甲還是像貝殼

般的閃亮..根本一根寒毛都沒傷到..

面對眼前的景象..整個軍隊的人都愣在原地..

(哇!太弱小了吧!這樣子站著不動給你們打..一點感覺都沒有..)

于蕾心裡正取笑著這些軍隊:

突然..于蕾的表情好像想到什麼:

(這些跳蚤們雖然很小..但也是人類阿!他們那麼弱..在他們眼裡..

那~~我算是神吧..對了..女神..那就~~于蕾女神!嘻嘻!)

的確..人類跟于蕾比較起來..人類太弱小了..于蕾就像神一樣..

輕易的就能讓人類消失..巨大女神于蕾終於誕生了..

接下來..于蕾女神低頭看著腳下的軍隊..臉帶微笑著..

這樣子的表情很像天使般的微笑..但是于蕾女神接下來的動作..

足以認為她可是個小惡魔阿!

這時候..于蕾女神終於說話了:

[你們這些小跳蚤們~~竟敢侵犯我~~于蕾女神~~]

[于蕾女神就給你們一點小小的懲罰呦~~]

于蕾女神的聲音非常甜美..但是對女神腳下的軍隊而言..

那等於被判了死刑..

于蕾女神的右腳開始動作了..巨大的涼鞋往天空慢慢的抬高..

這時..較靠近于蕾女神右腳的噴射機..瞧見地面上的巨大涼鞋..

正往上抬高..這些噴射機一碰到于蕾女神的右腳背..就被摧毀了..

大約一半數量的噴射機..只因于蕾女神的這個小動作..而消失了..

當然發生這樣子的事..對于蕾女神來說根本毫無感覺..

此時巨大的涼鞋已經抬高至定點..這時剩餘的噴射機跟地面的

坦克部隊只能看見到于蕾女神右腳巨大涼鞋的鞋底..在鞋底腳跟前

還可以看到一些像字及23.5的數字..23.5號的女用涼鞋不就是代表

空變成漆黑..面對這樣巨大的腳..可想而知全部的軍隊很快就消失了..

于蕾女神開始把她的右腳慢慢的朝向地面踩..在空中剩餘的噴射機就是

接下來的犧牲品..全部消失在于蕾女神的右腳涼鞋鞋底..當然地面的坦
