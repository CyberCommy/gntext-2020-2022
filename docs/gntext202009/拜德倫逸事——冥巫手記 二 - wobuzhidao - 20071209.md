# 1

已經是三天以後了，當晚莫莫跟我耍了個小小的把戲，我附身的根本就不是什麼藥丸，而是一塊魔法水晶！當然，法陣也不是那麼容易就會失效的，所以我到現在還是一顆小石頭，至於莫莫，則絲毫沒有放過我的意思，她騎馬的時候，我一般被放在她的內褲中，馬背顛簸，我不知不覺就會滑進她的前後庭院，飽受“水深火熱”之煎熬（或許是享受）。步行的時候，她也不會忘記把我放在襪子中，由於腳底和鞋子之間是容不下這麼大的石頭的，所以我基本被夾在她的腳趾之間，她的腳趾還不安分，常常要晃動，結果就是我翻跟頭，基本把四面八方都親吻過了。
    直到最後幾天，已經深入修斯帝國，我是作為項鍊掛在她的胸前，也是我第一次在她的雙峰間擺動，莫莫算不上巨乳，我也不喜歡大乳房，因為顯得不協調，只有這樣標準的雙峰才能挺拔，雪白的肌膚在陽光下反射著耀眼的光芒，從莫莫不算保守的服裝中放射而出，不斷地引發著交通事故。
    最後，我們在圣西索購置了房產——是導師老人家親自挑選的，這裡是富人圈，又靠近圣西索學院，同時周圍有許多中心建築，比如皇宮、皇家圖書館、皇家騎士團總部、魔法師工會等。
    入夜。
     “明天就是圣西索學院報導的日子了，需要幫忙嗎？”我問莫莫，而後者正半躺在新居臥室中的豪華大床上，嘴裡塞滿了各種點心——這些點心的原料不是魔族那貧瘠的土地可以產出的，而且那邊的人也沒有這麼好的手藝，只有貴族的食品還稍微過得去。到了這裡，終於安頓下來，莫莫自然會大吃特吃，而我，當然附身在她柔軟的舌頭上享受了，不過剛剛想起了正事而已。
     “唔？唔用額，偶凹既幾然餓”莫莫由於嘴裡塞滿東西，話都說不清楚了，不過還好我能聽懂，她要憑自己的力量，通過入學考試還是沒什麼問題的。
     “那今天晚上我們做點什麼好呢？”我有所希冀地問。莫莫把東西咽了下去說：“今天不要，明天我要考試，我要保留體力的，反正圣西索這麼大，你出去逛一圈也不會無聊的。”說完，她起身去刷牙，我受不了牙膏的味道，只好離開她溫軟的舌頭。
    離開家，我邊飛邊尋思著該去什麼地方，突然發現已經到了學院的上空。 “不如去這裡摸摸情況吧，反正也算是任務的一部分。”我降下身形。
    校園裡並不是空無一人，看來高年級的人應該已經提前回來了。教室和圖書館零星地亮著燈，大概是用功的學生在學習魔法、古文之類的吧，公園裡還有成對的情侶在幽會，吃完消夜的人從食堂走出來……不知不覺，我竟然晃到了女浴室。據說女生進入圣西索學院是要經過嚴格選美的，裡面風景一定不錯吧。美女的引力超過了地心引力，“反正我愛的只有莫莫一個人，好色應該不是什麼大罪吧，英雄本‘色’嘛。”我毫不憂鬱地飛進了浴室。
    春光無限啊！ ！ ！這是我的第一映像，更衣室中不斷有赤裸的美女進出，我開始盤算怎樣享受這些美女了。
    首先，從腳開始，每一雙玉足都那麼讓人垂涎，我附身到更衣室門口的地毯上，開始慢慢品位。一雙雙赤足踩過毛茸茸的地毯，也踩在我的臉上，幾十公斤的體重對我來說就像按摩一樣，誰讓我現在是地毯呢，換做普通人，可能會吃不消吧。一雙雙玉足中，不乏精巧到及至的，有的腳上還帶有微香酸甜的腳汗，是剛脫下衣服進來洗澡女生，而洗完澡離開的女生，腳上的水還未乾，帶著沐浴露和體味的芬芳浸入我的身體。在觸覺和味覺的享受中，我的視覺也同時經歷著考驗，因為這個角度，女生們私密的地方我一覽無餘，她們大多剛剛發育，稀疏的陰毛下，兩片粉嫩的小花瓣若陰若現。要是我有鼻子，恐怕少女們的腳底就要沾上我的鮮血了。
    過了一會，我稍微有些厭倦，就離開了地毯，找了一個“各方面”都不錯的女生，附身在她的右手中指上，此刻，她正把沐浴露塗上自己的手，然後，她的手指開始遊遍她的全身，甚至還在股溝中多停留了一會，大概她很愛乾淨，不願留下一點污垢吧。不過說實話，這裡的女生的身體還真比不上莫莫的——不論皮膚還是身材。不過誰讓我的莫莫今晚要休息呢，我姑且屈尊來陪你們。
    過了許久，這個女生終於洗完了澡，我剛想離開她的身體，不過一想也不知道去哪裡，乾脆跟她回寢室看看。
    女生寢室反而沒有浴室那麼多風景，女生們都穿著睡衣，簡單地聊天后就各自入睡了。我覺得無趣，剛想離開，卻發現這個女生有了動作，我附身的手指慢慢移動，最後停在了她的胯間，然後，我的臉貼上了她的內褲，隨著她手指的移動在毛絨的內褲上摩擦著。 “原來是手淫啊，我就留下來好了。”我這樣想著，透過內褲，可以看到她的兩片陰唇開始蠕動，繼而一張一合，好像要吞下一切似的。
    不久，她那粉紅的門口變得濕潤，愛液開始流出，沾濕了我面前的布料，隨後，大量的愛液湧出，隨後在內褲附著的魔法作用下化為氤氳的蒸汽。最後的一幕上演了，女生左手撩開內褲，而右手中指長驅直入，我頓時被溫暖的肉壁包圍了，黑暗中，我依然能看清周圍可愛的粉紅色，而愛液的味道是那麼濃厚，就像陳年的紅酒一樣香醇。手指不斷地抽拉，兩個人的心情都愉快到極點。對於她這樣的女生，小穴尚緊，手指雖然纖細了些，但效果還是不錯的。
    不知過了多久，女生似乎累了，動作遲緩下來，我也覺得似乎該干些正事了，於是脫身而出。
    憑著記憶，我很快就找到了學校的圖書館，學院裡有各種學生，那麼作為學生課外學習的地方，這裡應該有不少有用的書籍，甚至可能有我沒見過的魔法。想到這裡，我加速飛向圖書館的大門，然而就在到達大門的瞬間，強烈的魔法波動將我彈了回來。 “居然還有防禦，看來里面是真有好東西了。”
    隨著魔法波動趨於平靜，圖書館大門打開，裡面出來兩個守衛模樣的人，
     “咦？見鬼了，什麼人也沒有，魔法牆怎麼會有波動？”
     “別瞎想了，大半夜哪來的人，多半是哪個笨鳥撞上來了，走，回去吧。”
    看來這裡的魔法防禦很強大，我有把握破除這裡的防禦，但是不可能不驚動裡面的人，思索一下，我直接飛向後面那個守衛。也許走在前面的守衛沒有註意到他身後的同伴身體微微震了一下。
    利用這個守衛的身體，我順利地進入了圖書館，以我的能力，控制一個凡人的意識還是能做到不為人知的。離開守衛的身體，還聽到他在嘀咕剛才發生了什麼事。
    圖書館裡只有少部分的魔法燈還亮著，但絲毫不影響我的視力。圖書館非常大，單是第一層就有數百排書架，而且種類齊全，我粗略地瀏覽一下，大部分書都是關於語言學、歷史學、文學、兵法的，這些書只有以後慢慢來吸收，或者乾脆讓莫莫來看吧。
    到二樓，內容與一樓差不多，到了三樓，才發現有一些低級的魔法出現。再往上，發現到四樓的門是鎖著的，而且同樣有魔法防禦，看來暫時是進不去了。至少也要等到明天有人能進到四樓以上我才有機會。正準備原路返回，卻發現剛才的兩個守衛竟然上樓來了，
     “我總覺得有些不對勁，保險起見，還是檢查一下好了。”
     “你這個人就是多疑，不過另一方面講，你還是滿敬業的。”
      ……
    他們的對話不是重點，而重點是他們把門打開了！真是雪中送炭啊，我跟隨他們往上爬，果然，從四樓開始都是中段以上的魔法與鬥氣的訓練方法，但是我沒有時間細看，因為這裡每往上一層都有一扇門，而且這兩個守衛都是隨手關門的。
    一直跟隨他們到了第六層，這裡已經是最高層了，這裡的書籍甚至都閃著魔法的光芒，映射著我貪婪的眼神。守衛檢查完後離開了，這裡就剩我一個，於是我開始尋找需要的書籍。這裡的書籍不乏有禁咒，而且黑暗魔法的書籍也不少，但很可惜，大部分黑暗魔法我早在魔域就已經學會，而其他的魔法於我來說並無太大益處，失望之見，忽然瞥見角落裡一本不起眼的書，上面落滿了灰塵，但是我仍然感應到上面有微弱的黑暗魔法氣息。懷著無所謂的心理，我打算看看這本書上寫了什麼。
    由於灰塵太厚，看不見封面，我只能用靈魂來感受書中的內容。
    原來，這是一種古老的，已經失傳的黑暗魔法，主要是講如何能以黑暗之力巧妙地侵蝕人的身體，這種侵蝕不是局部的，而是整體的，黑暗之力由內而外地侵蝕人的身體和意志，而且由於是同步地侵蝕，人體內的各個器官同時被腐蝕掉，或者剩下一副皮囊，或者身體因為侵蝕而變小。這種魔法曾用來製作傀儡，但是等到更便捷的精神係法術出現，就日漸勢微了。 “也許能派上點用場。”這樣想著，我開始鑽研起來……
    直到牆上的魔法燈熄滅，我才感覺到外面的陽光已經射入水晶窗——天亮了。我一邊尋找著出路，一邊想著昨晚新學到的知識，正巧門開，一個法師模樣的人進來，剛好為我的離開提供了便利。
    外面，學院的大操場上已經站滿了人，看來招生考試已經開始了。
