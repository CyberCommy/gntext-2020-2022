這幾年下來，平均每年大約有十萬件『變身』的案例。但是照現在的情況看來，今年有突破２０萬件的趨勢。單純計算的話是往年的２倍。

「最近變身的人增加超多的耶。到底是為什麼呢？」七海問著我。

人類把「某天忽然變成了別的種族」的情況稱為『變身』。

說是「別的種族」，不過並不是白人與黒人的那種區別。
而是根本上與人類相異的種族。
雖然這麼說，也不是指狗跟貓之類普通的物種。
那是至今地球上從未存在的種族。

例如說下半身是魚的人魚。
身體的一半覆蓋著鱗片、而且還會噴火的龍人。
手臂變成翅膀的哈維。
狼人、貓人、其他像是水妖、精靈、還有fairy(＝妖精)，還有很多。

世界上第一起『變身』事件大約發生於２０年前。在美國，『變身』後的種族是精靈。
以那起事件為開端，之後世界各地就陸續發生『變身』事件了。

當時，世間皆以異樣、偏見的眼光看待『変身者』們。以『變身』為理由而被開除、或者被居住的地方放逐之類的『變身者』陸續的增加。
特別是變身後姿態越是偏離人類的話，這種情形就越明顯。甚至被稱作「惡魔附身」 而遭受迫害的例子也不少。

為了反抗這種情形，各國的『變身者』為了確保他們自身的人權而大大的努力奮鬥了一番。而那個結果就是現在世間對『變身者』的偏見減少很多、還修訂了相關法律，某種程度上可以說『變身者』們已經融入一般人的生活中了。

不過，現在還沒有『變身者』變回人類的例子。只要『變身』過一次，就會以那個樣子度過一生了。
世界上的科學家們也全力研究著有關『變身』的機器、『變身』的條件、從『變身者』復原成人類的方法等等，不過成果並沒有很明顯的樣子……。

我以前，也曾經是普通的人類。

雖然這麼說，當初也不明白為什麼會變成這樣。
從覆蓋在自己身上的衣服中脫離後，看到周圍的東西全部變大了，還有看見變成巨人的七海正用困惑的表情看著我。

然後跟著慌張的七海回家(不如說，失神發呆狀態的我被七海抓起來帶了回家)，鏡子中映照的是與之前的我完全不同的姿態。

雖然臉跟以前沒什麼兩樣，不過感覺變的有點年輕了。
頭髮的顏色也與以前一樣，但是眼珠的顏色變成了漂亮的綠色了。
接著是，從背上長出翅膀了。看起來像蜻蜓的翅膀一般，有著透明的淡綠色……。

現在，我在七海的家跟她一起生活著。
原本家裡就因為我爸媽去國外出差而空著，他們只有偶爾才會回來。而雖然我原本就在日本一個人生活，但是變成妖精的話生活就變的困難了。
於是就去住在身為青梅竹馬、而且離我家很近的七海家了。正好七海的父母也在國外出差，房間多少也空著一些。

……不過，被那樣用力握住身體還挺恐怖的。
雖然七海是我的好朋友，而且剛剛那也是開玩笑的動作而已，實際上七海也有控制了力道，但是，即使是那樣還是有一點點怕。
再怎麼關係好，七海還是人類，而我還是小小的妖精啊……。

　*

晚飯之後，七海問著
「我想要洗個澡，繪美也一起洗吧？」

嗯ー、電視現在正播到好看的地方呢，怎麼辦呢。
妖精的身體很小。一個人不會用到那麼大的浴室，但是打開浴室的門、轉開水龍頭等也是很費力的動作呢。
跟七海一起洗的話，就不用那麼辛苦囉。

最後，還是跟七海一起洗了。
「等等哦ー」 我說完後，就飛到七海的肩膀上了。

　*

「好冷喔ー！」
脫掉衣服後才剛踏入浴室，冬天的寒冷氣息就包住了全身。
我們快速的把身體沖一遍之後，馬上就跑進浴缸裡了。
七海抱住了雙腳泡在浴缸裡，然後我坐在七海的膝蓋上。為了能剛好泡到肩膀，七海調整了膝蓋的高度。一泡進浴缸裡身體很快的就溫暖起來了。

呼ー真舒服的水啊……正這麼想著時、
「哪、哪、繪美」 七海說。
「什麼事？」
「自從繪美變成妖精後，到今天正好半年了耶」
「耶？是哦？」
我完全忘記這件事情了，稍微驚訝了一下。
「嗯」
「這樣啊、已經半年了阿∼。不長不短的，奇妙的感覺呢」

「剛開始的時候，很辛苦呢」七海說。
「對阿。真不曉得發生了多少慘事耶！」 我說。
「還被貓襲擊……」
「對對。從來沒想到貓有那麼呢」 我同意的說著。
「第一次飛的時候也很辛苦呢」
「嗯嗯，沒錯。最初還不太能飛的起來呢。所以那時候身上都是撞傷擦傷、甚至骨折了呢。
還有掉進廁所啦、被小孩子當做玩具玩啦……。
真的、沒有變成這麼小的身體的話，什麼事都沒有呢」
「當初還會被狗叫聲嚇的要死呢。還有好像看到蟑螂的時候也是的樣子」
「啊哈哈」
兩個人想到當時的事情都一起笑了。當初真的對我來說，所有看到的東西都很可怕，小小的事情就能把我嚇的要死。被貓襲擊的時候還不太習慣飛行，真的很危險呢。
如果那個時候沒有七海的話，我一定早就死了。

所以

「不過，那個時候每次都是七海幫助了我呢」
從我的嘴中說出了那句話。
「那是因為、不能不管嘛。理所當然的說」
「謝謝妳…」
「嗯…」

然後、

「……雖然我變成了妖精、非常微小、脆弱，即使如此，還是覺得是件好事哦」
「什麼？」
「第一件事就是，能夠在天空飛行了」
「那第二是什麼呢？」
「第二件事就是，能夠像這樣跟七海在一起」

嗚哇。這種台詞，不適合我啦。
說完的時候、我感覺自己的臉很快的發燙了。

「……」

偷偷的、往七海的方向看去時、她也跟我的臉一樣變的通紅了。

然後持續的沉默。
雖然想說些什麼，但是說不出來。

打破沉默的是七海。
「哪、繪美」
「……什麼？」
「身體、我幫妳洗吧？」
耶？剛剛說什麼……？

「耶？剛剛說什麼……？」
「身體、我來幫妳洗」
「哈？」
我不小心直接叫了出來。

「今天是繪美變成妖精的半週年紀念嘛、吶、這是禮物喔」
七海清楚的說著。
「唉呦、這個……那個……」
看到我困惑的表情，七海的臉又通紅了，往下看著、
「那個、呃、不、不要的話也……也就算了啦……想說都是女生、應該也沒關係嘛……抱歉……」
越來越小聲、感覺快要哭出來的微小聲音這樣說著。

我總算知道了。七海是打算鼓勵我，才提出這麼大膽的建議的。
但是、因為看見我困惑的樣子，才忽然又覺得不好意思的吧。七海的臉仍然往下看著。
另一方面，我也是，跟七海一樣臉紅紅的，往下看著。

小學還有國中的畢業旅行的時候，常常跟七海一起洗澡。所以並不會抗拒兩個人一起洗澡、現在也是一起進來的。
不過幫別人洗身體的事情當然是一次也沒有。變成妖精後也是一次也沒有。
平常總是我對七海玩鬧著，七海戲弄我的情況倒是幾乎沒有。雖然如此但要幫我洗澡的話……。

……不對、都是女生哪有什麼色色的呢、而且是人類跟妖精、種族又不一樣、就像人類幫養的狗洗澡一樣嘛……不對、錯了錯了。我又不是七海的寵物……
七海為什麼會這麼說呢？有可能是想讓我困擾的玩笑吧。還是說是喜歡我呢？喜歡身為妖精的我嗎？還是說從我還是人類的時候彼此就有的朋友以上的關係呢……

腦中浮現一堆事情出現又消失。從開始到消失又太快，一直說不出話。
不過、從我的口中擠出來的是這句話。

「嗯……。啊、洗吧……」

　*

「那、我洗了哦……」
「嗯……」
七海溫柔的握住我的身體，然後坐在浴缸外的椅子上。大腿合起來之後把我放了上去。
「如果會痛的話要說哦……？」
「嗯……」

七海把沐浴乳擠在手上之後，開始溫柔的洗著我的身體。

首先是大腿。然後把我的腳舉起來，洗著小腿還有腳掌。我則是只有、只有隨著她去做了。
腳趾頭也幫我洗著。我的腳拇指被七海的指紋磨擦著。不過七海的手指不會傷到我。

接著是手臂。
七海用左手的拇指與食指把我的手臂抬起，然後用右手的拇指與食指洗著。
握著我的手掌的七海的拇指與食指大概有我的大腿那麼粗。
真實感覺到了自己的微小。

然後是翅膀。
妖精的翅膀其實比看上去來的結實，不過七海並不曉得。
七海像為了不傷到妖精的翅膀一樣，細心注意的的打開來洗著。
「會痛嗎？」 她多次的問著。
「不會痛哦」 我也多次的答著。

最後是幫我洗身體。
七海把我的身體拿起來，用兩手夾著。
從我的頭頂到腳底都被七海的手給收納起來了。
在那裡面，我感覺到了七海手中的體溫。
感覺到了七海手掌的柔軟。
然後，把我包住的兩手開始慢慢的摩擦著。
「嗯……」 感覺有一點癢。
「啊、對、對不起……痛嗎…？」
七海被我小小的叫聲嚇到、馬上停止了手上的動作。
「唔嗯、不是啦……。只是有一點點癢而已……」
七海的手再次動了起來。果然還是有點癢。不過很舒服。
感覺到了沐浴乳的香味、還有溫柔的青梅竹馬的香味。對被七海包住了的事情感到高興。我果然、最喜歡七海了。

身體洗完後接著是頭。
七海再次把我放在大腿上，然後開始抹洗髮精。
雖然不好意思對盡可能仔細的洗著我的七海說，但是這有一點點痛。

這樣那樣的搓著之後終於搓好頭了。
「好、結束了！那要沖了哦ー」
我為了不讓泡泡跑進眼睛裡面，閉上了眼睛。

刷ーーーーーーーー！！！

大量的洗澡水從頭上往下沖著。雖然七海到剛才為止都很細心，不過最後還是大意了吧。水量太多了啦。
「呀ーーーーー！！」
因為水量太多，我從七海的大腿上被沖到了浴室的花瓷磚上了。
直到快接近排水溝時，才終於停止了。
「沒、沒事吧？」 七海擔心著。
「七海ー…。水太多了啦！」 我嘟著嘴說著。
「啊哈哈、對不起啦ー」
「不過我洗的很舒服哦！就像到美容院一樣耶！謝謝妳、七海」
「不客氣」 七海搞笑的做動作說著。
「希望下次也能這樣呢」
我用一半玩笑、一半期待著的心情笑著說著。
「不ー行。幫妖精洗澡太累了啦。這麼花力氣的事情才不想再做呢」 七海笑著說。
什麼嘛ー、真失望。
不過、七海又繼續說著。
「…所以、半年後再來吧」

　*

那天晚上、是今年第一次的冷天氣。
我雖然躺在妖精用的被窩裡睡覺，但是太冷睡不著。
太冷而縮進被窩、然後蓋住頭的時候，想起了小學時養的貓。
－－那孩子，冬天的時候也常常鑽進我的被窩裡面呢。常常早上醒來的時候發現牠躺在身旁呢。

對了。鑽進七海的被窩吧。

「已經感情好到一起洗澡了，而且妖精又跟人類種族不同，一起睡也沒關係嘛」
那樣想著時，我往七海的被窩飛了過去。

打開了七海房間的門。
七海已經睡的很沉了。很好很好……不對、醒著也沒什麼關係啦。
「打擾了∼……」
自言自語著鑽進了七海的被窩裡。
被窩裡面果然很溫暖。現在我很了解那隻貓的感覺。
看到了身體與手臂間的空隙，在那裡睡也不怕翻身。
感覺著七海睡眠中的呼吸還有心跳，馬上就睡著了……。

　*

……結果那一晚，變成了狼狽的一夜了。
我睡著之後被睡著的七海打到。
我還變成了七海的墊子直到早上。
早上被七海發現的時候，我因為又熱呼吸又困難，滿身都是汗。不幸中的大幸是，還好沒有受傷。

不過、被七海「為什麼做出這麼危險的事！？」 生氣的罵著。
「可是貓都沒關係了嘛……」 這樣反駁的時候、「那是什麼？」 無情的回答著。

　*

這樣那樣的，我作為『變身者』的生活呢，雖然有很多的操心事，多少還是順利的進行著。
只不過、雖然知道是辦不到的，但是，如果可以的話，還是想變回人類。
不對不對。笨拙的願望還是別奢想的好。
總之，我作為妖精，決定一邊享受每天的日子，努力的生活。七海也在嘛！
