# 1

不定期深夜更新

随笔1

手心的男孩正在颤颤发抖，捂住脸从指缝里偷看我的样子还真是可爱。

我有点不耐烦了，伸出手指轻轻把他的手臂拉开，露出了惊慌的面孔。

十五六岁的样子，眼睛已经红肿，遮不住眼神里的恐惧。

我蹲下来，手掌平放，他看了我两眼，飞快地跳起来跑掉了。

真是个神奇的地方，我看着四周要塞的废墟感叹道。

这是什么地方来着？我歪着头询问旁边的另一个女生。

她很认真地掏出一幅地图，许久之后，抬起头来，“地球。”

随笔2

面前的黑暗中有一丝亮光，那是名为太阳的恒星。

按照人类的命名法来看，这个星系叫做太阳系，而其中有一个星球是人类的发源地。

第三个……是从哪边开始数呢？

看着那颗灰色的星球，应该不是这个吧？

这么小，而且也没有人居住的迹象。

更小的灰色卫星星球撞碎在我的睫毛上，让我眨了下眼睛。

咦，那颗星球就这样被睫毛斩断了。

人类那种聪明的生物，不会住在这种脆弱的星球上的。

我这么想着，伸出舌头把星球的一点点残渣卷进嘴里。

那么，那颗有着27个卫星星球，应该就是人类发源地了吧？

随笔3

不可一世的家伙，正跪在脚下呢。

这种感觉真是不错。

自以为是，自称女神的小不点女生，未经同意就毁掉了亿万个宇宙。

怎么能这样呢，一定要制裁。

虽然我也经常那么做，但是那是我的私有物品。

这种家伙是不能随便碰的。

生气的眼神像是有质量一样压迫着她，让她无法抬头。

相对只有我的脚趾高，但我可以瞬间变大到用细胞把她吸收，这种感觉让她很害怕吧？

是那种绝对的差距。

“我不想听你的废话呢。”就这样宣判了死刑。

不，不是死刑，她的身影越来越小，终于变成了细菌的大小，飞进了我的靴子里。

就做亿万清洁工中的一个吧。

其实，这和死刑又有什么区别呢？

不要乱动人家的私有物品呀。

随笔4

静静地坐着也会有人来找麻烦，这就是身体太大的坏处。

两万米高对于人类来说的确很难想象。

已经是第四批来要求我参加战争的人了，满嘴的国家安全和国家富强。

只不过是要利用我简单地赢得战争而已。

早就看穿了。

依旧是拒绝了，但是这次他们没有马上离开。

“你的家人现在在政府的照顾下很安全，请你不要担心家人安危了。”

我的眼神变得危险起来，伸了个懒8腰，然后站起来拍了拍身上的土。

就是这样的动作，几百米大的巨大土块从短裙上掉落，造成了大破坏。

“带我去吧。”

前线的军队，正在严谨的战备状态。

“我的家人在哪里呢？”

很快被带出来的妹妹和父亲让我放心了许多。

伸出一根手指挑起他们，放进了胸口的口袋。

然后，带着残忍的微笑对周围的士兵抬起了脚。

一分钟之内，十万人的军队就全灭了。

真是太简单了，看着对面溃散的敌人，心里有种胜利感。

就这样征服世界，好像也很简单吧？ 

# 2

菌斑

脱下运动鞋之后，一股肉眼可见的蒸汽从袜子上散发出来，旁边的佳儿捂着鼻子看着。

“干什么这样的表情？”我看到佳儿的样子，抄起地上的运动鞋，把鞋口朝着她的脸上盖去，46码的运动鞋，鞋口完全罩住了她的口鼻。

“呜呜呜呜呜！”佳儿拼命挣扎着推开我，“姐！你的鞋里都有霉斑了啦！”

我好奇地看着鞋内，的确有一个个圆圆的小黑点。

“胡说什么，那不是霉斑。”我不高兴道，“那是小人的城市。”

的确，鞋里那一个个半厘米大小的斑点，是一个个千万人口的城市呢。

靠着每天流下的汗液，和脚上的污垢来过日子的人类。

托他们的福，脚上才能干干净净的。

只不过，一滴汗水就可以杀掉一个城市的人口呢。

这么想着，一滴汗水从脖子上滴下，正好落进了鞋口，朝着一个城市砸了下去。

虽然我听不到声音，可是难以估量的大量酸臭汗水就这样湮没了多少人，多少建筑物呢？

人类真是太微不足道了。 

# 3

鞋里的环境很恶劣，刚刚被脱下的运动鞋腾腾地冒着蒸汽，味道更是几乎让人窒息。

鞋子的主人穿着被汗水浸湿的棉袜走开了，那双46码，但是又柔软的脚让人欲罢不能。

欲罢不能的还有一群人。

鞋子内的环境虽然极其恶劣，甚至传说中的地狱可能都没有这么可怕，但是里面是有生物的。

这种生物不是细菌，是人类。

虽然说是人类，但是大小却和细菌无异，至少和我相比是这样。

作为一个女生，喜爱运动的代价是出汗，这是不能忍受的。

身体上出汗还可以马上洗澡，而被汗水淹没的运动鞋就会有很大异味。

还好有人类的存在呢。

说起来，相对比细菌还小，是一千万亿分之一的大小，生存在鞋内。

就算塞进去几十万亿，也丝毫感觉不到，就是每天都要更换，太麻烦了。

鞋里。

一个个小小的都市正面临最大的危机。

刚刚运动时的闷热还有汗水已经席卷了城市，几十万亿人只剩下少于百分之一的存活率。

即使那样，也是几千亿的“庞大”数目。

存活的人类在汗水的海洋中挣扎着，光是一滴汗水就足以淹死所有幸存的人类了，而为了活下去，他们只有大口喝着我的脚汗。

嘴巴里是咸咸的汗水，但是鼻子里也不轻松。

鞋子里的味道自然不可能多好，虽然可以控制自己不产生任何味道，但是那样怎么有趣呢？

为了玩乐，只好让这几千亿人呼吸着臭不可闻的空气。

至于城市和科技，早就没用了，面对那么巨大的汗珠，城市和不存在没有两样，准备好的潜水艇无法承受那么强的水压，也全灭了，少部分的军舰勉强撑了下来，飘在一望无际的汗珠上。

我坐在王位上，伸出两只脚，一群群女神扑了上来，贪婪地舔舐着被汗珠打湿的棉袜，其中体型小的，试图从袜子的纤维处钻进去。

享受着脚底的按摩，我渐渐困了，慢慢睡了过去。 

# 4

“进来了！”随着响亮的警报声，人们立刻蜂拥而上。

一只穿着白色棉袜的脚丫钻进了鞋里，宽敞的空间被它霸道地占满了。

一个城市和白色的棉袜比起来根本看不见，就算是一个星球也不过是个小点，大概是人类的一百亿倍大。

在运动鞋里居住久了，所有的人类都被那不可抵挡的巨大脚丫所折服。

轻轻动一下就是地震，流一滴汗就是洪水，一点异味就是毒气，根本是不可匹敌的存在。

什么天灾简直弱爆了，台风会被异味瞬间瓦解，干旱是一滴汗就能解决的问题，超越了理解的强大存在。

越来越多的人类相信，如果穿越那厚度超过二十万公里的棉袜，舔舐里面从未在人类面前显示真实面目的脚趾的话，就能获得神的恩赐。

因此，所有人类的科技开始朝着高速移动发展，对那个梦想的渴望超越了一切，真正让科技达到了日新月异的地步。

喔，我的鞋里当然是没有太阳和月亮的。

每天脱下棉袜，都会在里面发现一些残骸，大多是奇形怪状的交通工具，不得不说，信仰的力量是很惊人的，在穿着运动鞋的短短的一个小时内，居然有的人穿越了棉袜厚度一半的距离，看来很快，他们就能实现梦想了。

对于这么努力的种族，应该给予赏赐。

我笑着将冒着热气的赤脚伸入了鞋里，希望他们会喜欢这个礼物。

 

# 5

“又要上课了，真不想去……”我一边努力从床上爬起来，一边抱怨道。

“请您务必去上课啊，大人。”床头一个小小的液晶屏幕上，一个老师模样的人恭敬地说。

“知道了，烦死了。”我不耐烦地打断他的话，然后站了起来，“我会去的。”

洗漱完毕，吃了早饭后，我打开鞋柜，选出了一双运动鞋，“今天进行难一点的课程吧～”

将鞋子放在地上，我又拿出一个盒子，里面装了许多1mm大小的长方体，小心翼翼地把它们排列在鞋里，我将穿着白棉袜的脚伸了进去。

教室在剧烈震动后，终于在主人的鞋里被排列好了。

然后，一只遮天蔽日的白袜足伸了进来。牢牢地压住了这些小小的教室。

如果不是用最最坚固的合金打造，这些教室根本撑不住一只脚的重量。

即使现在，也都被压得吱嘎响。

老师和学生都努力忽视自己在一个女生的运动鞋里上课，被她的白袜脚践踏着。

在脚下苟且偷生，完全没有尊严。

女生似乎出门了，不断的颠簸让老师和学生只有把自己捆在座位上，才不会到处乱飞。

进行体育课更是苦不堪言，被露天体育场上方是一片稍微发黑的白袜底，散发着浓烈的气味。

老师和学生像是除臭球一样大口呼吸着女生的足的味道。

渐渐的，女生的鞋里温度增加了，也开始出汗了。

学生们也到了课间，要排队领取午饭。

是女生的汗凝固成的脚垢，是唯一的食物，就像脚汗是唯一的水源一般。

吃着脚垢，喝着脚汗，闻着脚臭，看着脚底。

让人感到无比卑微。

“真是够了！我不读了！”一个女生终于忍不住丢下餐具，大声抗议。

所有人突然站起来拼命逃跑，尽可能远离刚刚激情抗议的女生。

噗叽。

巨大的脚趾点下，将刚刚的女生和一部分来不及逃跑的学生压成了肉酱。

“开除了哟，不愿意学习的坏学生～”我笑着宣布道。
