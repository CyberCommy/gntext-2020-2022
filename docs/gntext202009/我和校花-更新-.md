# 我和校花（更新）

*本帖最後由 xiaomi 於 2020-5-30 20:58 編輯*

看了一部电影，叫《缩小人生》，派拉蒙拍的，说的是一个叫保罗的傻缺，为了节省消耗，把自己缩小了。我就想啊，我要是也能缩小就好了。我要是缩小了，我得先到校花家去看一看，看看她现在怎么样了。听说她现在还单着呢。这事，要是放以前，我是连想也不敢想啊，一看见她就脸发热，心乱跳，连喘气都觉着费劲，更别说去她家了。可现在好了，哥们我缩小了，目测着身高也得一厘米多吧，反正校花家的门缝在我眼里有一米多高，一低头就钻进去了。进了校花家，立时把我给震惊了。我去，这房子也太阔气了吧，一眼望去都是客厅啊，足有500多米远。近处是一个鞋柜，看上去也得有三十层楼那么高。鞋柜旁边放了一双高跟鞋，估计是校花的，光是鞋跟就有十米高。（微信公众号：穷雕狂想录。想看下集，马上关注）我看了看远处的钟楼，不对，那应该就是一座钟；现在正是校花家时间上午10点10分。家里一个人也没有，我自己先逛了一遍。由于地方太大，我一边走一边留心着周围的标志性建筑。比如，那是茶几，从茶几左转，沿沙发辅路直行150米，右转，进入第一间卧室。再比如，从卧室出来，远远地看见一座大厦，那是冰箱。看见它，就不要拐弯了，直接去，大约走一公里多点，就到餐厅了。溜达了两圈，你还别说，校花还真讲卫生，连床底下都打扫得干干净净，除了一根曲里拐弯的绳子，啥也没有。我开始还纳闷呢，这是哪几个逗比想在这里拔河呢，也特莫不找根直点的绳子。突然，我似乎想起点什么，这特莫哪是绳子，分明是校花的头发嘛。校花的头发。哎呀妈呀，哥们真是有点小激动啊。我拿起头发闻了闻，嗯，上面好像隐约还有点她的体香呢。我抚摸着它，是越看越爱啊。多光滑，多顺溜啊，要不是我们校花，谁能长出这么曲线优美的头发。不过，我忽地想起来，校花娘娘的头发好像不是卷的……我又仔细端详了一番，越看越特莫不对劲。我擦，哥们我都不好意思往下想了，好羞羞啊。（微信公众号：穷雕狂想录。想看下集，马上关注）接下来，我又发现了几本小H书，还有一个套套。我心想，这套套总不会是她一个人用的吧。不过又想，那么一个冰清玉洁、不食人间烟火的仙女，这个东东肯定对她还有其他用处。对，肯定是……走了整整半天，感觉有点累了，刚想找个地方休息会，门突然响了。校花回来了。以前，我记得她走路可轻了，就像小猫似的。如今，我靠，整个一女奥特曼啊。咚咚咚，震得地板都有些发抖。听脚步声，就她一个人。我有心想喊她，又觉得不妥。这么多年不见面了，万一她认不出我，再把我当成虫子，那可惨了。（微信公众号：穷雕狂想录。想看下集，马上关注）校花进门后，直接去了卫生间，哇哇吐了好一阵子。我想，她该不会是怀孕了吧？想到这，心里多少还有些酸酸的。不过，马上哥们就知道了真相。怀妈蛋的孕啊，原来是这位仙女姐姐喝醉了。随着客厅方向叮啦咣当一阵乱响之后，我就听见，她好像是奔卧室来了，接着就是轰的一声，床嘎吱嘎吱响了一阵，感觉好像一座山倒了，再之后就啥动静也没有了。（微信公众号：穷雕狂想录。想看下集，马上关注）黑夜，我和她，还有一张床……哥们是越想越激动啊。这么多年来，梦寐以求的场景就在眼前，我是一刻也不想等了。我顺着床单就往上爬。你还别说，如今这粗布床单就跟软梯似的，也就一刻钟，哥们就来到了床上。这时，我的心都跳成一个点了。我感觉全身都在哆嗦，也不知道是兴奋，还是吓得。我在心里大喊，校花，我来了！可现实也忒特莫骨感了。哥们上床后，竟找不见她。明明知道她就躺在这里，可一眼望去，都是黑压压的大山，鬼知道这都是哪儿对哪儿啊。妈妈的，哥们还有好多想法没实现呢，这丫黑咕隆咚的，哪辈子再找到俺朝思暮想的那个地儿啊。早知道这么费劲，就带个手电筒来了。大约得有二十多分钟，哥们我才搞清楚，原来现在正处在她两只脚的中间。我心里这个骂呀，非丫把个脚洗那么干净干啥嘛，还特莫隐隐约约有股香味；要跟我似的，哥们早就利用生物识别技术，完成嗅觉定位了。（微信公众号：穷雕狂想录。想看下集，马上关注）我沿着山谷往里走，初极宽，能过车，复行数十步，原来越窄，空气中弥漫着一种让人亢奋的气味。我的心跳更快了。大约又走了几十步，前面已无路可走。我放眼四周，仍然是漆黑一片，但周围环境的温度、湿度和气息告诉我，这里就是目的地。（微信公众号：穷雕狂想录。想看下集，马上关注）我在心里反复想着一个问题，是喊醒她再……还是……不过我觉得，哥们堂堂正人君子，怎么能干那种下三滥的事呢……正这时，她突然动了一下，我就感觉整个大地都在颤抖翻滚。还没等我明白过来，就听一声巨响，随后就是狂风大作。哥们我身不由己，被狂风卷到空中，一路筋斗云，就向床下栽去了。

我和校花（二）幸亏校花家床下铺了一块地毯，要么就算有十个我，也早特莫摔死了。半夜里，哥们饿醒了。你想啊，折腾一天了，啥也没吃，换谁也受不了啊。天快亮的时候，我废了九牛二虎之力，来到了客厅的茶几上。我往下一看，我去，忒特莫高了，就跟站在公司大楼顶上往下看一样。我心话，要不是校花娘娘把衣服扔在茶几上，要不是丝袜耷到地上，打死我也爬不上来啊。女孩子家里一般都不缺小零食。哥们大致一，这茶几上光吃的就有好几样。坚果我是不想了，这东西如今都变得比我还大，那壳估计不拿电锯来是弄不开的。果冻和威化饼就更别提了，就那一包装，我想也就是齐天大圣来了才能打开。比较适合哥们我的，也就是苹果了。这东西虽然皮厚，但只要你肯下力气，还是可以吃到里面的瓤的。就眼前这俩苹果，我靠，比两个集装箱大卡车都大。哥们爬到顶上，挖下有半尺厚的皮，才吃到里面的果果。擦！哥们心里这个骂呀！丫吃个苹果，就累成这样子，要是……突然，苹果动了，哥们我站立不稳，差点从上面摔下去。我心话，这特莫又怎么了，扭头看，可不得了，只见校花正拿起苹果往嘴里送呢。那嘴巴比我们公司门口的水池还大，牙齿就跟一扇门似的。我登时浑身就软了。“妈呀！别吃我，别吃我啊！”[size=21.3333px]（微信公众号：穷雕狂想录）我的呼救起了作用。校花发现了我。她轻轻把我捏起来，送到离眼睛特近的地方，惊愕地问我是人还是虫子。我说我当然是人了，不但是人，而且还是你的同学。她又问我，为什么变得这么小，怎么跑到她家里了。我对她说，你赶紧松开手吧，我都快被捏出屎了。[size=21.3333px]（微信公众号：穷雕狂想录）于是，校花把我拿回了卧室，放在枕头上，两只大眼睛忽闪忽闪地看着我，问了这，又问那，还说起了好多其他同学。比如谁去了哪国，谁已经生了孩子，谁已经混成了公司的副总，谁辞职创业去了等等。我俩越聊越投机。校花说，她早知道，我暗恋她。我跟她说，别瞎掰了，根本不可能，因为这事我跟任何人都没说过。她扑哧一笑，捏起我放在她胸前的两座大山中间，让我看着她的眼睛。一时把我搞得手足无措。而她却毫不在乎，说上学那会，一看我的眼神，就知道我心里想的什么。我心话，别扯犊子了，上学的时候，我压根就没敢正眼瞅过她。正这时，校花的手机响了，她忙坐起身接电话，竟忘了我还在她的大山中间。我去，这下可麻烦大了。哥们我是一路跟头，滚到下面的一个沟里去了。（微信公众号：穷雕狂想录）
