# 2

頭好疼……

頭好疼……

頭好疼……

我睜開眼睛，眼前朦朧地出現一些觸手狀的機械，在我周圍忙碌著……

身體好難受，脊椎像是有電流通過一般灼痛。

好難受……

“這是最後的希望了……”

失去意識之前，我聽到這樣的聲音。

…………

再次睜開眼睛，頭還是很疼，身體還是很疼。

我試著動動自己的右手，發現右臂異樣地沉重，……那是我的右手嗎？包覆著奇怪的外殼與機械，熟悉而又陌生的感覺。

然後我發現，不只是右手，我的全身都是這樣，現在的我簡直不像一個人類……

不像人類……那我又是誰？

對了，我究竟是誰？

想不起來…………………………無論如何都想不起來…………

好累，我還是再睡會吧。

…………

…………

“轟——！”巨大的爆炸聲將我拉回現實。周圍的槍聲，尖叫聲和建築的毀滅聲也清晰起來。

手中端著的蓄電槍還在向前方傾瀉著火力，一連串的子彈落在前方龐大的物體上。

不……與其說是物體，它長得倒是比我更接近我周圍的人類，甚至可以說是“她”，而且還她在“她”們當中，還有著足以自豪的年齡、身材、容貌，尤其是身上沒有任何遮蔽的衣物，這本來是可以讓所有男人熱血沸騰的景象，但是問題在於，她實在是太巨大了。

有多大呢，我無暇去思考這個問題，我所站的大街只是剛好足夠她用兩只腳來行走，而街兩邊的商場還不及她的膝蓋。雖然距離我們有好幾百米，不過以她的身高，走過來的話只是幾步的問題，而且她正在往這邊走。

當然在我和我周圍的那些穿軍裝的人眼中，更直觀的景象是，兩只巨大的腳交替地向我們靠近，每次腳掌落下，都能在零點幾秒後感受到大地的震動，以及迎面而來的充滿沙塵和焦味的強勁氣流。落地後的腳掌會陷進地面兩三米，並且偶然有火焰從腳掌底下噴出——那些是受到重壓而爆炸的載具。而當腳掌再次抬起，各種各樣的殘渣碎屑從腳底脫落，原地則留下一個鋪滿扁平殘骸的腳印。

“轟——！”只是幾秒鐘後，一只巨大的左腳落在我們前面大約20公尺的地方，那腳掌與地面接觸前，我還清楚地看到了粘在上面的破碎的人類屍體和被塵土沾染的血跡。

“轟——”右腳也跟上來了，停在了左腳的旁邊，周圍的槍炮聲頓時消失了，士兵們早就丟下武器四散逃開，裝甲車的蓋艙門打開後，裏面的機師也爭先恐後地往外逃……

我的內心感覺到一種不安，那是動搖？還是恐懼？我不明白……但是身體依舊沒有要離開原地的意思，而手中的蓄電槍還在鳴叫著……

電漿彈落在巨大少女的腳背上，散射出美麗的火花，但是卻沒有對其造成任何傷害。

我明白我和我面前的她不會一只保持現在的姿勢。

她動了，動的是那只右腳，只一個瞬間，那只足以占據一個球場的右腳掌懸在了我的頭頂，四周變的陰暗了，不停有灰塵和肮髒的屍塊掉落在我身邊。

“轟——！”沒有遲疑，周圍暗下來了…… 

# 3

頭好暈……

頭好暈……

頭好暈……

幾乎無法站穩，身體好重……

我這是在哪裏？我想問別人，但是無人可問，周圍沒有熟悉的人，沒有陌生的人，甚至沒有“人”。既然無人可問，那我還是沉默吧……

啊，硬要說的“人”話，在我腳邊四散逃開的東西倒是長著人的樣子，但是那太虛假了，還有那些樓房，那些車子，都跟玩具一般大小。

太假了……

太虛假了……

既然是虛假的，那就不用在意吧？就算是弄壞了，也沒有關系對吧？

我漫無目的地走在街道上，思考著我究竟是何時站在這裏的，這裏又是什麼地方……腳底時不時傳來癢癢的感覺，好像是被我踩到的車子爆炸的樣子。

這樣子破壞是不行的吧……但老實說腳底癢癢的感覺很不錯，我就睜一只眼閉一只眼好了。

“痛！”腳背傳來針紮一樣的感覺，這時候我注意到，還有那麼些沒有逃跑的小人，正用他們手中的武器轟擊著我的腳背。他們就在離我一米遠的地方，我打算走的更近一點仔細看看他們，但是我剛走到他們跟前，他們也像其他小人一樣四散跑開了——真遺憾。

不過腳背上的刺痛感並沒有消失，原來還有一個小人沒有逃走。

這就對了，你們為什麼要逃走呢……又不對……我能感覺到，這個沒有逃跑的小人，是特別的，是不同於那些家夥的……

“痛！”你就不能停下來嘛……這樣的想法形成之前，身體的本能先有了反應。等我回過神來，我的右腳已經踏在了那個小人站的位置。腳底再次傳來癢癢的感覺，但不是那個小人帶來的，我這才想起來，在他的周圍還有一些裝甲車……不知怎麼的，心裏有點遺憾。

周圍徹底清淨了，又只剩下我一個人了……

為什麼我總是一個人……

腦中浮現出一些不清晰的畫面，離我而去的背影，無論如何也捕捉不到……

頭又開始疼了，我搖了搖頭，將奇怪的想法趕出大腦，開始重新打量周圍。

這是一座城市，雖然是那麼的小，但是他有著一個城市應有的一切……除了人。

不，大概是因為我的緣故，他們躲起來了吧……怕我傷害到他們嗎……

原來，在人們的心目中，我是會傷害他們的啊……

那麼，既然你們是這麼希望的……

不對，不應該是這樣……

開始混亂了，我究竟是誰，又為什麼在這裏……

“啊啊啊啊——！”仿佛在掙紮一般地跺了跺腳，等回過神來，才發現街道旁的建築都在我的腳下化為了細小的瓦礫。還有少數的小人逃離了倒塌的廢墟，朝遠離我的地方跑去。

我不清楚有多少小人在這些由我隨意制造的廢墟中喪命，但是我沒有把我的腳收回來。鋼筋與混凝土在腳下崩毀的感覺順著神經傳到我的全身，讓我踏實地感覺自己存在著……

我需要這種感覺……

於是，我開始踐踏更多的房屋，甚至開始踐踏那些逃跑的小人……

有什麼東西壞掉了，在我的裏面……我這麼感覺到，但是已經無法回頭了……

漸漸地，我發現，踐踏那些逃跑的小人更能讓我找到存在的感覺，他們逃跑時的絕望與即將在我腳下化為模糊血肉前的尖叫，都實實在在地證明著我的存在……

還不夠，還要讓這種感覺更加強烈，這樣想著，我不知為何向虛空中伸出了手。

不只是被誰賦予了這種力量，城市裏的小人開始從他們隱藏的地方飛了出來，聚集在我伸著的手掌前方，懸浮著……

我倒是被嚇了一跳，將手收了回來，那種力量也消失了，小人紛紛從空中跌下去。

我沒有看到最先落地的小人是什麼下場，我只知道，這裏已經匯聚了足夠多的小人，而我只需要將我的腳踏上去……不，這次我直接並著雙腳跳進了小人堆中。

“噗嘰——”從來沒聽過的聲音，從來沒感受過的感覺，要說相似的話，就像一瞬間捏碎了防震包裝膜上所有的泡泡，但是這裏的感覺要強烈、鮮明不知多少倍。

這種感覺會上癮，我知道我該怎麼做了…… 

# 4

身體沒有知覺了，為什麼意識還在？

那個巨大的少女到底是什麼存在，又是什麼時候來到這裏的……

腦中閃過一些破碎的影像，我曾面對她，逃跑了，但那似乎是犯下了什麼重大的過錯……

所以我這次又站在這裏，這次一定不能逃跑。

也因此，我才會被她踩在腳下吧。

那一腳，我應該已經化為一灘血肉了才對。

但是現在的我確實地活著。

視覺恢複了，感覺也慢慢回來，我吃力地爬起身，發現自己正處於一個巨大的坑中，不用說這只是她的腳印而已，周圍盡是被壓成制片一樣的裝甲車——沒被壓扁的地方也嵌到地下去了。

再看看自己，托這身奇怪的外殼的福，我感覺不到身體受到了任何損壞。手邊的蓄電槍還能使用，我必須去找她……這次不能逃跑。

費了點力氣，我爬出了那個足有四米深的腳印，她已經走到離這裏幾公裏遠的地方去了，但是我依然能看到她巨大的身影。

從路邊撿起一輛還能用的摩托，我轉滿了油門向她所在的地方開過去，這次不能逃…… 

# 5

這次我的目標是市區裏最高的那棟大樓，好想破壞掉，只有親手破壞掉才能感覺到我的存在。

“痛！”異樣的感覺從腳後跟傳來，我轉身低頭，發現一輛摩托車證倒在我的腳邊，輪在還在飛快地轉著——就是這個東西撞上了我吧。

而再前面一點的地方，我看到了那個熟悉的，奇怪的小人，本來應該在我的腳下化為亡魂的小人……他居然又來了。

雖然對於他為什麼沒有死抱有疑惑，但是心裏卻有點慶幸的感覺。

其實不用費力氣破壞城市來證明自己的存在，在這裏，就有人在注視著我，雖然他此刻正拿著槍向我開火，但就是這樣，卻比那些犧牲在我腳下的普通小人和建築要來的有意義得多。

但是好疼，能不能請你停下來……雖然我未曾受到傷害，但是這裏唯一注視著我的人卻是來傷害我的，這讓人稍微有點不能接受。

再一次，我將右腳抬到了他的頭頂，他仍舊沒有逃走，不過這次我只是將腳輕輕地壓了下去，只要能讓他停下當前的動作就好。

這次的街道很空曠，除了那個小人之外沒有其他的東西，所以當我的腳掌和地面接觸時，我能清楚的感覺到他在我的腳心處。

我能感覺到，隨著我的腳掌下壓，他的姿勢由站立變為仰面躺下，身體陷進我腳心的皮膚中。

掙紮的感覺從腳底傳來，此刻我的心裏感到無比的滿足。

“這次一定不會讓你丟下我一個人”心裏不知怎麼會產生這樣的想法，或許，我和我腳下的小人，本身就存在什麼特別的聯系也說不定吧。

很奇怪吧，居然因為這麼一個小人，我就能得到滿足，很奇怪吧……

腦中無端地生出了其他的念頭，那個壞掉的聲音似乎又想起來了，右腳下意識地加大了力量，腳掌很快便陷入了地面。

腳下的掙紮很快停止了，回過神來的我慌忙將腳拿開，蹲下來觀察腳印裏的他。

還好他看上去沒事，即使是足以將地面踩塌的力量也沒有對他造成實質性的傷害，他爬了起來，向遠離我的地方跑去……

終於，你也要走了麼，不知為何我覺得，如果他要離開，我是無論如何都留不住的……

“痛！”疼痛的感覺再次傳來，原來，他只是跑出一段距離，便回頭向我開槍了。

“你是叫我跟著你麼……”我重新站起來，向他走過去，心中有一種莫名的興奮。

他花費好大的力氣才跑出的距離，我一步就能追上，然而他還是在繼續跑，並不時地回頭向我開槍，那一定是在叫我跟上。

我想我一定做出了好笑的表情，他居然怕我跟不上麼。 

# 6

又一次被她踩在腳下了，不知為何，這次的一腳並沒有那種毀滅一切的氣勢，只是想把我壓在下面這種感覺……或許對於她來講，這種程度的反抗就跟遊戲一樣吧。

我試著鑽出她的腳下，但很快發現這是徒勞的，她的腳心皮膚很柔軟，像被子一樣蓋在我身上……真難想象這樣柔嫩的肌膚是如何抵禦槍炮的侵襲，又是如何將一片片房屋化為瓦礫的……

“滴——”身上的某處傳來電子訊號的聲音，腦中浮現出某個地方的坐標，我需要把這個巨大的少女帶到那裏去……

原來我的身上還攜帶著這樣的設備麼。

踩著我的腳突然加大了力道，我隨著地面向下陷進去，但是已經有了一次經曆的我知道，這並無大礙。

再次爬出腳印，我向預定的方向跑去，但是跑了一段距離，發現巨大的少女並沒有跟上來，而是頹然地蹲在那裏，兩眼無神地望向地面。

如果我不能吸引她的注意力，她又會像那樣到處破壞了吧，我此刻才想到，我應該是來阻止她的，對，一開始，我就應該是為了阻止什麼而誕生的，所以我才在這裏……

於是我向巨大的少女開了槍。

然後她注意到了我，用一種不可思議的表情看過來，似乎有一點喜出望外的感覺，然後她起來了，一邊盯著我一邊走了過來。

為了避免被她踩中，我只能繼續邁開步子跑，但她總是追上幾步便又停下，我怕她中途走掉，便不停地回頭開槍，但後來我發現，她停下只是因為她走的比我快太多了，是她要等我而已……

既然她也決定跟著我，我幹脆集中精力跑吧……

忽然眼前光線變暗了，雖然沒有回頭，但是周圍熟悉的氣氛讓我很快了解到接下來要發生的事情……

在倒地之前我一個轉身，讓自己面朝上，這意味著我的臉再次埋進她的腳掌中——這總比埋進地面要好…… 

# 7

雖然沒有語言的溝通，但是我和這個小人的意見居然達成了一致，他是要帶我去什麼地方吧，那個地方一定有什麼，能解開我的疑惑。

於是我默默地跟著他，起初，他還不時回頭開槍，來確保我在跟著他，但是漸漸的，他也只顧自己趕路了，雖然我明白他還是在為我帶路，但是孤單的感覺再一次包圍過來。

於是，不能自己地，我再一次將他踩住。

終於有閑心打量四周，才發現自己已經被這個小人帶著離開了原先所處的城市。

這裏沒有硝煙也沒有廢墟，只有寧靜的草地，夕陽放射著殘酷而晦暗的光芒，更加讓我感覺到，這個世界上，只有自己和腳下的那個小人是確切存在的。

夜幕降臨時的風，把涼意從皮膚直吹進我的靈魂深處，唯一能感覺到的溫暖，來自於腳底的那個小人，不過，這就夠了，只要能感覺到他，就能感覺到自己。我開始確信，這個小人是和自己有著什麼聯系的，不是因為他不懼死亡地糾纏上我，而是因為更加直接的原因……

好困……也許是這幅景象過於讓人寂寞，又過於讓人安心，我感到困倦了。

從何時開始，我沒有合上眼好好睡過一覺呢？已經記不清了，從前只是不知疲倦地毀壞於破壞著的我，現在想要在這原野之上，閉上眼睛，好好地睡一覺。

但是如果一覺醒來，那個小人又不見了怎麼辦，我又會回到一個人的世界……那樣絕對不要……一定要防止……

對了……可以這樣……
