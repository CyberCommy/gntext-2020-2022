# 【新年快乐】奥古斯特的鞋中大陆

TID：32535

# 1

*本帖最後由 内置冰刺的水柜 於 2022-1-31 21:09 編輯*

新年特别稿QWQ希望新的一年能够有更多人支持咱工作的论坛GW当然，以下就是为新年摸的文章）随着坐在皇位上的奥古斯特晃了晃自己的脚，在那之上的黑色高跟鞋，就不断的在脚尖之上晃动，如同摇摇欲坠的大陆一般。在那高跟鞋里的城市可就倒了霉。随着那高跟鞋的摇晃，那高跟鞋里的鞋垫大陆便随之晃动，令小人们在这晃动之中不断的尖叫和咆哮着。不过，虽然数百万人为此咆哮，为此不满，但他们的生命，甚至不如少女脚上寄生的细菌，此等暴乱，自然是无法令少女感到分毫差别了。“要开始了哦?在我的脚完全压下之前赶快逃出去吧～嘻嘻～”随着那脚跟着地的巨大脚底正不断向着自己压下，整座城市早就慌作了一团。但，对于这个还没有她脚底面积大的小小城市，以及当中所居住的弱小微生物来说，再怎么挣扎都是毫无意义的。那几乎完美的粉嫩脚底，所散发出的气味，正不断的令下方城市中的小人们迷失，而不断碾近的皮肤，正将他们，以及他们弱小的造物笼罩在那完美的足底曲线之下。“攻击！”随着小人们的导弹不断的发射，那足底也也被如此之多导弹的火光所照亮。“哦?看来你们这些小人还是稍微有点本事呢?”但，对于逃跑多时的小人来说，在无数次的尝试之后，他们终于明白了这黑色高跟鞋所形成的囚笼，是多么的坚不可摧。无论小人们爆破，钎凿，甚至都无法令那黑色的高墙所损毁一毫！不过此刻，已经按不住脚痒的奥古斯特，渐渐的踩下了她的神足。随着如同千万年漫长的失重感，无数的小人和他们的造物被如此震飞到空中，但随着高跟鞋着地，那处于奥古斯特脚掌下，挣扎的最激烈的士兵们，率先随着他们所自豪的高楼大厦，以及他们称作“坦克”的兵器，一起被那玉足压碎在了黑色的鞋底，变为了灰色的尘埃。兴许是为了将鞋完全穿上，那被丝袜所包裹的五指，正沿着城市向着鞋尖，如风暴席卷大地一般。可是，对于鞋尖和脚趾间残存城市的小人们来说，如此存活下来几乎不可能。即使是少女刻意减慢了自己脚趾的速度，但在这已经几乎被东倒西歪的大楼所阻碍的城市之上，也只有那些本来就处于鞋尖的小人们，得以在巨大的，沿着鞋底不断往前的脚趾下幸存。也许，他们的余生，就是依靠残骸里残存的食物，以及少女脚趾皮肤上所分泌的晶莹剔透的汗水维生。最后，当少女的脚趾完全碰触到了鞋尖的时候，二分之一的大陆，近一亿名的小人，除了那些在脚趾缝隙间苟延残喘的小人，连同他们百年来所建设的一切，都成为了少女脚尖所吞噬的，亦或是前脚掌所无情碾碎的弱小微生物。而少女此刻舒展身体，完全踩着大地站了起来的时候，那在鞋根之上，残存的小人城市，也在一瞬间被少女落下的足跟所消灭。数个城市几乎都在同一时间，就如此消失在了一位少女的足底当中，即使是脚趾间还有那奄奄一息的小人们，下一秒也随着脚趾的揉搓，那残存的绝大部分幸存者和他们所依靠的残骸，也被脚趾所反复碾碎，变成了彻彻底底的废墟。也许，在那两亿多的幸存者当中，只有数百个躲在脚趾和鞋尖之间的夹缝，亦或是在少女足跟落下之时被黑色纤维困在足弓间的小人们，得以在未来彻底成为只能依靠奥古斯特足底代谢废物所生活的细菌。虽然这一脚就几乎踩碎了整个大陆，在数分钟间屠杀了数以亿计的小人们，但奥古斯特只是嗤笑了一下小人的弱小，便走了几步。剩下的幸存小人也就这样被她巨大的脚趾所吞没，碾碎，消失在了她丝袜的纤维中。随着少女迈出自己的房门，那辉煌的大陆，和他的历史，就这样在数分钟间葬送在了少女的仅仅是一只足下。而换来的，仅仅是女神的一点愉悦感，以及对弱小之辈的鄙视罢了。
