# 1

反正论坛里的坑那么多，也不差我一个，而且，我也不是初犯。

最近闲着的时候，一直在翻看一些老电影，多数是些港产片，太高雅的我看不明白，从小就喜欢港产片，因为它通俗易懂，剧情简单，场面热闹，结局喜庆，我就喜欢皆大欢喜善恶有报的故事，这正适合我的品味档次，看到感动的时候，会有写文的冲动。

其实，我只是写了点开头，没有长远的规划和持久战的思想准备，因为我发现，那股冲动劲来的很快，去的更快，有时候还没动笔，就觉得无聊了，所以下一章什么时候写，我也不知道，所以坑的可能性很大，这也不是会让人内疚的事情了吧。

我打算，gts剧情不超过十分之一（争取有突破），而且，该属性的出现，只为合乎论坛的主题思想，如果和故事的整体显得不和协太突兀，就再降低比例，也不打算出现和性爱有关的任何描述，时髦的话，叫无“撸点”。

简单的说，这应该是一篇比较清淡的文，很难让人兴奋，但也绝不是温柔系，所以，看之前请三思，莫要抱有过多期待。

我不太清楚同人文的具体概念，如果照搬了一堆别人作品里的姓名和片段，然后乱炖一番，算不算是同人文呢？因为搀和了太多的作品的东西，所以，我也不知道算是哪一个的同人文。

从故事的主线来看，大多是抄袭了《无间道》的人物背景，电影我看过，小说我也看过，我都喜欢。但我只指的是前两部，那个第三部我不喜欢，因为配角的戏份不精彩，尤其是康熙爷的加入，只有最后刘建明泪眼倔强的以枪抵喉的那瞬还有点感觉，可怎么看也比不过第二部倪永孝死的精彩，当然，最精彩的还是开篇时候黄Sir高空坠落那幕-------再见，警察，来的悲凉入骨无法释怀。

呵呵，对，我大概写的就是这些过时的东西，虽然很难写出那种味道，但不要失望不要气馁，女人迟早会出现，一定会杀人，不然怎么叫做脂粉杀呢---------

我决定，将无视任何人的评论，拒绝任何人的建议，一意孤行的写下去，呵呵。

**黄志诚** 

-------------那一年，我刚入行，还是个巡警。

某晚，我正值当班，接报位于永安路的新世界歌厅内有事发生。

我和一个师兄赶至现场，歌厅里到处是吸毒后疯狂的人渣，几十个人正在殴斗，十六七个流氓倒地呻吟，地上血迹斑斑，在我脚边是一个仍紧握水果刀的断手。

我从未见过这种场面，顿时慌了心神，眼看着一个黄毛小子往我身前奔来，意图冲逃。

我张臂欲拦，却发现那小子手里握着一根铁管，面带杀气，举管要刺。

师兄叫我掏枪，我便去掏，岂料枪袋还未打开，我就被师兄一脚踢倒，再回头时，师兄已经倒在我面前。

那根一米多长的铁管，一下子捅进了师兄的腹部。

我抱着师兄，鲜血从他喉管涌出，直喷我身。

黄毛小子弃械而逃。

我被恐惧冲昏头脑，对着他的背影狂扣扳机，待我定下神来，只见我伸直双手，食指仍在不断的扣动扳机，发出空洞的卡擦响声。

黄毛小子满身鲜血倒卧地上，奄奄一息。

两天后，我重见师兄，地点在停尸房。

我不敢抬眼看师兄的尸体，因为，躺在那里的人，本应是我。

师兄一家老小四口默默无语，我却哭的泣不成声。

上司叫我几天后上法庭，因为那黄毛小子并未死。

结果他被判入狱几年我忘了，我只知道，我师兄死了，而他还活着。

最近我再遇见他，是在一个聚会上，他不再是当年蓬头垢面的臭小子，一身笔挺的西装，手带金表，好不光鲜，站在身边的十几号人马，全是他的手下。

我当时就坐在他身后，死死的盯着他的脖子看，因为那里有一处枪伤，是我当年留给他的。

他没有注意到我，他和我几位上司，觥筹交错，聊的正欢。

杀人放火金腰带，修桥补路无尸骸。

我已不太记得师兄的模样，我只是后悔当年没把那六发子弹瞄准黄毛小子的头发射，哪怕一颗致命也好。

这个世界不该是这样，做人不该是这样。

我只是个警察，有上司，有手足，有纪律，有约束，终究无法改变什么。

但十年前，我带领一干手下在师兄坟前上香的时候，就对自己说：既然我还活着，那我无论用什么手段，都一定要让这世上的所有人渣都记住我的名字-----黄志诚。
