# 2

背景：隨意

初始地：被埋在地底下

屬性：無口

突發事件：豬腳遭到襲擊而家破人亡

關鍵要素：鐵錘 可疑的卷軸

0

基德睜開眼睛的時候，並沒有像往常一樣迎來光明——等一下，自己是什麽時候失去意識的？這裡又是什麽地方？在他能夠開始思考這些問題之前，周身的疼痛和壓迫感便隨著他的意識一起朝他湧來。

基德開始感到呼吸似乎變得困難了，可能這是一個密閉的空間。拋開其他念頭，基德把注意力集中在“尋找離開這裡的方法”上，但他似乎被什麽重物壓在身上，手腳幾乎都無法動彈。

“開什麼玩笑……”咬著牙縫擠出這幾個字，基德使出渾身的力量想要揮開身上的重物，但它（們）紋絲不動。他試著揮動了幾次手，卻只能橫向敲打到一些木頭似的東西。

空氣似乎越來越稀薄了。基德的注意力也開始逐漸散開。他開始回想自己爲什麽會陷入這個境地，甚至是更為久遠的一些東西。

姐姐。

頭似乎又要開始發暈了。難道自己就要這麼死去嗎？不知何時何地，也不知道被什麽東西——這樣地殺死。更糟糕的是，死前沒有人陪在自己身邊。基德自詡精心準備好了臨死前的遺言，絕對是帥氣又感人的，現在看來也派不上用場了。

要說嗎？還是不要說嗎？

正當基德開始認真地煩惱這個問題的時候，問題得到了解決。

1

基德躺在一個坑里，大口大口地喘著氣。事實上，他從來也沒有覺得天空是如此地美麗過，這也不單單是對夕陽、雲彩，亦或是那位少女的奉承話。

得救了。

從現場的狀況來看，基德是被地板、瓦礫等等亂七八糟的東西一起被埋在了地底下。所幸的是有位剛才提到的，現在再補充說明一下或許是剛好路過的穿著披風的少女把東西及時搬開，使得基德免於窒息而死之災。而把東西都移開后，地面上就多出了一個坑。

等等，就憑那雙纖細的黑色手套？還是說那是什麽增強力量的神奇道具么？

但是這裡沒有其他任何人。

粉紅色中長髮的少女靜靜地站在坑的邊緣。她並沒有任何多餘的動作，臉上也沒有任何表情，也沒有說一句話，她所做的只有用那雙紅寶石般的的大眼睛盯著基德的眼睛。少女就這樣一動不動地站著，甚至連披風也沒有隨著微風搖擺，仿佛從亙古以來便一直如是站在那裡一樣，又仿佛從來也不曾站在那裡。

她的眼睛很美，但令人遺憾的是，那雙不輸于夕陽的紅寶石總是猶抱琵琶般地半眯著，令人不禁想要一探其中的究竟：在那雙看似朦朧的明眸中，究竟隱藏著怎樣的悲傷？基德不禁看得出了神，甚至忘了要從坑中爬起來，就這麼直直地仰視著美麗的少女。不光是眼睛，她那小巧精緻的臉龐、蓬鬆而明亮的秀髮、纖細無骨的四肢也都同樣極具魅力，但最吸引基德的還是那雙鮮紅如血的眼睛，裏面好像有一個璀璨的世界，令基德無法將注意力移開半分。

至於自己爲什麽身於此處？究竟發生了什麽事？眼前的少女究竟是什麽人？要理清的疑問簡直堆積如山，要發出的疑問層出不窮，但基德一點也不關心它們了。他關注的只有那雙美麗的紅寶石……哪怕注視著它會化為石像，基德覺得也值得。

及其輕微地，少女櫻唇微張，但又欲言又止般地合上了。她的表情仍舊平靜如水般毫無變化，但卻默默地合上了雙眼，像是下了很大決心一樣，把白皙的脖頸轉到一邊。

忽然間，朦朧的世界，美麗的夕陽，全部都消失不見了——雖然夕陽還是夕陽，少女也還是少女，但是那如夢似幻的氣氛卻驟然消失不見。飛走的思緒回來了，周身的疼痛也隨之一擁而上，但所幸的是基德傷得也不太嚴重。他站起身，拍了拍身上的塵土，撿起他那標誌般的帽子——基德一直認為，若是沒有這頂帽子，牛仔就不像是牛仔了，可他忘了他自己根本就不是一個牛仔，哪怕他有模有樣地穿著馬靴、掛著槍套。

——戴上。

“妳好！是妳救了我嗎？真是感激不盡！”基德三兩步跳上地面，想要去握對方的手，“我叫基德……”但少女卻把身影轉了過去，還向前走了幾步，像是沒有注意到基德的存在一樣，逕自走入原本是房屋的廢墟中。

“……哎？”基德感到很奇怪，他上前幾步，伸手去拉少女的肩膀。“妳怎麼了……”但少女卻頭也沒回，靈巧地避開了他。

“……喂，”基德露出了疑惑的神情，卻又忽然僵住，甚至連伸出去的手都忘記收回來，就這麼直直地伸在空中。

“混蛋……到底是怎麼回事。”

真是奇怪。怎麼會忘記了呢？明明不可能忘記的，自己來這裡的目的，已經發生了什麽事。但一見到那個少女，就像是被催眠了一般，忘記了要做的事……哪怕是一秒鐘，那也是不可原諒的。

暗罵了自己一句，基德趕緊跟了上去。那原本是一幢寬大的木屋，但此刻人們所能見到的，就只有被折斷的屋樑，移位的屋頂，被壓扁、碾碎、一片狼藉的各種傢具物事，以及……一隻巨大無比，不僅將整幢房子壓塌，並且在周圍數公尺內都留下清晰凹痕的，巨大腳印。

2

基德•卡文迪許是一名信客。其實說白了，就是給人送信的郵差或者信使，但基德本人非要如此自稱不可了，大概是有什麽重要的理由吧。實際上，在通訊技術發達的今天，郵差……我是說信客，反而是一個逐漸消失的職業，因為人們可以透過互聯網和手機等先進設備來傳達訊息，不必再依賴效率低下的傳信。不過既然這個職業還存在，就說明還是有著哪怕兩個要寫信的人了。

總而言之，基德今天本來是打算到姐姐家——或者說回家，因為那幾乎是他唯一的親人——送一封信，然後再將事先說好的姐姐寫好的信回送給對方。僅此而已，并不會有久違的姐弟寒暄、熱情的重逢之類的親情戲碼。

沒辦法，基德和姐姐的關係就是如此——毫不親密的親人。會委託他送信也是處於他的業務能力，而非作為弟弟可以信賴的理由吧。但基德對此也毫不在意，不如說如果姐姐忽然像個姐姐一樣對待他的話，他反而會不自在吧。

但現在基德覺得糟糕透了。

因為他面對的是，血液已經凝固、屍體已經開始腐爛的，八十歲老人的屍體——他姐姐的屍體。

在回憶的攻勢之下事情的經過并不難推斷：基德造訪的時候，一位巨大的不速之客忽然登門，并毫無禮儀地一腳將整棟房子踏爛了三分之二。其中基德十分“幸運”地被天花板、倒下的書架、以及各式各樣的東西埋在地下兩天而沒死；而姐姐則不幸被巨大的鞋子命中紅心，當場化為一灘肉泥。

很奇怪，雖然基德率先想到的是兇手是誰這個疑問，但他居然不是要為姐姐報仇，也沒有要把姐姐好好安葬的樣子。他只是靜靜地注視著屍體好一會兒，才開始在房間里——至少是曾經被稱為房間的地方——走動，翻動各種物事。說不定還能找到姐姐要他送的東西或者書信。他往不遠處的樹林邊上一瞥，發現自己的愛馬愛德華還被栓在樹上，兀自吃著草。謎一般的粉紅色頭髮少女也在不遠處……

這時他才注意到，粉紅色頭髮的少女也在翻找著什麽，而且有意無意地注視了他好久。

忘記了此前的失利，基德再次上前和陌生的少女搭話，“我的名字是基德•卡文迪許，再次感謝妳救了我一命。這裡的主人……是我的姐姐，我也算是對這裡有一定瞭解，如果有什麽地方可以幫得上忙的話請務必讓我效勞。”雖然他一貫粗魯，但如果仔細思考過還是能儘量有禮貌地說話的。

但少女仍舊無言，只是指了指自己的喉嚨，擺擺手，似乎示意她不會說話。

“那、妳會寫字嗎？”基德掏出自己的手機，很幸運的是它居然沒被壓壞。少女半眯著紅寶石一般的大眼睛，像是看著什麽新奇的事物一樣。“這個要這樣……”基德想辦法教給她最低限度的使用方法。幸好她很聰明，很快可以在上面寫出字來。

【我是尼路斐尼亞•托爾。在尋找米爾納。現在的我無法說話。】她的字句很簡潔。

“叫我基德就行了。那麼尼路醬……”不知不覺中基德的言辭又開始有些失禮了，不過面無表情的少女似乎不是很關心這種小事，非常爽快地點了點頭。

“請問米爾納是什麽？”

粉紅色頭髮的少女眨了眨眼睛，並沒有在手機上打字，而是用手作了個敲打的動作。

“……是一把錘子？”

她點點頭。但像是想起了什麽似的，在手機上補充道，【有可能被封印起來了】

這樣可能就麻煩了。基德嘟囔道。基德的姐姐是一個愛好古董的收藏家，可以說是什麽樣奇奇怪怪的東西都有。而且大部份是不在這間房子里而是在保險庫、密室之類更加隱蔽的地方。但是現在姐姐已經死了，沒人知道那個所謂的米爾納到底被放在哪裡。

……只能一件件來了么。基德把自己的帽檐扶正。

嘿小子。你就是用這樣的態度面對你的姐姐的嗎？

“吵死了，旁白就該在旁邊閉上嘴巴好好看著。”

真是可憐的姐姐啊，孤苦伶仃地終老，連死去了都沒有人來替她收尸，這就是一個弟弟所應為的嗎？

“……”

啊我知道了，你一定是口嫌體正直的類型吧？嘴上雖然很臭，但沒人的時候你還是會悄悄地安葬她，還會傷心欲絕地流下幾滴眼淚呢對吧？搞不好還會把殺人兇手碎屍萬段對吧？

哦好可怕好可怕，不要把槍口指著我∼

“你夠了哦，我不做是因為我知道有人會做！”

是嗎？

“是……吧。”

是嗎。

3

尼路斐尼亞看著那個二十出頭的青年手忙腳亂地翻開各種已經壞掉的傢具。

沒有……沒有……

儘管一次又一次地失敗了，他還是幾乎狂亂地翻找著，卻不知他到底希望找到，抑或不希望找到？問題的答案，或許在他自己的心中也迷失了。有的地方已經是第三次翻開了，木屑和灰塵在空中飛舞著。

盡力遠遠地，尼路斐尼亞注視著基德躬身的背影。乍看之下，她的臉上仍舊沒有一絲表情；但若仔細觀察，就會發現她的臉頰稍微有點發力——用力抿住自己的嘴唇。

事實上，她已幾乎拼盡全力。

尼路斐尼亞今天“早上”醒來的時候，是被一陣敲門聲吵醒的。當然，那是她還不知道那是基德被埋在地底下奮力求生時無意所致。但是，當她醒來時，她很意外地發現——她無意中把棺材的門打開了。

時隔不知幾個月，尼路斐尼亞又見到了“朝陽”，接踵而來的“早晨”，以及熟悉的眾生。然後她看到了——已然化為廢墟的，用來鎮壓她的封印的房屋。

尼路斐尼亞是被稱為吸血鬼的存在。

至於自己是什麽時候、被什麽理由所封印的，她已經記不太清楚了。就連封印她的那個紅白巫女的面孔，她似乎也記不得了。她唯一記得的就是這個她熟睡了許久的棺材，她就被封印在那裏面，終日過著睡了又醒，醒了又睡的生活，直到這個封印被打破。

很長時間沒有吃東西了，但吸血鬼卻不會因此餓死。這或許是他們眾多神秘的能力中的一項：極度旺盛、難以磨滅的生命力。或許這也是那個巫女沒有殺死她而選擇封印她的原因。

但在封印消失之前，她的食慾也隨之被壓制著，但現在卻不可阻擋地爆發出來了，這一點在她運用怪力移開重物救出那個年輕人類之前，並沒有被她意識到。

想要吸他的血嗎？

尼路斐尼亞沒有回答。

基德也好氊帽也好馬靴也好，人類的標誌對她而言毫無意義，在她的眼中，所有人類都是一樣的——都是她的食物。至少作為吸血鬼的她是這麼不思考的，這是她的本能，她的生理衝動，完全無需經過思考。

但不能吸他的血嗎？

尼路斐尼亞沒有回答。

基德也好氊帽也好馬靴也好，人類的標誌對她而言毫無意義，在她的眼中，所有人類都是一樣的——尼路斐尼亞•托爾，自從誕生以來，就不曾吸過任何一個活人的血。至少作為巨人的她是這麼思考的。

自相矛盾。

既是如此，當初爲什麽又要救他呢？還是說爲什麽要救人呢？爲什麽要救活著的生命呢？爲什麽要救死去的生命呢？爲什麽不吃了他呢？爲什麽不吃了可以見到的生命呢？爲什麽——不吃了我呢？

尼路斐尼亞沒有回答。

相信我，我只是旁白而不是作者，就算我死了故事也會若無其事地繼續進行的哦？

尼路斐尼亞——沒有回答。

“啊！這、這個是——！！”基德忽然驚愕地大叫，他的手中握著一個卷軸，激動地不住顫抖。

怎麼回事？她本想這麼問，但此刻的她無法開口。然而她甚至來不及開口，基德就毫無預兆地忽然仰面倒下。

顧不得吸血衝動，尼路斐尼亞趕緊衝上前去，卻只見基德奄奄一息地呻吟道：“餓死了~動不了啦~”

“……”這是在干什麽，撒嬌嗎？尼路斐尼亞不禁感到一陣暈眩，但她並不清楚，基德已經被活活埋在瓦礫堆下兩天了，滴水未進，能活下來就已經相當幸運了。畢竟人類的生命力可遠遠比不上吸血鬼。

尼路斐尼亞努力把視線從基德的脖子上移開，四下張望。畢竟這裡曾經是有人住的房子，如果裏面沒有食物和水那反而說不過去。她決定在大概是廚房和地窖的地方找找看，有沒有能夠從那隻巨腳下倖存的完好的食物和水。

……等等。如果這裡有人居住，那麼……這個“人”在哪裡？

偏偏這個時候，吸血衝動讓她的嗅覺變得異常靈敏，一股血腥味悄然無聲地鑽入她的鼻孔，她把視線又移回基德身旁的一個瓦礫堆。

她戰慄著將上面的重物統統移開，並非出於恐懼，而是出於興奮：她看到的是被壓扁的血肉模糊——雖然血液已經幹了——的肉塊。

在這裡的主人，恐怕已經死了。

但對尼路斐尼亞來說，卻是另一層意義上的恐慌……她恐怕無法再，按捺住自己了。

在黑夜之中，她的眼睛鮮紅如血。

4
