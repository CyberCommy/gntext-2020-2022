# 2

“呼....呼....呼....呼”气喘吁吁地奔跑在戈壁的夏夜是一种奇怪的感觉，尤其是坐了很长时间的车之后。这种感觉就像你踩到的一开始是硬地，紧接着你听到“咔嚓”一声，脚下一软，仿佛踩的东西又变成了一块棉花糖。不得不说，这真是个避暑的好地方。即使是盛夏，夜晚的温度也足矣让你瑟瑟发抖。一般看来，在这里跑步是极为不舒服的，可是，我并没有停下来的机会。

“咚........咚......咚....咚”仿佛是一个低音炮的音响膜一样，地面正有规律的震动着。这震动越来越近，越来越响，好像正在逐渐的接近我。我跑得更快了，可是那震动好像也变得更频繁。就在我回头想要看看是什么在追我的一瞬，一个黑影从我头上跨过，落在我前头“糟了，要撞上了”头还没回过来，我便和这黑漆漆的东西撞了一个满怀。犹如撞到墙一般，我的全身的骨头都似乎散了架子，即使有说明书也再也拼不回来。只是....这墙为什么有点软？

30分钟前

“啊！”我一下从座椅上醒过来，想跳，却发现自己受着安全带的禁锢。

“这梦真可怕....如果真的是这样我肯定就玩完了”一番自言自语过后，我扫试了一下周围，在发现大家都还在睡着而没有被我吵醒后我总算松了一口气。望向窗外，南方的天空中月亮安静地挂着，摩羯座的几颗疏星与其相映成趣。隐隐约约的，可以借着月光看到几口油井分散在或近或远的戈壁滩上。

“克拉玛依....还有漫漫长路啊....”想到这里，我忍不住苦笑。从喀纳斯到乌鲁木齐大概要做8个小时的汽车，按照位置来看，我们已经走过了一半。拿出手机，打开相册，静静的观赏着自己的作品，不，也许说大自然的鬼斧神工更合适吧。翻着翻着，突然看了到了一张极为突兀的照片，这才回想起来，此次旅行是多么的令人触目惊心：几乎每一个靠近水边的草地，总会有那么几个团体，不顾员工的多次劝阻，执着的向各种地方丢弃自己的垃圾。几乎每一个直角的小路的拐弯，都会有那么几个人，深谙“两点之间线段最短”的道理，将繁茂的草地踩出一条光秃秃的道路，也许就为省个一两米。除此之外折木摘花这一类主动破坏自然的环境的行为就更是不胜枚举。想到这里，不禁叹了一口气：景色很美，可惜有了人。

正在我自说自话的时候，车厢里突然亮如白昼，紧接着便是一个急刹车，然后震了一下，车腾空了一段时间，翻滚了几圈后重重的摔落在地上。

“我得出去”这是我在认清我还没有死后的第一个想法。掏出随身携带的小刀，割断了安全带后我爬出了已经着火的客车。首先让我清醒的是这刺骨的凉风，然后....我看到了路中间竖躺着的一具尸体，死相极为凄惨：下半身几乎变成了一滩肉泥而上半身却没有什么伤，好似用掉了一半的牙膏。定睛一看，是司机！我虽然经常玩一些血腥的游戏，像这样的画面见到的也不少，可是当我真正遇到这样的场景时，两条腿瞬间像两团泥一样瘫软下去。过了一会，好不容易振作起来，我突然发现手机还在自己的裤兜里，掏出来一看...忘插卡了。车上的火燃的越来越旺了，急中生智，我猛然想起来刚刚路过油田，而油田是一定有人的！想到这些，我极速向来的时候路过的油田跑去，跑的时候还特地绕开司机的尸体一大圈。
