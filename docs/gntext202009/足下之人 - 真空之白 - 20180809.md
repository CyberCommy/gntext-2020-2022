# 1

芙兰朵露，你这种恶趣味我还是第一次见咪。

    在鲜红的地毯铺盖的地上蹲着，懒散托住下巴抬眸看坐椅子上摇摆双腿看着心情不错的金发吸血鬼。另手夹持烟枪悠悠上下摇晃落撒许些烟灰。

    大抵是耐不了僵持不下与其冷面贴热面的气氛，苦思冥想许久，闷闷的运吐出缥缈紫烟，才毫不在意的在地板上磕磕烟枪抖灰绑回腰间收好，顺带伸脚摁灰在地上搓了搓。

    回身保持蹲着的姿势半思考吊着眼睛瞅悬在椅子垫板下面那双没有穿着鞋袜、白皙娇小的脚丫。

    说实在的我并不觉得你的身体不干净，不过真的要做起来我还是觉得，别扭。

    做了个叹气的样子，也许害羞了的迅速缩下脖子，为了保模样衡控制右膝盖贴于地面，探右手小心握住吸血鬼脚后跟。

    很凉。

    指尖在第一时间准确无误的传递了这样的信息，从而加剧了脸庞沁出的点滴冷汗。咽下突然大量分泌的唾液，默默将头埋下去，任凭生得杂乱的黑发掩饰稍微发烧的脸。

    轻启唇齿吻上恶魔的脚背，冰凉冷流直冲脑神经，惹得喉咙嗓子眼发紧，干燥渴水。此番，便为依附苍白的冰凉提供理由。

    开始升温的不中用身体。
    与逐渐被吞噬所谓理智。

   即便现在成为这双纤细玉足的奴隶也好。脑袋里出现了这样不理智的想法，并逐渐侵占整个思海。
与之而来的是严重的眩晕感、以及瞳孔中映出扭曲的风景。原本一手可握那样纤细的玉足而今变化成为了比同自己身躯大小的怪兽。
..不、那样称呼未免对这样精致的造物显得过于失礼。
“如同美神维纳斯般的圣物”。
如同虔诚的教徒似得把身体贴近在地面—之前她曾放置玉足的地面之上。贪婪的将其中甜美的香气吸入身体里。
或许是这样滑稽的动作被她瞧见了，从头顶传来了奖赏似得银铃般的笑声。
然后、结局骤然降临了。
自己的意识突兀陷入进了无边的黑暗，死亡来临了。
头顶的王不知是否是感觉到了无趣，随意的将死亡赐予了足下那个小小的虫蚁。
而现在、她足趾上那灵性的鲜红—就是自己存在所有的意义了。
或许在不久之后那抹血色就会被地摊抹下，但至少在这一刻—她紧紧拥住了她所爱、所崇尚、所敬之物。
这也是一种幸福，不是吗？

“人类..真是脆弱呢。说起来那家伙的名字是什么来着？好像是..琉璃吧。”
