# 1

“虽然我们这一辈子也只能在女神大人的身上生活，其实女神大人之外还是有其他存在的，和女神大人是同类的存在。”时钟敲响三点，女生的声音准时响起，似乎不在意到底有多少人入场。
        当然，就算她不在意，整个讲堂也是坐满了。
        “比如那位能力在女神大人之上的存在，就是另一名女神了。不过，这样的存在数量很少，历史上有记载的只有十七位，但是目前还存在的只剩三围，分别是女神大人，那位以及另一个很特殊的女神，史称媛媛。”
        “请问，其他的女神呢？”一个一边做笔记一边认真听的男生打断了故事。
        “这位同学。”女生抬起手，男生手中的笔记本飞到了她的手里。“吟游诗人的故事过程中严禁以除了大脑记忆之外的方式记录。正如同那位的力量凌驾于女神大人之上一样，女神的力量总有区分，而其他那十四位女神，其中有十二位力量在女神大人之下，就被女神大人驯服而成为了女神大人身上的女王。”
        “至于另外两位，虽然力量超过女神大人，可是和那位比起来实在是云泥之别，在和女神大人的战斗中被那位随意剥夺了力量，不过结局更惨，因为反抗的缘故，直接被变成了和人类没两样的东西，很快就灰飞烟灭了。”
        “那么，那位媛媛呢？”另一个戴蓝框眼镜女学生问。
        “不清楚，因为那位阻止了女神大人去和媛媛战斗，因此没人知道结果。”女生推了推眼镜。“当然了，这不是今天的主题，最重要的是，降临。”
        “女神大人的绝大多数时间都花在和那位以及其他女神接触上，但是极少数的时候，女神大人会以特殊的形态降临。这时的女神大人可能和女王一样大，也可能干脆只和人类一样大。”
        全场哗然。
        “当然了，大家可能有疑问，为什么你们从未听说过这种事呢？因为女神大人每次降临都是非常隐秘的，甚至，她可能现在就混在我们之间，比如刚才提问的那位同学。”
        “虽然是以相对较小的体型降临，可是力量却一成未变，把巨大到恐怖的身躯里的能量压缩到很小的身体里的时候，就会发生可怕的实情。曾经有女王在降临之时试图伤害女神大人，结果女神大人的一个眼神就让她消失了。”
        时钟敲响了三点一刻，女生站起来离开了讲堂。

        我慢悠悠地从前门走到后台，“辛苦了。”
        女生摇摇头，“哪里有。还不是女神大人给面子，没有在我讲故事的时候把这个地方毁灭。”
        我摘下蓝框眼镜，“我才没做过那种事。”
        “是吗？上次在脚上的时候，还有在牙缝中……”
        “啊，诗人，最近天气不错，是吧。跟我出去逛逛？”
        女生笑着，并未揭穿我转移话题的事实，“怎么会突然有这种想法？”
        脸色暗了一下，“没事，被大招打到有点痛。”
        “那就去吧。”

        某个城市首当其冲成为了我发泄的对象。我很少会对人类发脾气，因为我觉得那是一种很丢面子的事情，但是凡事总有例外。
        以九千公里的身高行走是一种奇妙的感受，既可以有效地毁灭一切，却又不会因为太大而完全失去知觉。
        用脚趾残忍地点在一座大都市上，一点感觉也没有，又杀掉了五百万人吧？又或者是一千万呢？无所谓，平时只要流一滴汗，死亡人数又是这样的多少倍呢？
        蹲下来，脸部贴近地面，然后又发现了几座城市，所有人都瑟瑟发抖地跪在地上，他们知道我是谁，那种气息是女王那种蝼蚁不可能有的啊。带着一丝恶趣味，我把鼻头凑近城市，用力抽了一下，再看的时候，城市已经不见了。
        被吸入女生鼻孔的感觉如何呢？
        没有军队反抗的破坏总是无聊的啊。我坐了下来，把十几座城市压在身下，反正他们也不可能活下去的。伸直双腿，然后，一些人很快凑了上来。被称为女王的生物跪着爬向我，然后把自己缩小到几十公里高，认真舔着我的脚，不让一丝城市的残骸留在那里。
        好无趣。我厌恶地皱了一下眉头，然后用脚趾夹起一个女王，轻轻并拢，一丝嫣红从脚趾缝中流下，两个女王赶紧凑上去舔舐起来。
        恶心。舔着自己同类的血液。
        但是，她们却又不敢不舔，否则，接下来变成一抹红色的可能就是她们。
        权力啊，真是让人上瘾的东西呢。
        看着人类眼中高贵的女王却在我脚下瑟瑟发抖，我感到了一丝满足，不过很快就逝去了。欺负蚂蚁，能带来多少快感呢。

        回到身体内，我看着脚背上的一块区域，“这里的人类看到了我降临呢，看来是不能留了啊……”我歪着头想着处理他们的方法。
        “女神大人……”旁边，一个女仆走了过来。
        “就是你了。”我点点头，“舔干净，要是有一个人类幸存下来，你就不要干了。”
        “是……”女仆伸出小小的舌头，尽心尽力地舔舐着。

        “怎么我就这么倒霉……”被厚度可以用光年形容的唾液覆盖的诗人不甘心地说着，“又要花几百年才能离开了……”
