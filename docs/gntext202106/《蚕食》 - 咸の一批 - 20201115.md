# 《蚕食》

作者：咸の一批

TID：29816

 

# 1

“帕克！你真是瘋了！要是你踏出城裡半步，你會被那些巨人給吃掉的。”

“我家裡已經沒有糧食了，現在唯一的辦法就是去墻外找到補給品帶回來賣掉，韋德！不要妨礙我。”

帕克甩開勸他的好友韋德，自顧自地從城裡的密道裡離開了城裡。

十二天前帕克就已經依靠著韋德的救濟才活到現在，可是現在韋德也快沒有餘糧了，而那些該死的地主卻佔據著土地不允許耕作。

“他們會害死自己的！”

帕克怒罵一句，從長長的暗道盡頭走出。

五十年前，泰拉星上開始出現了巨人，二十年前，雌性巨人消滅了雄性巨人贏得了統治權，從那一天開始，她們就蠶食著人類的土地，直到把人類分散成了數十個小城市，人口也從八十億銳減到了六百萬不到。

帕克住著的默克爾城裡就有一萬人——這座城市坐落在茂密的叢林里，即使是視力最好的巨人要發現這個默克爾城也沒那麼簡單，所以才一直存活到現在。

可是自從地主佔據土地後，這裡的日子一天比一天差，面對擁有武器的人，人們完全無法抵抗。

不過話雖如此，適應了城外強烈的光線之後，帕克對自己的目標猶豫了起來。如今，巨人掌控的世界裡，植物和建築物都成為了為了適應這些巨人的大小，在帕克面前的，就是一顆兩千米高的三葉草。

也就是說即使是最小的巨人也有十萬倍，不過現在不能叫巨人了，如果帕克知道外面只有雌性巨人，那麼就會直接簡稱女巨人。

“哼，不管怎麼說，終於從那個地獄一樣……”

就在帕克自言自語的時候，突然天空暗了下來，隨後……

“轟隆隆隆隆！！！！！！！”

什麼東西落到了帕克身後的地面上，差點把帕克的耳膜和五臟六腑都震破掉。

“！”

帕克回頭望過去，只能看到直入雲霄的一望無際的一根雪白的柱子。

“是，那些巨人嗎？”

沒等帕克細細地觀察，緊跟著兩次震動，那跟柱子便消失了。

“默克爾城！……韋德……怎麼會這樣……”

帕克一屁股坐到了地上，當那隻高跟鞋的鞋跟離開後，帕克眼前的景象讓他的恐懼一次性爆發了出來——一望無際地範圍裡只有一個數百千米深的巨坑，包括默克爾城在內，包括默克爾城周圍的大片森林在內，被夷為了平地，光潔而整齊，因為那樣巨大的動能產生的能量，瞬間將那些物質變成了分子態，即使是厚厚的土壤層也差點沒承受住。

後記

帕克後來被一位心地善良的女巨人捉到，養在家裡，過上了衣食無憂的生活，在這期間，他知曉了很多關於這些可怕的存在的信息，成為了人類當中第一位也是古往今來唯一一位對女巨人有著研究的人。
