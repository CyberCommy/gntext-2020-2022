# 1

本来想找几个九月没推荐的理由，不过想想理由这种东西说出来也是虚的，毕竟还是没更新把期待的读者给鸽了，只能表示惭愧。

九月共有新文章28篇，十月共有新文19篇。不得不承认的是文章区的活力近期有些衰退，但即便是这样仍有许多从未发表过gts文的作者大佬写文发布。这让我相信一时的黑暗遮蔽不了gts的魅力。

和我的无能形成对比的是eventually大佬在九月份依旧做了推荐，再此除了感谢也不知道还能说些什么了。

　　「我想问小师傅，如来还在世否？」

——叶凡

七月的时候我们提到了（虽然并没有人记得）将巨大系写出缩小系感觉的作品，那么本篇则是反其道而行之，在缩小系的作品里写出了一个庞大的世界。魔女与凡人的冲突，纯血与混血异同，泉水与神的秘密，失落的巨大化秘术，在重重的铺垫之后一并爆发出来的话一定会让人爽快无比吧。不过本文为了照顾福利剧情还是稍显急促了一点，要是能够再打磨一下剧情设定，让主角来牵引线索的发展，就不失为一篇佳作了。

以前我经常用「轻小说」的标准来评估未能出版的作品——其实现在也是——是因为当时的轻小说还远没有像现在这样web化，商业化，媚宅化，所写出来的东西姑且还是有经过编辑们的严格审核的，至少行文上相对而言是在保持了轻松易懂的同时还要文笔流畅许多的，虽然咱们已经在无形中把标准一降再降了。那么这次推荐的作品是琳姐近来发布的几部作品中较为行云流水的一作，如果能够把握住这个写作的基调再充实剧情的话，就非常有看头了。此外，本作也是近来，甚至是今年度，最为接近「轻小说」的一作——这里并不是我夸口，而是因为越是接近事物的本质，就越容易迷失事件的真相。
