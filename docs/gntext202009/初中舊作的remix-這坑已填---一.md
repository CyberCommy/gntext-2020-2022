# 1

這是年少無知時寫作的yy文，現在經過重新透露方式重現眼前。

女郎資料

年紀:19-22歲

身高:160cm

在一個城市中，有一個女子要進行報復。她要消滅這個城市。

女子是一個著名科學研究所的研究人員，她最近成功研究改變物體大小的藥物。

於是她將藥物偷了，不過當她執行偷竊時，不巧被保安人員見到，於是保安跟她說:

-妳走不了!

女子回答:

-你來見證我的偉大成果吧!

說完，將藥物飲了一半，於是四周在震動，保安人員見著女子的所有一切，包括身體和衣服，都在擴張。不經不覺，女郎由原來的1米6，變成了16米。驚慌的保安人員拿著手鎗向眼前的女巨人發射，子彈射到女郎的腳眼，子彈反彈了，女郎絲毫無損。

這時女郎明白到，這個高度的大小不足以消除這班可恨的保安，更不要說城市了。於是她做了一個決定。

正如之前所說，藥物將女郎的一切都變大了10倍，所以連未飲完的藥物也變大了10倍。女郎想，假如她飲用完餘下的藥物，她會變成萬以十計巨大的女郎，到時這個城市的一切都是她眼中的灰塵。

想到這裡，她愉快地笑了。沒有意會腳上的痛楚，她一口氣飲完餘下的藥物。

飲完，她靜候著自己變得巨大。

地上的保安人員，這時嚇得動彈不能。

女郎變得更巨大，現在比研究所還要高大。不過大家不知道，女郎有些不滿意。

因為她現在只有160米。

-本來想有萬以千計的巨大，現在卻只有160米，實在不合預期。不過，160米也足夠破壞吧!

於是女郎也不花太多時間，就直接用腳上長靴將保安人員踏死，然後用拳頭敲破研究大樓，很多內部的人走避不及，被她的拳頭和瓦礫壓死。

-哈，不堪一擊。

女郎看看自己身上衣著，恤衫短裙以及高跟長靴，最好給這個城市報復。

然後她慢慢地拖著腳步向高樓密集的地區走。

xxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxx

這個時候的城市，車水馬龍，人來人往，沒有人知道將來發生的事，只知道現來很塞車，於是他們發瘋似地按喇叭，整條街道頓時嘈雜得聽不到其他聲音。

除了一種很低很沈重的震撼。

沒錯，就是高160米的那位女郎，她本來沒有決定破壞那裡，只是漫無目的的向高樓大廈方向走，於是就走著走著，

聽到了嘈吵的汽車鳴笛聲。這是女郎這麼痛恨城市的原因，污染的空氣、嘈雜的聲音污染，這樣的城市會令容貌娟好的年輕女郎變得年老醜陋。

沒有任何顧慮，女郎就從車龍的後方直接向前走。

排著隊的車龍，就這樣被插隊的女郎腳上長靴的尖鞋跟刺穿、壓垮。

被惡意踐踏的車輛不但冒出濃煙，甚至爆出火花。至於那些司機，很多的還未知道發生甚麼事，就被壓扁。更多的司機是知道了巨大女郎的惡意踐踏後，正要逃生，卻因為交通變得更加擠塞而走投無路，被長靴活活踏死。

女郎就只是走著，就將地上的人們趕盡殺絕，女郎會輕鬆地吹著口哨，然後隨心所欲地去拳打四周的大廈，於是想逃避馬路的人們，又發現人行道更危險，終於也難逃女郎的靴底。一雙復仇的黑色鐵鎚敲擊著大地，所引起的震盪不單震碎人類建築的玻璃，也震懾了人類的心神。

xxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxx

一路上沿著車道行走，很容易到達高樓密集的地區。踐踏車龍的玩意變得沈悶，女郎感覺不到復仇的快感。之前的地方，不是沒有高樓大廈，只是密度不陪高，也太矮，沒有趣味。

幸好，真正的高樓密集地區終於出現眼前。。。。。

(lz說:看了『全球60億人口 來舔我的鞋』後，覺得這句很搞鬼..................)

女郎可以用雙腳感覺大廈材料的質感，於是她不理此路不通這個問題，就自己一個走入了大廈群中。當然人們建設大廈群時可沒有計劃到有足夠空間給一個160米的巨大女郎走動，更何況讓她擠身到樓群之間。所以，大廈可以破碎的東西都隨同女郎輕巧身形穿越時撞得破碎；沒有被撞碎的，例如一些還會走會跳的上班一族們，則因為大廈本身超過70%都是容易破碎的物質，而遇上了地震才會發生的情節:大廈倒塌。對於未經歷地震的人，現在所有都是夢境，巨大的女郎，倒塌的大廈，都只有電影的CG才可做出的效果，在現場活生生出現。高處墮落的人變成肉醬，有些肉醬被倒塌瓦礫掩埋，地面除了一道一道裂縫外，就是灰白的以及血紅的點滴。

當然這些只是她用雙腳走過大廈之間所發生的事，也就是她甚麼也沒有搞，只是在走路，就垮了幾座樓，甚至不用計算周圍因地震而搖擺不定的建築物與老人們。聽起來女郎好像走直線，其實她計劃來回的走，達到整個大廈群都佈滿她的鞋印。於是，女郎不一會走了個循環8字路出來，而在8字兩個圈中未受直接破壞的建設，得到一小段的和平安穩；走出一個明顯的8字在地上，女郎由8 字的底部，直接走上了頂端，周而復始，直到地上不再是8字，而是一根竹籤穿插了兩顆魚蛋的圖案。

這使還未死的人，以及女郎，都回憶起孩童的玩意，小孩玩泥沙。在泥沙中畫畫，是兒童消磨時間的好方法。女郎的復仇也採用了這種形式。女郎的下一步，相信不用說，大家也估計到。

『這個世界死不盡的過剩人口，來舔我的靴!』穿靴子玩泥沙，不用說，不單鞋底，鞋面甚至靴筒都是污垢。女郎也慢慢蹲下，再由蹲轉為臀部坐地上、雙腿往前伸的姿勢。

女郎相信，總有一大群人未死，世界的人口過剩這麼嚴重，總不會她兩三下就解決了。她有信心，於是在見到一堆疑似人類走近靴子時，她開心，自己的長靴可真由鞋底至靴筒都焕然一新。

『拿你們的身體來擦。』
