                                                                                                  剥皮新郎
   “咿呀~！！”
   深埋与山谷中茂林里的一个小村落，惊恐嚎叫声轻松贯穿这个只有不到百户的弹丸小村，霎时间鸡飞狗跳，从寂静到喧嚣，除了行动不便的老人小孩，百余衣衫不整的轻壮男女冲出家门，手持钉耙，擀面杖，柴刀等农活物事向传来惨叫的村东口汇聚。
   村东有颗老槐树，堪堪一人合抱，时至深秋，老槐叶疏枝枯，顶上盘旋着十余只被惊起却不愿离去的乌鸦，哇哇叫着，残月老槐乌鸦，鬼气森然。
   普通人在这样的深夜里是断然不会来村东口，更别说走到老槐树下，不过个把月前村子里接连失踪了几个少女，又有人朦胧睡眼中看到屋顶有怪异黑影飞掠。小村平静几十年，徒然接连发生如此诡异的事件，由于地处偏远，报官也无人理会，于是在村长乡绅的组织下，家家枕戈而卧，又有十余耳聪目明胆大心细的青年轮流巡夜，不管是什么山精野兽，势要尽快打杀，先不说村里女人本来就少，一众光棍更加眼巴巴望着未来媳妇失踪了好几个。
村落虽然落后而迷信，敬畏鬼神，此事也似阴鬼作祟，但事关村落种姓延续与自己今后几十年的下半身幸福，那可是玩命也要上啊。
   今天夜里轮到顺子和二蛋值夜，两人属于人傻胆大的豪迈型号，加上出门前浊酒入肚，两人凑成一队胡吹牛皮，渐渐开始耻笑对方手中所拿柴刀钉耙等神器太过于大巧不工，接着再赞自己手可拎日月，最后热血烧脑，互操了同一祖宗后变换队形，成单兵战术反向突进。
   于是一个去了村西，一个来到村东口。
   二蛋缩着脖子，腋下夹着耕地的钉耙，腰里别着生锈柴刀，一边搓手呵气抵御深秋的夜寒，嘴里一边骂骂咧咧:“狗日的王小顺，敢操老子祖宗，我祖宗不是你狗日的祖宗？要不是为了俺那窝心肝的小翠花睡个安稳觉，老子非打的你爬回家不可。”
   从铁瘸子家闺女无故失踪，今天已经是第四日，二蛋已经连续两个晚上没有捂热炕头，从最初的心虚胆寒到现在几日不见动静，逐渐没有了警惕，其实巡夜的十数人都是这样，年轻心燥，白日里耕田劳作，夜里还要漫无目的的满村乱窜，村子本就不大，除了狗吠梦吟，就剩下风吹叶动的声音，于是侥幸之下都想着就算有山鬼作恶，也已经吃饱离去，村子看样子将再度平静安详下去。
    二蛋低着头，从腰里把嗝的难受的柴刀抽出拿在手里，走到老槐树下，慢慢贴树坐倒。老话说槐树属阴，容易栖息鬼怪，平常二蛋断然是不敢夜里来这遛弯，今夜气血沸腾，只感觉俯瞰天下无所惧，一颗破树来回看了十几年，怕个锤子，便要靠树打个小盹，顺顺鸟气。
   左手将柴刀插在泥里，将钉耙横放膝间，下巴整着双臂，正要合眼，突然感觉到屁股下湿漉漉的，伸手摸到一把湿泥，残月下看不清楚，二蛋以为哪个野狗在这撒过尿，正待要骂，哔哔剥剥听到几声轻响，有几滴液体打在二蛋头脸上。
    原来是落雨了，二蛋用衣袖在头上擦拭，闻到一股淡淡腥味，借着月光一看，灰黑肮脏的衣袖上再添一抹艳红。
   那明显不是雨水，是血色。
   二蛋痴痴抬头，一双惨白的脚丫正悬在自己头顶上方五尺处，血水顺着脚尖，正缓慢往下滴。
    一滴血水恰好滴进二蛋半开的嘴里，二蛋不自觉吧嗒了一下嘴，有点咸。
    猛然回过神的二蛋连滚带爬的向前翻，一边回头看。
    一个人正被吊在老槐树的一条斜错出来的粗壮枝丫上，肚腹间黑红淋漓，内脏外翻，更有一截肠子打了几个转，缠在那人脖子上，一头缠住老槐枝丫，将将吊住，还在微风里轻摇弋。
     老村愚民，见过最血腥的事情就是年节杀猪屠羊，如今大半夜的，一个人满身下水外流，被自己肠子像风干鸡一样吊在树枝上，可想而知脸上表情也是相当的抽象精彩，双眼外凸，舌头垂在下巴外，依稀还能辩出脸貌，是村子里最招人厌的大混混狗剩。
     这人平日游手好闲，家里农田荒废，四处偷鸡摸狗，今天爬墙偷窥寡妇洗澡，明日便顺一条小娘子肚兜，好些黄花闺女都被他言语猥亵调戏过。
     二蛋拿着柴刀的手抖个不停，胯下已经湿到裤管，跟狗剩对峙了好一会，直到尿液流满两条腿，打湿单薄布裤，被夜风一吹，猛然打了个寒颤。
    于是扯开嗓门，大吼一声。

    周禄俞推开娘子缠在脖子上的白嫩胳膊，喜妹不依的再度搂住他，嘟着嘴嗔道：“相公，别去，大半夜的，我害怕。”
    周禄俞不耐烦的甩开喜妹，一边穿衣服一边下床，微怒道：“现在村子里什么情形你又不是不知，刚才那声叫喊满含惊惶，肯定是有人看到了什么，你听听外面，全村人都东起来了，你若害怕，就在家里睡觉，别要出去了着凉，合着我还得伺候你。”
    喜娘把被子一掀蒙在自己头上，骂道：“赶紧滚，跟你好声好气，还不是怕你出事，你一个连宰鸡都不会的酸秀才，出去凑什么热闹，就知道对我发火。”
    周禄俞顺手在灶头拿了一把菜刀，推门直接看到老丈人脸色铁青的站在院子里，不由一怔，道：“爹，您这是？”
    王祥山瞪他一眼，转身回房，猛然摔上门，骂声从屋里传出：“凑个狗屁热闹，烂泥扶不上墙。”
    周禄俞脸色黯然，深吸一口气压下怒火，往外走去。
    村子里唯一的平整主道上三三两两的人在向着村东口惨叫处奔跑，看到周禄俞都脸现鄙夷，他也不理会，跟着人群疾走。
    此时老槐树下已经围满了人，二蛋正被他爹娘搀在怀里，几个年长者问东问西，其他人都远远围成个大圈，对着吊在树上的狗剩指指点点。狗剩父母早死，在村子里也是人神共愤，围观者都一味吸着冷气表示惊讶和害怕，并无伤心表现。不知道谁喊了一句“村长来啦！”人群分开一条缝，一个胡子灰白，戴着一顶瓜皮帽的七旬老人在几个乡绅搀扶下走了进来，脸色铁青的看着随风晃动的狗剩，一挥拐杖，说道：“像什么话，赶紧先把狗剩放下来！”人群你看我我看你，却都没有动作，一来是惧怕死者惨状，不敢靠近，二来是厌恶狗剩生前作为，不想替他收尸，村子那么小，谁没在这无赖跟前吃过点小亏。
    村长气得胡子乱颤，喝骂中终于有几个手脚灵活的青年爬上老槐，远远够着用钉耙勾了几下，狗剩肠子受力已久，啪一声断了，再摔在地上，心肝脾肺肾混着半凝固的腥臭血水溅了一地。
    周禄俞站在人群外围，被慌乱散开的人群挤得差点摔在地上，直起身时看到一具惨不忍睹的尸体，再仔细一看，却是终日恨不能生食其肉的狗剩，惊吓中反而有七成快意，他退出人群，站在一旁，知道自己这个外姓人自从娶了喜娘后就越发不为众人待见，也不去跟人问话，静看事情发展。
    村长让人把狗剩散落一地的下水和尸体扫成一堆，找了两件破衣裳草草盖住，问了下缘由，便拄着拐来到二蛋跟前，先好言安抚一阵，看着二蛋脸上惊惶在众人劝慰下逐渐消退，闻到一阵尿骚，不动声色的看了看二蛋下身，和颜问道：“二蛋哪，这狗剩的尸体，是你发现的？”
    二蛋点点头，摸了把脸，看着众人都眼瞧着自己，挣脱娘亲双手，从地上爬起来，自豪道：“那个回村长啊，是俺第一个发现的。”
   “可看见是谁……或是什么东西做的恶？”
   “那可没瞧见，我跟顺子吵了一架，自己走来这里时，狗剩他已经吊在树上了。”
   村长摇摇头，身边有一乡绅说道：“二蛋也没瞧见，让我看哪，这明显不是山里来了猛兽，咱王家村向来平和，也从不见鬼神传说，只怕……这是人为哪！”说完，瞄了一眼独独站在人群外的周禄俞。
    众人立刻安静下来，看向周禄俞。村长一挥手，高声道：“事情越发古怪，不要乱说，此事和前些日子发生的怪事有些不同，你们先给狗剩收尸，大家完了都回吧，管事的来我家议一议。巡夜的不要停了，要三人一伍，莫要分散。"
    周禄俞在众人满含警惕的目光中，只能先人一步走回家，放好手里菜刀，喜娘穿着肚兜，抱着被子缩在床角，看见周禄俞回来，本来有些紧张的脸立刻冷了下来，说道：“又是哪家女子不见了？”
    周禄俞脱了外衣鞋袜，刚才看到喜娘表情，心里也有些软了，伸手抱住她，喜娘顺势躺进他怀里，感觉到他身上冰冷，忙拉过被子将两人裹住，反抱住周禄俞。
   “这次不是有人失踪，是闹了人命。”
   喜娘呀的惊呼一声，夜色里格外大声，她赶忙捂着嘴，小声道：“出了人命？是谁死了？”
   “狗剩，死的很惨，被人开膛破肚，用自己肠子掉在村东那颗老槐树上。”
   喜娘被吓的身子一跳，再紧紧抱住周禄俞，抖着嗓子问道：“这……这……是谁做的恶？”
   “二蛋第一个发现狗剩尸体的，已经吊在树上，不曾见谁行凶，听他们言语，也没谁发现别的怪事。”
   两人也说不出个所以然，讲了会话，喜娘困意来袭，渐渐睡了。
   周禄俞看着喜娘梦里依然微微皱着的眉头，用手轻轻抚摸，想着狗剩的惨死，感觉心里一直堵着的石头终于不见，半年多来郁结在心里的一股气，慢慢顺了下来。

# 2

周禄俞曾是镇上小有才气的一个秀才，五年前兵祸牵连，家产被抄，自己跟老母四处流浪，到了王家村。山野孤村人心淳朴，见着周禄俞满身风尘，背着一个年迈母亲，好心收留他们住下，没想到周母没过两天安稳日子，便感染风寒撒手归西，周禄俞守了岁，在村民挽留下便在王家村扎了根，在村民帮衬下起了泥草房子，平日里给孩子们教授文字，帮王家村整整族谱，写写对联，慢慢安稳下来。
   再过了一年，跟同样父母早死的王秀秀日久生情，周禄俞去往村长处提亲。王秀秀吃百家饭长大，长成了村子里最水灵的姑娘，早在周禄俞之前便有许多年轻小伙死死盯着。村长虽然对周禄俞这个村子里难得的会识文断字的书生颇喜，但毕竟是外姓人，对周禄俞的提亲不置可否，便给按了下来。
两人情渐浓时，待春花烂漫把持不住，自然而然的做了违礼之事。
村子就那么大，王秀秀寄住在村里一个老寡妇家，周禄俞的破茅草房又树在村子中央，破门破窗档不住光也遮不了人。那年春末夜里，两人偷偷摸到了村东那处入夜便阴森森的老槐附近，你侬我侬，水乳交融。
其实这本与旁人无关，村长甚至存了这个意思，等他们饭做熟了，就在后边推一把，将秀秀许给周禄俞，虽然免不了肥水流进外人田，但能给村子留下个读书种子，替老村愚民开开拙智，想来还是划算。
可惜两人太心急。
要是等到三更后，狗剩可能也回家睡了。偏巧那天这歪货偷了只鸡，也想跑来老槐树下烤来吃，一眼撞见了在老槐旁草丛里半裸着的两人。
狗剩是什么人，看到王秀秀白晃晃的胸腩，红了眼，炸了肺，抓起手里那把准备剥鸡的菜刀就往周禄俞头上招呼，好在脚下被树根绊了个狗吃屎，才让两人衣衫不整，小脸煞白的逃回村里。
当时狗剩在后面追，一面怒吼。偷情的一对男女在前边逃，王秀秀在尖叫。
于是王家村沸腾了。
对于王秀秀来说，王家村老一辈人，都是她的长辈，这挺好。但一个姑娘没出闺阁就和男子厮混到那个地步，还被当场曝光，闹的人尽皆知，那就会很严重。
王家村里的人愤怒着，厌恶着，心底里扭曲着不为人知的念想，唾尽恶语在秀秀身上。那些精力过剩的小伙子们，更如狗剩一般，嫉妒而悲愤。淳朴与善良在此刻泯灭，他们叫嚷着起哄，开始要赶走两个人，又发觉这样岂不成全了她们，于是渐渐变成了要用石头，用木棍砸死两个人。
大家心里明白一些事情，老一辈在旁无声而阴沉，姑娘们眼里悯然而厌恶。看着被人群包围而依然颤抖着抱在一起的小情人，狗剩丢出了手里那块拳头大的石头。
石头落在额头。
周禄俞满脸是血的醒来时，依旧躺在村子里那唯一平整的硬泥路上。
人群不见了。
秀秀也不见了。
周禄俞满脸是血，半身是血，飞奔在小小的王家村里。他跑掉了秀秀亲手纳的黑布鞋子，跑红了裹脚的白布袜子。
王家村里家家闭户，人人闭嘴。
老寡妇家里只有老寡妇。
狗剩用颤抖的柴刀指着手不能提，肩不能担的周秀才，被追的哇哇乱吠。
然而秀秀在哪里呢。
那天的天特别阴沉，黑的特别早。
周禄俞满身的血和泥，满眼的血红和黑暗，一半爬，一半走的来到了老槐下。
他看到了秀秀。
秀秀在老槐上。

# 3

他曾说执子之手。
秀秀生在小村，长在小村，去过最远的地方是山外西边的小镇。小镇很小，秀秀吃过最好吃的东西是八岁那年村长爷爷元宵时给买的糖葫芦。小镇很矮，陷在山外的群山里，一眼望去，全是山，全是树，秀秀看过最好看的物事，是六岁大年初三晚上看过的皮影戏。小镇很古旧，古的像前些年村外老槐下挖出的青铜小鼎表面那层绿绿的锈;小镇很旧，一如夕阳里那沧桑的鸦叫。秀秀在村里，在小镇，没有看过山外那山，纯的好像老槐初春时那枯枝上抽出的新芽，纯的很纯粹。
他鼓起勇气执起秀秀微微颤的手，那手软软的，手心里都是羞涩的汗水。

他曾说共白头。
他轻轻揽起她的头，轻轻放上自己的肩，他的头和她的头轻轻靠在一起，夕阳下老槐旁，那两个融合在一起的剪影是小村不曾有的风景。

他曾说同生。
她爱穿淡黄色的裙子，布质粗陋，穿在她身上，有风吹过时，微微荡漾的很好看。他写了好多对联，在穷的让人牙酸的小村里攒了点钱，去镇上买了最好的一条碎花围巾。他在腊月里雪地上亲手戴在她颈间，围巾很长，他替她包了好多圈，遮住了半张被风吹的有点红，却很好看的脸。他看着她笑成一对月牙儿的眼睛，怎么看也看不够。

他曾说共死。
她好不容易在他的指导下学会了写自己的名字，他说的好多话她还未能听懂。这一次他看着她的眼睛，她从他的眼里读懂了那浓厚而热切的含义。她在寒风里解下一半围巾，绕在他的脖子上，把两个人牢牢的绑在了一起。
要执子之手，要白头偕老，要同生，要共死。
