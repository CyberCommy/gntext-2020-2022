2012 12 23

教室里出奇的安静，几个早早来到学校的学生---只有几毫米身高，正坐在教室的地板上商讨着什么。

'玛雅预言成真了呢，人类确实迎来了灭亡...' 瘦高的男生说道。

'张凯，真不一定呢，即使所有人类都被缩小了，那也可以继续生活下去，只不过采集食物困难了点而已。'

'你确定？窗外的鸟声可是和以前一样的响呢...假如只有人类缩小了的话，其他动物在我们看来比怪兽还可怕。'

'也许只是个别动物没有被缩小...'

'别太抱希望啦。'瘦高的男生看着天，似乎已对这个话题不再感什么兴趣。

就在这时，巨大的推门声打破了宁静，一个熟悉的身影进入教室。

'叶岚！她还没被缩小，太好了！'之前反驳瘦高男生的人几乎高兴地跳起来。

'喂，等等...'其中一个女生似乎有些担心...但那几个男生直接冲向叶岚，边跑边大喊着。然而叶岚却没有任何反应，只是一直在把文具和书本从书包里拿出来。直到男生们喊的嗓子都快嘶哑的时候，叶岚才感觉好像听到了什么动静。

顺着声音来源，她渐渐将目光下移，似乎注意到了那些人。

'我在这！我们都在！叶岚？叶...'

她确实注意到了他们，但是，她的眼神却变得轻蔑。当小人们注意到这一点时，似乎已经晚了，叶岚轻轻抬起左脚，跑在前面的三个男生瞬间被阴影笼罩，根本来不及做出任何举动，巨大的帆布鞋便重重踏下。

所有的人都惊慌的往后逃，可叶岚似乎不想这么轻易地放过这些惹人生厌的虫子，再次抬起脚，重复刚刚的动作，可这次却只是悬在空中停下了，然后收回了脚。逃过一劫的学生们瞬间瘫软了下来，趴在地上喘着粗气。

'奇怪，怎么是红色的血...'

叶岚蹲了下来，仔细看着这些瘫软在地上的人们，轻轻地，将手伸向他们。

一根巨大的粉红色柱子从天而降，萧雨刚想爬起来，可面对巨大的叶岚他的腿似乎使不了劲，只能坐在地上边大叫着边用手撑着身子慢慢后退。

'叶岚！是我呀！'这时之前有些担心的女生突然大声喊了出来，由于此时叶岚蹲下了，注意力也很集中，所以很快地分辨出了发出这个声音的人，当她看到站在桌脚瑟瑟发抖的，只有蚂蚁大小的欣玲也瞬间意识到了这些之前当作虫子的东西其实就是曾经朝夕相处的同学们...

叶岚瞪大眼睛看着欣玲，然后捂住嘴，向着那三个小红点看去，这才注意到被巨大的力量碾压后的尸体只有一部分留在了地面，曾经的同学现在却在不经意间被自己剥夺了生命。

'张凯！李清！王居明！你知道他们看见你的时候多高兴么！'另一个男生指着叶岚大声说道。

但叶岚只是低着头，什么都没有反驳。

'昨天我们还一起打过篮球！你竟然...'由于已经在一起上了两年的学，都很熟悉，平时也是经常互相开玩笑，所以现在才能这样随心所欲的指责着叶岚。

'...我不是故意的...'

'混蛋！'他跑向叶岚放在地上的手指，狠狠地踢着指甲，把失去朋友的怒火通通发泄在叶岚身上。'你个杀人犯！'

可是突然，叶岚觉得之前的杀戮没有那么严重了，她就这样微笑着，看着手底下的小人对着自己的手指进行着微不足道的攻击。欣玲似乎感觉一直都很准，她注意到了叶岚的表情后，便立刻想去阻止男生的这些拳打脚踢。但是已经晚了，叶岚轻轻动了一下他脚正踩着的手指，可即使是这样，那个男生几乎在空中翻了一圈才落地。

'你干什么！'男生愤怒的喊道，'反正你也不会疼！'

叶岚没有说话，只是慢慢的用手指靠近男生，不理会他的指责，轻轻弹一下，男生便飞出去好远，半天都没有爬起来。叶岚似乎还不满足，再次伸向男生，可这时欣玲及时赶到男生身边，用身体挡住他。

'再这样下去他会死掉的！'

'死了...又怎么样呢？'叶岚嘴角微微上翘，眼神也变得有些轻蔑。

'对...对不起，他之前说话有些激动，你也要理解...'听到叶岚那句话后，欣玲莫名感到非常紧张，这句话几乎是边颤抖着边说出来的。

但是叶岚看到这种表现后，不知道为什么，心里有种奇怪的感觉，莫名的有些...兴奋？
