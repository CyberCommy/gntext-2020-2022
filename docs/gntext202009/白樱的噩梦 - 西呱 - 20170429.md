# 1

我已经不是写手了，文都坑了，假借他人设定来写一篇吧
加群457681764，被我误踢的一百多人希望回来，我真不是故意的。。
村里，又有人犯事被抓起来了
老张接着看热闹，这台子挺干净
村里尘土飞扬的一般没什么干净地方
这个方台不大，且20米见方，如同天权石一样的洁白
这回，又是老张一邻居，他跪在方台上，
头顶一只巨大的脚伸了过来
这脚太洁白了
巨大的洁白的脚非常的细腻，老张却是非常的看呆
呆立着内心赞叹着美好，可是这是要踩死邻居了
老张漠视着，邻居被踩成稀烂
第二天，老张被人绑了要被处死
他可没偷核桃啊
说他偷了核桃，神明要踩死他
这时看着那白嫩的脚底，泛着微微的粉色
他不再有心情欣赏，抱着头，
一旁的人呵斥着他跪好，就这么跪着被踩成了稀烂
头顶，白樱拿着核桃正津津有味的吃着
刚要踩死这人，却感到了强烈的愤怒和憎恨
她醒了
为什么要这么做
偷核桃干什么，还做这么无聊的事情？
这是我吗？
她迷迷糊糊的醒了，这真是个荒唐的梦
鞋柜里，天权石闪耀着光芒，不为人知的闪耀着
