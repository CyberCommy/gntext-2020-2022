# 1

“巨人”这个词，你们是如何定义的呢？ 
       请原谅我使用了“你们”这样一个听起来稍有距离感的词语，但正如这个词所表述的含义一样，“我”的确和“你们”不太一样。事实上，如果按照你们对于“巨人”这个词的传统定义来看的话，我确实算得上是一个“巨人”。 
当然，或许对你们而言，我还是与这个词的传统意义“略”有不同。 
当我的身体和其他女孩子一样正常的时候，我知道你们习惯称呼那些身材高大的人为“巨人”，所以像篮球运动员啦，模特啦很容易就会得到这么一个听起来有些别扭的“雅号”。 
       但是对于我，或者说，对于现在这种状况的我来说，“巨人”这个用来描述一个人比起其他人而言仅仅是“高人一头”的词语，就略显乏力了。我的意思是说，如果你能够见到现在这个样子的我，你就会在自己的词汇库里完全更新自己对“巨人”一词的理解。到那时候，我相信全人类的文学作品中凡是形容人类身高的地方，全部只有两句话：“像我们这么高”，“像她那么高”，虽然第二句话我觉得几乎不会出现。 
       说到这里，我相信你们已经知道我有着异于常人的身高了。但是对于我的身高在你们心中到底有多高，这个问题对我而言始终是个疑问，因为你们的想象力总是受到所谓一般性常识地约束，我想绝大多数人也就仅仅停留在“这个姑娘可能真的很高吧”这种程度上了，不过这不怪你们，因为在我像你们这般大小的时候，对于事物的认知也停留在和你们同样的水准上。 
为了方便你们对于我的身体有一个更加直观的概念，我决定拿这些耗费了你们巨额财富建造而成的高楼大厦作对比，希望能有所帮助。 
        比如说，这栋大楼吧。在我还是个普通小姑娘的时候，我从书上得知它的高度是430米，这座城市的标志性建筑，全体纳税人血汗的结晶，政府和市民的骄傲。当我站在这栋你们所谓的摩天大楼前时，我衷心希望顶层的那些小家伙们不要抬头去看我的内裤，因为谢天谢地，我变成这样子的时候还穿着短裙，而且我的T恤没有撑破。 
        每当我在你们这座不大不小的城市里漫步的时候，我总是觉得这个建在市中心位置的建筑物跟周围的环境是那么的格格不入。怎么说呢，如果我还和你们一样，我是说一样大小的时候，每当我路过它，我总是幻想自己今后能够成为在这座建筑里工作的OL，也时常产生跟其他小女生一样“想跟自己心中的白马王子在楼顶的旋转餐厅共进烛光晚餐”的浪漫幻想，但如果你们有机会（虽然这种机会比较少）变得和我一样大的话，我相信你们也会产生和我一样的不愉快感，那就好像不知道是哪个家伙在你家客厅正中间摆放了一个晾衣架一样。 
虽然我觉得自己把这栋建筑挪走并不比搬掉一个晾衣架费事多少，但我总是在心中告诫自己，当一个像我这么漂亮的女孩子长到这么大的时候（无论是年龄还是身体），总该要学会尊重他人的劳动成果。
       但我怀疑这种象征性的自我约束究竟能持续多久，因为你知道，至少在这座城市里应该还不存在能够阻止我发一顿小姐脾气的人或者别的什么东西，至于你们相不相信，我反正是信。

# 2

       刚刚提到了在这座城市中漫步，我想你们应该能够理解，一个像我这么大的女孩子，一个正处在泡吧，和很多帅哥在KTV里唱歌，夜不归宿，每周外出开两次房年龄的女孩子，让我每天坐在海边面朝大海，或者坐在你们口中的高山或者我眼中的土堆上望着夕阳发呆的话，你不如发动你们能够调集到的所有军队，朝着我把你们火药库里的弹药全部打光，然后看看这样做能让我发多大的火气——虽然我觉得这些小动作基本不会起到你们预想的作用。 
所以，每天从城市的一边漫步到另一边，再从城市的一角漫步到另一角，就成了排遣我无聊生活的重要事情。 
       我非常希望你们能把城市建的大一些，因为现在看来你们这座不足2000平方公里的小城市比我以前看男孩子打球的地方实在大不了多少。好吧，也许算上周围的郊区还有城市边缘的园林的话，也许能让我的旅程变得稍微长那么一点。 
       在穿越城市的过程中，一开始我还能够告诫自己，小心你们那些造的过于低矮和不起眼的房子，但是它们实在太小了。也许对于你们来说，一栋10层高的建筑就可以令你们产生高度的眩晕感，但是要避开它们，对于现在的我而言，这无疑于比走路时不要被一本书或者一个易拉罐绊倒更加困难。 
       所以到后来，我对于踩碎一些你们眼中的高层建筑的事情，心中的内疚感已经小很多了，我甚至渐渐喜欢上了那种踩碎钢筋水泥时脚底的愉快触感，当然你们会用你们的方式对我表示抗议，但你们知道，我除了用一个美丽女孩子的美丽微笑外，实在给不了你们更多的回复。 
       对于一个整天无所事事的人来说，脑袋里偶然蹦出来的奇思妙想便成了她消遣度日的重要手段。比如有一天，我突发奇想地想知道自己的乳沟到底有多深，而你们辛辛苦苦修建的那些高楼大厦则成了我最好的，或者说身边唯一能使用的标尺了。虽然我现在变得比你们绝大多数人都要大一些，但是对于自己身材的自信并不来源于此，因为在我像你们这般大小的时候，我就已经对自己的身材相当有自信了。我想我的胸部应该有E或者FCUP，其实我并没有刻意地去量过自己乳房的尺寸，但通过和我上过床的男孩子的手，我确定它们的尺寸一定不小。 
“你的乳沟就像马里亚纳海沟一样深邃。” 
       我忽然间想起之前和我厮混过一段日子的某个男生说过的肉麻情话，这更加激发起了我进行这恶作剧般小实验的欲望，不过对你们而言，也许会是个不大不小的灾难。当然这一定不能全怪我，因为你们小小的房屋建筑的实在太密。我花了几分钟时间来寻找一个相对独立的建筑物，但是很快我的耐性就被你们建筑的密集性消磨光了，我希望在这个小小的测试结束时不会给你们造成太多的损失，但我仍坚持认为这更多地取决于你们建筑的质量。 
       我双膝跪地，然后俯下身去，拉低T恤的领口，尽可能地露出更多部分的乳房，然后用双臂给乳房一个向中间挤压的力道，以求能塑造出一个更加迷人的乳沟。当然，你们这座城市不可能提供一片足够大的空地来让我进行这个邪恶的小小测试，所以在我做出上述一系列动作的时候，你们的一片，或者说几片建筑区成为废墟就在所难免了。我吐了吐舌头，做出一个抱歉的表情，仅此而已了。

# 4

我虽然知道自己变成了一个巨人，但我没想到自己的乳房随之变大后，挤出来的乳沟能变的这样“宏伟壮观”。我现在用乳房抵住了一栋看起来二三十层高的建筑，我惊喜地发现自己的乳沟自地表处算起，比这栋楼房的屋顶还要小小超出一点，这无疑令我更加兴奋。试想，有哪个男孩子能抵抗一道30层写字楼一般高度的迷人乳沟呢？

        我不想如此草率地结束这次小小的试验，既然量完了乳沟，为什么不趁此机会好好量一量我的乳房呢？于是我脱掉那碍事的T恤，随手丢到一边。虽然这个举动在我看来是无比的自然，但是我忽然想到对于你们来说或许是毁灭性的打击。

      “可你们也看到了我的裸体，这样就扯平了吧。”

        假设你们也变成像我这般巨大的话，如果还像以前那样处处考虑细节，会让你觉得很头疼的。

        我解下了胸罩的搭扣，这次我记得把它轻轻地放在一处住宅区的上面，但我显然低估了现在身为巨人的我手指的力道，以及这巨大化不知几百倍的胸罩的重量，于是，这片住宅区就在一个女孩子的胸罩下变成了一片废墟瓦砾。真是的，我明明让你们把房子造的更坚固的。
        我俯下身子仔细寻找可以作为标尺的建筑物，但是这真的很难，但我可以理解，因为你们建造这些“高楼大厦”的初衷并不是为了给一个少女丈量她乳房的尺寸。你们建造的这些房屋不是太窄就是太矮，我想让自己的乳房完全垂下来，可你们建筑物的间隙连这点空间都提供不了。所以没有办法，为了满足我的实验，我必须手动清除一些不必要的障碍物，你们得记住，这本来不是我愿意做的。

        如果你们变成像我现在这么大，你们会发现拆除一片小区或者移走几座商场其实并不是一件十分困难的事情，和你在家中拾起掉在地板上的书并把它放回原位的感觉有些类似。现在，我终于制造出了一个容许我俯下身子时使自己的乳房完全放松下来的空间。接下来的工作，应该是开始“测量”了吧。

        结果是显而易见的，你们所建造的那些低矮建筑物很多都被我这对E或者是F CUP的乳房比了下去，也许你们应该考虑一下，为我这样一个爱美的女孩子多准备几件内衣是多么重要的事情，虽然这或许会耗费你们所有的布料。

        应该说，比起我以前的姐妹们，我跟那些男孩子们上床的次数并不算多，但是你要求一个充满活力的女孩子长时间禁欲，我想这并不是一个合理的要求。何况在我进行这个小小测试的时候，你们的那些小小建筑总是不可避免地碰到我的乳头，那可是我的敏感带啊。

        “哎，好想做一次啊~”

        我知道，能够满足我的男孩子或许永远不会出现，但是我的乳头却由于刚才的刺激挺立了起来，我想这对挺立的乳头对于你们和你们的城市来说并不算是一个很好的信号吧。

        如果明天一早你们身边的某片小区，或者某些数十层甚至上百层高的建筑物从地表消失的话，请你们一定不把这件事和一个女孩子自慰联想到一起，那样我会很不好意思的。哦对了，请你们也要一定无视那些一夜之间形成的湖泊哦~

        就这样，我的生活还在继续。我发现自己不再会渴也不再会饿，除了每隔一段时间就要发作的性欲之外，对于你们这些被欲望填满身体的人类而言，我实在可以称得上是无欲无求了。

        我发现自己变得比以前更加接近天空，当然，也更加接近你们制造出的那些已经起飞的飞行机器。我尽量让自己的行为受到内心道德标准的约束，但你们知道，这种虚无缥缈的东西作用实在有限得很。所以你们只能祈祷了，祈祷我哪天心情不是很好的时候，不会给你们那栋标志性建筑来上那么一脚。

        我大多数在人迹罕至的郊外休息睡眠，但偶尔也会尝试在城市中裸睡。我之所以只是偶尔尝试，是因为要清理出一片能够让我当做“床”的空间，确实有些“辛苦”，当然我知道给你们造成的“辛苦”会大得多。

        我每晚入睡前都在想，如果第二天醒来时发现自己又变回了和你们一样大小，那又会是怎样一番情景，你们会把我当做中世纪的巫婆一样处以极刑吗？又或者我为了不让自己以后被你们如此处置，现在把你们都“清理干净”？哎，做一个有道德的巨人真难。

        也许当我在你们的城市生活腻的时候，我会去这世界的其他地方转转，但在那之前我会先做一件事，这件事在我心中忍了好久了。

        “啊，果然还是好想跟那个晾衣杆做一次啊~”
