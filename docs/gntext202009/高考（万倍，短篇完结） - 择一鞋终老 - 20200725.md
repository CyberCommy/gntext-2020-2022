# 1

    公元3030年，由于人口密度过大，资源紧缺，缩小技术被开发出来，用以节省一部分人生存所占用的空间和资源。
    而划分缩小与否的标准就是高考，在每个人18岁的时候必须参加高考，根据分数分为：
    A级：675—750分，维持身高
    B级：600—674分，缩小十倍
    C级：525—599分，缩小一百倍
    D级：450—524分，缩小一千倍
    E级：375—449分，缩小一万倍
    只有A级才是精英社会，B级C级多是低端产业，而D级E级的生活，这些中学生们完全不知道，他们的老师也不知道，可能跟正常身材的人比，已经不属于一个世界了。
    考试结束后，有人欢喜有人忧，在某重点中学，一对情侣正相互拥抱准备暂时分别，他们即将通过传送塔变成不同的大小，再被传送到各自的学校或者居住城市。
    “我估分735，报了青北大学，你呢？”
    “720，我们青北见。”
    “青北见。”
    男孩名叫李青，常年霸占重点中学校的第一名，女孩叫徐北，垄断了这三年来的第二名。

    当李青从传送仓醒来后，只见舱门外站着几个健壮的男人，纷纷对他驱寒问暖，介绍着当学徒的工作，一副热情好客的样子。
    “这就是A级市区吗？个个是人才，说话又好听，爱了爱了。请问青北大学怎么走？”
    几个人沉默了一番，随后纷纷道：
    “这小伙子是不是精神错乱了？”
    “这里不是A级市，这儿是E级市。”
    “青北大学？怎么走可还行，坐飞机也到不了吧？”
    “我看看啊，高考分数449分，未录取，分配等级E，难怪，差一分进D级，是受打击太大了。”
    “我的，七百，三十，五分……”李青全身瘫软，快要晕过去了，他实在想不明白发生了什么能扣掉三百分，除非理综没写名字，那也太假了。
    “少一分不是事儿，跟着哥也能吃香喝辣，猎食队今天打了只大蚂蚁回来，来了这么多新人，今儿个我请大伙吃蚂蚁肉！”说着，众人把李青扶上了一辆破旧的卡车，向西城区驶去。
    “那是什么？”李青看到市区中央有一座几百米高几千米长的粉色的大山，上面筑有不少房屋。
    “几年前一个小女孩把鞋子丢在这里了，砸坏好多建筑，我们找不到她赔偿，也搬不走，就只能留在那里了，它叫做鞋山，有时会有昆虫入侵，是个很好的防空洞。”
    “我小时候打哭过不少这样的小女孩，没想到高考分配后，那样的小脚丫上的鞋子都比我大。”有人说。
    李青沉默了。

    E等公民的城市里没有手机电脑那种高端设备，因为没有利润，就没有人为他们生产，但市中心却有一间网吧，安放着他们从D级市搞到的十几台破旧电脑，这对E级市来说是最高档的奢侈品，网费贵得惊人。
    李青在一家为D级、C级市制造细小精密零件的零件场卖力地打工赚钱，三个月后才和几个兄弟一起凑到了一小时的网费，平均每人用12分钟。
    “李青，抱歉啊，只剩5分钟了。”
    “还不是因为某人，说什么在网上撩到了C级的妹子，还说机会难得多占几分钟，以后被包养了富贵了不会忘掉我们。也不瞅瞅自己什么德行，C级的妹子怎么会看上他？”
    李青没搭话，赶紧走进了电脑间，这电脑屏幕都比他高，键盘也是十倍大小，打起字来很麻烦。
    打开QQ，别人的几百条消息看都不看，迅速找到了那个名字，徐北，看见了她这三个月发来的消息。
    “青，我在青北找遍所有新生也没找到你。”
    “这届也有个叫李青的，不过是个女生，考了736分。”
    “你是不是报错学校了？你现在在哪呢？”
    “那个女生期中成绩特别低，我感觉她有问题。”
    “你到底怎么了，这么长时间没回话。”
    “我好想你。”
    ……以及一些表达思念的话和分享的学校日常。
    只剩三分钟了。
    “我不知怎么被分成E级了，现在在XX市。”李青点了一下发送位置，他记得在中学正常身高时看过地图，地图上不会显示C级以下城市的位置。
    “我觉得我很可能是被人调包了，736是我考的才对，我想见一下那个女生。”
    “我这里不方便上网，时间有限，你有空来找我吧，不过要注意脚下。”
    #对方邀请您进行视频通话#同意
    网络加载中……
    这破网吧终于网速不够用了。
    在时间结束之前，两人终于见了一面，刚张嘴，还没说话。

    徐北来了。
    她戴着白色的遮阳帽，穿着粉色的卡通短袖，蓝色的牛仔短裙，白嫩的小腿下是黄白条纹的短袜，以及一双运动鞋，充满了青春的气息。
    只是这样一个女孩，她的鞋底都有三百米高，鞋上的网洞能装得下一栋别墅。
    其实这个E级城市是在一间被废弃了的小学的一个角落里，而大一些的C级市、D级市则占据了操场。
    徐北饶有兴致地看着这些比她小一百倍，一千倍的小人和建筑，小心翼翼地绕开了他们。
    当她走到李青发送的坐标位置时，却只看见地上有一只粉色的小鞋子。
    “李青，你在这儿吗？”说着，她缓慢地蹲下，才勉强看清地面上沙粒大小的建筑，这些建筑最大的也不过米粒大小。
    “也太小了。”她心想。
    整个E级市对她来说不过是1.6米长宽的大小，以前有着C等人给他们做的围墙，防止昆虫入侵，但现在早已残破了，徐北没有看出来，已经走进E级市好几步。
    李青看着那两千多米长的巨大鞋底压了下来，来不及逃跑，在这么大的物体前逃跑几乎是没有意义的，那只鞋子还是他送给她的，结果现在自己比它下面卡着的沙粒还要渺小。
    “轰”的一声巨响传来，大地震颤着，厂里的设备翻倒砸下伤到了不少工友，裹着尘沙的气浪掀飞了厂房的顶盖，撕扯着差点把他们也卷了出去。在她脚下的建筑，片刻都无法支撑，直接变为齑粉。
    李青骇然地望着那只巨大的鞋子，自己平时居住的楼区已经不复存在，因为受到重压，地面隆起了十几米高的土山，环绕在她的脚边。
    还好她已经停下脚步，但蹲下造成的大风又一次摧残了她脚下的人们。
    得赶紧联系她。

    E级市其实是有市内电话的，李青跑到电话亭，联系到了网吧老板。
    于是整个E级市都听见了徐北手机上特别关心的提示音。
    徐北有些脸红地打开手机一看，只见李青给她发了消息：
    “李青说他在你左脚侧面大概1厘米的地方，叫你先不要动。我是网吧老板。”
    徐北不敢动，但还是缓缓侧过头来瞥了一眼自己的左脚，却看不到小小的李青。
    吱呀吱呀的声音传来，身体的转动引起了脚部偏移1毫米，这一毫米却是10米的位移，她脚上的鞋带垂落了下来，扫碎了一大片楼房。
    “李青说鞋带倒了，他得跑了。”手机上显示。
    徐北心中咯噔一下，她知道哪怕只是鞋带的末端对只有0.1毫米左右的小人来说也是二三十米宽的庞然大物，砸下来一定十分可怕，很是担忧。
    徐北放下一根手指，压在地面上，想让李青爬上她的指甲，失败后又垂下一根头发，李青终于抱着头发，来到了她的手心。
    好在徐北发现及时，不过还是无意中踩死了五万余人，市长坐着直升机来到徐北面前，喊话要求她赔偿1000亿元E民币，其实折合成A民币只有1毛钱。徐北很大方地转账了1块钱，表达了自己的歉意，使这个E级市一下子多了10年的GDP。
    1万亿元相当于平均个活着的人都能分到100万，极大地改善了他们的生活。在市长宣布赔偿方案后，很多人欢呼不已，甚至恋恋不舍地看着天空上那位巨大的女孩离去的背影，希望她再踩几脚。

    青北大学。
    李青站在徐北手心上，用天文望远镜观看这所他本来应该在这里学习的大学。
    “徐北学姐！”有一个捧着箱子的少女在向徐北打招呼。李青望向那箱子，发现里面竟然密密麻麻的全是小人。
    “你这是在做什么？”
    “做社会实验啊！花一点点钱在网上发招募，这些D级的人都疯了似的抢着要来参加实验呢。”说着，少女在盒子中放了一小块饼干作为他们的伙食。
    “你知道李青在哪里吗？就是我们这届那个李青。”
    “那边。”
    徐北找到了那个顶替李青的女孩。
    “你认识他吗？”徐北伸出手指给她看。
    “这是啥？”女孩看着这一根手指，不明所以。
    “他叫李青，高考736分，最后却被分到了E级市，你觉得你不应该解释一下吗？”
    “我为什么要给一个E级的虫子解释？”那女孩眯了眯眼，冲着李青说道：“认识我，是你的荣幸。”
    无意中一个吐沫星子砸到了李青身上，一万倍的唾沫星子足有三层楼的大小，李青不会水，拼命地挣扎着，徐北连忙拿出卫生纸吸干了水分，李青才得以从一摊液体中爬出，气喘吁吁地趴在徐北白嫩的巨大手指上。
    竟然真有人会被吐沫星子淹死，笑得那女孩花枝乱颤。
    “说吧，你要多少钱。2000块够不够？”
    2000元A民币，相当于2000万亿元E民币，这是E级市两万年的GDP，别说他一辈子花不完，就算整个E级市100万人口，花上两万年也花不完。
    “这是钱的问题吗？你毁了人家的人生！”A等公民随便一点小钱，对李青来说就是天文数字，实在是种悲哀。
    “不要拉倒，尽管去告，你连诉讼费都出不起。”那女孩不屑说道，转身离开了。
    确实出不起，E级市全部人口奋斗一辈子的产值，只值A民币4块钱，甚至还买不了一杯奶茶。
    “我帮你告。”徐北对李青说。
    徐北把这件事发在了网上，人们纷纷谴责高考顶替者，在舆论压力下，官方派出调查组调查此事，青北大学也对该同学做出了开除学籍的处罚。
    但就算证据确凿，又能怎样呢？那个女孩最多会再被追究一笔赔偿，敢于在传送塔做手脚的老师估计会进局子，而E级的李青却没有机会变回原来的大小了。

    当徐北和李青再次回到E级市时，地上只剩下一片乱七八糟的脚印，城市的废墟卡在了脚印之间，而那座整个倾尽E级市之力都无法搬走的鞋山，也被人一脚踢飞。
    “都怪你！”有幸存者们从断壁残垣中走了出来，指责李青。
    “愿意原谅他的人，可以爬到这张白纸上，以后你们的食宿，我包了。”徐北说道。
    幸存者们闻言大喜，高举起了李青，欢呼不已。李青却高兴不起来，只感觉有些累。
    几千人站在这张白纸上，也很不起眼，徐北把他们带回宿舍，安放在一个鞋盒里。
    E级市的幸存者们从此过上了前所未有的美好生活，徐北购买了造好的微雕，每个人都住进大别墅，每个人都配备了电脑，毕竟D级市的电脑对于A等公民来说花费实在是微乎其微。每天都有新鲜的面包山（面包渣）被放在盒市的西边，每天都会有一滴水滴在盒市的东边，形成一个湖泊，供给人们的饮用。
    人们再也不用劳作，日复一日地吃喝睡，玩电脑。最后都变成了肥宅。
    李青觉得这样的人生毫无意义。
    在某一天徐北去上课时，李青给她留言道：“我不想被人可怜，过着被施舍的生活。原谅我不辞而别，走了，再见。感谢在最美好的年华遇到你，可惜不能陪你一起走一辈子了，我们没有可能，你去找个更好的吧，不要想我。”
    李青拉着行李，走到鞋盒的角落处，从缝隙中钻出，看了看鞋盒外边的一座鞋山，继续向前走。
    走了很远很远，当他再次抬起头时，却发现自己只是从鞋后走到了鞋前，原来走了这么久，也不过是一鞋之地。
    他有些惘然地跌坐下来。
