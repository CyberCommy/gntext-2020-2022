# 雕刻家和助手

作者：yanmiaoyan

TID：17044

 

# 1

*本帖最後由 yanmiaoyan 於 2014-6-15 13:14 編輯*

一间宽敞的房间里，一男一女正在努力的工作，那个男人正在一块自己两倍大的石头上努力凿着，而旁边的金发少女正在努力打磨一块同样大小的石头，只不过有一个不对劲的地方，那块有男人两倍大的石头在女孩面前是那样的小巧。

“好想揉胸部。。。”这时那个男人停了下来，站在地板上，突然自言自语道。

“啊。。。”那女孩显然听到了，停下了手里的工作，站起来低头看着这个男人，这下直观的体现了两个人的差别，那个男人的身高只勉强到那个女孩的膝盖。

“不是因为h的理由哦。”那男人抬头看着怀疑的看着自己的女孩，连忙拿出图纸，“是为了雕刻的更加真实，我想以最棒的质量来完成这次雕刻，光看图像实在做不到。”那女孩蹲了下来，无语的看着那个男人。

“但是我没有揉过胸部。”那男人继续说道。

“我没问你哟。”女孩低头看着他，说。

“所以不实际揉下在自己脑海中留下深刻印象是不行的。”那男人没在意女孩的吐槽，继续说道，“虽然不是说想揉就可以揉的。。。但是如果为了工作，有什么人愿意协助一下的话。”说着，他声音小了下来，偷瞄起女孩那在他上方的胸部。

“老师真的没有别的心思吗？”那女孩说道。

“额，稍微有一点。”男人犹豫了下，说道。

说到这儿，该介绍一下这到底是怎么回事了，那个男人名叫青木，是一个雕刻家，如今已经有了缩小技术，所以为了能够在城里的房子也能方便雕刻，缩小了身体，而那个女孩则是他的助手真子小姐，缩小后的石头就算是这样的一个小女孩也能轻易的搬运，同时能够用女性特有的细腻辅助青木的工作。而这时候青木正在雕刻一个女神的雕像，没谈过恋爱的他对自己所雕的胸部总是不满意，于是就出现了开头的那一幕。

“那。。。那么，真子小姐你能不能低下身子，让我近距离看你的胸部，只是为了工作，是真的！”青木强调。

“我，我知道了。”真子跪了下来，弯下了身子，靠近了青木，“如果这样就能雕刻出好的作品的话。”

“谢谢了，这样就好了。”青木抬头看着靠近自己的胸部，突然摸了上去。

“啊啊啊，干什么啊。”真子吓得直起了身子。

“太过漂亮的胸在眼前，不经意就。。。”看着青木无辜的脸，真子默默地站了起来，伸出了自己的脚，踩了下去。

后来这件雕像在拖迟了两天后完美的完成了。
