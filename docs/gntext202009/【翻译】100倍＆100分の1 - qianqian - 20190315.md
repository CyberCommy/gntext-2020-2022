# 2

我：“呐！要不要来玩一下sizeplay啊？”

莎拉：“诚司君真的很好这口呢~！如果不是像人家这样的女孩子的话，怕是会被你吓到的吧？”

我的女朋友莎拉，是能够接受我这种癖好的人。虽然最初刚知道我有这种奇怪的癖好时她有觉得很惊讶，但是和我相处了一段时间后，现在，她也变得妄想要巨大化，也经常妄想着虐待变得小小只的我，我认为我们能作为一对都有着相同癖好的情侣，真是太好了。

我：“莎拉最近不也觉得这样的play还不错吗？而且这一次我要告诉你个很宝贵的消息哦♪“。

莎拉：“……你说的sizeplay不就是前段时间玩的VR游戏嘛，难道是出了新作吗？”

我：“啊！那个真的是超棒的！呀~现在的时代真的很美好呢……不不不对，我是想说，我真的找到了能让人变大和缩小的药物了！是不是很厉害啊？！”

莎拉：“不管诚司君有多么喜欢sizeplay也不能说慌哦~♪还是说睡迷糊了呢？”

莎拉笑眯眯地说着，这是完全不相信的表情呢。嘛，想来也是，这种话听起来简直就像是开玩笑，所以不相信也是没办法的事。

我：“看啦！就是这上面写的这个啦！不过因为要10万日元，显得很可疑，所以谁都没有买过。”

莎拉：“可能这个只是用来展示而不是出售的，所以才卖那么贵的吧。”

我：“虽然没有得到很详细的消息，但是我从熟人那得知这个可是真品来的哦！这样我们就能在现实中也能一起玩sizeplay了呢！！”

莎拉：“那么诚司君喜欢多大的尺寸呢？”

我：“我想先试试所谓的‘一百倍’的差距，因为这种机会可不多，所以……费用就由我来出，然后拜托你，陪我一起，帮我实现这个愿望吧！”

莎拉：“好吧好吧……就当是被骗了吧，如果真的是被骗了的话人家可不管了哦！”

就这样，今天我们幻想着“一百倍差距”的场景，然后各自归家解散了。

一周后。

我：“还没有跟莎拉说，前些日子订购的缩小药其实已经送到了呢，事不宜迟，赶紧试试看吧！？而且今天莎拉也打算要来我家的，如果她看到我变小的样子一定会很惊讶的吧？呵呵，真期待呢！”

打开送达的包裹，发现里面有类似喷雾器和饮料一样的两种东西。
看来如果是用喷雾的话，不光是肉体，喷在穿戴的东西上，那些东西也会跟着一起变大或变小，这似乎很方便呢。而饮料则是分为“100分之1的缩小药”和恢复大小所用的“100倍的巨大化药”。

我：“这东西做得比想象中的还要正规嘛，这样看来真的能期待一下呢……”

然后我立刻把喷雾喷到衣服和手机上，而自己则喝下了缩小药水。

我：“哦哦！视线渐渐下降了！这东西，果然是真货啊！！”

我感动得浑身发抖，但现在我可没有时间沉浸在感动之中了。
我现在的身高已经缩小到了不足原来的一般。如果就这么呆在地板上，会有被其他人一个不小心而踩到的危险。我有预料出现这种情况，所以我买了一个玩具屋，但是我把它放在桌子上，因此我得把椅子当作垫子才能爬上桌子。这时我的身高已经缩小到了原本的20分之1，刚好和玩具屋的大小匹配。

嘎吱！

我：“这就是玩具屋的里面吗……嘛，因为缩小到刚好匹配的尺寸，所以看起来和普通房子没两样啊。”

但是，我是将要缩小到100分之一的。在这20分之1的玩具屋中，看着眼前的空间逐渐变得广阔，最终这玩具屋也变成了仿佛是我的5倍大的巨人的房子。

我：“卧槽！这玩具房简直就是巨人的房子了啊！这楼梯，仅仅是一层阶梯就有我肩膀高了啊！桌椅什么的，根本爬不上去！”

幸运的是，因为是玩具屋，所以玄关的门没有锁，而且还是塑料制作成的，虽然是寻常（在我看来）的5倍之大，但稍微出点力，还是能推开的。

环顾了下四周。

我：“哇！我的家……已经变大100倍了啊！超厉害的！”

那么，现在是万事俱备，只差莎拉了，这样想着，我便试着联系莎拉。

咚！咚！！

突然响起了巨大的声响，并伴随着剧烈的震动，在这种时候遇上地震实在是倒霉啊！
我慌忙地逃进玩具屋中，躲在了虽然只有20分之1，但对我来说大得过分的桌子下，蜷缩着身子瑟瑟发抖。

把时间稍稍往前回溯一下，回到我刚服下缩小药的时候。

莎拉：“嗯姆嗯姆……这东西意外地很有感觉呢。”

我并不知道的是，莎拉偷偷买了和我一样的药，其实她是为了吓我一下才买的，并打算在事后再向我所要费用，但是为了相互给彼此一个惊喜便没有说出来，所以在巧合之下，我们同时都买下了这种药。

莎拉：“衣服，智能手机还有包包都喷上了喷雾，这样就行了吧？这个看起来也不像是假的东西……慎重起见，还是在附近的公园喝下药水吧！♪不过那样的话，怕是会让其他人看到“百倍差”的场景吧，虽然很害羞，但是如果诚司君喜欢的话，就这点小事……”

就这样，莎拉便在公园里喝下了“巨大化药水”。

莎拉：“诶？！……我，我真的在变大！？”

莎拉的身高在增长着，逐渐超过了公园厕所的高度，然后比电线杆也还要高，再到公园种植的大树都要低头俯视……

莎拉：“这，人家这便是巨大化成了100倍的大小吗！？虽然有在VR中体验过好几次了，这种视野也已经完全习惯了，但现在却能很好地感觉到周围的存在呢，和游戏不一样，这就是现实吗？”

莎拉因为被周围的人围观着而感到很害羞，但是如今真的变成了巨人，莎拉感到非常的激动、兴奋。

咚！咚！！

当人们看到身高160m的莎拉在路上行走时，都吓得一溜烟就跑了，道路上基本没什么车，显得很宽广，即使是巨大化后的莎拉也能轻易地踏出脚步，然后伴随着巨大的声响穿过街道，莎拉走向了男朋友的家。

莎拉：“这就是诚司君的家嘛？看起来好小呢~就像玩具屋一样♪”

对于莎拉来说，只有100分之1的房子，尽管有着2层的高度，但莎拉想要把它拿起放到手心上也是轻而易举的事。

莎拉：“不趴下来的话看不到房子里面呢……把这个给压坏的话不要紧吧？”

如果这是在密集的住宅区的话，莎拉肯定会犹豫不决的，但是我家前面的，是一排近日预定要拆毁的旧式住宅区。这样的话，就算是破坏了，也不会有人受伤，也不会让人为难吧？

嘎嘎嘎~噶嚓！！！

莎拉：“诶？怎么这么脆弱啊？还以为这是在住宅区，房子会挺结实的说……”

当莎拉一趴下，在那下方的有着四层楼建筑的小区，在一瞬间便有四栋楼给压塌了。

莎拉：“唔……算了，不管了，让我看看房子里面……emmm，看不清楚啊，得想想办法。”

只是简单地趴下的话，对于莎拉来说视线还太高了，虽然能轻易地窥视到二楼，
但是要想看到诚司所待在的一楼的客厅，就必须把脸侧着贴在地上，这样莎拉的视线高度才能接近普通人，才可以看到一楼的客厅。

莎拉：“啊咧？已经到了约定的时间了啊，怎么会没有人在？”

在莎拉感到疑惑的时候，手机突然响了起来。

我：“莎拉！我家附近发生了强烈的地震，甚至还有建筑物倒塌了，你还好吗？在来我家的路上没有被卷进危险之中吧？”

莎拉：“呵呵♪地震啊……是在说人家吗？因为想玩「百倍差」的游戏，想着要让诚司君吃惊一下人家就偷偷地喝下药水巨大化到100倍了呢♪但是人家现在在看着房子里面，却看不到诚司君的身影呢，你在哪里啊？”

我：“不会吧？！你说的‘100倍’？！哇哦哦哦！！！莎拉你真的，变成巨人了？！！”

我从玩偶屋出来站在桌子上，看向窗外。于是，能看见那双巨大的眼睛在盯着我，仿佛覆盖了整个窗户一样。一开始听到莎拉巨大化了100倍的时候，老实说没什么实感，虽然很吃惊，但不怎么相信，但实际看到后，我才意识到这是真的。

莎拉：“你有看见人家吗？你是不是在家里啊？在哪个地方呢？”

我：“我就在莎拉你眼前哦，不如说，从刚才开始我们就一直在盯着对方啊……我就站在桌子上啊，再仔细瞧瞧。”

莎拉：“在桌子上？虽然能看见有个小小的盒子，但完全看不到诚司君的身影哦。”

对于我来说，莎拉巨大的眼睛充满了我的整个视野，但对于莎拉来说，根本看不到过于渺小的我吧。

我：“那么，把窗户打开，慢慢地把桌上的小盒子拿起来，注意点不要捏碎了哦！窗户并没有锁上，莎拉你应该很简单就能打开了。”

莎拉：“呜呜……太小了，很难做到啊……”

莎拉把指甲扣在一楼的大窗户上，轻轻地打开了。因为打开的窗户只容得两根手指进入，所以莎拉得首先把桌子拿出来放到院子里，再慢慢地捏起大概只有1cm的玩具屋，放到了另一只手的手指上。

我：“这摇晃好剧烈啊……那么接下来试着把玩具屋的屋顶给拆下来吧。”

莎拉：“诶哆…诶~哆……这样嘛？但是里面什么都没有啊？”

我：“我就在那个看起来什么都没有的玩具屋里面哦！因为我也想让莎拉吓一跳而缩小到了100分之1呢。”

莎拉：“……真的？”

我：“是真的哦！”

莎拉：“也就是说，人家现在是普通人的100倍之大，而诚司君则是普通人的100分之1……我们彼此实现了相反的‘百倍差’，那我们之间岂不是相差了1万倍吗？！”

我：“嘛，就是这么回事吧……”

莎拉：“只有一万分之一大小的诚司君啊……虽然在VR里已经习惯了，但现在可是在现实中呢，一个不小心就会把诚司君给碾碎了吧，真可怕呢……不过却意外的dokidoki地感到兴奋呢♪“。

# 3

莎拉从包里掏出放大镜，再次观察期玩具屋的内部。

莎拉：“本来是为了能看清100分之1的诚司君才带来的放大镜，居然是以这样的形式派上用场呢……诶哆，这样总算能看到了呢，那个看起来有些微妙的、在移动着的点就是诚司君了吧？”

我：“我想大概是吧……嘛，虽然比当初预定的尺寸差了很多，但是难得的机会，我们来相互接触一下，体验一下尺寸的差距吧？”

莎拉：“原本是有这个打算的，不过人家现在这么大，而诚司君又这么小，人家可是碰不到你哦。”

我：“那我就站在莎拉的手指上吧，让我好好看一下此刻的景象。”
我从莎拉食指上的玩具屋中跑出来，然后为看到的景色而大为惊叹。

我：“这可真厉害啊……本来100倍的话大概就是大厦级别的大小吧，但是现在相差了1万倍，这看起来已经完全是超越了大厦，甚至超过了高山的级别了啊！！”

我从手指上站起来，全身都能感觉到从莎拉的手指中释放出来的热量、以及在皮肤之下因血液流动所产生的强力脉动。放眼四周，周遭有着比学校运动场更为广阔的空间，但实际上，这只是莎拉那只有2m大小的手指而已，但对于现在的我来说，则是宽200m，整根手指的长度更是超过1km，是一块极其宽阔且巨大的大地。

我：“那么，以现在的我的体感来说的话，从指尖这里到手掌的距离差不多是1km吗？要不要去看一下呢……哇啊！这是什么？！”

我突然从指尖上摔了下来。不对，不该说是从手指上掉下来，而是说掉进了手指里比较适合，也就是说我突然掉进了指纹的缝隙里了。

我：“不会吧？！这是指纹？这可是有2m高啊！”

对于竖立眼前这面超过我身高的墙壁，我怎么也爬不上去，实在没办法，只好打电话给莎拉说明情况。

莎拉：“诶诶？！诚司君掉到了人家的指纹里了？！就算你叫人家帮忙……人家也很难控制好力度啊，要是一不小心就会把诚司君捏死的啊，而且，要怎样才能捡起掉进指纹里的东西啊？太小了根本办不到啊！”

说的是呢，那根本做不到的啊。没办法，只能自己想办法了，决定了之后，我静下心来观察指纹的墙壁。其实仔细一看，这堵墙并不像水泥墙那样平坦，因为是是构成人体的物质，所以意外地凹凸不平，能像攀岩那样有可以抓住的地方，而且着力的地方也很合适。我暂且是找到了容易攀登的地方，总算是从超过2米的指纹谷里逃了出来。

我：“哈……哈……我从来没想过从指纹中爬出来是这么个吃力的活。”

为了不会再次掉进指纹之间，我在指纹上跳跃着移动，然后成功回到了玩具屋。本来还想着借着这难得的尺寸差和莎拉做一些色色的事情的，但是，这个大小，就连想在莎拉的皮肤上移动都成问题。

我：“嘛，总而言之，先恢复到原本的大小再说吧！莎拉也一起恢复吧？”

莎拉：“也是呢，那么下次再来享受sizeplay吧♪”

我和莎拉一起，在事先喷了喷雾、一同改变大小的包包里拿出了药水……

我：“阿勒？为什么两种药水的量是一样的？”

刚才变小的时候喝的缩小药，所以药量相比起另一瓶是绝对较少的才对，但是现在不知为何，两瓶药水的量居然是一样的。虽然仔细地再确认了好几次，但还是看不出有什么不同。仔细看了下说明书，这个药似乎是一种特殊的纳米机器，如果保留了一定程度的量的话，放着它便会自动增殖，直到瓶装满为止，虽然这种东西在经济上是非常的实惠啦，但是现在可是有个非常迫切的问题要解决啊……

莎拉：“巨大化和缩小化的药上面的标签被剥落了，完全分不清哪瓶是什么药了……”

莎拉的巨大声响起，没有载着我的另一只手在包里搜寻着，莎拉所拥有的药水也和我一样自动补充了。但是，由于标签的粘着力低下，被完全剥落了，再加上两种药水都是无色透明的，因此无法辨别到底哪一瓶是什么药水。本来的话，用量来区别两种药水是没问题的，但没想到会自动补充回来，现在的话根本就无法分辨出来。

我：“我这边也一样啊……可是，不喝下药水的话就无法变回去啊，那要不我靠直觉来试试吧！”

莎拉：“那祝你好远！！”

咕噜！

我：“都是二分之一的机率，我应该没那么……卧槽，别吧！”

看起来我的运气实在不好，再次喝下了缩小的药水，眼看着就缩小了。
倍率的重叠，100分之一的100分之一，如今的我，变成了1万之一大小的微生物了。本来20分之1的玩具屋对我来说是5倍的大小，那么现在变成了500倍的，超大巨人的房子了啊，现在的我身处其中，就像是巨人感觉不到的蝼蚁一样。如果我就这样回家的话，就现在这个尺寸，遇上虱子或跳蚤的话是绝对不会平安无事的。

轰隆！！！！

我：“怎么了怎么了？！怎么会有这么强烈的晃动？！！”

总之我得先给莎拉打个电话，告诉她我尝试失败了才行。

我：“真的很不幸啊，喝错了药水，现在我变得更小了，只有普通人的1万分之一，再加上这药水一天好像只能服用两次，所以看来今天是没办法复原了呢，所以说，莎拉啊，能不能想办法把我和玩具屋一起，保护起来？”

莎拉：“诶哆……那个，关于这件事，其实，人家也把药水给搞错了的说……”

我：“不，不会吧，那岂不是说……”

听到莎拉的话，我不禁冒出了冷汗，本来就已经是普通人大小的100倍了，如今再变大100倍的话……

莎拉：“人家现在好像是普通人的1万倍了哦……哎呀~真是糟糕呢~♪”

我：“为什么听起来感觉你好像很开心的样子？”

莎拉：“嘛~♪虽然不是故意失败的，但是呐，能变成身高16km这么厉害的大小，其实人家还是挺高兴的呢♪因为你看呐，就连富士山也够不着人家的膝盖哦！？云和飞机什么的，都只能在裙子下方漂浮着呢~而且现在，人家要是愿意的话，就算是国家，也能轻~松~毁~灭~哦！呵呵，开玩笑的啦♪。”

我：“不不不，这根本不是玩笑了吧！你现在这种级别的大小，是真的能做到的啊！根本笑不出来啊好不好？！”

咚！

传来一阵剧烈的晃动，就连玩具屋也摇晃了起来。

我：“哇啊啊！怎么了？！莎拉你动了手指吗？！”

莎拉：“没有哦，人家没有动过哦。”

咚！咚！

每次震动的声音响起，玩偶屋都会剧烈地摇晃，甚至被震得慢慢地移动了起来。

我：“难不成这是莎拉的血液在指尖里的毛细血管流动的原因？只是单纯的血流流动产生的‘脉动’，就能让人感觉到剧烈的地震吗？！”

对于现在的莎拉来说，玩具屋就像当初只有她的1万分之一的我吧？而如今，再缩小100分之一，只有普通人大小的1万分之一的我与是普通人的1万倍巨大的莎拉，这尺寸的差距是达到了无法想象的1亿倍了啊……对于现在的我来说，莎拉已经是比行星更为巨大的女神了啊。那么，即使血液流过这个指尖所产生的，“单纯的脉动”，对我造成难以想象的影响，也不是不可理解了。

嘎咚！嘎咚！！！

我：“哇啊啊啊啊！！！”

被连续不停的地震侵袭的玩具屋，一点点地移动着，最终是无可奈何地掉进了指纹的山谷里。好不容易才保护自己不受伤，我拼命地往这超巨大的玩具屋的玄关跑去，透过门缝看清了外头的情况。

我：“这……想要爬上去是根本不可能的吧……”

在刚才这指纹对于我来说就应经是超过了2m高得墙壁了，那么我现在又缩小了100分之一，那么2m的高墙就变成了200m高的悬崖了，而在此基础上莎拉还变大了100倍，那这样的话……这个指纹，就如字面的意思，变成了无比深邃的山谷，甚至看不见这山谷的顶部。

莎拉：“哇哦！这居然是人家的指纹吗？好厉害呢~完全想象不到这是人家的身体的说！”

莎拉看见我发给她的照片，不由得发出感慨。

嘛，不过这张照片，可能是这世上任何的专业摄影师都无法拍到的吧。

我：“总之，等到明天就喝药复原，把一切都结束吧！”

但是，到了第二天早上，忘了给两种药做上标记的我们，又要为喝哪种药而烦恼了，嘛，这只是其中一件让我担忧的事，因为如果接下来又搞错药水的话，我们就分别成了普通人大小的100万分之一和100万倍了啊，那相对来说的话，我们就成了体型相差1兆倍的情侣了啊……唔，老实说，我没有能够喝下正确的药水的自信啊。
