# 縮小故事1（六日小劇場）

作者：4399小影

TID：21238

 

# 1

今天是一個平凡的早晨

但是不平凡的事情是....

我從床上醒來的瞬間的景色不是普通的樣子

我...變小了 我觀察了一下房間的環境

我大概剩下3公分左右

這並不是一件好事....我自己那麼覺得

但是這不是重點阿

下一個瞬間

房間傳來巨大的聲響

哥~哥~ 上學要遲到了喔~

一個巨大的身影走進了我的房間

呃...那是我的妹妹 她目前17歲是一個高中生

...為什麼只包著一條毛巾來我房間 而且還是早上

妹妹朝著我的方向走了過來....

我大聲的喊著她 妹~我在這裡阿

但是她卻沒有看到我....因為我實在太小了

不對她的手怎麼朝著我伸過來

我看了腳底下...對齁

妹妹發出了無奈的聲音 自言自語著

討厭~~哥妳又拿我的內褲去幹什麼了啦

不對~~妹妹 我在這裡阿

我就這樣在那內褲的中間

被沒有注意到我的妹妹 夾在她的內褲中被穿上了

奇怪  哪裡怎麼癢癢的

妹妹露出了不舒服的表情

用手指來回摩擦了她的小嘴 發出了些許的呻吟

我在妹妹的那裡...不....不要

那手指的力道把我推了進去她的陰道裡

我....這個味道實在沒辦法

妹妹這裡面的味道重的嚇人 那濕滑又黏稠的感覺

我跟本無法動彈 被緊緊的夾在了裡面 連一下子都不能動 全身感覺....

我的內臟已經承受不住那陰道的夾合力

加上妹妹匆忙騎著腳踏車去上學更是夾的用力

不到一下子 我的器官被壓的四分五裂 全部都留了出來

....就這樣 哥哥死了

妹妹那天剛好下課去了廁所

排了MC的血出來 變這樣不以為常的過了一天

哥哥就這樣從此消失在了世界上
