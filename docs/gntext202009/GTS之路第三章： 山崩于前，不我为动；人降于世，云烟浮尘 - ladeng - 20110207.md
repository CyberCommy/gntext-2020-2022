# 2

第三章

维克多·雨果所说的：“一种思想的时代到来时，没有什么可以阻挡它的前进。”

-----------------------------------

“历史将记取的社会转变的最大悲剧不是坏人的喧嚣，而是好人的沉默。”

梅梅是那种坚强固执倔强同时也会温柔体贴妩媚，善良开朗大方却又极端偏激邪恶。
而莲莲则是精干顽强支配欲极强也富有细腻的感情，并且不是一个患得患失的人，潜力很大。

莱阳的药厂出现骚动，工程被迫停止下来。大家都在忙自己的一亩三分地，而我却没法帮助他们。
这个事情，看来单总自己不出面的话，事情就对我们越来越没有利。

我先给姜华打了一个电话，姜华说这次事态有点失控，已经有人开始满墙贴大字报了。非要政府给一

个说法，而且先行运来的药理实验室设备和建材被工人扣押了。

我心情郁闷到了极点，当天晚上，单总来到上海，休息了一下，准备第二天带着我一起去莱阳。

单总明显瘦了一圈，明显的，他感到这里面的事情是他能力所无法掌控的。

我问他“和王总沟通了吗？”

“说了好几次了，他推说手上有事不想管，他想冷处理。我感觉他就是想要我好看，因为股权分配是

1：1，而新药的研发掌握在我手里，他会不会想通过这个搞垮我，然后夺取工厂的控制权。伊娜，你

把合同书复印件给小汤看看。”

姚伊娜递过来一叠薄薄的材料，是他和王总签署的投资协议。里面有一句话吸引了我：

《双方合作初期股权分配为1：1，如果生产需要资金入股，双方可以监督和吸纳第三方投资。投资额

不得超过现有股价的30%》

我把这句话指给单总：“这句话啥意思，你了解吗？”

“我也是后来才了解的，如果王总想吞了我们，他完全可以在最需要钱的时候控制一个第三方基金投

资进来，然后他看到药厂的技术和知识产权的注入后有发展了，再购入这个基金，达到大量控股的目

的。现在他缺的就是一个最需要钱的机会。所以他现在搞这个事，因为他知道了我们现在有资金，故

意来探我们的底。”

单总燃了根香烟：“目前是担心的是，我收拾不了这个局面，如果答应工人的要求，我们就背上一个

沉重的包袱，多少钱都不够砸的。”

还是去当地看看吧。

------------------------

闲话少说

我和单总，娜娜赶去莱阳。我们先去厂子附近看了一看，厂子里果然停工了，地面上一篇狼籍。我们

直接跑到梁厂长的办公室，问了问情况。梁厂长说道：“这几天来，工人闹的厉害，开始的时候还只

是上访，告到市里政府不管，说这是商业上的事情。这几天在酝酿去省城告状，我这也没办法了。”

单总说：“他们要求什么条件？”

“简单，就是要岗位呗，不能有下岗的。你说整改意见里，全自动流水线一上，起码一半岗位要被砍

。要不怎么闹成这样”

“问题是没有这么多岗位啊，整个人事计划也就是100名工人，30名研发人员，这不是难为人吗？”

单总表示没办法

“毕竟这是国有企业，职工都当铁饭碗啃着。要不你们多投下资，多上几条生产线”

“不行，这样投资利用率太低了，没那么多产品，上了也是闲置，到时候还得裁员。”

“……”

谈话有点僵，我们满腹心思走出梁厂长的办公室，回到宾馆。

姜华在宾馆等着。

“单总，我观察了梁厂长的眼睛，有点飘忽，好像注定就是来谈条件的。你看，增加投资，正好合王

总的意思，虽然我们现在不缺钱，可还是比不上他的后盾厉害。这一投资，条件估计要重新谈了。”

“是的，估计我说什么工人都不会答应。”

“这样吧单总，咱们不如将计就计……”

讲述一下我给单总出的计策，既然王总想利用员工挑事，我们就让他自食苦果。
首先我想先晾着王总和梁厂长，我们在上海浦东星火开发区先建立临时药理实验室，科研进程不能因

为这个推迟。如果事件有好转，再整体搬到莱阳。

其次我们把合同上的资金注入单总自己的医药公司账户，以事件未解决投资有风险的理由，向法院申

请延迟注资。省的注入资金和知识产权后被王总捞走。

这样如果建厂不成，还可以拿着新药向其他药厂求助，我们自己建厂，资金也足够。即使法院的延迟

注资到期，我们只损失200万的合同费，也比被王总整锅端走强！

单总显然也心疼这两百万，从他的表情可以看出来。这意见他不是很赞同。

“单总”姜华看他发愁，终于开口了

“这几天我就在这里观察，一般这种事情，政府会出面干预，做工作的。如果政府不出面，而王总又

冷处理的话，我想他盘下这个药厂的决心下的很大。”

“唉，先这么处理吧，咱们全回去，新药研发不能慢。今天就叫车，把先期的实验室设备拉走。”

“如果职工不让拉走怎么办？”

“怎么办，只能哄了。”

--------------------------

我们叫了卡车来拉货的时候，果然十几个工人拦着门不让进，设备也不让拿。

单总不高兴：“你们这么做，是侵犯公私财产的，这些设备是将来建工厂的，你们保管不当，损坏了

责任谁来负？”单总咄咄逼人的说。

“我们不管，你们资方一天不解决我们就业问题，这个厂子一天就别想进。”一个40岁左右的职工大

声说，声如洪钟。

“你们组织这个事情的，谁说了算，我们谈下吧！”单总不无霸气地拆了一包烟，自己衔了一根，递

过去几根，算是给他们个面子。

果然他们反应不那么强烈了，那个工人说道：“我们都这个意见，没有谁带头。”

“那我就和你谈”单总深吸了一口。“我来之前，和财务做了这么个预算。”

“新工厂呢，暂时品种比较单一，但是我们研发的药品是1类新药，市场上的地位绝对是垄断的！”

“什么药？”大家纷纷议论

“暂时不能和你们说，不过我保证，我们就以这个药开始发展，逐步将片剂，胶囊，贴膏都整上生产

线，这样的话可以多上几条。”

“什么时候？”大家问道

“这要看你们的态度了，设备扣在这里一天，研发就慢一天。现在各种关系都在围绕这个药开始活动

了，别关系打通了以后，药拿不出来，到时候就不是下岗100个200个了，一个都剩不下，赔个精光。

”

显然这话引起一场骚动，姜华补充道：“现在厂子建设资金不到位，新药拖延研发就是损失金钱，我

们已经投资这里了，药肯定要在这里生产的。大家放心吧！”

显然职工的战线被单总的谎言说动了，没有人再阻拦，我们顺利的把设备都抬上卡车。刚开出厂门，

前面刷得来了3辆小轿车，下来7，8个人。

“停，都别走，谁敢把东西拉走我揍谁！”原来这些人是打手，不用说，一定是王总得到风声派来的

。

“你们是谁！”单总觉得王总未免过份。

“我们是这个厂子里职工，今天你们偷设备，是不是想进局子啊！”

“开什么玩笑呢朋友？”单总虽然心里没底但是嘴上依然不依不饶。
“这些设备可都是我们投资的，不信问梁厂长去！”

“投资到厂子里的东西，任何人不能拿走！”这群人翻到卡车上，开始驱赶司机，控制住了驾驶室，

不让卡车开走。

“等会！”我问单总“给王总打个电话吧，看他什么反应？”

“哼，他当然不会同意，说不定我这么一打，他就知道我们怕了他了。”

“也对，主动打过去效果不好，我们等他电话！”

这群人里有个穿深蓝色夹克的小平头，走到职工面前大喊：“你们怎么能被他们骗了？他们是要撤资

，拖你们，等拖到最后，吃亏的是咱们职工！”

单总也没和他废话，他知道怎么解释这些人也不会放过这批设备。于是我们暗中交代司机，如此这般

。

当夜，这个司机从破烂的厂房模进去，用备用钥匙启动了卡车。

引擎一响，立刻从门房里冲出两个人，就要往卡车上攀。

司机也是个社会上模爬多年的狠人，驾着东风140卡车就往厂门上撞过去，破烂的铁门立刻从中间分

开来，两条人影如败叶搬从车身旁飘落。

卡车拐上国道，飞快得往上海方向驶去。

-----------------------

事后得知，被卡车带倒的两个人，都是白天的打手。一个摔断了脊椎，还是粉碎性挫伤，估计这辈子

就长期卧床；另一个摔断了胳膊，还比较轻一点。

单总给司机一笔钱，让他把汽车的发动机号和底盘号都凿了去，将车子报废给当地的修配厂。换了车

，将设备连夜拉回上海。

王总气冲冲打来电话，我不知道单总是怎么和他说的，总之这事闹大了。单总也不承认，就说是司机

的个人行为。总之，没有证据，没有目击者。

单总很自然地报案，备案，并把“丢失”的设备损失记在王总头上……

商场里的水，不是一般的深，单总居然因为这个事情，召来了杀身之祸。

----------------------------

我们在星火开发区租了一座2层的小楼，把先后采购来的设备安装调试完毕。一座还凑合的理化实验

室建成了。

但是当我用之前的配方---700PPM的SORA，3-4PPM的左氧氟沙星和1PPM的万古霉素配置的缩小药物后

，只见到小白鼠毒理反应加重，没有缩小现象。

问题出在哪了？

王教授采购了一批中成药，主要成分是云南灵芝孢子成分。
姜华和王教授按照王教授之前的研究开展了一系列药物提纯实验，通过提纯灵芝三萜和矩化反应开发

脑神经新药。当冷冻粉碎法提纯灵芝三萜的效果达到了预期值以后，我们就更忙了---从理论过度到

临床试验。这期间按要打通的关节，就太多太多了。

首先你要去药监局提交新药临床许可。然后临床至少2年，期间的大量报告自不必说。不过有了王教

授的知名度和单总的秘密活动，仅仅1个月，就进入了专家论证阶段。

这13名专家是从全国医院专家库里随机抽出来的，具有一票否决权，所以个个都是大爷。有的人心知

这种发财机会可能一辈子就那么一次，所以格外变本加厉。单总自然不敢怠慢，除了用王教授的名声

来施加压力外，每个人还塞了厚厚一只信封。这些事情，就和出差旅游一样私下里就成了。

------------------------------

莲莲和梅梅带着“旅游团”回来了，西门子成功进入者5家顶级医院，销售额5亿美元。囊括了丰田、

东芝、索尼、三洋等世界巨头的三井财团虽然也侧重做了大量工作，以世界商社经济实力第一的地位

，竟然也没撼动西门子！更别说实力稍弱的第一劝银财团的富士通电子了。

晚上，两位美女请我出去吃饭，顺便秀秀她们在国外买的衣服。

她俩现在是名副其实的富姐了，家财千万，尼桑肖克都不稀罕开了，莲莲干脆买了辆英菲尼迪FX55，

我一看这排量：乖乖，这是喝油的老虎啊，太符合莲莲的性格了……

莲莲在德国给我带了块TAG HAUER的手表，梅梅从英国给我买了一件大波波的大衣。看来，我晚上又

要有勤劳的劳动来报答这两位御姐了！

-------------------------------

莲莲要连夜向大区经理报道，梅梅和我先回来。

一到家，梅梅就坏笑着说：“汤，我在国外吃了好多好吃的，现在……”指了一下性感的翘屁股，咬

过耳朵来说：“就等你继续品尝啦！”

好家伙，今儿一回来就整这么重的口味！

梅梅吃吃笑着，拉着我进入卫生间：“姐姐吃的东西可好呢，特意留着给你吃吃。”

我下面的小小汤忍不住了，我们脱了衣服，在温热的热水淋浴下，互相厮磨着。

梅梅让我仰面在地上，两只脚踏在我的胳膊上，臀部压倒我脸上。我两只手受到限制，却也忍不住去

一亲那肥美的屁屁。

当我的舌头划过她粉嫩的菊门时，她娇哼一声，顺势一屁股重重坐下来，我一阵晕眩，脑袋像要爆开

始的。然而却迫不及待地用舌尖抵进她的菊门。

刚插进去那么一点，她浑身一震，我感觉有那么一块硬硬的东西，呼之欲出！

这是她留给我的外国美食吗？在没被她消化之前，会是什么呢？wagyu beef? tiger prawn?

pamersan salad? 我现在有这个能力进入她的消化道了，不知道我出来时会是什么形式呢~~

那边梅梅已经张嘴套住我的小小汤了，她舔了几下，转过头来笑道：“我们互相交换哦，我只吸你一

点点，但我会有很多好吃的给你吃哦。”

说话间她的菊门一鼓，满满一大肠的黄金倾巢而出，瞬间就铺满了我的脸。

粘稠的黄金混合着滑溜溜的肠液，慢慢流到我脖子上。里面充斥着幼滑的纤维，还有一些种子状的颗

粒物。我望着还在源源不断喷涌而出的黄金，心已经飞到了她的肚子里。

我的缩小药品一旦研制成功，我就可以实现进入她体内的梦想了。我会以什么形式从眼前的菊门里出

来呢？难道是消化的稀烂的肌肉，支离破碎的骨头，残留的毛发吗？

梅梅使劲吸着小小汤，我慢慢地品尝她的大粪。初尝有点腥，里面还有一些软软的流质物。温水将很

多黄金冲掉，我则对准她已经开始收缩的菊门，舔舐着她直肠内部。然后转战她的阴部，攻击G点…

…

梅梅很快就招架不住了，舔吸变成了乱咬。我的小小汤在她牙齿的绞磨下，狠狠地射了。她深深吞入

小小汤，贪婪地吞咽着。

----------------------------------

2007年初，在一场混乱中，单总抢出了他的设备，得罪了王总。又秘密地在上海组织新药的研发和原

材料的采购，这种原材料以灵芝孢子为原材料，现在一类新药准字和批号下来已经是指日可待，关键

还出在这个原材料采购上。如果原材料跟不上，那么所有一切的努力就是浮云里的浮云。

单总早就在着手准备原材料的事情了，这个事情是他亲自来抓的。所谓事必躬亲，一定要学习程天送

的职业劲。所以他亲自去云南，广西和四川去采购药物，联络药商。并投资在云南设立了一个加盟连

锁的公司，传授山民养殖灵芝的技术，让山民投资，然后公司高价收购灵芝成品。

单总带来的致富信息，加上王教授的成果，让山区一些贫困县政府雀跃不已。这真是等天等不到金凤

凰，竟然来了财神。于是单总在这些贫困县地位之高，简直就是钦差大臣一样的待遇。

于是乎，十万农民种灵芝，这在西南一些省份绝对是大手笔。比起当年的虫草，丝毫不减踊跃。

不过，王教授的灵芝三萜是从紫色灵芝里提纯的，这种灵芝只有野生才有，需要3-4年才能成熟，成

熟后颜色接近黑色，其孢子里灵芝三萜含量丰富。而单总却着急走产量，让山民大量种植黑芝，这种

灵芝是紫芝的近亲，但是有效成分几乎是0，但这种黑芝只要养分够，人工一年就能长到盘子大小。

您说这个黑芝就是能糊弄人啊，成熟后的紫芝和黑芝，放一起不拿仪器分析，根本就分不出来。紫芝

只有在幼年期时加以标记，才能区别开。别看这两个灵芝样式差不多，但是区别呢……嗯，基本上就

类比牛奶和牛屎……
