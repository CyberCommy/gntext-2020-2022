# 1

第三次世界大戰後，人類的文明被破壞殆盡，大量的幅射和污染使的人類瀕臨滅絕，只有極少數的倖存者適應了新的環境而生存了下來，並在數百萬年間有了不可思議的演化。男人為了減少對糧食和能源的需求而越變越小；女人為了克服惡劣的生存條件而越變越巨大，同一個物種朝著完全不同的方向演化著。歷史記載，遠古的時代男人有女巨人的姆指這麼大，那時人類過著和諧沒有殺戮的生活。但隨著演化的進行，男人漸漸變得只有指甲的大小，一些不為法律所允許的殘酷事件開始出現。有些奴隸販子將男人抓起來秘密地賣給了女富豪，女富豪介由玩弄和虐待小男人得到快感。更有許多小男人被抓來幫主人塗指甲油、舔指甲縫的汙垢，工作完成後就被踩扁。食用這些小男人也是女富豪們的娛樂之一，他們常常聚在一起，一人帶一罐小人(約四五十隻)，有些人帶中國的小人，有些人帶歐洲的，交換品嚐討論味道。

「小雅啊~我今天帶了台北的小人要不要吃吃看啊，剛抓的很新鮮喔」

女巨人從袋中拿出那罐裝滿小人的瓶子，裡面發出尖叫與吶喊聲。

「台北人看起來又白又嫩的，一定很好吃」

於是女巨人打開蓋子，用姆指和食指伸進罐中，長長的指甲挾起一隻小人，小人馬上勃起精液狂射，同時又不停得尖叫掙扎，被女巨人放入嘴中齟嚼，成為討論的話題，他一定感到相當榮幸吧！

一些秘密的地下餐廳，提供了許多用小人料理的餐點，全都是精心調理設計，保證好吃又殘酷。比方說將小男人的肚子用指甲剖開，拉出腸子做成的麵、將陰莖劃十字翻開的拼盤、用竹籤從屁眼穿到嘴巴的小人串燒或是小人燒烤等，都是當代經典之作。(待續)
