# 【小囧】精灵宝可梦同人：小绿用大师球收了男训练师

作者：lyh971722424

# 1

大家好，我是小囧，活跃在gts圈内群多年，也在论坛潜水四年了，一直想为论坛做些贡献。前不久忙于某些大事，没有时间，现在空闲了下来，终于可以着手为论坛做一些事情了。本文翻译并改写英语原版跨圈福利文《green》，我自己改写的内容特别多，可能更适合gts圈内人的口味。这个帖子是小囧在论坛的处女帖，因英语水平和写作能力有限，文中不当之处，希望各位大佬多多包涵。

正文：

精灵宝可梦出现以来，人们的生活出现了很大的变化，越来越多的人，特别是年轻人，选择训练师作为他们的职业。然而据说在一个不起眼的平和的小镇，可能也就是我们常说的新手村，出现了一个十分稀有的宝可梦——曦鸹，使这个小镇变得举世瞩目。虽然官员们试图消除“谣言”，但毕竟纸包不住火，引来全世界众多训练师欣然前往这个小镇，大家都想捉到曦鸹，因为这可能是最厉害的宝可梦之一。

曦鸹的常出没地点在冠军之洞，这个洞穴离这个小镇不远，但只允许具有各种冠军头衔的优秀训练师进入，因为这个洞穴内的环境十分复杂，并不适合没有经验的训练师，而且曦鸹很强，训练师如果处理不好可能会连命都丢了。小绿就是优秀训练师的一员，她有着黑色的长发，头戴绿色某水果图案发卡，身穿日本高中的jk制服，身高虽然只有155cm，但是她是一位非常强的训练师。她一路击败了很多想捕捉曦鸹的训练师，击败他们时，她露出可爱不失帅气的微笑，继续前行，去往冠军之洞。

洞穴里一片漆黑，小绿只能让她的宝可梦放出【照明术】，借助微弱的灯光向前快速前进。当她在洞穴里盲目奔跑寻找曦鸹时，突然踩到一个富有弹性的物体，她以为是踩到了什么恶心的物体，又使劲剁了两脚。她脚下传出一阵惨叫，小绿赶忙向脚下看，原来是一个受伤的年轻男训练师，他痛苦地躺在地上，脸上深深地刻着小绿的鞋印。小绿“啊”地叫了出来，慌忙给她脚下的男训练师道歉，从他的脸上挪开了脚，并试图扶起他。男训练师挣扎地坐起来，小绿对他说一定要多加小心，要在曦鸹发现他之前尽快离开洞穴。男训练师微笑着说，不用担心曦鸹了，因为他已经成功捉到它了。话音未落，小绿立即放开了男训练师的手，男训练师因为跟曦鸹苦战，身体非常虚弱，应声瘫倒在地上。小绿露出迷人的微笑，请求男训练师，把曦鸹让给她，她会十分开心。但男训练师怎么可能为了一个可爱的妹子就放弃了他费劲气力捉到的曦鸹，于是，男训练师婉拒了她。小绿沉默了一会，脸上的表情变得十分可怕，说：“哥哥，我已经给过你机会了，可你没有把握住，我最后再问你一次，你可不可以把曦鸹让给我。”男训练师坚定地拒绝了她。

小绿从她的鞋子中抽出她穿着可爱的白色过膝袜的足，由于小绿进洞穴中跑了很久，足部出了不少汗，袜底渗出淡淡的黄色，热腾腾的蒸汽被宝可梦的照明术映在男训练师脸上，女孩子足部特有的气味充斥着男训练师的鼻腔，正所谓【雾失楼台，月迷津渡，桃源望断无寻处。】，可男训练师并不M，他依然坚定地说，绝对不把曦鸹让给小绿。小绿见他不服，一脚狠狠地踏在他的脸上，接着把另一只鞋子也脱了，把她的全部体重压在男训练师上。男训练师身体十分虚弱，已无反抗之力，就算有点反抗，也是在咸湿的足下不断挣扎和被狠狠踩下去的不断循环，但他还是颤抖地说绝对不把曦鸹让给她，还说等他出了洞穴，一定要把小绿的暴行向全世界公布。

小绿听到这句话，怒不可遏，拿出了真·大师球，这是她在以前帮助怪博士获得的世界唯一能捕捉所有生物并使他们完全听持球者的话的【精灵球】，而且被这种球捉到的生物的大小和技能也是可以根据主人的意愿决定的，这种球是秘密开发的，因为涉及到伦理问题，被紧急叫停。男训练师并不知道小绿手中的球的来头，正在男训练师嘲笑她之时，她对着男训练师说，如果哥哥不把曦鸹让给我，那么没办法了，哥哥只能做我的奴隶了。说罢，小绿朝着男训练师将球扔去，一阵白光照亮了洞穴，洞穴的墙壁上映着男训练师惊恐地脸，几秒之后，一米八几的男训练师消失了，只留下一个白色的真·大师球。小绿将男训练师放出来，将他变成18cm左右的大小。男训练师惊恐地说，你对我做了什么。小绿笑了笑，“我给过哥哥机会，这是哥哥自己选择做我的奴隶的。”

小绿慢慢脱下她的白色过膝袜，对男训练师轻轻说，“舔”。男训练师的身体根本不受他自己的控制，瞬间钻进小绿的脚趾间，大口大口地舔起来。所谓逃不过【真香定律】，也只不过是在斗争中实践选择服从对生活的妥协的表现而已。小绿对他说：“以后，你就是我的特殊的宝可梦了，不过你不是用来进行战斗的，而是用来满足我的需求的，所以你的技能不是普通宝可梦的技能，而是服侍我的技能，听明白了吗？”男训练师频频点头，小绿透过脚趾看着他，露出迷人的微笑。“我命令你学习一项新的技能，【可爱之吻】，你目前的任务就是保持主人的足部清洁，【可爱之吻】是一个像舌吻一样舔主人的足部的技能，无限技能点数，现在一直使用【可爱之吻】技能。”话音未落，小绿将男训练师放进她的白色过膝袜里，穿上了过膝袜，又将男训练师缩小了一些，在她的足底，感受到了男训练师的存在。“真是好孩子，在好好地清洁呢”，小绿动了动脚趾，以示鼓励。被真·大师球封印的宝可梦有不死之身，所以不必担心被踩死，可能在小绿走路过程中有可能会被踩扁，但也可以立即复原。

小绿拿起男训练师原来的背包，把他的精灵全部上传到精灵系统电脑中，当然包含曦鸹。她在冠军之洞收获颇丰，有了曦鸹还有了一个可以服侍她的小奴隶，满意地离开了洞穴。
