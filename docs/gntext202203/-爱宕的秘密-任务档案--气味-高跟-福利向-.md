# 1

“哈啊～～累死了啊。”

经过了一上午的工作，爱宕终于返回港区。炎热的夏天，大大提高了舰娘们的体力消耗速度，仅仅才到中午，爱宕就已浑身被汗水打湿，瘫倒在房间里的她，已经完全不想再将身体挪动一尺了。

夏天，一直是舰娘们的大敌。

几乎没有舰娘会喜欢在夏天这个季节执行任务，不论是追着船只跑的护卫任务，或是镇守海域的防卫任务，甚至是要去攻城略池的攻击任务。

不管是哪一种，对于温度感知十分敏感的舰娘来说，都是如临大敌一样的存在。

大口喘着气的爱宕，将身上装载的舰装卸下，敞开战斗服，任凭凉风吹进自己的身体。

顿时，整个房间都充斥着一股神秘的腥味，那是她的香汗随着空气流动，蒸发到整个房间的奇特现象。

为了优秀的防御性能，像爱宕这样的重型舰娘，都会搭载厚实到几乎无法透气的战斗服。

更可怕的是，也许是由于之前有过类似的案例，在爱宕这一代之后的舰娘，穿着的战斗服全都是将全身上下都裹住的紧密服装。

可惜的是，还没等爱宕得到充足的休息，指挥部又一次指派了新的任务。

要是像上午一样的护卫任务，那就再好不过了，只需要追着一艘货船跑个几小时，在这过程中，清凉的海风还会带走身上的燥热。

况且，一旦遇到想要接近的不明舰艇，不论对方是谁，拥有绝对处置权的爱宕拥有会不由分说地将其直接击沉的权利，而她一般也会那么做。一方面是因为经过了数次改造的爱宕，已经拥有在港区中数一数二的实力了，而另一方面，则是由于夏天带来的燥热，让她无法平复心情，只好找些什么发泄。

但可惜的是，那样简单的任务，一般不会在下午出现。

果不其然，当看到任务内容的时候，爱宕的心都为之一振。脑中冒出的第一个想法，便是想要将其无视，躺在床上度过一个下午，等到晚上再匆忙地前去将其完成。

原本以为，这样天衣无缝的计划，能够一直持续到夏天结束。但当爱宕看到任务期限的时候，她的表情立即从得意变为了失望，随即又变成了绝望。

“啊啊啊啊——！！”

明明海洋是如此广阔，天空是如此蔚蓝，却让人完全静不下心来。爱宕一边朝着任务地点缓慢驶去，一边怒吼着宣泄心中的不满。

这一次，又是敌方将原本中立的城市占领，随后将城市中的许多设施都转为了军用。原本，这驻扎着大量敌军的都市，应该是由复数位舰娘组成的编队来执行任务。但奇怪的是，由于上次的大事件，爱宕已经成为了港区赫赫有名的神级舰娘，崭露头角的她，自然被安排了更多的任务。

以一人之力，灭掉由数量不等的航母、重巡、驱逐、潜艇构成的庞大编队。不论怎么说都有些太过魔幻，甚至让人怀疑是否有夸大的成分。然而，这样的事情，就发生在了爱宕的身上。

“哈啊，下次还是故意失败比较好吗。”

原先搭载的三门重型火炮，爱宕全都将其卸下，放在了港区的宿舍里。带着这些笨重的武器，只会徒增她的体重，想要尽可能保持轻盈身体以便散热的爱宕，几乎卸下了所有的舰装，只留下侦测用的雷达以及两门超轻型的火炮。

大部分时间，爱宕都是在海上作战，因而那些能一下击溃一艘航母的重型火炮得以派上用场。至于陆地作战，情况就大有不同了。

那些重型火炮射出的弹药，一发便能让一个街道变为火海，只需十发就能毁掉一座城市。这次的任务，只是单单镇压这些叛乱的军队，与消灭城市本体毫无关联，因此爱宕会选择抛弃那些舰装，也不是毫无理由的。

“啧，被发现了吗。”

还未靠近陆地，周围戒备着的几艘轻型巡洋舰上，就冒出了火光。从爱宕的视角看，这些在海上漂浮着的舰船，可能还不如她的脚趾头来的大。有着接近两千米的傲人身高，爱宕光是体型上，就足以碾压一众舰娘了。

至于各方面都远不如舰娘的普通军舰，就更不值得一提了。如果不是这些巡洋舰自寻死路，跑到爱宕的脚边，她也许都不会理会这些只有黄豆大小的船只。

沐浴了一轮流弹的攻击，又挨了十几下舰艇重炮的打击，如此猛烈的进攻，理应能让世界上任何一艘舰船遭受重创，甚至当场沉底。但是，这些疯狂的火力，打在身上却毫无感觉，甚至就连大口径的炮弹也无法击穿爱宕腿上的轻纱。

至于为何人们不攻击爱宕腿部以上的部位，真实原因则让人不禁发笑了。那些小人可能都没有认识到此时的爱宕有多么巨大，就自顾自地将自己那渺小的舰艇开向她的方位。直到那裹着轻纱的脚掌落在他们周围的时候，那激起数十米高的海浪，才激起了他们仓促发动攻击的想法。奈何舰艇上的火炮，最高也只能拉到六十度的迎角，这也正是爱宕膝盖的高度而已。

“烦人。”

原本漂浮在海面的脚掌，随着爱宕的一声抱怨，猛地从海中升起。人们这才真正看清，那将婀娜的玉足整个包住的，原来是一双精致无比的水晶高跟鞋。

虽然外貌与普通高跟毫无差别，但整个鞋面，包括鞋跟鞋底的材质，均是由湛蓝的水晶所制成。可以看出，爱宕似乎十分爱惜这双高跟，就连鞋面都擦拭得一尘不染，在阳光下还隐约反射着太阳的光芒。

被脚掌覆盖的那艘舰船，上方的人们顿时陷入了呆滞，就像是魂魄都被爱宕那陡峭的鞋跟勾走一样。他们目不转睛地看着那遮天蔽日的玉足缓缓移动到头顶，将天空都染成鞋底的颜色。正当他们还在欣赏着这唯美的玉足时，爱宕却忽然压下脚掌，结束了他们悲惨的生命。

另一只脚也以同样的方式，将那艘试图逃跑的军舰同样踩碎。

收走了千人左右的性命，爱宕依旧一如往常，用散漫的姿势朝着地平线尽头的陆地移动着。然而，当两艘在外海部署的军舰被她一脚踩碎之后，这个消息立马就传到了陆地那边的指挥部。在爱宕的身影还未从海雾中出现时，人们便已经得知了她即将袭击陆地的情报。

然而，即便是以如此散漫的姿势前进，爱宕的速度依旧超乎了陆地上人们的想象。仅用了不到一分钟，横跨整个海域的爱宕，便来到了距离岸边仅千米左右的距离。

调动任何军舰，都为时已晚了。人们看着步步逼近的爱宕，只好选择让陆上的军队参战。

从一开始，爱宕就没有打算从后方给这里的人们来一个突然袭击。她径直走向城市的港口，决定在那里登上陆地。看着那自海雾中出现，逐渐变得清晰的身影，人们的目光顷刻间全都转向爱宕，被这接近万双眼睛盯着的爱宕，也感受到了背后的一股凉意。

不过，首先吸引人们的，并不是那让人胆寒的恐怖大小，而是爱宕完美的身材。

两座高高凸起的雪峰，就这样挂在爱宕的胸前，随着她摆动着双手，那对爆乳也开始摇晃起来，形成了壮观的乳摇。紧贴着爱宕身体线条的战斗服，设计的初衷似乎就是为了将她身材的完美体现得淋漓尽致。

如果说只是单纯的爆乳，那对人们的杀伤力可能还没有那么夸张。爱宕的全身，似乎都是为了衬托她的爆乳而特殊设计的。先是那毫无赘肉的小腹，让胸部的大小顿时提高了好几个尺码。

转到背后，那高傲的丰腴臀部便再次吸引人们的目光。似乎是要与前面突起的巨相对应，这丰满的翘臀，仅有一层薄薄的黑色裤袜遮挡，透过交错的尼龙纤维，还能隐约看到覆盖着臀部的那一层雪白肌肤。

岸边的火力，毫不留情地洒在了爱宕的身上。

然而，就连重炮都没能给她造成一点损伤，这些与之相比威力小到可怜的子弹怎么可能会派上用场。

“咚——！！！”

只见爱宕高高抬起脚掌，越过岸边的提防，朝着陆地落下了第一脚。

试图阻挡她的小人被巨大的脚掌直接砸碎，就连血肉都被拖平。这径直朝着人群落下的高跟，在地上碾出了一个脚印一样的坑洞，至于刚刚还站在那里的小人，则早已随着破裂的大地，与地面融为了一体。

爱宕的第一个目标，便是坐落在港口附近的某个军用机场。然而，由于情报不够详细，爱宕也无法判断眼前这两个机场，究竟哪一个才是任务的对象。

既然不能判断，那就只能全部将其毁灭了。

从港口到机场，还有一段距离，中间分布着高低不一的老式居民楼。虽然建造时间已然久远，但直到现在里面还居住着不少的小人。爱宕毫不在乎，就这样一脚踩了进去，碾碎无数楼房。

正值中午，里面许多小人还在享受安详的午觉，吹着和谐的海风，忽然之间，爱宕的巨足从天而降，将整个楼房自上而下地碾压过去，当场变为一个平面的楼房，直接将里面的小人活生生地夹死了。

在留下两个脚印，夺走数千条性命之后，爱宕又把轻炮对准居民区，随后无情地扣动了扳机。

矗立地面的数千楼房，一瞬间就被蔓延的火焰所吞噬。尽管屠杀平民是不被允许的，但像爱宕这样的舰娘，只需要说是以任务为目的，一般都不会有人深究。而她也并非刻意想要把那些小人烧死，只是因为自己路过的时候，不慎踩碎了他们的楼房，底下骚动的人群就开始不停地尖叫，为了和港区保持联络而不得不开启通讯设备的爱宕，就这样受着下方惨叫与哀嚎的折磨，终于忍受不住朝着人群开火，这样看起来，也并非没有道理。

抵达机场的时候，那里已经聚集起了妄图攻击她的人群。

爱宕再次笃信自己的选择，能够聚集起装甲车、坦克之流的机场，毫无疑问属于军方。但奈何即便是坦克的穿甲弹，也无法击穿爱宕的裤袜，爱宕随即将自己的炮口对准地面，打算一击解决这些无礼的小人。然而，她忽然想到，假如让整个机场变为一片火海，原本就燥热的气温，岂不是又要升高不少。这种自掘坟墓的想法，爱宕立马将其抛弃了。

仔细一看，地面上这声势浩大的陆军，实际上也只是由寥寥几辆坦克，为数不多的装甲车，大部分都是由穿着军装的小人组成的编队。

爱宕抬起脚掌，下方的小人随即开火。可惜的是，爱宕十分疼爱的水晶高跟，自然也是用上了世界上最为珍贵的宝石制作而成。这些子弹打在上面，连留下痕迹都做不到。在落地之后，爱宕再度拖动脚掌，碾过人群，压扁一切试图阻挡她的愚蠢事物。

就这样轻轻一抹，那些小人就变为了地上的一滩血渍。爱宕冷笑一声，再度抬脚，将地上溃散的坦克与装甲车，连同那些小人的血肉一同碾入地底。此处，成为了埋葬千位小人的乱葬岗。

碾过机场的铁丝网，爱宕踏上数千米长的跑道。在爱宕进入机场的一瞬间，里面的人群立即骚动起来。就连爱宕本人也觉着有些奇怪，明明是军用机场，为何会有如此多的平民。人满为患的航站楼，匆忙起飞的客机，这一切都不像是一个军用机场的模样。

“嗯，只是错觉。”

既然这里有着人类的存在，爱宕就必须一个不留地将它们全都碾死。

“轰——！！”

漂浮着的轻炮，对准了朝着空中的客机，一阵炮响之后，空中出现了一个巨大的火球。随后，爱宕又接连瞄准几架飞机，从地面到天空，无一幸免。

在解决完这些试图逃走的小人后，爱宕又把目光瞄准了宏伟的机场本体——航站楼。

宛如一把利剑，陡峭的鞋跟径直落向那占地巨大的航站楼顶端。只不过，此刻里面的人们，都被爱宕的美貌与优雅的动作所吸引，全然忘了悬浮在头顶的鞋跟。轻轻施加压力，由坚固混凝土铸成的航站楼，便自上而下地被整个贯穿。至于恰好在鞋跟附近的小人，则随着碎裂的地板一同被碾死了。

接连落下几次脚掌，原先看着十分庞大的航站楼也变为了爱宕脚印的一部分。有如屠杀机器的她，在机场中大肆破坏，收走无数生命。

此时，她的炮口正朝向天空，对准千米之外的另一座机场。连续几发炮响之后，经过精准计算的燃烧弹，直接落入了机场的内部。这样一来，两个目标都在同一时间被摧毁了。

被脚掌蹂躏过后的航站楼，尽管已经变为了一片废墟，但实际上里面还有少数的幸存者。他们也许还抱着，在爱宕走后能够有人前来支援，把她们救出的幻想吧。

然而，用雷达侦测到废墟中的生命气息，爱宕又一次将炮口对准了地面。

“轰——！！”

在一片火海之中，心情舒畅了不少的爱宕，缓缓朝着下一个目标的方向移动。

但是，余光扫过跑道之时，爱宕发现了一架搭载了至少千位小人的巨型客机。原来，为了躲避爱宕的攻击，这架方才没有出现的客机，一直藏匿于机库之中，这才躲过了她的肉眼扫描。可在爱宕搭载的最为先进的SG实验型雷达上，他们的踪迹便一览无余了。

“来得正好，呵呵。”

爱宕大跨步地走向那架即将起飞的客机，见到已经走远的爱宕再度返回，那些小人顿时慌乱起来，以最大功率运行的四台发动机，与爱宕进行着关乎生死的赛跑。

机舱里瞬间被欢呼声所占据，看到逐渐离地的机身，终于逃过一劫的人们，开始相拥庆祝。而为了拉开与爱宕的距离，在离地之后，飞机依旧以满功率运行，速度甚至可以与音速媲美。

无论怎么看，爱宕都不可能追上这架飞机了吧。

“嘿呀，抓到了～”

然而，只是一个突然的加速，整架飞机便被爱宕用手掌裹住了。现代工业最为先进的合金，都阻挡不了爱宕手掌的重压，当场变形断裂。至于机翼，则更是凄惨，在与爱宕发生接触的一瞬间便发生不可逆转的形变。最终，躺在爱宕手里的，除了一大群奄奄一息的小人之外，就是一堆已经扭曲到看不出原来形状的废铁。

原本想要就这样把整架飞机捏碎，但正当爱宕试探性地握紧手掌，做好撰碎这架巨无霸客机的准备时，她却感到了一丝凉意。

“嗯～”

突然，一个绝妙的想法浮现眼前。

缓缓地、爱宕逐个解开胸前的衣扣，胸前那对完全称得上是「爆乳」的雪峰，在解除了衣物的束缚之后，立即像皮球一样弹了出来。

白皙无瑕的皮肤，透着脂肪特有的细腻质感。因重力稍稍形变的爆乳，让人无不垂涎欲滴，就连手掌上的小人，也顿时忘却了恐惧，全都目不转睛地盯着窗外摇晃的爆乳。

“呵呵～”

爱宕的嘴角，勾起微妙的弧度。

稍稍扭动身子，那对爆乳便将最后一枚纽扣挤爆。不过，人们想象中那种胸部直接从衣服中蹦出的场景，实际上并没有发生。看着那已然毫无遮拦，完全显现的乳房，客机中的人们顿时像停止了呼吸一样，无不感叹着这对爆乳的唯美。

翘起一根玉指，爱宕纤细的手指，拨弄着这对乳鸽。随后，伸入乳沟的手指，将被汗液紧紧黏在一起的双乳逐渐拨开。

蒸腾的热气，带着无比浓烈的汗腥味，散发到空气中。这股扑面而来的臭味，直冲人群。顷刻间，所有人的肺部都被来自爱宕身体的汗臭充满。

拖动着脚掌，把废墟都一同抹平的爱宕，用审视的目光看着两栋大致相同的高楼。为了防止下方还埋藏着隐患，爱宕特意确认了一次结构图，随后抬起脚趾，在两栋大楼的上方来回晃动着。

就连多看一眼那些小人脸上绝望的表情都没有，爱宕拍下巨足，连带着周围的街区也一同被抹去了踪迹。

配合着搭载的分析模块，爱宕再次扫描了整个城市，尽管外围区域已经被彻底踏碎。但综合分析下来，整座城市的破坏率还不到一半。

有些恼火的爱宕，将炮口对准发电厂，装填上最大威力的子母炸弹。一瞬间，整个城市都被火光与轰鸣所掩盖，超过百分之三十的城市因此被炸毁。就连实战中都少有使用的终极武器，仅仅为了满足爱宕的报复心理，就被她搬上台面。当然，此时已经没有人能够责备她了。

又走了几步，不远处另一座发电厂也惨遭子母炸弹的袭击。

由于周围已经被轰炸一空，虽然规模等同于上次爆炸，但也仅造成了千人左右的伤亡——而就在刚刚，这个数字是现在的一百倍。还没等人们反应过来，第三发炮弹就又打在了地面上，但这次，是落在了人口极端稠密的居民区。

正当爱宕打算射出第四发的时候，她余光瞥了下城市，惊愕地发现身后竟然已经变为了一片火海，而城市的破坏率也骤然升到了百分之八十。如果再射出这一发，那么一整个城市都将化为焦炭。

用指甲划过大地，割出一条深数十米的峡谷。诺大的城市核心区，顿时成了一座孤岛，等待着爱宕审判他们的命运。

“轰隆隆隆隆隆——！！！”

看着徐徐升起，最后在空中“砰”地炸开的烟花，爱宕觉得自己似乎也没那么讨厌夏天了。然而，仅仅是一座城市的毁灭，对她来说太不过瘾了。

爱宕干脆脱下战斗服，赤裸着身子躺在城市上。身边漂浮着的炮艇对准天空，时而发出火光，半径十公里之内的战机，无一例外均被她击落。

被压在身下的建筑，应该有百栋之多吧。

爱宕毫不在乎身边小人的挣扎，只是一心享受着这只属于她一人的日光浴。假如地面派遣了武装部队，只需偶尔扭动身体，重心的转移便会让身体周围的地块开裂，吞没试图攻击她的坦克与装甲。

“嗯，各位都是我珍贵的玩具呀。所以，请变成我的一部分吧～～”

随手抓起远方的几栋大楼，爱宕挑开两瓣涂着口红的嘴唇，向指间的小人们展示自己的口腔。随后，在某一个时刻，爱宕的双指悄然松开。

甚至没有在嘴中停留一秒，那些大楼就直接落进了爱宕的食道中。不过，这也意味着他们要承受比在嘴中就被消化的小人们多出无数倍的痛苦。因为，爱宕的胃酸，就连地球本身都能溶解，这些大楼光是碰到胃酸的蒸汽，估计就要瞬间蒸发了吧。

已经不仅仅是为了进食，而是为了戏弄这些小人。爱宕偶尔将大楼聚集在一块，随后一个个将其挑起，扔到空中，张开嘴唇接住他们。偶有失误，则那些大楼会落在爱宕身体上的某处，亦或是直接砸回城市，造成二次破坏。

其中，当属不幸落到爱宕唇上的小人最为悲惨了。

由于在大楼被抛起的过程中没有抓稳，这些被抛弃的小人，径直落向了爱宕的红唇。那妖娆的嘴唇，甚是令人垂涎，但当人们真正意识到她的大小时，却连惊讶都来不及，就被唇上附着的唾液黏住了身体。

结果显而易见，那些小人在爱宕伟岸的红唇上挣扎着，想要逃离唾液的束缚。但最终，落到唇上的足有十三位小人，他们中没有一人活过了十秒。

捕食游戏玩腻之后，爱宕又摇晃了一下胸前的爆乳。随后，一大把的土地被她攥在手中，稳稳当当地将惊恐之中的他们放置在平滑的乳沟中。

而且，为了寻求刺激，爱宕特地挑选了一个存放着军火的仓库。在双乳的重压下，一切都被平面化，那满是危险品的仓库自然也不例外。一阵火光，伴随而来的是一股热意，回过神来的时候，乳沟之间已经荡然无余，就连火焰也被夹紧的爆乳扑灭了。

“呀，你们应该不介意我翻个身吧？”

爱宕的语气中，充满了戏谑与轻蔑。

随后，她的身体开始翻转，在大地上任意地滚动，碾平下方的街道楼房。当然，由于胸部实在太大，以至于小腹与胸部之间还会留下一小块无法碾过的区域。爱宕特意拖动黑色的长发，让万根千米之长的发丝碾过此处。坚韧的秀发，在地上碾动着，让地面深深烙印上了发丝的形状。

“好，任务完成咯～”

当破坏率到达八十之时，看着几乎变为废墟的城市，就算想要接着玩弄，由于楼房与楼房，街道与街道之间距离太远，不仅没有什么连续感，还有可能白白浪费时间。

撑起一千六百米的骄傲身躯，爱宕的三门主炮，一门轻炮，一齐瞄准了这座城市。 直到世界上最后一座城市消失之时，也许爱宕才会停止她的破坏吧。
