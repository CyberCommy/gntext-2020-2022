# 变（第一章：变狗1)

# 1

接着，我跟她来到不远处的租住的房子里.‘这里是我前两天借的房子，就是为了

方便了解我们的目标。’小静说着打开了门，我跟进去看了下，虽然地方不大，但

是被她布置得很温馨，一看就是女孩子的房间。虽然她是搞科学研究的，而且似乎

取得了不小成果，但是她并不象什么科学疯子不注意个人形象，更没有不注意周围

的卫生整洁，所以即使是她暂时借的房子，但还是一尘不染。

只见她拿出了支针剂说，‘去厕所把衣服裤子都脱后把手臂从门缝里伸出来.'看来

她是算准了我会答应的，东西都已经准备好了。不过还好她没让我当着她面脱，不

然还真不好意思。走进厕所我转身关上了门，接着把衣服脱完后就把整个左手手臂

伸了出去，忽然感觉手臂臂弯处被扎了一下，接着感觉体内一阵燥热，人好象开始

都站不直了，只能趴下，同时手臂和腿都开始缩短，身上迅速得长出了厚谋的毛。

看到厕所里有面镜子，看着自己变化的样子，就和《西游记》里妖怪被打回原形那

样。当完全变好后，发现镜子中的自己就和早晨看到的那条狗完全一模一样，就这

么趴在了地上，我尝试着用四肢站起来，开始还不着么习惯，但走几步后就好了。

当我还想再适应一下的时候，门忽然开了，接着一双白嫩的穿着拖鞋的脚出现在我

的面前.

‘你快出去，我要上厕所了！’小静的声音从上面传来。大概是狗的耳朵比较好吧

，我觉得声音比平时响了好多。

听到她的话，我就向前走去，当我刚走过她旁边时，忽然一阵风吹起了她的裙角，

竟然把我的头给罩进去了，周围顿时暗了下来。我本能的一回头去看她，结果看到

在我面前不到一厘米的地方是她那雪白丰润的臀部，被一条粉红色的内裤紧紧的包

着，随着呼吸还能清晰的闻到她那特殊部位散发出来的味道。也许是变成狗后嗅觉

变强了？当我还在想的时候，听到上面传来声‘啊呀’，随着她把裙角从我头上拉

了下来，周围又恢复了明亮。

‘呵呵，不好意思呀。’我想说，但嘴里出来的是‘汪汪’的几声叫声。

她看了看我后说’快出去吧’。于是我跑了出去，身后传来了关门声。跑到客厅我

跳到沙发上坐下等她出来。不一会我竟然听到了她小便冲击到马桶内壁发出的声音

，也许是因为变成狗的原因？这不禁使我联想翩翩。没多久声音小了下来，又过了

大概几分钟，只见她拿了包东西出来，透过袋子看到里面是我的衣服裤子什么的。

她把我的那包衣服放到了一个柜子里，转过头来对我说‘你的衣服我先放这里了啊

，你记住了哦。’

'哦‘我说，但嘴里出来的还是’汪‘的那声。

她放完衣服后就坐到我对面的椅子上，说’你先在这里和我一起呆一段时间.等我

找到机会了就把你弄进她们家去。‘

什么？现在还没机会？那这么早把我变成狗干什么呀？我又汪汪叫了两声。

’别吵！等你住进去后找到什么证据如果晚上出来不方便的话白天可以放在门口的

信箱里。为了安全，我只会在每个星期六晚上把研究所那个复原的设备拿到这里。

所以如果你想复原的话只有星期六才行。'接着她又和我说了会话，但看我回应的

只能是‘汪汪’声，也没兴趣和我说话了。

‘我先去做午饭了，你自己在这呆着吧。’说完她向厨房走去。想起我昨天玩网游

到凌晨，今天一早又忙到现在，于是趴在沙发上想睡会。

当我睡得迷迷糊糊的时候，感觉有人拍了拍我的头，接着听到‘别睡了，起来吃饭

了。’睁开眼睛，看到小静已经在桌上放满了菜。我跳下沙发，跳上了餐桌边的椅

子上。

‘啊呀，你着么把沙发弄得这么脏呀。你看上面都是你的毛!'她刚说完，我想去拿

面前的饭碗，没发现伸出来的是前爪，结果又把饭碗给砸了.

'你。。。算了，看样子你也不适合在桌上吃饭了。我去拿个盆以后你就在桌子下

吃算了。还有，你身上毛这么多，以后别上椅子沙发了，客厅也干净，前委屈你一

下，你睡觉就睡厕所里吧。‘说完她把地上的碎片收拾了后从厨房里拿出个装了点

菜和饭的盆放到桌子边的地上，不顾我的反对，自顾自的坐回去吃饭了。没办法，

我只能低下头去吃地上盆子里的饭菜了。吃着吃着，偶然间抬头看到小静裙子下一

双秀长白嫩的腿就在我面前，不时的还会晃两下。再往上看只能看到她的小腹，再

上面就被桌子挡住了。此时我发现一早到现在还没吃过，也饿了。不管着么着，开

吃吧。吃完后我只能趴在客厅的地上和她一起看了会电视。到大概十点左右，她说

‘我要睡觉了。你也去厕所睡觉吧。’还真让我睡厕所了呀。没办法，只能老实的

跑到厕所睡觉去了。早晨还在睡梦中的时候她把我拍醒，说早晨了她要上厕所洗涑

了，让我出去一下。没多久她就从厕所出来，然后去客厅拿了包就出门了,我则只

能在屋里无聊的呆一天。就这么一天天过去了，我发现她随着时间的推移竟然一点

点不把我当人看而是当狗了。比如一开始早晨她进厕所是用手拍我的脑袋，到后来

变成用脚轻轻的踢几下了。直到一天晚上我当时就在马桶边趴着，忽然只见厕所门

开了，她冲进来后并没有叫我出去，而是直接关上门就连忙撩起裙子褪下内裤后坐

到了马桶上。没多久就一股恶臭传来，好象是拉肚子了。此时我到是想出去，但是

厕所的门是往里面开的，我变成狗后只要门关上我自己是出不去的。没办法只能跑

到她面前想让她帮忙，此时她还坐在马桶上，两条雪白的大腿一展无余，继续往里

看到大腿的交汇处是那浓密的黑色毛毛。此时她看到我看着，竟然没有一点羞涩，

而是竟然伸出手摸了摸我的头。这是。。。真的把我当狗了？

从那天后她上厕所也再也没有叫我出去了。

而有一天晚上向往常一样吃着晚饭。吃着吃着，忽然从上面掉下来块骨头，接着她

低下头来看着我‘不好意思，筷子没夹牢。吃了一半掉下来了。上面还有好多肉呢

，你。。。吃吗？别浪费了哦。’接着她也不把头抬上去，就这么看着我。没办法

，只能伸过头去先伸出舌头轻轻舔了一下，感觉上面好象还有点她口水的味道，不

过肉是挺多的，就张开嘴吃了起来。‘呀，你还真吃了呀，呵。’说着她又回到桌

上继续吃饭了。此时我真想去咬她一口，但想想万一真把她咬伤了她不给我恢复了

可就完了。没办法只能忍。没多久又听到她在上面说‘既然你不介意，那这有几块

骨头你也吃了吧。’接着只听几声上面掉下来不少骨头，有几块还滚远了。然后她

竟然伸出脚把几块滚远的骨头踢到了我面前.'快吃吧‘她说道。我只能把她扔下来

的那几块骨头一起吃了。

吃完晚饭她洗完碗后回到客厅开了电视。此时她忽然说’前面给你吃了这么多好吃

的，你是不是也应该表示一下呀。‘我趴在地上抬起头疑惑的看着她，她把脚从鞋

中伸了出来说’给我舔舔脚着么样？我都累了一天了。‘说完她往沙发后一靠闭上

了眼睛等着我，好象看死我会去舔的一样。此时我在想她会不会就这么一直把我当

条狗养在她身边呀，不会忘记我曾经是个人了吧，不会忘了我们的任务了？但没办

法，都给她看死了。我现在跑出去没人相信我是个人，也只有她能帮我恢复，我真

有点怀疑她是不是骗我故意把我变成她的狗的。我跑到她脚边，先把头凑到她脚边

闻了闻，还好刚洗好澡味道不重，可能是因为狗鼻子的关系，有点略微的酸臭味，

我想如果我现在是人的话是绝对闻不出来的。看着面前那五根可爱的脚趾，不时的

还动两下，也不觉得着么恶心讨厌了。于是伸出了舌头舔了下，味道微酸略咸，还

能忍受。于是一下一下开始舔了起来。她也配合着，脚趾一个个的分开，于是我舔

遍了每个脚趾缝，接着是脚底。不时的听到上面她传来舒服的呻吟声，好象很享受

呀。舔完了一只后又是另一只。

从此，我就完全过上了狗的生活。她下班回家后用嘴去帮她叼着拖鞋跑到她面前，

再把她换下的鞋子叼回鞋架。晚上在她桌底吃饭，不时的她还会用脚踢我两下，扔

下几块骨头。饭后还要帮她舔脚。白天则是在家被关一天，吃她前一天吃剩下的东
