# 1

問我會否再寫下去? 也許吧... 因為這個是惡搞作,所以需要特定的時間才能發出來. 原本應在昨天發的,但又沒時間...

(看下去就知道我說甚麼了)

-----------------------------------------------------

今天是一個普通的日子. 她如常在特定的時間起床,然後進行例行的工作. 吃完早餐後,她如常在出門上班時用一小時上網看最喜愛的內容 - GTS.

在多年前她便有了GTS這個興趣. GTS擁有超越常理的力量,可以為所欲為,用以前的那些"同類"來滿足她. 只從那時,她便每天上班前抽一小時來觀看GTS界的最新內容.

經過了一年,她對GTS渴望得已經以細小的昆蟲來充當小人.

第二年她家中更擺放了多間娃娃屋. 每天她睡覺前她會以GTS的角度玩弄屋子.

她會根據那天是星期幾來決定該做甚麼,例如在星期一踏在屋頂上,星期二窺視屋子的內部,星期三...在星期天她甚至會將其中一棟西式的屋子的煙囪插入自己的私處中. 由星期一到星期天她的瘋狂程度是遞增的.

她把電腦開啟後,只見桌面的背景是一張由GTS圖制作神人所制的圖. 她迅速地把滑鼠移向一個腳印圖案並雙按,打開了陪伴她瀏覽GTS網站三年的firefox.

她打開我的最愛,然後按了第一個連結. 這個連結的名字是Giantess Night.

在網頁完成讀取的同時,她發現了流量計寫著"1000000",並且網頁下方多了一行字: 恭喜你,你是這個網站第1000000個訪客,能夠存取本站一級機密GTS片段. 按此進入

"那個好像有甚麼好看的"她沒有考慮就按入去了.

裏面則時令她吃一驚的內容:

"為了慶祝元宵,第一百萬個訪客能夠進入這裏.在這裏您可以找到神秘的,完全真實的GTS影片." 她看到這,已經十分興奮,忍不住要看下去.

"詳請有待情人節公佈"

"甚麼?還要等呀?!"這似乎令她很失望. 她把這個頁面加進了我的最愛,然後便開始上班.

-----------------------------------------------

我想我還會寫下去吧(這不是一個長篇故事),如果我在14日有時間的話.
