# [公告]你文章沒人看？這裡有些小技巧∼

作者：Dante

TID：497

 

# 1

　　各位在本區發表文章的朋友們，是否曾覺得自己的文章受到冷落，或是缺乏回應呢？

　　撇開兄弟登山、各自努力的基礎文筆不談，當你在論壇上發表文章時，若能運用一些小技巧，其實是能增加你的讀者的。這道理很簡單，就是站在一個讀者的角度、試著去想自己會喜歡看見什麼東西，又不喜歡看見什麼東西？請注意我說的是「東西」而非「文章」，文章的內容百分之百由作者決定，但你可以試著去美化你的文章，好讓讀者閱讀起來輕鬆一些。只要能做到這個，就能在一定程度之內，有效吸引你文章的人氣。當閱覽數增加後，回應也才會跟著來，沒有人會回應一篇沒看過的文章的。

　　現在，也許你會問我，到底是什麼小技巧？其實說難很難，說簡單也很簡單，最重要是要有耐心：

　　「把你的文章適度排版後再貼出」。

　　不難理解吧？文章不比普通的主題討論，文章的字數是很多的。若你文章連段落都不分、幾千字一氣呵成擠在一塊，誰人看了都要眼花撩亂啊！這樣又怎能期待讀者把你的大作給看完呢？若文章連看都看不下去，又有誰會給你回應？以這個論壇的編排來講，我會建議各位在段落之間隔出一行空白，讓文字不要那麼擠，讓讀者的眼睛不會那麼累，這是只有作者才做的到的事情。畢竟現在的各位都是寫作個體戶，沒有編輯會幫你排版啊。

　　還有一點，可能並非必要，但如果在每個段落的首行加上兩個全形字以上的縮排，看起來會更賞心悅目。普通我們在進行討論時是不用這樣，但別忘了文章的字數是遠超過那些討論串，任何美化版面的動作都不會是多餘的。

　　最後，是我個人的建議。若你是簡體字用戶，在本區發表文章前，不妨將你的大作先轉成繁體字。兩岸用詞的差異就讓他留著，至少把文字轉為繁體，台灣同好才比較容易讀得下去。當然，愛很夠的人根本不會在意這些，但事實的現狀是大家比較喜歡跑圖片區，圖片是超越語言的存在，而且視覺上更加完整，讀者的反應也是又快又直接，這是文章永遠及不上的地方。文章是需要時間咀嚼的，而通篇簡字的文章對台灣同好來說，確實是必須花更多時間來閱讀；更別說GTS文章是需要醞釀情緒來閱讀的，而這股情緒往往會被文字的差異給打斷，這可是很掃興的。綜合以上的原因，讓為數不少的台灣同好在看見簡體字的同時，就選擇跳出主題。

　　同樣的問題也發生在女神那邊，有人抱怨過繁體字的文章難以閱讀，而事實上女神的繁體文章也確實較不受人青睞，不管是閱覽數或回應數都明顯偏低，表示文字的差異確實對閱讀者產生了影響。而既然女神主要通行的是簡體字，那我把本論壇定位為繁體本位制，應該是很公平的。當然，目前兩邊的管理群都沒有對文章的繁簡體做出限制；但既然讀者的反應是如此明顯，那麼作者選擇在本站發表作品的時候，將自己的文章以繁體字格式發表，無疑是非常聰明的。

　　以上，提了三個文筆外的小技巧，希望每位作者都能廣集人氣、大受歡迎。
