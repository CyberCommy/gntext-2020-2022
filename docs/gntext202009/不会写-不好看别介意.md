# 1

白天梦里，许删坐在我床问我，愿意成为我的用品吗？比如：内裤，鞋这些。永远是我的用品，永不为人……我会赐橡胶般的体质，你的形状可以让我随意变换。赐你不灭之身，不会损坏，也不会痛。许删脸上泛着妖异的光芒。我看许删坐在我床上，戴着黑色胸罩，穿着粉色内裤对着我看。我顿时被她迷住了，况且成为她的内裤和鞋是我的愿望。我一直都幻想着变成她的内裤和鞋。连做梦都想。我马上回答她说我意。好。嗯，你先把我内裤吃了吧，反正我可以变成我的内裤。说着就把粉色的内裤脱下来，噻进我嘴里。我看见她密密的毛心跳加速就直接把她的内裤吞下去了。她看见我把她内裤吞了就站起来双脚放在我身体的两边。脸向我头部。然后坐在我头部来。我感觉有一股力量压住我头部立刻醒来，睁开眼睛一看，只能看见农密的黑毛和和一个小腰。我马上想起刚才那个梦，突然我的头被往上面按住，我的嘴巴好像贴在很热很潮湿的地方，眼睛什么都看不到。不用说我都知道是哪里了。身体也被往上弯，脚弯到前和手绑在一起。我知道刚才那个梦不是真的，我现在已经是许删的内裤了。一股很甜美的声音传来，爽吧，你的愿望实现了。我知道是许删在认我说话。我就问她，你说我永远是你的用品，不能为人了是真的吗？她嘻嘻，用手动了动我，不是，应该是她的内裤。当然是真的了，谁叫你那么贱，一直幻想成我的内裤和鞋。5年了，你足足幻想了5年，现在我帮你变成了现实。我成仙一千年还是第一次见到你这样的人。哦，你不是人，你是我的用品，现在是我的内裤，嗯，明天...做我的鞋吧。以后做我的什么用品还没想好。反正我只要不死你就只能是我的用品，等我死了你也会和我一起死。等我至少要几十万年。呵呵，你安心的享受吧！出去玩了。我的全身被东西夹住，被夹的很紧很紧。许删又嘻嘻笑了一声。穿紧身裤感觉不错吧。我感觉到闷热，潮湿。之后听到走路声，每走一步我的嘴巴就和她的私密部位摩擦一下。想说话也说不了。不多久，有一种黏黏的液体流进我嘴巴，顺着我的喉咙下去。我知道我都是许删主人的用品了……永不为人
