扭曲的光線，無序的雜音，淡綠的背景，這是我無比熟悉的世界。

當然，我現在還沒有身體。

我於是開始回憶自己“應該有”的外貌，175的身高，略顯瘦削的體型，然後是長袖襯衫與牛仔褲。

OK，現在身體構造完成，我從口袋裡取出一根煙，嫻熟地點上。

深吸一口，煙草的氣息是如此真實。

“我回來了。”我對著空無一人的世界說道。

“你回來了，哥哥！”月兒笑著回答。

我轉過身體，第一眼看到的只是一對白嫩嫩的膝蓋。

再往上，則是白衫藍裙的校服，以及月兒微笑的臉龐。

嘖，居然也變成巨人了。

她跪坐在我的面前，我依然不到她膝蓋的高度。

“是啊，我回來了。”我點點頭，作出回應。

“這次也是很快就離開嗎？”她低下頭，例行問了一句。

“是啊，你把老爸殺了，我總得做點善後吧，大概幾分鐘就結束。”

“只有幾分鐘......哥哥不能留下來陪我嗎？”她眼簾低垂，似乎就要流下淚珠。

“我的答案依然是不能啦，畢竟現實中還有一個月兒在等我會去呢。”我狠下心腸，再次拒絕。

她抬起頭來，悲傷地看著我：“難道哥哥認為這裡不是現實嗎？”

“也不能這樣說啦......”我再次深吸一口煙草，“對我而言，兩邊都是現實，兩邊的月兒一樣重要哦。”

“那為什麼不一直留著陪我呢？”她伸出雙手輕輕圍住我，似乎害怕我會突然消失。

“一直留在這裡，不就死了嘛？”我推開她的手指，解釋道，“既然死後我會一直在這裡陪你，那麼還活著時當然要陪著那個月兒啦。”

“但是......”

月兒還想說什麼，但我果斷打斷了她。

“沒什麼可是了，就這樣吧。”

然後，她就消失了。

畢竟她只是我意識中的投影，存在與否僅在我一念之間。

該做正事了。

我叼著煙頭，點點手指，開始搜尋關於老爸的一切資料。

-------------------------------------------------- -------------------------------------------------- ---------------------------

“即使身在果殼之中，我也自認為是無限世界之王！”

七歲時，我就理解了這句話的含義。

我有超能力，我知道。

這能力大概是類似靈魂出竅的技能，我可以不受身體的束縛，自由地以意識狀態在空間中飛翔，並且看到、聽到周圍的一切。

但這能力真正的作用，直到網絡時代才體現出來。

我是網絡中的幽靈。

這不是比喻。

沒有東西可以阻擋我，沒有東西可以限制我。

我感到，這個世界才是我的歸宿。

-------------------------------------------------- -------------------------------------------------- --------------------------------------

要讓一個人不存在，到底有多難？

很難。

要讓一個死人不存在，難嗎？

不難。

在可愛的網絡時代，只要你的身份證明，出生記錄，工作記錄等等等等的記錄全被刪去或篡改，你在法律上就不存在了。

能證明你存在的，只剩下你的親人與朋友。

而我老爸則只有同事卻無朋友，親人中有來往的也只有我們兩人。

很乾淨的人際關係，不是嗎？

但我不能簡單地刪除。

我自己姑且不論，月兒還是要正常上課，工作的，她需要一個合理的出身，一個存在的父親。

這有點麻煩，但不困難。

首先，我找到了一個月前發生的大型車禍，死了十多人。

然後，這個事故已經處理完畢，資料被塵封，想必也沒人會去檔案室查這個。

那麼，只要在警局中的死亡人數上小小添上一筆，一切就解決了。

當然啦，還有車票記錄、死亡證明、遺產分配以及諸如此類的繁雜手續，也要一一辦理。

唯一的缺憾是，檔案室的實體文件與電腦資料會產生脫節，但問題不大。只要沒人懷疑，就不會去翻查這些無聊的數據；若有人懷疑，無論如何也總會被發現，改不改就無所謂了。

至於老爸的公司......

哼哼，金融危機中不幸被裁撤的白領可不少。

所以說，我愛英特網。

-------------------------------------------------- -------------------------------------------------- -------------------

“完美結束了。”我睜開雙眼，伸個懶腰，對一直緊張看著我的月兒說道。

“結束了？”她疑惑地睜大眼睛，“這麼快？”

“當然，我也是有能力的，只要是與網絡有關的問題，都不是問題。”我笑著自誇，“記住咯，我們可憐的老爸在一個月前的車禍中升天了，請節哀。不過他的遺產都留給了可愛的女兒，這點是還是很值得開心的。 ”

“記住了。”她用力點點頭，“那哥哥你的失踪呢？”

“我的房子反正是自己買的，而且剛剛已經賣掉了，這裡留不下線索；而我的人際關係比老爸還單薄，許多我認識的人並不認識我，所以不可能出事，你就安心吧。”笑話，作為一個宅男即使哪天死了都不會有人知道，何況失踪而已。

“那麼，不會有警察來找我了？不會有人來抓我了？”她似乎沒怎麼聽懂我的解釋，直接提出了最本質的問題。

“恩，不會了，只要你小心，不要在人前使用能力。”我便很簡單地給她結論。

“萬歲！哥哥萬歲！辛苦您了，請喝茶吧！”她誇張地鞠了個躬，卻沒想過以這樣的體型差我差點被她帶起的風給吹走。

“唔......”等我好不容易站穩，第一眼看見的是一杯熱茶。

“雖說你請我喝茶我很開心啦......”我摸摸鼻子，“但這對我而言是個蓄水箱吧？”

“請別擔心這個。”月兒挺起小小的胸膛，自豪地看向了眼前的茶杯。

我觀察到，她的瞳孔變成了紫色，緊接著，大概一秒後，杯子就瞬間縮小。

果然，一切物體都可以啊。我上前捧起大小合適的茶杯，輕輕啜了一口。

“我可以把什麼都變小哦，哥哥不用擔心生活會不方便！”她追加了一句，然後盯著我，表情興奮又緊張。

月兒啊，你的臉上寫滿了“來誇我嘛”四個大字哦。

“真不錯呢，月兒很厲害啊。”

我依她的意思說出口後，月兒就“嘿嘿”地傻笑著。

大學生的情商，這麼低不會有問題嗎？當哥哥的很苦惱啊。

所以我必須潑點冷水。

“月兒啊，以後得靠你來照顧我了呢。”

“是的，我一定會做好的！”

“但是呢，你如何再往這杯子裡加水呢？”

沒錯，縮小了二十倍的杯子，是不可能用正常的水壺續杯的。

大概這個問題太尖銳了，她猶豫了一下才回答：“我可以先倒進另一個杯子，再把它變小。”

“那我們的杯子就成一次性的了啊，而且不止杯子，碗啊碟啊也都不能再次使用，你是學生，沒有收入，靠著老爸的積蓄能支持多少年呢？”

“唔......哇！怎麼辦啊，我沒想過這個問題啊!”月兒無力地趴在了書桌上，把我震得幾乎站不住。

我太殘忍了吧，居然問一個天真少女這麼現實的問題......

“乖啦，別怕，有我在就沒事。”作為贖罪，我立刻進行安慰。

“但是......哥哥已經被我害得無法工作了啊......”她抬起沮喪的臉，自責地說。

“沒有哦，哥哥我算是黑客，無論變成多小，只要有電腦，有網絡，我就能賺錢。”

“黑客？好帥哦！”小星星，滿眼小星星。

“帥？咳咳，反正錢的問題你不用費心，絕對夠的。”天哪，現在的小女生審美觀太怪了吧，我國的政治法律教育還真失敗——我本來以為她會說“黑客是罪犯哥哥不能做”而準備了大量說辭的呢......

算了，反正無所謂。

說起來，為什麼我們兄妹都有超能力啊，真的是遺傳？

莫非，老爸死前一直嗚嗚著的話是“其實我也有超能力啊，就是不管怎麼吃都只會微微發胖絕對不會有很明顯的啤酒肚”？

吶，死人就不去理他好了，我從腦中抹去這可笑的一幕。

總而言之，我和月兒的新生活開始了。

 混沌的空間，混沌的時間。

我孤身一人懸浮。

還是比較熟悉這個世界啊，雖然不是不能讓背景變得豐富一點，可是只有這樣才能分辨兩個世界的差距吧？

不過，一個人還是無聊了點。

“哥哥，你回來啦！”於是月兒的聲音就出現了。

這次她趴在地上，以手支頤，笑嘻嘻地看著我。

唔，為何還是......

自從真實世界中的月兒將我縮小後，這裡的月兒就一直保持二十倍的相對身高。

算了，儘管不是很適應，但也無所謂吧。

為了改變因為體型差異過大而造成我看不清她下巴以上部位這一事實，我退後兩步，然後抬頭，看著她的眼睛。

“啊，我回來了。”照例回答。

“哎呀呀呀呀，我發現了！”她突然瞪大了眼睛，以幾乎震破耳膜的聲音叫了起來。

“......你發現什麼了？”雖然我很想說“淑女是不該這樣大叫的”，但又因為“什麼啊又不是現實世界這種小事無所謂的吧”而放棄了說教。

“我才發現呢，哥哥一直以來和人說話時都避免眼神接觸的，只有對著我說時才會看眼睛呢。”

“啊，是嗎......我都沒注意呢。”雖然這樣說，但其實......

“太好了呢！Double的開心啊！”

“為何是Double啊？還有請說國語。”

“哥哥回來了我很開心，哥哥特別地對待我我很開心！”毫不猶豫，乾脆利落的解釋，月兒你的屬性是熱血運動少女嗎？我一直以為你是天然系的啊！

雖然很想那樣吐槽，但還是算了吧。

“當然咯，你是我重要的妹妹嘛，和那些路人A至Z完全不同啊。”我的屬性也暫時變成溫柔系的了，請多指教！

“啊呀呀，哥哥變成溫柔系的了，大驚詫啊！”她很敏銳地發現了，月兒的腦子沒那麼好使吧，你真的是月兒嗎？

“我對月兒你一直很溫柔的吧，剛才的台詞很傷人啊！”我憤然抗議。

“但是哥哥只對那邊的月兒溫柔，對我一直很毒舌啊。”她果斷反駁。

“啊，是呢。”我不得不承認。

“不要在大段沉默之後只說這幾個字啊，哥哥你得充滿誠意地道歉啊！”

月兒狠狠一拍地面，我猝不及防下就輕易被震倒在地，然後她在我起身前伸出食指壓在我的胸口，完全剝奪了我的移動力，居高臨下地用她那黑亮的眼睛盯著我說道。

“就是因為那邊的月兒不會像你這樣粗暴，我才會溫柔對她的呀！質問別人前先考慮自己的行為吧！”我用盡雙手的力量，徒勞地推著那隻食指。

“我......很粗暴嗎？”月兒放輕了一點力氣，疑惑地問我。

“這是廢話吧！”我指指胸口說。

“但我是哥哥意識的投影啊，是你的意識希望我這樣的。”她側過頭，邊思考邊說，“所以，哥哥就是覺得那邊的月兒太溫柔了，所以你需要我在物理上S你，同時你在語言上S 我。沒錯，哥哥你的愛好好複雜哦！ ”

“停止！STOP！終了！”我咆哮著，“作為一個投影你不應該說這些的！對話發展到奇怪的地方去了！給我刪去之前的記憶！”

“好，記憶刪除，我會裝作之前沒發生過任何對話的。”月兒放開了我，用右手行了一個奇怪的軍禮。

“......雖然我知道你知道，但算了，大家都裝沒發生就好了......果然兩個人的話對話會很危險，再加一名聊友吧。”我摸摸鼻子，打了個響指。

清脆的一聲“啪”，第三人出現在了這個世界。

“呦，好久不見啊，阿軍！啊，還有月兒！”並對我們若無其事地打著招呼。

“明明剛見過才沒兩天啊，為什麼明明只有四十多歲卻要像個老頭子般因為大限將至而特別珍惜時間一樣對我們說出'好久不見'這種毫無道理的台詞啊！ ”因為在這個世界裡，我毫不猶豫地進行了吐槽。

“啊，恩，那個，你好，爸爸......”而月兒則完全無法像我一樣灑脫，無論是在那個世界。

不過這也是當然的吧，剛被自己殺掉的老爸又出現在眼前並且活力十足地向自己打招呼，是人都會拘束的吧？

無所謂啦，反正就是這樣而已。反正只是人類會感到拘束這種程度而已。

“吶，老爸，現在感覺如何啊？”

我點燃一支煙，隨意地問道。

“要說感覺嗎，倒是一般般吧，反正我就是這麼平庸的人，什麼都只是一般般而已。不過如果真要說點什麼特別的事情，就是那個啦......”

“到底哪個啊？”

“就是那個啊，下水道的氣味真令人受不了呢！”

“嗚哇哇！對，對不起爸爸，我不該把你丟進下水道沖走的！”月兒立刻因為罪惡感而淚流滿面，跪著向老爸道歉。

“月兒別向這個傢伙道歉啊！而且把它丟進廁所沖掉是我的主意！”一個好哥哥怎麼可以眼看著妹妹哭泣呢，我立刻安慰道。

“你說它？我沒聽錯吧？你剛才說的是'它'而不是他？你是這麼稱呼自己的親生父親的嗎？”似乎這邊也受傷了，老爸雙手摀住胸口很難過地說道。

拜託，中年大叔就別學西子捧心了！

“就是它沒有錯！你已經是個屍體而已了啊！屍體沒有人權！”

“好，好乾脆冷酷的台詞啊，阿軍在我不知道的時候已經長成如此有個性的男人了嗎？”

“這明明是母親才會說的話啊！你的語言比你的行為更噁心了呀老爸！”

“是這樣嗎？可能是因為我太傷心了吧。那麼能不能給傷心的父親一根煙呢？”完全不帶任何傷心的表情，老爸向我伸出右手做出了索要物品的經典手勢。

“自己隨便在哪裡找一張廁紙然後放入雜草做成捲菸吧！”我果斷地一巴掌扇開了他的手。

“嘰呀吱！”

這是什麼怪異的聲音！

我扭頭看去，老爸的脖子正呈現出違反人體構造的扭曲，像一灘爛泥般歪向一邊。

“嗚哇哇！對，對不起爸爸，我之前不該把你的頭扭斷的！”月兒的罪惡感瞬間加劇，眼淚跟瀑布已經毫無分別。

“你贏了！煙拿去！”我立刻投降，遞過去煙的同時以小弟的姿態獻上了打火機。

“啊，有個孝順的長子真幸福。”老爸的脖子立刻回復正常，陶醉地享受著天倫之樂（？）。

這是個什麼樣的家庭關係啊？

哥哥，哥哥

“好啦別哭啦月兒，老爸的脖子不是好端端的嗎？”

哥哥，哥哥

“怎麼還在叫我？”

哥哥，哥哥

啊，原來......

“就這樣吧。”我輕輕抬手，巨大而愛哭的月兒以及卑劣地搶了我一根煙的老爸都消失了。

精神世界，或者夢世界，怎麼都好

現在該醒了。
