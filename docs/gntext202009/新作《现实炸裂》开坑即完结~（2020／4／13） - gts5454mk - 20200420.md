# 1

    大家好呀，潜水多年的我又回来了。
    前段时间，我在聊天区看到有人在问，推荐女友入坑的话，究竟应该看论坛里的哪篇文章呢？我觉得这个问题很有意思，所以就有了下面这篇文章。（虽然我并没有女朋友，哭泣）
    论坛里面的很多文章呢，它其实是我们赤裸裸欲望的体现，而且大多数都涉黄或者有暴力倾向的元素在里面。（包括我自己曾经写过的文章）作为这个小众圈子里的我们来说，其实是很怕被人打上恶心、下流这样的名号的，被人带着有色眼镜看待并不好受。
    所以我在想，第一篇推荐给圈外妹子看的文章，它首先不能太过分，至少不应该有很多涉黄的元素。不给别人的第一印象是，这个圈子里的人都是变态那种感觉。
    其次呢，它应该解释，这个圈子里的人喜欢的，究竟是什么？这样一个问题。（关于这一点我尽力了，毕竟我的观点不能代表圈内所有人的观点，我尝试涉及了一些）
    最后，它不能是一篇无趣的科普文章，不然妹子看完了，可能都对这个圈子提不起兴趣了，又谈什么入坑呢？
    我一直觉得，无论是交普通朋友也好，还是谈女朋友也罢，我们不能强求别人理解自己的爱好，但是当我们介绍自己时，我们需要有拿的出手的一篇文章。它不是用下半身写的快餐文一样的东西，它只是客观公正向别人描述我们所憧憬的世界，并试图去解释，我们产生这样欲望的原因，仅此而已。
    这也是我觉得“推荐女友入坑要看哪篇文章”这个问题很有意思的原因。下面这篇文章，便是我的答案。

    对我而言，今天是个重要的日子。我右手拿着刚刚喷过水的花，上面凝结成的小水珠，在不同颜色的花瓣上，折出宝石一般的光芒。

    黄昏的阳光斜射进房间里，它不刺眼，也不灼热，甚至有些慵懒，这缕光披在这束花上，也打在我喜欢的人的脸庞上。

    “我们在一起三周年了。”我走近雪儿，从背后把花掏出来，移到正玩手机的她眼前。“今天我会和你坦白，关于我的所有事情。”

    “自从你和我说你前世是狐狸之后，我听到什么都不觉得奇怪了啦！”雪儿接过花，凑近闻了闻。

    “我的前世是一名自由斗士，生活的世界也和现在完全不一样…”

    “诶？你之前还说你前世是狐狸的。”雪儿打断我，一脸怀疑的看着我。

    “你别急嘛，我一会就会讲到的。”我拿出小画册和铅笔，坐到雪儿身旁，试图在她面前，勾勒出我前世的那个世界。

    我的前世是一名自由斗士，生活的世界是一个完完全全的母系社会，女人统治着近乎所有的社会资源，并在各种意义上，统治着男人。

    而我为男人的自由而战。

    社会原本是父系的，男人无论在体力上，地位上，甚至是魔力的拥有量上，都比女性要高的多。

    “诶？魔力吗？魔力是什么东西？”雪儿歪着头问我。

    “魔力对于前世的我们来说，是极为普遍的东西，就好像白天无处不在的阳光一样，每个人都好比一颗植物，可以从阳光中获取能量。而至于魔力到底是什么，它是连接现实与精神，连接有与无之间的一种能量。”

    我转过头，看着雪儿：“假设现在你的脑海中有一个想法，或者说一张蓝图，可以用木头做出一个小凳子，那么这张蓝图，或者这个在你脑海中的小凳子，究竟是存在还是不存在呢？”

    “emmmm～不存在吧，小凳子不是还没有做出来吗？”

    “可是你已经有了想法啊，拿到几块木头，加上几颗钉子，稍微动动手，这个小凳子就做出来了。你看，靠谱的想法总是能转变为现实，如果你说脑海中的小凳子是不存在的话，魔力就是连接存在和不存在的一种中间能量，它介于有与没有之间，这种能量，就等同于你去动手钉钉子，把脑海中的蓝图转变为现实。”

    “哇～这个太超现实了吧！”雪儿皱着眉头，她愈发怀疑的望着我：“我不太能理解你说的这个世界。”

    “对我而言也是一样的啊，对于同时拥有两个世界记忆的我来说，现在这个没有魔力的世界，让我不能理解。”我说。

    “可是我总感觉你在骗我。”

    我笑了笑，将雪儿搂进我怀里，把小画册翻个面，继续我的故事。

    “嘛，固执己见可是要吃亏的。”我摸着她的头。

    低阶的魔法可以将想法转变为现实，就像凭空在地上创造出一个小凳子一般，其代价是魔力量的消耗。而高阶的魔法可以改变世界的规则，物理法则甚至是存在方式，只不过，这样的魔法是无法被个人实现的。

    就如同单个人无法建出五角大楼一般。

    我总是在怀疑，现在生活的这个没有魔力的世界，是不是在先前的某个时间点，有些人达成了协议，运用极高阶的魔法，修改了整个世界。

    “我愿这个世界再无魔法！”于是世界成为了现在的世界。

    是的，修改整个世界这件事情并不奇怪，只是它需要极高的魔法造诣，这也是为什么在我前世的世界中，男性的体力和魔力都高于女性的情况下，世界也在很短的时间内，就从父系社会转为母系社会。
