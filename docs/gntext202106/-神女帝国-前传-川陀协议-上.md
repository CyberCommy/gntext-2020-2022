# 2

战争结束了。

这场被后世称为“天堂之战”“能终结一切战争的战争”在经过短短的一年多的时间后，终于落下帷幕。整个银河系的所有星际文明都被卷入了这场战争中，连银河系边缘的一些刚迈入星际文明的门槛的文明都被迫以各种各样的方式参与到这战争中。

战争的战败方，是以第三象限的凯恩斯帝国和瓦尔希帝国为首的，成员国达两百多个国家的“反神女帝国联盟”。

而战胜方，自然是孤立无援、被团团包围的，却在这场战争中把国土面积扩大了近十倍的神女帝国。

在联盟中，谁都没想过会有这样的结果。即使是那些保守派，也从未想过神女帝国的反攻会那样的猛烈和迅速。 

# 3

川陀星系，第三环世界。

这里曾是联盟的最重要的前线指挥部。在这里，联盟里来自各个帝国的将领在这里指挥那些深入神女帝国领地的舰队。

以及眼睁睁地看着他们的舰队被神女帝国的舰队一个个剿灭。

在神女帝国反攻至这里时，她们没有对这个星系下毒手，反而是将其保护得很好。这个星系也是被控制地区中唯一一个没被神女们蹂躏的可居住星系。

现在，这里变成了签署无条件投降协议的场地。

而这个环世界的首府，川陀城，一个不算特别发达的城市，因需要为签署协议作准备，这城里的居民全部被撤离，留下来的是联盟所剩无几的陆军军队。

川陀市最大的空地——政府广场——被清理得干干净净，在那之上只有一张摆了二十一个席位的圆桌。

还有相应的，二十一位联盟各国领袖以及他们的秘书。

……

“真是难得啊，大家齐聚一堂，共讨国事。”来自银河系第一象限边缘的，希维尔帝国的虫族领袖王尔德阴沉沉地说道。

“难得个屁，谁他妈想这样？”在他右手边的，和王尔德一同来自第一象限的，凯美瑞帝国的机械种族K1024，愤怒地大叫，“难道你想要这个结果？”

“我本意并非如此，但看到你这样恶心的机器人能吃到苦头，那我倒是挺乐意的。”

“你说什么？”

“别吵了！”一道听上去年老的声音在众人的心中响起。

“你们两个从几千年前就一直吵，吵到现在，还没够？”

两个领袖立马闭上了嘴，虽然他们仍然能轻易地吵起来——至少得给联盟的最高领袖，凯恩希帝国的凯恩希五十八世一个面子。

一团散发着白色光芒的灵能物体，正漂浮在圆桌一角的上方——那便是凯恩希五十八世的本体。

“与其在这里吵吵嚷嚷，还不如好好想想该怎么面回去面对自己国家的人民。”瓦尔希帝国的领袖，海星一族的族长，海格尔说。

“都成了流亡政府了，还能怎么面对？”不知是从哪里传来的，带有深深悲哀痛苦之情的话传到每个人的心中。

是啊，还能怎么面对？这里的领袖大半都是领导着一个几乎毫无作用的流亡政府，该去哪里面对他们各自的国民？

 

# 4

就在他们沉默不语之时，一道金光出现在圆桌中间——看上去是一位高十几米的神女传送了过来。

“贵安，联盟的领袖们。我是神女帝国政府的秘书长。索尔·洛丝薇瑟。”

按地球人的方法去描述，这位秘书长穿着普通白领秘书一样的衣服：黑色外衣配白内衬，黑色短裙配黑色高筒袜，以及黑色高跟。

虽说是秘书职位，可带给众人的那份威压感却不是一个普通秘书该拥有的——毕竟是一位顶级的天空级神女，在这里要论实力她仅比那灵能大能，凯恩希五十八世稍逊一筹。

“你来这里干什么？”凯恩希率先开口，对这位神女表达不善之意。

“我是来这里给各位递送协议的，并且被命令要监督各位的签字。”索尔微笑着对那位灵能体说。

“监督？就你一个？你主子不来吗？”王尔德阴沉地说。

“啊，我的主人说不来这里呢，”索尔笑了笑，“毕竟这里没她的位置。”

“什么意思？”

“这里才二十一个座位，不是么？根本没有属于我主人的位置呢。”

“这不是你们说只要二十一个座位就行了？你们这样两面做派是脑子秀逗了？”

“啊，确实有这样说过呢。”索尔歪着头，思索了一会，“因为你们没必要准备我主人的位置。”

“毕竟我主人跟我说，你们根本没资格和她平起平坐呢。”

“臭婊子，你TM是想死吗？！”不少领袖被激怒了，他们可从未被这样轻视。

“啊，请不要大喊大叫，我主人可不喜欢听到惹人讨厌的虫子的乱喊乱叫。”

“各位请冷静点。”海格尔强压着内心的愤怒，劝阻大家，“你说你主人不来，那协议是双方的，你主人不来签的话那协议无法生效。”

“啊，我主人确实不来这里呢。”

“倒不如说，我的主人要来的地方，并不是这里。”索尔意为深长地笑了笑。 

# 5

轰隆——

大地突然剧烈地震动起来，天边传来巨响，耀眼的光芒在远处突然出现。

当异象结束，众人发现天边出现了四个金光璀璨的，遍布华美花纹的长条物体。抬头极目远眺，就会发现那四个不明物体似乎在支撑着什么东西，但由于距离实在太远，目力所不能及——只有凯恩希和一些机械生命运用他们的能力才能隐隐约约地感觉到那是什么。

“那恐怕是一把椅子……”凯恩希喃喃低语。

众人顿时炸开了锅。

“神TM椅子，给谁坐的？”连海格尔都不淡定了，对着凯恩希问道。

“当然是给我主人坐的啊。”索尔优雅地笑了笑。“这椅子高度都快八万公里了哦，全银河系也只有五位神女才有资格坐这种椅子呢~”

“TM的她肯定只是吓唬我们，别被这婊子吓到了！”k1024不敢相信，在他的数据库里还没有这样的生命体能有如此庞大的体型。

“是不是在虚张声势，我想凯恩希您应该很清楚吧~您应该是在场的唯一一个见过我主人的真实实力的生命体。您不妨和他们分享一下那天您的感受~”

“不过现在不用劳烦您了，大家马上可以见识到，作为蝼蚁是什么样的感觉~”

话音刚落，黑暗笼罩了整个会场。

如若他们所有人都有凯恩希那样的灵能实力，那么就能看到这片黑暗下发生的事

——天崩地裂，山河破碎，万物毁灭，生灵涂炭。

会场方圆数千公里内，无论是什么东西，都变成了大地上的残渣碎屑。

如果从宇宙中望去，会赫然发现，两个数千公里长的高跟鞋鞋印出现在了这个环世界的表面上。

那川陀城，完好无损，孤零零地伫立在鞋印之中。

很快，黑暗褪去了。出现在众人面前的是一望无际的棕黄色大地，头顶上不知到底有几十公里宽的，红色规律纹路，以及从那纹路上掉下来的断壁残垣——若不是有着防护罩的保护，川陀城就会毁灭在这残骸雨之中。

“这究竟是……”海格尔震惊地道，这对他而言是不可理喻的事情。

“快看，那是什么？”一道声音让所有联盟领袖看向了一个伫立在大地上的巨大物体。

尽管有着头顶上那巨大物体的阻挡，无法看到远处那物体的全貌，但依然能辨别出那究竟是什么。

那是一只超然巨大的黑色红底高跟鞋。

明白了，他们都明白了。

占据这些领袖心中的，只有对这高跟鞋的主人的恐惧，以及发动战争的懊悔。

他们深深地明白了，当蝼蚁遇见神的时候，能做的只有祈祷神不要踩向他们的无力与恐惧。

他们也深深地明白了，凯恩希五十六世和五十七世为何会去了一趟神女帝国首都之后疯癫而死。

他们更明白了，对这样的存在发动战争，无异于自杀。

“所以，还没看够我的高跟鞋么？”

高贵，优雅的声音从上方传来，就像是那高跟鞋在和他们对话。

那言语中蕴含着庞大的神女之力，除了灵能生命和机械生命之外，包括海格尔在内的普通生命体因为这句话而直接爆体而亡。

“啧，真是有够弱的，就只是放了点力量进去而已。”

“弱成这样的你们，都敢对我们发动战争，真不知道是谁给你们的勇气。”

“是你吗，小家伙？”

一瞬间，凯恩希就从精神上感受到了近乎无穷的威压。

“我记得你的上一代，还有上上一代，都和我见过一面呢。”

“那时候你们种族还没进化到纯能量体，还有个躯壳在。其实你们不进化更好，这样你就可以像那两位一样，舔我的脚来求我不踩死你。”

凯恩希彻底火了。他不相信他最尊敬的五十六世和五十七世有过如此屈辱的往事。

但由于那强大的精神压力，他连最基础的精神交流都无法做到。

“嗯，怎么都不说话？是害怕到说不出话来了还是怎样？”

“算了算了，不给你们太大压力了。免得待会你们连签字都做不到。”

话音一落，幸存的领袖便感觉那话语中的力量消失不见了，凯恩希的精神压力也随之消失。

“索尔，拿协议出来让他们签字。”

“是，主人。”一直在一旁站着的秘书长打了个响指，把纸质的协议书“打印”了出来，放在各个领袖的面前。

“请各位认真阅读协议书，如若同意协议条款，请在签字处签上您的名字。”

到了这样的情况，不签字和无异于自杀——头顶上的高跟鞋底可是会随时踩下来，将他们碾死。

很快，领袖们都签完了字。那些死去的领袖无法签署协议，只能默认他们同意协议并由索尔帮他们签字。

“好了，各位既然签完了协议，那么我就告辞了。”索尔向联盟领袖们鞠了个躬。

“希望你们在最后一段时光里，好好取悦我的主人~”语毕，这位秘书长带着一叠协议书传送回她的战舰上，只留下不明所以的领袖们。

“她这句话是什么意思？”其中一位领袖困惑不已。

“还能是什么意思，”凯恩希绝望的声音在他们的脑海中响起，“死亡。”

“小家伙说得很对，你们就快要死在我手上了~”那高跟鞋的主人似乎很开心。

“你们大概有不少遗言，可我不想听你们这些虫子的叽叽喳喳。”

“除了凯恩希以外。”

“这算是对我的特别关照吗……”凯恩希“苦笑”道。

“你可以这么想。我是想听听，你的遗言和他们两个有什么不同~我会帮你实现未完成的愿望~”

“……”

“可以对我的人民好一点吗？”

“这个不可以哦，换一个愿望吧。”

“为什么这个不可以？”

“嘛，本来是不打算告诉你们的，不过你们快要死在我鞋底下，告诉你们也无妨~”

“一直以来，我有一个愿望，那就是把整个银河系当做我的玩具。”

“除了我们神女帝国的人民以外，银河系的其他文明的所有生命体都要成为我想用就用，想怎么用就怎么用的奴隶。”

“你觉得，我会好好对待奴隶么？”

凯恩希这下算是明白了，即使他们不发动战争，这个施虐欲极强的神女也迟早会让神女帝国主动出击，吞下整个银河系。

他内心的懊悔消失了，取而代之的是平静。

“我可以给你一个忠告吗？”

“哦，说来听听。”

“你迟早有一天遭到惩罚的，包括你的帝国。”

“哈哈哈哈哈哈哈哈哈哈哈！！！遭到惩罚？就凭你们这些我一脚能踩碎几千亿个的蝼蚁？”

“惩罚你的不是我们，而是天理。你这样的存在，天理难容。”

“天理？天理就是你们这样的蝼蚁生来就是个错误！生来就要被我们神女所支配所蹂躏！！！”

“你既然说天理会惩罚我，那你说的天理，能阻止我把你踩成连鞋底残渣都不如的东西么？”

说完，联盟领袖们头上的红色天空以极快的速度压了下来。

这个过程只有两秒。两秒后，这场战争的发动者们，悉数死在了这高跟鞋下。

“应该都死了吧。”她转了转踩下去的右脚脚尖，看着脚尖又造成了巨大的灾难。

此时的她，正优雅地翘着二郎腿，坐在红色为主调的华美王座上，左手拖着装有红色葡萄酒的高脚杯，右手正扶在那和这环世界一样宽的王座的扶手上。身穿黑色却不失华丽的宴会礼服，低领开胸加上短裙也凸显了她的绝世性感。

“啊，感觉来了呢~居然被这群蝼蚁搞得有感觉了~”她扭了扭她那巨大的美臀，并把她的右手伸向了那短裙之中。

即使隔着手套和蕾丝内裤两层布料，那份湿润与粘滑的感觉还是被那纤细手指所感觉到。

“已经湿了吗……如果就这样回去的话感觉会溜走的呢~”

“那么，就先从你们这里开始吧。”

此时，环世界的某处地方，地面被精确地分离开来，宽达数百公里，长度持续不断增加的地块飞上天空，飞向那巨大王座，连带着在那上面生活的数百亿的生命体。

“我可爱的蝼蚁奴隶们~”

“为了你们至高无上的女皇，为了你们未来的银河系女神，”

“好~好~做~哦~”

在响彻整个第三环世界的撞击声中，银河系的奴隶时代开始了。
