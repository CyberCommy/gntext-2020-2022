Mega Orin

直到那一天前，每天都和和平平的渡過了
天氣還相當的溫暖，少許雲朵點綴萬里晴空，人們在和熙的陽光下，享受美好的假日午後。

突然間，尖叫聲從城市邊緣傳來，
上空明明沒有雲層，但是人們卻不尋常地被陰影籠罩，
他們抬頭尋找任何遮蔽陽光的物體，但很可惜的，眼前所見盡是一片黑暗。
沒有人想的到這片黑暗竟是鞋底的陰影。數個人放聲驚叫，他們驚覺到黑色的天空正在慢慢下降，但，來不及了。

同一時間，在20公里高的上方，女孩露出滿足的笑臉，
她有一頭漂亮的紅髮，結成一對辮子，紅色眼瞳慵懶的在地面上來回掃視著，
黑色貓耳不時抖動，分岔的尾巴不安分地左右甩動，

你應該猜到女孩是誰了：她就是**火焰貓** **燐**

此時燐正在玩耍，她聽不懂八雲紫的解釋，但那一點也不重要。
諾大的城市對燐而言，就像鋪平的地毯一樣，少有超高層比她漂亮的黑色皮鞋高，更遑論腳踝了。
燐知道人類建築物的高度已經到極限了：雖然沒有雲層高，但從她的角度還能看到一部分的建築物。

但問題在於，雲層僅僅漂浮在她的膝蓋附近，最最上層的雲也只到她的腰而已。

甩甩尾巴，或是動動膝蓋都能輕易的把雲層攪碎，燐不明白這些小如麵包屑般的人們能耐在哪？就算他們沒這麼矮，結果也差不多。
燐慵懶地伸個懶腰「喵~~」，她滿意地嚼嚼嘴，要玩什麼遊戲好呢？雖然八雲紫給她一籮筐的東京必玩景點，但那些地方都太渺小了，燐找的到它們嗎？就算找到了，她知道該怎麼小心翼翼的玩耍嗎？

------

當無比巨大的貓娘出現在東京市中心時，全市的動作都停滯了。她看起來過於荒謬、龐大，使人們忘記了究竟該忽視、愛慕或是恐慌，多數市民呆滯在原地。
只有與燐靠的夠近，卻恰巧沒被踩碎的人們，被皮鞋踏在地上吹出的暴風吹上天空，而他們剛剛站立的地方瞬間被壓縮的小到難以辨認。
長度超過3公里，寬度超過500公尺的黑色皮鞋，踏出每一步的一瞬間，沒有任何物體可以存活。

東京塔暸望台全程目睹了慘不忍睹的畫面：
觀光客、路人和工作人員停下動作，一雙雙眼緊盯著天空：一對漂亮的膝蓋略過他們的上空，
他們首先注意到的，是低沉的隆隆聲，大地喳吱作響，寬廣的雲層被掠過的長腿撕成碎片，女孩蹲下身，她的鞋尖由於重量集中而沉入地面，
下一個瞬間，他們的天空被粉紅色的高牆遮蔽，在他們感到驚恐之前，刺耳的金屬撕裂聲、巨大的加速度、擠碎的壓力、劇烈降低的氣壓，排山倒海而來：來自地獄的緋紅雙眼以極近的距離盯著他們，看來就像一面紅色的牆壁。
只有燐知道，小人們在她巨大的瞳孔裡是什麼模樣，
因為下一個瞬間，人們來不及感到驚恐，窒息，甚至是發出一絲哀鳴，就從世界上消失了。
--------

拿東京塔來當試試好了。燐這麼想著。

燐蹲在東京塔旁邊，巨大的鐵塔在燐眼中是個長度不滿3公分的迷你紅針，包覆著脆弱的鋼網，
鐵塔的頂端有些微小的東西，看不清楚呢...
燐伸出纖細的食拇指，靠近微小的東京鐵塔，用指尖輕易的撕起。
她拿到眼前仔細的觀察...嗯，的確有些小東西，非常微小的點，和塵埃差不多大，是人類呢。微小的人類在針尖上擠啊，跑啊，也許還在尖叫呢，太有趣了！
在發生任何事之前，寂靜壟罩著瞭望台。燐會怎麼殺了人們？用睥睨螻蟻巢穴的目光？少無聊了。
燐闔上手指，東京塔瞬間被捏碎。她搓搓指尖，組成鐵塔的鋼構鐵骨在巨量的壓力下瞬間熔成液態，高溫的鐵水順著手指緩緩流下。

名勝清單中的一個旅遊景點就這麼被燐消滅了

很糟糕嗎？不，有名的景點還多的是呢。
