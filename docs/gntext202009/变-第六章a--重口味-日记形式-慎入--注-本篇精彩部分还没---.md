# 1

第六章

忽然感到下面好象有什么东西湿湿的,顶着我,把我都弄湿了,因为刚变内裤,还没习惯,想挪动下自己的身子,结果没想到刚想动发现自己忽然就复原了,而我本来是被装在行李箱里的,人一复原就把整个行李箱撑开了,我连忙爬出来一看还好还没走出浴室,就连忙向更衣间跑去,而王珍忽然感觉着么行李包忽然就爆了,回头一看看到一个人光着身子跑向更衣室,顿时她和她身上所有的每个细胞都惊呆了...

回到更衣室我也没心思洗澡了,穿回衣服走出去,看到王珍还在那站着,包大开,她竟然出差还带洗衣液....怪不得我能变回来.她看到我后呆呆的说'小周,我可能最近压力太大了,刚竟然看到个人从我行李箱里跑出来...'

'恩,是呀,正好趁这次出差,你也好好放松下休息下.'我正想着么说呢,她自己帮我找到借口了,我说着帮忙把她行李箱拉上,和她一起走了出去.

王珍出差了,而我也趁着这段时间,把唐静留下来的资料全部都看了遍,经过努力学习,我也知道了一二,同时为了不要将来说这技术是唐静的,我把她的成果封装到了一个大型机器里,把有生命的物体变成无生命的物体.同时申请了独立的研究室,平时不允许别人打搅,同时在王珍回来后决定在2013年4月6日进行第一次实验,因为那天是星期六,来上班的人不多,即使有什么问题也应该没什么吧.

2013年4月6日  星期六

到底我还是太年轻了,不是专业的,实验出现了意外.本来实验的对象只是实验室的一只小老鼠,结果不小心把发生器对准了我,而我就变成了一件没有生命的东西了,而最令人尴尬的是当时存储在计算机里的数据是只抽水马桶,所以我就可悲的变成了抽水马桶.只是我实验室的电脑里着么会储存着一只马桶的数据呢?然而不幸中的万幸是我还能说话,还保留了大多数的感觉,而我的视线只是从马桶内部看出去,所以我只能通过我自己的马桶圈看到实验室的天花板,除非有人直接站在我边上直接看向马桶内.而王珍则不断的拿着仪器对我进行各种检测,看看我究竟是由什么构成的.结果可以说是非常好也非常另人沮丧,我的主体和水箱是陶瓷的,我的坐垫,马桶盖完全是塑料的,而排水管什么的都是金属的.做完检测后她发现没有任何有生命迹象的物质存在,即使那时变成了内裤,着么说也是棉的丝的,是由有生命的东西产生的呀,而这次一点都没有.这也让我和她都很好奇,我到底是着么会继续存活的.而最让我感觉不舒服的是我还能好象感觉到我以前的形体.最糟糕的是我的马桶内壁好象是我的嘴,而且好象还有味觉,而那个抽水的把手感觉是我的JJ,当王珍碰到那把手的时候,我就有一股很爽的感觉,而储水箱好象是我的储精袋,尽管里面现在空空的,感觉好象很奇怪,就感觉好象水箱本来就应该储满了水.

一开始我就想就当我死了算了,可越想越觉得窝囊.我现在被变成了只抽水马桶,虽然我还有自己的意识,但我还是一只马桶,而做为马桶的工作就是...感觉自己真是生不如死,也不知道自己现在还算不算活着.

王珍的笔记:

我真没想到会发生这样的事,小周竟然变成了....在实验室当中有只马桶,然后从马桶里面发出声音和你对话,那种感觉真是很奇怪.而最另我内疚的是其实那是我的原因,我们租的地方好象马桶总漏水,所以我淘宝上定了只抽水马桶,谁知道当时没注意,写了单位的地址,结果昨天下午就送过来了,而昨天下班前又无聊,就看到旁边有个摄像头样的东西连着电脑,就把那只马桶扫进电脑了.本来我想换只新马桶给小周一个惊喜,结果没想到是这样的惊喜.他现在还很乐观,总感觉自己有办法变回来,即使变不回来,他也想变成象<时间简史>的作者霍金那样,而我还是觉得他还是太乐观了,霍金即使一点都动不了,还写出了<时间简史>,但他着么也是个人,大家都能接受他,但大家会接受一只马桶的科学家吗?而且我想我应该把他藏起来,免得其他同事发现这个秘密后拿他去做实验.

2013年4月18日  星期四

自从那次事故后,不知不觉的我变成马桶已经快两个星期了.尽管第一天白天有王珍会陪着我说话不算太坏,但到了晚上我就觉得非常寂寞空虚,特别是我的水箱里空空的,更感到空虚的要疯了.所以第二天我就让王珍帮我把水箱灌满.当她打开我的水箱盖子,把一大桶水倒进去的时候,我就觉得自己全身都舒服,好象自己更完整了,尽管我感觉我想要打飞机.接下来的几天我就一直呆在实验室的当中,而王珍也暂停了她自己的项目,不断的对我研究.今天她不小心碰到了冲水把手,那是我从没体验过的快感.当水从水箱冲到马桶内,立即疏缓了里面的干燥,然后我的出水管又把水排了出来,流满了一地.王珍马上不停的和我道歉,说她不是故意的,接着拿着拖把把周围弄干净,又在我水箱里倒满了满满的一桶水.而我也越来越担心我未知的命运了,先不说能不能变回来,就算被其他同事知道了有我这么个怪物,肯定还不会把我拿去拆了研究?

王珍的笔记:

看着小周一天天的越来越沮丧,我都不知道着么办了.其实我今天是故意去按冲水把手的,他太需要一种发泄的方式了.也许我应该帮助他接受他自己新的'身体',但我真的不知道着么去让一个人接受他变成了一只马桶的事实.

2013年4月27日 星期六

今天是我变成马桶第三个星期了,我发现我也变了好多.现在王珍至少每天会帮我冲一次水让我疏缓一下,而和她在一次长谈后,她也决定先接手我的项目来帮我.而至于我是着么从一个有机的生命体变成这么个没任何生命特征的物品有待深入研究,现在最重要的是先把我现在这个样子变一下,我总不能一直做一只马桶吧.既然不能把无机的变成有机的,那我想要么先变成个象服装商店里的那种假人?但由于没有样本源,最后王珍脱下了自己的一只黑色的高根鞋作为样本源.但也许因为我现在是没有生命的物体,那个发射枪即使对准我发射也没有任何效果,难道我注定只能永远是只马桶了吗?

而在那几天后最不想发生的事情终于发生了.唐静以前手下的另个助手,现在也有自己独立的项目了,算起来和我也有点熟,他那天也许空了,竟然过来看我了.一开始当他进来的时候我就一直保持沉默,而王珍则在不断的应付着他.但最后不小心给他看到了王珍还放在桌上的工作笔记,结果他就什么都知道了.不过他着么也不相信我还活着能说话,而我当时也死活不发出任何声音.接着也不知道他当时是脑抽还是着么的,对王珍说,'要不也让我试试?'而我完全没有想到的是王珍竟然真的试了,当我看到那台机器发出轰鸣声的时候我就知道机器已经完全启动了,接着只听一声尖叫,男的竟然也能发出这种尖叫?我那时也是这样的吗?只见那个唐静曾经的助手消失了,而他原来所在的地方出现了一只黑色的高根鞋.隐约听到不断的发出'我到底着么了,我到底着么了...'的声音.接着王珍走过去捡起鞋子走出了实验室,回来的时候就空着手了.我也没多问,因为刚才他的出现已经让我意识到事情的严重性了,最后我们决定,我还是需要一个藏身之所的.她说她会帮我想办法解决的.

王珍的笔记:

实验一次次的失败,即使把小周变成其他东西好象也不行,这可真是麻烦了.最主要的是今天唐静以前的助理小张也来看他被他发现这个秘密了,没办法为了保守这个秘密我也只能先把他变成了一只高根鞋,小周是经过我确认之后才知道自己变成一只马桶的,所以小张应该自己也不知道自己变成了什么,我总不能告诉他我把他变成了一只黑色高根鞋.我今天晚上下班就把她放到我的包里带回家,等想到解决的办法后一起把他变回来,今晚就让他在我包里过算了.记得上个月研究所的男厕所在重新装修,现在这个月在弄女厕所,而我们实验室对面就有一个女厕所,平时只有我一个人上.明天我也只能故意去把对面厕所的那只马桶砸一个洞了,然后去和所长说有一只马桶不小心送错了地方送到实验室来了,正好我对面厕所的马桶坏了,让施工人员去把小周安装上去,然后我马上去找到他确认好后,把那间厕所的门锁起来,反正平时也只有我一个人上,所以即使锁上不用也没什么大问题,我可以去其他楼层上,在找到把他变回来的方法前也暂时只能先这样了.哦,不对,现在应该是他们了,还有那只黑色的高根鞋小张,哎,真是越来越乱了.
