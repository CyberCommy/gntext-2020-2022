满月被厚厚的云层覆盖着。街道被黑暗所覆盖，星星点点的路灯照亮着虚无。为了不被注意到这完美的寂静秩序，少女压抑着喘息声埋头自慰。在不被她发现的情况下偷食的小矮人。在一个房间里，只有这两个人作为生命发挥着作用。

“啊……嗯!来吧……”

小女孩的声音在大床上响起。据说是她父亲从南方买来的棒状水果，大小正好和她的阴道一样大，所以她用它自慰。昂贵的食物被插入少女的腹股沟，在沾满黏液的肉压下变得不再是食物。这种背德感让少女的快感增加了好几倍。

“啊啊啊……往里一些……”

伴随着湿漉漉的叹息，她发出了声音。我一边慢慢品尝水果摩擦的感觉，一边把它塞到子宫口。生物看准她的空隙，小心翼翼地接近床边的小桌。他很小，饥饿到了极限。如果能把少女带进房间的好几根水果带回家的话……。他是这么想的，为了不让少女发现才来的。

当他用微弱的力气爬上小桌时，

“啊啊，好啊!啊，我又崩溃了……”少女喃喃说道。

然后，她把手伸向桌子。从床上探出身子，拿着水果走了。

“啊，危险……”差点被发现的小矮人小声说。这样一来，暂时不会再对水果感兴趣了吧。想到这里，他急忙爬上小桌。

少女剥去果皮，再次插入胯下。她的爱液和溃烂后变成液体状的“水果”溢出来的洞里，慢慢地。

“啊……果然很舒服啊……”

少女不停地拔插着水果。确认了这一点后，小矮人又确认了剩下的水果的数量。

“还有……一根?刚才还有那么多……”

小矮人是在地板上走过来的，所以看不见，但就在他绕到床上的时候，小女孩把几根棒状的水果变成了水果泥。

他慌忙掀开果皮，想要咬进果肉里。但是在那里，

“啊啊啊，我又去了。最后一根也拿来用了吧……”少女嘟囔着，向他伸出手。

她拿起了水果。小人被触碰。少女察觉到了异样。被云彩覆盖的满月，开始一点点探出头来。

“啊?是小矮人!变态!太差点儿了!”

少女抓住了和水果一样大的他。然后，

“作为惩罚，你要陪我自慰!”生气。

小矮人破破烂烂的衣服被小心翼翼地脱掉，他被浸在咕嘟咕嘟的液体里。是爱液和水果混合果汁。然后，少女毫不客气地将他插入阴道。小矮人已经陷入绝望之中。

“啊，太好了……”少女略带羞涩地说。

阴道壁分泌的爱液量会进一步增加。她那挤碎了好几根果肉的阴道，紧紧地束缚着小矮人的全身。封闭的阴道内空气稀薄，温度和湿度都异常高。就像水果生长一样，在高温多湿的环境中，一个生物的生命即将结束。

“我又不能高潮，就很痛苦，我也一直在忍耐……但我已经快忍不住了。”

伴随着“哇”的叹息，低语也传到了小矮人的耳朵里。他下定了决心。为了响应这个，少女摆弄阴部的速度变大，阴道内的环境也变得严酷起来。然后，在小矮人意识到死亡之后，她达到了顶峰。

“啊，太好了，好舒服啊……”

从全身虚脱的少女下半身，各种液体源源不断地溢出来。其中还混杂着小人。他勉强还活着。

“我还以为你会死呢……”

少女红着脸对他说。

“怎么可能把你杀死在我身体里面呢，流出的内脏什么的会很恶心的。”

彼此满身疮痍，静静地等待着下一次的展开。

小女孩注意到小矮人的体形，

“难道小矮人肚子饿了?”问。

小矮人点了点头。小女孩沉思片刻，把剩下的一根水果放在他身边。

“那你就这么说吧……我给你饭吃……”

一脸惊讶的少女温柔地低语。满面笑容的小矮人拼命地咬着水果。

当他吃完了10%的水果时，一副再也吃不下去的表情时，

“喂，我有件事想求你。”少女羞涩地开口。

“什、什么事?”小矮人警惕地回答。

“我给你很多饭吃，以后也这样自慰……你能帮我吗?”

“啊，我可以的话……”

就这样，本应笼罩着这座城市的黑暗变得有点粉红色了。满月光明正大地照耀着少女柔嫩的肌肤和沾满爱液的小人。今天，少女用小矮人自慰的事情还在继续。
