# 1

我叫明飞，出生在一个非常特殊的家庭，不过在我小的时候，我并不知道我的家庭有多么与众不同，因为我一直都在家里从来没外出一次，也就不知道我家究竟有多么不寻常。
说是特殊，是因为我家特别大，正如刚才说的一样，大的我5岁之前从来没出去过，家里有妈妈和四个姐姐，从小我就在她们的呵护中长大，我从来都是被她们抱来抱去，或者在她们的胳膊里，或者被夹在她们的巨大的胸部中间，我说，长大了我也要有大大的身子和大大的奶子，姐姐妈妈每次听我这么说，都咯咯笑了，笑的乳房不停的抖动，尤其是妈妈的那对，一只就比我要大两倍，边抖乳头里还边荡漾出奶水。过了会妈妈不笑了，摸了摸我的头，乖宝宝，你是长不到妈妈姐姐那么大的，也没有那么大的奶奶哦！
为什么啊！
因为你是男的，男的没有大奶奶哦！大姐林佳靠了过来，胸部中央的豪沟贴到了我的后背。和妈妈硕大的奶子一一相对紧贴在一起，四个乳房把我团团围住，然后大姐的脑袋向前跟了点，对我说:但是你有个大宝贝，永远是我们的乖宝宝啊！
我一直没有明白大姐这句话，一直到我六岁那年。
那几天姐姐妈妈看我的眼神总是不大对劲，一直盯着我的下体看，时不时舔舔嘴唇，还把玩一会儿，说来奇怪，那时我开始感到被玩的时候开始痒痒。就说:好痒！
舒服吗？
我不做声了。她们又笑了，把我放下后，一起走开，不管我了！<title>2</title> <link href="../Styles/Style.css" type="text/css" rel="stylesheet">
