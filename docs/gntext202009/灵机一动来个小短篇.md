# 1

怎么说呢，为了经验已经开始不择手段了吗？

……………………………………………………

我是G城三流高中的一名高一新生，和其他新来的一样，我对崭新的学校生活充满了美好的想象：比如被漂亮学姐眷顾啊，被学校帮会的老大看中啊……总之，和学习是没有半毛钱的关系。

开学第二天，我便注意到了一个高二的长发美女。她长得酷似叛逆的神马修里的CC，只不过头发颜色是黑色。从那时起，每天我都做着关于她的白日梦，幻想着她把那穿着雪白色棉袜的玉足伸到我的嘴边……甚至有一天晚上我还梦遗了，哎，即使我承认自己算是半个文艺青年，不过屌丝的本性还是有的。我从来不信自己能逆袭，所以迟迟不敢上去和她搭话。

知道屌丝是怎么度过那些没有女神的寂寞日子吗？很简单，只需要一卷纸，一张女神的照片还有一个没有其他人的隐秘空间。

既然不敢追女神，那我就偷拍她足部，然后……嘿嘿嘿……果然我是屌丝啊。

说干就干，今天一放学，我便拿着ipad缓缓走在女神的身后，装成玩游戏的样子，其实是在伺机拍照。我胆子不算大，所以我与女神保持着一定的距离，借助放大功能来勉强获取最佳效果。

突然，女神和她闺蜜撞见了，我心中一喜，看来今天能拍到两双玉足啊！

女神和闺蜜在一旁的长凳上坐下，聊了一会竟然开始拖鞋！难不成我的春天来临了？只见两双雪白的棉袜缓缓出现在我那饥渴的眼中，机会难得，我立即拍了五六张。可是……问题来了，我忘关闪光灯了……这尼玛，光天化日之下偷拍女生的流氓自己把自己给暴露了。

果然，女神疑惑地望了过来，表情似笑非笑，她对我勾了勾手指，示意我过去。我环顾四周，发现没别的人看到，于是就屁颠屁颠地跑了过去。

“喜欢我们的脚是不是？”女神眯着双眼，温柔地说道。

“不不不，……没……”我的第一反应就是死不承认，但我的眼睛一直死盯着她的玉足，傻子都能看出来我在说谎。

“不承认是吗？呵呵，那就接受我的惩罚吧！正好上一个玩具玩坏了。”这句话听得我一头雾水。

当她说完这句话后，我忽然感到四肢无力，并且头疼欲裂……渐渐地，我承受不了如此剧烈的疼痛，于是便昏了过去。

醒来后，我意识到自己可能变小了。因为面前正是一双摄人心魄的巨大玉足！由于我变小了，所以我清晰地看到袜子上有很多汗液。

还没来得及缓缓神，我就被踩在了脚下。头顶上传来阵阵银铃般的笑声……

我和女神的玉足一样大，所以并没有太恐惧。眼见她们很喜欢把脚贴在我身上，我便放开了胆子卖力地舔了起来。在我张嘴的一瞬间，数不清的脚汗流入到我的口中，差点没呛死我。舔了一会，我的下体开始迅速膨胀……“要射了吗?我偏不让你射！”女神把脚移开，然后三下五除二把我的衣服脱光，她的闺蜜见状好像明白了什么，转身从包里掏出一个橡皮筋……敢情她们想榨干我啊！我既期待，又恐惧。万一真成人干了该如何是好啊。

女神弹了弹我膨胀的下体，嬉笑道：“我让你射，你才能射哦！”

说罢，她闺蜜便一脸坏笑地用橡皮筋绑住了我的下体。

“你在一边光看，是不是也想折磨折磨这个小倒霉蛋啊？”女神笑着问闺蜜。

“还是你懂我，呵呵！”

………………四只诱人的白袜巨脚将我团团围住，她们开始疯狂地蹂躏我……原谅我可耻地屈服了。

她们那独特的脚香已经取代了我鼻子中的空气，我浑身上下都湿透了，仿佛浸泡在脚汗堆积的水池里。我拼了命地享受着这一切……最终，橡皮筋被我的下体撑断了……一条银白色的光柱射到了女神的脸上……

女神舔了舔银白色的液体，笑着说：“……从现在起，你就是我的御用宠物了~”

听到这句话，我竟然幸福地晕倒在地……我勒个去，我这才发现原来我是个M足控啊！！
