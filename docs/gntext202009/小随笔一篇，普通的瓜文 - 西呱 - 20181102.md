# 1

他已经和一个同班女同学说好，以后就缩小做她的宠物了，会住在她的鞋里了

当然他不清楚这意味着什么

没有穿袜子的穿着脏兮兮的小红鞋，跳房子，追逐打闹，抓鬼，跳了一下午的女孩，回到家粗暴的把鞋扣了扣，湿透，已经晕过去的小人被熏得臭臭的了，便嫌弃的丢在了一旁。

食物是剩饭，投喂给他的是鸡骨头，橘子皮之类的垃圾

他的惨象对女孩来说挺好玩的

用脚玩玩，然后带着小人洗脚
晚上依然睡在可爱的小红鞋里(鞋子十天一洗)
小人的健康状态:每况愈下

第二天，在女孩鞋里度过了上课，课件运动，体育课，湿透，气味攻击，缺水缺食物的一天后，男孩被倒出来后已经病倒
晚上依然是鸡骨头之类的投喂
洗脚的时候被女孩踩进水里玩，溺水
依然住在小红鞋里
身体状况:虚弱

第三天，
“救救我”男孩趴在鞋口哀求着
而作为回应，她用脚尖把他拨弄回了鞋里，然后很平常的穿好了小红鞋
鞋里如同满是脚味的禁闭室，男孩开始严重的缺乏补给
女孩跑跑跳跳的，让男孩彻底晕了
晚上从鞋里出来时已经失去意识
“站起来”这样的命令也没了反应
女孩已经玩腻了，既然玩腻了，当然就要处理掉
踩死了男孩
尸体彻底粉碎在了拖鞋的鞋底
皆大欢喜
