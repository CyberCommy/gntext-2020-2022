# 1

虽然调查团里人人都叫我指引明路的苍蓝星，但我知道，她们只是在欺负我这个哑巴工具人。

只有每天狩猎狩猎龙娘，给我的艾露娘换换装才能维持生活这样子。

说是狩猎，其实就是给龙娘们修修脚，趁她们累到喘息的时候，偷剪点头发脚趾甲什么的。龙娘浑身都是宝，毛发唾液可以作为炼金素材，而坚硬的趾甲死皮这类则可以做成猎人们的盔甲和武器。用龙娘的脚趾甲和死皮做成衣服总觉得怪怪的，感觉身上充盈这她们的味道，不知道调查团里的人有没有和我一样的感受，想来是不会有的，反正我从来没见过她们正儿八经的去狩猎。难道被龙娘的脚臭变成足控的就我一个…………呸呸呸，难道这么大个调查团就我一个人在干猎人该干的事吗？

从下位的龙娘到上位的龙娘，我越来越强，装备也越来越好，龙娘神奇的力量赋予了防具特殊的能力，这让我在给龙娘修脚……啊不是，狩猎的时候越来越得心应手。终于，我能挑战站在生态位顶端的古龙娘们了。

看起来元气满满，但被打疼了就会在地上打滚撒娇的灭尽娘；喜欢正坐在岩浆区看起来很文静，但一进入战斗就变成多动症的炎王娘和炎妃娘这对百合；最讨厌别人摸她头，一旦被摸到一定次数，就会气到四肢无力瘫倒在地一段时间看起来既优雅又聪明，其实笨笨的钢娘；整天勾引人给她脱衣服，真的脱到只剩内衣就会恼羞成怒的纯情不良绚辉娘…………

原本以为故事就会这么平淡无奇的发展下去，直到我接到“毁灭来临，喇叭作响”这个任务。

狩猎的目标是灭尽娘，但今天的她看起来似乎有点不一样，眼眸里透出一摸金色，脚上的白丝在接线处也用黑金走线，体型也比往常更加高大，大约有25米左右，不像往常会吼两声想把我喝推，直接一个箭步冲了上来想把我踩在脚下。

灭尽娘的一双白丝会随着战斗的过程逐渐被弄脏，当变成黑丝的时候她就会用龙娘神奇的力量将脚臭一口气排出去，虽然脚上恢复了干净的白丝，但排出去的这团黑烟味道却不可小觑，而让灭尽娘不释放脚臭的办法就是主动在狩猎的过程中，多用武器帮她刮去白丝上的杂质。这也是多次狩猎失败被艾露娘拖回营地的经验之谈。但今天的灭尽娘似乎格外有活力，两只大脚踩个不停，时不时还会弯下腰想一下子抓住我，这让我没空隙给灭尽娘做足部护理，果然在弄脏了自己的白丝后，随着灭尽娘的一声咆哮，释放出的浓郁脚臭将我掀翻在地，正当我晕头转向时，她跟上前来一记“如来神脚”将我踩晕了过去。

等我醒过来的时候，身上的装备都被脱光了，连条内裤都没给我留，只有脖子上套着个项圈，用链子固定在墙上。

“你醒啦？”灭尽娘听见了我的动静，回过头来。

龙，龙，龙娘说话了？！
一时间我震惊的说不出话来。

“人家可是历战王哦，学会你们这些下等种族的语言不是轻而易举。”灭尽娘看起来有点小得意。“终于赢了，之前被你这个小家伙打跑了好多次，今天我灭尽娘终于一雪前耻啦，怎么样，我厉害吧，一脚就把你踩昏过去了。”

看着灭尽娘在桌前蹲下，脸上满是掩饰不住的窃喜，不像是要对我动粗的样子，但脖子上的锁链却又让我紧张不安，何况虽然和龙娘们打了这么久的交道，但这么近距离以非战斗的方式接触，还是头一回。她嘴里喘的粗气都吹到了我身上，让我起了一身鸡皮疙瘩，更何况她还会说话呢。

灭尽娘直起来身，自豪的俯视着她的战利品，嘴里满是些“等我成了历战王，就把你们都杀了”这些既恐怖又好笑的话。

“咩咩子，我的……”既然她看起来没有什么恶意，我就想问下我的艾露娘去哪了，没有将力尽倒下的我抢救回营地，这还是头一次，但我一开口就后悔了，怎么顺嘴把灭尽娘的爱称说出来了。

“嗯？！”灭尽娘一个弓步直接将脚踏上了桌面，还就踩在我的链条上，拉得我一个踉跄，失去平衡一屁股坐在她脚趾的旁边。战斗时都没注意，龙娘的脚真的好大啊，怪不得一脚就把我这种猎人踩晕过去了，而且灭尽娘的脚的味道在白丝状态下只有一丝丝淡淡的汗味，真不敢想象在战斗过程中出汗弄脏之后，会是那种令人印象深刻的脚臭。

“这是爱称啦，爱称，只有灭尽娘你有哦。”我扯了扯被灭尽娘踩着的链条，但果然和古龙娘比力气是没有结果的。
“那你倒是说说为啥叫人家这个名字啊！”灭尽娘扭动了她的脚，带着脚下的链条一起滚动，拉扯着我一下子扑倒在她的脚背上。隔着白丝也能注意到龙娘的皮肤很好，平时狩猎时刀砍斧劈的真是对不住了。
“因为咩咩子在被打疼之后，就会倒在地上咩咩叫，和刚见面时看起来有点凶的样子完全不一样嘛。”看见灭尽娘在沉思，我赶紧补上一句:“这就是反差萌哦！”

“嗯，本大人就饶了你了，看在你之前欺负人家但最后都会放人家走的份上。”灭尽娘似乎觉得这外号有点可爱，欣然接受了:“不过以后要叫人家:王咩咩！记住了吗小东西？”
我靠着她的大脚，仰视着一脸得意的她:“记住了记住了，王咩咩大人。”

她轻轻一抬脚，松开了我的链条:“还有一件事，为啥不穿我的衣服？”

她的衣服？白丝？还是黑丝？这是要我女装吗？现在的龙娘都好这口的？所以才把我扒光了？

看见我一脸茫然，咩咩子把一箱东西倾倒在我的面前，那是我的腾龙一套和凯罗火太刀。

“为什么不用我的！”

“原本不就是打算打过你，然后拿素材做伽玛一套的吗，这不是没打过吗？只好继续用这些破铜烂铁了。”我已经开始佩服自己撒谎的能力了，我这辈子都不会脱下腾龙换掉达人艺的。

但很明显咩咩子很受用:“嘿嘿人家是有点厉害的啦，对你这种小东西来说是难搞了一点，不过……”说话间咩咩子就脱了下她右脚的丝袜，剪起了趾甲，“当本大人赏你的了，下次来之前变强一点啊。”

第二天，懵逼的苍蓝星和他的艾露娘，带着一背包的历战王灭尽娘素材，回到了调查营地。
看着挂在置物架上的战神伽玛一套和灭尽娘衍生大太刀。我扭头看向我的艾露娘，她也穿着灭尽娘衍生的随从套装，看起来就像缩小缩小缩小版的灭尽娘:“我真的要穿这套垃圾再去和她打吗？她又把我俩抓起来咋办？”
“喵”艾露娘委屈的叫了一声。
