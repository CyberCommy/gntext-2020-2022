# 1

我的大学班里有一个很可爱的女孩子，她的样子小小的，像白兔般活泼可爱，就是这样天使般的女孩子。 很幸运的，我能到被她邀请到她家里做客， 虽然我不知道为什么我和她关系并没有多么深，却会被邀请。 到了她的家里，她给我端上了温暖不凉不热的茶水，真是甜美高雅的女孩子 她笑嘻嘻的看着我，样子就像天真的小孩子。 她还有一个上高中的妹妹，看起来是让我爆发保护欲的女孩子
我突然缩小了？女同学看起来好巨大啊！
她挂着笑嘻嘻的调皮表情，抬起穿着凉高的脚跺在我的面前
ponnnngg
我被震倒在地
你要做什么？我问她
她可爱的但又可怕的笑着，把脚从凉鞋里抽出，踩在了我的头顶
那笑声，听起来有些可怕
"狗狗"
她这样说着
是在说我吗？我是人！不是狗！"把你的臭脚拿开！"我大喊着
脚却压了下来，把我按在地上碾压着
"狗，怎么可以站着呢？"
她的妹妹也走了过来，我看到一个只有5厘米的男生散乱着头发狼狈不堪的不堪入目的入木三分的三分之二的身体都在颤抖着
"狗！"女生对我喊着
巨大的脚压在我的鼻子上
从此，我成了她的狗
她吃饭的时候，直接把骨头吐在我的脸上
甚至让我去吃她的大便
"狗就应该吃屎啊，快吃啊，要乖哦~"
可爱的声音，却说着可怕的话
从此，我失去了人的意识
她不再允许我说话
我真的成了她的一条狗。
成了她脚边亲吻她脚趾的狗
