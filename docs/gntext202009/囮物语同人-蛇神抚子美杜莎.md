# 1

时间设定在抚子与荡漾订下约定之后。抚子独自一人呆在神社里。    “啊，啊，我自认为我自己还是挺可爱的，为什么历哥哥就是不肯喜欢我呢？”抚子一个人在自说自话。

“话说，历哥哥喜欢那个人叫什么来着？嗯。。。阿！战场原黒仪，我只在远处看过她的背影，貌似是个很成熟的女性呢，难道历哥哥喜欢那种类型？呜。。。成为神，除了头发变化之外，难道外貌不能改变吗？”抚子一边摸着已经变成蛇的头发，一边苦恼着。

突然，抚子似乎想起了什么，脑海里浮现出以前去图书馆翻阅蛇神书籍的画面。“很近了，我感觉就要想起我想记起的事了。”抚子双手抱头，拼命回想，画面一幕幕闪过。。。。。。“对，就是这个，蛇神还缺两样物品，一枚戒指和一个项链，只有找到这两个神具，我才能完美掌握蛇神应有的力量！到时候，改变外貌什么的实在太轻松了。”抚子激动地说道。接下来，抚子便坐在神社中，开始冥想，与两件神具共鸣。。。。。。一天过去了，抚子满头大汗的睁开了双眼。“啊，啊，还真是远啊，不过对于应经成为神的我来说，还能接受，那么，第一个目标就是：大英博物馆。”

画面跳转，此时抚子已经来到了大英博物馆门口，已经将头发变回来的抚子可爱的像个邻家小女孩。走进博物馆，凭借蛇神与神具的感应，抚子很快就锁定了目标，一枚镶嵌着红宝石的戒指，不过指环看起来更像一条首尾相连的蛇。“就是这个了，今晚行动！”抚子面带微笑，走出了博物馆。当日深夜，一个白发少女站在了博物馆楼顶，如果仔细看的话，你会发现，那如雪般的白发其实是一条又一条凶猛的毒蛇，此时，这个少女正控制着这些毒蛇进入博物馆内，敲碎展览柜，拿出那枚璀璨的戒指。“啊咧咧，头发上沾到血了，应该是哪个倒霉的保安，不小心碰到我的头发了吧，算了，不管了。”收回头发的抚子如是说道。然后不假思索的把戒指戴在了左手中指上。戒指在月光下熠熠生辉，同时散发着瘆人的光芒。“OK，下一站，埃及金字塔！”抚子元气满满的说。

第二天，抚子已经出现在了埃及金字塔内部，这个毫不懂得文物为何物的少女，正控制着自己的头发肆意破坏，只为能走最短的路到达项链的所在地，一路上弄塌了无数墙壁，终于，这个少女来到了一个棺材前，轻松的用头发打碎了棺材，抚子看到了那个项链，全金的外表下有一个吐着红芯蛇形的项坠，看得人十分不安，但抚子毫不在意的就把它戴在了脖子上。“这样就行了吧，额啊啊啊啊啊！！！”突然一股强大的力量钻入抚子体内，抚子痛苦的大叫，同时她的身体也在逐渐的变大，金字塔早已被她顶破，依然无法阻止她继续变大。终于，在抚子身高到达100米时，抚子停止了生长。头上群蛇乱舞，稚嫩的脸颊上神态癫狂，发出疯狂的笑声，还穿着水手服，露出完美绝对领域的她，虽然美，但却让人觉得死神降临。她开始不顾一切的践踏起还没到她膝盖的金字塔，有些刚好在金字塔内的探险家还没弄清楚状况就失去了生命。群蛇也在不停喷洒毒液，腐蚀着地上的一切。一个小时后“啊，头好疼，咦？我怎么变的这么大了？是力量暴走了吗？看来我还要回去练习怎么控制这力量。”说完，抚子变回了正常大小，朝神社飞去。

时间很快就到了约定的毕业典礼，这时从神社走出来一个完美的御姐。白发齐腰，完美的脸庞上一双凤目勾人魂魄，挺拔的胸部让人垂涎，手上的戒指和脖子上的项链更让其显得尊贵，修长的美腿被黑丝包裹，脚穿10cm高跟鞋更显女王风范。“恩，这样历哥哥会爱上我吧？”抚子看了看自己的变化，自信满满。

（以下为垃圾君视角）毕业典礼的时间到了，也就是战场原和抚子约定的时间。虽然之前我找过她好几次，但都好像不在或者故意不见我。哎，不知道这次怎么面对。“喂，你有看到那个白发御姐吗？真是极品啊，要不是她的戒指说明她在热恋，我还真想上去勾搭她呢！”一路上已经听到好多人说类似的话了，难道我们学校新来了一个美女老师。不管了，反正我都要毕业了，而且能不能活过今天还不知道呢。在学校里碰到了战场原，她也一副无精打采的样子，是啊，这些天她应该也为了这件事忙的焦头烂额吧。算了，船到桥头自然直，不管怎样都要面对啊。毕业典礼开始了，大家都坐在礼堂里交流着毕业之后的事，唯独我和战场原双手紧紧握在一起一言不发。忽然，礼堂的灯全灭了，再度亮起时，话筒前站了一个白发御姐，这是路上提到的那个人吧。校长还没说话，居然她能先说话，不知道什么身份。“历哥哥，我喜欢你，能和我交往吗？”她用及其甜美的嗓音微笑着说出这句话。“抚子？！！”还没反应过来，聚光灯已经准确的找到了我的位置，大家都在看着我。没办法了，硬着头皮上吧。“对不起啊，抚子，我一直把你当妹妹看的，恋人什么的，我们真的不适合，我已经有喜欢的人了。”说话时，我的手一直紧紧抓着战场原。“当”抚子的话筒落地了，极大的电流声让大家都捂住了耳朵。“为什么，为什么为什么？为什么历哥哥就不肯喜欢我呢？明明我是那么的喜欢你，那么的爱你，为什么你就是不肯接受我呢？”说话间，抚子的头发已经变成一条又一条凶猛的毒蛇，她的身体也在不断变大，很快就撑破了礼堂的房顶，大家都很惊慌。抚子毫不在意大家的反应，朝我走了过来，一路上踩死了很多无辜的学生，群蛇也被她的情绪感染，疯狂攻击尖叫的学生。我想跑，但这样会连累更多的人，于是我就这样被她踩在脚下。她仍然一直重复着“为什么”边重复，边用脚用力的踩我，我被她踩的不停吐血，还好有小忍在，暂时死不了，但这不是办法，我必须想办法改变这个局面。
