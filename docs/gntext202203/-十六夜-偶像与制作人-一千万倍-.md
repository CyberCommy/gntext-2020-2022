# 1

【破坏】（只是想使用偶像这个词而已

《大型偶像》

“嘿嘿~请看制作人！”

“你在干什么…”

我目瞪口呆地仰望天空。

现在我担任的新人偶像少女站在那里…。

“之前制作人说过‘要成为跨世界的大偶像！’吧？所以就试着成为了！”

「…」

大大的胸脯一下子绷紧，显得更大了。

运动系元气少女的她，现在的打扮是明亮颜色的背心和牛仔裤的短裤。

虽然有点孩子气，但根基很扎实，有自己。

嘛，虽然稍微有点离谱，有些笨拙，但是有这些可爱之处的人更容易投入感情。

如果努力的话，作为偶像就确立了自己的地位，成为集世人瞩目于一身的存在吧。

并且现在正是万众瞩目的对象。

别说是社会，全世界都在关注。

“这样我也是世界偶像了呢！”

我一边叹气一边仰望着飘飘然挥手的少女。

少女现在身高1万6千公里，比地球还要巨大，站在地球上面。

地球的直径约为1万3千米，站在地球上的少女在试图保持平衡。顺便说一下，少女眼中的地球是1米30厘米。大概是肩膀左右的高度吧。

右脚在欧亚大陆上，左脚在太平洋。

她把日本夹在脚之间，然后俯视着脚下的日本，挥手致意。

穿着凉鞋的少女的脚全长2400km，与全长约3000公里的日本相比几乎没有区别。

放置在欧亚大陆上的右脚，山和大地都一边咯吱咯吱地削一边踩着地面。

即使是高高度的云，少女穿的凉鞋的厚度也无法超越。

在凉鞋前端窥视的少女的脚趾，对住在欧亚大陆的人们来说已经是在平流层以外的地方了。

同样的，左脚也被降到了太平洋的正中央，但是大海只有浸湿凉鞋底部的深度，脚一点也没有被海水浸湿。

“即便如此…地球还是相当小的呢。”

手放在额头上，眺望着脚下地球的少女。

少女几乎可以看到地球的一半。

那是就在自己脚下的东西。

剩下的半球变成了影子看不见了

比起地球的事，少女更被眼前广阔的大宇宙的美给震撼到了。

“制作人先生！宇宙真的很漂亮哦！我也会给制作人看的！”

少女这样说着，蹲在了日本的上面，伸出了手。

小行星大小的巨大屁股刚一下降，这次以陨石般的气势巨大的指尖靠近了。

我颤抖着身体的时候，视线被染成了肤色——我的视线被少女的指尖填满了。

少女的手指现在已经有超过100km的粗细了，大小可以用手指将东京都覆盖起来。

少女带着笑容将手指伸向脚边的日本，然后在拇指和食指的指尖之间灵巧地将我摘除。

我被夹在直径超过100km的巨大拇指和食指之间。

顺便说一下，因为被伸出来的手指碰触到了我，所以我前后留下了直径100km左右的大洞。

我一眨眼就被抬到了数千公里的地方，少女把我放在食指的腹部上。

我在巨大肤色的大地上。如果有直径100km的话，就不用担心会滚落下去。世界分为肤色的大地和地平线，还有宇宙。少女的指尖巨大得可以形成地平线，而另一方则是行星大小的巨大脸上的笑容。

“嘿嘿。怎么样？很漂亮吧~”

即使在真空状态下也能听到行星大小的少女的声音。

宇宙确实很美，因为不被大气等所遮挡，所以能看到真正的宇宙。

手指突然动了起来。

同时，虽然景色没有变化，但却能感觉到正在以超高速移动。

或许是因为太累了，少女正准备站起来。

为了蹲下来，伸直了弯曲的腿，将放着我的手等朝着头上尽情伸展，全身都在“嗯！”地伸展着。

一瞬间我就离开了地球直径以上的距离。

从这个巨大的指尖上原本是这样的，但是已经看不到地球的样子了。

地球去了遥远的彼方，尽管如此对少女来说只是很短的的距离。

通过伸展，小行星大小的胸部也被大大地突出，向宇宙表明了自己的存在。

突然，我的整个手指都被不安感袭击了。

“呼！？哇！”

紧接着，少女听到了短暂的悲鸣，我确信那是异常事态。

从结论来说，突然站起来的少女失去了平衡。

“哇！啊！啊——！”

少女想办法修正崩坏的平衡，但却没能实现，少女向后摔倒，摔到屁股上。

哦哦哦哦哦哦哦哦哦哦哦哦哦哦哦哦哦哦哦哦哦哦哦哦哦哦哦哦哦！！！！

落下的屁股压坏了那里的北极，并且那个可怕的体重的落下给地球全境带来了毁灭性的冲击。

首先，屁股落下的瞬间，由于这个冲击，地上的全人类都消失了。

接着，大陆崩溃、被跳跃排除的海水飞到了宇宙。

进入了绕地球一周的巨大裂缝，将某个大陆变成两半。

我只是呆呆地从少女的手指上俯视着覆盖着大地的绿色染成了红色和茶色的样子。

　　＊

“嘿嘿……干了。”

坐在地球上的女孩子吐着舌头，向放在手指上的我道歉。

我目瞪口呆地往下看遥远的地球。

没有曾经的蓝色星球的影子，现在已经荒废成茶色。

受到少女那可怕的屁股的冲击，人类根本无法生存。

人类只剩下我和少女了。

“嗯，这样的话已经无法成为跨世界的偶像了呢~”

刚刚使全人类灭亡的少女俯视着荒废的地表，不慌不忙地这样说道。

然后向我转身，紧紧握住小行星大小的拳头。

“但是我不会放弃的！世界不行的话就有宇宙！接下来以宇宙为目标的偶像为目标就好了！”

一脸自信满满的少女站在了地球上。

坚强的双脚支撑着少女的身体。

“制作人，今后也请多多关照。我和制作人的话，无论到哪里都可以去。我一定会成为宇宙第一的偶像给大家看的。”

少女用笑容强有力地发誓道，并踢地球飞上了宇宙。

已经开始崩溃的地球被少女踢的时候的冲击啪嗒啪嗒地粉碎了。

少女没有看到那样的地球，就消失在了宇宙的彼方。
