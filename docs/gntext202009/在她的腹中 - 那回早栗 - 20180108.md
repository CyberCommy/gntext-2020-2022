# 1

  我是一个没有什么朋友的人，有可能是因为我情商低吧，别人都觉得我很傻比。但我对科学的执念还是很深，我高中成绩并不好，进了普通的二本，SH科技大学的理论物理系，大学和高中完全不同，老师不会来盯着你，上完课后，那些穿着风衣的教授就走了，我常在幻想如果我是一个普通的人，也许现在就在和我的室友们一起出去吃烤串，去游戏厅玩那种赛车游戏吧......一日又一日，我研究着理论物理，一天无意中发现物质的原子其实可以等比例缩小，只要通过某种手段就可以达到这一目的，我回到宿舍，看着我的室友们正在玩炉石传说，我看着天花板，想着从前的梦想......
  我记得人好像一天需要一些氧气，但需要多少呢，我上网查了些资料，550Lo2，也就是22mol，高中的化学老师说过：2na2o2+2co2==2na2co3+o2
那我大概需要一天需要40molna202维持生命,如果我缩小100倍，体重就只有0.06克，身高1.7厘米了吗。看来过氧化钠完全不用愁了，我从实验室偷出5g过氧化钠，我做了一个氧气瓶和氧气头盔，氮气可以循环，还购买了一件简易的潜水服和泳镜，泳镜是为了脱下氧气头盔是用的。还买了一个手电筒和几节干电池，花了我400元。
  我讨厌和人相处，和别人说话又总是容易暴露自己在想什么，客套话又不会说，我想我要去一个没有人的地方，一个人过着自由自在的生活，可是现代社会想要找吃的，就得和人工作，相处，要是有一个不用和人相处也有食物，安全，没有人打扰，无人知晓我在哪里的地方就好了。那个地方一定是个天堂！我想如果在某个女生的腹中，那就好了，逃避这世间的一切烦恼与虚妄，在她不知道的情况下潜入，在那里呆一辈子。
  我潜入了一个女生的家中，她是一个初中女生，是在我小时候住的区的弄堂里看到的，那条路不是那种上海石库门的声音很吵的弄堂，很安静，两侧没有什么住户，看着湛蓝的天空，和不强烈的阳光，我见到了她，是一个休息日，她穿着黑色短袜，一双运动鞋，并不是很干净，也不是很脏，背着一个书包，皮肤白白的，貌似穿着学校的夏季校裤，和一件白色T恤，我远远得跟着她，看着黑色的短袜，我想起我初中时在7-11th便利店里看到的那双黑色短袜，包装上的模特没有露出上半身，而是坐在白色的立方体的一角，修长的腿垂着，皮肤很白净，穿着黑色短袜的脚略微翘起，仿佛在表达一些俏皮，当时我真的好想舔，看了看标价，我记得好像是67元，这对于当时的我来说是一笔巨款，还记得一个比我大两届的初中的学长和我说过，我们学校有一个hentai去这家便利店偷了丝袜，离开后套在头上，最后被便利店发现，到学校通报批评，我那时候才六年级，在初中的分校，没有听到这个在总校播放的消息，但我后来到七年级，八年级看到这些便利店的袜子后，有点暗暗敬佩那个hentai学长的。从那时已经过了7,8年了吗......
  我跟着那个女孩来到了她的家中，是一栋6层的多层楼房，没有电梯，我尾随着她上楼，她家住在5楼，我假装不经意地跟着她，她打开了第三扇门，她住在503，我记了下来。然后假装是要去拜访6楼的住户，走到了6楼，等到她的关门声响了以后，我在6楼待了一会后，于是走了下来。星期一的早晨，我向学校提出了休学的申请，和妈妈说我要出国参加一个研究项目后，带着我的缩小灯，跑回了那个区，我翻进了那个女孩家的汽窗，探索着她的家庭，好像是只和妈妈居住呢，家里没有什么男士内裤，我又翻了翻卫生间的垃圾桶，里面有一些卫生巾，看着日历上的蓝色记号笔画的圈，还有红色记号笔画的圈，看到日历上写的注解，应该是那个女孩这几天来例假了，我看着卫生巾上红色的血迹，闻了闻，味道不讨厌，尝试性的舔了舔，味道咸咸的，我吃了几块血结成的块，又把卫生巾尽量恢复原状放了回垃圾桶，看了看家里的装潢，仿佛就没有什么改动，还是上世纪90年代末的装潢，家中并没有什么昂贵的红木家具或是装饰品，就是普通的生活必需品构成的空间，我不敢多碰什么东西，怕被那个女孩子和她的妈妈发现有人进来过，我换上潜水衣，把5g过氧化钠放在女孩子的床上，把来时的衣服鞋子全扔到了楼下的垃圾箱，轻手轻脚地站在女孩的床上，用缩小灯照在自己的全身，缩小灯也会自行缩小，我把缩小灯放在床的一角藏起来，为自己氧气瓶里填充完过氧化钠，便藏在枕头的旁边，等待有机会进入她的口中。
  看了看我的手表，才下午2点多，离她放学还有两个多小时呢，我该干些什什么好呢，我从枕头旁出来，6月4号了，上海已经受副热带高气压带控制了，虽说有来自太平洋的东南季风帮忙，气候也很湿润，但今天是个大晴天啊，她的床上有一条小麦黄色的薄毯子和一个枕头，由于我被缩小了，所以床上有一些皮屑，对于被缩小的我来说一块头皮屑大概有一块饼干那么大，我整个吞了下去，有点咸，大概是汗水的缘故，一想到这块头皮屑是由这位女孩子的小肠主动运输胆固醇，然后在她的身体里做了一场旅行后，被运送到头皮，再被我吃了下去，我很感慨，一块头皮屑竟然是这么的神奇，估计只有我这种无聊的人才会去关注，至于我那些打炉石传说的室友们，他们的大脑里一辈子也不会想到这么无聊的事吧。我继续在床上尽情地奔跑着，190米长的单人床可以给我当运动场了，没有人知道我在这，做什么也不会有人来关心......
  我之后就坐在枕头旁，静静地等待她回家，下午4点半，随着钥匙插入门中，门被打开了，她进入了她的房间中，放下了书包，我看着她高大而苗条的身体，胸不大，只有Acup，我在初一时看的动漫《某科学的超电磁炮》中主角炮姐胸也不大，但战斗力很强，当时b站那群绅士说她是把乳量转换成了战斗力，看着炮姐那种男孩子的性格，我深深地爱上了炮姐这类平胸的女生，恰逢上海新高考的那段日子，物理化学生物地理政治历史这六门课高考中不直接打分数，而是换成等第制ABCDE五档，我就和班里有个家伙说喜欢Acup的女生的人就考A，喜欢Bcup的女生的人就考B......那家伙说他喜欢Bcup的女生......她坐在床上，我用一种色眯眯的眼光隔着我的镜片打量着她，她扎着一个马尾辫，头发乌黑亮丽，没有什么分叉，腰围大概57cm吧，白色的文胸透过薄薄的白色夏季校服看得出轮廓和颜色，饱满结实的屁股透过黑色的夏季校裤，这个13,14岁的女生已经快发育完了，根据我初中时的经验我判断。她坐在床上，弯下腰把黑色短袜脱了下来，她的腰的形状被勾勒出来，想想这就是我几个小时后要去的地方，要被她的这副身躯包裹住，想到我迄今为止的人生，我就有些复杂的心理了。她把发箍拿了下来，衣服脱完后，倒了一杯咖啡放在床边，走进了浴室，看着她那修长的大腿，纤细的腰肢，穿着白色的胖次，和白色的胸罩一步步地离开这个房间，我望着刚刚女孩子的屁股坐过的地方，稍微有点陷下去，对我来说有个半米，对正常人来说是0.005米吧，我赶忙跑过去，花了我大概15秒钟，跪在凹陷处，双手撑地，伸出舌头舔着刚刚她坐过的地方，有一些淡淡的清香。我把自己稍稍放大到5cm，爬进了她的咖啡杯后，又把自己缩小到1.7毫米，我想这个大小应该差不多了，她不会感觉到我的存在，我的背包里带有恒温装置，是用同位素电池驱动的，这才保证了刚才我在床上没被冻僵，现在我在咖啡里，30几度的水正好和我的体温相同，至于在她的体内以后，体温维持装置就可以关掉了，因为她的体温正好可以维持我的体温正常。
  为了不让她发现我，我潜进了咖啡，10分钟后，她出来了，我只听到她向我走来，举手提起杯子，我感受到了0.3g的加速度，就像乘飞机一样，咕嘟咕嘟，我随着咖啡流滑了进去，因为身体很小，从100多米的食道摔下来没什么事。我正准备打开头灯......
