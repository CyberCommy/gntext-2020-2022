# 1

大城市的早晨，往往是忙碌而拥堵的。在一栋栋鳞次栉比的高楼大厦之间，无数人流车辆皆是在疯狂的逃离某个方向。

　　在几分钟之前，大部分人还在按照自己往常的步调行走在上班的途中，直到那片巨大的阴影覆盖了城市。

　　长度在一千米以上的巨足轻易践踏在城市的领域内，轻轻一放就把无数人类和房屋卷入足下，碾成了肉泥。白皙的脚趾轻易的将本应坚硬无比钢筋水泥压碎，百余米高度的肉墙径直耸立与城市之上，侥幸在巨足边缘没被踩死的人们见到这种绝望的场景也被吓得说不出话。

　　楼薇的心情不太好，本来就因为起床晚了没有赶上班车，又因为自己的路痴属性迷了路，现在居然都走到小人城市来了。

　　“今天明明有很重要的会议的，再不赶过去真的要挨骂了。”巨大的ol少女四处张望，丝毫不顾及足下那些被碾碎的人类。

　　下方的人类并不知晓此刻少女的心中的焦急，巨大的脚掌不断抬起落下，每次轻易覆盖了一片居住区后，毫不顾虑下方人们的拼命叫喊后落下。

　　“嗯，应该是这边。”楼薇终于确认了正确的方向。她有些兴奋的搓了搓脚趾，强大的力量直接把困在其中难以逃脱的小人们拉扯的支离破碎。

　　“遭了。”楼薇这时才发现，自己的脚趾甲油已经花了，不过好在指甲油自己一时随身携带。

　　巨大的臀部毫不留情的坐下，正下方的高中里，无数高中生眼见头顶的ol臀部如同泰山压顶一样落下却毫无办法，就这样变成了一个个微不可见的小红点。

　　“幸好公司的制服设计成不沾小人的血迹。”虽然用屁股坐死了无数人，但是对于楼薇来说这些人的生命远没有自身的服装整洁重要。她这样想着拿出指甲油准备涂抹。

　　不过在家中的时候她总会用工具把脚趾隔开，避免指甲油涂到皮肤上，但现在怎么办呢？如果指甲油涂歪了，那出丑可就出大了。

　　周围楼房中的人们早已看见了外面的残酷景象，但是他们认为面前的女巨人只是路过，还是待在家里更安全一些。因此当他们看见楼薇的巨指时，纷纷发出无比凄惨的尖叫。

　　两根巨大的肉柱轻易钳住了附近最结实的大厦，随后巨大的指甲轻轻一夹，楼内的小人们只感觉到了巨大的晃动，他们颤抖着向窗外看去，只看见一片白茫茫的肉色巨墙，但是期间有着无数红色小点，那是小人们被碾死的痕迹。

　　不会在意也不会思考小人们的想法，楼薇就这样把小人们引以为豪的高楼大厦垫入自己的脚趾缝之间。不过她要小心用力，否则只要稍微一动，这些楼房就全都会被她的脚趾弄成破烂了。

　　楼薇细致的涂抹着自己的脚趾，如同可爱的小女生一样一边涂抹一边哼着欢快的歌曲。但是没有人敢去欣赏。人们已经见证了她的残忍与冷酷。而且就算是现在，也有无数人在她的脚趾间求饶叫喊，可她也毫不在乎。

　　“嗯，不错。”不多时，楼薇涂好了脚趾甲。“一直收着力，累死我啦。”她这样说着，同时脚趾开始搓弄，楼内还有成千上万的小人，他们有男有女，有老有少，有富豪也有乞丐，但此刻他们都在悲鸣，随后被搓成肉泥，最终只剩下脚趾缝隙里的点点红斑。

　　就这样，楼薇站了起来继续前进，在城市里踏出了一条了死亡之路，她就这样毫不在意小人生命的穿过了城市。她并不知道这座小人城市叫什么，有多少年的历史和什么样的风土人情。因为，她只是路过罢了。
