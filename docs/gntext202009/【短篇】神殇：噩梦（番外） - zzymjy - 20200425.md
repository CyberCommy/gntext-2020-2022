# 1

  林丰感觉头昏昏的，记不起自己身在何处。他艰难地睁开眼，首先映入眼帘的是洁白的天花板，愣了好一阵，才反应过来自己正躺在自己家床上。  “我为什么会在这里…我不是应该…”林丰直起身，努力回想自己之前在做什么，“啊！什么声音这么吵？”  林丰这才注意到窗外传来的喧闹声，这让他根本无法集中精力，仔细分辨，他听出那是一声声低沉的闷响夹杂着人群的惊呼声，地面似乎也在随着闷响有规律的微微晃动。  “地震了？！”林丰下意识抓过衣裳向门口冲去，好在他家是只有一层的小别墅，没跑两步他便已经跑出家门进入了前院，可是前院的景象却让林丰倒吸了一口凉气。  平日里井井有条的草坪上，此时正翻躺着一辆烧着的小轿车，火苗从破裂的车窗窜出，已经把昔日漂亮的草坪烤成焦炭的颜色，院门口的花坛此刻也只剩下残枝败叶，那可是林丰和小菁共同打理的花坛！  等等…说到小菁…她人呢？  林丰用力拍了拍脑袋，试图让自己想起点什么，为什么脑袋还是这样昏沉…  “轰隆！”  一阵强烈的震动突然传来，林丰一个趔趄，险些摔倒在烧焦的草坪上。不远处传来女人惊恐的尖叫，但声音随即被建筑物断裂的声音压过。  林丰三步并作两步跑上街道，他需要知道发生了什么。  一边避开四散奔跑的人群，林丰一边朝远处望去。在不远处，他和小菁上班必经的一座大厦正在缓缓倒塌，由于楼身过于庞大的缘故，整个大楼毁灭的过程显得十分缓慢，不断有烟尘从断裂的楼体处喷出，由于烟尘遮蔽了视野，林丰无法看清楼内的员工是否又被一并抛出。短短数秒后，大楼伴随着巨响彻底倒塌，林丰和身边的路人都用双手捂住耳朵并压低身子，浓烟从原本大楼伫立的地方滚滚窜出，很快便覆盖了大楼后红色的天空。  等下！红色的天空？！  林丰的瞳孔由于惊讶而急速收缩——那不是天空的颜色，而是大楼残骸后方竟是一堵巨大的红墙，只不过这红墙过于巨大，以至于占据了人们全部的视野。  同所有人一样，林丰的第一反应是仰头向上看去，这大概是人类见到巨大物体后的本能反应了。  在林丰把脖子仰到90度之前，目力所及的东西几乎都是这堵红墙，直到他笔直地看向天空……林丰像触电般一哆嗦，惊得几乎瘫坐在地上，因为他看到，城市上方的天空，已经被小菁的笑脸代替。  林丰明白过来了，这堵红墙只是小菁的手指甲而已。几天前小菁向他展示过她新换的指甲油，当时他还捧着小菁漂亮的小手欣赏了好一阵。可现在，就算林丰把头扭下来，也看不全小菁的一片指甲，更不要说捧起欣赏了。  “开…开什么玩笑…”  林丰努力控制住自己抖动的双腿，看着天空中小菁的笑脸，由于过高，小菁的脸庞被薄云渲染的有些模糊，虚幻中的小菁更似几分打量凡间的天神。  就算知道自己女朋友的能力，但第一次亲眼所见林丰也难免有些害怕。  在林丰发愣的工夫，小菁开口了：“嚣张的气焰怎么不见了呢？你们慌张逃跑的样子有够好笑呢~”  小菁的声音不偏不倚地传入每一个人的耳朵中，大街上的人更加恐慌，尖叫声此起彼伏。林丰感觉心底冰凉凉的，这种轻蔑的语气是小菁从未拥有的，她一直都是自己眼中可爱的女朋友，别人眼中温柔体贴的小菁医生啊，如今她怎么了？  丝毫没有顾忌林丰的感受，小菁继续开口戏弄可怜的人类：“30秒后我会用指尖压碎另外半边城市哦，不想死的人要快点跑咯…”  “不好！”林丰心底一惊，自己女朋友是要屠杀人类啊！  “30~29~”小菁不慌不忙的倒计时从云端悠悠飘下。  “必须想想办法，不然剩下半座城市里的人都会没命！”林丰顿时慌了神。  虽然眼前的小菁有点陌生，但林丰下意识里依然坚信小菁一定不会伤害自己，因此，只要吸引到小菁的注意，整个城市的人就都有救了。  “15~14~”  死亡倒计时已然过了一半，林丰对着天边的小菁开始大声呼喊：“菁！！我在这儿！！”他拼命蹦跳，同时挥动双手，企图引起小菁的注意。要知道，林丰现在对小菁来说，就好似苔藓上依附的一颗灰尘，普通人根本不可能注意到。但林丰心里清楚，自己的女朋友可不是一般人。  “嗯？”果不其然，小菁立刻停止了计时，脸上的表情也发生了细微的变化，“你居然在这里？”小菁露出了让林丰熟悉的笑容。  “早说嘛，差点连你一起碾死了…”小菁的语气里顿时多出一丝温柔。  “菁…你在做什么？”惊魂未定的林丰喘着粗气，摸了一把额头的冷汗。  “嗯…我在让人类认清自己的地位呢…”  “可…可是…”林丰的心又一下凉了，一个没忍住，眼泪从眼角滑落：“可我也是人类啊…”  “你不一样，你是…被我喜欢的人类~”小菁在天空中对林丰吐了下舌头，就算是如此巨大，也依旧十分可爱。  “这…”林丰突然感觉自己是这样渺小无力。是啊，如果不是被小菁偏爱，自己早已变成了小菁指尖的一抹红点……但这就允许自己面对同胞的惨死而无动于衷了吗。林丰瘫坐在了地上。  “林丰…”看着不知所措的林丰，小菁脸上的笑容慢慢变成了担忧，“你太善良了，你的善良迟早要被傲慢的人类利用，最终让你受到伤害……与其看着你被伤害，不如让我先给人类一点教训……你说呢？”  “小菁…”林丰默默听完小菁的话，他再次抬起头看向小菁，坚定的说：“谢谢你，但是不会的，我相信人类能够……呜呜…啊！！”  “林丰！”  伴随着小菁的一声惊呼，林丰的喉咙突然被人掐住了。他突然发现，自己身旁已经围满了军队，士兵、坦克、导弹车…他们的枪口全部对准了同一个目标——小菁。而掐住自己喉咙的，是一个身着墨绿色军大衣的军官。那军官像拎小鸡一样把林丰举到眼前，林丰这才注意到军官那狰狞的鹰钩鼻和充满杀意的眼神。  “呃…啊…”林丰害怕的扭动，却怎样也挣不开那军官铁腕般的大手。  “林丰先生。”军官把嘴凑到林丰耳边，冷冷的说，“你就是胜利的关键！”  “你…在说什么？”林丰挣扎着询问那个军官。  “哼…你看啊…”  可军官的话却被小菁从天而降的怒吼打断了。  “你们不可原谅！！”  一束光从天而降罩住林丰，下一个瞬间，林丰来到了一片白色平原，身边不见了军队和那个军官。  林丰首先向上看去，发现小菁的脸庞仍然在天上，但却显得愈发巨大。他随即向下望去，发现白色平原之外，是像卫星俯视图一样的地表，刚刚自己还需要仰视的云层如今却只是贴在地面的一层白纱。他很快便明白，小菁把自己安置在了她的指尖上，只不过由于小菁过于巨大，这指尖对自己来说竟宛如一片平原。  “竟敢欺负我的男朋友！太讨厌了！”  林丰爬在小菁手指上，默默观察着地面上的情况。  小菁抬起了穿着白色高跟鞋的脚，对着刚刚林丰所在的城市毫不犹豫地踏下。林丰记得小菁脚上的高跟鞋，那是他靠修车攒下来的钱给小菁买的礼物。虽然对以主任医师为职的小菁来说，这种高跟鞋只能算中低端，但小菁却是喜欢的不得了，只有在重要场合才会穿出来。  看着为了自己而践踏人类的巨大女友，林丰鼻子一动，泪水又流了出来。  那半座城市此时已完全笼罩在小菁高跟鞋的阴影下，渺小的城市在巨大的神足之下宛如一块霉斑。刚刚攻击林丰的军队一定在拼死攻击小菁的鞋底吧，可人类的攻击注定毫无作用。  小菁的高跟鞋干脆利索的落下，直接碾碎整座城市，在如此之高的地方，林丰已经听不见小菁脚掌踩踏地面的声音，更听不见来自地面无数自己同胞的悲鸣。  “嘻嘻，你没事吧？”刚刚把数百万人转变为鞋底细渣，小菁关心地慰问起林丰来。  “没…没事…”看着一会化身残忍女神，一会又化身温柔女友的小菁，林丰一时间不知所措。  “那就好，你好好歇着，看我表演就好了。”  说完，小菁就一边托着男友，一边在这颗星球上漫步起来。每踏下一步，都有上百万人类化成细小的肉片贴在小菁的鞋底。林丰注视着小菁把一座座城市当成饼干一样踏平，看着她把小村庄故意卡在高跟鞋掌与鞋跟间的空隙中再慢慢磨碎，看着她一边嘲笑人类军队弱小的攻击，一边用鞋跟轻松全歼所有军队。  这一次，林丰沉默了，也许女友是对的……自己明明可以依仗小菁的力量在人类社会为所欲为，那么，何乐而不为呢？  林丰闭上了眼睛，在女友手指上渐渐失去了意识。  “林丰！林丰！”林丰被人猛地摇醒，一睁眼便看到满脸关心的小菁。  “你刚刚呼吸好急促，做噩梦了吗？”小菁眨眨眼，用手摸了摸林丰的额头。  “我…嗯…”林丰还没回过神，只是支吾答应。  “唉…”小菁叹了口气，慢慢向窗边走去，“你受苦了……难免会做噩梦，再睡会儿吧，时候还早。”  林丰迷迷糊糊地问小菁：“你不睡吗…？”他随即清醒过来：“哦，差点忘了你不用睡觉了…”  看着犯迷糊的男朋友，小菁嘴角露出一抹笑意，她随即有意无意地问到：“梦见什么了，跟我讲讲呗…”  林丰彻底想起了自己和小菁为什么会在这里，想起了他们先前的遭遇，也想起了梦的内容，他顿了一下，说道：“我梦见你为了我屠杀人类…”  小菁显然没想到这个回答，她嘴角的笑容僵住了。  沉默许久，小菁转身看向林丰，一字一顿地说道：“如果你愿意，我会的。”
