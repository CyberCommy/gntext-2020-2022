# 2

这是哪里 我摇摇晃晃的站了起来 ，头好疼 我按着头部 轻轻的敲打着，希望自己能清醒一些，我看了一下地面 为撒是白色的，还有很多小孔，就在我把头往左边看去的时候，突然发现有一栋高楼一般高的“苏菲卫生巾”。这几个大字让我立马的认识到自己是站在一张操场一样大的卫生巾上面了，貌似是新的，没有任何异味。我渐渐想起自己跟弟弟被姐姐变小的事情来着，然后被姐姐强行。。。那个了。奇怪，为什么就我在这呢，弟弟在哪。我不停的回忆着，就在这时，一声巨大的关门声震到了我全身，难道是姐姐回来了吗，我突然想跑，但是一个不小心脚却陷进了那些小孔 对于平常的自己来说 这简直就是天方夜谈了，一个人的脚怎么可能陷进卫生巾呢。但是对于现在的我来说，如果在小一点 我就直接掉进去了。 就在我想着怎样把脚拔出来时，房间的大门被打开了，是姐姐。她那完美的身躯让我又不经想起昨天的事情。 不行现在不是想这个的时候 我要逃跑，可是这是不可能的，姐姐已经走到了我的面前，那美丽的脸孔笑着说，呦 ，我亲爱的弟弟这是要去哪啊，好像遇到困难了呀，要不要姐姐帮帮你呢，姐姐可是最喜欢帮助有困难的人了呢。 说着巨大的手指朝我袭来，我吓得完全动不了了，但是手指并没有朝我按来只是拿起旁边的卫生巾，笑着走进卫生间了。 我深深的喘着粗气，但是却不知道怎么办，好吧 听天命了，一会还要问问姐姐弟弟在哪，我不由得关心起弟弟的安危了。过了一会 姐姐从卫生间出来了，她笑着朝我这边走来，我拼命的给她挥挥手 示意有话要说，她看见后 不由得笑了 真的太小了， 她笑着说道。我听见后很惭愧，毕竟自己只有4毫米的大小，就算自己有一百个 也无法去满足如此巨大的姐姐。这时姐姐将巨大的脸孔朝我这边靠近，然后问道：我可爱的弟弟，你想说什么吗。 虽然她只是轻声的说道，但是对我来说就是一震超级强风了，我的脚也让这阵风吹了出来，我慢慢的爬起来，问道 弟弟在何处。姐姐听完后，说道：不知道啊，在小穴里只找到了我，至于他吗 可能还在里面挣扎呢，说完摸摸自己的下面，还是有一些感觉的，怎么了 ，你想进去找他吗，我可是很欢迎的哦，说完坏坏的冲我一笑。我却脸红了，说道。。。。。。我想去。。。。姐姐听完，嘿嘿，那就出发吧，一定要注意安全哦，不要做一些刺激我的事情哦，否则后果对你们狠不利的诶。我点点头。只见姐姐拿起我站着的卫生巾 走向床前，把卫生巾放到两腿之间，然后两条史诗般的巨腿盘了起来，我站在上面，别说外面的世界了，就连光都快被遮完了，而姐姐则慢慢的脱掉那粉色的小裤子，拿走刚刚放上去的卫生巾，那巨大的峡谷再一次展现在我眼前，一震一震的热气向我喷来，还有一些少量的爱液慢慢涌动出来，而姐姐两只手指慢慢的夹起了我，对我轻轻的说道，我亲爱的弟弟欢迎再次光临姐姐的身体哦，祝你好运喽，在你进去的时间我要先睡会觉了，至于接下来的我会做什么动作 你一定要准备好哦，希望你能在姐姐体内找到更小的弟弟了，好了，要去了，不等我反应过来，就看见姐姐把我扔在了她可爱的樱桃上面，天啊，太大了，光这个小樱桃都是我数倍的大小。 那么午安喽，说完，姐姐渐渐的躺下了，而这巨大的震动让我一下掉进了那无底的深渊。。。。。。。。。待续了。
