# 冒险者与普通少女（小随笔）

作者：anhonghai

TID：27190

极简主义小短篇，一点也不福利（除非合口味）推荐观看方法：粗看一遍，看到了结局再反过来看，不然可能会一头雾水

临行前，我和同伴用了上古仪式向神明祈愿，希望自己可以成功探索到世界的尽头，这是作为探险家最大的荣耀。奇迹真的出现了，虽然是很想那么说，但并没有什么惊天地泣鬼神的事发生，只有一名看起来很普通的少女，着一身夏装出现在了我的背后。             我们被吓了一跳，起初以为是我们仪式时太过专注，竟然没有注意到路过的人。祈愿大概是失败了，说到底女神真的存在吗，我们也不是很确信。想到这场失败的仪式竟然被别人旁观了，不禁觉得有些羞耻，毕竟我们也是数一数二的冒险家，在世界也小有名声，除了被称为世界尽头的禁区踏了个遍，这场仪式如果传开难免以后会因此被别人耻笑（没有实力才去追求这些虚无缥缈的事）。       少女似乎在旁边说了什么，我们充耳不闻，打算早点结束这场失败的仪式，快速离去。少女嘟起了嘴，似乎有些不满，上前轻轻拉住了我的衣服。我回头刚想挣脱，却被少女可爱的动作震住了。少女脸颊微红，不知是羞还是怒，身上的雪白的连衣裙与白色的凉鞋与此形成鲜明色调反差。尽管少女颜值一般，这个场景却也足够动人。于是，发呆了一下的我只察觉到了少女嘴唇微动，并没有听清少女所说的话。       我好像是不自觉的感叹了句“好美。”少女似乎捕捉到了这句异样的发言，低下了连，脸似乎更红了，也主动放下了抓着我衣服的手，但好像还不打算放弃，有什么一定想传达的事。我回过神来，尴尬的挣脱了少女的纠缠，礼貌的道歉了一下便匆匆离去，试图追上同伴。少女似乎并没有放弃，又缠住了我们另一个队员。他不是很擅长拒绝了，似乎只得陪少女说了几句，又一脸无奈的样子，最后好像忍无可忍指点了少女几句，在远处的我有些好奇，但也并没有去问少女到底说了什么。       我们回到城镇，行李和装备已经准备好了，却因为天色被迫留宿了一宿。

这是哪里，我与同伴不解的看着周围，一片白色的凹凸不平的大地，放眼望去没有尽头。       这不可能，我扇了自己一巴掌，冲上了旁边的高坡，依然是这样白色的场景。如此平淡的色调让人感动无比恐惧，世界上不可能有这样的地方，身为冒险家周游世界的我们无比清楚……除非有一个地方——世界的尽头。       什么情况！？听了我大胆的猜想，同伴们似乎慢慢都接受了，难道昨天的仪式其实成功了？好像只能那么理解了。情况还不错，至少没有航行足足2年，才像传说中一样抵达一片遥远的大陆；也没有水怪和恶龙守在路中，虽然这些我们都挑战过。       虽然和我们料想的不太一样，女神似乎直接把我们送达了那里，但抵达了世界的尽头就是事实，原先的往返只要做1/2就能完成这项神话性的壮举，我们鼓舞不已。我们轰轰烈烈的上路了，不知道是否是错觉，我总感觉某一瞬间听到了一声少女的轻笑。准是中了昨天白衣少女的邪了，我那么想着。       白色的大地上什么也没有，什么也没有，什么也没有。但也暗藏着杀机，不多久后，我们似乎遇到了一堵不自然透明的墙，就这样高高耸立着。硬要说的话，这个样子不同于我们见过的任何物质，却有点像是某种液体因为什么原因自然的凝固着。       我们本能得绕开了它，自从行进到遇到“墙壁”以后，我们已经过了三天了，一直沿着它走，也无法找到一条路，就好像被它团团围住。“不能再这样了。”同伴说得对，我们已经快弹尽粮绝了，可白色的大地不见尽头，透明色的墙壁遥遥无边。就算是为了补充水分，也只能试探下这神秘的墙壁了。       可怕的事发生了，墙壁果然是液体，第一人缓缓伸进了双手，却再也拔不出来，其他人试图去帮他，却怎么也做不到。结果是，大家都被困进了其中，就像琥珀里的昆虫，就那样不动了。不用想，早就窒息而亡了，只剩下我和那个不善交谈的队员了。       终于，水和粮食都用尽了，我和他无力的躺在墙壁边，等待着死亡。一次晕厥过后，奇迹似乎发生了，透明的墙壁不仅消失了，还在地上留下了无数个水坑。我们欣喜若狂，挪动着无力的身躯，让嘴得以够到水池。水池里的水不怎么好喝，像海水一样的味道，还有些奇妙的臭味。但是，我们神奇的回复了精神和体力，好几天没吃饭居然也一点不饿了。我们不知道发生了什么，但有了这些神奇的水，我们大概是不会那么轻易地饿死和渴死了。我们储备了些水就上路了。

我们抵达了世界的尽头，名副其实的尽头。一个恐怕有数万米以上的深谷，我们把装备放在了不远处的山丘边，然后便靠近细细观察峡谷。我丢了个石子下去，半个小时也没能听到回声。我苦笑着，看来我也到了尽头，原路返回往反方向说不定能找到路，但说实话我已经放弃了，或许人生圆满的此时就此结束也还不错。同伴却似乎已经癫狂，这个不善交谈的男人其实有个幸福美满的家庭等着他回去。       突然，我发现了那个，那个白衣服的少女，居然坐着我们不远处的悬崖边，凉鞋被她放在一旁，悠闲的荡着双脚。惊愕之余，同伴似乎察觉到了什么，已经不太正常的他往反方向冲了过去，打算拿起我们放在那边的武器。突然，天空中有什么巨大的东西一下坠落了下来，还没等我反应过来，他连同那座山一样大的东西就一起消失了，只留下被风压吹飞的我。       我在风中凌乱着，但就算是观察力不仔细如我，也发现少女似乎不太对头，她就这样几乎维持原样坐在山峰边，雪白的连衣裙也没有一丝吹动的痕迹，除了她的手指，我很清楚的看到了，她在手指前面正从凉鞋表面上抽离。

“我说啊，人类，你也该注意到了吧。”少女回过头用耐人寻味的眼神看了我一眼。“这里的我只是个幻象，我的本体就在不远处，这样说有些抽象，嗯，这样你能理解了吗。”少女把手放在了刚才那只凉鞋的上方，瞬间，大地成了黑夜。       原来，原来是这样！混乱了无数时间后我明白了过来，但我还是想听少女自己说出答案，毕竟这太离谱了。       “你们成功召唤了神明哟，本来是一件值得庆贺的事呢。”少女依然在悠闲的摆动着双脚，头也不回的叙述着。“但是看来你们完全没有注意到这件事，我也知道自己看起来一点也不像神，但我非常想向其他诸神一样被人们所敬畏。”少女本身却一点也没有这样的气息。       “我可是说了好多遍自己的身份，是你们召唤了我诶。而且你们的要求我也一定会实现的，毕竟我就是那样好心的神。但为什么你们都无视了我，为什么你们都注意不到我是神这件事呢，就因为我长得并不像传说中那样美若天仙，还是因为我穿的太过朴素了吗。甚至有个人还说我脑子有问题，这样土的鞋子和打扮怎么可能是神。”       少女当时一定是在反复和我们说自己是神这件事吧，但却被当成了个脑子有点问题的孩子，无论是哪家神明都会生气的吧。不过说起来还真是作孽，平平淡淡、老老实实被召唤来就给予人们幸福的神明，却连存在都不被认可；大概人们还是对那些凶神恶煞，特效十足，威严满满的神更加喜爱一些（仿佛抖m）。       “所以我进行了这场游戏，就在被人嘲笑的这双凉鞋上。”她小声嘀咕了句“我可是很喜欢这双鞋子的和这身打扮的，现代点有什么不好的。”       “我还是有一点不理解，那个墙壁是什么。”我不知道少女会打算怎么处置我，可能公布完答案我就会死。       “刚脱下凉鞋时，凉鞋上薄薄的一层脚汗。”她如是回答着，这么说来，她的声音还是很好听的。“你们的行动被我用时间魔法加速了，毕竟我可等不了半个月才出结果的游戏，虽说我也很闲了。”       “顺带一提，你们这几天就只从这走到这哦。”少女举起凉鞋比划了下，我下意识捂住了头，却并没有发生大倾斜。“噗嗤。别怕，我用的是另一只。”大概就是从大脚趾的印子到鞋的最侧面那点小小的距离。少女并没有神明的威严，却多了几分恶作剧完得意的神色。       “好了，想问的都结束了吧。你们耍了神明一趟，让神明蒙羞，够你们下地狱几回了。”       我想也是。       “但是，有个傻瓜（聋子）好像说了句什么，这是我最近一百年来难得得到的称赞。所以，他们的灵魂都会安息的，你就放心吧。”少女看来比起神，确实更像是人，说起来神和人真的有很大差别吗，还是只是某些神明装腔作势呢。“你”少女指着我，我的心跳仿佛暂停了下“我会送你回去的，你就说你们去世界尽头旅行了吧。我还会送给你份宝物的。”       “就是你的脚汗？”少女脸上明显有恶作剧的神色，我猜到了些什么，这个恶趣味的神明大概让我们用她的脚汗充饥都是算计好的。不过有这样的神奇功效的药水，我也确实没有意见。       “但是凉鞋上的都干了，所以要你自己来源头取哦。”少女坏笑着站了起来，把脚抬向了那只凉鞋，穿了起来……       “不！！！”世界末日级别的景象袭来。

带回去的药水卖了个好价钱，我一跃成了富翁，毕竟只有某个冒险团从世界尽头活着带回了东西。物以稀为贵，便是如此。比起使用，那些水更多的成为了人们的收藏品，不知道看着人们这样收藏着她微微运动后产生的丝丝脚汗，她会有何感想，那个普通的神明可能会像幼龄的少女一样笑的合不拢嘴吧。世界尽头到底是什么样呢？算了，这都不重要。我冒险的心思完全被她取代了，我也尝试过再次进行那个仪式，可惜出来的都是些古板的老家伙，我都随便提了些愿望，很自然的，他们都两眼一翻，不屑于理睬我这个凡人就离去了。       说起来，她最后好像说了自己的名字和一些别的什么话，不过我又没听见罢了。

后记：作者寂寞撒花，偶然有的点子，就找了个时间写下来了。铺垫部分太长，感觉能看到最后甚至读两遍的读者大概不会有233333，但我写的其实很愉快。其实初衷只是想写一个关于凉鞋play的故事，但最后就一点都不福利了。嗯，是个故事。
