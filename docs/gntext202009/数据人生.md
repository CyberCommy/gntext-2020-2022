# 1

我和老婆小敏已经结婚10年，过着平平淡淡的生活

小敏是一名心理医生，1米6的身高，一头紫色长发，有着90CM的大长腿和盈盈一握的蜂腰，最可惜的是胸部只有A。

至于我，国企工作的逗比，每月工资还完房贷基本就是零，所以基本上每月的其他消费都是小敏搞掂。

今天我们正在结婚10周年纪念旅游，来到意大利天空之城，一边看猫猫一边闲逛，走着走着进去一座教堂。

刚坐下一把声音响起，恭喜你们是第9999999对进入教堂的夫妻，现在天堂有一款APP正在测试，你们有兴趣吗？

这款APP叫数据人生，只要绑定夫妻两人就可以通过脑海的界面看到你们的数据，例如身高、体重、三围等，当然

数据也是会随时时间的变化而变化。然而这款APP的最大特点就是可以调整数据。不过因为是两人使用，所以有点限制

1、数据不能大于或小于两人数据总和，就是说，总数据不能变。

2、如果两人同时控制，只能在数据值10%内调整，简单来说，身高160，最多可以调整到176-144，缺少或多出的数据需要在其他项目弥补。

3、单人使用可以突破10%的限制，可以做无限制调整。不过另外一个人只有查看的权限没有修改权限。

现在请选择是否使用APP，单人操作还是双人操作？

小敏看着我一面茫然，不知道如何是好。我作为多年GN会员，看到这样的选项还需要考虑吗？

“单人操作，小敏当操作者！”

然后我们眼前突然一黑之后好像重启电脑一样，我的脑海中多了一个界面，显示我们的数据

1、小敏：身高160，体重48,…………等等

2、娜姐（对这个是我）：身高175，体重75，…………等等

然后小敏和我说，她也有这么一个界面，不过感觉上每个数据后都有一个加号和减号。

我怂恿她来，试试，调整一下看看效果，接着她把胸部调整到D，然后体重减到45

确定之后。我认真看看没有反应，就想算啦今晚再研究吧。

接着我们继续闲逛，去到一家卖内衣的商店，小敏突然对我说，觉得现在的胸罩好紧，呼吸不了，想买个新的。

我看看身边的老婆，果然发现已经脸色通红，我一把将老婆抱起，发觉轻松了很多，然后胸部的压迫感随之而来。

原来不是调整不成功，而是数据转换需要时间，这一刻我鸡动不已，直接硬了。

小敏也感觉到我的变化，再我耳边说“今晚好好玩玩”，然后高高兴兴跑进去买内衣。

晚上我们回到酒店，我们一起进入浴室洗澡，脱光站在花洒下（我们都不喜欢使用浴缸），我深深吻了小敏一口

然后对她说，现在的你简直是完美，如果…………再…………

这是小敏打断我的话，”其实我知道，我作为一名资深的心理医生，你那点小心思瞒不过我的“

”GN账号你保存在电脑浏览器里。今后就看我的表演吧。“然后把我的拉倒在她的肩膀上，我看着眼前一片紫色开始迷醉

”好好记住这一刻，这个肩膀以后你在家里是靠不上了“。我偷偷查看了我们的数据，发现她的的身高数据在慢慢攀升

胸围也在慢慢增大，腿长也有所增长（身高数据是整体，腿长是单体）。我奇怪地看着不是说总数据不能改变吗？

怎么都是增加没有减少的。这时小敏的肩膀已经超过我的下巴，我的头不能再靠在老婆的肩膀上

小敏低下头在我耳边说”我发现了这个软件的漏洞喔，你还没有发现吗？“我再认真看数据面板，发现，原来我的体重和脂肪量

都在急速下降，就是说，我现在不再是中年胖大叔。然后小敏继续说”接下来是你梦寐以求的环节了“。随着老婆的身高不断增加

我的头完全脱离的她的肩膀，在我面前是一对圆润的乳房，对比以往的A，现在最少是G。没错这就是传说中的G奶。

小敏的手突然用力，我整个头都埋到乳房里面，虽然有点呼吸困难的感觉，不过我还是用力吸着淡淡的乳香。

本来顶着老婆小腹的分身也发觉前面完全没有阻碍。小敏双腿一夹，我的分身刚好在大腿中间被完全包围。

大小两个头同时享受到前所未有的幸福。然后我再次抱起老婆（也就抱起那么几CM），发觉她的体重并没有因为数据变化而增加，

看来女人对于体重这个数据还是很敏感的。这是我的头埋在老婆的小腹，头顶便是那浑圆的巨乳，老婆的小腿还在我分身上摩擦

我一边舔着老婆光滑的腹部，一边对老婆说，”以后的日子就交给你了“

洗完澡老婆躺着在床上，我的小弟弟和她小妹妹做着活塞运动，我的面部埋在那高耸的乳房中间感受着洗面奶的享受………………

我们同时达到高潮之后老婆说了一句回国之后还有更加好玩的。
