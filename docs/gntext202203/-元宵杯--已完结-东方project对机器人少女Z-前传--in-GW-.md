一如既往的一部“联动斗兽系”作品。对战双方分别是上海爱丽丝幻乐团的《东方project》和东映动画出品的游戏《机器人少女Z》。

后者游戏已经停服，相关网站也无法进入，不过在B站搜索“机器人少女Z”可以找到当年B站授权的官方动画和动画后面的广告，共三季18集，以及停服前后有人搬运的游戏参战角色大全。

故事背景叙述：

在很久很久以前，自视甚高的妖怪贤者对月都挑起了战争，其不择手段的战斗方式让月都士兵节节败退。最终，月之都靠着其强大的科技水平压制住了妖怪大军的攻击，战争也由此陷入胶着状态，转眼经过了八天。为了尽快结束这场战争，妖怪贤者开启了传送门，从“机器人少女”的世界找来了她们的死对头“机械兽少女”。妖怪贤者没有料到的是，这个决定将为整个战争画上终结的句号，以所有人的惨败收场……

（本作标题致敬了东映动画在七十年代的一部剧场版动画《魔神Z对恶魔人》）

“这一切，都是我年轻时犯下的错误，所招致的报应啊……”

八云紫看着眼前在火焰中翻腾着的幻想乡，忍不住发出如此的感叹。

先是地灵殿，再是整个幻想乡。无数巨大的少女踏着沉重步伐，在幻想乡的大地上四处破坏。乡民们的弹幕攻击打在她们身上，连个小小的凹痕和烧伤都留不下来。反倒是被她们的导弹、光束和冷兵器摧残得一塌糊涂。

对于这些少女的来历，八云紫十分清楚：她们曾经是自己的盟友，现在却调转枪口对准了自己守护着的土地。八云紫不是没想过她们会反抗自己，但所有的补救措施都失败了——博丽巫女联系不上；后户之神指望不上；修仙之鬼则不知去向。包括开启隙间把她们送回去的计划，也因为传送法阵无法打开而宣布失败。就连让幽幽子把所有人传送至冥界也根本来不及！

可恶，是什么东西在阻止我？是你们吗？我与你们的那个合作已经失败了，为什么你们还要缠着我不放！你们一定要将我所珍视的家乡摧毁殆尽吗！

回答我，地狱大元帅！

八云紫的怒吼声，低沉到连她的式神都没能听见。因为这些声音，跟幻想乡芸芸众生的惨叫声相比，根本不值一提。

我已经认识到当时的错误了，现在的我只想保护好幻想乡，求求你们，不要把这片最后的乐土从我身边夺走！

就在这时，一只黑色的身影钻入隙间，幻化成人形：“紫大人，妖怪山的山脚下出现了另一个巨大少女！黑白魔法使和茨木大人也在那里。”

“出来了吗……”

听到这个消息的八云紫，用双手托着下巴，尽量挡住自己的苦笑。没错，一切都会和那时一样，一切都会毁灭，拜那个消失了一千年的机器人少女所赐！

“你也是来嘲笑我的吗？黑铁要塞——魔神Z……”

**篇一：无须在意，全员都已准备就绪！**距今一千多年前，那时对于月之都来说不是什么好日子。每年的单数月份对付赫卡特，双数月份跟纯狐勾心斗角。好不容易碰上个闰月，某个高层决定发表重要讲话：“闰月不会有任何敌对势力进攻月都，大家可以好好放个假了！”结果那一月，第一次地月战争爆发了。**八云紫**率领一众妖魔鬼怪攻打月之都，其不择手段的战斗方式让月都士兵节节败退。最终，月之都靠着其强大的科技水平压制住了妖怪大军的攻击，战争也由此陷入胶着状态，转眼经过了八天……面对越来越疲惫地妖怪大军，**八云紫**也十分头疼。用什么办法都无法打破驻守月之都的士兵，就算用隙间潜入内部，也会立刻被月之民发现。为了打破僵局，八云紫叫来了自己当时的亲信，商讨具体解决措施。“紫大人，这场仗我们就不该打。这一切的失败都是因为您的‘芥操’……”**前鬼**说完这句话就被扔进隙间里接受惩罚了，看得一旁的**后鬼**瑟瑟发抖。关键时刻，还是**蓝**比较靠谱：“既然月之都有先进的技术，那我们用更先进的技术不就可以了？”一句话，点醒了**八云紫**。对呀，我们可以找个科技实力雄厚的合作对象拖住月之都，等把月都占下来再回应合作对象的要求。这等于一张空头支票，但**八云紫**很清楚，有些对象陷入的情况比自己还要麻烦，她们不可能拒绝自己的条件。而她看上的合作对象，正在另一个宇宙，名叫“机器人少女”世界里……这个名叫“机器人少女”的宇宙，科技已经发展到了21世纪。这是一个和我们的世界很像的世界。只不过在这个世界的历史上，出现了一种名叫“空中元素固定装置”的超高速打印技术，以及一种名叫“光子力”的无限清洁能源。于是，由光子力的原料打造的超合金Z，在打印技术下成为了“超合金纤维Z”，并被改为少女们的外衣——一种被称为“机器人少女”的新型技术正式登场。而作为技术的开发者，第一个吃螃蟹的人，**兜十藏博士**让自己的孙女“兜Z”成为了第一个机器人少女**“魔神Z”**。当然，每个世界里都有些不安生的家伙：一个自称“**地狱大元帅**”的女孩子在暗中成立了组织“地下帝国”，并制造了一系列改造人：“机械兽少女”，指派她们袭击了**兜十藏博士**。虽然袭击以失败告终，但**兜博士**还是不知去向。之后，**魔神Z**投奔了兜博士的弟子“**弓教授**”，并结识了一众快乐的小伙伴们：有外星人、人工智能、盖塔、新人类、古代生物、超能力者、忍者……并和地下帝国的机械兽们展开了物理层面的愉快交流。具体来说，就是每次见面都要用“光子力射线”犁一遍东京，过分程度仅次于李梅。两个世界发生交集的几分钟前，机械兽**“德比拉X1”**正带着一众飞行机械兽，躲避**魔神Z**的追杀。而在附近空地上，地狱大元帅正和自己的亲信“阿修罗男爵”一起，把机甲“天火P1”改造超时空传送装置。只要成功，自己就可以把机器人少女关进另一个世界，自己就可以趁机夺走光子力进而支配整个世界了。然而人算不如天算，被击中的**德比拉**正好砸进空地中央，把机器人少女们的注意力全吸引回来了。于是机器人少女们和机械兽少女们在空地上爆发了一场混战。这场混战最终“烈阳光弹”的洗礼而画上句号，不过**德比拉**等机械兽少女都闪开了主要杀伤区，只有**盖塔龙**中招了。“你这家伙，知道你做了什么吗？”**魔神Z**一把揪起来被震晕的德比拉，恶狠狠地质问道。“你说什么？躲开烈阳光弹吗？我也是珍惜生命的，那玩意挨一下不死也得掉层皮。”“我是问你为什么要偷拍光子力温泉的女汤池！”“你说那个吗？只是普通的收集情报罢了，这些情报在网上能赚得一笔不小的钱呢！”“那为什么只有**波士**的照片呢？”“她好歹还有五千万粉丝呢，谁像你似的。拍你的照片我们只能自己消化。”“神马？！”听到自己被这么评价，魔神Z的火一下子就到了头顶。她刚打算一拳挥下去，就被一只手拉住了。“那个……地狱大元帅她们要逃跑了。”站在后面的**高巴里安**指了指远处正准备爬起来的大元帅。“而且我们还不知道她们集合在这里的原因……”“管他什么原因，到时候挨个揍飞就是了！”见自己被无视了，**高巴里安**只能探口气，她早就怀疑这些家伙正义的身份了。环顾四周，跟自己一起的**古洛伊萨**被扫下来了、**盖塔龙**在用牙咬放烈阳光弹的家伙、**钢铁吉克**正在挖出被埋起来的路人、其他机器人少女都在控制着机械兽少女……看来能管事的只有自己了。她走到大元帅的跟前，蹲下来温柔地伸出手去。“抱歉让你遇到这种事情……能告诉我你们到底在干什么吗？”**地狱大元帅**一抬头看见是**高巴里安**，整个人吓得像只炸毛的猫一样坐起来了，正好一屁股坐到了自己装置的启动开关上。瞬间，头顶上一阵炸裂，空地上的所有人一齐望向天空——天空竟然像镜子一样的裂开了。从裂缝里不断翻涌着黑紫色的气体，以及无数不可名状的眼睛。“那是什么？”“我们说了太多的mac队笑话，然后亚波人生气了。”“说mac队的笑话跟亚波人有什么关系啊！”在所有的机器人少女和机械兽少女中，只有**地狱大元帅**看着头顶的裂缝，逐渐露出疯狂的笑容：“成功了，妾身终于成功了！连接其他世界的通道被打开了！你们这些家伙，等着被永远地从这个世界抹除掉吧！”**地狱大元帅**一个翻身跳了起来，却发现自己的双脚够不着地面了……其他的机械兽少女们也一样在风暴中越飞越高，逐渐靠近空中的那个裂缝。“大元帅大人，这是怎么回事？”**阿修罗男爵**一边抓着两个机械兽，一边焦急地问道“这种事……这种事妾身也不清楚啊！要被吸进去了！救命啊！”“我们是想救，可是不知为什么飞不动了！”机械兽**赫特霍克M3**喊道。“怎么竟在关键时刻掉链子？”“毕竟我们这些机械兽一个也就花费五千日元。”**卡拉达K7**应声回到。**达布拉斯M2**也应和着“但凡多加一千也不至于一点用都没有……”“可恶啊……”**地狱大元帅**的哭喊声越来越远，连同机械兽少女们一起，逐渐消失在隙间的另一头。隙间也仿佛露出了满意的表情，开始渐渐合拢起来。“别想跑，机械兽们！”**魔神Z**张开翅膀一跃而起，朝着隙间直冲过去。“别冲动，Z酱！”她的同伴马上追上去想要拉住她，但**魔神Z**已经钻进了隙间里。隙间也完全闭合起来，仿佛从未存在过一样。万里无云的天空中，只剩下追上去的**古兰戴萨**的身影。**大魔神**、**维纳斯****A**也赶忙跟上来，却也是一无所获。她们不得不面对一个事实：机械兽少女还有Z酱，已经失踪了。
