# 2

这是漆黑的夜晚。
新来到这个城市的少女在一个已经关门的报亭门口踌躇着。
要不要敲门呢。
“请进”是一位中年男子的声音。
门开了，从外面看不见里面。
少女慢慢地走了进去，门关上。
“有什么可以帮忙的吗？”
中年男子询问道。
“那个，有没有这个城市的地图？”
少女挠了挠自己的耳根，似乎对深夜造访感到不好意思。
“地图，什么地图都有呢。”
“麻烦了。”
男子从一堆打印纸中抽出了一张地图，递给少女：“20元。”
少女付过钱，男子送她出门。
“欢迎下次光临。”
少女的身影消失在暮色的黄昏中，男子的嘴角浮现出了不经意的一丝浅笑。

少女向前走着，夜晚的马路上很空旷，没有一辆车。少女拿着地图，在寻找着，“这里有个旅馆。”少女的手指按在地图的一个地方，实际位置是在她正前方的五百米左右，少女抬起头，有只通天的巨柱挡在前面，而那家旅店，已经被压扁了。“救命啊。”少女尖叫着，手里的地图滑落地面，那巨柱也随之消失。少女的声音又小了下去，“那柱子，很像是手指。”少女看着自己的右手，与刚刚的柱子同一颜色，质感也是相同的。
“这到底，是怎么回事。”
少女再次捡起地图，用手指按了按刚刚的地方，和她所想象的一样，每次她按下去，就会有一根巨大的柱子出现，而且，那就是她的手指，只不过，有近30米的粗细。少女对自己的力量感到莫名的恐惧，她将地图折叠起来，收进包里，坐到路旁的椅子上，对刚刚的事情仍心有余悸。
心跳的好快
……思绪好乱
我到底干什么，我杀了人？
没有，没有。少女看着自己的右手，不相信地摇了摇头。
并没有经过多久。
那地图，到底是什么。
好奇心战胜了恐惧，少女又拿出了那地图，端详着。
“我在这里。”少女不敢按到地图上。
“前面，还有一家旅店，今晚就住在那吧。”
拿着地图，又走进了夜幕。<title>3</title> <link href="../Styles/Style.css" type="text/css" rel="stylesheet">
