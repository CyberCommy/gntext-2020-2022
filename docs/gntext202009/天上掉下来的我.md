# 天上掉下来的我

作者：hiahia

TID：18032

 

# 1

*本帖最後由 hiahia 於 2014-11-2 21:18 編輯*

我住在一个“巨大的世界”里，说是“巨大世界”其实也只是真对我，在我有懂事的时候，就发现我与别人不同，因为我只有10厘米，听我的养母说，我是从天上掉下来的，正好掉在她的床边，她发现我的时候我才4厘米，不用放大镜的话都看不出来我是人还是虫子，她看我这么可爱，所以就收养了我，她是个非常有钱的人，为了我在这个世界里呆的舒服，她花了很多的钱，在学校和各种娱乐场所里都设置了小人专用通道，和专用座位，所以我非常的感激她，不过像她这样的好人本来是可以长命的，不知道为什么，她有一天生了一场大病，她说她感觉自己快不行了，就开始立遗嘱，说把她的全部家产给我，因为她想到我以后一定会有女朋友，怕女朋友欺负我，所以又多说了一句，就是这个钱必须要我本人去取才能取的出来，这样的话她就不担心有人谋财害命了，然后她就这样去世了，看着我的养母就这样去世了，我感觉很伤心，哭了整整一夜，我的朋友来过多次说让我想开一点，还和我说让我去我家对面的舞厅一定会让我的心情好起来的，我当时很好奇想和他一起去来的，不过后来我放弃了，因为我的养母告诉过我那是大人们才能去的地方，我要想去一定要让她陪着才能去，可是她已经去世了啊，我想到着里又有点伤心了，所以就回我的小床上睡觉去了。

过了一个星期后，我把持不住对舞厅好奇，所以早早的起床，走出大门，开着我的电动玩具车，到对面的舞厅去了，到了舞厅门口，我开始找专门为我准备的小人专用门，找到后从小门里进到舞厅内，里面非常的吵，这是我对舞厅的第一印象，我继续往前走，我看到了一个非常高非常小的座位，那肯定就是我的专座了，我开始往上走，走了很长时间才到我的小座位，我累的坐在上面呼呼喘气，我缓过来后，看见在我面前有一个“巨大”的女人在台上跳舞，她美极了，我很想大声说我喜欢她，可是在舞厅里的音乐太吵了，就算我喊破喉咙她也听不见，就在我失望的时候我突然想起来什么，我可以把钱擦在牙签上来回挥舞，她一定会注意到我的(其实这个舞厅里是这样的，所以人拿出钱，舞厅小姐，看上谁就回去拿谁的钱，并且很那个人做爱，当然对于第一次来这的我来说，我并不知道这里有这样的规则）。

我那起我的迷你电话给我的保姆打了个电话，告诉她给我拿一个牙签，和一个100元美金钞票（有钱人嘛，肯定有保姆），打完电话后很快保姆就把东西带来了，然后我开始准备，当我整好美元钞票旗子的时候，整好舞厅那个漂亮女人跳完舞了，我就开始使劲的挥舞那个美元旗子，我看见漂亮女人扫视台下的时候，她看到了我，她的扫视在我这停了下来，她走下台后，就往我这边走，走到我面前后，我刚想自我介绍一下，可是她突然把我我挥舞的美金拿走了，并且把我握在手里，另一只手则掀开她的内裤，她把我提到内裤口的正上方后，握这我身体的那个手就松开了，我直接掉到了她的内裤里，之后，她也松开了掀开内裤的那只手，我的全身一下子处于一片黑暗之中，我的周围有一股很骚的味道，难道女人的内裤里都有骚味吗？不过这骚味让我全身火辣辣，我的小弟弟也不知道在什么时候站了起来，突然外面有东西顶着我的后背，让我贴在了那个漂亮女人的阴唇上，我感觉到了，她的阴唇已经湿透了，阴唇里面的洞口好像在欢迎我似的，来回收缩，外面的压力也越来越大，我感觉我的头顶在了阴唇里的阴道口上，骚臭味开始越来越浓，我头顶上的阴道口蠕动的越来越快，我感觉她的阴道已经把我的眼睛和鼻子吸进去了，外面的压力好像有大了，我感觉有东西在顶我的屁股，让我更快的被她的阴道吞噬，她的很快她的阴道已经把我的腰部吞了进去，我的上半身在她的阴道里被她阴道里那淫乱的肉壁来回的侵犯，我感觉非常的舒服，可是就是氧气太少了，我现在只能呼吸她阴道内骚臭的气息，慢慢的我的意识开始模糊，在我昏迷之前，我感觉我的屁股也已经被她的阴道吞噬，然后我就昏了过去。（待续。。）
