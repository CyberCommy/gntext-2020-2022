# 我的危險搭檔(繁體)

作者：rockzero

TID：1127

 

# 1

“公民方雷，這個城市已經宣布進入緊急狀態，請配合我們的工作，謝謝！”就在我剛剛收到政府廣播的戒嚴令之後不久，一個警察敲開了我的家門。

我一個平頭百姓不想也沒有那個能力來惹麻煩，所以我簡單的收拾了一下行李，很配合得登上了那個警察開來的大型懸浮車，他會送我到緊急避難所。然後在那裡我會分到一個平衡倉，在平衡倉中我可以接入虛擬現實網絡，在虛擬城市中通過虛擬現實界面工作。這讓我想起了當年地球時代時的一部叫《黑客帝國》的電影。這是政府為了減少能源消耗的一個很重要的措施，當然，也不僅僅是為了減少能源消耗。

無論是哪個時代，在人群聚集的地方總是會流傳著一些亂七八糟的八卦、新聞，還有小道消息，無論是當年的茶館，酒館，互聯網……還是現在的虛擬城市。這種時候議論最多的話題當然就是政府的那個看起來異想天開的“危險搭檔”計劃了。

比方說虛擬城市的某個論壇上就有一幫人在爭論到底該不該佔領那個近在咫尺的“斯卓”星。贊成的說只有留在這個星域才有可能重新和星際聯盟取得聯系，而且這個星球資源儲量極為豐富，足以支持到星際聯盟發射的超時空探針的到達。至于怎樣佔領，政府不是正在實施一個叫“危險搭檔”的計劃嗎？

至于反對的則對此嗤之以鼻，先不說別的，單說培養出巨人來了怎麼把它運到那個星球上去？現在奈吉星上最大的飛船也那些土著人的巴掌大小。更何況，奈吉星上哪有那麼多資源供養得起這麼個大家伙。就算退一萬步說，把這個大家伙送到“斯卓”上去了，它就一個“人”，兩雙手，用句老話說，就算它是混身是鐵又能打幾個釘。那裡可是有土著文明的啊！土著們的城市裡有些地方，就算是我們的探測器也看不到裡面的情況。天知道這些土著們有什麼稀奇古怪的本事，歷史上星際聯盟因為土著文明陰溝裡翻的船還不算多嗎？政府這個所謂的“危險搭檔”計劃純粹是在揮霍民眾節衣縮食省出來的能源！我們應該搭乘飛船太空移民，憑我們的科技，找個星球活下去根本不是什麼問題。

這時正方又說了，太空移民？真是站著說話不腰疼，你以為政府沒有進行天體探測嗎？探測出來的結果是這一片星域內的天體質量都大得嚇人，這個奈吉星只能算是個小石頭，我們的飛船可能勉強能算得上是星際塵埃了。在茫茫的宇宙中進行超時空跳躍？腦子進水了嗎？沒有坐標哪個敢亂來？胡亂跳躍的結果只會跳進地獄。

我看著兩幫人吵得唾沫亂飛，一會覺得這邊說得對，一會覺得那邊也有道理，最後弄得一腦子漿糊，晃晃腦袋離開了論壇。這種高深的問題像我這種笨蛋是想不出結果的，還是讓那些高人去消耗腦細胞吧。

我晃了晃腦袋離開了論壇，打開了我的工作界面，檢查了一下數據，像往常一樣指揮我的那些機器人們干活去了。

像這種幾乎所有的人都被或友好或強制的送進緊急避難中心的平衡倉的時候，城市中除了某些特殊的崗位一定要有人值班外，從某種意義上可以說，這已經是座死城了。我以前是一家機器人公司的售後服務人員，現在接受了政府的僱佣，我的主要工作就是指揮著四五個機器人輸送能源，但我更重要的工作則是監視這些機器人的數據，將智能突變扼殺在萌芽狀態，現在的奈吉星根本承受不了一次新的機器人叛亂。

過了大概十萬周的時間，政府宣布“危險搭檔”進入正試的實施階段，第一個“戰天使”出世了。政府一共培養了數十個“戰天使”。她的基因綜合素質在所有的“戰天使”當中是最優秀的。與人們想象中的不同，這個剛剛出世“戰天使”身高還不到10厘米，盡管她身體的平均密度比正常人少十分之一，卻可以舉起相當于自身體重七倍的重物，身體綜合素質類比接近B級體系能力者。她的基因素質在所有的“戰天使”中是最優秀的。最重要的是，她的身體沒有普通人體型的限制，只要給予足夠的營養，她可以近乎無限的生長。預計在她身高達到千米階段時投放到“斯卓”星，這已經是奈吉的極限運載能力了。雖說體型相對土著人來說實在是太小了，但體型小不容易被發現，依然可以做很多事情。

這麼大的家伙如果鬧起脾氣來可不得了。為了“把風險降到最低”政府使用了來自帕蘭星的“心語”技術。這是一種不可思議的技術，星際聯盟的科學家們至今也沒有成功的解析出它的秘密。

“心語”技術更像是一種神奇的魔法。她可以讓兩個智慧生命的心靈聯系在一起，這種聯系是超越空間的，哪怕相距再遠，雙方都可以聽到彼此的聲音，分享彼此的感受。而且可以將“心語”雙方的能力放大到不可思議的地步（當年聯盟艦隊就吃過“心語者”的虧），“心語者”的雙方的能力越接近，增幅效果也就越好。但是，“心語”也是有弱點的，最明顯莫過于“心語者”的雙方任何一方死亡，另一方也會跟著死去。

僅僅是憑著“心語者”共享死亡和“心語者”可以彼此隨時交流這兩點，就足以讓政府選擇這種神奇的魔法了。

不過不是什麼人都可以成為“心語者”的，“心語者”的雙方必須都是有感情的思想單純的並且至少有一方身體發育成熟的碳基智慧生命。剛出生的嬰兒理論上也是可以成為“心語者”的，成年人與嬰兒的搭檔也是可以成立的，在這種情況下，嬰兒通過吸收成年人的記憶來進行心智上的成長，心智成長過快往往會造成心智與生理上的脫節，這樣的組合往往只有不到300兆周的生命。而且相對于成年人的組合來說，這樣這種組合實力往往相對較差，所以這種組合方式也漸漸的被遺忘了。現在很明顯，政府正是採用這種被遺忘的組合方式來控制那些“戰天使”們。

心思單純的人，在這種競爭激烈的社會早就很難找了，盡管相對于星系聯盟那龐大的人口基數來說，這種人的絕對數量還算不少的。不過，在奈吉星上，這樣的人還是不多見。

就在我又一次檢驗數據的時候，工作界面上的外界呼叫指示器響了起來，我趕緊讓那幾個機器人放下手中的活，停機離線。再粗略的校驗了一下數據，斷開了和計算中心的連接。

平衡倉蓋打開了，映入我眼帘的是凱因大哥那胖乎乎的圓臉還有那漂亮的何雨姿姐姐。

凱因大哥微笑著說：“方雷，恭喜你成為第二個幸運兒。”

附注：兆周、千周、周：由于在太空時代，各星球自轉公轉周期各不相同，並且偶爾會出現奈吉星這種意外變更軌道的情況，通常的時間單位年、月、日已經不適用了，故採用“銀河標准時間周期”簡稱“周”這種時間單位。1周約等于1秒，一年大約為31.536兆周。
