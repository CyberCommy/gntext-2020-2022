# 1

2012.3.26
这两天一直在忐忑中度过,今天白天和领导请了三天假,还好我和院长有着点亲戚关系,不然刚工

作就请假好象还是挺难的.
到了晚上刚吃过晚饭敲门声就响起了,打开门一看黄瑛的妈妈已经站在门口了.她进来后就问

我,'小郑,你考虑得着么样了呀?不过首先要提醒你这存在一定的风险的哦.但如果你通过了第一

关,收益也是非常大的.'
'哦?着么说呢?'我好奇的问道.
'小郑,你也可以说是学医的,应该知道干细胞吧.'
'恩,当然.干细胞是人体内的一种特殊的细胞,可以变成我们任何一种细胞.'
'那就好,其实我们家祖祖辈辈都在进行人体的研究.既然你已经答应了,那我就把你要遇到的问

题告诉你.首先我会抽取你一部分的干细胞,放在我带来的培养液里,利用我特有的技术,可以完

全再制造一个你出来...'
'那不就等于有两个我了吗?那...'
'你先别打断,听我说.虽然我们制造出了另个你,但是这个你只有非常微弱的意识.我会把他弄到

...弄到黄瑛平时上的厕所下面.你要知道我们家厕所有两个,所以我和她父亲平时用一个,她自

己用专门的一个.而那个厕所下面有一段中空的,我们会在明天白天把另个你放在下面.然后我会

把你缩小到你前两天看到的小人的大小,放到药丸里面.明天就是黄瑛吃药丸的日子了,我们会把

你给她吃了.然后她一般每天早晨起床后会在家里大便,当然明天一早另个你还没到厕所下面去.

但是现在的你就会被她吞下去,被她消化吸收...'
'那我不就死了吗?'
'可以这么说,但也不一定对.首先我要问你,你是什么?你是肉体?还是意识?如果我把你所有的记

忆,习惯,性格等完全复制到另个身体你,你说你还是活着的还是死了呢?'
'这....这我也说不知道.'
'是呀,这没人知道.但对于我们其他人来说,如果我把你所有的一切记忆习惯性格等复制到另个

你的身体里,你就还是活着的.'
'那着么复制呢?'
'这就是关键所在,也是你会遇到危险的地方.到明天晚上的时候,现在的你已经被完全消化了,而

另个用干细胞做出来的你还只有微弱的意识,并没有你的记忆等.那时另个你会躺在黄瑛的厕所

下面,等星期三一早黄瑛上厕所大便的时候,把你变成的大便拉到用干细胞做出来的你的嘴里的

时候,你的所有一切就都会恢复.而我们会对另个你进行身体适当的改造,你会比现在更聪明,更

强健...但是如果她星期三并没有在家里上厕所或者遇到其他什么意外的话...那你的另个身体

就会在星期四被我再次缩小放进药丸,然后在将来的某一天再次被我女儿吃掉,那时你就彻底的

失败了.'
'你的意思是我死了?'
'对的.'她点了点头说.
如果我被黄瑛吃了,而由我干细胞变成的我再次获得我所有的记忆等,那我还是我吗?我也很好奇

这个问题.
'对了,以前...黄瑛有过男朋友吗?我的意思有别人经历过这种考验吗?'
'呵呵,当然有.以前有过两个,可惜都失败了.一个就是在被她吞了后第二天她们公司正好体检,

另个就是被你发现的那具骷髅.不过你放心,如果失败的话,我们会对黄瑛的记忆进行处理,她会

彻底忘记你,她以前两个准男友就是,她一点印象都没有了.'
'对了,你的意思是...我还要吃黄瑛的大便吗?那她明天回家上厕所小便我也都要喝了?'
'那当然,你不愿意吗?'顿时伯母瞪了我一眼.
'不...没有没有...'虽然我觉得恶心,但是和自己的性命相比,也就没什么了.而且黄瑛着么说也

是一等一的美女,而且将来可能成为我的老婆.
'额....这个....'虽然有点犹豫,但是一开始我都答应了,而现在伯母已经说到了这种程度,如果

我不答应的话她还会放过我吗?
'那...那好吧.现在就开始吗?要我准备点什么吗?'
'不用,随时都可以开始.'
'那就开始吧.'我说完她就拿出了早已准备好的设备,抽取了我骨髓中的干细胞,然后就放到了她

那所谓的培养液中.
'好了,接下来我该把你变小了...'
'等等...我能看到另个我完全出来后再变吗?'
'好吧.反正另个你我已经放到了你的床上,大概两小时后就能完全成型.那我过两小时再来吧.别

想着逃跑哦,都到这步了,如果你是聪明人应该知道着么做了.'
'恩,当然.'我紧张的回答
于是她就先回去了,我一个人留在房间里就写着日记.现在两个小时已经到了,我已经完全看到另

个和我完全一模一样的人赤裸的躺在我的床上,一动不动.但是能感觉到有生命的存在,因为我能

感觉到他微微起伏的呼吸.那就是另个我吗?如果我死了,我真的会变成他吗?那时我还是我吗?会

成功吗?虽然今天天很好,但我内心却波涛汹涌.也许...今天是我最后一天写日记了,明天这个时

候我就应该在黄瑛的肚子里,已经被完全消化完了.
Good Lucky! 祝福自己,也祝福此时躺在床上的另个自己!
