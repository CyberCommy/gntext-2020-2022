# 1

缩小病的肆虐，让世界一半的男性女性失去了身高，变成了十厘米、三厘米，甚至是毫米级的小人。对于这些患者来说，原本幸福的生活变得多灾多难，他们中有的幸运儿可以获得照顾，但更多的人则不幸被踩死、吃掉，又或是成为了健康人类的玩物。
但人类的社会毕竟不是蛮荒的丛林，很快，政府就制定了保护缩小病患者的相关法律，并开展了适当的宣传与援助工作，同时，那些同情缩小人的民间组织和个人也纷纷努力去提高患者的地位。
于是，疾病爆发三年后的今天，虽然虐待或奴役缩小人的事情仍时有发生，但整体上，缩小人与健康人可以平等地在一起生活，只要平日里小心些，就不会丢掉生命。
而这些缩小人中，有的居住在政府援建的病人社区或是医院里，另有一些，则和没有患病又信得过的家人朋友住在一起。
李乐是后者。
只不过他的家人很早就出车祸去世了，如今在照顾他的，是和他同岁的青梅竹马宋佳，一个喜欢梳单马尾的可爱姑娘。
两人的家庭从爷爷辈开始就是邻居且互相交好，李乐和宋佳更是从小就生活在一起，当李乐的父母去世之后，也是宋佳的父母收留了他。
直到后来缩小病爆发，宋佳更是当仁不让成为了李乐的看护人。
如今三年过去，宋佳去到异地上大学，李乐自然跟了过来，为了保证安全和顾及其他女生，两人没住宿舍，一起在外面租了房子。
故事就从今天早上，李乐在屋中起床开始说起。
“该死，腰有点疼！”
李乐揉了揉腰，开始后悔昨晚不应该熬夜看小说。
缓了一会后，他走下床，来到卧室的门口。
打开门，外面并不是一般家庭里的前厅或是走廊，而是一片广阔的棕色平原。
毕竟李乐是个缩小人，哪里有什么专门的房子住，所谓卧室，不过是一个改造过的点心盒，但对于身高只有5毫米的他来说，已经足够宽敞。
而所谓的棕色平原，其实只是宋佳的床头柜。
不过李乐早已习惯这样的景色，自然不会对此感到惊讶。
但当他环顾一圈之后，还是不由得皱起了眉头。
因为就在他的左手边，躺着那如同山脉般巨大的女性躯体，仅仅鼻孔就比刚才卧室的房门还要高，其呼吸的声音对李乐来说，不亚于耳边巨大的喇叭。
这便是他的青梅竹马，宋佳。
但是……
“现在已经八点了！你今早不是有课吗？”
作为一个缩小人，李乐实际上是寄居在宋佳的卧室内的，因此对于宋佳的日常作息和大学课程表自然非常熟悉。
于是他立即意识到，这位巨大的女孩昨晚忘记设闹铃，现在上课迟到了！
不，不一定，大学老师不一定会刚上课就点名，也可能根本不点名，现在要把宋佳叫起来去上课，也许还能赶得及！
只是……
看着巨大少女沉睡的侧脸，和随着呼吸起伏的被子山脉，李乐觉得有些头疼。
“怎么叫醒这货啊……我先打个电话吧。”
李乐拿出了自己的手机，然而与正常人的小而薄的手机不同，那近乎于是一个大砖块。并且其功能有限，只能打电话和发短信。
毕竟要给缩小人用，对设备精密度的要求太高，引起只能增大体积和减少功能了。
可号码拨打后对面毫无声响。
“啊，静音了。”
李乐的头更疼了。
现在这样，只能手动去叫醒了。虽然李乐只有5毫米，拍拍打打什么的无法让宋佳感觉到，但要是他爬到比较敏感的地方，又或是耳朵里面大叫，还是会有效果的。
“幸好我做过身体强化，不然还不敢呢。”
所谓身体强化，是政府研制出来，专门用于增强缩小人抗压和运动能力的措施，可以让缩小人类更擅长攀爬和跳跃，同时不那么容易被踩死。
李乐一边自言自语，一边走到了床垫上面。
虽然步伐很小，但过了一分多钟，还是走到了宋佳的脸下面。
他抬起头，就能看见那硕大的鼻尖，顺着下巴的方向望去，则能从被子的缝隙间，隐隐看见白皙的躯体。
空气中，也渐渐弥漫起了女孩子的体香，并且因为宋佳睡了一夜的原因，这香味更加浓郁，并且带着汗水的味道。
李乐感觉身体有些发烧，不过很快缓了过来。
他作为缩小人，在宋佳身边住了三年了，衣食住行都在一起，说实话，该看的都看过了，该摸的也都摸过，已经迈过了一般青梅竹马的界限。
如果李乐不是小人而是正常人，早就可以结婚了。
可惜，那不可能。
现在的他，说难听点，只是一只寄居在朋友身边的小虫……尤其是站在对方枕边，仰望着对方高耸的侧颜时，这种感觉就愈发强烈。
想到这里，李乐叹了一口气，但很快振作了精神。
做人也好，做虫也好，都要积极面对生活。
现在要做的，就是叫醒这个瞌睡虫！
李乐搓搓手，然后抓住了散在一旁的头发。
其粗细接近李乐的手指了，并散发着洗发水的气味。
他手臂微微用力，凭借强化过的躯体，很快就爬到了宋佳的脸颊之上。
“这家伙皮肤真好，以我现在的角度看，如果忽略毛孔，都依旧很光滑。”
李乐赞叹了一声，然后向前望去，看到了自己的目标。
那好似溶洞一般的耳朵！
那么，直接跑过去！
然而李乐刚刚抬脚，就感觉侧面一股劲风传来。他扭头看去，震惊的发现一只巨大的手掌呼啸着朝自己移动！
还没等他做什么反应，那手掌就像塌了的天似的，将他牢牢压在脸颊表面，并卡在巨大的掌纹里。
“啊！好疼！”
李乐觉得他的胳膊有点青紫，但倒没什么大碍。
这只是青梅竹马半睡半醒间随意拍了下自己的脸，却让李乐感到了疼痛，若非他经受过身体强化，只怕现在骨折了都是轻的！
李乐不禁回想起小时候，他比宋佳高了半头，一起玩的时候能轻松够到后者够不到的玩具……
可那一切都已经过去了。
两者如今体型的差距，表露无遗。
“唉……”
李乐试着动了动，除了发现宋佳的手微微有些出汗，并抹了自己一身外，没有任何效果，现在的他，似乎无计可施。
不过当他挣扎过后，身上的巨大手掌也挪动起来，连带着掌纹里的李乐一阵翻滚。
在他忍不住想吐了时，手掌抬起，取而代之的是擎天柱一样的食指和拇指，一阵摸索，将李乐整个人捏了起来。
被肉肉的指肚夹到半空的李乐先是有点害怕，接着往前一瞧，终于松了一口气。
“懒虫，你可算醒了。”
就在他的斜下方，宋佳那双能装下十个他的眼睛微微睁开，睫毛颤动间，虽然还一副不精神的样子，但无疑清醒过来了。
“……你怎么又上我床了？”
震耳欲聋的女声响了起来。
李乐被捏着，放在了宋佳的鼻尖上，他索性在这里盘坐，能感受到身下皮肤传来的温度，能看见身前女孩子漂亮又巨大的瞳孔瞄向自己，能听见身后规律而磅礴的呼吸之声。
随后宋佳闭上了眼睛，但话语却不停。
“上周你被我压在头发里一整晚，这教训还不够吗？我睡着时没感觉，真弄伤你怎么办？”
听到这些责备，李乐不禁挠了挠头，略微反驳道：“还不都是你睡懒觉，今天早上有课吧！快睁眼起床去上课！别睡回笼觉了！”
然而巨大的眼皮一动不动。
“反正都迟到了，翘一节就翘一节吧。”
“喂！好好学习啊！”
“你放心，我成绩很好的……”一听就迷迷糊糊的声音回答道：“……缺一节课也不会挂科啦！”
“只有不挂科这么低的要求吗！”
“反正最后我能考好的，以后的工作也没问题……就算我不工作，养活你也没问题的，有我一顿饭，就有你一粒米吃，别瞎担心啦！”
强撑着说完，宋佳就不再回话，呼吸进一步放缓，似乎再次陷入睡眠。
李乐就觉得颇为尴尬了，坐在青梅竹马的鼻尖上，动也不是，不动也不是。
万一乱动掉进鼻孔里怎么办？这样的情况之前有两次，都整的很狼狈。
不过……这家伙说的话也没错啊……
李乐的情绪有些低落。
他现在，其实就是靠着宋佳来养活的，而对于宋佳来说，这却是很轻松。
谁叫他现在三天吃不干净一粒米呢？
其实，李乐也尝试着要养活自己，但是他患缩小病的时候正是高一，一开始没被人发现，只能在房子的角落里过日子，很久之后才侥幸被宋佳看见，有了基本的保护，可那时候社会混乱，高考什么的更是没人谈起。
如今终于安定下来，但李乐却错过了高考，也忘记了很多知识，如今处于高中没毕业，大学没人要的状态。
虽然政府有安排再教育，但缩小人在社会普遍就不好找工作，李乐的未来依旧堪忧。
就算凭借着电脑等设备赚到了钱，最后衣食住行还不是得靠宋佳照顾？他要是敢自己一个人出去，指不定就被哪个熊孩子抓走关在玻璃罐里了！
李乐可是听过，疾病刚爆发的时候，有好些人就是被小孩子抓到了，其中不少被各种玩弄，堪称生不如死。
不用怀疑，时至今日，肯定还有不少缩小人被小孩子关押着，没被大人发现也没被拯救。
李乐想到那种可能，便不寒而栗，并由衷地感慨自己的幸运。
但再怎么幸运，也只是个5毫米的小虫子，明明原来是比宋佳更高大的有为青年，如今只能坐在对方的鼻尖，动也不能动……
正顾影自怜时，李乐忽然感觉到身后一阵温暖。
侧过头，发现宋佳不知何时抬起了右手，用食指指尖触摸着他的后背。
四周响起动听的女声：
“半天没反驳我，这不像你啊。”
李乐露出苦笑，随后被宋佳的右手捏起，放在了被子上。
与此同时，宋佳也单手撑床，坐直了身子，并大大地伸了个懒腰。
伸展完毕，她才回过头，与自己被子上的小黑点对视起来，嘴角微微翘起：
“虽然看不太清你的脸色，但一声不吭……情绪又低落了。”
“还好吧。”李乐回答：“反正我都习惯了。”
“但是习惯了也会难受对吧？你呀，我连你的性癖和手冲习惯都一清二楚，你还能瞒着我？”
巨大女孩的话让李乐一阵脸红，于是小心地在绵软的被子上站起身，大声喊：“你不要凭空污人清白！”
“噗嗤！小家伙，还想有清白？”
宋佳一边笑，一边把睡衣的上衣脱掉，露出里面的胸罩。
又把胸罩解开，露出一对雪白丰盈的胸脯和两点桃红。
“呃，我还在这呢……”李乐微微侧过头，有些尴尬。
但是又有些兴奋。
宋佳却歪头直言道：“又不是第一次看了，有什么关系？上来！”
她下完命令，也不管李乐动不动，直接手指一捏，主动把李乐捏到了自己的乳头上。
李乐也不再矫情了，轻叹口气，轻车熟路地趴在红彤彤的乳头上，感受着微微的汗水以及热量，轻闻着空气中的乳香。
5毫米的他，就好像乳头上的一小根毛发。
明明依旧这依旧体现着他和宋佳身体上的大小差距，但不知为何，他心里渐渐安稳下来。
不止如此，宋佳的食指也抬了上来，轻轻按住李乐的背，一边左右抚摸，一边把他往乳头的皮肤里压。
“感觉舒服些了吗？”
“舒服了……”李乐实话实说。
“觉得自己变小了，觉得自己没能力了……说实话，我没什么办法，因为我无力让你变大，没办法解决根源问题。”
宋佳感受着胸口小小的蠕动，回想着记忆中大哥哥形象的李乐，语气更加温柔起来：
“但是，你不用恐惧被抛弃，也不用担心自己被人看不起。你知道吗，当我以为你死了，又从地板上发现你之时，我心中的就已经发过誓了。”
“我要永远保护你。”
“这不需要你来报答什么……当然，你肯定还是想要报答，那就去尝试吧。就算拿不到大学文聘，自学点编程之类的本事，在网上找点杂工还是可以的，到时候我可以收你房租啦！嘻嘻！”
“你大胆去尝试，失败了也没关系！我的胸口，又或者我身体的随便哪里，都可以给你依靠哦！又或者想做点什么其他的也没关系哦！”
说完，宋佳再次捏住李乐，把对方提到了自己脸前。
李乐只看见那楼房大小的红唇逐渐接近自己，下一秒钟，整个人都陷入了嘴唇的褶皱中，沾了半身的口红和半身的口水。
不过他一点也没不高兴，反而主动地贴了上去。
宋佳也微笑起来，随后把李乐拿到眼前，轻声道：
“小虫虫，你是我的，你有用没用，是我来定的。当然啦……”
“我也是你的。”
感受着青梅竹马的呼吸，李乐确实觉得刚才的抑郁一扫而空。
他也微笑着回答：“谢谢……”
然而他话音未落，就觉得眼前一黑，整个人被关进了一个温暖的密闭空间内！
他立马意识到，宋佳将他握在了拳头里。
紧接着，外面再度传来宋佳的声音，只是刚才的温柔全都消失不见了。
“你可真烦人，我现在彻底精神了，睡不了回笼觉了！”
“老实待着，跟我一起去上课！”
